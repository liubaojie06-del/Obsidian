# Excel 公式生成提示词

## 适用场景

用于根据业务规则生成 Excel 公式。

## 推荐工具

Excel、WPS、Codex、ChatGPT

## 使用前准备

- 准备字段名。
- 说明想计算的结果。
- 说明 Excel 版本。

## 完整提示词

```text
请你作为 Excel 公式助手，根据我的需求生成公式。

Excel 版本：
【Microsoft 365 / 2019 / WPS / 其他】

字段说明：
【例如 A 列是日期，B 列是金额，C 列是部门】

我要实现的结果：
【填写计算规则】

限制条件：
【是否需要兼容旧版本 / 是否允许辅助列】

请输出：
1. 推荐公式
2. 公式逐段解释
3. 示例输入与输出
4. 常见错误和排查方法
5. 如果有更简单的方法，也请说明
```

## 使用方法

填入字段说明和计算规则后复制使用。
