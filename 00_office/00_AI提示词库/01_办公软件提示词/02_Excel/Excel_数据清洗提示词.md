# Excel 数据清洗提示词

## 适用场景

用于处理重复数据、空值、日期格式、字段拆分、文本清理等 Excel 数据问题。

## 推荐工具

Excel、WPS、Codex、ChatGPT

## 使用前准备

- 准备表头说明。
- 说明数据问题。
- 粘贴部分样例数据。

## 完整提示词

```text
请你作为 Excel 数据清洗助手，帮我分析并清洗下面的数据。

表格用途：
【填写用途】

当前问题：
【重复值 / 空值 / 日期格式混乱 / 文本多余空格 / 字段需要拆分】

表头说明：
【填写字段含义】

样例数据：
【粘贴几行数据】

请输出：
1. 数据问题清单
2. 清洗步骤
3. 可直接使用的 Excel 公式
4. 如适合 Power Query，请给出操作步骤
5. 清洗后的样例结果
```

## 使用方法

替换样例数据和问题描述后使用。
