# Excel 表格美化提示词

## 适用场景

用于优化 Excel 表格的可读性、颜色、边框、冻结窗格和打印效果。

## 推荐工具

Excel、WPS、Codex、ChatGPT

## 使用前准备

- 准备表格用途。
- 提供列名和表格规模。
- 说明是否用于打印或汇报。

## 完整提示词

```text
请你作为 Excel 表格美化助手，帮我设计一套清晰、专业、适合办公场景的表格样式。

表格用途：
【填写用途】

列名：
【填写列名】

使用场景：
【屏幕查看 / 打印 / 汇报截图 / 数据录入】

请输出：
1. 表头样式
2. 字体、字号、行高、列宽建议
3. 边框和底色建议
4. 条件格式建议
5. 冻结窗格、筛选、打印区域设置
```

## 使用方法

替换表格用途和列名后使用。
