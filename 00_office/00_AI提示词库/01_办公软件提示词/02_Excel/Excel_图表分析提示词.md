# Excel 图表分析提示词

## 适用场景

用于分析数据并选择合适的图表类型。

## 推荐工具

Excel、WPS、Codex、ChatGPT

## 使用前准备

- 准备数据样例。
- 说明分析目的。
- 说明报告对象。

## 完整提示词

```text
请你作为 Excel 数据分析与图表设计助手，帮我分析下面的数据并推荐图表。

分析目的：
【趋势分析 / 对比分析 / 占比分析 / 异常发现】

目标读者：
【领导 / 客户 / 团队 / 自己查看】

数据样例：
【粘贴表头和部分数据】

请输出：
1. 数据中值得关注的结论
2. 推荐的图表类型
3. 每张图表的横轴、纵轴、图例设置
4. 图表标题和说明文字
5. 在 Excel 中制作图表的步骤
```

## 使用方法

把数据样例和分析目的替换进去。
