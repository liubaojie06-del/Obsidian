# PPT 高级视觉提示词

## 适用场景

用于制作更有设计感、更适合正式展示的可编辑 PPT，例如产品介绍、方案汇报、课程展示、品牌展示。

## 推荐工具

Codex、PowerPoint、`$presentations skill`

## 使用前准备

- 准备品牌色、Logo 或参考风格。
- 准备内容资料。
- 说明希望呈现的气质，例如高级、科技、专业、清爽。
- 如有参考图，可一起提供。

## 完整提示词

```text
请使用 $presentations skill 帮我制作一份高级视觉风格的可编辑 PowerPoint PPTX。

核心目标：
把我的内容整理成一份视觉更高级、结构更清晰、适合正式展示的 PPT。

硬性要求：
1. 输出必须是可编辑 PPTX。
2. 不要整页图片，不要把文字压进截图或背景图里。
3. 所有文字必须是 PowerPoint 可编辑文本框。
4. 流程图、架构图、时间线、步骤图尽量使用 PowerPoint 原生形状和连接线。
5. 表格必须用 PowerPoint 原生表格。
6. 图片只能作为辅助视觉元素，不能替代可编辑内容。
7. 配色要克制，不要花哨；版面要有留白、对齐、层级和节奏。
8. 完成后必须做排版 QA 检查，重点检查文字溢出、元素遮挡、对齐问题、配色一致性和页面风格统一性。

工作流程：
第一步：先输出视觉方案和完整大纲。
- 包括推荐配色
- 字体层级
- 页面结构
- 每页标题与内容重点
- 建议使用的图表、流程图或信息图

第二步：先生成 1 页样张。
- 样张必须体现最终视觉风格
- 样张中文字和形状都要可编辑
- 样张不能是整页图片

第三步：等待我确认样张。

第四步：确认后再生成完整 PPTX。

第五步：对完整 PPT 做排版 QA。
- 检查文字是否溢出
- 检查是否有遮挡
- 检查图形和文本是否对齐
- 检查页面边距和留白
- 检查字体、字号、颜色是否统一
- 修复问题后再给最终文件

PPT 主题：
【填写主题】

使用场景：
【产品介绍 / 商务汇报 / 课程展示 / 品牌介绍 / 其他】

目标听众：
【填写听众】

页数要求：
【填写页数】

视觉偏好：
【高级简约 / 科技蓝 / 黑白极简 / 清爽办公 / 品牌风格】

品牌或配色要求：
【填写品牌色、Logo、禁用颜色等】

资料内容：
【粘贴资料】

请先输出视觉方案和 PPT 大纲，不要直接生成完整 PPT。
```

## 使用方法

适合对视觉质量要求较高的 PPT。先让 Codex 输出视觉方案和大纲，再确认 1 页样张，最后生成完整 PPTX。
