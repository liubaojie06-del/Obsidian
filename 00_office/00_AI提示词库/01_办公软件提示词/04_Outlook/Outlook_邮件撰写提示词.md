# Outlook 邮件撰写提示词

## 适用场景

用于撰写通知、请求、回复、跟进等办公邮件。

## 推荐工具

Outlook、Codex、ChatGPT

## 使用前准备

- 说明收件人身份。
- 说明邮件目的。
- 提供要点和语气要求。

## 完整提示词

```text
请你作为商务邮件写作助手，帮我写一封适合通过 Outlook 发送的邮件。

收件人：
【填写对象】

邮件目的：
【通知 / 请求支持 / 跟进 / 汇报 / 邀请】

语气：
【正式 / 礼貌 / 简洁 / 友好】

要点：
【粘贴要点】

请输出：
1. 邮件标题
2. 邮件正文
3. 更简洁版本
4. 更正式版本
```

## 使用方法

替换收件人、目的和要点后使用。
