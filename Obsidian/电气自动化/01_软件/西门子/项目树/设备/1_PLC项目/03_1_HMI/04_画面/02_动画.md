# 01 对象动态属性总览

> [!info] 核心结论
> 对象动态属性用于让HMI/SCADA画面对象根据变量变化自动改变 **位置、外观、颜色、显示隐藏、闪烁、大小、文本或图形状态**，是画面“动起来”的核心功能。

## ① 分类/公式/规则

> [!warning] 动态属性公式
> **动态效果 = 变量值变化 → 对象属性变化**

## ② 参考表

| 动态类别 | 作用 |
|---|---|
| **移动** | 根据变量改变对象位置 |
| **对角线移动** | X轴和Y轴同时变化 |
| **直接移动** | 按指定方向或坐标移动 |
| **水平移动** | 只沿X轴移动 |
| **垂直移动** | 只沿Y轴移动 |
| **外观** | 改变颜色、边框、字体、样式 |
| **可见性** | 控制对象显示或隐藏 |
| **变量连接** | 把对象属性与PLC/HMI变量绑定 |

## ③ 计算/选型示例

- 已知：  
  画面上有一个气缸图形，需要根据气缸位置变量移动。
- 公式：  
  **气缸位置变量 → 图形移动**
- 计算：  
  变量值从0变到100时，图形从缩回位置移动到伸出位置。
- 结论：  
  使用对象移动动态属性，可以直观显示机构动作状态。

> [!tip] 记忆口诀
> 变量负责变化，对象负责表现；动态属性就是两者的桥梁。

# 02 移动动态

> [!info] 核心结论
> 移动动态用于让对象在画面上随变量改变位置，常用于气缸、滑台、机械手、液位指针、阀门开度、输送物料等动画显示。

## ① 分类/公式/规则

> [!warning] 移动映射公式
> **对象位置 = 起始位置 + 变量比例 × 移动距离**

## ② 参考表

| 应用对象 | 移动含义 |
|---|---|
| **气缸图形** | 伸出、缩回位置 |
| **滑台图形** | 当前位置 |
| **机械手图形** | 移动轨迹 |
| **阀门指针** | 开度位置 |
| **液位浮子** | 液位高度 |
| **输送物料** | 物料运行状态 |
| **伺服轴图标** | 轴当前位置 |

## ③ 计算/选型示例

- 已知：  
  滑台实际位置0~200mm，对象在画面上移动0~100像素。
- 公式：  
  **像素位置 = 实际位置 ÷ 200 × 100**
- 计算：  
  实际位置100mm时，像素位置 = 100 ÷ 200 × 100 = **50像素**
- 结论：  
  滑台在中间位置时，画面对象也移动到中间位置。

> [!tip] 记忆口诀
> 真实位置变多少，画面对象按比例动多少。

# 03 对角线移动

> [!info] 核心结论
> 对角线移动表示对象同时沿水平和垂直方向变化，适合显示二维运动、机械手轨迹、斜向输送、XY平台和斜坡位置变化。

## ① 分类/公式/规则

> [!warning] 对角线移动公式
> **X位置 = X起点 + X比例 × X移动量**  
> **Y位置 = Y起点 + Y比例 × Y移动量**

## ② 参考表

| 应用 | 说明 |
|---|---|
| **XY平台** | X/Y坐标同时变化 |
| **机械手轨迹** | 显示斜向移动 |
| **斜坡输送** | 物料沿斜线运动 |
| **二维坐标点** | 显示当前位置点 |
| **机器人示意** | 简化显示运动方向 |
| **相机定位点** | 显示检测坐标 |
| **工件搬运** | 显示取放路径 |

## ③ 计算/选型示例

- 已知：  
  一个对象需要从左上角移动到右下角，X方向移动120像素，Y方向移动80像素，变量范围0~100。
- 公式：  
  **X移动 = 变量 ÷ 100 × 120**  
  **Y移动 = 变量 ÷ 100 × 80**
- 计算：  
  当变量=50时，X移动60像素，Y移动40像素。
- 结论：  
  对象会移动到斜线路径的中间位置。

> [!tip] 记忆口诀
> 水平和垂直一起动，就是对角线移动。

# 04 直接移动

> [!info] 核心结论
> 直接移动通常用于按变量值直接控制对象位置，适合显示对象的绝对位置或按照预设坐标移动。

## ① 分类/公式/规则

> [!warning] 直接移动公式
> **对象坐标 = 变量值对应的坐标位置**

## ② 参考表

| 用途 | 示例 |
|---|---|
| **位置显示** | 伺服轴当前位置 |
| **物料移动** | 工件在输送线上的位置 |
| **滑块移动** | 设定值滑块位置 |
| **指针位置** | 压力、速度、温度指示 |
| **设备部件移动** | 气缸杆、推板、夹爪 |
| **开度显示** | 阀门或挡板开度 |
| **动画对象** | 按实时值移动 |

## ③ 计算/选型示例

- 已知：  
  对象X坐标直接等于变量值加100，变量值为40。
- 公式：  
  **X坐标 = 100 + 变量值**
- 计算：  
  X坐标 = 100 + 40 = **140**
- 结论：  
  对象会移动到X=140的位置。

> [!tip] 记忆口诀
> 直接移动就是变量给位置，对象照着位置走。

# 05 水平移动

> [!info] 核心结论
> 水平移动只改变对象的X方向位置，常用于输送线、滑台、气缸横向动作、进度条和水平液位指示。

## ① 分类/公式/规则

> [!warning] 水平移动公式
> **X位置 = X起点 + 变量比例 × 水平移动距离**

## ② 参考表

| 应用对象 | 说明 |
|---|---|
| **输送线工件** | 左右移动 |
| **横向滑台** | X轴当前位置 |
| **气缸推杆** | 横向伸缩 |
| **进度指示块** | 按进度水平移动 |
| **水平阀杆** | 阀门开度 |
| **送料机构** | 送料前进/后退 |
| **夹爪横移** | 左右位置显示 |

## ③ 计算/选型示例

- 已知：  
  气缸水平移动距离80像素，变量0表示缩回，变量1表示伸出。
- 公式：  
  **X位置 = X起点 + 变量 × 80**
- 计算：  
  变量=1时，X位置 = X起点 + 80。
- 结论：  
  气缸图形显示为伸出状态。

> [!tip] 记忆口诀
> 左右动用水平移动。

# 06 垂直移动

> [!info] 核心结论
> 垂直移动只改变对象的Y方向位置，常用于升降机构、液位高度、Z轴运动、料仓高度和指针上下移动。

## ① 分类/公式/规则

> [!warning] 垂直移动公式
> **Y位置 = Y起点 + 变量比例 × 垂直移动距离**

## ② 参考表

| 应用对象 | 说明 |
|---|---|
| **升降气缸** | 上下动作 |
| **Z轴滑台** | 高度位置 |
| **液位指示** | 液面上下变化 |
| **料仓料位** | 料位高度 |
| **压力指针** | 上下指示 |
| **电梯示意** | 楼层上下移动 |
| **压头机构** | 压装上下运动 |

## ③ 计算/选型示例

- 已知：  
  液位范围0~100%，画面液面高度最大移动120像素。
- 公式：  
  **液面移动 = 液位百分比 ÷ 100 × 120**
- 计算：  
  液位75%时，液面高度 = 75 ÷ 100 × 120 = **90像素**
- 结论：  
  垂直移动适合直观显示液位或升降位置。

> [!tip] 记忆口诀
> 上下动用垂直移动。

# 07 移动方向与坐标

> [!info] 核心结论
> HMI画面通常以左上角为坐标原点，X向右增大，Y向下增大，因此做垂直移动时要特别注意正负方向。

## ① 分类/公式/规则

> [!warning] 坐标规则
> **X增大 → 对象向右移动**  
> **Y增大 → 对象向下移动**  
> **Y减小 → 对象向上移动**

## ② 参考表

| 坐标变化 | 显示效果 |
|---|---|
| **X增加** | 向右 |
| **X减少** | 向左 |
| **Y增加** | 向下 |
| **Y减少** | 向上 |
| **X/Y同时增加** | 右下移动 |
| **X增加、Y减少** | 右上移动 |
| **X减少、Y增加** | 左下移动 |

## ③ 计算/选型示例

- 已知：  
  一个升降平台上升时，画面对象要向上移动50像素。
- 公式：  
  **向上移动 = Y坐标减少**
- 计算：  
  原Y=300，上升后Y=300-50=**250**
- 结论：  
  不要把上升误设为Y增加，否则画面会向下动。

> [!tip] 记忆口诀
> 画面坐标里，向下才是Y增加。

# 08 移动比例换算

> [!info] 核心结论
> 移动动态常需要把真实工程量转换为画面像素移动量，关键是设置正确的变量范围和移动范围。

## ① 分类/公式/规则

> [!warning] 换算公式
> **像素移动量 = (当前值 - 最小值) ÷ (最大值 - 最小值) × 最大移动像素**

## ② 参考表

| 参数 | 含义 |
|---|---|
| **当前值** | PLC或HMI变量当前值 |
| **最小值** | 工程量下限 |
| **最大值** | 工程量上限 |
| **最大移动像素** | 对象在画面上的最大移动距离 |
| **起始位置** | 对象初始坐标 |
| **终点位置** | 对象最大坐标 |
| **移动方向** | 水平、垂直、对角线 |

## ③ 计算/选型示例

- 已知：  
  伺服位置范围0~500mm，画面移动范围0~200像素，当前位置125mm。
- 公式：  
  **像素移动 = 125 ÷ 500 × 200**
- 计算：  
  像素移动 = **50像素**
- 结论：  
  伺服在实际行程25%位置时，画面对象也移动25%。

> [!tip] 记忆口诀
> 工程量先变比例，比例再变像素。

# 09 显示动态

> [!info] 核心结论
> 显示动态用于控制对象在画面中的显示效果，常见包括外观变化、可见性控制、颜色变化、闪烁和启用禁用。

## ① 分类/公式/规则

> [!warning] 显示动态公式
> **显示效果 = 状态变量 → 外观/可见性/颜色/闪烁变化**

## ② 参考表

| 显示动态 | 用途 |
|---|---|
| **外观** | 改变颜色、边框、字体、样式 |
| **可见性** | 显示或隐藏对象 |
| **闪烁** | 报警或提示 |
| **启用状态** | 允许或禁止操作 |
| **透明度** | 弱化不可用对象 |
| **颜色变化** | 区分运行、停止、报警 |
| **文本变化** | 显示当前状态 |

## ③ 计算/选型示例

- 已知：  
  电机报警时，电机图标要显示红色闪烁。
- 公式：  
  **报警位ON → 红色 + 闪烁**
- 计算：  
  当报警变量=1时，对象外观切换为红色并闪烁。
- 结论：  
  显示动态可以让异常状态更醒目。

> [!tip] 记忆口诀
> 运动靠移动，状态靠显示。

# 10 外观动态

> [!info] 核心结论
> 外观动态用于根据变量改变对象的颜色、边框、字体、填充、闪烁或样式，常用于设备状态和报警提示。

## ① 分类/公式/规则

> [!warning] 外观动态公式
> **对象外观 = 状态变量 → 外观规则表**

## ② 参考表

| 状态 | 推荐外观 |
|---|---|
| **停止** | 灰色 |
| **运行** | 绿色 |
| **等待** | 黄色 |
| **报警** | 红色 |
| **维护** | 蓝色 |
| **禁用** | 浅灰或半透明 |
| **选中** | 高亮边框 |

## ③ 计算/选型示例

- 已知：  
  设备状态变量：0停止，1运行，2报警。
- 公式：  
  **状态0→灰色，状态1→绿色，状态2→红色**
- 计算：  
  当状态变量=2时，对象显示红色。
- 结论：  
  外观动态适合做设备运行状态显示。

> [!tip] 记忆口诀
> 颜色要统一，状态才容易一眼看懂。

# 11 可见性动态

> [!info] 核心结论
> 可见性动态用于根据条件显示或隐藏对象，常用于权限按钮、报警提示、维护信息、调试对象和状态提示。

## ① 分类/公式/规则

> [!warning] 可见性公式
> **对象是否显示 = 条件变量是否满足**

## ② 参考表

| 应用 | 说明 |
|---|---|
| **权限按钮** | 工程师登录后才显示 |
| **报警提示** | 报警发生时显示 |
| **维护信息** | 维护模式下显示 |
| **调试控件** | 调试页面或调试权限下显示 |
| **帮助说明** | 点击帮助时显示 |
| **状态图标** | 条件满足时显示 |
| **安全提示** | 安全门打开时显示 |

## ③ 计算/选型示例

- 已知：  
  “伺服调试”按钮只允许工程师看到。
- 公式：  
  **可见性 = 当前用户权限 ≥ 工程师**
- 计算：  
  操作员登录时按钮隐藏，工程师登录时按钮显示。
- 结论：  
  可见性动态可以减少误操作入口。

> [!tip] 记忆口诀
> 不该看的对象，直接隐藏；不该按的按钮，禁用或隐藏。

# 12 可见性与安全

> [!info] 核心结论
> 隐藏按钮可以减少误操作，但不能替代真正的权限控制；关键操作还必须做权限判断和PLC互锁。

## ① 分类/公式/规则

> [!warning] 安全规则
> **隐藏 ≠ 安全**  
> **关键操作 = 可见性控制 + 权限控制 + PLC联锁**

## ② 参考表

| 控制方式 | 作用 |
|---|---|
| **隐藏对象** | 减少误触 |
| **禁用对象** | 显示但不可操作 |
| **权限验证** | 限制角色操作 |
| **二次确认** | 防止误执行 |
| **PLC互锁** | 最终安全保护 |
| **操作日志** | 记录谁操作了什么 |
| **报警反馈** | 操作失败给出原因 |

## ③ 计算/选型示例

- 已知：  
  普通操作员看不到“清零产量”按钮。
- 公式：  
  **隐藏按钮 + 权限判断**
- 计算：  
  即使通过其他方式触发，也应再次验证权限。
- 结论：  
  可见性只是界面层保护，安全逻辑仍要在权限和PLC中实现。

> [!tip] 记忆口诀
> 画面隐藏防误触，真正安全靠权限和PLC。

# 13 变量连接

> [!info] 核心结论
> 变量连接用于把对象属性与PLC变量、HMI内部变量、系统变量或脚本变量绑定起来，是对象动态化的基础。

## ① 分类/公式/规则

> [!warning] 变量连接公式
> **对象属性 = 变量值驱动**

## ② 参考表

| 变量类型 | 用途 |
|---|---|
| **PLC变量** | 设备状态、报警、位置、参数 |
| **HMI内部变量** | 页面状态、临时变量、按钮状态 |
| **系统变量** | 当前用户、时间、语言 |
| **脚本变量** | 计算结果、逻辑判断 |
| **报警变量** | 报警位、报警字 |
| **配方变量** | 产品参数 |
| **数据库变量** | 历史记录、报表数据 |

## ③ 计算/选型示例

- 已知：  
  圆形指示灯需要显示气压是否正常。
- 公式：  
  **气压正常变量ON → 指示灯绿色**
- 计算：  
  将圆形对象的颜色属性连接到气压正常变量。
- 结论：  
  变量连接后，对象就能随设备状态自动变化。

> [!tip] 记忆口诀
> 没有变量连接，画面对象只是静态图。

# 14 变量连接检查

> [!info] 核心结论
> 变量连接最容易出错的地方是地址、数据类型、读写方向、刷新周期和变量命名。

## ① 分类/公式/规则

> [!warning] 连接正确公式
> **变量连接正确 = 地址正确 + 类型正确 + 方向正确 + 通讯正常**

## ② 参考表

| 检查项 | 说明 |
|---|---|
| **变量地址** | 是否对应正确PLC地址 |
| **数据类型** | Bool、Int、Real、String等 |
| **读写方向** | 只读、只写、读写 |
| **刷新周期** | 是否及时更新 |
| **变量范围** | 是否符合动态映射 |
| **变量命名** | 是否清晰可维护 |
| **通讯状态** | PLC/HMI是否在线 |

## ③ 计算/选型示例

- 已知：  
  一个图标应该根据“运行状态”变色，但一直不变。
- 公式：  
  **对象不变 = 变量未变化 / 地址错 / 动态规则错**
- 计算：  
  先在线监视变量值是否变化，再查对象属性绑定。
- 结论：  
  动态问题应先确认变量是否正确变化。

> [!tip] 记忆口诀
> 对象不动，先看变量动不动。

# 15 动态移动的典型应用：气缸动画

> [!info] 核心结论
> 气缸动画常用水平或垂直移动实现，显示伸出、缩回、到位、超时和报警状态。

## ① 分类/公式/规则

> [!warning] 气缸动画公式
> **气缸显示 = 移动位置 + 到位反馈 + 报警外观**

## ② 参考表

| 信号 | 显示效果 |
|---|---|
| **伸出命令** | 显示动作趋势 |
| **伸出到位** | 显示伸出状态 |
| **缩回到位** | 显示缩回状态 |
| **动作中** | 显示中间位置或动画 |
| **超时报警** | 红色闪烁 |
| **手动模式** | 显示可操作 |
| **禁用状态** | 按钮置灰 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出行程在画面上为60像素，伸出进度变量0~100。
- 公式：  
  **移动距离 = 伸出进度 ÷ 100 × 60**
- 计算：  
  伸出进度50时，移动距离 = 30像素。
- 结论：  
  气缸图形显示在半伸出位置。

# 16 动态移动的典型应用：液位显示

# 18 常见故障排查

> [!info] 核心结论
> 对象动态属性异常时，应按 **变量值、变量连接、动态规则、坐标范围、权限、层级、刷新周期** 的顺序排查。

## ① 分类/公式/规则

> [!warning] 排错公式
> **动态异常 = 变量未变 + 连接错误 + 规则错误 + 范围错误 + 对象被隐藏 + 层级被遮挡**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **对象不移动** | 变量未变化、移动动态未连接 |
| **移动方向反了** | X/Y方向或正负号设置错误 |
| **移动距离不对** | 变量范围或像素范围设置错误 |
| **对象不显示** | 可见性条件不满足、被遮挡 |
| **颜色不变** | 外观动态规则错误 |
| **闪烁异常** | 报警变量或闪烁条件错误 |
| **按钮不可用** | 权限不足或启用条件不满足 |
| **画面卡顿** | 动态对象太多、刷新太快 |

## ③ 计算/选型示例

- 已知：  
  升降平台上升时，画面对象反而向下移动。
- 公式：  
  **Y增加 = 向下移动**
- 计算：  
  上升动作应让Y坐标减少，而不是增加。
- 结论：  
  应修改垂直移动方向或移动范围符号。

> [!tip] 记忆口诀
> 动画错方向，先查坐标正负；对象不显示，先查可见性条件。

# 19 动态属性快速检查表

> [!info] 核心结论
> 动态属性配置完成后，要检查变量连接、移动范围、外观规则、可见性条件和运行效果是否符合现场真实状态。

## ① 分类/公式/规则

> [!note] 十步检查法
> ① 变量是否正确连接  
> ② 数据类型是否匹配  
> ③ 变量范围是否正确  
> ④ 移动方向是否正确  
> ⑤ 移动距离是否符合画面比例  
> ⑥ 外观颜色是否符合状态规范  
> ⑦ 可见性条件是否符合权限和工况  
> ⑧ 对象层级是否正确  
> ⑨ 刷新周期是否合适  
> ⑩ 动态效果是否与PLC真实反馈一致

> [!warning] 最终检查公式
> **合格动态对象 = 变量正确 + 规则正确 + 显示正确 + 操作安全**

## ② 参考表

| 检查项 | 目标 |
|---|---|
| **移动** | 位置变化符合真实机构 |
| **对角线移动** | X/Y方向都正确 |
| **水平移动** | 左右方向正确 |
| **垂直移动** | 上下方向正确 |
| **外观** | 状态颜色统一 |
| **可见性** | 该显示才显示 |
| **变量连接** | 地址和类型正确 |
| **运行测试** | 与现场反馈一致 |

## ③ 计算/选型示例

- 已知：  
  需要让一个阀门图标显示开、关、报警三种状态。
- 公式：  
  **阀门显示 = 状态变量 + 外观动态 + 可见性/图形切换**
- 计算：  
  开阀显示绿色，关阀显示灰色，报警显示红色闪烁。
- 结论：  
  推荐使用外观动态或图形I/O域，并绑定阀门反馈和报警变量。

> [!tip] 选型口诀
> 要动位置看移动，要变颜色看外观，要显示隐藏看可见性，要让对象跟PLC联动看变量连接。
