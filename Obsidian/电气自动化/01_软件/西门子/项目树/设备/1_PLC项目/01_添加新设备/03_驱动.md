# 01 驱动和启动器总览

> [!info] 核心结论
> 驱动和启动器的核心作用，是控制电机的 **启动、停止、调速、转矩、定位和保护**。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 普通调速看 **SINAMICS G**；  
> 高性能运动控制看 **SINAMICS S**；  
> 中压大功率电机看 **SINAMICS MV**。

> [!warning] 🟧 通用选型公式
> **驱动选型 = 电机功率 + 电压等级 + 负载类型 + 控制精度 + 通讯方式 + 环境条件**

## ② 参考表

| 系列 | 主要定位 |
|---|---|
| **SINAMICS G** | 通用变频调速 |
| **SINAMICS S** | 高性能伺服 / 运动控制 |
| **SINAMICS MV** | 中压大功率驱动 |

## ③ 计算/选型示例

- 已知：  
  一台三相异步电机，功率 **7.5kW**，用于风机调速。
- 公式：  
  **三相电机电流 I ≈ P ÷（√3 × U × η × cosφ）**
- 计算：  
  假设 U = 380V，η = 0.9，cosφ = 0.85  
  I ≈ 7500 ÷（1.732 × 380 × 0.9 × 0.85）≈ **14.9A**
- 结论：  
  风机属于通用调速负载，可优先考虑 **SINAMICS G**，具体型号需按电机铭牌电流、过载能力和厂家手册校核。

> [!tip] 记忆口诀
> 普通调速选G，高端运动选S，中压大功率选MV。

# 02 SINAMICS 驱动

> [!info] 核心结论
> SINAMICS 是西门子驱动产品族，覆盖从普通变频调速到高性能伺服、中压大功率驱动的应用。

## ① 分类/公式/规则

> [!note] 🟦 驱动核心任务
> 驱动器不是简单“给电机供电”，而是根据工艺需求控制电机的 **速度、转矩、位置、加减速和保护**。

> [!warning] 🟧 驱动能力判断公式
> **驱动能力 = 电流容量 + 过载能力 + 控制方式 + 制动能力 + 通讯能力**

## ② 参考表

| 选型项目 | 关键参数 |
|---|---|
| 电机类型 | 异步电机、同步电机、伺服电机 |
| 电压等级 | **低压 / 中压** |
| 功率范围 | 按电机铭牌功率校核 |
| 控制方式 | V/f、矢量、伺服、转矩控制 |
| 通讯 | PROFINET、PROFIBUS、USS等 |
| 环境 | 温度、海拔、粉尘、防护等级 |

## ③ 计算/选型示例

- 已知：  
  项目中既有风机变频，又有伺服定位轴。
- 公式：  
  **驱动类型 = 负载对象 + 控制精度 + 动态响应**
- 计算：  
  风机只需要调速，定位轴需要高动态、高精度控制。
- 结论：  
  风机可选 **SINAMICS G**，定位轴应评估 **SINAMICS S**，具体型号需按厂家手册校核。

> [!tip] 记忆口诀
> 先看电机，再看负载；先算电流，再选驱动。

# 03 SINAMICS G

> [!info] 核心结论
> SINAMICS G 是通用型变频器系列，适合风机、水泵、输送线、搅拌机等常规调速场景。

## ① 分类/公式/规则

> [!note] 🟦 定位
> SINAMICS G 主要用于 **普通异步电机调速**，强调经济、稳定、通用和易维护。

> [!warning] 🟧 G系列判断公式
> **适合G系列 = 普通调速 + 中低动态 + 非高精度定位 + 通用机械负载**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 典型负载 | **风机、水泵、输送、搅拌** |
| 控制目标 | 调速、节能、软启动 |
| 控制精度 | 普通速度控制 |
| 动态要求 | 一般 |
| 重点参数 | 功率、电流、过载、通讯 |
| 注意 | 重载应用需校核过载能力 |

## ③ 计算/选型示例

- 已知：  
  一台 **11kW** 水泵电机，需要变频节能控制。
- 公式：  
  **变频器额定电流 ≥ 电机额定电流**
- 计算：  
  根据电机铭牌查额定电流，再选择额定输出电流不小于电机电流的变频器。
- 结论：  
  水泵属于典型通用调速负载，可优先考虑 **SINAMICS G**，型号需按电机铭牌、电压等级、过载能力校核。

> [!tip] 记忆口诀
> 风机水泵输送线，普通调速G优先。

# 04 SINAMICS S

> [!info] 核心结论
> SINAMICS S 是高性能驱动系列，适合伺服控制、运动控制、多轴同步和高动态设备。

## ① 分类/公式/规则

> [!note] 🟦 定位
> SINAMICS S 面向高性能运动控制，常用于定位轴、同步轴、电子齿轮、飞剪、包装、机器人等场景。

> [!warning] 🟧 S系列判断公式
> **适合S系列 = 高精度 + 高动态 + 多轴同步 + 复杂运动控制**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 典型负载 | **伺服轴、定位轴、同步轴** |
| 控制目标 | 速度、转矩、位置 |
| 动态要求 | 高 |
| 精度要求 | 高 |
| 应用场景 | 包装、印刷、飞剪、搬运 |
| 校核重点 | 轴数、编码器、控制周期、通讯 |

## ③ 计算/选型示例

- 已知：  
  一台包装机有多个伺服轴，需要同步运行和精准定位。
- 公式：  
  **运动复杂度 = 轴数 + 同步关系 + 定位精度 + 节拍要求**
- 计算：  
  多轴同步 + 高节拍 + 位置控制，普通通用变频器难以满足。
- 结论：  
  应优先考虑 **SINAMICS S**，具体驱动模块、电机、编码器和控制器需按厂家手册校核。

> [!tip] 记忆口诀
> 要定位、要同步、要高动态，就选SINAMICS S。

# 05 SINAMICS MV

> [!info] 核心结论
> SINAMICS MV 是中压驱动系列，适合中压电机、大功率设备和重工业场景。

## ① 分类/公式/规则

> [!note] 🟦 定位
> SINAMICS MV 主要面向 **中压大功率电机驱动**，常用于泵、风机、压缩机、磨机、矿山、冶金、水泥等场景。

> [!warning] 🟧 MV系列判断公式
> **适合MV系列 = 中压电机 + 大功率 + 高可靠性 + 重工业应用**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 电压等级 | **中压** |
| 功率等级 | 大功率电机 |
| 典型负载 | 风机、泵、压缩机、磨机 |
| 应用行业 | 冶金、水泥、矿山、电力 |
| 重点参数 | 电压、功率、冷却、谐波、防护 |
| 注意 | 必须按项目规范和厂家手册校核 |

## ③ 计算/选型示例

- 已知：  
  工厂有一台中压大功率风机，需要调速节能。
- 公式：  
  **MV需求 = 中压等级 + 大功率 + 长周期运行 + 高可靠性**
- 计算：  
  普通低压变频器不适合中压大功率电机，需要专用中压驱动系统。
- 结论：  
  应评估 **SINAMICS MV**，电压等级、功率、冷却方式、谐波要求和防护等级需按实际规范/厂家手册校核。

> [!tip] 记忆口诀
> 低压通用看G，高端伺服看S，中压大功率看MV。

# 06 负载类型选型

> [!info] 核心结论
> 驱动选型不能只看功率，还要看负载是轻载、重载，还是高动态运动负载。

## ① 分类/公式/规则

> [!note] 🟦 负载分类
> 风机、水泵多为平方转矩负载；  
> 输送、搅拌多为恒转矩负载；  
> 伺服定位多为高动态负载；  
> 中压大设备多为重工业连续负载。

> [!warning] 🟧 负载判断公式
> **负载等级 = 启动转矩 + 运行转矩 + 加减速时间 + 过载频率**

## ② 参考表

| 负载类型 | 推荐方向 |
|---|---|
| 风机、水泵 | **SINAMICS G** |
| 输送、搅拌 | **SINAMICS G，重载需校核** |
| 伺服定位 | **SINAMICS S** |
| 多轴同步 | **SINAMICS S** |
| 中压大功率 | **SINAMICS MV** |
| 重工业连续运行 | **SINAMICS MV / 按项目校核** |

## ③ 计算/选型示例

- 已知：  
  同样是15kW电机，一个用于风机，一个用于提升机构。
- 公式：  
  **驱动选择 = 功率 + 负载转矩 + 过载要求**
- 计算：  
  风机负载较平稳，提升机构启动和过载要求更高。
- 结论：  
  两者不能只按15kW直接套型号，提升机构必须重点校核过载能力、制动单元和安全要求。

> [!tip] 记忆口诀
> 同功率不同命，负载不同选型不同。

# 07 驱动快速选型表

> [!info] 核心结论
> 驱动选型先分清 **通用调速、运动控制、中压大功率** 三类，再细化功率、电流和通讯。

## ① 分类/公式/规则

> [!note] 🟦 四步选型法
> ① 看电机铭牌  
> ② 看负载类型  
> ③ 看控制精度  
> ④ 看通讯和环境

> [!warning] 🟧 最终选型公式
> **推荐驱动 = 电机额定电流 + 负载特性 + 控制精度 + 通讯需求 + 环境条件**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 普通电机调速 | **SINAMICS G** |
| 风机水泵节能 | **SINAMICS G** |
| 输送搅拌调速 | **SINAMICS G，重载校核** |
| 伺服定位 | **SINAMICS S** |
| 多轴同步 | **SINAMICS S** |
| 中压电机 | **SINAMICS MV** |
| 大功率重工业 | **SINAMICS MV** |

## ③ 计算/选型示例

- 已知：  
  项目有三类电机：风机、水泵、伺服定位轴。
- 公式：  
  **驱动系列 = 负载类型 + 控制要求**
- 计算：  
  风机、水泵属于通用调速；伺服定位轴需要位置控制和高动态响应。
- 结论：  
  风机、水泵优先选 **SINAMICS G**；伺服定位轴优先选 **SINAMICS S**。若遇到中压大功率电机，则评估 **SINAMICS MV**。

> [!tip] 选型口诀
> G管通用调速，S管伺服运动，MV管中压大功率。
