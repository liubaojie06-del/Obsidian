# 01｜PID_Temp 是什么

> [!info] 核心结论
> **PID_Temp 是 TIA Portal 中专门面向温度控制的 PID 功能块，适合加热、冷却、恒温、炉温、热封、模温等闭环控制场景。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>图片识别</strong><br>
截图中的 <strong>PID_Temp V1.1</strong> 是西门子 PLC 中用于温度闭环调节的 PID 控制块。
</div>

## ① 分类/公式/规则

| 项目 | 说明 |
|---|---|
| **指令名称** | `PID_Temp` |
| **版本** | `V1.1` |
| **中文含义** | 温度 PID 控制器 |
| **控制对象** | 加热器、冷却阀、模温机、炉温系统 |
| **控制目标** | 让实际温度稳定接近设定温度 |
| **典型输出** | 加热功率、PWM、模拟量、冷却控制量 |

<div class="card orange" style="background:#FFF7F3;border-left:6px solid #FF6B35;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>一句话理解</strong><br>
PID_Temp 就是专门用来让温度“升得稳、停得准、不乱冲”的 PID 控制块。
</div>

## ② 参考表

| 名称 | 中文含义 | 工程理解 |
|---|---|---|
| **Setpoint** | 温度设定值 | 想要达到的温度 |
| **Input / PV** | 实际温度 | 传感器测到的温度 |
| **Error** | 温度偏差 | 设定温度 - 实际温度 |
| **Output** | 控制输出 | 加热/冷却强度 |
| **Heating** | 加热控制 | 温度低时增加输出 |
| **Cooling** | 冷却控制 | 温度高时降低温度 |
| **AutoTune** | 自动整定 | 帮助寻找 PID 参数 |

## ③ 计算/选型示例

- **已知：** 热封设备需要温度稳定在 `180℃`
- **公式：** 温度闭环 = `设定温度 - 实际温度 → PID计算 → 控制加热器`
- **计算：**
  - 设定值：`180℃`
  - 实际值：热电偶或 PT100 反馈
  - 实际值低于设定值：增加加热输出
  - 实际值接近设定值：逐步减小输出
- **结论：** 恒温控制场景适合使用 `PID_Temp`。

> [!tip] 记忆口诀
> **控温找 PID_Temp，调速找 PID_Compact，三步阀找 PID_3Step。**

# 02｜温度闭环原理

> [!info] 核心结论
> **温度 PID 的核心是根据温度偏差调节加热或冷却输出，使实际温度逐渐稳定在设定值附近。**

<div class="card orange" style="background:#FFF7F3;border-left:6px solid #FF6B35;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>重点公式</strong><br>
温度偏差 = 设定温度 - 实际温度
</div>

## ① 分类/公式/规则

| 偏差状态 | 控制动作 | 工程含义 |
|---|---|---|
| **实际温度低** | 增加加热 | 让温度上升 |
| **实际温度高** | 减少加热 / 开冷却 | 让温度下降 |
| **接近设定值** | 小幅调节 | 防止超调 |
| **稳定区间内** | 保持输出 | 维持恒温 |

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>PID 三项口诀</strong><br>
P 管当前偏差，I 管长期偏差，D 管变化趋势。
</div>

## ② 参考表

| 控制项 | 作用 | 调太大的风险 |
|---|---|---|
| **P 比例** | 提高响应速度 | 温度振荡 |
| **I 积分** | 消除稳态误差 | 超调明显 |
| **D 微分** | 抑制温度变化趋势 | 对噪声敏感 |
| **死区** | 减少频繁动作 | 控温精度下降 |
| **输出限幅** | 防止输出过大 | 升温变慢 |

## ③ 计算/选型示例

- **已知：**
  - 设定温度：`120℃`
  - 当前温度：`95℃`
- **公式：** 偏差 = `SP - PV`
- **计算：**
  - `120 - 95 = 25℃`
- **结论：** 当前温度偏低 `25℃`，PID_Temp 会增加加热输出。

> [!tip] 记忆口诀
> **温度低就加热，温度高就收力，快到设定要慢慢来。**

# 03｜适合什么场景

> [!info] 核心结论
> **PID_Temp 适合热惯性明显、需要稳定温度的系统，不适合普通开关动作或快速定位控制。**

<div class="card green" style="background:#F1FBF5;border-left:6px solid #27AE60;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>工程定位</strong><br>
只要核心目标是“温度稳定”，优先想到 PID_Temp。
</div>

## ① 分类/公式/规则

| 场景 | 是否适合 | 说明 |
|---|---|---|
| **电加热炉** | 适合 | 恒温控制 |
| **热封刀温控** | 适合 | 控温精度要求高 |
| **模具温度** | 适合 | 热惯性大 |
| **烘箱温度** | 适合 | 慢过程 |
| **冷却水温度** | 适合 | 加热/冷却调节 |
| **气缸伸缩** | 不适合 | 离散动作 |
| **伺服定位** | 不适合 | 应用运动控制 |

## ② 参考表

| 控制对象 | 执行器 | 常见反馈 |
|---|---|---|
| **加热棒** | SSR / 接触器 / 模拟调功器 | 热电偶 |
| **加热炉** | 可控硅 / 固态继电器 | 热电偶 / RTD |
| **模温机** | 加热 + 冷却阀 | PT100 |
| **热封机构** | 加热器 | 热电偶 |
| **冷却系统** | 冷却阀 / 风机 | 温度传感器 |

## ③ 计算/选型示例

- **已知：** 烘箱温度需要稳定在 `80℃`，允许小范围波动
- **公式：** 慢速温度对象 + 恒温目标 → PID_Temp
- **计算：**
  - 温度变化慢
  - 有传感器反馈
  - 有加热执行器
  - 需要稳定控制
- **结论：** 烘箱温控适合使用 `PID_Temp`。

> [!tip] 记忆口诀
> **温度慢慢变，PID 慢慢调；别用开关硬怼温控。**

# 04｜输入与传感器

> [!info] 核心结论
> **PID_Temp 的实际值必须来自可靠的温度反馈，常见传感器有热电偶、PT100、热电阻和温度变送器。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>输入链路</strong><br>
温度传感器 → 模拟量/温度模块 → 工程量温度 → PID_Temp 输入
</div>

## ① 分类/公式/规则

| 传感器 | 特点 | 注意 |
|---|---|---|
| **热电偶** | 适合高温 | 类型 K/J/T 等要匹配 |
| **PT100** | 精度较好 | 三线/四线接法需正确 |
| **温度变送器** | 输出 4-20mA | 需工程量换算 |
| **NTC/PTC** | 成本低 | 需电路和曲线换算 |
| **红外测温** | 非接触 | 受表面 emissivity 影响 |

<div class="card orange" style="background:#FFF7F3;border-left:6px solid #FF6B35;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>重点规则</strong><br>
PID 之前必须先确认温度值真实可信，否则再怎么调参数都不稳定。
</div>

## ② 参考表

| 检查项 | 作用 | 常见错误 |
|---|---|---|
| **传感器类型** | 决定模块设置 | K型接成PT100 |
| **量程范围** | 决定换算比例 | 量程配置错 |
| **接线方式** | 决定测量稳定 | 补偿线/三线制错误 |
| **滤波** | 抑制温度波动 | 滤波太强响应慢 |
| **断线诊断** | 发现传感器故障 | 未启用诊断 |

## ③ 计算/选型示例

- **已知：**
  - 温度变送器输出：`4-20mA`
  - 对应温度范围：`0-200℃`
  - 当前换算后为 `12mA`
- **公式：** 温度 = `(当前电流 - 4) ÷ 16 × 量程`
- **计算：**
  - `(12 - 4) ÷ 16 × 200 = 100℃`
- **结论：** 当前温度约为 `100℃`，可作为 PID_Temp 的实际值输入。

> [!tip] 记忆口诀
> **先测准温度，再谈 PID；温度值错了，输出一定乱。**

# 05｜输出控制方式

> [!info] 核心结论
> **PID_Temp 的输出通常控制加热功率，也可以配合冷却阀、风机或调功器实现加热/冷却双向调节。**

<div class="card orange" style="background:#FFF7F3;border-left:6px solid #FF6B35;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>输出理解</strong><br>
PID_Temp 输出可以理解为“当前需要多少加热或冷却能力”。
</div>

## ① 分类/公式/规则

| 输出方式 | 适合设备 | 说明 |
|---|---|---|
| **PWM** | SSR、固态继电器 | 按占空比控制加热 |
| **模拟量** | 调功器、可控硅 | 连续调节功率 |
| **数字量开关** | 接触器 | 适合慢速低频控制 |
| **冷却输出** | 风机、冷却阀 | 温度过高时使用 |
| **百分比输出** | 上层程序换算 | `0~100%` |

## ② 参考表

| 执行器 | 推荐控制 | 注意 |
|---|---|---|
| **SSR** | PWM | 周期不宜过短或过长 |
| **接触器** | 慢周期开关 | 避免频繁吸合 |
| **可控硅调功器** | 模拟量 | 线性和范围校核 |
| **冷却风机** | 开关/PWM/模拟量 | 防止过冷 |
| **冷却水阀** | 模拟量/三步控制 | 与阀门类型匹配 |

## ③ 计算/选型示例

- **已知：**
  - PID 输出：`40%`
  - PWM 周期：`10s`
- **公式：** 导通时间 = `输出百分比 × PWM周期`
- **计算：**
  - `40% × 10s = 4s`
- **结论：** 一个周期内加热输出导通约 `4s`，关闭约 `6s`。

> [!tip] 记忆口诀
> **SSR 看占空比，调功器看模拟量，接触器别频繁跳。**

# 06｜加热与冷却控制

> [!info] 核心结论
> **温控系统可能只有加热，也可能同时有加热和冷却，控制方向必须在调试前确认清楚。**

<div class="card purple" style="background:#F8F1FF;border-left:6px solid #9B59B6;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>控制口诀</strong><br>
低温加热，高温冷却；方向弄反，越调越偏。
</div>

## ① 分类/公式/规则

| 控制类型 | 说明 | 典型场景 |
|---|---|---|
| **单加热** | 只控制升温 | 烘箱、热封 |
| **单冷却** | 只控制降温 | 冷却水、风冷 |
| **加热+冷却** | 双向温控 | 模温机、恒温箱 |
| **分段控制** | 不同温区不同策略 | 多区炉温 |
| **保温控制** | 稳态小输出 | 恒温过程 |

## ② 参考表

| 现象 | 可能原因 | 处理 |
|---|---|---|
| **温度越控越远** | 控制方向反 | 检查加热/冷却方向 |
| **超温明显** | 加热太强 / 惯性大 | 降低输出或优化参数 |
| **升温太慢** | 加热功率不足 | 查功率和负载 |
| **降温太慢** | 冷却能力不足 | 查风机/阀门 |
| **温度来回震荡** | 参数过激 | 降低 Kp 或减弱积分 |

## ③ 计算/选型示例

- **已知：** 实际温度低于设定温度，但冷却阀却打开  
- **公式：** 温度偏低时应减少冷却或增加加热  
- **计算：**
  - 当前温度低
  - 冷却阀打开会继续降温
  - 控制方向明显错误
- **结论：** 应先修正控制方向，再做 PID 整定。

> [!tip] 记忆口诀
> **温控先看方向，再看参数。**

# 07｜参数整定

> [!info] 核心结论
> **PID_Temp 整定的目标是让温度升温不过冲、稳态不震荡、长期无明显静差。**

<div class="card orange" style="background:#FFF7F3;border-left:6px solid #FF6B35;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>整定口诀</strong><br>
慢了加 P，差了加 I，冲了降 P，抖了弱 I。
</div>

## ① 分类/公式/规则

| 参数 | 作用 | 调整方向 |
|---|---|---|
| **Kp** | 响应快慢 | 太小升温慢，太大超调 |
| **Ti** | 消除静差 | 太强易超调 |
| **Td** | 抑制趋势 | 温度噪声大时慎用 |
| **输出限幅** | 防止功率过大 | 保护加热器 |
| **采样周期** | 控制节拍 | 慢过程不宜过快 |

## ② 参考表

| 现象 | 可能原因 | 调整方向 |
|---|---|---|
| **升温慢** | Kp 小 / 功率不足 | 增大 Kp 或查加热能力 |
| **超温明显** | Kp 大 / 积分强 | 降 Kp，减弱积分 |
| **稳定后偏低** | 积分不足 | 增强积分 |
| **温度振荡** | 参数过激 | 降 Kp 或加大死区 |
| **输出频繁跳变** | 反馈噪声 | 加滤波或优化周期 |

## ③ 计算/选型示例

- **已知：** 温度长期稳定在 `78℃`，设定值为 `80℃`
- **公式：** 稳态偏差 = `设定值 - 实际值`
- **计算：**
  - `80 - 78 = 2℃`
  - 长期存在静差
  - 可检查积分作用是否不足
- **结论：** 稳态偏差明显时，应检查积分参数、输出限幅和加热能力。

> [!tip] 记忆口诀
> **先稳住，再调快；温控最怕猛。**

# 08｜调用周期与温度惯性

> [!info] 核心结论
> **温度系统通常是慢过程，PID_Temp 调用周期应稳定，参数整定要考虑热惯性和滞后。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>温控特点</strong><br>
加热有滞后，散热有惯性；现在的输出，过一会儿才体现在温度上。
</div>

## ① 分类/公式/规则

| 特点 | 影响 |
|---|---|
| **热惯性大** | 升温和降温都慢 |
| **滞后明显** | 输出变化后温度延迟响应 |
| **过冲常见** | 加热过猛容易超温 |
| **采样不宜乱** | 周期波动影响 PID |
| **冷却能力有限** | 超温后恢复慢 |

## ② 参考表

| 控制对象 | 过程特点 | 调试重点 |
|---|---|---|
| **热封刀** | 中等惯性 | 防超温 |
| **烘箱** | 大惯性 | 慢速整定 |
| **模具** | 热容量大 | 分区均匀性 |
| **水温** | 中慢过程 | 搅拌/流量影响 |
| **空气加热** | 滞后明显 | 风量影响大 |

## ③ 计算/选型示例

- **已知：** 加热器开启后，温度 20 秒后才明显上升  
- **公式：** 大滞后系统 → 参数不能太激进  
- **计算：**
  - 输出变化有延迟
  - 如果 Kp 和积分太强，会在温度未反应前持续加热
  - 容易造成超温
- **结论：** 有明显滞后的温控系统，应采用更温和的 PID 参数。

> [!tip] 记忆口诀
> **温度有惯性，调参别着急。**

# 09｜常见故障排查

> [!info] 核心结论
> **PID_Temp 排查顺序：先看温度反馈，再看输出方向，再看执行器，再看 PID 参数。**

<div class="card red" style="background:#FFF4F3;border-left:6px solid #E74C3C;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>排查顺序</strong><br>
传感器 → 量程 → 输出方向 → 加热器/冷却器 → 输出限幅 → PID 参数。
</div>

## ① 分类/公式/规则

| 问题 | 常见原因 |
|---|---|
| **温度一直为 0** | 传感器断线 / 量程错误 |
| **输出有但不升温** | 加热器故障 / SSR 不动作 |
| **越控越远** | 加热冷却方向反 |
| **超温严重** | 输出过大 / 参数太激进 |
| **温度振荡** | Kp 太大 / 积分太强 |
| **长期不到设定** | 功率不足 / 散热过大 / 输出限幅 |

## ② 参考表

| 现象 | 排查方向 | 解决建议 |
|---|---|---|
| **PV 异常** | 传感器和模块量程 | 先修正温度反馈 |
| **Output=100% 但不升温** | 加热器/SSR/接触器 | 查执行器回路 |
| **接近设定仍超温** | 热惯性过大 | 降输出、优化参数 |
| **温度上下摆动** | 参数过激 | 降 Kp 或减弱积分 |
| **Error=1** | 块状态异常 | 查状态码和帮助文档 |
| **冷却不动作** | 冷却输出未接/方向错 | 手动验证冷却 |

## ③ 计算/选型示例

- **已知：** 设定 `150℃`，实际显示始终 `25℃`，输出已经 `100%`
- **公式：** 满输出无升温 → 先查执行器和反馈
- **计算：**
  - 可能加热器未通电
  - SSR/接触器未动作
  - 传感器安装位置错误
  - 温度反馈未正确接入
- **结论：** 不要先改 PID 参数，应先确认加热回路和温度反馈。

> [!tip] 记忆口诀
> **温度不准先查传感器，输出不动先查执行器。**

# 10｜一页速记总卡

> [!info] 核心结论
> **PID_Temp 的核心是温度闭环：用设定温度和实际温度的偏差，自动调节加热或冷却输出。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px 14px;border-radius:12px;margin:10px 0;">
<strong>总口诀</strong><br>
先测温，再控温；先手动，再自动；先方向，再参数。
</div>

## ① 分类/公式/规则

| 主轴 | 关键词 |
|---|---|
| **对象** | `PID_Temp V1.1` |
| **控制目标** | 温度稳定 |
| **输入** | 设定温度、实际温度 |
| **输出** | 加热、冷却、PWM、模拟量 |
| **关键参数** | Kp、Ti、Td、输出限幅、采样周期 |
| **典型设备** | 加热炉、热封、模温、烘箱 |

## ② 参考表

| 需求 | 推荐 |
|---|---|
| **温度闭环** | `PID_Temp` |
| **普通连续量调节** | `PID_Compact` |
| **三步电动阀** | `PID_3Step` |
| **开关气缸** | FB 顺序控制 |
| **伺服定位** | TO / MC 运动控制 |
| **加热器 PWM** | PID 输出 + PWM/SSR |

## ③ 计算/选型示例

- **已知：** 一个热封刀需要稳定在 `180℃`
- **公式：** 温控系统 = `温度反馈 + PID_Temp + 加热输出`
- **计算：**
  - 传感器采集实际温度
  - PID_Temp 比较设定值和实际值
  - 输出控制 SSR 或调功器
  - 稳定后小范围调节
- **结论：** 热封刀控温是 `PID_Temp` 的典型应用。

> [!tip] 记忆口诀
> **控温三件事：反馈要准，方向要对，参数要稳。**
