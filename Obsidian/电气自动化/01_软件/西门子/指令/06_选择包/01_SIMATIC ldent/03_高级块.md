## 14 高级块 | 选型速记

> [!info] 核心结论
> 高级块的核心是：**UID 用 Read_UID，天线用 Set_Ant，UHF 盘点用 Inventory，EPC/TID 用对应读写，复杂系统用 Ident_Profile。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
读身份 UID，控天线 Ant；UHF 盘点 Inventory，读写 EPC/TID，配置看 Config 和 Profile。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 高级命令入口 | **Advanced_CMD** |
| 读取 UID | **Read_UID** |
| RF300 天线控制 | **Set_Ant_RF300** |
| RF600 组态读写 | **Config_Upload / Config_Download** |
| RF600 标签盘点 | **Inventory** |
| EPC 读取 | **Read_EPC_Mem** |
| TID 读取 | **Read_TID** |
| RF600 天线控制 | **Set_Ant_RF600** |
| UHF 参数修改 | **Set_Param** |
| EPC 写入 | **Write_EPC_ID / Write_EPC_Mem** |
| Ident 配置文件 | **Ident_Profile** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 读取标签唯一身份 | **Read_UID** |
| 控制 RF300 天线 | **Set_Ant_RF300** |
| 控制 RF600 天线 | **Set_Ant_RF600** |
| 识别 UHF 标签列表 | **Inventory** |

### ③ 计算/选型示例

- 已知：RF600 工位需要先识别区域内标签，再读取 EPC 数据，最后写入新的追溯信息
- 公式/规则：先盘点，再读取，再写入
- 操作：先用 **Inventory** 识别标签；再用 **Read_EPC_Mem** 读取 EPC 数据；最后用 **Write_EPC_Mem** 写入追溯信息
- 结论：这是典型的 RF600 UHF 标签高级读写流程

> [!tip] 记忆口诀
> **UHF 先盘点，确认目标再读写；EPC 是应用码，TID 是芯片码。**
