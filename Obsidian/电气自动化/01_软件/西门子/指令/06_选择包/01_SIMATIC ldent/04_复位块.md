| 功能描述 | **复位 MOBY D 系统** |
| 版本 | **V4.4** |
| 核心作用 | 复位 MOBY D 识别系统 |
| 使用重点 | 复位前确认当前读写任务是否允许中断 |

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 通信异常 | 读写器或系统无响应 |

| 系统卡滞 | 任务状态无法正常结束 |
| 维护调试 | 恢复系统到可重新操作状态 |

### ③ 计算/选型示例

- 已知：MOBY D 系统连续读取失败，状态无法恢复
- 公式/规则：系统异常后执行复位
- 操作：使用 **Reset_MOBY_D**
- 结论：系统复位后，可重新执行读写任务

> [!tip] 记忆口诀
> **MOBY D 异常卡住，Reset_MOBY_D 先复位。**

## 08 复位块 | 选型速记

> [!info] 核心结论
> 复位块的核心是：**哪个系统异常，就选对应 Reset；型号不确定或统一封装，用 Reset_Univ。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
MOBY 找 MOBY，RF 找 RF，光学找 MV；通用复位用 Univ。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| MOBY D 系统复位 | **Reset_MOBY_D** |
| MOBY U 系统复位 | **Reset_MOBY_U** |
| 光学识别系统复位 | **Reset_MV** |
| RF200 系统复位 | **Reset_RF200** |
| RF300 系统复位 | **Reset_RF300** |
| RF600 系统复位 | **Reset_RF600** |
| 通用 Ident 系统复位 | **Reset_Univ** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 复位 MOBY D 系统 | **Reset_MOBY_D** |
| 复位 MOBY U 系统 | **Reset_MOBY_U** |
| 复位光学识别系统 | **Reset_MV** |
| 复位 RF200 系统 | **Reset_RF200** |
| 复位 RF300 系统 | **Reset_RF300** |
| 复位 RF600 系统 | **Reset_RF600** |
| 复位通用 Ident 系统 | **Reset_Univ** |
| 读写异常排查 | 先查状态，再复位 |
| 复位后处理 | 重新初始化，再执行读写 |

### ③ 计算/选型示例

- 已知：RF600 工位连续盘点失败，阅读器状态异常
- 公式/规则：RF600 系统异常用对应专用复位块
- 操作：先读取状态信息，确认任务可中断后执行 **Reset_RF600**
- 结论：复位后重新执行盘点和 EPC 读写流程

> [!tip] 记忆口诀
> **先查状态再复位，复位完成再重试。**
