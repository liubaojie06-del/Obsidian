# 01 Polyline｜线性插值

> [!info] 核心结论
> **Polyline 用于根据已知点表进行线性插值，把输入值转换成对应输出值。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#173B63;">功能定位</strong><br>
适合做传感器标定、非线性曲线补偿、工程量换算。
</div>

## ① 分类/公式/规则

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br><br>

$$
y = y_1 + \frac{x - x_1}{x_2 - x_1} \times (y_2 - y_1)
$$
</div>

| 规则 | 说明 |
|---|---|
| 输入在两点之间 | 按比例线性插值 |
| 输入小于最小点 | 通常限幅或外推，需校核 |
| 输入大于最大点 | 通常限幅或外推，需校核 |
| 点表顺序 | 一般要求 X 值递增 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **X_Input** | 当前输入值 |
| **Y_Output** | 插值输出值 |
| **X_Array** | X 坐标点表 |
| **Y_Array** | Y 坐标点表 |
| **PointNum** | 点数量 |
| **Error/Status** | 点表错误、越界等状态，需按厂家手册校核 |

## ③ 计算/选型示例

- 已知：
  - 点 1：`x1 = 0`，`y1 = 0`
  - 点 2：`x2 = 100`，`y2 = 200`
  - 当前输入：`x = 40`

- 公式：

$$
y = 0 + \frac{40 - 0}{100 - 0} \times (200 - 0)
$$

- 计算：

$$
y = 80
$$

- 结论：
  - 输入为 `40` 时，输出为 **80**。

> [!tip] 记忆口诀
> **两点定直线，输入找区间，比例算输出。**

# 02 SplitRange｜分程 PID 控制

> [!info] 核心结论
> **SplitRange 用于把一个 PID 输出拆分成多个控制段，分别驱动不同执行器。**

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">典型场景</strong><br>
一个 PID 同时控制加热/冷却、大阀/小阀、粗调/精调、两路执行机构。
</div>

## ① 分类/公式/规则

| 分程方式 | 说明 |
|---|---|
| 顺序分程 | 先开 A，再开 B |
| 互补分程 | A 增大时 B 减小 |
| 重叠分程 | A、B 在部分区间同时动作 |
| 死区分程 | 中间留死区，防止频繁切换 |

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#173B63;">常见规则</strong><br><br>

- PID 输出 `0% ~ 50%`：执行器 A 动作  
- PID 输出 `50% ~ 100%`：执行器 B 动作  
- 是否设置死区/重叠区：**需按工艺和厂家手册校核**
</div>

## ② 参考表

| 参数 | 作用 |
|---|---|
| **PID_Out** | PID 总输出 |
| **Out_A** | 执行器 A 输出 |
| **Out_B** | 执行器 B 输出 |
| **Range_A** | A 的工作范围 |
| **Range_B** | B 的工作范围 |
| **DeadBand** | 死区，减少抖动 |
| **Overlap** | 重叠区，平滑切换 |
| **Mode** | 分程模式，需按手册确认 |

## ③ 计算/选型示例

- 已知：
  - PID 输出：`70%`
  - A 段：`0% ~ 50%`
  - B 段：`50% ~ 100%`

- 公式/规则：
  - `PID_Out ≤ 50%`：A 动作
  - `PID_Out > 50%`：B 动作

- 计算：
  - `70%` 已超过 A 段
  - B 段开始承担调节

- 结论：
  - 当前主要由 **B 执行器**控制。

> [!tip] 记忆口诀
> **一个 PID 管全局，多个执行器分段出力。**

# 03 RampFunction｜限制信号变化率

> [!info] 核心结论
> **RampFunction 用于限制信号变化速度，让输出按斜坡平滑变化。**

<div class="card green" style="background:#F4FBF7;border-left:6px solid #27AE60;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#27AE60;">功能定位</strong><br>
防止设定值突变、执行器冲击、电机/阀门动作过猛。
</div>

## ① 分类/公式/规则

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br><br>

$$
\Delta y_{max} = Rate \times T_s
$$
</div>

| 规则 | 说明 |
|---|---|
| 上升限制 | 限制输出增加速度 |
| 下降限制 | 限制输出减少速度 |
| 双向不同速率 | 上升、下降可设置不同斜率 |
| 复位 | 输出可回到初始值，需按手册确认 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **Input** | 目标输入值 |
| **Output** | 斜坡处理后的输出 |
| **RateUp** | 最大上升速率 |
| **RateDown** | 最大下降速率 |
| **Ts/CycleTime** | 采样周期 |
| **Enable** | 功能使能 |
| **Reset** | 复位输出状态 |

## ③ 计算/选型示例

- 已知：
  - 当前输出：`20`
  - 目标输入：`80`
  - 上升速率：`10 /s`
  - 采样周期：`0.1 s`

- 公式：

$$
\Delta y_{max} = 10 \times 0.1 = 1
$$

- 计算：
  - 每个周期最多增加 `1`
  - 输出变化：`20 → 21 → 22 → ...`

- 结论：
  - 输出不会从 `20` 突跳到 `80`，而是按斜坡平滑上升。

> [!tip] 记忆口诀
> **目标可以跳，输出慢慢跑。**

# 04 RampSoak｜斜坡与保持

> [!info] 核心结论
> **RampSoak 用于生成“斜坡升降 + 平台保持”的工艺曲线。**

<div class="card purple" style="background:#FAF5FF;border-left:6px solid #9B59B6;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#9B59B6;">典型应用</strong><br>
温控炉、烘箱、热处理、老化测试、工艺配方曲线。
</div>

## ① 分类/公式/规则

| 阶段 | 说明 |
|---|---|
| Ramp | 按设定速率升高或降低 |
| Soak | 到达目标后保持一段时间 |
| Segment | 一个斜坡段或保持段 |
| Profile | 多段组合成完整工艺曲线 |

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br><br>

$$
Rate = \frac{Target - Start}{RampTime}
$$
</div>

## ② 参考表

| 参数 | 作用 |
|---|---|
| **StartValue** | 起始值 |
| **TargetValue** | 目标值 |
| **RampTime** | 斜坡时间 |
| **SoakTime** | 保持时间 |
| **SegmentNo** | 当前段号 |
| **Run/Hold/Reset** | 运行/暂停/复位 |
| **Done** | 曲线完成标志 |
| **Status** | 当前状态，需按厂家手册校核 |

## ③ 计算/选型示例

- 已知：
  - 起始温度：`25 ℃`
  - 目标温度：`125 ℃`
  - 升温时间：`20 min`
  - 保温时间：`30 min`

- 公式：

$$
Rate = \frac{125 - 25}{20}
$$

- 计算：

$$
Rate = 5 ℃/min
$$

- 结论：
  - 系统先以 **5 ℃/min** 升温到 `125 ℃`
  - 再保持 **30 min**

> [!tip] 记忆口诀
> **先爬坡，到平台，稳住时间再换段。**

# 05 Filter_PT1｜一阶惯性滤波

> [!info] 核心结论
> **Filter_PT1 是一阶低通滤波，用于平滑信号并抑制高频噪声。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#173B63;">功能定位</strong><br>
适合压力、温度、液位、模拟量采集等轻度抖动信号。
</div>

## ① 分类/公式/规则

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br><br>

连续形式：

$$
G(s)=\frac{1}{T s + 1}
$$

离散近似：

$$
y(k)=y(k-1)+\alpha[x(k)-y(k-1)]
$$

$$
\alpha=\frac{T_s}{T+T_s}
$$
</div>

| 选型规则 | 结果 |
|---|---|
| **T 小** | 响应快，滤波弱 |
| **T 大** | 响应慢，滤波强 |
| Ts 小 | 输出更平滑 |
| Ts 大 | 可能滞后明显 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **Input** | 原始输入信号 |
| **Output** | 滤波后信号 |
| **T/TimeConstant** | 时间常数 |
| **Ts/CycleTime** | 采样周期 |
| **Enable** | 滤波使能 |
| **Reset** | 复位滤波状态 |

## ③ 计算/选型示例

- 已知：
  - 当前输入：`100`
  - 上次输出：`80`
  - 时间常数：`T = 1 s`
  - 采样周期：`Ts = 0.1 s`

- 公式：

$$
\alpha=\frac{0.1}{1+0.1}=0.091
$$

- 计算：

$$
y=80+0.091 \times (100-80)
$$

$$
y=81.82
$$

- 结论：
  - 输入突变到 `100`，输出先变为约 **81.82**，不会立刻跳变。

> [!tip] 记忆口诀
> **T 小跟得快，T 大滤得稳。**

# 06 Filter_PT2｜二阶惯性滤波

> [!info] 核心结论
> **Filter_PT2 是二阶低通滤波，比 PT1 滤波更强，但响应更慢。**

<div class="card purple" style="background:#FAF5FF;border-left:6px solid #9B59B6;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#9B59B6;">功能定位</strong><br>
适合噪声较大、需要更平滑输出的模拟量信号。
</div>

## ① 分类/公式/规则

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">典型传递函数</strong><br><br>

$$
G(s)=\frac{1}{T^2s^2+2DTs+1}
$$

其中：  
`T` 为时间常数，`D` 为阻尼系数。具体形式需按厂家手册校核。
</div>

| 对比项 | PT1 | PT2 |
|---|---|---|
| 阶数 | 一阶 | 二阶 |
| 滤波强度 | 中等 | 更强 |
| 响应速度 | 较快 | 较慢 |
| 参数复杂度 | 简单 | 较高 |
| 适用信号 | 普通抖动 | 噪声明显 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **Input** | 原始输入信号 |
| **Output** | 二阶滤波输出 |
| **T/TimeConstant** | 时间常数 |
| **D/Damping** | 阻尼系数，需按手册确认 |
| **Ts/CycleTime** | 采样周期 |
| **Enable** | 滤波使能 |
| **Reset** | 复位状态 |

## ③ 计算/选型示例

- 已知：
  - 温度采样值抖动明显
  - PT1 滤波后仍有波动
  - 控制系统允许一定滞后

- 公式/规则：
  - 噪声大 → 增强滤波
  - 响应要求快 → 不宜过大时间常数

- 计算：
  - 先选择较小 `T`
  - 观察波动与滞后
  - 若仍抖动，再逐步增加 `T` 或调整阻尼参数

- 结论：
  - PT2 适合 **更平滑但可接受滞后** 的场景。

> [!tip] 记忆口诀
> **PT1 快，PT2 稳；越稳越慢要权衡。**

# 07 Filter_DT1｜一阶微分滤波

> [!info] 核心结论
> **Filter_DT1 用于实现带滤波的一阶微分环节，常用于提取变化趋势并抑制微分噪声。**

<div class="card green" style="background:#F4FBF7;border-left:6px solid #27AE60;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#27AE60;">功能定位</strong><br>
适合速度估算、变化率检测、PID 微分前处理、趋势判断。
</div>

## ① 分类/公式/规则

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">典型传递函数</strong><br><br>

$$
G(s)=\frac{T_d s}{T_f s + 1}
$$

其中：  
`Td` 为微分时间，`Tf` 为滤波时间常数。具体实现需按厂家手册校核。
</div>

| 规则 | 说明 |
|---|---|
| 微分作用 | 关注输入变化速度 |
| 滤波作用 | 抑制微分放大的噪声 |
| Td 增大 | 微分作用增强 |
| Tf 增大 | 滤波更强，但响应变慢 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **Input** | 原始输入信号 |
| **Output** | 微分滤波输出 |
| **Td/DiffTime** | 微分时间 |
| **Tf/FilterTime** | 滤波时间 |
| **Ts/CycleTime** | 采样周期 |
| **Enable** | 功能使能 |
| **Reset** | 复位内部状态 |

## ③ 计算/选型示例

- 已知：
  - 上次输入：`50`
  - 当前输入：`55`
  - 采样周期：`0.1 s`

- 公式：

$$
Rate = \frac{x(k)-x(k-1)}{T_s}
$$

- 计算：

$$
Rate = \frac{55-50}{0.1}=50/s
$$

- 结论：
  - 输入变化率为 **50/s**
  - 实际 DT1 输出还会经过滤波处理，具体结果需按厂家算法校核。

> [!tip] 记忆口诀
> **微分看变化，滤波防噪声。**
