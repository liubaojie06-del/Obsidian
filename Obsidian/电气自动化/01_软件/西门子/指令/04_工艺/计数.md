# 01 CTRL_HSC_EXT｜扩展高速计数器

> [!info] 核心结论
> **CTRL_HSC_EXT 用于扩展高速计数器功能，适合高速脉冲采集、频率测量、位置计数和复杂计数控制。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#173B63;">功能定位</strong><br>
用于高速脉冲信号处理，比普通计数更适合编码器、流量计、测速脉冲、位置反馈等场景。
</div>

## ① 分类/公式/规则

| 类型 | 说明 |
|---|---|
| 单相计数 | 只统计一个脉冲输入 |
| 双相计数 | A/B 相编码器计数 |
| 加减计数 | 根据方向信号增减 |
| 频率测量 | 根据单位时间内脉冲数计算频率 |
| 位置测量 | 根据累计脉冲换算位移或角度 |

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br><br>

频率计算：

$$
f = \frac{N}{T}
$$

转速计算：

$$
n = \frac{f \times 60}{PPR}
$$

位移计算：

$$
S = \frac{Count}{PPR} \times L
$$
</div>

| 规则 | 说明 |
|---|---|
| **脉冲频率高** | 必须使用高速计数器 |
| **编码器计数** | 优先确认 A/B 相模式 |
| **方向判断** | 看方向信号或 A/B 相相序 |
| **清零/装载** | 需明确复位条件 |
| **硬件限制** | 最大频率需按 PLC 手册校核 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **HSC_ID** | 高速计数器编号 |
| **Enable** | 功能使能 |
| **CountValue** | 当前计数值 |
| **PresetValue** | 预置值 |
| **Direction** | 当前计数方向 |
| **Reset** | 清零计数 |
| **Load** | 装载预置值 |
| **Gate** | 门控允许计数 |
| **Frequency** | 频率值，需按实际接口确认 |
| **Status/Error** | 状态与错误信息 |
| **Version** | 截图显示：V1.0 |

> [!warning] 注意
> 参数名称、支持模式、最大计数频率、硬件通道映射，**需按实际 PLC 型号和厂家手册校核**。

## ③ 计算/选型示例

- 已知：
  - 编码器每转脉冲数：`PPR = 1000`
  - 测得频率：`f = 5000 Hz`
  - 需要计算电机转速

- 公式：

$$
n = \frac{f \times 60}{PPR}
$$

- 计算：

$$
n = \frac{5000 \times 60}{1000}
$$

$$
n = 300 r/min
$$

- 结论：
  - 当前转速约为 **300 r/min**。
  - 若使用 A/B 相 4 倍频计数，实际 PPR 需乘以 4，需按编码器模式校核。

<div class="card green" style="background:#F4FBF7;border-left:6px solid #27AE60;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#27AE60;">选型口诀</strong><br>
脉冲慢用普通计数，脉冲快上 HSC；要测速看频率，要定位看累计。
</div>

> [!tip] 记忆口诀
> **高速脉冲别硬扫，HSC 专门来计数。**

# 02 CTRL_HSC｜控制高速计数器

> [!info] 核心结论
> **CTRL_HSC 用于控制高速计数器的启动、停止、清零、装载和状态读取，是 HSC 的基础控制功能块。**

<div class="card blue" style="background:#F7F9FC;border-left:6px solid #173B63;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#173B63;">功能定位</strong><br>
适合对高速计数器进行基础控制，例如启停、复位、读取当前值、判断到达目标值。
</div>

## ① 分类/公式/规则

| 控制动作 | 说明 |
|---|---|
| 启动计数 | 允许 HSC 开始计数 |
| 停止计数 | 暂停或禁止计数 |
| 清零计数 | 当前值归零 |
| 装载预置值 | 将指定值写入计数器 |
| 读取当前值 | 获取高速计数结果 |
| 比较判断 | 判断是否到达设定值 |

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">重点规则</strong><br><br>

高速计数器不是普通程序扫描计数，通常由硬件高速通道完成计数，程序只负责配置、控制和读取。
</div>

| 判断场景 | 常用逻辑 |
|---|---|
| 到达目标值 | `CountValue >= PresetValue` |
| 反向到达 | `CountValue <= PresetValue` |
| 清零条件 | 原点信号、启动前复位、批次结束 |
| 报警条件 | 超范围、硬件错误、配置错误 |

## ② 参考表

| 参数 | 作用 |
|---|---|
| **HSC_ID** | 高速计数器编号 |
| **Enable** | 使能控制块 |
| **Start/Run** | 启动计数 |
| **Stop** | 停止计数 |
| **Reset** | 清零当前计数 |
| **Load** | 装载预置值 |
| **PresetValue** | 目标值或预置值 |
| **CurrentValue** | 当前计数值 |
| **Done** | 操作完成标志 |
| **Busy** | 操作执行中 |
| **Error** | 错误标志 |
| **Status** | 状态码，需按厂家手册校核 |
| **Version** | 截图显示：V1.0 |

## ③ 计算/选型示例

- 已知：
  - 目标切断长度：`L = 500 mm`
  - 编码器一圈脉冲数：`PPR = 1000`
  - 轮周长：`C = 200 mm`
  - 需要计算目标脉冲数

- 公式：

$$
Pulse = \frac{L}{C} \times PPR
$$

- 计算：

$$
Pulse = \frac{500}{200} \times 1000
$$

$$
Pulse = 2500
$$

- 结论：
  - 当高速计数器累计到 **2500 脉冲** 时，长度约为 `500 mm`。
  - 实际还需考虑机械滑差、编码器倍频、轮径误差，需现场校准。

<div class="card purple" style="background:#FAF5FF;border-left:6px solid #9B59B6;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#9B59B6;">控制口诀</strong><br>
先配硬件通道，再使能 HSC；先清零，再计数，到点输出动作。
</div>

> [!tip] 记忆口诀
> **CTRL_HSC 管动作，HSC 硬件管脉冲。**

# 03 高速计数器｜选型速记

> [!info] 核心结论
> **只要脉冲频率超过普通程序扫描可靠计数能力，就应优先考虑高速计数器。**

<div class="card orange" style="background:#FFF7F2;border-left:6px solid #FF6B35;border-radius:14px;padding:14px;margin:12px 0;">
<strong style="color:#FF6B35;">选型重点</strong><br>
高速计数器选型不是只看功能块，还要看 PLC 型号、输入点类型、最大频率、编码器模式和接线方式。
</div>

## ① 分类/公式/规则

| 选型问题 | 判断方法 |
|---|---|
| 是否需要 HSC | 脉冲快、普通扫描易漏计 |
| 是否需要方向 | 看是否有方向信号或 A/B 相 |
| 是否需要测速 | 看是否要计算频率 |
| 是否需要定位 | 看是否要累计脉冲 |
| 是否需要清零 | 看是否有原点/复位信号 |
| 是否需要中断 | 看是否到点立即动作 |

## ② 参考表

| 项目 | 校核内容 |
|---|---|
| **PLC 型号** | 是否支持 HSC |
| **输入点** | 是否为高速输入 |
| **最高频率** | 需大于现场最大脉冲频率 |
| **电气类型** | PNP/NPN、24V、差分信号等 |
| **计数模式** | 单相、双相、A/B 相 |
| **滤波时间** | 过大会漏高速脉冲 |
| **中断/比较** | 是否支持高速比较输出 |
| **厂家手册** | 所有边界参数必须最终校核 |

## ③ 计算/选型示例

- 已知：
  - 编码器：`1000 PPR`
  - 最高转速：`1500 r/min`
  - A/B 相 4 倍频计数

- 公式：

$$
f = \frac{PPR \times n}{60} \times 4
$$

- 计算：

$$
f = \frac{1000 \times 1500}{60} \times 4
$$

$$
f = 100000 Hz
$$

- 结论：
  - 最大输入频率约为 **100 kHz**。
  - PLC 高速输入能力必须大于 `100 kHz`，并留有余量，需按实际规范/厂家手册校核。

> [!tip] 记忆口诀
> **算频率，看通道，查手册，留余量。**
