## 01 `MB_CLIENT` | 通过 PROFINET 进行通信

> [!info] 核心结论
> **MB_CLIENT** 用于让 PLC 作为 **Modbus TCP 客户端**，通过 PROFINET/以太网主动访问 Modbus TCP 服务器设备。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">版本</strong><br>
MODBUS TCP 分类版本：<strong>V6.0</strong><br>
指令版本：<strong>V6.0</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **其它 / MODBUS TCP** |
| 指令名称 | **MB_CLIENT** |
| 功能描述 | **通过 PROFINET 进行通信** |
| 版本 | **V6.0** |
| 通信角色 | **Modbus TCP Client 客户端** |
| 核心作用 | 主动读写 Modbus TCP 服务器数据 |
| 使用重点 | IP 地址、端口、功能码、寄存器地址、数据长度需按设备手册校核 |

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点规则</strong><br>
Client 是<strong>主动方</strong>：主动连接、主动读取、主动写入。
</div>

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 读取仪表数据 | 如电表、流量计、温控表 |
| 写入设定值 | 向变频器、温控器下发参数 |
| PLC 采集第三方设备 | 通过 Modbus TCP 获取数据 |
| 主动轮询设备 | PLC 定时读取多个寄存器 |
| 风险提醒 | 地址偏移、大小端、数据类型需校核 |

### ③ 计算/选型示例

- 已知：PLC 需要读取一台电表的有功功率寄存器
- 公式/规则：PLC 主动读取第三方 Modbus TCP 设备
- 操作：选择 **MB_CLIENT**，设置服务器 IP、端口、寄存器地址和读取长度
- 结论：PLC 可作为客户端读取电表数据

> [!tip] 记忆口诀
> **主动去读写，用 MB_CLIENT。**


## 02 `MB_SERVER` | 通过 PROFINET 进行通信

> [!info] 核心结论
> **MB_SERVER** 用于让 PLC 作为 **Modbus TCP 服务器**，等待客户端读取或写入 PLC 数据。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">版本</strong><br>
指令版本：<strong>V5.3</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **其它 / MODBUS TCP** |
| 指令名称 | **MB_SERVER** |
| 功能描述 | **通过 PROFINET 进行通信** |
| 版本 | **V5.3** |
| 通信角色 | **Modbus TCP Server 服务器** |
| 核心作用 | 等待外部客户端访问 PLC 数据 |
| 使用重点 | 数据映射区、访问权限、端口、寄存器对应关系需校核 |

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">重点规则</strong><br>
Server 是<strong>被动方</strong>：等待别人连接、读取或写入。
</div>

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 上位机读取 PLC 数据 | PLC 提供寄存器数据 |
| 第三方系统采集 PLC | MES/SCADA 读取状态和产量 |
| 外部设备写入命令 | 允许客户端写设定值或控制字 |
| 数据共享 | PLC 开放一段寄存器映射区 |
| 风险提醒 | 写入类数据必须加权限和联锁 |

### ③ 计算/选型示例

- 已知：上位机需要通过 Modbus TCP 读取 PLC 的产量、运行状态和报警字
- 公式/规则：外部客户端访问 PLC，PLC 应作为服务器
- 操作：选择 **MB_SERVER**，配置数据映射区
- 结论：上位机可读取 PLC 中开放的数据区域

> [!tip] 记忆口诀
> **等别人来读写，用 MB_SERVER。**

## 05 MODBUS TCP | 选型速记

> [!info] 核心结论
> MODBUS TCP 选型先看角色：**主动访问别人用 Client，等待别人访问用 Server；普通系统用 MB，冗余系统用 MB_RED。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
主动读写 Client，被动开放 Server；普通通信 MB，冗余通信 MB_RED。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 普通客户端通信 | **MB_CLIENT** |
| 普通服务器通信 | **MB_SERVER** |
| 冗余客户端通信 | **MB_RED_CLIENT** |
| 冗余服务器通信 | **MB_RED_SERVER** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| PLC 主动读取电表/仪表 | **MB_CLIENT** |
| PLC 主动写入第三方设备参数 | **MB_CLIENT** |
| 上位机读取 PLC 数据 | **MB_SERVER** |
| 外部设备写入 PLC 数据 | **MB_SERVER** |
| 冗余 PLC 主动访问外部设备 | **MB_RED_CLIENT** |
| 外部系统访问冗余 PLC | **MB_RED_SERVER** |

### ③ 计算/选型示例

- 已知：PLC 要读取第三方电表，同时 SCADA 也要读取 PLC 数据
- 公式/规则：PLC 主动访问电表 = Client；SCADA 访问 PLC = PLC 做 Server
- 操作：读取电表用 **MB_CLIENT**；开放 PLC 数据给 SCADA 用 **MB_SERVER**
- 结论：一个项目中 Client 和 Server 可以同时存在，关键看谁主动访问谁

> [!tip] 记忆口诀
> **我去访问别人是 Client，别人来访问我是 Server。**
