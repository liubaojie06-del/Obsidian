## 01 `FileReadC` | 从存储卡中读取文件

> [!info] 核心结论
> **FileReadC** 用于从 PLC 存储卡中读取文件内容，适合读取配方、参数、日志或外部数据文件。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">版本</strong><br>
分类版本：<strong>V1.1</strong><br>
指令版本：<strong>V1.0</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **文件处理** |
| 指令名称 | **FileReadC** |
| 功能描述 | **从存储卡中读取文件** |
| 版本 | **V1.0** |
| 核心作用 | 从存储卡读取文件数据 |
| 使用重点 | 文件路径、文件名、缓冲区大小需按实际项目校核 |

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 读取配方文件 | 从存储卡加载产品参数 |
| 读取配置文件 | 加载设备配置或运行参数 |
| 读取日志文件 | 读取历史记录内容 |
| 读取通信文件 | 将外部数据导入 PLC |
| 参数细节 | 需按实际软件/厂家手册校核 |

### ③ 计算/选型示例

- 已知：存储卡中有一个参数文件，需要 PLC 读取后用于运行
- 公式/规则：从存储卡读文件用 **FileReadC**
- 操作：设置文件路径、文件名和接收缓冲区
- 结论：文件内容被读取到 PLC 变量区，后续可参与逻辑处理

> [!tip] 记忆口诀
> **读文件用 FileReadC，存储卡数据拿进来。**


## 02 `FileWriteC` | 将文件写入存储卡中

> [!info] 核心结论
> **FileWriteC** 用于把 PLC 中的数据写入存储卡文件，适合保存参数、日志和运行记录。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">版本</strong><br>
指令版本：<strong>V1.0</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **文件处理** |
| 指令名称 | **FileWriteC** |
| 功能描述 | **将文件写入存储卡中** |
| 版本 | **V1.0** |
| 核心作用 | 将数据写成存储卡文件 |
| 使用重点 | 写入频率、文件大小、存储卡寿命需校核 |

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 保存运行参数 | 将当前参数写入文件 |
| 保存生产记录 | 把批次数据写入存储卡 |
| 保存故障日志 | 记录报警、故障、时间戳 |
| 数据备份 | 断电或维护前保存关键数据 |
| 风险提醒 | 高频写入可能影响存储卡寿命 |

### ③ 计算/选型示例

- 已知：每批生产结束后，需要保存一份生产数据文件
- 公式/规则：PLC 数据写入存储卡用 **FileWriteC**
- 操作：准备写入数据、设置文件路径和文件名
- 结论：生产数据被保存到存储卡文件中，便于追溯

> [!tip] 记忆口诀
> **写文件用 FileWriteC，数据落盘好追溯。**

| 核心作用 | 删除指定存储卡文件 |

> [!tip] 记忆口诀

## 04 文件处理 | 选型速记

> [!info] 核心结论
> 文件处理的核心是：**读文件用 FileReadC，写文件用 FileWriteC，清理文件用删除文件。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
读卡用 Read，写卡用 Write，清空间才 Delete；路径文件名，执行前核对清。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 读取文件 | **FileReadC** |
| 写入文件 | **FileWriteC** |
| 删除文件 | **删除文件** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 从存储卡加载参数 | **FileReadC** |
| 从存储卡读取日志 | **FileReadC** |
| 保存运行参数到文件 | **FileWriteC** |
| 保存生产数据到存储卡 | **FileWriteC** |
| 删除旧日志文件 | **删除文件** |
| 释放存储卡空间 | **删除文件** |

### ③ 计算/选型示例

- 已知：设备需要先读取配方文件，生产结束后写入日志，并定期清理旧日志
- 公式/规则：读、写、删分别使用对应文件处理指令
- 操作：用 **FileReadC** 读取配方；用 **FileWriteC** 写入日志；确认后用 **删除文件** 清理旧文件
- 结论：这是典型的 PLC 存储卡文件管理流程

> [!tip] 记忆口诀
> **文件三步走：读用 Read，写用 Write，删前先核对。**
