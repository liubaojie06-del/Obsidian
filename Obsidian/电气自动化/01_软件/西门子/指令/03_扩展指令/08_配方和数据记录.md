## 01 `RecipeExport` | 导出配方

> [!info] 核心结论
> **RecipeExport** 用于把 PLC/HMI 中的配方数据导出到文件，方便备份、复制和维护。

</div>

| 项目 | 内容 |
|---|---|
| 所属分类 | **配方和数据记录 / 配方函数** |
| 指令名称 | **RecipeExport** |
| 功能描述 | **导出配方** |
| 版本 | **V1.2** |
| 核心作用 | 将配方数据导出为文件 |
| 使用重点 | 文件路径、配方编号、数据格式需按实际项目校核 |

### ② 参考表

| 使用场景 | 说明 |
|---|---|
| 配方备份 | 将当前配方保存到外部文件 |
| 配方迁移 | 把设备 A 的配方复制到设备 B |
| 生产换型 | 保存不同产品的工艺参数 |
| 维护恢复 | 出现异常时可恢复原配方 |

### ③ 计算/选型示例

- 已知：设备当前产品配方需要备份到存储卡
- 公式/规则：导出配方数据用 **RecipeExport**
- 操作：设置配方来源、目标路径和文件名
- 结论：配方被导出为文件，便于后续恢复或复制

> [!tip] 记忆口诀
> **配方要备份，RecipeExport 导出去。**

## 10 配方和数据记录 | 选型速记

> [!info] 核心结论
> 配方和数据记录的核心是：**配方导入导出用 Recipe，过程数据保存用 DataLog。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
配方导出 Export，配方导入 Import；日志先 Create，再 Open，写入 Write，结束 Close。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 配方导出 | **RecipeExport** |
| 配方导入 | **RecipeImport** |
| 创建日志 | **DataLogCreate** |
| 打开日志 | **DataLogOpen** |
| 写入日志 | **DataLogWrite** |
| 清空日志 | **DataLogClear** |
| 关闭日志 | **DataLogClose** |
| 删除日志 | **DataLogDelete** |
| 新建日志文件 | **DataLogNewFile** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 备份当前配方 | **RecipeExport** |
| 恢复产品配方 | **RecipeImport** |
| 第一次建立数据日志 | **DataLogCreate** |
| 准备写入已有日志 | **DataLogOpen** |
| 保存一条生产数据 | **DataLogWrite** |
| 清除日志内容 | **DataLogClear** |
| 结束日志操作 | **DataLogClose** |
| 删除旧日志文件 | **DataLogDelete** |
| 按批次新建日志 | **DataLogNewFile** |

### ③ 计算/选型示例

- 已知：每个产品批次开始时导入配方，生产过程中记录温度和压力
- 公式/规则：配方管理用 Recipe，过程追溯用 DataLog
- 操作：用 **RecipeImport** 导入配方；用 **DataLogCreate / DataLogOpen / DataLogWrite / DataLogClose** 记录过程数据
- 结论：配方负责“用什么参数生产”，DataLog 负责“生产过程发生了什么”

> [!tip] 记忆口诀
> **配方管参数，日志管过程；Import/Export 管配方，DataLog 管记录。**
