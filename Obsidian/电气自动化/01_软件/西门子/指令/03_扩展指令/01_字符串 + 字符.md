## 24 字符串 + 字符 | 选型速记

> [!info] 核心结论
> 字符串处理的核心是：**移动用 S_MOVE，转换用 S_CONV，数值互转用 STRG_VAL/VAL_STRG，截取查找用 LEFT/RIGHT/MID/FIND。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
搬字符串用 S_MOVE，转字符串用 S_CONV；字转数 STRG_VAL，数转字 VAL_STRG；拼接 CONCAT，查找 FIND。
</div>

### ① 分类/公式/规则

| 类型     | 常用指令                                               |
| ------ | -------------------------------------------------- |
| 字符串移动  | **S_MOVE**                                         |
| 字符串转换  | **S_CONV**                                         |
| 字符串与数值 | **STRG_VAL / VAL_STRG**                            |
| 字符串与字符 | **Strg_TO_Chars / Chars_TO_Strg**                  |
| 长度获取   | **LEN / MAX_LEN**                                  |
| 字符串拼接  | **CONCAT**                                         |
| 字符串截取  | **LEFT / RIGHT / MID**                             |
| 编辑字符串  | **DELETE / INSERT / REPLACE**                      |
| 查找字符   | **FIND**                                           |
| 运行信息   | **GetSymbolName / GetInstanceName / GetBlockName** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 复制字符串 | **S_MOVE** |
| 字符串转数值 | **STRG_VAL** |
| 数值转字符串 | **VAL_STRG** |
| 字符串拆成字符 | **Strg_TO_Chars** |
| 字符组合成字符串 | **Chars_TO_Strg** |
| 获取当前长度 | **LEN** |
| 获取最大长度 | **MAX_LEN** |
| 合并两个字符串 | **CONCAT** |
| 从左/右/中间截取 | **LEFT / RIGHT / MID** |
| 查找字符位置 | **FIND** |
| 读取块名称 | **GetBlockName** |

### ③ 计算/选型示例

- 已知：HMI 输入 `"123"`，PLC 需要把它作为数值参与比较
- 公式/规则：字符串不能直接按数值比较
- 操作：先用 **STRG_VAL** 转成数值，再参与比较
- 结论：这是典型的**字符串转数值**场景

> [!tip] 记忆口诀
> **显示用字符串，计算用数值；两者互转，STRG_VAL 和 VAL_STRG。**
