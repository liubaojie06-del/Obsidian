## 01 `CTRL_PWM` | 脉宽调制

> [!info] 核心结论
> **CTRL_PWM** 用于控制 PWM 脉宽调制输出，通过调节占空比来控制平均输出效果。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">版本</strong><br>
分类版本：<strong>V1.1</strong><br>
指令版本：<strong>V1.0</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **脉冲** |
| 指令名称 | **CTRL_PWM** |
| 功能描述 | **脉宽调制** |
| 版本 | **V1.0** |
| 核心作用 | 输出 PWM 脉冲信号 |
| 关键参数 | 频率、周期、占空比等需按实际 CPU/模块手册校核 |

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br>
占空比 = 高电平时间 ÷ 周期 × 100%
</div>

### ② 参考表

| 参数/概念 | 说明 |
|---|---|
| 周期 | 一个完整 PWM 波形的时间 |
| 频率 | 每秒输出多少个周期 |
| 高电平时间 | 输出为 1 的持续时间 |
| 占空比 | **高电平时间占整个周期的比例** |
| 输出效果 | 常用于模拟量效果控制、速度/亮度/功率调节 |

### ③ 计算/选型示例

- 已知：PWM 周期为 `10 ms`，高电平时间为 `3 ms`
- 公式：`占空比 = 3 ÷ 10 × 100%`
- 计算：`占空比 = 30%`
- 结论：该 PWM 输出为 **30% 占空比**

> [!tip] 记忆口诀
> **PWM 调宽度，占空比决定输出力度。**


## 02 `CTRL_PTO` | 以指定频率输出一个脉冲串

> [!info] 核心结论
> **CTRL_PTO** 用于按指定频率输出脉冲串，常用于步进电机、伺服脉冲输入或高速脉冲控制。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">版本</strong><br>
分类版本：<strong>V1.1</strong><br>
指令版本：<strong>V1.0</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **脉冲** |
| 指令名称 | **CTRL_PTO** |
| 功能描述 | **以指定频率输出一个脉冲串** |
| 版本 | **V1.0** |
| 核心作用 | 输出 PTO 脉冲串 |
| 关键参数 | 频率、脉冲数、方向、加减速等需按实际 CPU/模块手册校核 |

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">重点公式</strong><br>
脉冲数 = 目标位移 ÷ 单脉冲位移
</div>

### ② 参考表

| 参数/概念 | 说明 |
|---|---|
| 频率 | 决定脉冲输出速度 |
| 脉冲数 | 决定运动距离或动作次数 |
| 方向 | 控制正转/反转，需结合硬件接线 |
| 加减速 | 防止启动/停止冲击 |
| 应用对象 | 步进驱动器、伺服脉冲输入、高速输出模块 |

### ③ 计算/选型示例

- 已知：步进机构每个脉冲移动 `0.01 mm`，目标移动 `50 mm`
- 公式：`脉冲数 = 目标位移 ÷ 单脉冲位移`
- 计算：`50 ÷ 0.01 = 5000`
- 结论：需要输出 **5000 个脉冲**，可使用 **CTRL_PTO**

> [!tip] 记忆口诀
> **PTO 管脉冲串，频率管速度，脉冲数管距离。**


## 03 脉冲 | 选型速记

> [!info] 核心结论
> 脉冲指令选型先看目的：**调功率/亮度/平均输出用 PWM，控制步进/伺服位置用 PTO。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
调占空比用 PWM，发脉冲串用 PTO；PWM 看宽度，PTO 看频率和脉冲数。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 脉宽调制 | **CTRL_PWM** |
| 脉冲串输出 | **CTRL_PTO** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 调节 LED 亮度 | **CTRL_PWM** |
| 调节加热平均功率 | **CTRL_PWM** |
| 控制步进电机运动 | **CTRL_PTO** |
| 控制伺服脉冲输入 | **CTRL_PTO** |
| 改变输出强弱 | 看 **PWM 占空比** |
| 改变运动距离 | 看 **PTO 脉冲数** |
| 改变运动速度 | 看 **PTO 频率** |

### ③ 计算/选型示例

- 已知：需要控制步进电机移动固定距离
- 公式/规则：固定距离需要按脉冲数控制
- 操作：选择 **CTRL_PTO**，计算所需脉冲数并设置输出频率
- 结论：这是典型的**脉冲串输出**场景

> [!tip] 记忆口诀
> **PWM 调强弱，PTO 控位移；一个看占空比，一个看脉冲数。**
