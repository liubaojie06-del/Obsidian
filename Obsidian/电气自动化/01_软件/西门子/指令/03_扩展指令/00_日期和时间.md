## 01 `T_CONV` | 转换时间并提取

> [!info] 核心结论
> **T_CONV** 用于进行时间格式转换，并可从时间数据中提取所需信息。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
时间格式不一致时，先用 <strong>T_CONV</strong> 做转换或提取。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **日期和时间** |
| 指令名称 | **T_CONV** |
| 功能描述 | **转换时间并提取** |
| 版本 | **V1.2** |
| 核心作用 | 时间格式转换 / 时间字段提取 |

### ② 参考表

| 常见需求 | 说明 |
|---|---|
| 时间格式转换 | 不同时间类型之间转换 |
| 提取日期 | 提取年、月、日等信息 |
| 提取时间 | 提取时、分、秒等信息 |
| 参数细节 | 需按实际软件/厂家手册校核 |

### ③ 计算/选型示例

- 已知：需要从系统时间中提取小时和分钟
- 公式/规则：时间数据需要先转换或拆分
- 操作：使用 **T_CONV**
- 结论：适合时间格式转换和字段提取

> [!tip] 记忆口诀
> **时间类型要转换，T_CONV 先来帮。**


## 02 `T_ADD` | 时间相加

> [!info] 核心结论
> **T_ADD** 用于把一个时间值与时间间隔相加，得到新的时间点。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点公式</strong><br>
新时间 = 起始时间 + 时间间隔
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **日期和时间** |
| 指令名称 | **T_ADD** |
| 功能描述 | **时间相加** |
| 版本 | **V1.2** |
| 核心作用 | 在时间点上增加一段时间 |

### ② 参考表

| 参数理解 | 含义 |
|---|---|
| 起始时间 | 原始时间点 |
| 时间间隔 | 需要增加的时间 |
| 输出时间 | 相加后的新时间 |

### ③ 计算/选型示例

- 已知：当前时间为 `10:00`，需要 30 分钟后执行动作
- 公式：`新时间 = 10:00 + 30min`
- 计算：`10:30`
- 结论：选择 **T_ADD 时间相加**

> [!tip] 记忆口诀
> **未来几点做，当前时间加一段。**

## 12 日期和时间 | 选型速记

> [!info] 核心结论
> 日期和时间指令的核心是：**转换提取用 T_CONV，时间加减用 T_ADD/T_SUB，求时差用 T_DIFF，系统时钟用 RD/WR。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
转换提取 T_CONV，加减时间 T_ADD/T_SUB；求差 T_DIFF，组合 T_COMBINE；读写时钟看 RD/WR。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 时间转换 | **T_CONV** |
| 时间计算 | **T_ADD / T_SUB / T_DIFF** |
| 时间组合 | **T_COMBINE** |
| 系统时间 | **RD_SYS_T / WR_SYS_T** |
| 本地时间 | **RD_LOC_T / WR_LOC_T** |
| 时区设置 | **SET_TIMEZONE** |
| 运行计时 | **RTM** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 提取时间字段 | **T_CONV** |
| 当前时间加 30 分钟 | **T_ADD** |
| 计划时间提前 10 分钟 | **T_SUB** |
| 计算运行持续多久 | **T_DIFF** |
| 日期和时间合成完整时间 | **T_COMBINE** |
| 读取系统时间 | **RD_SYS_T** |
| 设置系统时间 | **WR_SYS_T** |
| 读取本地时间 | **RD_LOC_T** |
| 设置时区 | **SET_TIMEZONE** |
| 统计设备运行时间 | **RTM** |

### ③ 计算/选型示例

- 已知：设备启动时记录开始时间，停止时计算运行时长
- 公式：`运行时长 = 停止时间 - 启动时间`
- 操作：启动时用 **RD_SYS_T** 读取开始时间，停止时再读取一次，然后用 **T_DIFF** 求时差
- 结论：这是典型的运行时长统计逻辑

> [!tip] 记忆口诀
> **读时间用 RD，写时间用 WR；算时间用 T，记工时用 RTM。**
