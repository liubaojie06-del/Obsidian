## 01 `CTU` | 加计数

> [!info] 核心结论
> **CTU** 是加计数器，每检测到一次计数脉冲，当前计数值 **CV 增加 1**。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
CTU 用于<strong>向上累计次数</strong>，达到预设值 <strong>PV</strong> 后，输出 <strong>Q = 1</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **计数器操作** |
| 指令名称 | **CTU** |
| 功能描述 | **加计数** |
| 版本 | **V1.0** |
| 核心作用 | 每次触发计数值加 1 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| CU | 加计数输入 |
| R | 复位输入 |
| PV | **预设计数值** |
| CV | 当前计数值 |
| Q | 到达预设值输出 |

### ③ 计算/选型示例

- 已知：产品经过传感器 10 次后，输出满料信号
- 公式：`PV = 10`
- 计算：每来 1 个产品，`CV + 1`；当 `CV ≥ 10` 时，`Q = 1`
- 结论：选择 **CTU 加计数**

> [!tip] 记忆口诀
> **CTU 向上数，数够 PV 就输出。**


## 02 `CTD` | 减计数

> [!info] 核心结论
> **CTD** 是减计数器，每检测到一次计数脉冲，当前计数值 **CV 减少 1**。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点规则</strong><br>
CTD 用于<strong>倒计数</strong>，常见场景是剩余数量、倒数次数、库存扣减。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **计数器操作** |
| 指令名称 | **CTD** |
| 功能描述 | **减计数** |
| 版本 | **V1.0** |
| 核心作用 | 每次触发计数值减 1 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| CD | 减计数输入 |
| LD | 装载输入 |
| PV | **预设装载值** |
| CV | 当前计数值 |
| Q | 到达判断输出 |

### ③ 计算/选型示例

- 已知：料仓初始有 20 个工件，每取走 1 个，剩余数减 1
- 公式：`CV = CV - 1`
- 计算：装载 `PV = 20`，每触发一次 CD，`CV` 减 1
- 结论：选择 **CTD 减计数**

> [!tip] 记忆口诀
> **CTD 向下数，越数越少看剩余。**


## 03 `CTUD` | 加减计数

> [!info] 核心结论
> **CTUD** 是加减计数器，既能加计数，也能减计数，适合统计数量的增减变化。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点规则</strong><br>
有进入就加，有离开就减；需要同时统计<strong>增加量与减少量</strong>时选 CTUD。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **计数器操作** |
| 指令名称 | **CTUD** |
| 功能描述 | **加减计数** |
| 版本 | **V1.0** |
| 核心作用 | 支持加计数和减计数 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| CU | 加计数输入 |
| CD | 减计数输入 |
| R | 复位输入 |
| LD | 装载输入 |
| PV | **预设计数值** |
| CV | 当前计数值 |
| QU | 上限到达输出 |
| QD | 下限到达输出 |

### ③ 计算/选型示例

- 已知：停车场车辆进入加 1，车辆离开减 1，满 50 辆报警
- 公式：进入 `CV + 1`，离开 `CV - 1`
- 计算：设置 `PV = 50`，当 `CV ≥ 50` 时输出满位信号
- 结论：选择 **CTUD 加减计数**

> [!tip] 记忆口诀
> **只进用 CTU，只出用 CTD，有进有出用 CTUD。**


## 04 计数器操作 | 选型速记

> [!info] 核心结论
> 计数器选型先看计数方向：**只增加选 CTU，只减少选 CTD，既增又减选 CTUD。**

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">总口诀</strong><br>
CTU 往上加，CTD 往下减，CTUD 加减都能干。
</div>

### ① 分类/公式/规则

| 需求 | 推荐指令 |
|---|---|
| 产品累计数量 | **CTU** |
| 剩余数量倒计数 | **CTD** |
| 库存/人数/车辆增减 | **CTUD** |
| 清零重新计数 | 使用 **R 复位** |
| 预置初始值 | 使用 **LD / PV**，需按实际指令参数校核 |

### ② 参考表

| 指令 | 关键词 |
|---|---|
| **CTU** | 加计数、累计、到达数量 |
| **CTD** | 减计数、倒计数、剩余数量 |
| **CTUD** | 加减计数、动态数量、进出统计 |

### ③ 计算/选型示例

- 已知：统计传送带通过的产品数量，达到 100 个后报警
- 公式/规则：每通过 1 个产品，数量加 1
- 操作：选择 **CTU**，设置 `PV = 100`
- 结论：这是典型的**加计数**场景

> [!tip] 记忆口诀
> **加用 U，减用 D，加减都有用 UD。**
