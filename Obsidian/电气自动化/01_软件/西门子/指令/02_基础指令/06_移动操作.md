## 01 `MOVE` | 移动值

> [!info] 核心结论
> **MOVE** 用于把一个变量或常量的值复制到目标变量中，是最常用的数据赋值指令。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
MOVE 是<strong>复制值</strong>，不是搬走源数据；源变量原值不变。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **移动操作** |
| 指令名称 | **MOVE** |
| 功能描述 | **移动值** |
| 核心作用 | 源值复制到目标变量 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| IN | 源数据 |
| OUT | **目标数据** |
| EN | 使能输入 |
| ENO | 使能输出 |

### ③ 计算/选型示例

- 已知：`IN = 100`
- 公式：`OUT = IN`
- 计算：执行 MOVE 后，`OUT = 100`
- 结论：适合普通变量赋值

> [!tip] 记忆口诀
> **MOVE 是复制，源不动，值到目标去。**

## 21 移动操作 | 选型速记

</div>

### ① 分类/公式/规则

|---|---|

| 字节流转换 | **Serialize / Deserialize** |
| 块移动 | **MOVE_BLK / UMOVE_BLK** |
| 块填充 | **FILL_BLK / UFILL_BLK** |
| 位拆分 | **SCATTER / SCATTER_BLK** |
| 位组合 | **GATHER / GATHER_BLK** |
| 数组边界 | **LOWER_BOUND / UPPER_BOUND** |
| VARIANT 读写 | **VariantGet / VariantPut** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 复制一个变量 | **MOVE** |
| 复制一段数组 | **MOVE_BLK** |
| 关键数据整块复制 | **UMOVE_BLK** |
| 数组全部清零 | **FILL_BLK** |
| 通信数据打包 | **Serialize** |
| 通信数据解析 | **Deserialize** |
| 状态字拆成位 | **SCATTER** |
| 多个位合成状态字 | **GATHER** |
| 获取数组上下限 | **LOWER_BOUND / UPPER_BOUND** |

- 结论：这是典型的**位组合**场景
