## 01 `CONVERT` | 转换值

> [!info] 核心结论
> **CONVERT** 用于把一个数据值转换成另一种数据类型，是类型转换的基础指令。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
不同数据类型参与运算前，先确认是否需要 <strong>CONVERT</strong> 转换。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **CONVERT** |
| 功能描述 | **转换值** |
| 核心作用 | 数据类型转换 |
| 注意事项 | 溢出、精度损失需按实际数据类型校核 |

### ② 参考表

| 常见转换 | 说明 |
|---|---|
| INT → REAL | 整数转浮点，便于小数计算 |
| REAL → INT | 浮点转整数，可能丢失小数 |
| DINT → REAL | 大整数参与比例运算 |
| WORD / BYTE 转换 | 需注意位宽和符号 |

### ③ 计算/选型示例

- 已知：整数计数值 `Count = 25`
- 公式：`RealValue = CONVERT(Count)`
- 计算：`25` 转换为 `25.0`
- 结论：转换后可参与 REAL 类型比例计算

> [!tip] 记忆口诀
> **类型不同先转换，算得稳来不报错。**


## 02 `ROUND` | 取整

> [!info] 核心结论
> **ROUND** 用于对浮点数进行四舍五入取整。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点规则</strong><br>
ROUND 按最接近的整数取值，具体 .5 处理规则需按实际软件/厂家手册校核。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **ROUND** |
| 功能描述 | **取整** |
| 核心作用 | 四舍五入取整数 |

### ② 参考表

| 输入值 | ROUND 结果 |
|---|---|
| 3.2 | 3 |
| 3.6 | 4 |
| -2.3 | -2 |
| -2.8 | -3 |

### ③ 计算/选型示例

- 已知：测量值 `Value = 12.7`
- 公式：`OUT = ROUND(Value)`
- 计算：`ROUND(12.7) = 13`
- 结论：输出整数为 **13**

> [!tip] 记忆口诀
> **ROUND 看最近，四舍五入取整数。**


## 03 `CEIL` | 浮点数向上取整

> [!info] 核心结论
> **CEIL** 用于把浮点数向上取整，结果取不小于输入值的最小整数。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点规则</strong><br>
CEIL 永远向<strong>更大的方向</strong>取整。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **CEIL** |
| 功能描述 | **浮点数向上取整** |
| 核心作用 | 向上取整 |

### ② 参考表

| 输入值 | CEIL 结果 |
|---|---|
| 3.1 | 4 |
| 3.9 | 4 |
| -2.1 | -2 |
| -2.9 | -2 |

### ③ 计算/选型示例

- 已知：每箱装 8 件，当前有 25 件
- 公式：`箱数 = CEIL(25 ÷ 8)`
- 计算：`25 ÷ 8 = 3.125`，`CEIL(3.125) = 4`
- 结论：至少需要 **4 箱**

> [!tip] 记忆口诀
> **CEIL 往上抬，不够一箱也算一箱。**


## 04 `FLOOR` | 浮点数向下取整

> [!info] 核心结论
> **FLOOR** 用于把浮点数向下取整，结果取不大于输入值的最大整数。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
FLOOR 永远向<strong>更小的方向</strong>取整。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **FLOOR** |
| 功能描述 | **浮点数向下取整** |
| 核心作用 | 向下取整 |

### ② 参考表

| 输入值 | FLOOR 结果 |
|---|---|
| 3.1 | 3 |
| 3.9 | 3 |
| -2.1 | -3 |
| -2.9 | -3 |

### ③ 计算/选型示例

- 已知：每组需要 8 件，当前有 25 件
- 公式：`完整组数 = FLOOR(25 ÷ 8)`

- 结论：可以组成 **3 个完整组**

> [!tip] 记忆口诀
> **FLOOR 往下压，只算完整不凑数。**


## 05 `TRUNC` | 截尾取整

> [!info] 核心结论
> **TRUNC** 用于直接去掉小数部分，只保留整数部分。

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">重点规则</strong><br>
TRUNC 不四舍五入，只<strong>截掉小数尾巴</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **TRUNC** |
| 功能描述 | **截尾取整** |
| 核心作用 | 去掉小数部分 |

### ② 参考表

| 输入值 | TRUNC 结果 |
|---|---|
| 3.1 | 3 |
| 3.9 | 3 |
| -2.1 | -2 |
| -2.9 | -2 |

### ③ 计算/选型示例

- 已知：计算结果 `Value = 18.95`
- 公式：`OUT = TRUNC(Value)`

- 结论：OUT = **18**

> [!tip] 记忆口诀
> **TRUNC 不进位，小数直接丢。**


## 06 `SCALE_X` | 缩放

> [!info] 核心结论
> **SCALE_X** 用于把标准化数值按上下限缩放成工程量，常用于模拟量换算。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点公式</strong><br>
工程值 = MIN + VALUE × (MAX - MIN)
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **SCALE_X** |
| 功能描述 | **缩放** |
| 核心作用 | 标准化值 → 工程量 |
| 输入范围 | VALUE 通常为 **0.0 ~ 1.0**，需按实际指令校核 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| MIN | 工程量下限 |
| VALUE | 标准化输入值 |
| MAX | 工程量上限 |
| OUT | **缩放后的工程值** |

### ③ 计算/选型示例

- 已知：标准化值 `VALUE = 0.75`，量程 `0 ~ 100℃`
- 公式：`OUT = 0 + 0.75 × (100 - 0)`
- 计算：`OUT = 75`
- 结论：工程量温度为 **75℃**

> [!tip] 记忆口诀
> **SCALE 是放大尺，0 到 1 变工程值。**


## 07 `NORM_X` | 标准化

> [!info] 核心结论
> **NORM_X** 用于把工程量按上下限转换成 0.0 ~ 1.0 的标准化数值。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点公式</strong><br>
标准化值 = (VALUE - MIN) ÷ (MAX - MIN)
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **转换操作** |
| 指令名称 | **NORM_X** |
| 功能描述 | **标准化** |
| 核心作用 | 工程量 → 标准化值 |
| 注意事项 | MAX 与 MIN 不能相等，需校核量程 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| MIN | 工程量下限 |
| VALUE | 当前工程量 |
| MAX | 工程量上限 |
| OUT | **标准化结果** |

### ③ 计算/选型示例

- 已知：温度 `VALUE = 75℃`，量程 `0 ~ 100℃`
- 公式：`OUT = (75 - 0) ÷ (100 - 0)`
- 计算：`OUT = 0.75`
- 结论：标准化结果为 **0.75**

> [!tip] 记忆口诀
> **NORM 是归一化，工程值变 0 到 1。**


## 08 转换操作 | 选型速记

> [!info] 核心结论
> 转换操作的核心是：**类型不同用 CONVERT，小数处理看取整方向，工程量换算用 NORM_X 和 SCALE_X。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
转换用 CONVERT，四舍五入 ROUND；向上 CEIL，向下 FLOOR，截尾 TRUNC；归一 NORM，放大 SCALE。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 类型转换 | **CONVERT** |
| 四舍五入 | **ROUND** |
| 向上取整 | **CEIL** |
| 向下取整 | **FLOOR** |
| 截尾取整 | **TRUNC** |
| 工程值转标准值 | **NORM_X** |
| 标准值转工程值 | **SCALE_X** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| INT 转 REAL | **CONVERT** |
| 小数四舍五入 | **ROUND** |
| 计算箱数/批次数，剩余也算一组 | **CEIL** |
| 只统计完整组数 | **FLOOR** |
| 直接去掉小数 | **TRUNC** |
| 模拟量变成 0~1 | **NORM_X** |
| 0~1 还原成工程量 | **SCALE_X** |

### ③ 计算/选型示例

- 已知：模拟量工程值 `75℃`，量程 `0 ~ 100℃`
- 公式/规则：工程量转标准化值用 **NORM_X**
- 操作：`(75 - 0) ÷ (100 - 0) = 0.75`
- 结论：标准化结果为 **0.75**

> [!tip] 记忆口诀
> **先转换类型，再处理小数；工程量换算，NORM 与 SCALE 成对用。**
