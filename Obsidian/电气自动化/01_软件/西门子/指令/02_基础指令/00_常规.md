## 01 插入程序段 | 插入程序段

> [!info] 核心结论
> **插入程序段**用于新增一段程序逻辑，适合把不同功能分块管理。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">功能定位</strong><br>
把复杂程序拆成多个小段，方便阅读、调试和维护。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **常规** |
| 指令名称 | **插入程序段** |
| 功能描述 | **插入程序段** |
| 快捷键 | 照片未显示，**需按软件版本校核** |
| 核心作用 | 新增程序段 / 网络段 |

### ② 参考表

| 使用场景 | 建议做法 |
|---|---|
| 功能较多 | 按功能拆分程序段 |
| 逻辑较长 | 每段只放一类逻辑 |
| 后期维护 | 程序段名称要清楚 |

### ③ 计算/选型示例

- 已知：需要编写“启动控制”和“报警输出”
- 规则：不同功能建议分段管理
- 操作：分别插入两个程序段
- 结论：程序结构更清晰，后期排查更方便

> [!tip] 记忆口诀
> **功能一变就分段，程序清楚不混乱。**


## 02 空功能框 | 空功能框 F8

> [!info] 核心结论
> **空功能框**用于插入待填写的功能指令，是调用定时器、计数器、比较、运算等功能的入口。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">快捷键</strong><br>
<strong>F8</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **常规** |
| 指令名称 | **空功能框** |
| 功能描述 | **空功能框** |
| 快捷键 | **F8** |
| 核心作用 | 插入功能指令框 |

### ② 参考表

| 常见功能 | 功能框用途 |
|---|---|
| 定时器 | 延时启动 / 延时停止 |
| 计数器 | 计数到达后动作 |
| 比较指令 | 判断数值大小 |
| 运算指令 | 数据计算处理 |

### ③ 计算/选型示例

- 已知：需要做一个延时 5 秒启动
- 公式/规则：延时控制通常使用定时器功能框
- 操作：按 **F8** 插入空功能框，再填写定时器指令
- 结论：空功能框是复杂功能指令的入口

> [!tip] 记忆口诀
> **简单触点直接放，复杂功能用 F8。**


## 03 打开分支 | 打开分支 Shift + F8

> [!info] 核心结论
> **打开分支**用于创建并联逻辑，让多个条件可以分别控制同一个结果。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">快捷键</strong><br>
<strong>Shift + F8</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **常规** |
| 指令名称 | **打开分支** |
| 功能描述 | **打开分支** |
| 快捷键 | **Shift + F8** |
| 核心作用 | 新建并联支路 |

### ② 参考表

| 逻辑关系 | 梯形图理解 |
|---|---|
| 串联 | 条件同时满足，类似 **AND** |
| 并联 | 条件任一满足，类似 **OR** |
| 打开分支 | 开始创建并联路径 |
| 嵌套闭合 | 结束当前分支结构 |

### ③ 计算/选型示例

- 已知：按钮 A 或按钮 B 任意一个按下，都能启动电机
- 公式/规则：A 或 B = **并联逻辑**
- 操作：按 **Shift + F8** 打开分支
- 结论：并联分支实现“任一条件成立即可动作”

> [!tip] 记忆口诀
> **遇到“或者”开分支，多个条件并着试。**


## 04 嵌套闭合 | 嵌套闭合 Shift + F9

> [!info] 核心结论
> **嵌套闭合**用于结束当前分支或嵌套结构，保证逻辑回路完整。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">快捷键</strong><br>
<strong>Shift + F9</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **常规** |
| 指令名称 | **嵌套闭合** |
| 功能描述 | **嵌套闭合** |
| 快捷键 | **Shift + F9** |
| 核心作用 | 闭合分支 / 嵌套结构 |

### ② 参考表

| 操作 | 作用 |
|---|---|
| 打开分支 | 创建并联起点 |
| 嵌套闭合 | 结束并联或嵌套 |
| 未闭合 | 可能导致逻辑错误 |
| 多层分支 | 需逐层闭合 |

### ③ 计算/选型示例

- 已知：已经用 **Shift + F8** 打开一个并联分支
- 公式/规则：分支结构必须完整闭合
- 操作：完成支路后，按 **Shift + F9** 闭合
- 结论：逻辑结构完整，程序更容易通过检查

> [!tip] 记忆口诀
> **开了分支要闭合，逻辑才算真完整。**


## 05 `--|` | 插入输入

> [!info] 核心结论
> **插入输入**用于放置输入触点，常表示按钮、开关、传感器等输入条件。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">符号识别</strong><br>
照片中名称栏显示：<strong>--|</strong>，描述为：<strong>插入输入</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **常规** |
| 指令名称 | **`--\|`** |
| 功能描述 | **插入输入** |
| 快捷键 | 照片未显示，**需按软件版本校核** |
| 核心作用 | 插入输入触点 / 条件 |

### ② 参考表

| 输入类型 | 常见含义 |
|---|---|
| 按钮 | 启动、停止、复位 |
| 限位开关 | 到位检测 |
| 接近开关 | 物体检测 |
| 传感器 | 温度、压力、液位信号 |

### ③ 计算/选型示例

- 已知：按下启动按钮后，电机运行
- 公式/规则：启动按钮属于**输入条件**
- 操作：插入输入触点，并填写对应输入地址
- 结论：输入触点成立后，后续输出逻辑才会动作

> [!tip] 记忆口诀
> **按钮开关传感器，先用输入做条件。**
