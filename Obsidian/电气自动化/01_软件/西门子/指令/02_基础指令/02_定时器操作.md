## 01 `TP` | 生成脉冲

> [!info] 核心结论
> **TP** 用于在输入信号触发后，生成一个固定持续时间的脉冲输出。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
触发一次，输出保持 <strong>PT</strong> 设定时间。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **定时器操作** |
| 指令名称 | **TP** |
| 功能描述 | **生成脉冲** |
| 版本 | **V1.0** |
| 核心作用 | 输出固定宽度脉冲 |

### ② 参考表

| 参数 | 含义 |
|---|---|
| IN | 启动输入 |
| PT | **预设脉冲时间** |
| Q | 脉冲输出 |
| ET | 当前已计时间 |

### ③ 计算/选型示例

- 已知：按钮按下后，蜂鸣器响 3 秒
- 公式：`PT = T#3s`
- 计算：触发后 Q 输出保持 3 秒
- 结论：选择 **TP 生成脉冲**

> [!tip] 记忆口诀
> **TP 打脉冲，触发一下响一阵。**


## 02 `TON` | 接通延时

> [!info] 核心结论
> **TON** 用于输入接通后延时输出，常用于延时启动。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点规则</strong><br>
IN 持续为 1，计时达到 <strong>PT</strong> 后，Q 才变为 1。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **定时器操作** |
| 指令名称 | **TON** |
| 功能描述 | **接通延时** |
| 版本 | **V1.0** |
| 核心作用 | 延时启动 / 延时接通 |

### ② 参考表

| IN 状态 | TON 输出 |
|---|---|
| IN = 0 | Q = 0，ET 复位 |
| IN = 1，未到 PT | Q = 0，ET 计时 |
| IN = 1，到达 PT | **Q = 1** |

### ③ 计算/选型示例

- 已知：传感器检测到物料 5 秒后启动电机
- 公式：`PT = T#5s`
- 计算：IN 持续为 1 达到 5 秒后，Q 输出
- 结论：选择 **TON 接通延时**

> [!tip] 记忆口诀
> **TON 慢启动，接通以后等一等。**


## 03 `TOF` | 关断延时

> [!info] 核心结论
> **TOF** 用于输入断开后延时关闭输出，常用于延时停止。

<div style="background:#F7F9FC;border-left:6px solid #27AE60;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#27AE60;">重点规则</strong><br>
IN 从 1 变 0 后，Q 不会立刻断开，而是继续保持 <strong>PT</strong> 时间。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **定时器操作** |
| 指令名称 | **TOF** |
| 功能描述 | **关断延时** |
| 版本 | **V1.0** |
| 核心作用 | 延时停止 / 延时断开 |

### ② 参考表

| IN 状态 | TOF 输出 |
|---|---|
| IN = 1 | **Q = 1** |
| IN 由 1 变 0，未到 PT | **Q = 1** |
| 延时到达 PT | Q = 0 |

### ③ 计算/选型示例

- 已知：设备停止后，风扇继续运行 10 秒散热
- 公式：`PT = T#10s`
- 计算：IN 断开后，Q 继续保持 10 秒
- 结论：选择 **TOF 关断延时**

> [!tip] 记忆口诀
> **TOF 慢停止，断开以后再等等。**


## 04 `TONR` | 时间累加器

> [!info] 核心结论
> **TONR** 用于累计输入为 1 的总时间，适合运行时间统计和分段累计计时。

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">核心特点</strong><br>
TONR 会<strong>累加时间</strong>，是否断电保持需按实际 PLC 型号、存储区和项目设置校核。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **定时器操作** |
| 指令名称 | **TONR** |
| 功能描述 | **时间累加器** |
| 版本 | **V1.0** |
| 核心作用 | 累计运行时间 |

### ② 参考表

| 状态 | TONR 表现 |
|---|---|
| IN = 1 | ET 继续累加 |
| IN = 0 | ET 保持当前值 |
| ET ≥ PT | Q = **1** |
| 复位触发 | ET 清零，Q 复位 |

### ③ 计算/选型示例

- 已知：水泵累计运行满 8 小时后提示维护
- 公式：`PT = T#8h`
- 计算：水泵运行时 ET 累加，停止时 ET 保持
- 结论：选择 **TONR 时间累加器**

> [!tip] 记忆口诀
> **TONR 会记账，停了时间也不忘。**

## 11 定时器操作 | 选型速记

> [!info] 核心结论
> 定时器选型先看动作需求：**脉冲用 TP，延时开用 TON，延时关用 TOF，累计时间用 TONR。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
TP 打脉冲，TON 慢启动，TOF 慢停止，TONR 会累计。
</div>

### ① 分类/公式/规则

| 需求 | 推荐指令 |
|---|---|
| 触发后输出固定时间 | **TP / --(TP)--** |
| 输入成立后延时输出 | **TON / --(TON)--** |
| 输入断开后延时关闭 | **TOF / --(TOF)--** |
| 累计运行时间 | **TONR / --(TONR)--** |
| 清除定时器 | **--(RT)--** |
| 设置预设时间 | **--(PT)--** |

### ② 参考表

| 指令 | 关键词 |
|---|---|
| **TP** | 脉冲、单次、固定宽度 |
| **TON** | 接通延时、延时启动 |
| **TOF** | 关断延时、延时停止 |
| **TONR** | 累加、保持、运行时间 |
| **RT** | 复位、清零 |
| **PT** | 预设时间、持续时间 |

### ③ 计算/选型示例

- 已知：按钮按下后，设备延时 5 秒启动
- 公式/规则：接通后延时输出
- 操作：选择 **TON**，设置 `PT = T#5s`
- 结论：该场景属于**接通延时**

> [!tip] 记忆口诀
> **脉冲 TP，启动 TON，停止 TOF，累计 TONR。**
