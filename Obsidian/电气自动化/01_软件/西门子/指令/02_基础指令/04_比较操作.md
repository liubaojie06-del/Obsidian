## 01 `CMP ==` | 等于

> [!info] 核心结论
> **CMP ==** 用于判断两个数据是否相等，相等时逻辑结果为真。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
A 等于 B，则比较结果为 <strong>TRUE</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **比较操作** |
| 指令名称 | **CMP ==** |
| 功能描述 | **等于** |
| 判断公式 | **A == B** |

### ② 参考表

| A 与 B 关系 | 结果 |
|---|---|
| A = B | **TRUE** |
| A ≠ B | FALSE |

### ③ 计算/选型示例

- 已知：当前数量 `CV = 10`，目标数量 `PV = 10`
- 公式：`CV == PV`
- 计算：`10 == 10`
- 结论：比较结果为 **TRUE**

> [!tip] 记忆口诀
> **相等用双等，两个一样才通过。**

## 20 比较操作 | 选型速记

> [!info] 核心结论
> 比较操作的核心是：**数值比较看大小，范围判断看上下限，变量检查看类型和有效性。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
等不等用 CMP，范围内外用 Range，有效无效看 OK，类型变量先校验。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 相等判断 | **CMP ==** |
| 不等判断 | **CMP <>** |
| 上下限判断 | **CMP >= / CMP <=** |
| 范围判断 | **IN_Range / OUT_Range** |
| 有效性判断 | **OK / NOT_OK** |
| 类型判断 | **EQ_Type / NE_Type / IS_ARRAY** |
| NULL 判断 | **IS_NULL / NOT_NULL** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 判断是否等于设定值 | **CMP ==** |
| 判断是否超过上限 | **CMP > / CMP >=** |
| 判断是否低于下限 | **CMP < / CMP <=** |
| 判断是否在正常区间 | **IN_Range** |
| 判断是否超出正常区间 | **OUT_Range** |
| 判断数据是否有效 | **--\|OK\|--** |
| 判断数据是否无效 | **--\|NOT_OK\|--** |
| 判断是否为空 | **IS_NULL** |
| 判断是否为数组 | **IS_ARRAY** |

### ③ 计算/选型示例

- 已知：温度允许范围 `20℃ ~ 80℃`
- 公式/规则：正常运行要求温度在范围内
- 操作：选择 **IN_Range**
- 结论：比单独写两个比较更清晰

> [!tip] 记忆口诀
> **比大小用 CMP，查区间用 Range，防异常先 OK，查类型看 Type。**
