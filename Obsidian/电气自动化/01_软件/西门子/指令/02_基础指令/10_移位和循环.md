## 01 `SHR` | 右移

> [!info] 核心结论
> **SHR** 用于将数据位整体向右移动，常用于除以 2、位状态右移、数据压缩等场景。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
右移 1 位，相当于无符号数除以 <strong>2</strong>；右移 n 位，相当于除以 <strong>2ⁿ</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **移位和循环** |
| 指令名称 | **SHR** |
| 功能描述 | **右移** |
| 核心作用 | 数据位向右移动 |
| 常见用途 | 除以 2、状态位右移、位处理 |

### ② 参考表

| 操作 | 含义 |
|---|---|
| 右移 1 位 | 约等于 ÷2 |
| 右移 2 位 | 约等于 ÷4 |
| 右移 n 位 | 约等于 ÷2ⁿ |
| 空出高位 | 通常补 0，需按数据类型校核 |

### ③ 计算/选型示例

- 已知：二进制数据 `0000_1000`
- 公式：右移 1 位
- 计算：`0000_1000 → 0000_0100`
- 结论：数值从 **8 变为 4**

> [!tip] 记忆口诀
> **SHR 往右挪，数值一半往下落。**

## 03 `ROR` | 循环右移

> [!info] 核心结论
> **ROR** 用于将数据位循环向右移动，移出的最低位会回到最高位。

<div style="background:#F7F9FC;border-left:6px solid #FF6B35;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#FF6B35;">重点规则</strong><br>
普通右移会丢位，循环右移不会丢位；最低位绕回最高位。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **移位和循环** |
| 指令名称 | **ROR** |
| 功能描述 | **循环右移** |
| 核心作用 | 位序列向右循环移动 |
| 常见用途 | 跑马灯、循环状态、环形队列 |

### ② 参考表

| 操作 | 说明 |
|---|---|
| 右移 | 低位移出，高位补位 |
| 循环右移 | **低位移出后回到高位** |
| 是否丢位 | 不丢位 |
| 位宽影响 | BYTE / WORD / DWORD 需校核 |

### ③ 计算/选型示例

- 已知：4 位数据 `1001`
- 公式：循环右移 1 位
- 计算：最低位 `1` 移到最高位，`1001 → 1100`
- 结论：适合循环灯、循环位状态控制

> [!tip] 记忆口诀
> **ROR 向右转，掉下来的回顶端。**

## 05 移位和循环 | 选型速记

> [!info] 核心结论
> 移位和循环的核心是：**普通移位会补位，循环移位会绕回；左移多用于放大，右移多用于缩小。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
左移 SHL 翻倍，右移 SHR 减半；循环用 ROL/ROR，位不丢只绕圈。
</div>

### ① 分类/公式/规则

| 类型 | 常用指令 |
|---|---|
| 普通右移 | **SHR** |
| 普通左移 | **SHL** |
| 循环右移 | **ROR** |
| 循环左移 | **ROL** |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 数值快速除以 2ⁿ | **SHR** |
| 数值快速乘以 2ⁿ | **SHL** |
| 位状态向右循环 | **ROR** |
| 位状态向左循环 | **ROL** |
| 跑马灯效果 | **ROL / ROR** |
| 不希望移出位丢失 | **循环移位** |

### ③ 计算/选型示例

- 已知：需要做 8 个指示灯循环点亮
- 公式/规则：点亮位移动后不能丢失
- 操作：选择 **ROL** 或 **ROR**
- 结论：循环移位适合跑马灯和环形状态控制

> [!tip] 记忆口诀
> **要放大选左移，要缩小选右移；要循环不丢位，就用 ROL/ROR。**
