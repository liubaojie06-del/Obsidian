## 01 `--| |--` | 常开触点 F9

> [!info] 核心结论
> **常开触点**用于判断信号是否为 1，信号为 1 时逻辑导通。

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">重点规则</strong><br>
常开触点看的是操作数是否为 <strong>1</strong>。
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **位逻辑运算** |
| 指令名称 | **常开触点** |
| 符号 | **`--\| \|--`** |
| 快捷键 | **F9** |
| 导通条件 | 操作数 = **1** |

### ② 参考表

| 操作数状态 | 常开触点结果 |
|---|---|
| 0 | 不导通 |
| 1 | **导通** |

### ③ 计算/选型示例

- 已知：启动按钮地址为 `I0.0`
- 公式：`I0.0 = 1` 时触点导通
- 计算：按下按钮后，`I0.0 = 1`
- 结论：常开触点导通，后续逻辑可以执行

> [!tip] 记忆口诀
> **常开看 1，等 1 就通。**

## 03 `--|NOT|--` | 取反 RLO

> [!info] 核心结论
> **取反 RLO**用于把当前逻辑运算结果反转，1 变 0，0 变 1。

<div style="background:#F7F9FC;border-left:6px solid #9B59B6;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#9B59B6;">重点公式</strong><br>
输出结果 = <strong>NOT 当前 RLO</strong>
</div>

### ① 分类/公式/规则

| 项目 | 内容 |
|---|---|
| 所属分类 | **位逻辑运算** |
| 指令名称 | **取反 RLO** |
| 符号 | **`--\|NOT\|--`** |
| 核心作用 | 反转当前逻辑结果 |

### ② 参考表

| 原 RLO | 取反后 RLO |
|---|---|
| 0 | **1** |
| 1 | **0** |

### ③ 计算/选型示例

- 已知：当前 RLO = 1
- 公式：`NOT 1 = 0`
- 计算：经过取反后，RLO 变为 0
- 结论：适合需要反向判断的逻辑

> [!tip] 记忆口诀
> **NOT 一下就反相，1 变 0，0 变 1。**

## 20 位逻辑运算 | 选型速记

> [!info] 核心结论
> 位逻辑运算的核心是：**触点判断条件，线圈写入结果，边沿只打一拍，置复位可保持。**

<div style="background:#F7F9FC;border-left:6px solid #173B63;padding:12px;border-radius:12px;margin:10px 0;">
<strong style="color:#173B63;">总口诀</strong><br>
常开看 1，常闭看 0；S 置位，R 复位；P 上升，N 下降。
</div>

### ① 分类/公式/规则

| 类型 | 关键词 |
|---|---|
| 触点判断 | 常开、常闭、NOT |
| 输出控制 | 赋值、赋值取反 |
| 状态保持 | S、R、SR、RS |
| 批量位操作 | SET_BF、RESET_BF |
| 边沿检测 | P、N、P_TRIG、N_TRIG、R_TRIG、F_TRIG |

### ② 参考表

| 需求 | 推荐指令 |
|---|---|
| 信号为 1 时导通 | **`--\| \|--` 常开触点** |
| 信号为 0 时导通 | **`--\|/\|--` 常闭触点** |
| 输出跟随逻辑 | **`--( )--` 赋值** |
| 输出保持为 1 | **S / SR / RS** |
| 输出清零 | **R / RESET_BF** |
| 按下瞬间触发 | **P / R_TRIG** |
| 松开瞬间触发 | **N / F_TRIG** |

### ③ 计算/选型示例

- 已知：按钮按下时启动一次计数
- 公式/规则：按下瞬间属于 **上升沿**
- 操作：选择 **R_TRIG** 或 **P 触点**
- 结论：按钮长按不会重复计数

> [!tip] 记忆口诀
> **判断用触点，输出用线圈，保持用置复位，瞬间用边沿。**
