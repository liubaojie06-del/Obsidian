# 01 异步电机总览

> [!info] 核心结论
> 异步电机是工业现场最常见的交流电机，特点是 **结构简单、可靠耐用、成本低、维护方便**。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> “异步”指的是：  
> **转子实际转速永远略低于旋转磁场的同步转速**。  
> 正是因为有这个速度差，电机才能产生转矩。

> [!warning] 🟧 通用选型公式
> **异步电机选型 = 电压等级 + 功率 + 极数/转速 + 负载类型 + 启动方式 + 安装方式 + 防护等级**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **常见类型** | 三相异步电机、单相异步电机 |
| **工业常用** | **三相鼠笼式异步电机** |
| **典型负载** | 风机、水泵、输送机、搅拌机、压缩机 |
| **控制方式** | 工频、软启动、变频器 |
| **主要优点** | 可靠、便宜、维护少 |
| **重点参数** | 功率、电流、转速、极数、接法、防护等级 |

## ③ 计算/选型示例

- 已知：  
  一台输送设备需要连续运行，现场电源为三相AC380V。
- 公式：  
  **电机需求 = 三相380V + 连续运行 + 转矩满足**
- 计算：  
  输送设备常用三相异步电机，功率和转速按负载计算。
- 结论：  
  可优先选择 **三相鼠笼式异步电机**，功率、电流和安装尺寸需按厂家手册校核。

> [!tip] 记忆口诀
> 工业动力看异步，便宜耐用最常用。

# 02 底层原理

> [!info] 核心结论
> 异步电机靠定子旋转磁场感应转子电流，再由转子电流产生电磁转矩。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 三相交流电进入定子绕组后，产生旋转磁场。  
> 旋转磁场切割转子导体，转子感应出电流。  
> 转子电流与磁场作用，产生转矩并带动负载旋转。

> [!warning] 🟧 能量转换逻辑
> **三相交流电 → 定子旋转磁场 → 转子感应电流 → 电磁转矩 → 机械转动**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子** | 产生旋转磁场 |
| **转子** | 感应电流并输出转矩 |
| **气隙** | 磁场传递通道 |
| **轴承** | 支撑转轴 |
| **风扇** | 散热 |
| **接线盒** | 电源接入和Y/Δ接线 |

## ③ 计算/选型示例

- 已知：  
  电机上电后有嗡声但不转。
- 公式：  
  **电磁转矩 > 负载阻力 → 电机转动**
- 计算：  
  如果缺相、负载卡死或启动转矩不足，电机可能只响不转。
- 结论：  
  应检查三相电源、接线、负载是否卡滞和保护装置状态。

> [!tip] 记忆口诀
> 定子造磁场，转子被感应；有转差，才有力。

# 03 为什么叫异步

> [!info] 核心结论
> 异步电机的转子转速低于同步转速，这个差值叫转差。

## ① 分类/公式/规则

> [!note] 🟦 关键概念
> 如果转子和旋转磁场完全同步，就没有相对运动，也就不能感应电流。  
> 没有感应电流，就没有电磁转矩。  
> 所以异步电机必须存在转差。

> [!warning] 🟧 转差率公式
> **s =（nₛ - n）÷ nₛ**

## ② 参考表

| 符号 | 含义 |
|---|---|
| **nₛ** | 同步转速 |
| **n** | 转子实际转速 |
| **s** | 转差率 |
| **转差小** | 轻载运行 |
| **转差大** | 负载加重 |
| **转差过大** | 可能过载或堵转 |

## ③ 计算/选型示例

- 已知：  
  4极电机同步转速为1500rpm，实际转速为1450rpm。
- 公式：  
  **s =（1500 - 1450）÷ 1500**
- 计算：  
  s = 50 ÷ 1500 ≈ **3.33%**
- 结论：  
  该异步电机转差率约为 **3.33%**。

> [!tip] 记忆口诀
> 异步差一点，电机才有劲。

# 04 同步转速与极数

> [!info] 核心结论
> 异步电机的理论同步转速由 **电源频率和电机极数** 决定。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **nₛ = 120 × f ÷ P**

> [!note] 参数说明
> **nₛ**：同步转速，rpm  
> **f**：电源频率，Hz  
> **P**：电机极数

## ② 参考表

| 极数 | 50Hz同步转速 | 常见实际转速 |
|---|---:|---:|
| **2极** | 3000rpm | 约2800~2950rpm |
| **4极** | 1500rpm | 约1400~1480rpm |
| **6极** | 1000rpm | 约900~980rpm |
| **8极** | 750rpm | 约700~740rpm |

## ③ 计算/选型示例

- 已知：  
  一台6极异步电机，频率为50Hz。
- 公式：  
  **nₛ = 120 × 50 ÷ 6**
- 计算：  
  nₛ = **1000rpm**
- 结论：  
  6极电机同步转速为 **1000rpm**，实际转速会略低。

> [!tip] 记忆口诀
> 极数越多，转速越低；4极50Hz，大约1500转。

# 05 功率、转速与转矩

> [!info] 核心结论
> 异步电机选型不能只看功率，还要看转矩是否能带动负载。

## ① 分类/公式/规则

> [!warning] 🟧 功率转矩公式
> **P = T × n ÷ 9550**

> [!note] 参数说明
> **P**：功率，kW  
> **T**：转矩，N·m  
> **n**：转速，rpm

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定功率** | 长期输出能力 |
| **额定转速** | 正常运行速度 |
| **额定转矩** | 长期带载能力 |
| **启动转矩** | 启动时能否带动负载 |
| **最大转矩** | 抗冲击能力 |
| **额定电流** | 保护、线缆、变频器选型依据 |

## ③ 计算/选型示例

- 已知：  
  电机功率 **4kW**，额定转速 **1450rpm**。
- 公式：  
  **T = 9550 × P ÷ n**
- 计算：  
  T = 9550 × 4 ÷ 1450 ≈ **26.34N·m**
- 结论：  
  该电机额定转矩约为 **26.3N·m**。

> [!tip] 记忆口诀
> 功率看等级，转矩看力气，转速看快慢。

# 06 鼠笼式异步电机

> [!info] 核心结论
> 鼠笼式异步电机是最常见的工业异步电机，结构简单、可靠性高、维护量小。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 鼠笼式转子像一个“金属笼子”，由导条和端环组成。  
> 它没有电刷和滑环，所以维护简单。

> [!warning] 🟧 适用规则
> **普通工业负载 → 优先考虑鼠笼式异步电机**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **结构** | 转子导条 + 端环 |
| **优点** | 可靠、便宜、维护少 |
| **缺点** | 直接启动电流大 |
| **调速方式** | 变频器调速 |
| **典型应用** | 风机、水泵、输送线、机床 |
| **维护重点** | 轴承、散热、绝缘 |

## ③ 计算/选型示例

- 已知：  
  一台风机需要长期连续运行。
- 公式：  
  **长期连续 + 普通负载 = 鼠笼式异步电机优先**
- 计算：  
  风机负载稳定，鼠笼电机可靠性高，维护量低。
- 结论：  
  可选择 **三相鼠笼式异步电机**，并按风机功率和转速校核。

> [!tip] 记忆口诀
> 鼠笼电机最常见，便宜耐用少维护。

# 07 绕线式异步电机

> [!info] 核心结论
> 绕线式异步电机可通过转子回路外接电阻改善启动性能，适合重载启动场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 绕线式电机转子有绕组，并通过滑环引出。  
> 启动时串入电阻，可提高启动转矩、降低启动电流。

> [!warning] 🟧 应用规则
> **重载启动 / 大惯量启动 → 可评估绕线式异步电机**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **结构** | 转子绕组 + 滑环 |
| **优点** | 启动转矩大，启动电流可控 |
| **缺点** | 结构复杂，维护量大 |
| **维护点** | 电刷、滑环、转子电阻 |
| **应用场景** | 起重、破碎、卷扬、大惯量负载 |
| **替代方案** | 变频器重载启动方案 |

## ③ 计算/选型示例

- 已知：  
  破碎机启动时负载重，直接启动困难。
- 公式：  
  **重载启动 = 需要较大启动转矩**
- 计算：  
  绕线式电机可通过转子串电阻改善启动能力。
- 结论：  
  可评估绕线式异步电机或重载型变频启动方案，需按厂家手册校核。

> [!tip] 记忆口诀
> 鼠笼省心，绕线有劲；重载启动看绕线。

# 08 单相异步电机

> [!info] 核心结论
> 单相异步电机常用于小功率设备，通常需要电容或辅助绕组帮助启动。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 单相电源不能像三相电源那样天然产生旋转磁场。  
> 因此单相异步电机常用 **启动绕组 + 电容** 产生相位差，形成启动转矩。

> [!warning] 🟧 启动逻辑
> **主绕组 + 辅助绕组/电容 → 相位差 → 启动转矩**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **电容启动电机** | 启动转矩较大 |
| **电容运行电机** | 运行较平稳 |
| **电容启动运行电机** | 启动和运行性能较好 |
| **罩极电机** | 小功率、结构简单 |
| **常见应用** | 小风机、小水泵、家电、小设备 |
| **常见故障** | 电容损坏导致只响不转 |

## ③ 计算/选型示例

- 已知：  
  单相电机通电后嗡嗡响，但启动不了。
- 公式：  
  **启动失败 = 启动转矩不足**
- 计算：  
  常见原因是启动电容损坏、启动绕组异常或负载过重。
- 结论：  
  应检查启动电容、辅助绕组和机械负载。

> [!tip] 记忆口诀
> 单相启动靠电容，电容坏了只嗡不转。

# 09 铭牌参数

> [!info] 核心结论
> 异步电机接线和选型前必须看铭牌，重点确认电压、频率、功率、电流、转速和接法。

## ① 分类/公式/规则

> [!note] 🟦 铭牌重点
> 电机铭牌是接线、保护整定、变频器选型和故障排查的依据。  
> 不看铭牌接线，容易接错电压或接法。

> [!warning] 🟧 铭牌校核公式
> **电机可用 = 电源电压匹配 + 接法正确 + 电流保护正确 + 负载匹配**

## ② 参考表

| 铭牌项 | 作用 |
|---|---|
| **额定电压** | 判断电源是否匹配 |
| **额定电流** | 保护器/线缆/变频器选型 |
| **额定功率** | 负载能力 |
| **额定转速** | 匹配工艺速度 |
| **频率** | 50Hz/60Hz确认 |
| **接法** | Y接或Δ接 |
| **绝缘等级** | 温升能力 |
| **防护等级** | 防尘防水能力 |

## ③ 计算/选型示例

- 已知：  
  电机铭牌为 **380/660V Δ/Y**，现场电源为三相380V。
- 公式：  
  **低电压对应Δ接**
- 计算：  
  380V是铭牌低电压侧，应采用 **Δ接法**。
- 结论：  
  该电机在380V电源下通常接三角形，最终以接线盒说明和厂家手册为准。

> [!tip] 记忆口诀
> 铭牌先看清，接线才放心。

# 10 启动方式

> [!info] 核心结论
> 异步电机启动电流较大，启动方式要根据功率、负载、电网容量和工艺要求选择。

## ① 分类/公式/规则

> [!note] 🟦 常见启动方式
> 小功率可直接启动；  
> 中大功率可用星三角、自耦降压、软启动；  
> 需要调速或重载平稳启动时，优先考虑变频器。

> [!warning] 🟧 启动方案公式
> **启动方式 = 电机功率 + 启动电流 + 负载转矩 + 电网容量 + 是否调速**

## ② 参考表

| 启动方式 | 特点 |
|---|---|
| **直接启动** | 简单，启动电流大 |
| **星三角启动** | 降低电流，也降低转矩 |
| **自耦降压启动** | 适合较大功率 |
| **软启动器** | 降低冲击，不长期调速 |
| **变频启动** | 平滑启动，可调速 |
| **绕线转子启动** | 重载启动能力较好 |

## ③ 计算/选型示例

- 已知：  
  一台搅拌机需要带料启动，启动转矩要求高。
- 公式：  
  **重载启动 → 需要足够启动转矩**
- 计算：  
  星三角启动虽然能降电流，但启动转矩也下降，可能带不动。
- 结论：  
  应优先评估 **变频器启动或重载启动方案**。

> [!tip] 记忆口诀
> 轻载可降压，重载要转矩；要调速，用变频。

# 11 星三角启动

> [!info] 核心结论
> 星三角启动能降低启动电流，但启动转矩也会降低，适合轻载或空载启动。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 启动时先接成星形Y，让绕组电压降低；  
> 电机转速起来后，再切换成三角形Δ正常运行。

> [!warning] 🟧 星三角规则
> **星形启动电流降低，启动转矩也降低**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **适合** | 风机、水泵轻载启动 |
| **不适合** | 重载、频繁启动、大惯量 |
| **条件** | 电机必须支持Δ运行 |
| **接触器** | 主接触器、星接触器、角接触器 |
| **控制元件** | 时间继电器或PLC |
| **风险** | 切换冲击、接线错误、互锁失败 |

## ③ 计算/选型示例

- 已知：  
  一台设备空载启动，运行后才加载。
- 公式：  
  **空载启动 → 星三角可评估**
- 计算：  
  空载启动对启动转矩要求不高，星三角可降低电网冲击。
- 结论：  
  可评估星三角启动，但必须校核电机接法和电气互锁。

> [!tip] 记忆口诀
> 星三角省电流，也省启动力。

# 12 变频调速

> [!info] 核心结论
> 异步电机调速最常见方式是使用变频器，通过改变频率来改变电机转速。

## ① 分类/公式/规则

> [!note] 🟦 调速逻辑
> 频率越高，电机同步转速越高；  
> 频率越低，电机同步转速越低。  
> 变频器通常同时调节电压和频率，保持磁通合理。

> [!warning] 🟧 变频调速公式
> **nₛ = 120 × f ÷ P**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **V/F控制** | 风机、水泵、普通调速 |
| **无PG矢量** | 输送、搅拌、一般重载 |
| **有PG矢量** | 低速大转矩、高精度速度 |
| **PID控制** | 恒压、恒流、恒温 |
| **多段速** | 多工艺速度 |
| **通讯控制** | PLC集中控制 |

## ③ 计算/选型示例

- 已知：  
  4极异步电机，变频器输出频率为30Hz。
- 公式：  
  **nₛ = 120 × 30 ÷ 4**
- 计算：  
  nₛ = **900rpm**
- 结论：  
  4极电机在30Hz时同步转速约为 **900rpm**，实际转速略低。

> [!tip] 记忆口诀
> 异步调速靠变频，频率一变速度变。

# 13 风机水泵节能

> [!info] 核心结论
> 异步电机带风机、水泵时，变频调速常能明显节能，因为功率近似随转速三次方变化。

## ① 分类/公式/规则

> [!note] 🟦 节能逻辑
> 风机水泵属于平方转矩负载。  
> 降低转速后，流量、压力和功率都会下降，其中功率下降最明显。

> [!warning] 🟧 相似定律
> **Q ∝ n**  
> **H ∝ n²**  
> **P ∝ n³**

## ② 参考表

| 参数 | 与转速关系 |
|---|---|
| **流量Q** | 约与转速成正比 |
| **压力/扬程H** | 约与转速平方成正比 |
| **轴功率P** | 约与转速三次方成正比 |
| **挡板/阀门调节** | 能耗较高 |
| **变频调节** | 节能明显 |
| **工艺下限** | 需按现场校核 |

## ③ 计算/选型示例

- 已知：  
  风机转速降到额定的 **80%**。
- 公式：  
  **功率比例 ≈ 0.8³**
- 计算：  
  功率比例 ≈ **51.2%**
- 结论：  
  理论轴功率约降至原来的 **51.2%**，实际节能需结合系统效率和工况校核。

> [!tip] 记忆口诀
> 风机水泵降一点速，功率按三次方往下降。

# 14 电机保护

> [!info] 核心结论
> 异步电机保护重点是短路、过载、缺相、过热、堵转、接地和绝缘异常。

## ① 分类/公式/规则

> [!note] 🟦 保护思路
> 短路保护通常用断路器或熔断器；  
> 过载保护常用热继电器、电机保护器或变频器内部保护；  
> 高要求场景可加PTC/PT100温度保护。

> [!warning] 🟧 保护配置公式
> **电机保护 = 短路保护 + 过载保护 + 缺相保护 + 温度保护 + 接地保护**

## ② 参考表

| 保护类型 | 作用 |
|---|---|
| **断路器** | 短路、过流保护 |
| **热继电器** | 过载保护 |
| **电机保护器** | 缺相、堵转、过载 |
| **PTC/PT100** | 温度保护 |
| **接地保护** | 绝缘故障保护 |
| **变频器保护** | 过流、过压、过载、过热 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流为 **12A**。
- 公式：  
  **热继电器整定值 ≈ 电机额定电流**
- 计算：  
  热继电器可初步整定在 **12A附近**。
- 结论：  
  最终整定需按启动方式、工作制、环境温度和厂家说明校核。

> [!tip] 记忆口诀
> 短路靠断路器，过载靠热保护，缺相堵转不能漏。

# 15 安装与防护等级

> [!info] 核心结论
> 异步电机选型不仅看电气参数，还要确认安装方式、防护等级、绝缘等级和散热条件。

## ① 分类/公式/规则

> [!note] 🟦 机械匹配
> 电机必须与设备结构匹配，包括底脚、法兰、轴径、轴伸、键槽、出线方向和接线盒位置。

> [!warning] 🟧 安装匹配公式
> **机械可用 = 安装方式 + 法兰尺寸 + 轴径轴长 + 防护等级 + 冷却方式**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **B3** | 卧式底脚安装 |
| **B5** | 法兰安装 |
| **B35** | 底脚 + 法兰安装 |
| **IP等级** | 防尘防水能力 |
| **绝缘等级** | 影响允许温升 |
| **冷却方式** | 自扇冷、强冷、水冷等 |
| **环境条件** | 高温、粉尘、潮湿、腐蚀需校核 |

## ③ 计算/选型示例

- 已知：  
  现场粉尘较多，电机长期运行。
- 公式：  
  **环境适配 = 防护等级 + 散热方式 + 密封结构**
- 计算：  
  防护等级过低会导致粉尘进入，影响散热和绝缘。
- 结论：  
  应选择合适IP等级的异步电机，并制定清灰维护计划。

> [!tip] 记忆口诀
> 电气能用还不够，机械装得上，环境扛得住。

# 16 常见故障排查

> [!info] 核心结论
> 异步电机故障排查按“电源 → 接线 → 保护 → 电机 → 负载 → 控制设备”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查三相电压，再查接触器、热继电器、变频器或软启动器。  
> 先空载试运行，再带载测试。

> [!warning] 🟧 排错公式
> **异步电机异常 = 电源异常 + 缺相 + 接线错误 + 保护动作 + 负载卡滞 + 电机损坏**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **不转** | 无电源、缺相、保护未复位 |
| **嗡嗡响不转** | 缺相、负载卡死、启动转矩不足 |
| **发热大** | 过载、散热差、电压异常 |
| **电流大** | 负载重、轴承坏、绕组问题 |
| **振动大** | 轴承、联轴器、转子不平衡 |
| **频繁跳闸** | 短路、接地、过载、保护整定不当 |
| **方向反** | 任意两相对调即可反向，需停电操作 |

## ③ 计算/选型示例

- 已知：  
  电机空载正常，带载后电流大、转速上不来。
- 公式：  
  **带载异常 = 负载过重 + 机械卡滞 + 电机容量不足**
- 计算：  
  空载正常说明电机和电源基本可用，带载异常更可能是负载或选型问题。
- 结论：  
  应检查机械负载、轴承、联轴器、电机功率和保护整定。

> [!tip] 记忆口诀
> 不转先查电，发热先查流，振动先查轴，带载不行查负载。

# 17 异步电机快速选型表

> [!info] 核心结论
> 异步电机选型先确认电源和负载，再确定功率、转速、极数、启动方式和安装环境。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看单相还是三相  
> ② 看电压和频率  
> ③ 看负载类型  
> ④ 算功率和转矩  
> ⑤ 选极数和转速  
> ⑥ 看启动方式  
> ⑦ 判断是否需要变频调速  
> ⑧ 看安装方式  
> ⑨ 看防护等级和绝缘等级  
> ⑩ 校核保护、线缆和厂家手册

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电压等级 + 功率 + 极数/转速 + 负载类型 + 启动方式 + 安装方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 普通工业动力 | 三相鼠笼异步电机 |
| 小功率单相设备 | 单相异步电机 |
| 需要调速 | 异步电机 + 变频器 |
| 重载启动 | 启动转矩、变频/绕线方案 |
| 风机水泵 | 变频节能、V/F控制 |
| 频繁启停 | 热容量、启动方式 |
| 粉尘潮湿环境 | IP等级、散热、绝缘 |

## ③ 计算/选型示例

- 已知：  
  设备需要转矩约 **30N·m**，转速约 **1450rpm**。
- 公式：  
  **P = T × n ÷ 9550**
- 计算：  
  P = 30 × 1450 ÷ 9550 ≈ **4.55kW**
- 结论：  
  可初步选择 **5.5kW等级** 异步电机，并按启动转矩、负载冲击和厂家规格留余量。

> [!tip] 选型口诀
> 电源先确认，负载再计算；速度看极数，力气看转矩；要调速配变频，环境差看防护。
