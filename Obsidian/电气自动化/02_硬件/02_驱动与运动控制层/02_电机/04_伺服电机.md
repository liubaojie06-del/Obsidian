# 01 伺服电机总览

> [!info] 核心结论
> 伺服电机用于实现 **高精度位置、速度、转矩控制**，必须配合伺服驱动器和反馈元件使用。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 伺服电机不是普通电机。  
> 普通电机重点是“转起来”；伺服电机重点是 **转到哪、转多快、出多大力，并且能反馈修正**。

> [!warning] 🟧 通用选型公式
> **伺服电机选型 = 负载转矩 + 最高转速 + 加减速时间 + 惯量匹配 + 编码器类型 + 抱闸需求 + 安装尺寸**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **核心功能** | 位置、速度、转矩精准控制 |
| **配套设备** | 伺服驱动器、PLC/运动控制器 |
| **反馈元件** | 编码器、旋变、霍尔等 |
| **常见类型** | 交流伺服、直流伺服、低压伺服 |
| **典型应用** | 机械手、滑台、转盘、包装机、数控设备 |
| **重点参数** | 转矩、转速、惯量、编码器、抱闸、法兰 |

## ③ 计算/选型示例

- 已知：  
  设备需要滑台精准移动到指定位置。
- 公式：  
  **精准定位 = 伺服电机 + 驱动器 + 编码器反馈 + 控制器指令**
- 计算：  
  控制器发目标位置，驱动器控制电机运动，编码器反馈实际位置。
- 结论：  
  该场景应选择 **伺服电机系统**，具体功率和转矩需按负载校核。

> [!tip] 记忆口诀
> 普通电机管转动，伺服电机管精准。

# 02 底层原理

> [!info] 核心结论
> 伺服电机的本质，是通过反馈闭环不断修正误差，让实际运动跟随目标指令。

## ① 分类/公式/规则

> [!note] 🟦 闭环流程
> 控制器给目标，驱动器算偏差，电机执行动作，编码器反馈实际值，驱动器继续修正输出。

> [!warning] 🟧 闭环控制公式
> **控制偏差 = 目标值 - 反馈值**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **控制器** | 发送位置、速度、转矩指令 |
| **伺服驱动器** | 控制电机电流和运动 |
| **伺服电机** | 输出转矩和转速 |
| **编码器** | 反馈位置和速度 |
| **负载机构** | 丝杆、同步带、转盘等 |
| **闭环修正** | 减小误差，提高精度 |

## ③ 计算/选型示例

- 已知：  
  目标位置为100.00mm，实际位置为99.95mm。
- 公式：  
  **位置偏差 = 目标位置 - 实际位置**
- 计算：  
  位置偏差 = 100.00 - 99.95 = **0.05mm**
- 结论：  
  伺服系统会继续修正，使实际位置接近目标位置。

> [!tip] 记忆口诀
> 有目标、有反馈、有修正，才叫伺服。

# 03 伺服电机组成

> [!info] 核心结论
> 伺服电机通常由定子、转子、编码器、轴承、外壳和可选抱闸组成。

## ① 分类/公式/规则

> [!note] 🟦 组成理解
> 电机本体负责输出机械运动；  
> 编码器负责反馈位置；  
> 抱闸负责断电保持；  
> 驱动器负责控制电流和运动状态。

> [!warning] 🟧 系统组成公式
> **伺服系统 = 伺服驱动器 + 伺服电机 + 编码器反馈 + 控制器**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子绕组** | 产生磁场 |
| **转子磁钢** | 形成转子磁场 |
| **编码器** | 反馈位置/速度 |
| **轴承** | 支撑电机旋转 |
| **抱闸** | 断电保持位置 |
| **接插件** | 动力线和编码器线连接 |

## ③ 计算/选型示例

- 已知：  
  一台垂直升降轴断电后会下滑。
- 公式：  
  **防下滑 = 伺服控制 + 抱闸保持 + 机械安全**
- 计算：  
  断电后伺服无主动转矩，必须靠抱闸或机械结构保持。
- 结论：  
  垂直轴应优先选择 **带抱闸伺服电机**。

> [!tip] 记忆口诀
> 电机负责动，编码器负责看，抱闸负责停。

# 04 交流伺服与直流伺服

> [!info] 核心结论
> 交流伺服适合工业设备和高性能控制，直流伺服适合低压、电池供电和移动设备。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> **交流伺服**：常见于AC220V、AC380V系统，工业自动化应用广。  
> **低压直流伺服**：常见于DC24V、DC48V、DC72V系统，适合AGV、机器人、小型设备。

> [!warning] 🟧 类型选择公式
> **固定工业设备 → 交流伺服**  
> **电池供电 / 移动平台 → 低压直流伺服**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **交流伺服** | 滑台、机械手、包装机、数控设备 |
| **低压直流伺服** | AGV、AMR、移动机器人 |
| **小功率伺服** | 小型定位、送料、转盘 |
| **大功率伺服** | 主轴、重载传动 |
| **直驱伺服** | 高精度旋转平台 |
| **防爆伺服** | 危险区域，需防爆认证 |

## ③ 计算/选型示例

- 已知：  
  AGV使用DC48V电池供电，需要车轮速度闭环。
- 公式：  
  **电池供电 + 速度闭环 = 低压直流伺服优先**
- 计算：  
  低压伺服可直接接入电池系统，减少电源转换复杂度。
- 结论：  
  该场景优先评估 **DC48V低压伺服电机**。

> [!tip] 记忆口诀
> 固定产线多交流，移动平台看直流。

# 05 额定转矩与峰值转矩

> [!info] 核心结论
> 伺服电机选型必须区分额定转矩和峰值转矩，长期运行看额定，加速冲击看峰值。

## ① 分类/公式/规则

> [!note] 🟦 参数区别
> **额定转矩**：电机可长期连续输出的转矩。  
> **峰值转矩**：短时间加速、启动、冲击时可输出的最大转矩。

> [!warning] 🟧 选型规则
> **电机额定转矩 ≥ 连续负载转矩**  
> **电机峰值转矩 ≥ 加速峰值转矩**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定转矩** | 长时间带载能力 |
| **峰值转矩** | 短时间加速能力 |
| **额定电流** | 长期发热相关 |
| **峰值电流** | 瞬间输出能力 |
| **过载倍数** | 判断短时能力 |
| **热容量** | 影响频繁启停 |

## ③ 计算/选型示例

- 已知：  
  负载连续需要1.2N·m，加速时需要3.5N·m。
- 公式：  
  **额定转矩 ≥ 1.2N·m**  
  **峰值转矩 ≥ 3.5N·m**
- 计算：  
  电机长期转矩必须覆盖1.2N·m，短时峰值必须覆盖3.5N·m。
- 结论：  
  若峰值转矩不足，加速时容易过载或跟随误差过大。

> [!tip] 记忆口诀
> 长期看额定，瞬间看峰值。

# 06 功率、转速与转矩

> [!info] 核心结论
> 伺服电机不能只看功率，必须同时校核转速和转矩。

## ① 分类/公式/规则

> [!warning] 🟧 功率转矩公式
> **P = T × n ÷ 9550**

> [!note] 参数说明
> **P**：功率，kW  
> **T**：转矩，N·m  
> **n**：转速，rpm

## ② 参考表

| 参数 | 选型意义 |
|---|---|
| **功率** | 初步判断电机等级 |
| **转矩** | 判断能否带动负载 |
| **转速** | 判断能否满足节拍 |
| **连续区** | 长时间运行范围 |
| **峰值区** | 短时加减速范围 |
| **转矩曲线** | 最终选型需查看厂家曲线 |

## ③ 计算/选型示例

- 已知：  
  伺服电机功率为750W，额定转速3000rpm。
- 公式：  
  **T = 9550 × P ÷ n**
- 计算：  
  P = 0.75kW  
  T = 9550 × 0.75 ÷ 3000 ≈ **2.39N·m**
- 结论：  
  该电机额定转矩约为 **2.39N·m**，峰值转矩需按厂家手册校核。

> [!tip] 记忆口诀
> 功率是结果，转矩才是力气。

# 07 编码器类型

> [!info] 核心结论
> 编码器是伺服电机实现闭环控制的关键，决定位置反馈方式和上电是否需要回零。

## ① 分类/公式/规则

> [!note] 🟦 编码器区别
> **增量编码器**：断电后通常需要回零。  
> **绝对值编码器**：上电可读取当前位置。  
> **多圈绝对值编码器**：可记录多圈位置，常用于不允许回零的设备。

> [!warning] 🟧 选择公式
> **允许回零 → 增量编码器**  
> **不允许回零 → 绝对值编码器**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **增量编码器** | 成本较低，通常需回零 |
| **单圈绝对值** | 一圈内位置可知 |
| **多圈绝对值** | 多圈位置可记忆 |
| **电池多圈** | 需维护电池 |
| **无电池绝对值** | 需按型号校核 |
| **旋变反馈** | 抗干扰强，适合恶劣环境 |

## ③ 计算/选型示例

- 已知：  
  设备断电后不能重新回原点。
- 公式：  
  **上电直接知道位置 = 绝对值反馈**
- 计算：  
  增量编码器断电后位置基准可能丢失。
- 结论：  
  应选择 **绝对值编码器伺服电机**。

> [!tip] 记忆口诀
> 增量要回零，绝对记位置。

# 08 位置、速度、转矩模式

> [!info] 核心结论
> 伺服电机常用三种控制模式：位置模式管到点，速度模式管快慢，转矩模式管出力。

## ① 分类/公式/规则

> [!note] 🟦 模式区别
> **位置模式**：控制电机到指定位置。  
> **速度模式**：控制电机按目标速度运行。  
> **转矩模式**：控制电机输出指定力矩。

> [!warning] 🟧 模式选择公式
> **定位 → 位置模式**  
> **连续旋转 → 速度模式**  
> **力控/张力 → 转矩模式**

## ② 参考表

| 控制模式 | 典型应用 |
|---|---|
| **位置模式** | 丝杆滑台、机械手、分度盘 |
| **速度模式** | 输送带、主轴、滚轮 |
| **转矩模式** | 压装、张力、夹持 |
| **脉冲控制** | 简单定位 |
| **模拟量控制** | 速度/转矩给定 |
| **总线控制** | 多轴同步、复杂运动 |

## ③ 计算/选型示例

- 已知：  
  转盘每次需要旋转90°并停止。
- 公式：  
  **固定角度定位 → 位置模式**
- 计算：  
  控制器发送目标位置，驱动器控制伺服电机转到指定角度。
- 结论：  
  应使用 **位置控制模式**。

> [!tip] 记忆口诀
> 到哪里用位置，跑多快用速度，出多大力用转矩。

# 09 惯量匹配

> [!info] 核心结论
> 惯量匹配影响伺服响应和稳定性，负载惯量过大容易振动、报警或跟随误差大。

## ① 分类/公式/规则

> [!note] 🟦 惯量理解
> 惯量可以理解为“负载不愿意改变运动状态的程度”。  
> 负载越重、半径越大、加速度越高，对伺服要求越高。

> [!warning] 🟧 惯量比公式
> **惯量比 = 负载折算到电机轴的惯量 ÷ 电机转子惯量**

## ② 参考表

| 惯量比状态 | 影响 |
|---|---|
| **较小** | 响应快，易调试 |
| **适中** | 常见合理匹配 |
| **过大** | 易振动、响应慢 |
| **加速度高** | 对惯量更敏感 |
| **减速机** | 可降低折算惯量 |
| **最终范围** | 需按厂家推荐校核 |

## ③ 计算/选型示例

- 已知：  
  折算到电机轴的负载惯量为0.001kg·m²，电机转子惯量为0.0002kg·m²。
- 公式：  
  **惯量比 = 0.001 ÷ 0.0002**
- 计算：  
  惯量比 = **5**
- 结论：  
  惯量比为5，是否合适需按厂家推荐范围和实际响应要求校核。

> [!tip] 记忆口诀
> 惯量比太大，伺服不好拉。

# 10 丝杆应用选型

> [!info] 核心结论
> 丝杆伺服选型要把直线速度、推力、导程换算成电机转速和转矩。

## ① 分类/公式/规则

> [!warning] 🟧 丝杆转速公式
> **电机转速 rpm = 直线速度 ÷ 丝杆导程 × 60**

> [!warning] 🟧 丝杆转矩近似公式
> **T = F × L ÷（2π × η）**

> [!note] 参数说明
> **F**：轴向推力  
> **L**：丝杆导程，单位m/rev  
> **η**：丝杆效率，需按实际结构校核

## ② 参考表

| 参数 | 作用 |
|---|---|
| **丝杆导程** | 一圈走多远 |
| **直线速度** | 决定电机转速 |
| **推力F** | 决定电机转矩 |
| **效率η** | 影响实际转矩 |
| **加速度** | 影响峰值转矩 |
| **行程** | 决定限位和模组长度 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，直线速度500mm/s。
- 公式：  
  **rpm = 500 ÷ 10 × 60**
- 计算：  
  rpm = **3000rpm**
- 结论：  
  电机需要满足约 **3000rpm** 的运行速度，并继续校核转矩和惯量。

> [!tip] 记忆口诀
> 丝杆导程定一圈距离，速度除导程就是转速。

# 11 抱闸与垂直轴

> [!info] 核心结论
> 垂直轴、升降轴、断电后会下滑的机构，通常需要带抱闸伺服电机。

## ① 分类/公式/规则

> [!note] 🟦 抱闸理解
> 伺服抱闸主要用于 **断电保持位置**，不是用来频繁动态刹车。  
> 正常动作时，应先使能建立转矩，再释放抱闸。

> [!warning] 🟧 抱闸动作逻辑
> **使能建立转矩 → 松开抱闸 → 电机运动**  
> **停止保持 → 抱闸闭合 → 取消使能**

## ② 参考表

| 场景 | 抱闸需求 |
|---|---|
| **水平滑台** | 按需求选择 |
| **垂直升降轴** | 重点关注 |
| **重物提升** | 必须校核 |
| **断电防下滑** | 需要抱闸 |
| **安全保持** | 需安全评估 |
| **频繁急停** | 不能只靠抱闸 |

## ③ 计算/选型示例

- 已知：  
  Z轴负载断电后会下落。
- 公式：  
  **防下落 = 抱闸保持 + 机械结构 + 安全联锁**
- 计算：  
  断电后伺服电机无主动输出，必须依靠抱闸或机械机构保持。
- 结论：  
  应选择 **带抱闸伺服电机**，抱闸保持力矩需按负载校核。

> [!tip] 记忆口诀
> 水平轴看需求，垂直轴先看抱闸。

# 12 驱动器匹配

> [!info] 核心结论
> 伺服电机必须与对应伺服驱动器匹配，不能只按功率随意混用。

## ① 分类/公式/规则

> [!note] 🟦 匹配重点
> 驱动器要匹配电机功率、电压、编码器协议、电机参数和控制接口。  
> 不同品牌或不同系列的伺服电机，编码器和参数可能不兼容。

> [!warning] 🟧 匹配公式
> **可用组合 = 电机型号 + 驱动器型号 + 编码器协议 + 电压等级一致**

## ② 参考表

| 匹配项 | 说明 |
|---|---|
| **功率等级** | 驱动器容量与电机匹配 |
| **电压等级** | AC220V、AC380V、DC48V等 |
| **编码器协议** | 必须与驱动器接口一致 |
| **电机参数** | 驱动器需支持该电机 |
| **控制方式** | 脉冲、模拟量、总线 |
| **安全功能** | STO等按需求校核 |

## ③ 计算/选型示例

- 已知：  
  伺服电机为绝对值编码器，但驱动器只支持增量编码器。
- 公式：  
  **反馈接口不匹配 → 无法正常闭环**
- 计算：  
  驱动器无法正确读取电机反馈，可能报警或不能运行。
- 结论：  
  必须选择厂家推荐的匹配驱动器。

> [!tip] 记忆口诀
> 伺服电机和驱动器要成套，编码器协议不能错。

# 13 通讯与控制接口

> [!info] 核心结论
> 伺服电机的控制方式取决于伺服驱动器接口，常见有脉冲、模拟量和总线控制。

## ① 分类/公式/规则

> [!note] 🟦 接口选择
> 单轴简单定位可用脉冲控制；  
> 多轴同步和高速联动优先总线控制；  
> 速度或转矩给定可用模拟量。

> [!warning] 🟧 控制匹配公式
> **控制成功 = 控制器接口 + 驱动器接口 + 参数设置 + 信号逻辑一致**

## ② 参考表

| 接口 | 适合场景 |
|---|---|
| **PUL/DIR脉冲** | 简单位置控制 |
| **CW/CCW脉冲** | 正反脉冲控制 |
| **0-10V模拟量** | 速度给定 |
| **±10V模拟量** | 速度/转矩双向给定 |
| **EtherCAT** | 多轴同步、高实时性 |
| **PROFINET** | 西门子系统集成 |
| **Modbus/RS485** | 参数读取和简单控制 |

## ③ 计算/选型示例

- 已知：  
  设备有6个伺服轴，需要同步运动。
- 公式：  
  **多轴同步 = 实时总线 + 运动控制器 + 同步周期**
- 计算：  
  脉冲控制接线复杂，同步性能有限；总线控制更适合多轴联动。
- 结论：  
  应优先选择支持 **EtherCAT等实时总线** 的伺服方案。

> [!tip] 记忆口诀
> 单轴简单用脉冲，多轴同步用总线。

# 14 参数整定与刚性

> [!info] 核心结论
> 伺服电机调试的关键，是让系统既响应快，又不振动、不过冲、不报警。

## ① 分类/公式/规则

> [!note] 🟦 整定理解
> 刚性太低，响应慢、跟随误差大；  
> 刚性太高，容易振动、啸叫、过冲。  
> 参数整定要结合机械刚性、负载惯量和运动节拍。

> [!warning] 🟧 稳定公式
> **稳定运动 = 合适增益 + 合理滤波 + 机械刚性 + 惯量匹配**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **响应慢** | 增益低、加速度小 |
| **振动啸叫** | 增益高、机械共振 |
| **定位过冲** | 刚性过高、惯量大 |
| **跟随误差大** | 转矩不足、增益低 |
| **低速抖动** | 摩擦、间隙、参数不合适 |
| **报警频繁** | 加减速过猛、负载异常 |

## ③ 计算/选型示例

- 已知：  
  伺服定位速度慢，但运行很平稳。
- 公式：  
  **响应慢 = 增益偏低 / 加速度偏小**
- 计算：  
  可逐步提高刚性或加速度，每次调整后观察振动和报警。
- 结论：  
  参数整定要逐步进行，必要时使用厂家自动整定功能。

> [!tip] 记忆口诀
> 刚性低了慢，刚性高了抖；调伺服，要找平衡点。

# 15 常见故障排查

> [!info] 核心结论
> 伺服电机异常要先看驱动器报警代码，再查电源、编码器、负载、参数和通讯。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 上电报警先查电源和编码器线；  
> 运行报警先查负载和加减速；  
> 定位不准先查电子齿轮、脉冲当量和机械间隙。

> [!warning] 🟧 排错公式
> **伺服异常 = 电源问题 + 编码器问题 + 参数问题 + 负载问题 + 通讯问题 + 机械问题**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **不上电** | 电源、断路器、接线异常 |
| **编码器报警** | 编码器线松动、协议不匹配 |
| **过载报警** | 负载重、加速度高、卡滞 |
| **过压报警** | 减速太快、再生能量大 |
| **跟随误差大** | 转矩不足、参数偏软 |
| **定位不准** | 当量错误、间隙、打滑 |
| **电机抖动** | 增益过高、相序或反馈异常 |

## ③ 计算/选型示例

- 已知：  
  伺服低速正常，加速时过载报警。
- 公式：  
  **加速过载 = 加速度过大 + 负载惯量大 + 峰值转矩不足**
- 计算：  
  加速越快，所需峰值转矩越大。
- 结论：  
  可降低加速度、延长加速时间，或选更大转矩电机。

> [!tip] 记忆口诀
> 上电查接线，运动查负载；定位查当量，抖动查增益。

# 16 伺服电机快速选型表

> [!info] 核心结论
> 伺服电机选型必须从机械负载出发，先算转速、转矩和惯量，再选电机和驱动器。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看运动形式：旋转/直线  
> ② 看负载重量和摩擦  
> ③ 看最高速度  
> ④ 看加减速时间  
> ⑤ 算连续转矩  
> ⑥ 算峰值转矩  
> ⑦ 校核惯量比  
> ⑧ 判断是否需要抱闸  
> ⑨ 选择编码器类型  
> ⑩ 校核驱动器、法兰和安装空间

> [!warning] 🟧 最终选型公式
> **推荐型号 = 额定转矩 + 峰值转矩 + 额定转速 + 惯量等级 + 编码器 + 抱闸 + 法兰尺寸 + 驱动器匹配**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 精准定位 | 位置模式、编码器分辨率 |
| 高速往复 | 峰值转矩、加速度、散热 |
| 垂直升降 | 抱闸、保持转矩、安全 |
| 多轴同步 | 实时总线、运动控制器 |
| 不想回零 | 绝对值编码器 |
| 大惯量负载 | 惯量比、减速机 |
| 压装/张力 | 转矩模式、力控反馈 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，目标速度500mm/s，连续转矩需求1.0N·m，加速峰值转矩需求3.0N·m。
- 公式：  
  **电机转速 = 速度 ÷ 导程 × 60**  
  **额定转矩 ≥ 连续转矩**  
  **峰值转矩 ≥ 加速峰值转矩**
- 计算：  
  电机转速 = 500 ÷ 10 × 60 = **3000rpm**  
  额定转矩需 ≥ **1.0N·m**  
  峰值转矩需 ≥ **3.0N·m**
- 结论：  
  应选择满足 **3000rpm、额定转矩≥1.0N·m、峰值转矩≥3.0N·m** 的伺服电机，并继续校核惯量和安装尺寸。

> [!tip] 选型口诀
> 速度看转速，负载看转矩；响应看惯量，垂直看抱闸；电机驱动要配套。
