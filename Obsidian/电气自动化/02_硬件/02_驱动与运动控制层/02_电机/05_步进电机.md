# 01 步进电机总览

> [!info] 核心结论
> 步进电机是一种把 **电脉冲信号转换成固定角度位移** 的电机，适合低成本、简单定位和中低速控制场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> PLC或运动控制器每发一个脉冲，步进驱动器就控制步进电机前进一步或一个细分步。  
> 步进电机常用于 **滑台、转盘、送料、打印机、点胶机、小型自动化机构**。

> [!warning] 🟧 通用选型公式
> **步进电机选型 = 相数 + 步距角 + 保持转矩 + 电流 + 电压 + 转速 + 负载惯量 + 驱动器匹配**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制方式** | 脉冲控制 |
| **常见相数** | 两相、三相、五相 |
| **常见步距角** | **1.8°、0.9°** 等 |
| **优点** | 成本低、控制简单、低速力矩好 |
| **缺点** | 高速力矩下降、可能丢步、易共振 |
| **典型应用** | 小型滑台、送料、分度、低速定位 |

## ③ 计算/选型示例

- 已知：  
  小型丝杆滑台需要低速定位，负载较轻，成本敏感。
- 公式：  
  **低速 + 轻载 + 简单定位 → 步进电机可评估**
- 计算：  
  该场景对高速响应和闭环精度要求不高，步进电机可以满足基本定位。
- 结论：  
  可选择 **步进电机 + 步进驱动器**，转矩余量和最高速度需按厂家手册校核。

> [!tip] 记忆口诀
> 简单定位用步进，高速精准看伺服。

# 02 底层原理

> [!info] 核心结论
> 步进电机靠定子绕组按顺序通电，形成旋转磁场，带动转子一步一步转动。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 步进驱动器按顺序给电机绕组通电，定子磁场一步步移动，转子受到磁场吸引后跟随转动。

> [!warning] 🟧 运动逻辑
> **电脉冲 → 绕组通电顺序变化 → 磁场移动 → 转子转动 → 机械位移**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子绕组** | 产生磁场 |
| **转子** | 跟随磁场转动 |
| **齿槽结构** | 决定步距角 |
| **驱动器** | 分配绕组电流 |
| **控制器** | 输出脉冲和方向 |
| **负载机构** | 丝杆、同步带、齿轮等 |

## ③ 计算/选型示例

- 已知：  
  步进电机步距角为 **1.8°**。
- 公式：  
  **每圈步数 = 360° ÷ 步距角**
- 计算：  
  每圈步数 = 360 ÷ 1.8 = **200步/圈**
- 结论：  
  不细分时，电机转一圈需要 **200个脉冲**。

> [!tip] 记忆口诀
> 一脉冲走一步，步距角定精度。

# 03 常见类型

> [!info] 核心结论
> 工业自动化中最常见的是 **混合式步进电机**，兼顾转矩、精度和成本。

## ① 分类/公式/规则

> [!note] 🟦 类型理解
> 步进电机按结构常分为 **反应式、永磁式、混合式**。  
> 按相数常见 **两相、三相、五相步进电机**。

> [!warning] 🟧 选型判断
> **普通自动化定位 → 两相混合式步进电机最常见**  
> **更平稳 / 更高细分性能 → 三相或五相可评估**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **反应式** | 结构简单，转矩较小 |
| **永磁式** | 有永磁转子，低速力矩较好 |
| **混合式** | **常用，精度和转矩较好** |
| **两相步进** | 应用最广，成本低 |
| **三相步进** | 运行更平稳 |
| **五相步进** | 步距更细，成本较高 |

## ③ 计算/选型示例

- 已知：  
  设备为普通小型送料机构，要求成本低、控制简单。
- 公式：  
  **普通定位 + 成本敏感 = 两相混合式步进优先**
- 计算：  
  两相混合式步进电机驱动器选择多，调试简单。
- 结论：  
  可优先选择 **两相混合式步进电机**。

> [!tip] 记忆口诀
> 两相最常见，三相更平稳，五相更细分。

# 04 步距角与每圈步数

> [!info] 核心结论
> 步距角越小，每圈需要的步数越多，理论分辨率越高。

## ① 分类/公式/规则

> [!warning] 🟧 每圈步数公式
> **每圈步数 = 360° ÷ 步距角**

> [!note] 🟦 工程理解
> 1.8°步进电机每圈200步；  
> 0.9°步进电机每圈400步。  
> 步距角越小，单步角度越细，但实际精度还受负载、细分、机械间隙影响。

## ② 参考表

| 步距角 | 每圈基本步数 |
|---|---:|
| **1.8°** | **200步/圈** |
| **0.9°** | **400步/圈** |
| **7.5°** | 48步/圈 |
| **15°** | 24步/圈 |
| **细分后** | 每圈脉冲数增加 |
| **实际精度** | 需按机械系统校核 |

## ③ 计算/选型示例

- 已知：  
  步进电机步距角为 **0.9°**。
- 公式：  
  **每圈步数 = 360 ÷ 0.9**
- 计算：  
  每圈步数 = **400步/圈**
- 结论：  
  不细分时，电机转一圈需要 **400个脉冲**。

> [!tip] 记忆口诀
> 步距角越小，一圈步数越多。

# 05 细分与脉冲数

> [!info] 核心结论
> 细分能让运动更平滑、控制分辨率更高，但不代表实际定位精度无限提高。

## ① 分类/公式/规则

> [!note] 🟦 细分理解
> 驱动器把一个整步分成多个小步，例如8细分、16细分、32细分。  
> 细分越大，每圈需要的脉冲越多，对控制器脉冲频率要求也越高。

> [!warning] 🟧 每圈脉冲数公式
> **每圈脉冲数 = 基本步数 × 细分数**

## ② 参考表

| 细分 | 1.8°电机每圈脉冲 |
|---|---:|
| **1细分** | 200 |
| **2细分** | 400 |
| **4细分** | 800 |
| **8细分** | 1600 |
| **16细分** | 3200 |
| **32细分** | 6400 |

## ③ 计算/选型示例

- 已知：  
  1.8°步进电机，驱动器设置 **16细分**。
- 公式：  
  **每圈脉冲数 = 200 × 16**
- 计算：  
  每圈脉冲数 = **3200脉冲/圈**
- 结论：  
  电机转一圈需要控制器输出 **3200个脉冲**。

> [!tip] 记忆口诀
> 细分让它走得细，精度还要看机械。

# 06 转速与脉冲频率

> [!info] 核心结论
> 步进电机转速由脉冲频率决定，频率越高转速越快，但过高容易失步。

## ① 分类/公式/规则

> [!warning] 🟧 转速公式
> **rpm = 脉冲频率 ÷ 每圈脉冲数 × 60**

> [!note] 🟦 参数说明
> 脉冲频率单位是Hz，也就是每秒脉冲数。  
> 每圈脉冲数由步距角和细分数共同决定。

## ② 参考表

| 参数 | 影响 |
|---|---|
| **脉冲频率** | 决定转速 |
| **细分数** | 影响每圈脉冲数 |
| **电机转矩** | 高速会下降 |
| **加速时间** | 太短易丢步 |
| **负载惯量** | 越大越难加速 |
| **控制器频率** | 必须满足最高速度需求 |

## ③ 计算/选型示例

- 已知：  
  每圈脉冲数为 **3200脉冲/圈**，脉冲频率为 **16000Hz**。
- 公式：  
  **rpm = 16000 ÷ 3200 × 60**
- 计算：  
  rpm = **300rpm**
- 结论：  
  电机理论转速为 **300rpm**。

> [!tip] 记忆口诀
> 脉冲管位置，频率管速度。

# 07 转矩与转速特性

> [!info] 核心结论
> 步进电机低速转矩较好，高速转矩会明显下降，选型必须看转矩曲线。

## ① 分类/公式/规则

> [!note] 🟦 工程理解
> 步进电机绕组有电感，高速时电流来不及建立，输出转矩会下降。  
> 所以步进电机不适合盲目追求高转速。

> [!warning] 🟧 高速风险公式
> **高速失步风险 = 负载转矩需求 > 电机可用转矩**

## ② 参考表

| 区域 | 特点 |
|---|---|
| **低速区** | 转矩大，容易控制 |
| **中速区** | 常用工作区 |
| **高速区** | 转矩下降明显 |
| **启动区** | 加速度不能过大 |
| **共振区** | 可能抖动、噪声大 |
| **最终判断** | 看厂家转矩-转速曲线 |

## ③ 计算/选型示例

- 已知：  
  负载需要0.8N·m，目标转速为600rpm。
- 公式：  
  **可用条件 = 600rpm时电机输出转矩 ≥ 0.8N·m × 余量**
- 计算：  
  不能只看保持转矩，必须查600rpm对应动态转矩。
- 结论：  
  若600rpm时转矩不足，应换大电机、提高电压、降低速度或改用伺服。

> [!tip] 记忆口诀
> 步进低速有劲，高速力气下降快。

# 08 保持转矩与定位力

> [!info] 核心结论
> 保持转矩表示电机通电静止时能保持位置的能力，但它不等于高速运行转矩。

## ① 分类/公式/规则

> [!note] 🟦 参数区别
> **保持转矩**：电机通电静止时抵抗外力转动的能力。  
> **运行转矩**：电机转动时实际可输出转矩。  
> 高速时运行转矩通常小于保持转矩。

> [!warning] 🟧 选型规则
> **负载所需转矩 × 安全系数 ≤ 电机可用转矩**

## ② 参考表

| 参数 | 含义 |
|---|---|
| **保持转矩** | 静止保持能力 |
| **定位转矩** | 断电时残余保持能力，通常较小 |
| **动态转矩** | 运行时可输出转矩 |
| **启动转矩** | 从静止启动能力 |
| **安全系数** | 防止丢步 |
| **负载冲击** | 需额外留余量 |

## ③ 计算/选型示例

- 已知：  
  负载运行需要0.5N·m，建议留2倍余量。
- 公式：  
  **电机可用转矩 ≥ 0.5 × 2**
- 计算：  
  电机可用转矩 ≥ **1.0N·m**
- 结论：  
  应选择在目标转速下仍能输出 **1.0N·m以上** 的步进电机。

> [!tip] 记忆口诀
> 保持转矩看静止，运行转矩看曲线。

# 09 开环控制与丢步

> [!info] 核心结论
> 普通步进电机多为开环控制，控制器发了脉冲不代表电机一定准确走到位。

## ① 分类/公式/规则

> [!note] 🟦 丢步原因
> 丢步常见原因包括：负载过重、加速度过大、速度过高、电流不足、电源电压低、机械卡滞、共振。

> [!warning] 🟧 防丢步公式
> **可靠运行 = 转矩余量 + 合理加减速 + 机械顺畅 + 电流电压匹配**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **位置越来越偏** | 丢步或机械打滑 |
| **启动抖动** | 加速度过大、共振 |
| **高速失步** | 高速转矩不足 |
| **堵转啸叫** | 负载卡死或力矩不足 |
| **偶发偏差** | 干扰、脉冲丢失 |
| **带载不动** | 电机选小或负载过重 |

## ③ 计算/选型示例

- 已知：  
  滑台低速正常，高速时位置逐渐偏移。
- 公式：  
  **高速偏移 = 高速转矩不足 + 丢步风险**
- 计算：  
  步进电机高速转矩下降，如果负载转矩超过可用转矩就会失步。
- 结论：  
  可降低速度、延长加速时间、增大电机规格或改用闭环步进/伺服。

> [!tip] 记忆口诀
> 步进最怕丢步，速度越高越要留余量。

# 10 共振与振动

> [!info] 核心结论
> 步进电机容易在某些速度段出现共振，表现为抖动、噪声、失步或运行不稳。

## ① 分类/公式/规则

> [!note] 🟦 共振理解
> 步进电机是一格一格运动的，天然存在转矩脉动。  
> 当脉冲频率接近机械系统固有频率时，就可能出现共振。

> [!warning] 🟧 抑制共振公式
> **降低共振 = 合理细分 + 平滑加减速 + 避开共振区 + 提高机械刚性**

## ② 参考表

| 方法 | 作用 |
|---|---|
| **提高细分** | 运动更平滑 |
| **S曲线加减速** | 降低冲击 |
| **避开共振速度** | 减少抖动 |
| **增加阻尼** | 降低振动 |
| **优化联轴器** | 减少机械冲击 |
| **闭环步进** | 改善稳定性 |

## ③ 计算/选型示例

- 已知：  
  电机在某一速度附近抖动明显，换速度后正常。
- 公式：  
  **局部抖动 = 可能处在共振区**
- 计算：  
  如果只在某个转速段异常，多半与共振有关。
- 结论：  
  可调整细分、加速度，或避开该速度段运行。

> [!tip] 记忆口诀
> 步进抖不一定坏，可能踩到共振点。

# 11 电流、电压与发热

> [!info] 核心结论
> 步进电机发热较常见，电流决定转矩和温升，电压影响高速性能。

## ① 分类/公式/规则

> [!note] 🟦 参数理解
> 电流偏小，转矩不足；  
> 电流偏大，电机发热严重；  
> 电压偏低，高速性能差；  
> 电压超限，可能损坏驱动器。

> [!warning] 🟧 设置规则
> **驱动器电流 ≈ 电机额定相电流**  
> **供电电压 ≤ 驱动器允许最高电压**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **相电流** | 决定转矩和发热 |
| **静止半流** | 降低待机发热 |
| **电源电压** | 影响高速转矩 |
| **绕组电感** | 影响电流建立速度 |
| **散热条件** | 影响长期运行 |
| **环境温度** | 影响温升余量 |

## ③ 计算/选型示例

- 已知：  
  步进电机额定相电流为 **2.8A**。
- 公式：  
  **驱动器电流 ≈ 电机额定相电流**
- 计算：  
  驱动器电流档位应选择接近 **2.8A** 的档位。
- 结论：  
  最终需结合温升、负载和厂家手册校核。

> [!tip] 记忆口诀
> 电流小没力，电流大太热；电压低跑不快，电压高别超限。

# 12 接线与相序

> [!info] 核心结论
> 步进电机接线必须正确识别相线，接错会导致抖动、不转、反转或驱动器报警。

## ① 分类/公式/规则

> [!note] 🟦 接线重点
> 两相步进电机通常有 **A相、B相** 两组绕组。  
> 常见线数有4线、6线、8线，接法需按电机和驱动器手册确认。

> [!warning] 🟧 接线匹配公式
> **正常运行 = 相线分组正确 + 驱动器相序正确 + 电流设置正确**

## ② 参考表

| 线制 | 特点 |
|---|---|
| **4线** | 最常见，两相绕组直接引出 |
| **6线** | 每相有中心抽头 |
| **8线** | 可串联/并联接法 |
| **串联接法** | 低速力矩较好，高速较差 |
| **并联接法** | 高速性能较好，电流需求大 |
| **最终接法** | 需按厂家手册校核 |

## ③ 计算/选型示例

- 已知：  
  电机上电锁住，但运行时来回抖动不转。
- 公式：  
  **抖动不转 = 相线分组错误 / 相序错误 / 脉冲异常**
- 计算：  
  若A相和B相线混接，磁场顺序错误，电机无法正常旋转。
- 结论：  
  应重新确认绕组分组和驱动器端子接线。

> [!tip] 记忆口诀
> 有力不转查脉冲，抖动不转查相线。

# 13 丝杆步进应用

> [!info] 核心结论
> 步进电机带丝杆时，要把脉冲数换算成直线位移，并校核转速和推力。

## ① 分类/公式/规则

> [!warning] 🟧 丝杆位移公式
> **直线位移 = 脉冲数 ÷ 每圈脉冲数 × 丝杆导程**

> [!warning] 🟧 脉冲当量公式
> **脉冲当量 = 丝杆导程 ÷ 每圈脉冲数**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **丝杆导程** | 一圈走多远 |
| **每圈脉冲数** | 由步距角和细分决定 |
| **脉冲当量** | 每个脉冲走多远 |
| **直线速度** | 由转速和导程决定 |
| **推力需求** | 决定转矩 |
| **机械间隙** | 影响实际精度 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程 **5mm**，1.8°电机，16细分。
- 公式：  
  **每圈脉冲数 = 200 × 16**  
  **脉冲当量 = 5 ÷ 3200**
- 计算：  
  每圈脉冲数 = **3200脉冲/圈**  
  脉冲当量 = **0.0015625mm/脉冲**
- 结论：  
  PLC每发1个脉冲，理论移动约 **0.00156mm**。

> [!tip] 记忆口诀
> 导程除脉冲，就是每脉冲走多远。

# 14 步进电机 vs 伺服电机

> [!info] 核心结论
> 步进电机适合低成本简单定位，伺服电机适合高速、高精度、高动态和防丢步场景。

## ① 分类/公式/规则

> [!note] 🟦 快速区别
> 步进电机多为开环控制，结构简单；  
> 伺服电机有编码器反馈，可实时修正位置和速度。

> [!warning] 🟧 选择公式
> **低速轻载 / 成本敏感 → 步进电机**  
> **高速高精度 / 负载变化大 → 伺服电机**

## ② 参考表

| 对比项 | 步进电机 | 伺服电机 |
|---|---|---|
| 控制方式 | 多为开环 | **闭环反馈** |
| 成本 | 较低 | 较高 |
| 低速力矩 | 较好 | 较好 |
| 高速性能 | 较弱 | **较强** |
| 丢步风险 | 有 | 较低 |
| 调试复杂度 | 简单 | 较复杂 |

## ③ 计算/选型示例

- 已知：  
  设备高速往复运动，丢步会导致产品报废。
- 公式：  
  **高速 + 不允许丢步 = 伺服优先**
- 计算：  
  开环步进无法实时确认是否到位，高速下转矩下降明显。
- 结论：  
  应优先选择 **伺服电机** 或闭环步进系统。

> [!tip] 记忆口诀
> 低速简单用步进，高速精准用伺服。

# 15 常见故障排查

> [!info] 核心结论
> 步进电机异常时，按“电源 → 驱动器 → 接线 → 脉冲 → 电机 → 负载”的顺序排查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看驱动器是否报警，再查电机相线和控制信号。  
> 先空载低速试运行，再带载提高速度。

> [!warning] 🟧 排错公式
> **步进异常 = 电源异常 + 驱动器设置错 + 相线错误 + 脉冲异常 + 负载过重 + 共振**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **电机不锁轴** | 驱动器未使能、电源异常 |
| **锁轴但不转** | 无脉冲、控制模式错误 |
| **抖动不转** | 相线接错、相序错误 |
| **方向反了** | 方向信号反或相序反 |
| **高速丢步** | 转矩不足、加速度过大 |
| **发热明显** | 电流设置偏大、散热差 |
| **噪声大** | 共振、细分不合适 |

## ③ 计算/选型示例

- 已知：  
  步进电机低速正常，高速丢步。
- 公式：  
  **高速丢步 = 高速转矩不足 + 加速度过大 + 负载惯量大**
- 计算：  
  步进电机高速转矩下降，负载越大越容易失步。
- 结论：  
  可降低最高速度、延长加速时间、提高合规电源电压，或更换更大电机。

> [!tip] 记忆口诀
> 不锁查电源，锁住不转查脉冲，抖动不转查相线。

# 16 步进电机快速选型表

> [!info] 核心结论
> 步进电机选型先看负载转矩和速度，再看步距角、相数、电流、驱动器和是否需要闭环。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看运动方式：旋转/直线  
> ② 看负载重量和摩擦  
> ③ 看目标速度  
> ④ 看加减速时间  
> ⑤ 算所需转矩  
> ⑥ 选步距角和相数  
> ⑦ 选保持转矩和电流  
> ⑧ 选驱动器和细分  
> ⑨ 校核脉冲频率  
> ⑩ 校核发热、共振和安装尺寸

> [!warning] 🟧 最终选型公式
> **推荐型号 = 相数 + 步距角 + 保持转矩 + 额定电流 + 机座尺寸 + 驱动器电流 + 速度转矩曲线**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单低速定位 | 两相步进电机 |
| 运行更平稳 | 三相/五相步进可评估 |
| 高分辨率 | 小步距角 + 合理细分 |
| 高速运行 | 看转矩曲线和电源电压 |
| 怕丢步 | 闭环步进或伺服 |
| 丝杆滑台 | 脉冲当量和推力 |
| 长时间通电 | 温升和静止半流 |

## ③ 计算/选型示例

- 已知：  
  1.8°步进电机，16细分，丝杆导程10mm，目标移动100mm。
- 公式：  
  **每圈脉冲数 = 200 × 16**  
  **所需脉冲数 = 移动距离 ÷ 导程 × 每圈脉冲数**
- 计算：  
  每圈脉冲数 = **3200脉冲/圈**  
  所需脉冲数 = 100 ÷ 10 × 3200 = **32000脉冲**
- 结论：  
  PLC需输出约 **32000个脉冲**，滑台理论移动100mm，实际精度需按机械间隙和是否丢步校核。

> [!tip] 选型口诀
> 先算转矩，再看速度；低速用步进，高速看伺服；细分看平滑，精度看机械。
