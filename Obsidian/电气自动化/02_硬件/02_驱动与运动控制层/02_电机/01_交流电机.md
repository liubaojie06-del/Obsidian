# 01 交流电机总览

> [!info] 核心结论
> 交流电机使用交流电源驱动，常用于风机、水泵、输送线、机床、压缩机、搅拌机等工业设备。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 工业现场最常见的是 **三相异步电机**。  
> 它结构简单、可靠性高、维护方便，是自动化设备中最常用的动力源之一。

> [!warning] 🟧 通用选型公式
> **交流电机选型 = 电压等级 + 功率 + 转速 + 转矩 + 负载类型 + 工作制 + 安装方式 + 防护等级**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **供电方式** | 单相AC / 三相AC |
| **常见电压** | AC220V、AC380V、AC400V、AC660V等，需按现场校核 |
| **常见类型** | 异步电机、同步电机、单相电机 |
| **典型负载** | 风机、水泵、输送、搅拌、压缩机 |
| **控制方式** | 工频启动、星三角启动、软启动、变频启动 |
| **重点参数** | 功率、电流、转速、极数、绝缘、防护、安装 |

## ③ 计算/选型示例

- 已知：  
  一台输送线需要连续运行，现场电源为三相AC380V。
- 公式：  
  **电机需求 = AC380V + 连续运行 + 负载转矩满足**
- 计算：  
  工业输送线通常可选三相异步电机，再根据负载功率和转速确定规格。
- 结论：  
  可选择 **三相异步交流电机**，功率、电流和安装尺寸需按厂家手册校核。

> [!tip] 记忆口诀
> 工业动力看三相，简单可靠用异步。

# 02 底层原理

> [!info] 核心结论
> 交流电机的核心原理，是交流电在定子中产生旋转磁场，带动转子旋转。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 三相交流电通入定子绕组后，会产生旋转磁场。  
> 转子在旋转磁场作用下产生电磁转矩，从而带动负载旋转。

> [!warning] 🟧 能量转换逻辑
> **交流电能 → 旋转磁场 → 电磁转矩 → 机械旋转**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子** | 产生旋转磁场 |
| **转子** | 输出旋转运动 |
| **气隙** | 定子和转子之间的磁场通道 |
| **轴承** | 支撑转轴旋转 |
| **机座** | 固定和散热 |
| **风扇** | 给电机散热 |

## ③ 计算/选型示例

- 已知：  
  电机通电后有嗡声但不转。
- 公式：  
  **电磁转矩 > 负载阻力 → 电机转动**
- 计算：  
  如果缺相、负载卡死或启动转矩不足，电机可能无法转动。
- 结论：  
  应检查三相电源、接线、机械负载和保护装置。

> [!tip] 记忆口诀
> 三相造磁场，磁场带转子。

# 03 三相异步电机

> [!info] 核心结论
> 三相异步电机是工业现场最常见的交流电机，特点是结构简单、价格低、可靠性高。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> “异步”表示转子转速略低于旋转磁场的同步转速。  
> 这个差值叫 **转差**，异步电机正是靠转差产生转矩。

> [!warning] 🟧 转差公式
> **s =（nₛ - n）÷ nₛ**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **nₛ** | 同步转速 |
| **n** | 实际转速 |
| **s** | 转差率 |
| **优点** | 可靠、便宜、维护少 |
| **缺点** | 直接启动电流大 |
| **常见控制** | 工频、软启动、变频 |

## ③ 计算/选型示例

- 已知：  
  某4极电机同步转速为1500rpm，实际转速为1450rpm。
- 公式：  
  **s =（1500 - 1450）÷ 1500**
- 计算：  
  s = 50 ÷ 1500 ≈ **3.33%**
- 结论：  
  该异步电机转差率约为 **3.33%**。

> [!tip] 记忆口诀
> 异步不等速，差一点才有力。

# 04 同步转速与极数

> [!info] 核心结论
> 交流电机的同步转速由电源频率和电机极数决定。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **nₛ = 120 × f ÷ P**

> [!note] 参数说明
> **nₛ**：同步转速，rpm  
> **f**：电源频率，Hz  
> **P**：电机极数

## ② 参考表

| 极数 | 50Hz同步转速 |
|---|---|
| **2极** | 3000rpm |
| **4极** | 1500rpm |
| **6极** | 1000rpm |
| **8极** | 750rpm |

## ③ 计算/选型示例

- 已知：  
  一台6极交流电机，电源频率为50Hz。
- 公式：  
  **nₛ = 120 × 50 ÷ 6**
- 计算：  
  nₛ = **1000rpm**
- 结论：  
  6极电机在50Hz下同步转速为 **1000rpm**，异步电机实际转速会略低。

> [!tip] 记忆口诀
> 极数越多，转速越低；4极50Hz，大约1500转。

# 05 功率、转矩与转速

> [!info] 核心结论
> 交流电机选型不能只看功率，还要看转速和转矩是否满足负载。

## ① 分类/公式/规则

> [!warning] 🟧 功率转矩公式
> **P = T × n ÷ 9550**

> [!note] 参数说明
> **P**：功率，kW  
> **T**：转矩，N·m  
> **n**：转速，rpm

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定功率** | 电机长期输出能力 |
| **额定转速** | 正常运行速度 |
| **额定转矩** | 长期带载能力 |
| **启动转矩** | 启动时能否带动负载 |
| **最大转矩** | 短时抗冲击能力 |
| **额定电流** | 电缆、保护和变频器选型依据 |

## ③ 计算/选型示例

- 已知：  
  电机功率为 **2.2kW**，额定转速为 **1450rpm**。
- 公式：  
  **T = 9550 × P ÷ n**
- 计算：  
  T = 9550 × 2.2 ÷ 1450 ≈ **14.49N·m**
- 结论：  
  该电机额定转矩约为 **14.5N·m**。

> [!tip] 记忆口诀
> 功率看等级，转矩看力气，转速看快慢。

# 06 电压、接法与铭牌

> [!info] 核心结论
> 交流电机接线前必须看铭牌，确认电压、频率、接法、电流和功率。

## ① 分类/公式/规则

> [!note] 🟦 常见接法
> 三相电机常见 **星形Y接** 和 **三角形Δ接**。  
> 电机铭牌常见写法如 **220/380V Δ/Y** 或 **380/660V Δ/Y**，表示不同电压下对应不同接法。

> [!warning] 🟧 接线规则
> **低电压用Δ，高电压用Y**  
> 具体必须以电机铭牌为准。

## ② 参考表

| 铭牌信息 | 作用 |
|---|---|
| **额定电压** | 判断供电是否匹配 |
| **额定电流** | 保护器和线缆选型 |
| **频率** | 50Hz/60Hz需确认 |
| **接法** | Y接或Δ接 |
| **功率** | 负载能力参考 |
| **绝缘等级** | 允许温升能力 |
| **防护等级** | 防尘防水能力 |

## ③ 计算/选型示例

- 已知：  
  电机铭牌为 **380/660V Δ/Y**，现场电源为三相380V。
- 公式：  
  **低电压对应Δ接**
- 计算：  
  380V是铭牌低电压侧，应采用 **Δ接法**。
- 结论：  
  该电机在380V电源下通常接三角形，最终以厂家铭牌和接线盒说明为准。

> [!tip] 记忆口诀
> 铭牌不看就接线，轻则不转，重则烧电机。

# 07 单相交流电机

> [!info] 核心结论
> 单相交流电机常用于小功率设备，通常需要电容或辅助绕组产生启动转矩。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 单相电源本身不容易产生旋转磁场，所以单相电机常用启动绕组、电容或罩极结构来帮助启动。

> [!warning] 🟧 单相启动逻辑
> **主绕组 + 辅助绕组/电容 → 相位差 → 启动转矩**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **电容启动电机** | 启动转矩较大 |
| **电容运行电机** | 运行较平稳 |
| **电容启动运行电机** | 启动和运行性能较好 |
| **罩极电机** | 小功率，结构简单 |
| **典型应用** | 小风机、小水泵、家电、小型设备 |
| **注意** | 电容损坏会导致启动困难 |

## ③ 计算/选型示例

- 已知：  
  单相电机通电后嗡嗡响，但不能启动，用手拨一下才转。
- 公式：  
  **启动失败 = 启动转矩不足**
- 计算：  
  常见原因是启动电容损坏、启动绕组异常或负载过重。
- 结论：  
  应检查电容容量、启动绕组和机械负载。

> [!tip] 记忆口诀
> 单相启动靠电容，电容坏了只嗡不转。

# 08 同步电机

> [!info] 核心结论
> 同步电机的转子转速与旋转磁场同步，适合高效率、高功率因数或精确速度场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 同步电机正常运行时，转速等于同步转速。  
> 永磁同步电机常用于伺服系统、变频驱动和高效节能场景。

> [!warning] 🟧 同步特性
> **同步电机转速 = 同步转速**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **永磁同步电机** | 高效率、控制性能好 |
| **励磁同步电机** | 功率因数可调 |
| **磁阻同步电机** | 结构特殊，效率较高 |
| **伺服电机** | 多为永磁同步结构 |
| **优点** | 转速稳定、效率高 |
| **注意** | 通常需要变频器或专用驱动 |

## ③ 计算/选型示例

- 已知：  
  一台4极同步电机，电源频率为50Hz。
- 公式：  
  **nₛ = 120 × 50 ÷ 4**
- 计算：  
  nₛ = **1500rpm**
- 结论：  
  该同步电机理论运行转速为 **1500rpm**。

> [!tip] 记忆口诀
> 异步有转差，同步跟磁场。

# 09 启动方式

> [!info] 核心结论
> 交流电机启动方式要根据电机功率、负载惯量、电网容量和工艺要求选择。

## ① 分类/公式/规则

> [!note] 🟦 常见启动方式
> 小功率可直接启动；  
> 中大功率常用星三角、软启动或变频启动；  
> 重载或需要调速的设备优先考虑变频器。

> [!warning] 🟧 启动方式选择公式
> **启动方案 = 电机功率 + 启动电流 + 负载转矩 + 电网容量 + 工艺要求**

## ② 参考表

| 启动方式 | 特点 |
|---|---|
| **直接启动DOL** | 简单，但启动电流大 |
| **星三角启动** | 降低启动电流，启动转矩也降低 |
| **自耦降压启动** | 适合较大功率 |
| **软启动器** | 降低冲击，不能长期调速 |
| **变频器启动** | 平滑启动，可调速 |
| **绕线电机启动** | 特殊电机应用 |

## ③ 计算/选型示例

- 已知：  
  设备启动时对电网冲击大，且运行中需要调速。
- 公式：  
  **需要调速 + 降低启动冲击 → 变频启动**
- 计算：  
  软启动只能改善启动过程，不能长期调速；变频器可启动和运行调速。
- 结论：  
  应优先选择 **变频器控制交流电机**。

> [!tip] 记忆口诀
> 只想软启动用软启，既要启动又要调速用变频。

# 10 星三角启动

> [!info] 核心结论
> 星三角启动可以降低启动电流，但启动转矩也会明显下降，不适合重载启动。

## ① 分类/公式/规则

> [!note] 🟦 工作过程
> 启动时先接成星形Y，降低绕组电压；  
> 电机转速上来后，再切换成三角形Δ正常运行。

> [!warning] 🟧 星三角特点
> **星形启动电流降低，启动转矩也降低**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **适合场景** | 轻载或空载启动 |
| **不适合** | 重载、频繁启动、大惯量负载 |
| **接触器** | 通常需要主、星、角接触器 |
| **时间继电器** | 控制切换时间 |
| **风险** | 切换冲击、接线错误 |
| **前提** | 电机必须支持Δ运行 |

## ③ 计算/选型示例

- 已知：  
  搅拌机带料启动，启动转矩需求较大。
- 公式：  
  **星三角启动转矩较低**
- 计算：  
  带料搅拌属于重载启动，星形启动可能转不起来。
- 结论：  
  不宜盲目使用星三角启动，应评估变频器或重载启动方案。

> [!tip] 记忆口诀
> 星三角省电流，也省了启动力。

# 11 变频调速

> [!info] 核心结论
> 交流电机调速最常见的方法是使用变频器，通过改变输出频率来改变电机转速。

## ① 分类/公式/规则

> [!note] 🟦 调速逻辑
> 频率越高，同步转速越高；频率越低，同步转速越低。  
> 变频器通常同时调整电压和频率，保持电机磁通合理。

> [!warning] 🟧 变频调速公式
> **nₛ = 120 × f ÷ P**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **输出频率** | 决定电机速度 |
| **V/F控制** | 普通风机水泵常用 |
| **矢量控制** | 低速转矩更好 |
| **加速时间** | 影响启动电流和冲击 |
| **减速时间** | 影响母线过压风险 |
| **制动电阻** | 快速减速时消耗能量 |

## ③ 计算/选型示例

- 已知：  
  4极电机，变频器输出频率为30Hz。
- 公式：  
  **nₛ = 120 × 30 ÷ 4**
- 计算：  
  nₛ = **900rpm**
- 结论：  
  4极电机在30Hz时同步转速约为 **900rpm**，实际异步转速略低。

> [!tip] 记忆口诀
> 要调交流电机速度，先调频率。

# 12 电机保护

> [!info] 核心结论
> 交流电机保护重点是过载、短路、缺相、过热、接地、堵转和绝缘异常。

## ① 分类/公式/规则

> [!note] 🟦 保护思路
> 短路用断路器或熔断器保护；  
> 过载用热继电器、电机保护器或变频器保护；  
> 高要求场景可加温度传感器和绝缘监测。

> [!warning] 🟧 保护选择公式
> **电机保护 = 短路保护 + 过载保护 + 缺相保护 + 温度保护 + 接地保护**

## ② 参考表

| 保护类型 | 作用 |
|---|---|
| **断路器** | 短路和过流保护 |
| **热继电器** | 过载保护 |
| **电机保护器** | 缺相、过载、堵转等 |
| **PTC/PT100** | 电机温度保护 |
| **接地保护** | 绝缘或接地故障 |
| **变频器保护** | 过流、过压、过载、过热等 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流为 **8A**。
- 公式：  
  **热继电器整定值 ≈ 电机额定电流**
- 计算：  
  热继电器可初步整定在 **8A附近**。
- 结论：  
  最终整定需按电机铭牌、启动方式、工作制和厂家说明校核。

> [!tip] 记忆口诀
> 短路靠断路器，过载靠热保护，缺相堵转不能漏。

# 13 安装方式与防护等级

> [!info] 核心结论
> 交流电机选型不仅看电气参数，还要看安装方式、防护等级、冷却方式和现场环境。

## ① 分类/公式/规则

> [!note] 🟦 机械匹配
> 电机必须和设备安装结构匹配，包括法兰、底脚、轴径、轴伸、键槽、出线方向和接线盒位置。

> [!warning] 🟧 安装匹配公式
> **机械可用 = 安装方式 + 法兰尺寸 + 轴径轴长 + 防护等级 + 冷却方式**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **B3** | 卧式底脚安装 |
| **B5** | 法兰安装 |
| **B35** | 底脚 + 法兰安装 |
| **IP等级** | 防尘防水能力 |
| **冷却方式** | 自扇冷、强冷、水冷等 |
| **绝缘等级** | 影响允许温升 |
| **环境条件** | 高温、粉尘、潮湿、腐蚀需校核 |

## ③ 计算/选型示例

- 已知：  
  现场为粉尘较多的输送设备，需要电机长期运行。
- 公式：  
  **环境适配 = 防护等级 + 散热方式 + 密封结构**
- 计算：  
  粉尘环境中防护等级过低，容易导致内部污染和散热变差。
- 结论：  
  应选择合适IP等级的电机，并制定清灰和维护计划。

> [!tip] 记忆口诀
> 电气参数能用，还要机械装得上，环境扛得住。

# 14 常见故障排查

> [!info] 核心结论
> 交流电机故障排查按“电源 → 接线 → 保护 → 电机 → 负载 → 控制设备”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查三相电压是否正常，再查接触器、热继电器、变频器或软启动器。  
> 先空载试运行，再带载测试。

> [!warning] 🟧 排错公式
> **交流电机异常 = 电源异常 + 缺相 + 接线错误 + 保护动作 + 负载卡滞 + 电机损坏**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **不转** | 无电源、缺相、保护未复位 |
| **嗡嗡响不转** | 缺相、负载卡死、电容坏 |
| **发热大** | 过载、散热差、电压异常 |
| **电流大** | 负载重、轴承坏、绕组问题 |
| **振动大** | 轴承、联轴器、转子不平衡 |
| **频繁跳闸** | 短路、接地、过载、保护整定不当 |
| **方向反** | 任意两相对调即可反向，需停电操作 |

## ③ 计算/选型示例

- 已知：  
  三相电机启动后声音异常，电流很大，转速上不来。
- 公式：  
  **电流大 + 转速低 = 负载过重 / 缺相 / 电机异常**
- 计算：  
  可先断开负载空载试机，判断是机械问题还是电机问题。
- 结论：  
  应检查三相电压、接线、轴承、负载是否卡滞和绕组绝缘。

> [!tip] 记忆口诀
> 不转先查电，发热先查流，振动先查轴，跳闸先查保护。

# 15 交流电机快速选型表

> [!info] 核心结论
> 交流电机选型先确认电源和负载，再确定功率、转速、极数、启动方式和安装环境。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看单相还是三相  
> ② 看电压和频率  
> ③ 看负载类型  
> ④ 算功率和转矩  
> ⑤ 选极数和转速  
> ⑥ 看启动方式  
> ⑦ 判断是否需要调速  
> ⑧ 看安装方式  
> ⑨ 看防护等级和绝缘等级  
> ⑩ 校核保护、线缆和厂家手册

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电压等级 + 功率 + 极数/转速 + 负载类型 + 启动方式 + 安装方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 普通工业动力 | 三相异步电机 |
| 小功率家用/轻载 | 单相交流电机 |
| 需要调速 | 变频器 + 交流电机 |
| 重载启动 | 启动转矩、软启动/变频 |
| 风机水泵 | 变频节能、V/F控制 |
| 频繁启停 | 热容量、启动方式 |
| 粉尘潮湿环境 | IP等级和散热 |

## ③ 计算/选型示例

- 已知：  
  设备需要转矩约 **20N·m**，转速约 **1450rpm**。
- 公式：  
  **P = T × n ÷ 9550**
- 计算：  
  P = 20 × 1450 ÷ 9550 ≈ **3.04kW**
- 结论：  
  可初步选择 **3kW以上等级** 的交流电机，并按启动转矩、负载冲击和厂家规格留余量。

> [!tip] 选型口诀
> 先定电压，再定功率；速度看极数，力气看转矩；要调速配变频，环境差看防护。
