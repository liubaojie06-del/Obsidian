# 01 直流电机总览

> [!info] 核心结论
> 直流电机通过直流电源驱动，常用于调速、传动、移动设备、小型机构和低压控制场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 直流电机的特点是 **调速简单、启动转矩大、控制方便**。  
> 常见类型包括 **有刷直流电机、无刷直流电机、永磁直流电机、他励直流电机**。

> [!warning] 🟧 通用选型公式
> **直流电机选型 = 电压 + 功率 + 转速 + 转矩 + 工作制 + 调速方式 + 负载类型 + 安装环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **供电方式** | DC12V、24V、48V、110V、220V等 |
| **控制方式** | 调压、PWM、驱动器闭环控制 |
| **典型应用** | 小车、风机、泵、卷绕、升降、传送 |
| **优点** | 调速方便，启动转矩大 |
| **缺点** | 有刷型需要维护，易产生火花 |
| **重点参数** | 电压、电流、转速、转矩、功率、工作制 |

## ③ 计算/选型示例

- 已知：  
  小型输送机构需要低压供电，目标使用DC24V系统。
- 公式：  
  **电机需求 = DC24V + 所需转速 + 所需转矩 + 调速方式**
- 计算：  
  若需要简单调速，可选择DC24V直流电机配PWM调速器。
- 结论：  
  可选 **DC24V直流电机 + PWM调速器**，功率和转矩需按负载校核。

> [!tip] 记忆口诀
> 直流电机看电压，能不能带动看转矩。

# 02 底层原理

> [!info] 核心结论
> 直流电机的本质，是通电导体在磁场中受力，从而产生旋转运动。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 电枢绕组通入直流电后，在磁场中产生电磁力。  
> 多个线圈连续受力，形成持续旋转。

> [!warning] 🟧 能量转换逻辑
> **直流电能 → 电磁转矩 → 机械旋转 → 带动负载**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子磁场** | 提供磁通 |
| **转子/电枢** | 通电后产生转矩 |
| **换向器** | 有刷电机中改变电流方向 |
| **电刷** | 给转子绕组供电 |
| **轴承** | 支撑旋转 |
| **外壳** | 固定和散热 |

## ③ 计算/选型示例

- 已知：  
  直流电机通电后不转，但能听到轻微嗡声。
- 公式：  
  **电磁转矩 > 负载阻力 → 电机转动**
- 计算：  
  如果负载卡死或转矩不足，电机可能无法启动。
- 结论：  
  应检查负载是否卡滞、电源电压是否正常、电机是否损坏。

> [!tip] 记忆口诀
> 有电有磁才有力，力够负载才会转。

# 03 常见类型

> [!info] 核心结论
> 直流电机常见分为有刷直流电机和无刷直流电机，二者控制方式和维护要求不同。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> **有刷直流电机**：结构简单，成本低，但有电刷磨损。  
> **无刷直流电机**：寿命长、效率高，但需要电子驱动器换相。

> [!warning] 🟧 类型选择公式
> **低成本简单控制 → 有刷直流电机**  
> **长寿命高效率 → 无刷直流电机**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **有刷直流电机** | 控制简单，维护电刷 |
| **无刷直流电机** | 需要驱动器，寿命长 |
| **永磁直流电机** | 结构简单，常见小功率 |
| **他励直流电机** | 励磁独立，调速范围大 |
| **串励直流电机** | 启动转矩大 |
| **复励直流电机** | 兼顾转矩和稳定性 |

## ③ 计算/选型示例

- 已知：  
  设备要求长期连续运行，维护空间小。
- 公式：  
  **低维护需求 → 无刷直流电机优先**
- 计算：  
  有刷电机需要定期检查电刷和换向器。
- 结论：  
  可优先选择 **无刷直流电机BLDC**，驱动器需匹配电机参数。

> [!tip] 记忆口诀
> 有刷便宜好控制，无刷耐用效率高。

# 04 转速、转矩与功率

> [!info] 核心结论
> 直流电机选型不能只看功率，还要同时看转速和转矩是否满足负载。

## ① 分类/公式/规则

> [!warning] 🟧 功率转矩公式
> **P = T × n ÷ 9550**

> [!note] 参数说明
> **P**：功率，kW  
> **T**：转矩，N·m  
> **n**：转速，rpm

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定电压** | 电源匹配 |
| **额定电流** | 电源和保护选型 |
| **额定转速** | 正常运行速度 |
| **额定转矩** | 长期带载能力 |
| **峰值转矩** | 启动和冲击能力 |
| **额定功率** | 综合能力参考 |

## ③ 计算/选型示例

- 已知：  
  电机功率为 **200W**，额定转速为 **3000rpm**。
- 公式：  
  **T = 9550 × P ÷ n**
- 计算：  
  P = 0.2kW  
  T = 9550 × 0.2 ÷ 3000 ≈ **0.64N·m**
- 结论：  
  该电机额定转矩约为 **0.64N·m**，启动和过载能力需按厂家手册校核。

> [!tip] 记忆口诀
> 功率看总体，转矩看带不带得动，转速看跑不跑得快。

# 05 电压、电流与转速关系

> [!info] 核心结论
> 在负载和磁场基本不变时，直流电机电压越高，转速通常越高。

## ① 分类/公式/规则

> [!warning] 🟧 简化关系
> **转速 n ∝ 电枢电压 U**

> [!note] 🟦 工程理解
> 提高电压，电机转速上升；  
> 负载增大，电流增大；  
> 堵转时电流很大，容易烧毁电机或驱动器。

## ② 参考表

| 状态 | 电机表现 |
|---|---|
| **电压升高** | 转速升高 |
| **电压降低** | 转速降低 |
| **负载增大** | 电流增大 |
| **堵转** | 电流很大，风险高 |
| **空载** | 电流较小，转速较高 |
| **过载** | 发热、掉速、保护动作 |

## ③ 计算/选型示例

- 已知：  
  DC24V电机使用DC12V供电。
- 公式：  
  **电压降低 → 转速和输出能力下降**
- 计算：  
  12V只有额定电压的一半，电机转速和带载能力会明显下降。
- 结论：  
  电机应按额定电压使用，低压运行需确认负载是否允许。

> [!tip] 记忆口诀
> 电压管速度，负载管电流；堵转电流大，保护不能少。

# 06 调速方式

> [!info] 核心结论
> 直流电机常用调速方式有调压调速、PWM调速和闭环调速，其中PWM最常见。

## ① 分类/公式/规则

> [!note] 🟦 调速方式
> **调压调速**：改变电机端电压。  
> **PWM调速**：快速开关电源，通过占空比改变平均电压。  
> **闭环调速**：通过编码器或测速反馈稳定速度。

> [!warning] 🟧 PWM平均电压公式
> **U平均 ≈ U电源 × 占空比**

## ② 参考表

| 调速方式 | 特点 |
|---|---|
| **电阻调速** | 简单但效率低 |
| **调压调速** | 简单直观 |
| **PWM调速** | 效率高，应用广 |
| **闭环调速** | 速度稳定性好 |
| **励磁调速** | 特定直流电机使用 |
| **驱动器调速** | 功能完整，保护更好 |

## ③ 计算/选型示例

- 已知：  
  DC24V电机，PWM占空比为 **50%**。
- 公式：  
  **U平均 ≈ 24 × 50%**
- 计算：  
  U平均 ≈ **12V**
- 结论：  
  电机等效平均电压约为 **12V**，转速会低于满电压运行。

> [!tip] 记忆口诀
> PWM不是降电源，而是高速开关调平均。

# 07 有刷直流电机

> [!info] 核心结论
> 有刷直流电机控制简单、成本低，但电刷和换向器会磨损，适合成本敏感和维护可接受的场景。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 有刷直流电机通过电刷和换向器实现机械换向。  
> 正负极反接通常可改变旋转方向，但具体需按驱动器和电机手册校核。

> [!warning] 🟧 反转规则
> **改变电枢电压极性 → 电机反转**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 便宜、控制简单、启动转矩大 |
| **缺点** | 电刷磨损、火花、电磁干扰 |
| **维护点** | 电刷、换向器、轴承 |
| **调速方式** | 调压、PWM |
| **反转方式** | 改变电压极性 |
| **不适合** | 防爆、高洁净、免维护要求高场合 |

## ③ 计算/选型示例

- 已知：  
  一台DC24V有刷电机需要正反转控制。
- 公式：  
  **正反转 = 改变电机两端电压极性**
- 计算：  
  可使用H桥驱动器实现正转、反转、停止和制动。
- 结论：  
  应选择电流足够的 **H桥直流电机驱动器**。

> [!tip] 记忆口诀
> 有刷好控制，维护看电刷；要正反转，H桥来切换。

# 08 无刷直流电机 BLDC

> [!info] 核心结论
> 无刷直流电机用电子换向代替电刷换向，寿命长、效率高，但必须配套驱动器。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> BLDC电机通常由驱动器根据霍尔传感器或编码器位置，控制三相绕组依次通电，实现连续旋转。

> [!warning] 🟧 无刷控制逻辑
> **转子位置检测 → 驱动器电子换向 → 三相绕组通电 → 电机旋转**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 寿命长、效率高、噪声低 |
| **缺点** | 需要专用驱动器 |
| **反馈** | 霍尔、编码器、无感控制 |
| **控制模式** | 速度、转矩、位置 |
| **典型应用** | 风机、泵、机器人、AGV |
| **重点匹配** | 电压、电流、相线、霍尔线、驱动器参数 |

## ③ 计算/选型示例

- 已知：  
  无刷直流电机有三根相线和五根霍尔线。
- 公式：  
  **正常换向 = 相线顺序 + 霍尔顺序正确**
- 计算：  
  相线或霍尔线接错，可能导致抖动、不转、反转或报警。
- 结论：  
  接线必须按电机和驱动器手册匹配。

> [!tip] 记忆口诀
> 无刷没有刷，换向靠驱动；相线霍尔错，电机就乱动。

# 09 启动电流与堵转保护

> [!info] 核心结论
> 直流电机启动和堵转时电流很大，驱动器、电源和保护器件必须留足余量。

## ① 分类/公式/规则

> [!note] 🟦 启动特点
> 电机刚启动时反电动势很小，电流会明显高于额定运行电流。  
> 堵转时电机不转，电流可能接近最大堵转电流。

> [!warning] 🟧 堵转风险
> **堵转时间过长 → 电流大 → 发热快 → 电机/驱动器损坏**

## ② 参考表

| 状态 | 电流特点 |
|---|---|
| **空载运行** | 电流较小 |
| **额定负载** | 接近额定电流 |
| **启动瞬间** | 电流较大 |
| **过载运行** | 电流升高 |
| **堵转** | 电流最大，风险最高 |
| **频繁启停** | 温升明显增加 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流3A，启动电流可能达到额定的4倍。
- 公式：  
  **启动电流 ≈ 3 × 4**
- 计算：  
  启动电流 ≈ **12A**
- 结论：  
  驱动器和电源需要能承受短时 **12A** 冲击，保护参数需按厂家手册设置。

> [!tip] 记忆口诀
> 启动电流看瞬间，堵转电流最危险。

# 10 驱动器与H桥

> [!info] 核心结论
> 直流电机驱动器负责调速、正反转、制动和保护，H桥是常见直流电机正反转控制结构。

## ① 分类/公式/规则

> [!note] 🟦 H桥理解
> H桥通过四个开关器件改变电机两端电压方向，从而控制电机正转、反转、滑行或制动。

> [!warning] 🟧 驱动器选型公式
> **驱动器选型 = 电源电压 + 连续电流 + 峰值电流 + 控制方式 + 散热能力**

## ② 参考表

| 功能 | 说明 |
|---|---|
| **调速** | PWM调速 |
| **正反转** | 改变电压极性 |
| **刹车** | 短接制动或能耗制动 |
| **过流保护** | 保护电机和驱动器 |
| **过热保护** | 防止功率器件损坏 |
| **欠压保护** | 防止电源异常 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流5A，启动峰值电流20A。
- 公式：  
  **驱动器连续电流 ≥ 5A**  
  **驱动器峰值电流 ≥ 20A**
- 计算：  
  驱动器不能只满足5A，还要承受启动峰值。
- 结论：  
  应选择连续和峰值电流都满足要求的直流电机驱动器。

> [!tip] 记忆口诀
> 驱动器长期看连续，启动看峰值，正反转看H桥。

# 11 制动与反转

> [!info] 核心结论
> 直流电机可通过断电滑行、短接制动、反接制动、驱动器制动等方式停止，但必须注意电流和发热。

## ① 分类/公式/规则

> [!note] 🟦 停止方式
> **滑行停止**：断电后靠惯性慢慢停。  
> **短接制动**：电机端短接，利用反电动势制动。  
> **反接制动**：反向加电压，制动力强但冲击大。  
> **驱动器制动**：由控制器按策略处理。

> [!warning] 🟧 制动风险
> **快速制动 = 再生能量 + 大电流 + 发热风险**

## ② 参考表

| 制动方式 | 特点 |
|---|---|
| **自由停车** | 简单，停得慢 |
| **短接制动** | 停得快，发热增加 |
| **反接制动** | 冲击大，需谨慎 |
| **PWM制动** | 可控性较好 |
| **机械抱闸** | 断电保持 |
| **能耗制动** | 消耗再生能量 |

## ③ 计算/选型示例

- 已知：  
  电机带动大惯量转盘，需要快速停机。
- 公式：  
  **快速停机需求 = 电气制动 + 能量处理 + 机械安全**
- 计算：  
  大惯量减速会产生较大再生能量，驱动器可能过压或过流。
- 结论：  
  应选择支持制动功能的驱动器，并校核制动电阻或能量处理方案。

> [!tip] 记忆口诀
> 想停得快，就要处理能量；能量没地方去，驱动器就报警。

# 12 反馈与闭环控制

> [!info] 核心结论
> 普通直流电机开环调速会受负载影响，加入编码器或测速反馈后才能稳定速度和位置。

## ① 分类/公式/规则

> [!note] 🟦 闭环逻辑
> 开环控制只给电压或PWM，不知道电机实际转速。  
> 闭环控制通过编码器、测速电机或霍尔反馈，实时修正输出。

> [!warning] 🟧 闭环控制公式
> **速度偏差 = 目标速度 - 实际速度**

## ② 参考表

| 反馈方式 | 作用 |
|---|---|
| **编码器** | 位置和速度反馈 |
| **霍尔传感器** | 换向和粗略速度反馈 |
| **测速发电机** | 速度反馈 |
| **电流检测** | 转矩或过载判断 |
| **闭环驱动器** | 自动修正速度 |
| **PLC高速计数** | 读取编码器脉冲 |

## ③ 计算/选型示例

- 已知：  
  目标速度为1500rpm，实际反馈速度为1400rpm。
- 公式：  
  **速度偏差 = 1500 - 1400**
- 计算：  
  速度偏差 = **100rpm**
- 结论：  
  闭环驱动器会提高输出，使实际速度接近目标速度。

> [!tip] 记忆口诀
> 开环只管给电，闭环才知道转得准不准。

# 13 接线与保护

> [!info] 核心结论
> 直流电机接线要重点关注极性、电流、线径、保险、散热、反向电动势和电磁干扰。

## ① 分类/公式/规则

> [!note] 🟦 接线重点
> 低压大电流系统线缆压降明显，电机线应按电流选截面。  
> 控制线和动力线尽量分开，必要时加续流、滤波和屏蔽措施。

> [!warning] 🟧 线缆压降公式
> **压降 ≈ 电流 × 线缆电阻**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **电源极性** | DC+ / DC-不能接错 |
| **线缆截面** | 按电流和长度选 |
| **保险丝** | 防短路和过流 |
| **散热** | 电机和驱动器都要散热 |
| **续流保护** | 感性负载反向电压 |
| **EMI抑制** | 有刷电机会产生干扰 |

## ③ 计算/选型示例

- 已知：  
  电机电流10A，线缆电阻0.1Ω。
- 公式：  
  **压降 = I × R**
- 计算：  
  压降 = 10 × 0.1 = **1V**
- 结论：  
  若系统为DC12V，1V压降已经明显，应加粗线缆或缩短线路。

> [!tip] 记忆口诀
> 低压系统电流大，线细就会掉压发热。

# 14 常见故障排查

> [!info] 核心结论
> 直流电机故障排查按“电源 → 驱动器 → 接线 → 电机 → 负载 → 控制信号”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先量电压，再看电流；  
> 先空载测试，再带载运行；  
> 先查接线，再怀疑电机损坏。

> [!warning] 🟧 排错公式
> **直流电机异常 = 电源异常 + 驱动器故障 + 接线错误 + 电刷磨损 + 负载卡滞 + 参数不当**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **不转** | 无电源、驱动器未使能、接线断 |
| **转速慢** | 电压低、负载重、调速信号低 |
| **发热大** | 过载、堵转、电流过大 |
| **火花大** | 电刷磨损、换向器脏 |
| **噪声大** | 轴承磨损、机械偏心 |
| **方向反** | 电机极性反或驱动方向参数反 |
| **速度不稳** | 电源波动、负载变化、无闭环反馈 |

## ③ 计算/选型示例

- 已知：  
  电机空载能转，带载后不转并发热。
- 公式：  
  **带载不转 = 负载转矩 > 电机输出转矩**
- 计算：  
  电机空载正常，说明电源和电机基本可用；带载不转多半是转矩不足或负载卡滞。
- 结论：  
  应检查机械负载、电机功率、驱动器电流限制和减速机构。

> [!tip] 记忆口诀
> 空载能转带载不转，优先查负载和转矩。

# 15 直流电机快速选型表

> [!info] 核心结论
> 直流电机选型先看电压和负载，再看转速、转矩、功率、调速方式和驱动器电流。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看供电电压  
> ② 看负载类型  
> ③ 看目标转速  
> ④ 算连续转矩  
> ⑤ 校核启动转矩  
> ⑥ 选有刷或无刷  
> ⑦ 选调速方式  
> ⑧ 校核驱动器电流  
> ⑨ 看工作制和散热  
> ⑩ 看安装尺寸和环境

> [!warning] 🟧 最终选型公式
> **推荐型号 = DC电压 + 额定功率 + 额定转速 + 额定转矩 + 启动电流 + 驱动器匹配 + 工作制**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 低成本简单转动 | 有刷直流电机 |
| 长寿命低维护 | 无刷直流电机 |
| 简单调速 | PWM调速器 |
| 稳速运行 | 闭环直流驱动器 |
| 正反转控制 | H桥驱动 |
| 大启动转矩 | 峰值电流和堵转能力 |
| 电池供电 | 电流、压降、续航 |

## ③ 计算/选型示例

- 已知：  
  设备需要DC24V供电，目标转速1500rpm，负载需要0.8N·m连续转矩。
- 公式：  
  **P = T × n ÷ 9550**
- 计算：  
  P = 0.8 × 1500 ÷ 9550 ≈ **0.126kW = 126W**
- 结论：  
  电机额定功率应大于 **126W**，并按启动转矩、效率、温升和厂家手册留余量。

> [!tip] 选型口诀
> 电压先匹配，转矩要算清；调速看驱动，反转看H桥；长期运行看温升。
