# 01 直线模组总览

> [!info] 核心结论
> 直线模组是把电机旋转运动转换成直线运动的执行机构，核心看 **行程、负载、速度、精度、安装方向和传动方式**。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 直线模组通常由 **电机、传动机构、导轨滑块、滑台、限位传感器、安装底座** 组成。  
> 它的任务不是“转起来”，而是让负载 **直线移动、精准定位、稳定往复**。

> [!warning] 🟧 通用选型公式
> **直线模组选型 = 行程 + 负载 + 速度 + 精度 + 安装方向 + 传动方式 + 电机类型 + 使用环境**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **行程** | 决定模组长度 |
| **负载** | 决定导轨、滑块、电机能力 |
| **速度** | 决定传动方式和电机转速 |
| **精度** | 决定丝杆/同步带/直线电机 |
| **安装方向** | 水平、垂直、侧装差异大 |
| **环境** | 粉尘、油污、洁净、防护要求 |

## ③ 计算/选型示例

- 已知：  
  负载5kg，行程500mm，要求重复定位精度较高。
- 公式：  
  **方案选择 = 行程 + 负载 + 精度**
- 计算：  
  500mm属于中短行程，重复定位精度较高，可优先考虑丝杆模组。
- 结论：  
  可选择 **伺服电机 + 滚珠丝杆直线模组**，具体推力、速度和寿命需按厂家手册校核。

> [!tip] 记忆口诀
> 行程定长度，负载定规格，精度定结构，速度定传动。

# 02 底层原理

> [!info] 核心结论
> 直线模组的本质，是通过传动机构把电机的旋转运动变成滑台的直线运动。

## ① 分类/公式/规则

> [!note] 🟦 运动链路
> 控制器发指令，驱动器控制电机，电机带动丝杆、同步带或齿轮齿条，最后让滑块沿导轨直线移动。

> [!warning] 🟧 运动转换逻辑
> **控制指令 → 驱动器 → 电机旋转 → 传动机构 → 滑台直线移动 → 传感器/编码器反馈**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **电机** | 提供动力 |
| **驱动器** | 控制速度、位置、转矩 |
| **丝杆/同步带** | 完成运动转换 |
| **导轨滑块** | 承载和导向 |
| **滑台板** | 安装负载 |
| **限位/原点** | 定位和保护 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程为10mm，电机转5圈。
- 公式：  
  **直线位移 = 电机圈数 × 丝杆导程**
- 计算：  
  位移 = 5 × 10 = **50mm**
- 结论：  
  丝杆模组中，电机转5圈，滑台理论移动50mm。

> [!tip] 记忆口诀
> 电机负责转，传动负责变，导轨负责直，滑台负责带负载。

# 03 常见类型

> [!info] 核心结论
> 直线模组常见类型有 **丝杆模组、同步带模组、齿轮齿条模组、直线电机模组**，不同类型适合不同速度和精度要求。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 高精度中短行程看丝杆；  
> 长行程高速看同步带；  
> 超长行程大负载看齿轮齿条；  
> 高速高响应高精度看直线电机。

> [!warning] 🟧 选择规则
> **精度优先 → 丝杆 / 直线电机**  
> **速度优先 → 同步带 / 直线电机**  
> **长行程大负载 → 齿轮齿条**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **丝杆模组** | 高精度、中短行程 |
| **同步带模组** | 长行程、高速度 |
| **齿轮齿条模组** | 超长行程、大负载 |
| **直线电机模组** | 高速、高响应、高精度 |
| **气缸模组** | 简单两点动作 |
| **电缸** | 推力控制、压装、升降 |

## ③ 计算/选型示例

- 已知：  
  行程1500mm，负载轻，速度要求高，精度要求一般。
- 公式：  
  **长行程 + 高速度 → 同步带优先**
- 计算：  
  丝杆长行程高速运行容易受临界转速限制，同步带更适合。
- 结论：  
  可优先选择 **同步带直线模组**。

> [!tip] 记忆口诀
> 短精用丝杆，长快用同步带，重长用齿条，极限用直线电机。

# 04 丝杆直线模组

> [!info] 核心结论
> 丝杆模组适合高精度、中短行程和较大推力场景，是自动化定位最常见方案之一。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 丝杆导程越大，同样转速下直线速度越快；  
> 丝杆导程越小，同样电机转矩下推力越大。

> [!warning] 🟧 丝杆速度公式
> **直线速度 = 电机转速 × 丝杆导程 ÷ 60**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **导程** | 一圈走多远 |
| **直径** | 影响刚性和承载 |
| **精度等级** | 影响定位精度 |
| **预压** | 减小反向间隙 |
| **支撑方式** | 影响高速稳定性 |
| **润滑** | 影响寿命和噪声 |

## ③ 计算/选型示例

- 已知：  
  电机转速3000rpm，丝杆导程10mm。
- 公式：  
  **直线速度 = 3000 × 10 ÷ 60**
- 计算：  
  直线速度 = **500mm/s**
- 结论：  
  该丝杆模组理论速度为500mm/s，实际还需校核临界转速和负载。

> [!tip] 记忆口诀
> 导程大速度快，导程小推力大。

---

# 05 同步带直线模组

> [!info] 核心结论
> 同步带模组适合长行程、高速度、轻中负载搬运，但精度和刚性通常不如丝杆。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 同步带通过同步轮和齿形带啮合传动，结构轻、速度快、行程长，常用于搬运、上下料、喷涂、码垛等场景。

> [!warning] 🟧 同步带每圈位移公式
> **每圈位移 = 同步轮齿数 × 皮带节距**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **皮带节距** | 齿与齿之间距离 |
| **同步轮齿数** | 决定一圈移动距离 |
| **皮带宽度** | 影响承载能力 |
| **张紧力** | 影响精度和寿命 |
| **行程长度** | 同步带适合长行程 |
| **防尘结构** | 影响现场可靠性 |

## ③ 计算/选型示例

- 已知：  
  同步轮20齿，皮带节距5mm。
- 公式：  
  **每圈位移 = 20 × 5**
- 计算：  
  每圈位移 = **100mm**
- 结论：  
  电机每转一圈，滑台理论移动100mm。

> [!tip] 记忆口诀
> 同步带跑得快，长行程很划算；精度很高先看丝杆。

# 06 齿轮齿条模组

> [!info] 核心结论
> 齿轮齿条模组适合超长行程、大负载和现场拼接扩展，但精度和噪声需重点校核。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 齿轮齿条通过齿轮旋转带动齿条直线运动，行程理论上可以拼接很长，适合大型龙门、重载搬运和长距离移动。

> [!warning] 🟧 每圈位移公式
> **每圈位移 = π × 齿轮分度圆直径**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **模数** | 决定齿大小和承载 |
| **齿宽** | 影响承载能力 |
| **齿轮直径** | 决定一圈位移 |
| **背隙** | 影响反向精度 |
| **润滑** | 影响寿命和噪声 |
| **拼接精度** | 影响长行程平顺性 |

## ③ 计算/选型示例

- 已知：  
  设备行程5m，负载较重，速度中等。
- 公式：  
  **超长行程 + 大负载 → 齿轮齿条可评估**
- 计算：  
  丝杆不适合超长高速大负载，同步带刚性可能不足。
- 结论：  
  可选择 **齿轮齿条直线模组**，背隙和润滑需重点校核。

> [!tip] 记忆口诀
> 行程越长越看齿条，负载越大越看刚性。

# 07 直线电机模组

> [!info] 核心结论
> 直线电机模组没有丝杆、皮带等中间传动，适合高速、高响应、高精度应用。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 直线电机可以理解为“展开的旋转电机”，直接产生直线推力。  
> 它没有机械传动间隙，但对反馈尺、安装精度、散热和防护要求更高。

> [!warning] 🟧 选型公式
> **直线电机选型 = 连续推力 + 峰值推力 + 最高速度 + 加速度 + 行程 + 反馈精度**

## ② 参考表

| 项目 | 特点 |
|---|---|
| **优点** | 高速、高响应、无反向间隙 |
| **缺点** | 成本高、安装要求高 |
| **反馈** | 光栅尺、磁栅尺 |
| **适合场景** | 半导体、激光加工、精密平台 |
| **注意** | 磁吸力、散热、电缆拖链 |
| **防护** | 铁屑粉尘环境需谨慎 |

## ③ 计算/选型示例

- 已知：  
  负载20kg，加速度5m/s²。
- 公式：  
  **推力 F = m × a**
- 计算：  
  F = 20 × 5 = **100N**
- 结论：  
  仅加速所需推力为100N，实际还需叠加摩擦、工艺力和安全余量。

> [!tip] 记忆口诀
> 要极快、极准、无间隙，直线电机很有优势。

# 08 负载与安装方向

> [!info] 核心结论
> 同一个直线模组，水平安装、垂直安装、侧装时承载能力不同，不能直接套用同一参数。

## ① 分类/公式/规则

> [!note] 🟦 安装方向
> 水平安装主要看负载重量和摩擦；  
> 垂直安装要额外考虑重力、防下滑和抱闸；  
> 侧装要重点校核滑块力矩。

> [!warning] 🟧 负载校核公式
> **实际负载 = 重量 + 加速惯性力 + 偏心力矩 + 工艺外力**

## ② 参考表

| 安装方式 | 重点 |
|---|---|
| **水平安装** | 负载、速度、加速度 |
| **垂直安装** | 重力、抱闸、防坠落 |
| **侧向安装** | 侧向力矩、导轨刚性 |
| **倒装** | 滑块受力和防护 |
| **偏心负载** | 力矩校核 |
| **高速往复** | 惯性力和振动 |

## ③ 计算/选型示例

- 已知：  
  负载10kg，垂直升降。
- 公式：  
  **重力 F = m × g**
- 计算：  
  F = 10 × 9.8 = **98N**
- 结论：  
  垂直轴至少要克服约98N重力，还需考虑加速力、摩擦和安全余量，并配置抱闸或防坠措施。

> [!tip] 记忆口诀
> 水平看推力，垂直看重力，偏心看力矩。

# 09 精度与重复定位

> [!info] 核心结论
> 直线模组选型要区分 **分辨率、定位精度、重复定位精度**，三者不是同一个概念。

## ① 分类/公式/规则

> [!note] 🟦 三个概念
> 分辨率：系统能分得多细。  
> 定位精度：实际到目标点有多准。  
> 重复定位精度：多次回到同一点的一致性。

> [!warning] 🟧 实际误差公式
> **实际误差 = 控制误差 + 传动误差 + 反向间隙 + 刚性变形 + 温度影响**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **分辨率** | 理论最小控制单位 |
| **定位精度** | 到目标位置的误差 |
| **重复定位精度** | 多次到位一致性 |
| **反向间隙** | 换向误差来源 |
| **丝杆精度** | 影响实际定位 |
| **导轨刚性** | 影响受力变形 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，每圈指令脉冲10000个。
- 公式：  
  **脉冲当量 = 导程 ÷ 每圈脉冲数**
- 计算：  
  脉冲当量 = 10 ÷ 10000 = **0.001mm/脉冲**
- 结论：  
  0.001mm只是理论分辨率，不代表实际定位精度一定达到0.001mm。

> [!tip] 记忆口诀
> 分辨率是能分多细，精度是能到多准。

# 10 速度、加速度与节拍

> [!info] 核心结论
> 直线模组不仅要跑得快，还要在规定节拍内完成加速、匀速、减速和停稳。

## ① 分类/公式/规则

> [!note] 🟦 运动阶段
> 一次运动通常包括 **加速段、匀速段、减速段、等待/稳定段**。  
> 短行程时，可能还没达到最高速度就开始减速。

> [!warning] 🟧 基本运动公式
> **距离 s = 速度 v × 时间 t**  
> **加速度 a = 速度变化量 ÷ 时间**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **最高速度** | 决定最快移动能力 |
| **加速度** | 决定启动响应 |
| **减速度** | 决定停止距离 |
| **行程** | 影响能否达到最高速度 |
| **停稳时间** | 影响定位节拍 |
| **负载惯量** | 影响加减速能力 |

## ③ 计算/选型示例

- 已知：  
  行程300mm，目标平均速度600mm/s。
- 公式：  
  **时间 = 距离 ÷ 平均速度**
- 计算：  
  时间 = 300 ÷ 600 = **0.5s**
- 结论：  
  理论移动时间约0.5s，实际还需加入加减速和停稳时间。

> [!tip] 记忆口诀
> 只看最高速度不够，还要看加速、减速和停稳。

# 11 电机选择：伺服还是步进

> [!info] 核心结论
> 直线模组可配步进或伺服，低成本简单定位用步进，高速高精度和防丢步用伺服。

## ① 分类/公式/规则

> [!note] 🟦 选择主线
> 步进电机适合低速轻载、简单定位。  
> 伺服电机适合高速、高精度、高动态、多轴同步和不允许丢步的场景。

> [!warning] 🟧 选择公式
> **低速轻载 / 成本敏感 → 步进电机**  
> **高速高精度 / 负载变化大 → 伺服电机**

## ② 参考表

| 对比项 | 步进模组 | 伺服模组 |
|---|---|---|
| 成本 | 较低 | 较高 |
| 控制 | 简单 | 更强 |
| 高速性能 | 一般 | **更好** |
| 防丢步 | 弱 | **强** |
| 精度稳定性 | 一般 | 更好 |
| 调试难度 | 低 | 较高 |

## ③ 计算/选型示例

- 已知：  
  模组需要高速往复，位置丢失会撞机。
- 公式：  
  **高速 + 不允许丢步 → 伺服优先**
- 计算：  
  步进高速转矩下降且开环系统可能不知道丢步。
- 结论：  
  建议选择 **伺服电机直线模组**。

> [!tip] 记忆口诀
> 简单低速用步进，高速精准用伺服。

# 12 原点、限位与传感器

> [!info] 核心结论
> 直线模组必须设置原点和限位保护，防止位置丢失、超程撞机和机械损坏。

## ① 分类/公式/规则

> [!note] 🟦 常见传感器
> 直线模组常配 **原点开关、正限位、负限位、硬限位、防撞块**。  
> 高精度系统还可能使用光栅尺或磁栅尺反馈位置。

> [!warning] 🟧 安全位置逻辑
> **原点建立坐标，限位防止越界，硬限位防止撞坏**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **原点开关** | 建立坐标零点 |
| **正限位** | 防止正方向超程 |
| **负限位** | 防止负方向超程 |
| **硬限位** | 最后机械保护 |
| **光栅尺** | 高精度位置反馈 |
| **拖链线缆** | 保护运动线缆 |

## ③ 计算/选型示例

- 已知：  
  模组有效行程500mm，需要防止撞端。
- 公式：  
  **安全行程 < 机械极限行程**
- 计算：  
  软件行程、限位开关位置和机械端部之间应留安全距离。
- 结论：  
  应设置软限位、硬限位和机械缓冲，具体距离按速度和减速能力校核。

> [!tip] 记忆口诀
> 原点管坐标，限位保安全，硬限位兜底。

# 13 安装与调试

> [!info] 核心结论
> 直线模组安装质量直接影响精度、噪声、寿命和故障率，重点是平面度、同轴度、固定方式和润滑。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 模组底面不能强行拧变形；  
> 电机与丝杆/同步轮连接要同轴；  
> 导轨和丝杆要定期润滑；  
> 拖链不能拉扯编码器线和动力线。

> [!warning] 🟧 调试原则
> **先空载低速 → 再空载高速 → 再带载低速 → 最后带载高速**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **安装基准面** | 平面度和刚性 |
| **固定螺丝** | 受力均匀 |
| **联轴器** | 同轴度、锁紧 |
| **同步带张紧** | 不能过松或过紧 |
| **润滑** | 丝杆和导轨寿命关键 |
| **拖链** | 弯曲半径和线缆余量 |

## ③ 计算/选型示例

- 已知：  
  模组安装后运行噪声变大，拆下空跑正常。
- 公式：  
  **安装后异常 = 基准面不平 + 强制变形 + 同轴度差**
- 计算：  
  模组被安装面强行拉变形后，导轨和丝杆受力异常。
- 结论：  
  应检查安装基准面、螺丝锁紧顺序和联轴器同轴度。

> [!tip] 记忆口诀
> 模组选得好，还要装得正；装歪了，再贵也不准。

# 14 常见故障排查

> [!info] 核心结论
> 直线模组故障要同时检查电机驱动、传动机构、导轨滑块、传感器和安装状态。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看驱动报警，再看机械是否卡滞；  
> 先空载运行，再带载测试；  
> 先查传感器，再查控制程序；  
> 先低速验证，再高速运行。

> [!warning] 🟧 排错公式
> **模组异常 = 驱动报警 + 机械卡滞 + 传动松动 + 润滑不足 + 限位异常 + 参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **不动作** | 电源、驱动器、使能、限位触发 |
| **定位不准** | 脉冲当量、间隙、打滑、丢步 |
| **运行异响** | 导轨缺油、轴承坏、安装变形 |
| **抖动** | 增益不当、共振、负载惯量大 |
| **卡顿** | 导轨污染、丝杆损伤、异物 |
| **反向误差大** | 反向间隙、同步带松、联轴器松 |

## ③ 计算/选型示例

- 已知：  
  滑台正向定位准，反向定位偏差大。
- 公式：  
  **反向误差 = 反向间隙 + 弹性变形 + 传动松动**
- 计算：  
  丝杆预压不足、联轴器松动、同步带松弛都可能导致反向误差。
- 结论：  
  应检查反向间隙、联轴器锁紧、同步带张紧和丝杆预压。

> [!tip] 记忆口诀
> 单向准、反向偏，优先查间隙和松动。

# 15 直线模组快速选型表

> [!info] 核心结论
> 直线模组选型先定运动需求，再选传动方式、电机类型、导轨规格和传感器配置。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看运动方向：水平/垂直/侧装  
> ② 看有效行程  
> ③ 看负载重量  
> ④ 看速度和加速度  
> ⑤ 看定位精度和重复精度  
> ⑥ 选丝杆/同步带/齿条/直线电机  
> ⑦ 选电机：步进/伺服  
> ⑧ 校核导轨滑块和力矩  
> ⑨ 配原点、限位和拖链  
> ⑩ 校核环境、防护和维护方式

> [!warning] 🟧 最终选型公式
> **推荐模组 = 传动方式 + 有效行程 + 负载等级 + 速度等级 + 精度等级 + 电机方案 + 传感器配置**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 高精度短行程 | 丝杆模组 |
| 长行程高速 | 同步带模组 |
| 超长重载 | 齿轮齿条模组 |
| 高速高响应 | 直线电机模组 |
| 低成本简单定位 | 步进模组 |
| 防丢步高节拍 | 伺服模组 |
| 垂直升降 | 抱闸、防坠、重力校核 |

## ③ 计算/选型示例

- 已知：  
  行程800mm，负载8kg，速度800mm/s，重复定位精度要求一般。
- 公式：  
  **长行程 + 高速度 + 精度一般 → 同步带模组优先**
- 计算：  
  800mm行程下，同步带速度优势明显；若精度要求很高，则再评估丝杆或直线电机。
- 结论：  
  可优先选择 **同步带直线模组 + 伺服电机**，并校核加速度、负载力矩和同步带张紧。

> [!tip] 选型口诀
> 先定行程负载，再定速度精度；短精丝杆，长快同步带；垂直要抱闸，高速看伺服。
