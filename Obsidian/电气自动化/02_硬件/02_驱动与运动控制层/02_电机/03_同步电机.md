# 01 同步电机总览

> [!info] 核心结论
> 同步电机运行时，转子转速与定子旋转磁场转速保持一致，适合 **高效率、高精度速度控制、高功率因数、伺服控制** 等场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> “同步”就是 **转子速度 = 旋转磁场速度**。  
> 同步电机不像异步电机那样靠转差产生转矩，正常稳定运行时没有转差。

> [!warning] 🟧 通用选型公式
> **同步电机选型 = 电压等级 + 功率 + 转速/极数 + 转矩 + 控制方式 + 反馈方式 + 冷却方式 + 应用场景**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **常见类型** | 永磁同步、励磁同步、磁阻同步 |
| **核心特点** | 转速稳定、效率高、控制性能好 |
| **典型应用** | 伺服系统、主轴、电动车、压缩机、泵、风机 |
| **控制方式** | 变频器、伺服驱动器、专用同步电机驱动 |
| **重点参数** | 电压、电流、转矩、极数、转速、编码器 |
| **校核重点** | 驱动器匹配、温升、退磁风险、负载特性 |

## ③ 计算/选型示例

- 已知：  
  设备需要高效率、速度稳定、低速控制性能好。
- 公式：  
  **高效率 + 精准速度控制 → 同步电机优先评估**
- 计算：  
  同步电机转速与频率严格对应，配合驱动器可实现高性能控制。
- 结论：  
  可选择 **永磁同步电机或同步伺服电机**，具体型号需按厂家手册校核。

> [!tip] 记忆口诀
> 异步靠转差，同步跟磁场。

# 02 底层原理

> [!info] 核心结论
> 同步电机通过定子产生旋转磁场，转子磁场与定子磁场锁定同步旋转。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 三相交流电进入定子绕组后产生旋转磁场。  
> 转子磁场可以来自 **永磁体、励磁绕组或磁阻效应**。  
> 当转子磁场与定子旋转磁场同步后，电机稳定输出转矩。

> [!warning] 🟧 能量转换逻辑
> **三相交流电 → 定子旋转磁场 → 转子磁场锁定 → 同步旋转 → 输出机械转矩**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **定子绕组** | 产生旋转磁场 |
| **转子磁场** | 与定子磁场同步 |
| **永磁体** | 永磁同步电机提供磁场 |
| **励磁绕组** | 励磁同步电机提供磁场 |
| **编码器/位置传感器** | 高性能控制时反馈位置 |
| **驱动器** | 控制电流、频率、相位 |

## ③ 计算/选型示例

- 已知：  
  同步电机需要稳定运行在1500rpm。
- 公式：  
  **同步转速由频率和极数决定**
- 计算：  
  若为4极电机，50Hz时同步转速就是1500rpm。
- 结论：  
  同步电机运行速度与输入频率严格对应。

> [!tip] 记忆口诀
> 定子造旋转磁场，转子磁场跟着跑。

# 03 同步转速公式

> [!info] 核心结论
> 同步电机的转速由电源频率和电机极数决定。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **nₛ = 120 × f ÷ P**

> [!note] 参数说明
> **nₛ**：同步转速，rpm  
> **f**：电源频率，Hz  
> **P**：电机极数

## ② 参考表

| 极数 | 50Hz同步转速 |
|---|---:|
| **2极** | 3000rpm |
| **4极** | 1500rpm |
| **6极** | 1000rpm |
| **8极** | 750rpm |
| **10极** | 600rpm |
| **12极** | 500rpm |

## ③ 计算/选型示例

- 已知：  
  一台6极同步电机，输入频率为50Hz。
- 公式：  
  **nₛ = 120 × 50 ÷ 6**
- 计算：  
  nₛ = **1000rpm**
- 结论：  
  该同步电机在50Hz下同步转速为 **1000rpm**。

> [!tip] 记忆口诀
> 频率越高转速越高，极数越多转速越低。

# 04 永磁同步电机 PMSM

> [!info] 核心结论
> 永磁同步电机用永磁体产生转子磁场，效率高、功率密度高，是伺服、电动车、机器人中常见的电机类型。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 永磁同步电机不需要转子励磁电流，转子损耗低，效率高。  
> 但永磁体存在 **退磁风险**，过热、过流或强反向磁场都可能影响性能。

> [!warning] 🟧 选型公式
> **PMSM选型 = 转矩 + 转速 + 电压 + 驱动器 + 编码器 + 温升 + 退磁裕量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 高效率、高功率密度、响应快 |
| **缺点** | 成本较高，需防退磁 |
| **控制方式** | 伺服驱动、矢量控制、FOC |
| **典型应用** | 伺服、电动车、机器人、主轴 |
| **反馈方式** | 编码器、旋变、霍尔等 |
| **注意事项** | 温度、电流、磁钢安全裕量 |

## ③ 计算/选型示例

- 已知：  
  设备要求体积小、效率高、动态响应快。
- 公式：  
  **高效率 + 高响应 + 小体积 → PMSM优先**
- 计算：  
  永磁同步电机转子无励磁铜耗，功率密度高。
- 结论：  
  可选择 **永磁同步电机**，驱动器和反馈接口必须匹配。

> [!tip] 记忆口诀
> 永磁效率高，怕热怕退磁。

# 05 励磁同步电机

> [!info] 核心结论
> 励磁同步电机通过转子励磁绕组建立磁场，可调励磁电流，适合大功率和功率因数调节场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 励磁同步电机转子不是永磁体，而是通过励磁绕组通入直流电建立磁场。  
> 改变励磁电流，可以影响功率因数和无功功率。

> [!warning] 🟧 励磁调节逻辑
> **励磁不足 → 可能吸收无功**  
> **励磁适中 → 功率因数接近1**  
> **过励磁 → 可能发出无功**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **转子磁场** | 由直流励磁产生 |
| **优点** | 可调功率因数，适合大功率 |
| **缺点** | 励磁系统复杂 |
| **典型应用** | 大型压缩机、泵、轧机、发电机 |
| **维护点** | 励磁系统、滑环或无刷励磁 |
| **校核重点** | 励磁电流、功率因数、保护系统 |

## ③ 计算/选型示例

- 已知：  
  工厂希望大型电机既驱动负载，又改善功率因数。
- 公式：  
  **功率因数需求 + 大功率 → 励磁同步电机可评估**
- 计算：  
  通过调整励磁电流，可影响无功功率。
- 结论：  
  可评估 **励磁同步电机**，具体励磁和保护方案需按厂家手册校核。

> [!tip] 记忆口诀
> 永磁靠磁钢，励磁靠电流；想调功率因数，看励磁同步。

# 06 磁阻同步电机

> [!info] 核心结论
> 磁阻同步电机利用转子磁阻差产生转矩，结构简单、无永磁体，适合高效节能驱动场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 磁阻同步电机转子没有永磁体，也没有励磁绕组。  
> 它利用磁通总是倾向于走“磁阻小”的路径，从而产生同步转矩。

> [!warning] 🟧 特点公式
> **磁阻差越明显 → 可利用磁阻转矩越明显**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **转子结构** | 无永磁体、无绕组 |
| **优点** | 结构简单、无退磁风险 |
| **缺点** | 控制依赖驱动器 |
| **典型应用** | 风机、水泵、压缩机节能 |
| **控制方式** | 变频器矢量控制 |
| **校核重点** | 驱动器匹配、效率、噪声、转矩脉动 |

## ③ 计算/选型示例

- 已知：  
  风机水泵项目希望提高效率，同时避免永磁体退磁问题。
- 公式：  
  **高效节能 + 无磁钢需求 → 磁阻同步电机可评估**
- 计算：  
  磁阻同步电机无永磁体，材料成本和退磁风险较低。
- 结论：  
  可评估 **磁阻同步电机 + 匹配变频器** 方案。

> [!tip] 记忆口诀
> 不靠磁钢不靠励磁，靠磁阻差来出力。

# 07 同步电机 vs 异步电机

> [!info] 核心结论
> 异步电机靠转差产生转矩，同步电机转速与磁场同步，效率和控制性能通常更好。

## ① 分类/公式/规则

> [!note] 🟦 快速区别
> 异步电机：结构简单、成本低、应用广。  
> 同步电机：效率高、速度稳定、控制性能好，但驱动和控制更复杂。

> [!warning] 🟧 选择公式
> **普通工业动力 → 异步电机**  
> **高效率 / 高精度 / 高响应 → 同步电机**

## ② 参考表

| 对比项 | 异步电机 | 同步电机 |
|---|---|---|
| 转速关系 | 略低于同步转速 | **等于同步转速** |
| 转差 | 有 | **正常无转差** |
| 成本 | 较低 | 较高 |
| 效率 | 较高 | **通常更高** |
| 控制难度 | 较低 | 较高 |
| 常见应用 | 风机、泵、输送 | 伺服、主轴、高效驱动 |

## ③ 计算/选型示例

- 已知：  
  设备要求速度非常稳定，并且低速响应好。
- 公式：  
  **速度稳定 + 高响应 → 同步电机优先**
- 计算：  
  同步电机配合驱动器，速度跟随性能和动态响应更好。
- 结论：  
  可优先考虑 **永磁同步电机或同步伺服电机**。

> [!tip] 记忆口诀
> 异步便宜耐用，同步精准高效。

# 08 启动方式

> [!info] 核心结论
> 同步电机不能像普通异步电机那样简单依靠转差启动，通常需要变频器、辅助启动结构或专用驱动器。

## ① 分类/公式/规则

> [!note] 🟦 启动特点
> 同步电机要想稳定运行，转子磁场必须与定子旋转磁场同步。  
> 如果启动方式不合理，可能无法拉入同步，出现失步或振荡。

> [!warning] 🟧 启动方案公式
> **同步电机启动 = 驱动器控制 + 转子位置识别 + 加速到同步 + 稳定锁定**

## ② 参考表

| 启动方式 | 说明 |
|---|---|
| **变频启动** | 常见，平滑升频 |
| **伺服驱动启动** | 用于PMSM伺服系统 |
| **异步启动绕组** | 部分同步电机具备 |
| **励磁控制启动** | 大型励磁同步电机常见 |
| **直接启动** | 需特殊结构，不能随意使用 |
| **闭环启动** | 高性能场景常用 |

## ③ 计算/选型示例

- 已知：  
  永磁同步电机需要从静止状态平稳启动。
- 公式：  
  **PMSM启动 = 驱动器识别位置 + 控制电流相位**
- 计算：  
  若驱动器不知道转子位置，启动可能抖动或失败。
- 结论：  
  永磁同步电机通常需要匹配伺服驱动器或专用变频器。

> [!tip] 记忆口诀
> 同步电机要同步，启动不能太粗暴。

# 09 变频与矢量控制

> [!info] 核心结论
> 同步电机常用变频器或伺服驱动器控制，核心是控制电流大小、频率和相位。

## ① 分类/公式/规则

> [!note] 🟦 控制重点
> 同步电机高性能控制常用 **矢量控制FOC**。  
> 它把电机电流分解成励磁方向和转矩方向，从而实现精准转矩控制。

> [!warning] 🟧 控制逻辑
> **位置反馈 → 坐标变换 → 电流控制 → 转矩输出 → 速度/位置闭环**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **V/F控制** | 简单同步电机调速，需按型号校核 |
| **矢量控制** | 高性能速度/转矩控制 |
| **伺服控制** | 高精度位置控制 |
| **无传感器控制** | 成本较低，低速性能需校核 |
| **编码器闭环** | 低速、高精度、高响应 |
| **FOC控制** | PMSM常见高性能控制 |

## ③ 计算/选型示例

- 已知：  
  设备要求低速平稳运行并快速响应负载变化。
- 公式：  
  **低速高性能 → 闭环矢量控制优先**
- 计算：  
  闭环控制能利用编码器反馈实时修正速度和转矩。
- 结论：  
  应选择支持 **编码器闭环矢量控制** 的驱动器。

> [!tip] 记忆口诀
> 同步控制看相位，低速高性能看闭环。

# 10 功率、转矩与转速

> [!info] 核心结论
> 同步电机选型同样要同时校核功率、转矩和转速，不能只看kW。

## ① 分类/公式/规则

> [!warning] 🟧 功率转矩公式
> **P = T × n ÷ 9550**

> [!note] 参数说明
> **P**：功率，kW  
> **T**：转矩，N·m  
> **n**：转速，rpm

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定功率** | 长期输出能力 |
| **额定转矩** | 连续带载能力 |
| **峰值转矩** | 加速、冲击能力 |
| **额定转速** | 常用运行速度 |
| **最高转速** | 最高允许速度 |
| **过载能力** | 短时负载能力 |

## ③ 计算/选型示例

- 已知：  
  同步电机功率为 **1.5kW**，额定转速为 **3000rpm**。
- 公式：  
  **T = 9550 × P ÷ n**
- 计算：  
  T = 9550 × 1.5 ÷ 3000 ≈ **4.78N·m**
- 结论：  
  该同步电机额定转矩约为 **4.78N·m**。

> [!tip] 记忆口诀
> 功率看等级，转矩看力气，转速看节拍。

# 11 功率因数与效率

> [!info] 核心结论
> 同步电机效率通常较高，励磁同步电机还可以通过励磁调节功率因数。

## ① 分类/公式/规则

> [!note] 🟦 工程理解
> 永磁同步电机因转子无励磁铜耗，效率较高。  
> 励磁同步电机可通过调节励磁，改善功率因数甚至提供无功补偿。

> [!warning] 🟧 效率关注公式
> **输入功率 = 输出功率 ÷ 效率**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **效率η** | 输出功率与输入功率之比 |
| **功率因数cosφ** | 电网侧有功利用程度 |
| **永磁同步** | 高效率、高功率密度 |
| **励磁同步** | 可调功率因数 |
| **轻载效率** | 需按曲线校核 |
| **节能评估** | 需结合运行工况 |

## ③ 计算/选型示例

- 已知：  
  电机输出功率10kW，效率为95%。
- 公式：  
  **输入功率 = 10 ÷ 0.95**
- 计算：  
  输入功率 ≈ **10.53kW**
- 结论：  
  电机输入功率约为 **10.53kW**，损耗约为0.53kW。

> [!tip] 记忆口诀
> 效率高少发热，功率因数好少吃无功。

# 12 失步与拉入同步

> [!info] 核心结论
> 同步电机如果负载过大、扰动过强或控制不当，可能失去同步，称为失步。

## ① 分类/公式/规则

> [!note] 🟦 失步理解
> 同步电机稳定运行时，转子磁场跟随定子旋转磁场。  
> 如果负载突然过大，转子跟不上旋转磁场，就可能失步。

> [!warning] 🟧 失步风险公式
> **失步风险 = 负载冲击大 + 转矩裕量不足 + 控制参数不当**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **启动失败** | 未成功拉入同步 |
| **运行振荡** | 控制参数或负载波动 |
| **失步报警** | 负载超过能力 |
| **电流异常** | 控制失稳或负载冲击 |
| **速度波动** | 负载突变或反馈异常 |
| **转矩不足** | 电机或驱动器容量不够 |

## ③ 计算/选型示例

- 已知：  
  电机低速运行正常，但加速带载时报警。
- 公式：  
  **加速报警 = 加速度大 + 负载转矩大 + 峰值转矩不足**
- 计算：  
  加速过程需要额外转矩，若峰值转矩不足，容易失步或过载。
- 结论：  
  应降低加速度、增大转矩裕量，或选择更大规格电机/驱动器。

> [!tip] 记忆口诀
> 同步最怕跟不上，转矩裕量要留够。

# 13 温升与退磁风险

> [!info] 核心结论
> 永磁同步电机要特别关注温升和退磁风险，过热或过流可能导致性能下降。

## ① 分类/公式/规则

> [!note] 🟦 退磁理解
> 永磁体在高温、强反向磁场或过大电流冲击下，可能出现不可逆退磁。  
> 一旦退磁，电机转矩能力和效率可能下降。

> [!warning] 🟧 安全运行公式
> **防退磁 = 温度不过限 + 电流不过限 + 驱动参数正确 + 冷却良好**

## ② 参考表

| 风险 | 说明 |
|---|---|
| **过温** | 永磁体性能下降 |
| **过流** | 反向磁场风险 |
| **堵转** | 电流大、温升快 |
| **冷却不足** | 长期温升高 |
| **频繁过载** | 热积累明显 |
| **参数错误** | 可能导致电流异常 |

## ③ 计算/选型示例

- 已知：  
  永磁同步电机长时间过载运行。
- 公式：  
  **过载运行 = 电流大 + 损耗大 + 温升高**
- 计算：  
  温升过高可能影响永磁体性能和绕组寿命。
- 结论：  
  应检查负载、散热、驱动器限流和电机温度保护。

> [!tip] 记忆口诀
> 永磁电机效率高，温度电流别超标。

# 14 编码器与位置反馈

> [!info] 核心结论
> 高性能同步电机控制通常需要编码器、旋变或霍尔传感器反馈转子位置。

## ① 分类/公式/规则

> [!note] 🟦 反馈作用
> 同步电机控制不仅要知道转速，还要知道转子磁极位置。  
> 位置反馈越准确，电流相位控制越准确，转矩输出越稳定。

> [!warning] 🟧 反馈匹配公式
> **反馈可用 = 电机反馈类型 + 驱动器接口 + 参数设置一致**

## ② 参考表

| 反馈类型 | 特点 |
|---|---|
| **增量编码器** | 常见，需回零或初始识别 |
| **绝对值编码器** | 上电可读位置 |
| **旋转变压器** | 抗干扰强，恶劣环境常见 |
| **霍尔传感器** | 简单位置检测 |
| **无传感器控制** | 成本低，低速性能需校核 |
| **编码器线缆** | 屏蔽和抗干扰很重要 |

## ③ 计算/选型示例

- 已知：  
  驱动器支持增量编码器，但电机为绝对值编码器接口。
- 公式：  
  **接口不匹配 → 无法正常反馈**
- 计算：  
  驱动器无法正确读取电机位置，可能报警或无法运行。
- 结论：  
  必须选择编码器接口匹配的驱动器。

> [!tip] 记忆口诀
> 同步控制看位置，反馈接口别接错。

# 15 常见故障排查

> [!info] 核心结论
> 同步电机故障排查要同时检查驱动器、电机反馈、负载、参数、温度和电源。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 上电报警先查编码器和参数；  
> 启动失败先查反馈相序和驱动器匹配；  
> 运行报警先查负载、温升和电流。

> [!warning] 🟧 排错公式
> **同步电机异常 = 驱动器参数错误 + 反馈异常 + 负载过大 + 温升过高 + 电源异常 + 接线错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无法启动** | 转子位置识别失败、反馈异常 |
| **启动抖动** | 相序/反馈方向错误 |
| **过流报警** | 负载过重、参数错误 |
| **过热报警** | 散热差、长期过载 |
| **失步报警** | 转矩不足、负载冲击 |
| **速度波动** | 控制参数不稳、反馈干扰 |
| **编码器报警** | 线缆、接口、编码器故障 |

## ③ 计算/选型示例

- 已知：  
  电机启动时抖动，无法平稳旋转。
- 公式：  
  **启动抖动 = 相序错误 + 反馈方向错误 + 参数不匹配**
- 计算：  
  同步电机需要电流相位与转子位置匹配，反馈错误会导致转矩方向异常。
- 结论：  
  应检查电机相序、编码器方向、驱动器电机型号参数和自动识别结果。

> [!tip] 记忆口诀
> 同步电机启动抖，先查相序和反馈。

# 16 同步电机快速选型表

> [!info] 核心结论
> 同步电机选型先看应用场景，再确定类型、功率、转速、转矩、驱动器和反馈方式。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看应用场景  
> ② 选同步电机类型  
> ③ 看电压等级  
> ④ 算功率和转矩  
> ⑤ 看额定转速和最高转速  
> ⑥ 看启动方式  
> ⑦ 看控制精度  
> ⑧ 选反馈方式  
> ⑨ 匹配驱动器  
> ⑩ 校核温升、冷却和保护

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电机类型 + 电压等级 + 功率 + 转矩 + 转速 + 反馈方式 + 驱动器 + 冷却防护**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 伺服定位 | 永磁同步电机 + 编码器 |
| 高效率节能 | 永磁同步 / 磁阻同步 |
| 大功率功率因数调节 | 励磁同步电机 |
| 高动态响应 | PMSM + 伺服驱动 |
| 风机水泵节能 | 同步电机 + 变频器 |
| 恶劣环境 | 旋变反馈、冷却、防护 |
| 不允许失步 | 转矩裕量、闭环控制 |

## ③ 计算/选型示例

- 已知：  
  设备需要转矩 **10N·m**，转速 **1500rpm**，要求高效率和精准速度控制。
- 公式：  
  **P = T × n ÷ 9550**
- 计算：  
  P = 10 × 1500 ÷ 9550 ≈ **1.57kW**
- 结论：  
  可初步选择 **2kW等级同步电机**，并按峰值转矩、驱动器、电压等级和冷却条件校核。

> [!tip] 选型口诀
> 要效率看同步，要精准看反馈；永磁看退磁，励磁看无功，磁阻看驱动匹配。
