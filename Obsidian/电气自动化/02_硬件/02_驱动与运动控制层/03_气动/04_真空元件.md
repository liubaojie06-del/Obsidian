# 01 真空元件总览

> [!info] 核心结论
> 真空元件用于产生、保持、检测和释放负压，常用于 **吸附搬运、取放料、包装、板材搬运、玻璃搬运、薄片分离** 等场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 真空系统不是只有吸盘。  
> 一套完整真空回路通常包括 **真空发生器/真空泵、真空吸盘、真空过滤器、真空开关、破真空阀、单向阀、管路接头**。

> [!warning] 🟧 通用选型公式
> **真空元件选型 = 工件重量 + 表面状态 + 吸附面积 + 真空度 + 泄漏量 + 节拍 + 安全系数**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **真空发生器** | 用压缩空气产生真空 |
| **真空泵** | 电机驱动产生真空 |
| **真空吸盘** | 与工件接触并吸附 |
| **真空过滤器** | 防止粉尘进入真空元件 |
| **真空开关** | 检测吸附是否到位 |
| **破真空阀** | 快速释放工件 |
| **单向阀** | 防止真空泄漏回流 |
| **真空表** | 显示负压大小 |

## ③ 计算/选型示例

- 已知：  
  需要搬运一块平整塑料板。
- 公式：  
  **真空搬运 = 吸盘面积 + 真空度 + 安全系数**
- 计算：  
  工件表面平整，适合用真空吸盘吸附；若表面多孔或漏气严重，需要增大流量或换方案。
- 结论：  
  可选择 **真空吸盘 + 真空发生器 + 真空开关**，吸附力需按实际重量校核。

> [!tip] 记忆口诀
> 产生真空靠发生器，抓住工件靠吸盘，确认到位靠真空开关。

# 02 底层原理

> [!info] 核心结论
> 真空吸附的本质不是“吸盘主动吸住”，而是吸盘内部压力低于外界，大气压把工件压在吸盘上。

## ① 分类/公式/规则

> [!note] 🟦 原理理解
> 当吸盘内部形成负压后，外部大气压大于吸盘内部压力。  
> 这个压力差作用在吸盘有效面积上，形成吸附力。

> [!warning] 🟧 吸附力公式
> **F = ΔP × A × η**

> [!note] 参数说明
> **F**：实际吸附力，N  
> **ΔP**：压力差，Pa  
> **A**：有效吸附面积，m²  
> **η**：修正系数，需按泄漏、表面、姿态和厂家手册校核

## ② 参考表

| 参数 | 影响 |
|---|---|
| **真空度** | 越高，压力差越大 |
| **吸盘面积** | 越大，吸附力越大 |
| **表面泄漏** | 泄漏越大，吸附越差 |
| **工件重量** | 越重，需要吸力越大 |
| **加速度** | 搬运越快，惯性力越大 |
| **安全系数** | 防止掉料 |

## ③ 计算/选型示例

- 已知：  
  真空压差为60kPa，吸盘有效面积0.001m²。
- 公式：  
  **F = ΔP × A**
- 计算：  
  F = 60000 × 0.001 = **60N**
- 结论：  
  理论吸附力约 **60N**，实际需按泄漏、加速度和安全系数降额。

> [!tip] 记忆口诀
> 真空不是往上拉，是大气压往下压。

# 03 真空发生器

> [!info] 核心结论
> 真空发生器利用压缩空气高速流动形成负压，适合自动化设备中局部、快速、间歇吸附。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 真空发生器又叫真空喷射器、真空发生阀。  
> 它不需要电机，只要有压缩空气，就能通过文丘里效应产生真空。

> [!warning] 🟧 选型公式
> **真空发生器选型 = 最大真空度 + 吸入流量 + 耗气量 + 响应时间 + 工件泄漏量**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **最大真空度** | 能达到的最高负压 |
| **吸入流量** | 抽气速度，影响建立真空时间 |
| **耗气量** | 压缩空气消耗 |
| **供气压力** | 通常需稳定压缩空气 |
| **破真空功能** | 快速释放工件 |
| **集成开关** | 可带真空检测 |

## ③ 计算/选型示例

- 已知：  
  机械手每次吸取小工件，节拍较快。
- 公式：  
  **快速局部吸附 → 真空发生器优先**
- 计算：  
  真空发生器靠近吸盘安装，管路短，响应快。
- 结论：  
  可选择 **小型真空发生器 + 真空开关 + 破真空功能**。

> [!tip] 记忆口诀
> 小工件快节拍，用发生器最方便。

# 04 真空泵

> [!info] 核心结论
> 真空泵通过电机驱动持续抽气，适合大流量、多吸盘、持续吸附或泄漏较大的场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 真空泵适合集中供真空，能够给多个吸盘或多个工位供负压。  
> 与真空发生器相比，真空泵不消耗压缩空气，但设备体积和维护要求更高。

> [!warning] 🟧 选择规则
> **多吸盘 / 大流量 / 持续吸附 / 泄漏大 → 真空泵优先**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **抽速** | 单位时间抽气能力 |
| **极限真空** | 能达到的最低压力 |
| **泵类型** | 旋片、干式、侧流风机等 |
| **储气罐** | 稳定真空、缓冲波动 |
| **维护** | 滤芯、油、叶片等 |
| **噪声** | 需按现场要求校核 |

## ③ 计算/选型示例

- 已知：  
  一套设备有20个吸盘同时工作。
- 公式：  
  **多点吸附 = 总泄漏量 + 总抽气量**
- 计算：  
  若每个吸盘都用独立真空发生器，耗气量可能很大。
- 结论：  
  可评估 **真空泵集中供真空**，并配置真空罐和真空开关。

> [!tip] 记忆口诀
> 单点快吸用发生器，多点持续看真空泵。

# 05 真空吸盘

> [!info] 核心结论
> 真空吸盘直接接触工件，选型重点是 **直径、形状、材质、工件表面、吸附方向和安全系数**。

## ① 分类/公式/规则

> [!note] 🟦 吸盘作用
> 吸盘既要密封，也要适应工件表面。  
> 表面越平整越容易吸，表面粗糙、多孔、弯曲或有纹理时容易漏气。

> [!warning] 🟧 吸盘选型公式
> **吸盘选型 = 工件表面 + 工件重量 + 吸附方向 + 吸盘直径 + 材质 + 数量**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **平形吸盘** | 平整表面、定位稳定 |
| **波纹吸盘** | 曲面、柔性补偿 |
| **深形吸盘** | 弧面或不规则工件 |
| **椭圆吸盘** | 长条工件 |
| **海绵吸盘** | 粗糙、多孔表面 |
| **防静电吸盘** | 电子元件场景 |

## ③ 计算/选型示例

- 已知：  
  工件表面平整，重量1kg，水平搬运。
- 公式：  
  **所需吸附力 ≥ 重力 × 安全系数**
- 计算：  
  重力约9.8N，若取3倍安全系数：所需吸附力 ≥ **29.4N**
- 结论：  
  吸盘总实际吸附力应大于 **29.4N**，并按加速度和泄漏降额。

> [!tip] 记忆口诀
> 平面用平吸，曲面用波纹，粗糙漏气用海绵。

# 06 吸附力计算

> [!info] 核心结论
> 真空吸附力由压力差和有效面积决定，安全系数不能省，尤其是垂直搬运和高速搬运。

## ① 分类/公式/规则

> [!warning] 🟧 核心公式
> **F = ΔP × A**

> [!warning] 🟧 安全选型规则
> **总吸附力 ≥ 工件重量 × 安全系数**

> [!note] 🟦 安全系数参考
> 水平搬运通常可取较低安全系数；  
> 垂直搬运、高速运动、表面漏气、工件贵重时，应提高安全系数。  
> 具体需按实际规范/厂家手册校核。

## ② 参考表

| 场景 | 风险 |
|---|---|
| **水平搬运** | 工件主要受重力 |
| **垂直搬运** | 更容易滑落 |
| **高速搬运** | 惯性力增加 |
| **表面粗糙** | 泄漏大 |
| **多孔材料** | 难以保持真空 |
| **油污表面** | 可能打滑 |

## ③ 计算/选型示例

- 已知：  
  工件重量2kg，水平搬运，取安全系数3。
- 公式：  
  **所需吸附力 ≥ m × g × 安全系数**
- 计算：  
  所需吸附力 ≥ 2 × 9.8 × 3 = **58.8N**
- 结论：  
  吸盘总实际吸附力应不小于 **58.8N**，并校核真空度和吸盘面积。

> [!tip] 记忆口诀
> 先算重量，再乘系数；吸力不够，工件会掉。

# 07 真空开关

> [!info] 核心结论
> 真空开关用于检测真空度是否达到设定值，是判断吸附是否成功的重要反馈元件。

## ① 分类/公式/规则

> [!note] 🟦 作用理解
> PLC不能只看真空阀是否动作，还要看真空是否真的建立。  
> 真空开关能把“吸住没吸住”的状态反馈给PLC。

> [!warning] 🟧 判断逻辑
> **吸附成功 = 真空度达到设定值 + 信号稳定保持**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **机械真空开关** | 简单，精度一般 |
| **数字真空开关** | 可显示、可设定 |
| **模拟量真空传感器** | 连续反馈真空值 |
| **PNP/NPN输出** | 需匹配PLC输入 |
| **带迟滞设定** | 防止信号抖动 |
| **双输出型** | 可设置吸附/报警阈值 |

## ③ 计算/选型示例

- 已知：  
  设备要求真空度达到-60kPa后才允许搬运。
- 公式：  
  **搬运动作允许 = 真空反馈 ≤ -60kPa**
- 计算：  
  PLC检测真空开关ON后，才执行上升或移动动作。
- 结论：  
  应配置真空开关，并在程序中设置吸附确认和超时报警。

> [!tip] 记忆口诀
> 阀开不代表吸住，真空开关说了算。

# 08 真空过滤器

> [!info] 核心结论
> 真空过滤器用于防止粉尘、碎屑、液滴进入真空发生器、真空泵和真空阀。

## ① 分类/公式/规则

> [!note] 🟦 作用理解
> 真空系统是“往里抽气”，工件表面的灰尘、碎屑很容易被吸入管路。  
> 没有过滤器，真空元件容易堵塞、磨损或失效。

> [!warning] 🟧 安装规则
> **吸盘 → 真空过滤器 → 真空发生器/真空泵**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **过滤精度** | 按工况和厂家要求选择 |
| **流量能力** | 不能限制抽气速度 |
| **透明杯体** | 便于观察污染 |
| **滤芯更换** | 堵塞后真空建立变慢 |
| **防液体吸入** | 液体场景需专门方案 |
| **安装位置** | 尽量靠近污染源 |

## ③ 计算/选型示例

- 已知：  
  吸取纸箱时，真空建立越来越慢。
- 公式：  
  **建立慢 = 泄漏大 / 滤芯堵塞 / 管路过长 / 真空源不足**
- 计算：  
  纸屑粉尘可能堵塞真空过滤器。
- 结论：  
  应检查并更换滤芯，同时评估海绵吸盘和更大抽气流量。

> [!tip] 记忆口诀
> 真空系统越会吸，越要防脏东西被吸进去。

# 09 破真空与释放

> [!info] 核心结论
> 破真空用于快速释放工件，节拍快的设备必须重视破真空速度和释放稳定性。

## ① 分类/公式/规则

> [!note] 🟦 释放逻辑
> 吸附时需要建立负压；  
> 放料时需要快速让吸盘内部恢复大气压，甚至短暂吹正压，工件才能快速脱离吸盘。

> [!warning] 🟧 释放公式
> **释放速度 = 管路容积 + 破真空流量 + 吸盘密封状态**

## ② 参考表

| 方式 | 特点 |
|---|---|
| **自然泄压** | 简单，但速度慢 |
| **破真空阀** | 快速进气释放 |
| **正压吹气** | 释放更快 |
| **带节流破真空** | 防止吹飞轻薄工件 |
| **近端破真空** | 响应更快 |
| **集成发生器** | 常带吸附/破真空功能 |

## ③ 计算/选型示例

- 已知：  
  吸盘放料时工件粘在吸盘上，影响节拍。
- 公式：  
  **释放慢 = 负压消失慢 / 表面粘附 / 破真空不足**
- 计算：  
  若只靠自然泄压，管路越长释放越慢。
- 结论：  
  可增加 **破真空阀或短促吹气**，并调节吹气量。

> [!tip] 记忆口诀
> 吸得快不够，还要放得快。

# 10 真空单向阀与保压

> [!info] 核心结论
> 真空单向阀用于防止真空回流泄漏，在多吸盘系统中可提高单个吸盘漏气时的整体可靠性。

## ① 分类/公式/规则

> [!note] 🟦 使用场景
> 多个吸盘同时吸附时，如果某一个吸盘没有接触工件，可能导致整个真空回路泄漏。  
> 单向阀或真空保持阀可以减少这种影响。

> [!warning] 🟧 保压规则
> **多吸盘 + 工件可能漏吸 → 单向阀/真空保持阀要考虑**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **真空单向阀** | 防止真空回流 |
| **真空保持阀** | 断气短时保持吸附 |
| **节流阀** | 限制泄漏影响 |
| **真空蓄能罐** | 增加系统稳定性 |
| **分区控制** | 分区吸附更可靠 |
| **止回接头** | 小型局部防泄漏 |

## ③ 计算/选型示例

- 已知：  
  4个吸盘搬运板材，有时一个吸盘没有压到工件。
- 公式：  
  **单点漏气 → 总真空下降**
- 计算：  
  未接触的吸盘会成为泄漏口，影响其他吸盘。
- 结论：  
  可给每个吸盘支路增加 **真空单向阀或节流措施**。

> [!tip] 记忆口诀
> 多吸盘怕一个漏，单向保压来补救。

# 11 管路与接头

> [!info] 核心结论
> 真空管路越长、越细、接头越多，真空建立越慢，泄漏风险越高。

## ① 分类/公式/规则

> [!note] 🟦 管路原则
> 真空发生器或真空阀应尽量靠近吸盘；  
> 管路应短、直、少接头；  
> 软管内径不能过小，否则抽气速度受限。

> [!warning] 🟧 管路规则
> **短管路 + 大通径 + 少接头 = 真空建立更快**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **管径** | 影响抽气速度 |
| **管长** | 影响响应时间 |
| **接头数量** | 增加泄漏点 |
| **软管材质** | 影响耐磨和弯折 |
| **弯曲半径** | 过小会折扁 |
| **分支结构** | 影响各吸盘均衡性 |

## ③ 计算/选型示例

- 已知：  
  真空发生器距离吸盘2米，吸附响应慢。
- 公式：  
  **管路容积越大 → 抽空时间越长**
- 计算：  
  长管路需要抽走更多空气，真空建立时间变长。
- 结论：  
  应尽量把真空发生器靠近吸盘，或增大抽气流量和管径。

> [!tip] 记忆口诀
> 真空管越短，吸得越快越稳。

# 12 多吸盘布局

> [!info] 核心结论
> 多吸盘布局要保证受力均匀、重心稳定、避免翘曲，并考虑单个吸盘失效时的安全风险。

## ① 分类/公式/规则

> [!note] 🟦 布局原则
> 吸盘应尽量分布在工件重心周围，距离边缘不能太近。  
> 长板件、薄板件、软质工件要防止弯曲和下垂。

> [!warning] 🟧 布局公式
> **稳定吸附 = 吸盘合力中心接近工件重心 + 足够抗倾覆力矩**

## ② 参考表

| 工件类型 | 布局建议 |
|---|---|
| **小平板** | 1~2个吸盘可评估 |
| **大平板** | 多点均匀分布 |
| **长条件** | 沿长度方向布置 |
| **薄片件** | 防止弯曲下垂 |
| **软包装** | 用大面积或海绵吸盘 |
| **重物** | 多吸盘 + 高安全系数 |

## ③ 计算/选型示例

- 已知：  
  长条板材搬运时一端下垂。
- 公式：  
  **支撑点不足 → 工件弯曲下垂**
- 计算：  
  吸盘过于集中在中间，无法支撑两端。
- 结论：  
  应增加吸盘数量，并沿板材长度方向分散布置。

> [!tip] 记忆口诀
> 吸盘不只算吸力，还要看重心和力矩。

# 13 表面泄漏与材质

> [!info] 核心结论
> 工件表面状态决定真空吸附难度，平整不漏气最容易，粗糙、多孔、柔软表面最难。

## ① 分类/公式/规则

> [!note] 🟦 泄漏理解
> 真空系统一边抽气，一边可能从工件表面缝隙漏气。  
> 当泄漏量大于真空源抽气能力时，真空度就建立不起来。

> [!warning] 🟧 泄漏判断公式
> **真空建立失败 = 泄漏量 > 真空源抽气能力**

## ② 参考表

| 表面 | 选型重点 |
|---|---|
| **玻璃/金属板** | 平吸盘效果好 |
| **纸箱/木板** | 可能漏气，需大流量 |
| **薄膜/包装袋** | 易变形，需柔性吸盘 |
| **粗糙表面** | 可考虑海绵吸盘 |
| **油污表面** | 易滑移，需校核 |
| **高温工件** | 吸盘材质要耐温 |

## ③ 计算/选型示例

- 已知：  
  同样吸盘吸金属板正常，吸纸箱经常失败。
- 公式：  
  **纸箱多孔 → 泄漏大**
- 计算：  
  纸箱表面透气，真空度难以建立。
- 结论：  
  应改用大流量真空源、海绵吸盘或增加吸盘面积。

> [!tip] 记忆口诀
> 平整好吸，多孔漏气，粗糙要换吸盘。

# 14 真空回路控制

> [!info] 核心结论
> 真空回路通常包括吸附、保压、检测、搬运、破真空释放和异常报警几个步骤。

## ① 分类/公式/规则

> [!note] 🟦 控制流程
> 自动化设备中，真空动作不能只控制“吸”和“放”。  
> 还要判断真空是否达到设定值，是否超时，搬运中是否掉压。

> [!warning] 🟧 标准动作流程
> **下降到位 → 开真空 → 检测真空到位 → 搬运 → 到位 → 破真空 → 确认释放**

## ② 参考表

| 步骤 | 重点 |
|---|---|
| **开真空** | 启动发生器或阀 |
| **吸附确认** | 真空开关到位 |
| **超时报警** | 防止未吸住继续运动 |
| **搬运监控** | 防止中途掉料 |
| **破真空** | 快速释放 |
| **释放确认** | 防止工件粘连 |

## ③ 计算/选型示例

- 已知：  
  开真空后0.5s内必须吸附成功。
- 公式：  
  **吸附成功时间 ≤ 0.5s**
- 计算：  
  若真空开关0.5s内未到位，PLC应报警并禁止搬运。
- 结论：  
  程序中应设置 **真空建立超时保护**。

> [!tip] 记忆口诀
> 吸住再搬，放开再走；没反馈，不动作。

# 15 常见故障排查

> [!info] 核心结论
> 真空故障排查按“真空源 → 管路 → 过滤器 → 吸盘 → 工件表面 → 真空开关”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看真空源有没有负压；  
> 再看管路是否漏气或堵塞；  
> 再看吸盘是否老化、破损、位置偏；  
> 最后看工件表面是否漏气或变形。

> [!warning] 🟧 排错公式
> **吸不住 = 真空不足 + 泄漏过大 + 吸盘不合适 + 管路堵漏 + 工件表面不适合**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **无真空** | 供气无压、发生器堵塞、泵故障 |
| **真空低** | 漏气、吸盘破损、表面粗糙 |
| **建立慢** | 管路长、滤芯堵、抽气量小 |
| **搬运掉料** | 安全系数不足、加速度过大 |
| **释放慢** | 破真空不足、管路太长 |
| **真空信号抖动** | 阈值太近、泄漏波动 |
| **吸盘磨损** | 密封变差，需更换 |

## ③ 计算/选型示例

- 已知：  
  真空表显示负压正常，但工件搬运时掉落。
- 公式：  
  **搬运掉落 = 吸附力不足 / 加速度过大 / 摩擦不足 / 吸盘布局不稳**
- 计算：  
  静态能吸住，不代表动态搬运一定安全。
- 结论：  
  应提高安全系数、降低加速度、增加吸盘数量或优化布局。

> [!tip] 记忆口诀
> 静态吸住不算稳，搬运不掉才算真稳。

# 16 真空元件快速选型表

> [!info] 核心结论
> 真空元件选型先确认工件重量和表面，再计算吸附力，最后校核真空源、吸盘、检测和释放。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看工件重量  
> ② 看工件表面是否漏气  
> ③ 看搬运姿态：水平/垂直  
> ④ 看速度和加速度  
> ⑤ 计算所需吸附力  
> ⑥ 选择吸盘类型和数量  
> ⑦ 选择真空发生器或真空泵  
> ⑧ 配真空过滤器和单向阀  
> ⑨ 配真空开关做反馈  
> ⑩ 配破真空功能并测试释放节拍

> [!warning] 🟧 最终选型公式
> **推荐真空系统 = 真空源 + 吸盘 + 过滤器 + 真空开关 + 破真空阀 + 管路接头**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 小工件快速取放 | 真空发生器 |
| 多吸盘持续吸附 | 真空泵 |
| 平整表面 | 平形吸盘 |
| 曲面工件 | 波纹吸盘 |
| 粗糙多孔 | 海绵吸盘/大流量 |
| 防掉料 | 真空开关 + 单向阀 |
| 快速放料 | 破真空阀 |

## ③ 计算/选型示例

- 已知：  
  工件质量3kg，垂直搬运，取安全系数4。
- 公式：  
  **所需吸附力 ≥ m × g × 安全系数**
- 计算：  
  所需吸附力 ≥ 3 × 9.8 × 4 = **117.6N**
- 结论：  
  吸盘总实际吸附力应大于 **117.6N**，并按表面泄漏、加速度和厂家手册校核。

> [!tip] 选型口诀
> 先看工件，再算吸力；平面用吸盘，多孔看流量；吸住要检测，放料要破真空。
