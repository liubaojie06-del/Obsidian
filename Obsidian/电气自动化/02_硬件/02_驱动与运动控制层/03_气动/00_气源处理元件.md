# 01 气源处理元件总览

> [!info] 核心结论
> 气源处理元件的核心作用是把压缩空气处理成 **干净、稳定、合适压力、必要时带润滑** 的工作气源。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 压缩空气从空压机出来后，通常会带有 **水分、油雾、粉尘、压力波动**。  
> 气源处理元件就是保护电磁阀、气缸、气爪、气动执行器的第一道防线。

> [!warning] 🟧 通用选型公式
> **气源处理选型 = 用气量 + 工作压力 + 过滤精度 + 接口尺寸 + 排水方式 + 是否润滑 + 使用环境**

## ② 参考表

| 元件 | 主要作用 |
|---|---|
| **过滤器** | 去水、去杂质 |
| **减压阀** | 稳定输出压力 |
| **油雾器** | 给气动元件润滑 |
| **三联件** | 过滤 + 减压 + 油雾 |
| **精密过滤器** | 去除细小颗粒/油雾 |
| **压力表** | 显示当前压力 |
| **自动排水器** | 自动排出冷凝水 |

## ③ 计算/选型示例

- 已知：  
  设备总用气量约 **800L/min**，现场压力0.6MPa。
- 公式：  
  **处理元件额定流量 ≥ 实际用气量 × 余量系数**
- 计算：  
  若取1.3倍余量：800 × 1.3 = **1040L/min**
- 结论：  
  应选择额定流量不小于 **1040L/min** 的气源处理元件，具体按厂家流量曲线校核。

> [!tip] 记忆口诀
> 先过滤，再减压；要润滑，加油雾；流量不够，全线憋气。

# 02 底层原理

> [!info] 核心结论
> 气源处理的底层逻辑是：先去杂质和水，再稳定压力，最后按需润滑。

## ① 分类/公式/规则

> [!note] 🟦 气路顺序
> 压缩空气先经过过滤器，把水和杂质分离；  
> 再经过减压阀，把压力调到设备需要的范围；  
> 最后按需求经过油雾器，给部分气动元件提供润滑。

> [!warning] 🟧 标准处理顺序
> **气源入口 → 过滤器 → 减压阀 → 油雾器 → 电磁阀/气缸**

## ② 参考表

| 顺序 | 作用 |
|---|---|
| **第1步：过滤** | 去除水滴、颗粒、杂质 |
| **第2步：减压** | 稳定输出压力 |
| **第3步：润滑** | 给需要润滑的元件供油雾 |
| **第4步：检测** | 压力表/压力开关监控 |
| **第5步：排水** | 防止积水进入气路 |
| **第6步：维护** | 更换滤芯、补油、排水 |

## ③ 计算/选型示例

- 已知：  
  气缸动作无力，压力表显示压力波动大。
- 公式：  
  **稳定动作 = 气源压力稳定 + 流量足够 + 管路无堵塞**
- 计算：  
  若过滤器堵塞或减压阀流量不足，动作时压力会下降。
- 结论：  
  应检查滤芯堵塞、减压阀规格、管径和供气压力。

> [!tip] 记忆口诀
> 气先洗干净，再把压力稳，最后看需不需要加油。

# 03 过滤器

> [!info] 核心结论
> 过滤器用于去除压缩空气中的水分、油污和颗粒杂质，是气动系统的基础保护元件。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 过滤器内部通过旋流、挡水、滤芯等结构，把水滴和颗粒从气流中分离出来，保护后端阀岛、气缸和精密气动元件。

> [!warning] 🟧 选型公式
> **过滤器选型 = 流量 + 接口尺寸 + 过滤精度 + 排水方式 + 杯体材质**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **过滤精度** | 常见5μm、40μm等，按厂家校核 |
| **额定流量** | 必须满足设备用气 |
| **接口尺寸** | G1/8、G1/4、G1/2等 |
| **排水方式** | 手动、半自动、自动 |
| **杯体材质** | 金属杯/透明杯，按环境选择 |
| **滤芯维护** | 堵塞后压降增大 |

## ③ 计算/选型示例

- 已知：  
  设备阀岛前端需要普通过滤，现场粉尘较少。
- 公式：  
  **普通气动系统 → 常规过滤器可用**
- 计算：  
  可选择适合流量和接口的普通过滤器，过滤精度需按阀岛厂家要求校核。
- 结论：  
  若后端为精密阀、比例阀或仪表气源，应进一步增加精密过滤器。

> [!tip] 记忆口诀
> 过滤器不干净，阀和气缸都遭殃。

# 04 减压阀

> [!info] 核心结论
> 减压阀用于把较高且波动的气源压力，调成设备需要的稳定工作压力。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 减压阀不是单纯“降压”，而是让输出压力在用气波动时尽量稳定。  
> 它通常配压力表使用，方便现场设定和观察。

> [!warning] 🟧 压力设定规则
> **输出压力 = 执行元件所需压力 + 管路损失余量**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **调压范围** | 如0.05～0.85MPa，需按型号校核 |
| **输入压力** | 不能超过额定最大压力 |
| **输出压力** | 设备实际工作压力 |
| **流量能力** | 过小会导致动作时掉压 |
| **压力表接口** | 用于显示压力 |
| **带锁旋钮** | 防止误调压力 |

## ③ 计算/选型示例

- 已知：  
  气缸要求工作压力0.5MPa，管路损失估算0.05MPa。
- 公式：  
  **减压阀设定压力 ≈ 0.5 + 0.05**
- 计算：  
  设定压力 ≈ **0.55MPa**
- 结论：  
  可将减压阀初步设为0.55MPa，再根据实际动作速度和推力微调。

> [!tip] 记忆口诀
> 减压阀管压力，动作无力先看表。

# 05 油雾器

> [!info] 核心结论
> 油雾器用于把润滑油雾化后带入气流，为需要润滑的气动元件提供润滑。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 并不是所有气动系统都需要油雾器。  
> 现代很多电磁阀、气缸可使用无油润滑，是否需要加油应按元件厂家手册确认。

> [!warning] 🟧 使用规则
> **需要润滑的气动元件 → 可加油雾器**  
> **无油元件 / 洁净环境 / 食品医药 → 谨慎使用油雾器**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **油杯容量** | 决定补油周期 |
| **滴油量** | 需要现场调节 |
| **润滑油类型** | 必须按厂家要求 |
| **安装方向** | 通常需垂直安装 |
| **适用场景** | 气动工具、部分老式气动元件 |
| **禁用场景** | 洁净、喷涂、食品等需校核 |

## ③ 计算/选型示例

- 已知：  
  后端是无油润滑气缸和洁净环境。
- 公式：  
  **洁净环境 + 无油元件 → 不建议加油雾**
- 计算：  
  油雾可能污染产品、管路和阀岛。
- 结论：  
  可不配置油雾器，最终按气缸和阀岛厂家手册确认。

> [!tip] 记忆口诀
> 油雾不是越多越好，该润滑才加油。

# 06 气动三联件

> [!info] 核心结论
> 气动三联件通常由 **过滤器、减压阀、油雾器** 组成，用于常规气动设备的气源预处理。

## ① 分类/公式/规则

> [!note] 🟦 三联件组成
> 三联件常用于传统气动设备入口，完成基础过滤、压力稳定和润滑。  
> 若设备不需要润滑，也可使用二联件：过滤器 + 减压阀。

> [!warning] 🟧 三联件顺序
> **过滤器 → 减压阀 → 油雾器**

## ② 参考表

| 类型 | 组成 |
|---|---|
| **二联件** | 过滤器 + 减压阀 |
| **三联件** | 过滤器 + 减压阀 + 油雾器 |
| **组合式FRL** | Filter + Regulator + Lubricator |
| **带压力表** | 方便调压观察 |
| **带支架** | 便于安装 |
| **模块化组合** | 可加压力开关、软启动阀 |

## ③ 计算/选型示例

- 已知：  
  设备有普通气缸和电磁阀，但后端元件标注免润滑。
- 公式：  
  **免润滑元件 → 二联件优先**
- 计算：  
  过滤和减压仍然需要，油雾器可省略。
- 结论：  
  可选 **过滤减压二联件**，减少油雾污染风险。

> [!tip] 记忆口诀
> 二联管干净和压力，三联再多一个润滑。

# 07 精密过滤器

> [!info] 核心结论
> 精密过滤器用于去除更细小的颗粒、油雾和水雾，适合仪表气源、比例阀、精密阀和洁净用气。

## ① 分类/公式/规则

> [!note] 🟦 应用场景
> 普通过滤器只能做基础处理。  
> 若后端设备对气源洁净度要求高，需要增加精密过滤器、除油过滤器或活性炭过滤器。

> [!warning] 🟧 精密气源规则
> **精密元件前端 → 普通过滤 + 精密过滤 + 干燥处理**

## ② 参考表

| 类型 | 作用 |
|---|---|
| **主管路过滤器** | 初级除水除杂 |
| **精密过滤器** | 去除细小颗粒 |
| **除油过滤器** | 去除油雾 |
| **活性炭过滤器** | 去除异味和残油 |
| **无菌过滤器** | 食品/医药场景需校核 |
| **干燥器** | 降低露点，去除水汽 |

## ③ 计算/选型示例

- 已知：  
  后端使用气动比例阀，厂家要求洁净干燥气源。
- 公式：  
  **比例阀气源 = 稳压 + 精密过滤 + 干燥**
- 计算：  
  仅普通三联件可能无法满足比例阀洁净度要求。
- 结论：  
  应增加精密过滤器，具体过滤等级按比例阀厂家手册校核。

> [!tip] 记忆口诀
> 普通气缸看三联，精密阀前加精滤。

# 08 干燥器与水分控制

> [!info] 核心结论
> 压缩空气中的水分会导致阀芯卡滞、气缸生锈、冬季结冰和仪表故障，必须按需求做干燥处理。

## ① 分类/公式/规则

> [!note] 🟦 水分来源
> 空气被压缩后，温度和压力变化会让水汽凝结成冷凝水。  
> 过滤器只能去除部分液态水，若要控制水汽，需要干燥器。

> [!warning] 🟧 干燥需求公式
> **低温环境 / 精密气源 / 长管路 → 需要重点控制露点**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **冷冻式干燥机** | 常规降低水分 |
| **吸附式干燥机** | 更低露点要求 |
| **水分离器** | 去除液态水 |
| **自动排水器** | 自动排冷凝水 |
| **露点仪** | 监测干燥效果 |
| **储气罐排水** | 减少主管路含水 |

## ③ 计算/选型示例

- 已知：  
  冬季低温环境中，气动阀偶发卡滞。
- 公式：  
  **低温 + 水分 = 结冰/卡滞风险**
- 计算：  
  压缩空气含水量高，低温时可能结冰堵塞阀芯。
- 结论：  
  应检查干燥器、排水器、过滤器和管路积水情况。

> [!tip] 记忆口诀
> 气路最怕水，冬天更怕冻。

# 09 排水器

> [!info] 核心结论
> 排水器用于及时排出过滤器、储气罐和管路中的冷凝水，防止水进入气动系统。

## ① 分类/公式/规则

> [!note] 🟦 排水方式
> 手动排水需要定期操作；  
> 自动排水适合无人值守或冷凝水较多的场景；  
> 排水口应连接到安全排放位置，避免污染设备。

> [!warning] 🟧 排水规则
> **积水越多 → 排水频率越高 → 自动排水越有必要**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **手动排水** | 成本低，需要人工 |
| **半自动排水** | 无压时自动排水 |
| **自动排水** | 有水自动排出 |
| **电子排水阀** | 定时或液位控制 |
| **浮球排水** | 依靠液位动作 |
| **维护重点** | 防堵、防漏、防失效 |

## ③ 计算/选型示例

- 已知：  
  过滤杯经常积满水，现场人员不固定巡检。
- 公式：  
  **无人值守 + 水多 → 自动排水优先**
- 计算：  
  手动排水容易忘记，积水会进入后端气路。
- 结论：  
  建议选用 **自动排水型过滤器** 或单独配置自动排水器。

> [!tip] 记忆口诀
> 水不排出去，就会跑进阀和气缸里。

# 10 压力表与压力开关

> [!info] 核心结论
> 压力表用于现场观察压力，压力开关用于把气压状态反馈给PLC或报警系统。

## ① 分类/公式/规则

> [!note] 🟦 检测逻辑
> 压力表是“人看压力”；  
> 压力开关是“PLC看压力”。  
> 自动化设备通常需要把气压不足信号接入联锁。

> [!warning] 🟧 气压联锁规则
> **气压未达到设定值 → 禁止设备启动 / 输出报警**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **机械压力表** | 显示当前压力 |
| **数字压力表** | 显示更直观，可带输出 |
| **压力开关** | 气压到位/不足信号 |
| **模拟量压力传感器** | 连续压力反馈 |
| **低压报警** | 防止气缸无力 |
| **高压报警** | 防止元件超压 |

## ③ 计算/选型示例

- 已知：  
  设备要求气源压力低于0.45MPa时禁止运行。
- 公式：  
  **压力开关设定值 = 0.45MPa**
- 计算：  
  PLC检测到低压信号后，禁止启动气动动作并报警。
- 结论：  
  应配置压力开关或压力传感器，并在程序中加入气压联锁。

> [!tip] 记忆口诀
> 压力表给人看，压力开关给PLC看。

# 11 流量与接口尺寸

> [!info] 核心结论
> 气源处理元件流量和接口太小，会造成动作时压力下降、气缸无力、速度变慢。

## ① 分类/公式/规则

> [!note] 🟦 选型重点
> 接口尺寸不是越大越好，但过小会限制流量。  
> 应根据设备最大用气量、动作频率、管路长度和压力损失综合选择。

> [!warning] 🟧 流量余量公式
> **气源处理额定流量 ≥ 最大瞬时用气量 × 1.2～1.5**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **接口尺寸** | 影响最大流量 |
| **额定流量** | 决定供气能力 |
| **压力降** | 动作时压力损失 |
| **管径** | 影响流量和响应 |
| **管路长度** | 越长损失越大 |
| **同时动作数** | 决定瞬时用气量 |

## ③ 计算/选型示例

- 已知：  
  最大瞬时用气量为 **500L/min**，取1.5倍余量。
- 公式：  
  **额定流量 ≥ 500 × 1.5**
- 计算：  
  额定流量 ≥ **750L/min**
- 结论：  
  应选择额定流量不小于 **750L/min** 的气源处理元件，压力降需按厂家曲线校核。

> [!tip] 记忆口诀
> 接口小会憋气，流量小会掉压。

# 12 安装顺序与方向

> [!info] 核心结论
> 气源处理元件必须按气流方向和功能顺序安装，装反会影响过滤、减压和润滑效果。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 元件壳体通常有箭头标明气流方向。  
> 过滤器和油雾器一般要求竖直安装，方便排水和滴油。

> [!warning] 🟧 安装顺序
> **空压机/主管路 → 截止阀 → 过滤器 → 减压阀 → 油雾器 → 电磁阀 → 气缸**

## ② 参考表

| 安装项 | 要点 |
|---|---|
| **气流方向** | 按箭头安装 |
| **过滤杯方向** | 通常垂直向下 |
| **排水空间** | 方便排水维护 |
| **压力表位置** | 便于观察 |
| **油雾器位置** | 尽量靠近用油元件 |
| **检修阀** | 便于停气维护 |

## ③ 计算/选型示例

- 已知：  
  现场三联件装反，减压效果不稳定。
- 公式：  
  **方向错误 → 内部阀芯/滤芯工作异常**
- 计算：  
  气流方向与元件设计方向相反，会影响过滤和减压。
- 结论：  
  应按壳体箭头重新安装，并检查内部滤芯和密封件是否损坏。

> [!tip] 记忆口诀
> 看箭头，按顺序；先过滤，后减压，再润滑。

# 13 维护与更换

> [!info] 核心结论
> 气源处理元件需要定期排水、换滤芯、检查压力表、补润滑油，否则气动系统会逐渐变慢、无力或卡滞。

## ① 分类/公式/规则

> [!note] 🟦 维护重点
> 过滤杯积水要排；  
> 滤芯堵塞要换；  
> 油雾器缺油要补；  
> 压力表异常要校验；  
> 密封件老化要更换。

> [!warning] 🟧 维护判断公式
> **压力降增大 + 水分增多 + 动作变慢 = 气源处理需检查**

## ② 参考表

| 项目 | 维护动作 |
|---|---|
| **过滤杯** | 定期排水 |
| **滤芯** | 堵塞或到期更换 |
| **减压阀** | 检查调压是否稳定 |
| **压力表** | 检查指针是否正常 |
| **油雾器** | 补油、调滴油量 |
| **密封圈** | 漏气时检查更换 |

## ③ 计算/选型示例

- 已知：  
  设备动作越来越慢，压力表空闲时正常，动作时明显下降。
- 公式：  
  **动作掉压 = 流量不足 / 滤芯堵塞 / 管路压降大**
- 计算：  
  滤芯堵塞会导致动态压力下降。
- 结论：  
  应优先检查过滤器滤芯、主管路压力和气源处理元件流量规格。

> [!tip] 记忆口诀
> 气动慢慢变差，多半先查滤芯和积水。

# 14 常见故障排查

> [!info] 核心结论
> 气源处理故障要按“气源压力 → 过滤积水 → 减压设定 → 流量压降 → 后端元件”的顺序排查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看主管压力，再看处理元件出口压力；  
> 先看静态压力，再看动作时动态压力；  
> 先查过滤器，再查减压阀和管路。

> [!warning] 🟧 排错公式
> **气动异常 = 压力不足 + 流量不足 + 水分过多 + 滤芯堵塞 + 减压阀异常 + 管路漏气**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **气缸无力** | 压力低、流量不足、漏气 |
| **动作变慢** | 滤芯堵塞、管径小、供气不足 |
| **阀卡滞** | 水分、油污、杂质进入 |
| **压力波动大** | 减压阀规格小或气源不足 |
| **杯体积水多** | 干燥不良、排水失效 |
| **油雾过多** | 滴油量过大或不该加油 |
| **压力表不动** | 堵塞、损坏或安装异常 |

## ③ 计算/选型示例

- 已知：  
  多个气缸同时动作时压力从0.6MPa降到0.35MPa。
- 公式：  
  **动态掉压 = 瞬时用气量 > 供气能力**
- 计算：  
  同时动作时气量需求大，若处理元件或管路流量不足，就会明显掉压。
- 结论：  
  应增大气源处理元件流量规格、主管管径或优化动作顺序。

> [!tip] 记忆口诀
> 静态有压不代表够用，动作一掉压才是真问题。

# 15 气源处理元件快速选型表

> [!info] 核心结论
> 气源处理元件选型先确认用气量和压力，再确定过滤精度、接口尺寸、排水方式和是否需要润滑。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看设备总用气量  
> ② 看工作压力范围  
> ③ 看接口尺寸和管径  
> ④ 看过滤精度  
> ⑤ 看是否需要精密过滤  
> ⑥ 看是否需要油雾润滑  
> ⑦ 看排水方式  
> ⑧ 看压力表/压力开关  
> ⑨ 看环境温度和粉尘油污  
> ⑩ 按厂家流量曲线校核压降

> [!warning] 🟧 最终选型公式
> **推荐配置 = 过滤器 + 减压阀 + 压力表/开关 + 排水方式 + 按需油雾器/精密过滤器**

## ② 参考表

| 需求 | 推荐配置 |
|---|---|
| 普通气缸系统 | 过滤器 + 减压阀 |
| 传统需润滑元件 | 三联件 |
| 阀岛/比例阀 | 精密过滤 + 稳压 |
| 洁净环境 | 无油气源 + 精密过滤 |
| 水分多 | 干燥机 + 自动排水 |
| 无人值守 | 自动排水器 |
| 气压联锁 | 压力开关/传感器 |

## ③ 计算/选型示例

- 已知：  
  设备最大用气量600L/min，工作压力0.5MPa，后端为普通气缸和电磁阀，免润滑。
- 公式：  
  **额定流量 ≥ 600 × 1.3**
- 计算：  
  额定流量 ≥ **780L/min**
- 结论：  
  可选择额定流量不小于780L/min的 **过滤减压二联件**，油雾器可不配，最终按厂家手册校核。

> [!tip] 选型口诀
> 用气量定大小，工作压定范围；普通用二联，需油用三联，精密用精滤。
