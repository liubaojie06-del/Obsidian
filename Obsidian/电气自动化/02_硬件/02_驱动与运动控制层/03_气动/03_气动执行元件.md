# 01 气动执行元件总览

> [!info] 核心结论
> 气动执行元件把压缩空气的压力能转换成 **直线运动、旋转运动、夹持动作、吸附动作**，是气动系统真正“干活”的部分。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 气源处理元件负责“气干不干净”；  
> 气动控制元件负责“气怎么走”；  
> 气动执行元件负责“把气变成动作”。

> [!warning] 🟧 通用选型公式
> **气动执行元件选型 = 动作形式 + 推力/夹持力 + 行程/角度 + 速度 + 安装方式 + 负载方向 + 使用环境**

## ② 参考表

| 类别 | 主要作用 |
|---|---|
| **标准气缸** | 直线伸缩 |
| **薄型气缸** | 短行程、空间紧凑 |
| **导杆气缸** | 防转、承受侧向力 |
| **滑台气缸** | 高刚性直线导向 |
| **无杆气缸** | 长行程直线移动 |
| **旋转气缸** | 输出摆动角度 |
| **气爪** | 夹取工件 |
| **真空吸盘** | 吸附搬运 |

## ③ 计算/选型示例

- 已知：  
  设备需要把工件向前推送100mm。
- 公式：  
  **直线推送动作 → 气缸类执行元件**
- 计算：  
  需要确认推力、行程、速度、安装空间和是否受侧向力。
- 结论：  
  可优先选择 **标准气缸或导杆气缸**，具体缸径和行程需按负载校核。

> [!tip] 记忆口诀
> 直线看气缸，旋转看摆缸，夹取看气爪，吸附看真空。

# 02 底层原理

> [!info] 核心结论
> 气动执行元件的本质，是利用压缩空气压力推动活塞、叶片或夹爪机构产生机械动作。

## ① 分类/公式/规则

> [!note] 🟦 工作链路
> 压缩空气进入执行元件内部，作用在活塞或叶片面积上，产生推力或转矩，推动负载运动。

> [!warning] 🟧 能量转换逻辑
> **压缩空气 → 压力作用面积 → 推力/转矩 → 机械运动**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **气源压力** | 提供动力来源 |
| **活塞面积** | 决定理论推力 |
| **密封件** | 防止漏气 |
| **活塞杆** | 输出直线力 |
| **导向结构** | 保证运动方向 |
| **缓冲结构** | 减小末端冲击 |

## ③ 计算/选型示例

- 已知：  
  气缸压力0.5MPa，活塞有效面积0.001m²。
- 公式：  
  **F = P × A**
- 计算：  
  F = 0.5 × 10⁶ × 0.001 = **500N**
- 结论：  
  理论推力约为 **500N**，实际需考虑摩擦、效率和安全余量。

> [!tip] 记忆口诀
> 压力给力气，面积放大力。

# 03 单作用气缸

> [!info] 核心结论
> 单作用气缸只有一个方向靠气压动作，另一个方向通常靠弹簧复位。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 单作用气缸结构简单、耗气少，适合短行程、轻负载、复位方向要求不大的场景。

> [!warning] 🟧 选型规则
> **简单推出 / 自动弹回 → 单作用气缸可选**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制阀** | 常用3/2阀 |
| **动作方向** | 一边气压，一边弹簧 |
| **优点** | 结构简单，耗气少 |
| **缺点** | 行程短，回程力有限 |
| **典型应用** | 顶出、挡料、压紧 |
| **注意** | 弹簧力会抵消部分推力 |

## ③ 计算/选型示例

- 已知：  
  机构只需要短距离推出，断气后自动缩回。
- 公式：  
  **断气自动复位 → 单作用气缸**
- 计算：  
  弹簧可在断气后带动活塞回位。
- 结论：  
  可选择 **单作用弹簧复位气缸**，推力需扣除弹簧阻力后校核。

> [!tip] 记忆口诀
> 单作用一口气，出去靠气，回来靠弹簧。

# 04 双作用气缸

> [!info] 核心结论
> 双作用气缸伸出和缩回都靠气压驱动，是自动化设备中最常见的气缸类型。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 双作用气缸有两个气口，分别控制伸出和缩回。  
> 推出力和回拉力不同，因为有杆腔面积比无杆腔小。

> [!warning] 🟧 推力规则
> **伸出力 > 缩回力**  
> 因为缩回侧要扣除活塞杆面积。

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制阀** | 常用5/2阀或5/3阀 |
| **伸出** | 无杆腔进气 |
| **缩回** | 有杆腔进气 |
| **优点** | 双向有力，控制灵活 |
| **缺点** | 耗气量比单作用大 |
| **典型应用** | 推料、夹紧、升降、定位 |

## ③ 计算/选型示例

- 已知：  
  双作用气缸需要推送负载，要求推力大于300N。
- 公式：  
  **F = P × A × η**
- 计算：  
  应按气源压力、缸径面积和效率计算伸出力。
- 结论：  
  需选择伸出理论推力大于300N并留安全余量的双作用气缸。

> [!tip] 记忆口诀
> 双作用两个口，伸缩都靠气。

# 05 缸径与推力计算

> [!info] 核心结论
> 气缸推力主要由 **压力和缸径** 决定，缸径越大，推力越大，耗气量也越大。

## ① 分类/公式/规则

> [!warning] 🟧 推力公式
> **F = P × A × η**

> [!note] 参数说明
> **F**：实际推力，N  
> **P**：工作压力，Pa  
> **A**：有效面积，m²  
> **η**：效率，需按实际气缸和厂家手册校核

## ② 参考表

| 参数 | 影响 |
|---|---|
| **压力P** | 越高推力越大 |
| **缸径D** | 越大面积越大 |
| **活塞杆直径** | 影响回程力 |
| **摩擦** | 降低实际输出 |
| **安全系数** | 防止动作无力 |
| **效率η** | 需按厂家校核 |

## ③ 计算/选型示例

- 已知：  
  气缸缸径40mm，压力0.5MPa，效率暂按0.8估算。
- 公式：  
  **A = πD² ÷ 4**  
  **F = P × A × η**
- 计算：  
  A = 3.14 × 0.04² ÷ 4 ≈ 0.001256m²  
  F = 0.5 × 10⁶ × 0.001256 × 0.8 ≈ **502N**
- 结论：  
  该气缸实际推力约 **502N**，最终需按厂家手册校核。

> [!tip] 记忆口诀
> 缸径大一圈，推力涨一截；力不够，先看压力和缸径。

# 06 行程与速度

> [!info] 核心结论
> 气缸行程决定动作距离，动作速度主要由流量、节流阀、负载和管路决定。

## ① 分类/公式/规则

> [!note] 🟦 工程理解
> 行程不是越长越好，过长会增加弯曲、冲击和导向要求。  
> 速度不是只靠气压决定，气源流量和排气能力同样关键。

> [!warning] 🟧 基本关系
> **气缸速度 ∝ 流量 ÷ 有效面积**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **行程** | 决定移动距离 |
| **流量** | 决定速度上限 |
| **节流阀** | 调节动作速度 |
| **排气能力** | 影响动作平稳性 |
| **负载大小** | 影响速度稳定 |
| **缓冲** | 影响末端冲击 |

## ③ 计算/选型示例

- 已知：  
  气缸行程200mm，要求0.5s完成伸出。
- 公式：  
  **平均速度 = 行程 ÷ 时间**
- 计算：  
  平均速度 = 200 ÷ 0.5 = **400mm/s**
- 结论：  
  需确认气缸、阀、管路流量能支持约400mm/s动作速度。

> [!tip] 记忆口诀
> 行程定距离，流量定速度，缓冲定平稳。

# 07 薄型气缸

> [!info] 核心结论
> 薄型气缸适合安装空间受限、行程较短的场景，但导向和抗侧向力能力有限。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 薄型气缸轴向长度短，结构紧凑，适合夹具、挡料、压紧、顶升等短行程动作。

> [!warning] 🟧 使用规则
> **空间小 + 行程短 → 薄型气缸优先**  
> **侧向力大 → 不宜单独使用薄型气缸**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 短小紧凑 |
| **缺点** | 导向能力弱 |
| **适合** | 短行程压紧、顶升 |
| **不适合** | 长行程、高侧向力 |
| **安装方式** | 法兰、底座、侧面等 |
| **注意** | 避免活塞杆受偏载 |

## ③ 计算/选型示例

- 已知：  
  夹具内部空间很小，只需压紧20mm行程。
- 公式：  
  **空间小 + 短行程 = 薄型气缸**
- 计算：  
  标准气缸轴向长度可能过长，薄型气缸更适合。
- 结论：  
  可选薄型气缸，但压紧点应尽量同轴，避免偏载。

> [!tip] 记忆口诀
> 空间紧，用薄缸；偏载大，加导向。

# 08 导杆气缸

> [!info] 核心结论
> 导杆气缸自带导向结构，能防止活塞杆旋转，并承受一定侧向力和力矩。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 导杆气缸比普通气缸更适合推板、挡料、压装、顶升等需要防转和抗偏载的场景。

> [!warning] 🟧 选择规则
> **有侧向力 / 有偏心负载 / 需要防转 → 导杆气缸优先**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **导向方式** | 滑动轴承或滚珠轴承 |
| **优点** | 防转、抗偏载 |
| **缺点** | 体积更大、成本更高 |
| **适合** | 推板、夹具、顶升 |
| **重点校核** | 允许力矩、行程、速度 |
| **注意** | 不能超出厂家侧向力限制 |

## ③ 计算/选型示例

- 已知：  
  气缸前端连接推板，推板受力点偏离中心。
- 公式：  
  **偏心负载 → 产生力矩**
- 计算：  
  普通气缸活塞杆容易受弯，影响寿命。
- 结论：  
  应选择 **导杆气缸** 或外加直线导轨支撑。

> [!tip] 记忆口诀
> 普通缸怕偏载，导杆缸抗偏心。

# 09 滑台气缸

> [!info] 核心结论
> 滑台气缸把气缸和精密导轨集成在一起，适合高刚性、高重复定位的短行程直线运动。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 滑台气缸通常比导杆气缸刚性更好，适合搬运、定位、压入、夹具滑台等场景。

> [!warning] 🟧 选择规则
> **短行程 + 高刚性 + 需要导向精度 → 滑台气缸**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 刚性好、导向精度高 |
| **缺点** | 成本高于普通气缸 |
| **适合** | 小型搬运、夹具滑台 |
| **重复精度** | 通常优于普通气缸 |
| **安装** | 可水平、垂直、侧装 |
| **重点校核** | 负载、力矩、速度、寿命 |

## ③ 计算/选型示例

- 已知：  
  设备需要小工件横移30mm，要求重复位置稳定。
- 公式：  
  **短行程定位 + 稳定导向 → 滑台气缸**
- 计算：  
  普通气缸需要外加导轨，滑台气缸可直接集成。
- 结论：  
  可选择 **滑台气缸**，并校核负载质量和允许力矩。

> [!tip] 记忆口诀
> 要直线又要稳，滑台气缸更省心。

# 10 无杆气缸

> [!info] 核心结论
> 无杆气缸适合长行程直线移动，能节省安装长度，但负载和导向方式要重点校核。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 无杆气缸没有传统外伸活塞杆，滑块沿缸体移动。  
> 常见有磁耦式和机械连接式。

> [!warning] 🟧 选择规则
> **长行程 + 空间受限 → 无杆气缸可选**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **磁耦无杆气缸** | 密封好，负载有限 |
| **机械式无杆气缸** | 推力更大，结构复杂 |
| **优点** | 节省长度，适合长行程 |
| **缺点** | 抗偏载需校核 |
| **适合** | 搬运、推送、横移 |
| **注意** | 长行程需注意导向和缓冲 |

## ③ 计算/选型示例

- 已知：  
  需要移动距离1000mm，但安装空间不允许普通气缸外伸。
- 公式：  
  **长行程 + 安装长度受限 = 无杆气缸**
- 计算：  
  普通气缸总长度约为缸体长度加活塞杆伸出长度，占空间大。
- 结论：  
  可选择 **无杆气缸**，并按负载和导向要求校核。

> [!tip] 记忆口诀
> 长行程嫌太长，无杆气缸来帮忙。

# 11 旋转气缸

> [!info] 核心结论
> 旋转气缸用于输出一定角度的摆动动作，常见角度有90°、180°等，适合翻转、分度、摆臂动作。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 旋转气缸不是连续旋转电机，而是输出有限角度的摆动。  
> 常用于工件翻转、夹具转位、挡板开合等。

> [!warning] 🟧 选型公式
> **旋转气缸选型 = 输出角度 + 输出转矩 + 负载惯量 + 速度 + 缓冲方式**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **旋转角度** | 90°、180°等 |
| **输出转矩** | 带动负载能力 |
| **负载惯量** | 影响冲击和停止 |
| **缓冲方式** | 液压缓冲/气缓冲 |
| **重复精度** | 影响转位稳定 |
| **安装方式** | 轴端、法兰、底座 |

## ③ 计算/选型示例

- 已知：  
  夹具需要把工件翻转90°。
- 公式：  
  **有限角度翻转 → 旋转气缸**
- 计算：  
  需要按工件重量、重心距离和动作速度计算所需转矩。
- 结论：  
  可选择 **90°旋转气缸**，并校核转矩和缓冲能力。

> [!tip] 记忆口诀
> 直线用气缸，翻转用旋转缸。

# 12 气爪

> [!info] 核心结论
> 气爪用于夹取工件，选型重点是夹持力、开闭行程、手指长度、工件重量和摩擦系数。

## ① 分类/公式/规则

> [!note] 🟦 核心类型
> 气爪常见有平行气爪、角型气爪、三爪气爪、薄型气爪等。  
> 平行气爪适合常规夹取，三爪气爪适合圆形工件定心夹持。

> [!warning] 🟧 夹持规则
> **夹持力必须大于工件重力和加速度惯性力，并留安全余量**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **平行气爪** | 常规夹取 |
| **角型气爪** | 大开口夹取 |
| **三爪气爪** | 圆形工件定心 |
| **薄型气爪** | 空间紧凑 |
| **长手指** | 需降额校核 |
| **带磁开关** | 检测开闭到位 |

## ③ 计算/选型示例

- 已知：  
  工件质量0.5kg，垂直夹取，重力约4.9N。
- 公式：  
  **夹持力 ≥ 工件重力 × 安全系数**
- 计算：  
  若取3倍安全系数：夹持力 ≥ 4.9 × 3 = **14.7N**
- 结论：  
  气爪夹持力应不小于 **14.7N**，实际还需考虑摩擦、加速度和手指长度。

> [!tip] 记忆口诀
> 气爪夹得住，还要搬得稳。

# 13 真空吸盘

> [!info] 核心结论
> 真空吸盘通过负压吸附工件，适合搬运平面、薄片、包装袋、玻璃、板材等物体。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 真空吸附不是靠夹紧，而是靠大气压把工件压在吸盘上。  
> 工件表面越平整、越不漏气，吸附越稳定。

> [!warning] 🟧 吸附力公式
> **F = ΔP × A × 安全系数修正**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **吸盘直径** | 决定吸附面积 |
| **真空度** | 决定压力差 |
| **工件表面** | 平整度影响泄漏 |
| **吸盘材质** | 硅胶、橡胶、聚氨酯等 |
| **真空发生器** | 产生负压 |
| **真空开关** | 检测吸附到位 |

## ③ 计算/选型示例

- 已知：  
  吸盘有效面积0.001m²，真空压差为60kPa。
- 公式：  
  **F = ΔP × A**
- 计算：  
  F = 60000 × 0.001 = **60N**
- 结论：  
  理论吸附力约为 **60N**，实际需按表面泄漏、加速度和安全系数校核。

> [!tip] 记忆口诀
> 平面薄件用吸盘，漏气表面要谨慎。

# 14 磁性开关与到位检测

> [!info] 核心结论
> 气缸磁性开关用于检测活塞位置，是PLC判断气缸伸出、缩回到位的常用方式。

## ① 分类/公式/规则

> [!note] 🟦 检测逻辑
> 气缸活塞内置磁环，磁性开关安装在缸体外侧。  
> 活塞运动到对应位置时，磁性开关输出信号给PLC。

> [!warning] 🟧 到位判断规则
> **动作完成 = 电磁阀输出 + 到位开关反馈**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **磁性开关** | 检测气缸位置 |
| **接近开关** | 检测机构到位 |
| **压力开关** | 检测夹紧压力 |
| **真空开关** | 检测吸附状态 |
| **PLC输入** | 接收反馈信号 |
| **超时报警** | 动作不到位保护 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出后，PLC没有收到伸出到位信号。
- 公式：  
  **无反馈 = 开关位置不对 + 接线异常 + 活塞未到位**
- 计算：  
  可先查看开关指示灯，再微调磁性开关位置。
- 结论：  
  应检查磁性开关位置、接线、PLC输入和气缸是否真正到位。

> [!tip] 记忆口诀
> 阀让它动，开关证明它到了。

# 15 缓冲与冲击

> [!info] 核心结论
> 气动执行元件速度快、冲击大，必须根据负载和速度配置缓冲，防止撞击、噪声和寿命下降。

## ① 分类/公式/规则

> [!note] 🟦 缓冲方式
> 常见缓冲方式有橡胶缓冲、气缓冲、液压缓冲器和外部机械缓冲。  
> 负载越大、速度越高，越要重视缓冲能力。

> [!warning] 🟧 冲击风险公式
> **冲击能量 ∝ 负载质量 × 速度²**

## ② 参考表

| 缓冲方式 | 特点 |
|---|---|
| **橡胶缓冲** | 简单，适合轻载 |
| **气缓冲** | 气缸常见内置缓冲 |
| **液压缓冲器** | 适合高速重载 |
| **外部限位** | 机械定位保护 |
| **速度控制阀** | 降低末端冲击 |
| **软着陆控制** | 精密场景使用 |

## ③ 计算/选型示例

- 已知：  
  气缸高速伸出到末端时撞击声大。
- 公式：  
  **末端冲击 = 速度过快 + 缓冲不足**
- 计算：  
  可先降低排气流量或调整气缓冲，再评估外部缓冲器。
- 结论：  
  应优化速度和缓冲，否则会影响气缸、夹具和工件寿命。

> [!tip] 记忆口诀
> 气缸跑得快，末端要缓冲。

# 16 常见故障排查

> [!info] 核心结论
> 气动执行元件故障要按“气源 → 阀 → 管路 → 执行元件 → 负载 → 反馈”的顺序排查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看压力是否够，再看电磁阀是否换向；  
> 先空载动作，再带载测试；  
> 先查气路漏气，再查执行元件磨损。

> [!warning] 🟧 排错公式
> **执行异常 = 压力不足 + 流量不足 + 阀未换向 + 密封漏气 + 负载卡滞 + 到位检测异常**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **不动作** | 无气源、阀不换向、管路接错 |
| **动作无力** | 压力低、缸径小、漏气 |
| **动作慢** | 流量不足、节流太小、排气堵 |
| **爬行抖动** | 负载摩擦大、润滑差 |
| **不到位** | 行程受阻、负载过大、开关位置错 |
| **漏气** | 密封圈磨损、接头松动 |
| **冲击大** | 速度过快、缓冲不足 |

## ③ 计算/选型示例

- 已知：  
  气缸空载能动作，带负载后动作无力。
- 公式：  
  **带载无力 = 推力不足 / 负载卡滞 / 压力下降**
- 计算：  
  空载正常说明气路基本通，带载异常重点查推力和机械阻力。
- 结论：  
  应检查工作压力、缸径选型、机械卡滞和动作时动态掉压。

> [!tip] 记忆口诀
> 空载能动带载不动，先查推力和负载。

# 17 气动执行元件快速选型表

> [!info] 核心结论
> 气动执行元件选型先明确动作类型，再计算力、行程、速度和安装方式。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看动作形式：直线/旋转/夹持/吸附  
> ② 看负载重量和方向  
> ③ 算推力、夹持力或吸附力  
> ④ 确认行程或旋转角度  
> ⑤ 确认动作速度和节拍  
> ⑥ 选择气缸、气爪、旋转缸或吸盘  
> ⑦ 校核安装空间和固定方式  
> ⑧ 校核缓冲、导向和抗偏载  
> ⑨ 配磁性开关或到位检测  
> ⑩ 按厂家手册校核寿命和环境

> [!warning] 🟧 最终选型公式
> **推荐执行元件 = 动作类型 + 输出力 + 行程/角度 + 速度 + 安装方式 + 检测方式 + 缓冲方式**

## ② 参考表

| 需求 | 推荐执行元件 |
|---|---|
| 普通直线推拉 | 标准气缸 |
| 空间紧凑短行程 | 薄型气缸 |
| 偏载推送 | 导杆气缸 |
| 高刚性短行程 | 滑台气缸 |
| 长行程横移 | 无杆气缸 |
| 翻转摆动 | 旋转气缸 |
| 夹取工件 | 气爪 |
| 吸附搬运 | 真空吸盘 |

## ③ 计算/选型示例

- 已知：  
  需要水平推动负载，所需推力约300N，工作压力0.5MPa，取效率0.8。
- 公式：  
  **A ≥ F ÷（P × η）**
- 计算：  
  A ≥ 300 ÷（0.5 × 10⁶ × 0.8）= **0.00075m²**  
  对应缸径约 **31mm**
- 结论：  
  可初步选择 **缸径32mm及以上气缸**，再按行程、速度、缓冲和厂家手册校核。

> [!tip] 选型口诀
> 先定动作，再算力；行程看距离，速度看流量；偏载加导向，快速加缓冲。
