# 01 气动传感器总览

> [!info] 核心结论
> 气动传感器用于检测气缸位置、气压、真空度、流量和动作状态，是PLC判断气动系统是否正常动作的关键反馈元件。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 气动系统不是“阀一通电就一定动作到位”。  
> PLC必须通过传感器确认 **气缸是否到位、压力是否足够、真空是否吸住、流量是否异常**。

> [!warning] 🟧 通用选型公式
> **气动传感器选型 = 检测对象 + 信号类型 + 输出方式 + 安装方式 + 响应速度 + 防护等级 + PLC输入匹配**

## ② 参考表

| 类型 | 检测对象 |
|---|---|
| **磁性开关** | 气缸伸出/缩回到位 |
| **压力开关** | 气源压力、夹紧压力 |
| **真空开关** | 吸盘是否吸住 |
| **流量传感器** | 气体流量、泄漏异常 |
| **接近开关** | 机构位置、工件到位 |
| **压力传感器** | 连续压力值 |
| **真空传感器** | 连续真空值 |

## ③ 计算/选型示例

- 已知：  
  一个气缸伸出后，PLC需要确认动作是否完成。
- 公式：  
  **动作完成 = 电磁阀输出 + 到位传感器反馈**
- 计算：  
  只给电磁阀输出，不能证明气缸真的伸出到位。
- 结论：  
  应在气缸伸出端安装 **磁性开关**，把到位信号接入PLC。

> [!tip] 记忆口诀
> 阀让它动，传感器证明它真的到了。

# 02 底层原理

> [!info] 核心结论
> 气动传感器的底层作用，是把气动系统中的位置、压力、真空、流量等物理状态转换成PLC可识别的电信号。

## ① 分类/公式/规则

> [!note] 🟦 信号链路
> 传感器检测物理量，内部电路转换信号，最后输出开关量或模拟量给PLC。  
> PLC根据反馈判断是否继续下一步动作，或触发报警。

> [!warning] 🟧 控制链路
> **气动动作 → 传感器检测 → 电信号输出 → PLC判断 → 下一步动作/报警**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **气缸运动** | 产生位置变化 |
| **压力变化** | 反映夹紧、供气状态 |
| **真空变化** | 反映吸附状态 |
| **流量变化** | 反映耗气和泄漏 |
| **传感器输出** | 开关量或模拟量 |
| **PLC输入** | 读取反馈并执行逻辑 |

## ③ 计算/选型示例

- 已知：  
  气缸发出伸出指令后，2秒内没有收到伸出到位信号。
- 公式：  
  **动作超时 = 指令已发出 + 到位反馈未出现**
- 计算：  
  可能是气缸卡滞、压力不足、传感器位置不对或接线异常。
- 结论：  
  PLC应触发 **伸出超时报警**，停止后续危险动作。

> [!tip] 记忆口诀
> 没反馈，不放心；有超时，必须报警。

# 03 磁性开关

> [!info] 核心结论
> 磁性开关用于检测气缸活塞位置，是气缸伸出、缩回到位检测最常用的传感器。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 气缸活塞内部通常带磁环。  
> 当活塞运动到磁性开关附近时，磁场触发开关，输出信号给PLC。

> [!warning] 🟧 到位判断规则
> **气缸到位 = 活塞磁环到达磁性开关检测区**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **安装位置** | 气缸两端槽内或绑带固定 |
| **常见输出** | PNP、NPN、两线式 |
| **检测对象** | 活塞磁环 |
| **指示灯** | 方便调试到位状态 |
| **常见用途** | 伸出到位、缩回到位 |
| **注意** | 开关位置需现场微调 |

## ③ 计算/选型示例

- 已知：  
  气缸实际已经伸出，但PLC没有收到伸出到位。
- 公式：  
  **无信号 = 开关位置不对 + 接线异常 + 活塞未到检测区**
- 计算：  
  可先观察磁性开关指示灯，再沿气缸槽微调位置。
- 结论：  
  优先检查 **磁性开关位置、接线、PLC输入点**。

> [!tip] 记忆口诀
> 气缸到没到，先看磁开关亮不亮。

---

# 04 干簧管式与霍尔式

> [!info] 核心结论
> 磁性开关常见有干簧管式和霍尔式，干簧管结构简单，霍尔式寿命和抗振性更好。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> **干簧管式**：磁场触发机械触点闭合。  
> **霍尔式**：利用半导体霍尔效应检测磁场，无机械触点。

> [!warning] 🟧 选择规则
> **低成本普通检测 → 干簧管可用**  
> **高频动作 / 振动场合 → 霍尔式优先**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **干簧管式** | 成本低，触点式 |
| **霍尔式** | 无触点，寿命较长 |
| **两线式** | 接线简单，可能有漏电流 |
| **三线式** | PNP/NPN输出清晰 |
| **带指示灯** | 调试方便 |
| **抗振要求高** | 霍尔式更合适 |

## ③ 计算/选型示例

- 已知：  
  气缸动作频率很高，现场振动明显。
- 公式：  
  **高频 + 振动 = 无触点优先**
- 计算：  
  干簧管有机械触点，长期高频动作可能影响寿命。
- 结论：  
  建议选择 **霍尔式磁性开关**。

> [!tip] 记忆口诀
> 干簧便宜，霍尔耐用；频率高，选无触点。

# 05 压力开关

> [!info] 核心结论
> 压力开关用于检测气压是否达到设定值，常用于气源低压报警、夹紧确认和安全联锁。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 压力开关输出的是开关量信号。  
> 当气压高于或低于设定值时，输出ON/OFF状态给PLC。

> [!warning] 🟧 联锁规则
> **气压不足 → 禁止启动 / 停止动作 / 输出报警**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **检测对象** | 正压气源 |
| **输出信号** | 开关量 |
| **常见用途** | 低压报警、夹紧确认 |
| **设定值** | 按工艺压力设定 |
| **迟滞** | 防止信号频繁抖动 |
| **显示方式** | 机械式或数字式 |

## ③ 计算/选型示例

- 已知：  
  设备要求气源压力低于0.45MPa时禁止运行。
- 公式：  
  **压力开关设定值 = 0.45MPa**
- 计算：  
  当压力低于0.45MPa，压力开关输出低压报警信号。
- 结论：  
  PLC应设置 **气压不足禁止启动** 的联锁逻辑。

> [!tip] 记忆口诀
> 气压不到位，动作别继续。

# 06 压力传感器

> [!info] 核心结论
> 压力传感器输出连续模拟量，可让PLC实时读取压力值，适合比例控制、趋势监测和精密压力判断。

## ① 分类/公式/规则

> [!note] 🟦 与压力开关区别
> 压力开关只告诉PLC“到没到设定值”；  
> 压力传感器能告诉PLC“现在具体是多少压力”。

> [!warning] 🟧 模拟量换算公式
> **压力 = 模拟量比例 × 满量程压力**

## ② 参考表

| 信号类型 | 特点 |
|---|---|
| **0-10V** | 接线简单，短距离常用 |
| **4-20mA** | 抗干扰较好 |
| **0-5V** | 小型设备常见 |
| **IO-Link** | 可传参数和状态 |
| **数字显示型** | 调试直观 |
| **模拟量型** | 适合闭环控制 |

## ③ 计算/选型示例

- 已知：  
  压力传感器量程0-1.0MPa，输出0-10V，PLC测得5V。
- 公式：  
  **压力 = 5 ÷ 10 × 1.0**
- 计算：  
  压力 = **0.5MPa**
- 结论：  
  当前压力约为 **0.5MPa**。

> [!tip] 记忆口诀
> 开关看是否到位，传感器看具体数值。

# 07 真空开关

> [!info] 核心结论
> 真空开关用于检测吸盘是否建立足够真空，是判断工件是否吸住的重要反馈元件。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 真空阀打开，不代表工件一定吸住。  
> 只有真空度达到设定值并保持稳定，才说明吸附基本成功。

> [!warning] 🟧 吸附确认规则
> **吸附成功 = 真空度达到设定阈值 + 信号稳定保持**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **检测对象** | 真空负压 |
| **单位** | kPa、MPa、bar等 |
| **常见用途** | 吸附确认、掉料报警 |
| **输出方式** | PNP、NPN、模拟量 |
| **迟滞设定** | 防止信号抖动 |
| **显示型** | 便于现场调试 |

## ③ 计算/选型示例

- 已知：  
  工件吸附要求真空度达到-60kPa后才能搬运。
- 公式：  
  **允许搬运 = 真空值 ≤ -60kPa**
- 计算：  
  若真空开关未到位，机械手不允许上升或移动。
- 结论：  
  PLC应设置 **真空不到位禁止搬运** 和 **吸附超时报警**。

> [!tip] 记忆口诀
> 阀开不算吸住，真空到位才算数。

# 08 流量传感器

> [!info] 核心结论
> 流量传感器用于检测气体流量，可用于耗气监测、泄漏检测、吹气量控制和设备节能分析。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 气动系统中，流量异常常常意味着漏气、堵塞、阀异常或执行元件异常。  
> 流量传感器可以帮助判断系统是否处于正常耗气状态。

> [!warning] 🟧 判断规则
> **流量异常 = 泄漏增加 / 管路堵塞 / 阀动作异常 / 用气元件异常**

## ② 参考表

| 检测用途 | 说明 |
|---|---|
| **总耗气量** | 统计设备用气 |
| **支路流量** | 判断单支路异常 |
| **泄漏检测** | 停机仍有流量即可能漏气 |
| **吹气控制** | 保证吹气量稳定 |
| **节能分析** | 找出高耗气工位 |
| **堵塞判断** | 流量过小可能堵塞 |

## ③ 计算/选型示例

- 已知：  
  设备停机时，流量传感器仍显示有稳定流量。
- 公式：  
  **停机流量 > 0 → 可能存在漏气**
- 计算：  
  停机状态下正常应无明显用气。
- 结论：  
  应检查接头、气管、阀体、气缸密封和未封堵气口。

> [!tip] 记忆口诀
> 停机还耗气，多半有漏气。

# 09 接近开关与气动机构检测

> [!info] 核心结论
> 接近开关常用于检测工件、夹具、机构位置，可补充气缸磁性开关无法直接判断的机械状态。

## ① 分类/公式/规则

> [!note] 🟦 使用场景
> 磁性开关检测的是气缸活塞位置，不一定代表工件或机构真的到位。  
> 如果要确认机械夹具、挡块、工件位置，通常需要接近开关或光电传感器。

> [!warning] 🟧 判断规则
> **气缸到位 ≠ 工件一定到位**

## ② 参考表

| 类型 | 适合检测 |
|---|---|
| **电感式接近开关** | 金属件 |
| **电容式接近开关** | 非金属、液体等 |
| **光电传感器** | 工件有无、位置 |
| **磁性开关** | 气缸活塞位置 |
| **行程开关** | 机械接触到位 |
| **压力/真空开关** | 力或吸附状态 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出到位，但工件没有被推到最终位置。
- 公式：  
  **活塞到位 ≠ 工件到位**
- 计算：  
  气缸可能已经到位，但推板与工件之间存在间隙、工件卡滞或滑移。
- 结论：  
  应在工件目标位置增加 **接近开关或光电检测**。

> [!tip] 记忆口诀
> 气缸到位看磁开，工件到位看传感器。

# 10 PNP 与 NPN

> [!info] 核心结论
> PNP和NPN决定传感器输出电流方向，必须与PLC输入类型匹配，否则信号可能无法识别。

## ① 分类/公式/规则

> [!note] 🟦 快速理解
> **PNP**：输出正电，常见于源型输出。  
> **NPN**：输出负电，常见于漏型输出。  
> 实际接线必须按PLC输入模块类型确认。

> [!warning] 🟧 匹配规则
> **PLC输入类型 = 传感器输出类型匹配**

## ② 参考表

| 类型 | 常见理解 |
|---|---|
| **PNP** | 输出接近+24V |
| **NPN** | 输出接近0V |
| **三线制** | 棕+24V，蓝0V，黑输出常见 |
| **两线制** | 串入回路，注意漏电流 |
| **NO常开** | 检测到才导通 |
| **NC常闭** | 未检测时导通 |

## ③ 计算/选型示例

- 已知：  
  PLC输入模块要求PNP信号，现场传感器为NPN输出。
- 公式：  
  **类型不匹配 → PLC可能读不到信号**
- 计算：  
  NPN输出方式与PNP输入逻辑不一致。
- 结论：  
  应更换为 **PNP传感器** 或使用中间转换模块。

> [!tip] 记忆口诀
> PNP给正，NPN给负；接PLC前先看输入类型。

# 11 两线制与三线制

> [!info] 核心结论
> 两线制传感器接线简单，但可能存在漏电流；三线制信号更清晰，是自动化设备中常见选择。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 两线制像一个电子开关，串联在负载回路中。  
> 三线制有独立电源线和输出线，接线更清晰。

> [!warning] 🟧 选择规则
> **PLC输入稳定性要求高 → 三线制优先**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **两线制** | 接线少，注意漏电流 |
| **三线制** | 电源和输出分开 |
| **四线制** | 可能同时提供NO/NC |
| **模拟量型** | 输出连续压力/流量 |
| **IO-Link型** | 可读参数和诊断 |
| **带M8/M12接头** | 快速接线维护 |

## ③ 计算/选型示例

- 已知：  
  两线制磁性开关接入PLC后，输入点偶尔误亮。
- 公式：  
  **两线制漏电流 → 可能导致误触发**
- 计算：  
  漏电流经过PLC输入回路，可能让输入点不稳定。
- 结论：  
  可更换三线制传感器，或按厂家建议增加泄放电阻/适配模块。

> [!tip] 记忆口诀
> 两线省线，三线稳定；误触发先查漏电流。

# 12 NO 与 NC

> [!info] 核心结论
> NO表示常开，检测到目标才输出；NC表示常闭，正常未触发时输出，断线和异常更容易被发现。

## ① 分类/公式/规则

> [!note] 🟦 信号逻辑
> **NO常开**：平时OFF，检测到ON。  
> **NC常闭**：平时ON，检测到或异常时OFF。  
> 安全相关信号通常更偏向可诊断的逻辑设计。

> [!warning] 🟧 选择规则
> **普通到位检测 → NO常见**  
> **安全/断线监控 → NC或安全传感器需评估**

## ② 参考表

| 类型 | 状态 |
|---|---|
| **NO常开** | 未检测OFF，检测ON |
| **NC常闭** | 未检测ON，检测OFF |
| **互补输出** | 同时提供NO和NC |
| **断线风险** | NO断线可能像未检测 |
| **安全场景** | 需按安全规范设计 |
| **PLC逻辑** | 必须与程序一致 |

## ③ 计算/选型示例

- 已知：  
  程序要求气缸到位时输入点为ON。
- 公式：  
  **到位ON → NO输出更直观**
- 计算：  
  到位后传感器导通，PLC输入ON。
- 结论：  
  可选择 **NO常开型磁性开关**，但安全回路需另行评估。

> [!tip] 记忆口诀
> 常开到位才亮，常闭平时就亮。

# 13 安装与调试

> [!info] 核心结论
> 气动传感器调试重点是安装位置、检测距离、信号指示灯、线缆固定和PLC输入状态一致。

## ① 分类/公式/规则

> [!note] 🟦 调试步骤
> 先手动动作气缸，再观察传感器指示灯；  
> 再看PLC输入状态；  
> 最后固定传感器位置并做多次重复动作验证。

> [!warning] 🟧 调试原则
> **先看传感器灯，再看PLC输入，最后看程序逻辑**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **安装位置** | 对准检测区域 |
| **检测距离** | 不能超出范围 |
| **指示灯** | 现场判断信号 |
| **线缆固定** | 防止拉扯断线 |
| **防护等级** | 防水、防油、防尘 |
| **抗干扰** | 远离动力线和电磁阀线圈 |

## ③ 计算/选型示例

- 已知：  
  传感器指示灯亮，但PLC输入不亮。
- 公式：  
  **灯亮PLC不亮 = 接线错误 / 输入类型不匹配 / 输入点故障**
- 计算：  
  传感器本体已检测到目标，问题更可能在输出接线或PLC侧。
- 结论：  
  应检查PNP/NPN类型、公共端、PLC输入地址和输入模块供电。

> [!tip] 记忆口诀
> 灯亮查PLC，灯不亮查位置和电源。

# 14 常见故障排查

> [!info] 核心结论
> 气动传感器故障排查按“电源 → 指示灯 → 接线 → PLC输入 → 安装位置 → 被测对象”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先确认传感器供电；  
> 再看传感器指示灯是否随动作变化；  
> 再看PLC输入是否同步变化；  
> 最后检查安装位置和被测物状态。

> [!warning] 🟧 排错公式
> **传感器异常 = 无电源 + 类型不匹配 + 接线错误 + 位置偏移 + 线缆损坏 + PLC输入异常**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **完全无信号** | 无电源、接线断、传感器坏 |
| **指示灯亮PLC不亮** | PNP/NPN不匹配、公共端错 |
| **信号偶发丢失** | 安装松动、检测距离边缘 |
| **信号抖动** | 迟滞小、气压波动、振动 |
| **一直有信号** | 短路、常闭逻辑、位置错误 |
| **气缸到位无信号** | 磁开关位置偏、活塞未到位 |
| **真空误报警** | 泄漏、阈值设置不合理 |

## ③ 计算/选型示例

- 已知：  
  气缸动作正常，但到位信号偶尔丢失。
- 公式：  
  **偶发丢信号 = 检测点处于临界位置 / 振动 / 线缆松动**
- 计算：  
  磁性开关可能安装在检测边缘，气缸轻微波动就会失去信号。
- 结论：  
  应重新调整传感器位置，并固定线缆和开关本体。

> [!tip] 记忆口诀
> 偶发问题先查松动，边缘检测最容易抖。

# 15 气动传感器快速选型表

> [!info] 核心结论
> 气动传感器选型先明确检测对象，再匹配输出类型、PLC输入、安装方式和现场环境。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 明确检测对象：位置/压力/真空/流量  
> ② 确认开关量还是模拟量  
> ③ 选择PNP或NPN  
> ④ 确认NO或NC逻辑  
> ⑤ 确认两线制或三线制  
> ⑥ 确认电源电压  
> ⑦ 确认安装方式和尺寸  
> ⑧ 确认响应速度和精度  
> ⑨ 确认防护等级和环境  
> ⑩ 与PLC输入点和程序逻辑联调

> [!warning] 🟧 最终选型公式
> **推荐传感器 = 检测对象 + 输出方式 + 信号类型 + 安装结构 + 防护等级 + PLC匹配**

## ② 参考表

| 需求 | 推荐传感器 |
|---|---|
| 气缸伸缩到位 | 磁性开关 |
| 气源压力不足报警 | 压力开关 |
| 连续压力监测 | 压力传感器 |
| 吸盘吸附确认 | 真空开关 |
| 连续真空监测 | 真空传感器 |
| 耗气和泄漏监测 | 流量传感器 |
| 工件位置确认 | 接近/光电传感器 |

## ③ 计算/选型示例

- 已知：  
  双作用气缸需要检测伸出和缩回到位，PLC输入为PNP型，电源为DC24V。
- 公式：  
  **气缸两端到位 = 2个磁性开关**  
  **PLC为PNP输入 → 传感器选PNP输出**
- 计算：  
  需要DC24V、PNP输出、适配该气缸槽型的磁性开关2只。
- 结论：  
  推荐配置为 **DC24V PNP三线制磁性开关 × 2**，分别用于伸出到位和缩回到位检测。

> [!tip] 选型口诀
> 位置看磁开，压力看压开，真空看真空开；接PLC前，先分PNP和NPN。
