# 01 比例控制阀总览

> [!info] 核心结论
> 比例控制阀可以把 **电信号** 转换成连续可调的 **压力、流量或方向控制**，适合需要柔性调节的气动/液压系统。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 普通电磁阀多是 **开/关控制**；  
> 比例控制阀是 **连续调节控制**，输入信号越大，输出压力、流量或阀芯开度通常越大。

> [!warning] 🟧 通用选型公式
> **比例阀选型 = 控制对象 + 压力范围 + 流量范围 + 控制信号 + 精度要求 + 响应时间 + 介质条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制对象** | 压力、流量、方向、速度 |
| **常见信号** | 0-10V、4-20mA、0-5V、通讯控制 |
| **典型介质** | 压缩空气、液压油、水、惰性气体等 |
| **常见应用** | 张力、压紧、夹持、吹气、流量调节 |
| **优点** | 连续可调、可远程控制、自动化程度高 |
| **重点参数** | 压力范围、流量、精度、响应、迟滞、接口 |

## ③ 计算/选型示例

- 已知：  
  设备需要通过PLC模拟量控制气缸压紧力。
- 公式：  
  **压紧力 = 气压 × 气缸有效面积**
- 计算：  
  若需要连续调节压紧力，普通减压阀无法自动调节。
- 结论：  
  可选择 **比例压力阀**，由PLC输出模拟量控制压力，具体规格需按厂家手册校核。

> [!tip] 记忆口诀
> 电磁阀管开关，比例阀管大小。

# 02 底层原理

> [!info] 核心结论
> 比例控制阀的本质，是把输入电信号转换成阀芯位移或先导压力，再实现连续输出控制。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 控制器输出模拟量信号，阀内电子模块驱动比例电磁铁，比例电磁铁改变阀芯位置，最终改变压力或流量输出。

> [!warning] 🟧 控制链路
> **PLC模拟量 → 比例阀放大器 → 比例电磁铁 → 阀芯位移 → 压力/流量变化**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **控制信号** | 0-10V、4-20mA等 |
| **放大电路** | 把信号转换成线圈电流 |
| **比例电磁铁** | 产生可控电磁力 |
| **阀芯** | 改变通道开度 |
| **反馈传感器** | 闭环阀内部校正 |
| **输出端** | 输出压力或流量 |

## ③ 计算/选型示例

- 已知：  
  比例压力阀输入0-10V，对应输出0-0.8MPa。
- 公式：  
  **输出压力 = 输入电压 ÷ 10 × 满量程压力**
- 计算：  
  输入5V时，输出压力 = 5 ÷ 10 × 0.8 = **0.4MPa**
- 结论：  
  理论输出约为 **0.4MPa**，实际需考虑零点、线性度和厂家标定。

> [!tip] 记忆口诀
> 信号给多少，阀口开多少；开度变了，压力流量就变了。

# 03 常见类型

> [!info] 核心结论
> 比例控制阀按控制对象常分为 **比例压力阀、比例流量阀、比例方向阀**。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 比例压力阀管压力；  
> 比例流量阀管流量；  
> 比例方向阀管方向和速度；  
> 电液伺服阀精度更高，但成本和清洁度要求也更高。

> [!warning] 🟧 类型选择公式
> **要稳压力 → 比例压力阀**  
> **要调速度/流量 → 比例流量阀**  
> **要控制运动方向和速度 → 比例方向阀**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **比例压力阀** | 压紧力、夹持力、张力控制 |
| **比例流量阀** | 气体流量、速度调节 |
| **比例方向阀** | 液压缸方向和速度控制 |
| **比例减压阀** | 输出压力可调 |
| **比例溢流阀** | 系统压力限制 |
| **电液伺服阀** | 高精度、高响应闭环控制 |

## ③ 计算/选型示例

- 已知：  
  设备需要根据工件尺寸自动调整夹持压力。
- 公式：  
  **夹持压力可调 → 比例压力阀**
- 计算：  
  PLC根据工艺配方输出不同模拟量，比例阀输出不同压力。
- 结论：  
  应选择 **比例压力阀**，并校核压力范围、流量和响应时间。

> [!tip] 记忆口诀
> 压力用压阀，速度用流阀，方向速度一起管就看方向阀。

# 04 控制信号

> [!info] 核心结论
> 比例控制阀常用模拟量控制，最常见的是 **0-10V** 和 **4-20mA**。

## ① 分类/公式/规则

> [!note] 🟦 信号理解
> 0-10V接线简单，适合短距离控制；  
> 4-20mA抗干扰能力更强，适合较长距离或工业现场。

> [!warning] 🟧 信号换算公式
> **0-10V：输出 = 输入电压 ÷ 10 × 满量程**  
> **4-20mA：输出 =（输入电流 - 4）÷ 16 × 满量程**

## ② 参考表

| 信号类型 | 特点 |
|---|---|
| **0-10V** | 接线简单，常见 |
| **0-5V** | 小范围电压控制 |
| **4-20mA** | 抗干扰较好 |
| **PWM** | 部分阀或驱动模块使用 |
| **通讯控制** | 可读状态和参数 |
| **数字输入** | 用于使能、复位、模式切换 |

## ③ 计算/选型示例

- 已知：  
  4-20mA控制比例压力阀，满量程0.6MPa，当前输入12mA。
- 公式：  
  **压力 =（12 - 4）÷ 16 × 0.6**
- 计算：  
  压力 = 8 ÷ 16 × 0.6 = **0.3MPa**
- 结论：  
  当前理论输出压力约为 **0.3MPa**。

> [!tip] 记忆口诀
> 电压短线方便，电流长线抗干扰。

# 05 比例压力阀

> [!info] 核心结论
> 比例压力阀用于根据电信号连续调节输出压力，常用于压紧、夹持、张力和吹气压力控制。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 比例压力阀相当于“电控减压阀”。  
> 它可以由PLC或控制器远程改变压力，不需要人工旋钮调节。

> [!warning] 🟧 压力控制公式
> **输出压力 = 控制信号比例 × 压力满量程**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **压力范围** | 如0-0.5MPa、0-0.9MPa等 |
| **输入压力** | 必须高于目标输出压力 |
| **额定流量** | 决定供气能力 |
| **响应时间** | 影响调压速度 |
| **重复精度** | 影响压力一致性 |
| **排气能力** | 降压速度关键 |

## ③ 计算/选型示例

- 已知：  
  气缸有效面积为2000mm²，需要输出推力600N。
- 公式：  
  **F = P × A**
- 计算：  
  A = 2000mm² = 0.002m²  
  P = 600 ÷ 0.002 = **300000Pa = 0.3MPa**
- 结论：  
  比例压力阀需能稳定输出约 **0.3MPa**，并留有调节余量。

> [!tip] 记忆口诀
> 要控力，先控压；压力稳，推力才稳。

---

# 06 比例流量阀

> [!info] 核心结论
> 比例流量阀用于连续调节流量，常用于控制气缸速度、吹气量、冷却气量或液压执行器速度。

## ① 分类/公式/规则

> [!note] 🟦 核心作用
> 流量越大，执行器速度通常越快；  
> 流量越小，动作越慢。  
> 但实际速度还受负载、压力、管径和阀口特性影响。

> [!warning] 🟧 基本关系
> **执行器速度 ∝ 流量 ÷ 有效面积**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **额定流量** | 最大流量能力 |
| **控制范围** | 最小到最大可调范围 |
| **阀口特性** | 线性、等百分比等 |
| **压差要求** | 流量与压差相关 |
| **响应时间** | 流量变化速度 |
| **介质洁净度** | 防止阀芯卡滞 |

## ③ 计算/选型示例

- 已知：  
  气缸有效面积为0.001m²，希望活塞速度0.2m/s。
- 公式：  
  **流量 Q = A × v**
- 计算：  
  Q = 0.001 × 0.2 = **0.0002m³/s**  
  换算约为 **12L/min**
- 结论：  
  理论流量约12L/min，实际需考虑压缩空气状态、管路损失和厂家流量曲线。

> [!tip] 记忆口诀
> 流量管速度，压力管力气。

# 07 开环与闭环比例阀

> [!info] 核心结论
> 开环比例阀只按信号动作，闭环比例阀带内部反馈，输出更稳定、精度更高。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 开环比例阀：控制简单，成本较低，但受负载和压力波动影响较大。  
> 闭环比例阀：内部带压力、流量或阀芯位置反馈，可自动修正输出。

> [!warning] 🟧 选择规则
> **要求不高 → 开环可用**  
> **精度高 / 波动小 / 重复性好 → 闭环优先**

## ② 参考表

| 对比项 | 开环比例阀 | 闭环比例阀 |
|---|---|---|
| 反馈 | 无或少 | **有反馈** |
| 成本 | 较低 | 较高 |
| 精度 | 一般 | **较高** |
| 抗扰动 | 一般 | **较强** |
| 调试 | 简单 | 略复杂 |
| 应用 | 普通调节 | 精密压力/流量控制 |

## ③ 计算/选型示例

- 已知：  
  设备要求夹持压力重复性高，压力波动不能太大。
- 公式：  
  **高重复性压力控制 → 闭环比例压力阀优先**
- 计算：  
  闭环阀能读取实际压力并自动修正阀口输出。
- 结论：  
  应选择 **带压力反馈的闭环比例压力阀**。

> [!tip] 记忆口诀
> 开环听命令，闭环看结果。

# 08 PLC模拟量控制

> [!info] 核心结论
> PLC控制比例阀时，要确认模拟量类型、量程、接线方式、公共端和屏蔽接地。

## ① 分类/公式/规则

> [!note] 🟦 控制链路
> PLC模拟量输出给比例阀，比例阀根据输入信号调节压力或流量。  
> 同时可用压力传感器把实际值反馈给PLC，形成系统闭环。

> [!warning] 🟧 PLC控制公式
> **PLC输出值 → 模拟量模块 → 比例阀输入 → 压力/流量输出**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **AO类型** | 电压型或电流型 |
| **量程** | 0-10V、4-20mA等 |
| **公共端** | COM接线正确 |
| **屏蔽线** | 降低模拟量干扰 |
| **电源** | 24V供电需稳定 |
| **反馈信号** | 压力/流量传感器输入PLC |

## ③ 计算/选型示例

- 已知：  
  PLC模拟量范围0-27648，对应比例阀0-10V。
- 公式：  
  **输出电压 = 数字量 ÷ 27648 × 10**
- 计算：  
  数字量13824时，输出电压 = 13824 ÷ 27648 × 10 = **5V**
- 结论：  
  比例阀接收到约5V信号，对应约50%输出。

> [!tip] 记忆口诀
> 量程要对应，公共端别接错；模拟量怕干扰，屏蔽接地要做好。

# 09 响应时间与滞后

> [!info] 核心结论
> 比例阀不是信号一变输出立刻到位，响应时间、迟滞和重复精度都会影响控制效果。

## ① 分类/公式/规则

> [!note] 🟦 动态特性
> 响应时间决定调节速度；  
> 迟滞表示同一信号从上升和下降方向接近时，输出可能不完全一致；  
> 重复精度决定多次调节的一致性。

> [!warning] 🟧 动态选择规则
> **快速控制 → 看响应时间**  
> **精密控制 → 看线性度、迟滞、重复精度**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **响应时间** | 输出变化快慢 |
| **线性度** | 信号与输出是否成比例 |
| **迟滞** | 上升/下降输出差异 |
| **重复精度** | 多次输出一致性 |
| **分辨率** | 最小可调变化 |
| **温漂** | 温度变化造成偏差 |

## ③ 计算/选型示例

- 已知：  
  设备需要快速改变压力，节拍很短。
- 公式：  
  **节拍短 → 响应时间要小**
- 计算：  
  若比例阀响应慢，压力还未稳定，设备动作已经结束。
- 结论：  
  需选择响应更快的比例阀，并优化管路容积和传感器位置。

> [!tip] 记忆口诀
> 慢系统看精度，快系统先看响应。

# 10 气源与过滤要求

> [!info] 核心结论
> 比例阀对气源洁净度和稳定性要求较高，前端必须做好过滤、干燥和稳压。

## ① 分类/公式/规则

> [!note] 🟦 气源要求
> 水分、油污、粉尘会导致阀芯卡滞、输出漂移、响应变慢。  
> 比例阀前端通常建议配置过滤减压和精密过滤，具体按厂家手册校核。

> [!warning] 🟧 气源处理公式
> **比例阀稳定 = 气源压力稳定 + 空气洁净干燥 + 流量足够**

## ② 参考表

| 项目 | 要求 |
|---|---|
| **过滤器** | 去除颗粒和水滴 |
| **精密过滤器** | 保护比例阀阀芯 |
| **干燥器** | 降低水分 |
| **输入压力** | 高于目标输出压力 |
| **供气流量** | 不足会掉压 |
| **管路容积** | 影响响应速度 |

## ③ 计算/选型示例

- 已知：  
  比例阀输出压力不稳定，前端过滤杯有积水。
- 公式：  
  **水分进入比例阀 → 阀芯卡滞 / 输出波动**
- 计算：  
  压缩空气含水会影响阀芯和压力传感器稳定性。
- 结论：  
  应检查干燥器、过滤器、自动排水器和精密过滤配置。

> [!tip] 记忆口诀
> 比例阀要稳，气源先要干净稳。

# 11 安装与接线

> [!info] 核心结论
> 比例阀安装时要注意气流方向、供电电压、模拟量极性、接地屏蔽和散热空间。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 阀体通常有入口P、出口A、排气R等标识。  
> 接反气口会导致输出异常；模拟量线干扰会导致压力或流量抖动。

> [!warning] 🟧 安装接线规则
> **气路按标识接，电路按手册接，模拟量用屏蔽线**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **P口** | 供气入口 |
| **A口** | 控制输出 |
| **R口** | 排气口 |
| **24V电源** | 电压稳定、容量足够 |
| **模拟量线** | 屏蔽、远离动力线 |
| **接地** | 按厂家要求处理 |

## ③ 计算/选型示例

- 已知：  
  比例阀压力输出随变频器启动发生波动。
- 公式：  
  **模拟量波动 = 干扰 + 接地不良 + 线缆布线不当**
- 计算：  
  模拟量线若与动力线长距离并行，容易受到干扰。
- 结论：  
  应分开布线，使用屏蔽线，并检查接地和模拟量滤波参数。

> [!tip] 记忆口诀
> 气口别接反，信号线别乱走。

# 12 PID闭环控制

> [!info] 核心结论
> 当只靠比例阀内部控制不能满足精度时，可用PLC读取压力/流量反馈，做PID闭环控制。

## ① 分类/公式/规则

> [!note] 🟦 闭环逻辑
> PLC设定目标压力，压力传感器反馈实际压力，PLC计算偏差后调整比例阀输出，使实际值接近目标值。

> [!warning] 🟧 闭环公式
> **控制偏差 = 目标值 - 实际反馈值**

## ② 参考表

| PID项 | 作用 |
|---|---|
| **P比例** | 偏差越大，调整越大 |
| **I积分** | 消除静态误差 |
| **D微分** | 抑制变化过快 |
| **设定值SP** | 目标压力/流量 |
| **反馈值PV** | 实际压力/流量 |
| **输出MV** | 给比例阀的控制信号 |

## ③ 计算/选型示例

- 已知：  
  目标压力0.5MPa，实际压力0.46MPa。
- 公式：  
  **偏差 = 0.5 - 0.46**
- 计算：  
  偏差 = **0.04MPa**
- 结论：  
  PLC可根据PID算法增加比例阀输出，使压力接近0.5MPa。

> [!tip] 记忆口诀
> 有反馈才知道偏差，有PID才会自动修正。

# 13 常见故障排查

> [!info] 核心结论
> 比例阀故障排查按“电源 → 信号 → 气源 → 输出 → 反馈 → 阀体”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先量24V电源，再量模拟量信号；  
> 先看输入压力，再看输出压力；  
> 先查气源洁净度，再怀疑阀芯损坏。

> [!warning] 🟧 排错公式
> **比例阀异常 = 电源异常 + 信号错误 + 气源不足 + 阀芯卡滞 + 反馈故障 + 参数错误**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **无输出** | 无电源、无信号、未使能、气源无压 |
| **输出偏低** | 输入压力不足、量程设置错、流量不足 |
| **输出波动** | 气源不稳、模拟量干扰、反馈异常 |
| **响应慢** | 管路容积大、滤芯堵塞、阀规格小 |
| **无法降压** | 排气口堵塞、排气能力不足 |
| **压力漂移** | 温漂、污染、传感器异常 |
| **阀发热** | 线圈长期工作，需按手册判断 |

## ③ 计算/选型示例

- 已知：  
  PLC给定5V，但比例压力阀无压力输出。
- 公式：  
  **无输出 = 电源/信号/气源/使能任一异常**
- 计算：  
  先确认24V供电，再确认5V信号实际到达阀端，最后检查P口供气压力。
- 结论：  
  不要只看PLC程序，应在阀端实测电压、电流和气压。

> [!tip] 记忆口诀
> 比例阀不动，先量电，再看气。

# 14 比例控制阀快速选型表

> [!info] 核心结论
> 比例阀选型先明确要控制压力、流量还是方向，再按量程、流量、信号、精度和响应时间校核。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 明确控制对象：压力/流量/方向  
> ② 确认介质：空气/油/水/气体  
> ③ 确认最大压力和调节范围  
> ④ 确认最大流量  
> ⑤ 选择控制信号  
> ⑥ 确认开环或闭环  
> ⑦ 校核响应时间和精度  
> ⑧ 校核接口尺寸和安装方式  
> ⑨ 配置过滤、干燥和稳压  
> ⑩ 按厂家曲线校核压降和稳定性

> [!warning] 🟧 最终选型公式
> **推荐比例阀 = 控制类型 + 压力范围 + 流量范围 + 输入信号 + 精度等级 + 响应时间 + 接口尺寸**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 控制压紧力 | 比例压力阀 |
| 控制吹气量 | 比例流量阀 |
| 液压缸速度方向 | 比例方向阀 |
| 精密压力稳定 | 闭环比例压力阀 |
| 长距离控制 | 4-20mA优先 |
| 快速节拍 | 响应时间、管路容积 |
| 易污染现场 | 精密过滤、干燥处理 |

## ③ 计算/选型示例

- 已知：  
  需要0-0.6MPa连续调压，PLC输出0-10V，最大用气量300L/min。
- 公式：  
  **阀需求 = 0-0.6MPa + 0-10V + 流量≥300L/min×余量**
- 计算：  
  若取1.3倍余量：300 × 1.3 = **390L/min**
- 结论：  
  应选择压力范围覆盖0-0.6MPa、0-10V控制、额定流量不小于 **390L/min** 的比例压力阀，并按厂家曲线校核。

> [!tip] 选型口诀
> 先定控什么，再定量程；压力看MPa，流量看L/min；信号要匹配，气源要干净。
