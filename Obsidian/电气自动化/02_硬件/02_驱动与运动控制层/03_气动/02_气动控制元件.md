# 01 气动控制元件总览

> [!info] 核心结论
> 气动控制元件用于控制压缩空气的 **方向、压力、流量和动作顺序**，是气缸、气爪、真空元件实现自动动作的核心。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 气源处理元件负责“把气处理好”；  
> 气动控制元件负责“让气怎么走、走多少、什么时候走”。

> [!warning] 🟧 通用选型公式
> **气动控制元件选型 = 控制对象 + 工作压力 + 流量需求 + 阀位阀通 + 控制方式 + 接口尺寸 + 响应要求**

## ② 参考表

| 类别 | 主要作用 |
|---|---|
| **方向控制阀** | 控制气路通断和换向 |
| **电磁阀** | PLC控制气缸动作 |
| **手动阀** | 人工控制气路 |
| **机械阀** | 由机械碰撞触发 |
| **流量控制阀** | 调节速度 |
| **压力控制阀** | 调节或限制压力 |
| **单向阀/梭阀** | 控制气流逻辑 |

## ③ 计算/选型示例

- 已知：  
  一个双作用气缸需要PLC控制伸出和缩回。
- 公式：  
  **双作用气缸 → 需要换向控制**
- 计算：  
  双作用气缸有两个气口，需要控制两侧进排气。
- 结论：  
  可选择 **5/2电磁换向阀** 或 **5/3电磁换向阀**，具体流量和接口按气缸速度校核。

> [!tip] 记忆口诀
> 方向阀管动作，流量阀管速度，压力阀管力气。

# 02 底层原理

> [!info] 核心结论
> 气动控制的本质，是通过阀芯切换气路，让压缩空气进入或排出执行元件。

## ① 分类/公式/规则

> [!note] 🟦 工作链路
> PLC输出信号，电磁阀线圈得电，阀芯换位，压缩空气进入气缸一侧，另一侧排气，气缸完成伸出或缩回。

> [!warning] 🟧 控制链路
> **PLC信号 → 电磁阀动作 → 气路切换 → 气缸运动 → 磁性开关反馈**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **PLC输出** | 控制电磁阀线圈 |
| **线圈** | 产生电磁力 |
| **阀芯** | 改变气路连接 |
| **P口** | 进气口 |
| **A/B口** | 接气缸工作口 |
| **R/S口** | 排气口 |

## ③ 计算/选型示例

- 已知：  
  电磁阀线圈得电后，气缸没有动作。
- 公式：  
  **有电不动作 = 气源/阀芯/接管/气缸任一异常**
- 计算：  
  需检查P口是否有气、A/B口是否接反、排气口是否堵塞。
- 结论：  
  气动排查不能只看电信号，还要看气路是否真正切换。

> [!tip] 记忆口诀
> 电控阀芯动，阀芯改气路，气路推气缸。

# 03 方向控制阀

> [!info] 核心结论
> 方向控制阀用于控制气流方向，是气缸伸出、缩回、停止的核心元件。

## ① 分类/公式/规则

> [!note] 🟦 阀位阀通
> “几位”表示阀芯有几个工作位置；  
> “几通”表示阀有几个气口。  
> 例如 **5/2阀** 表示5个气口、2个位置。

> [!warning] 🟧 选择规则
> **单作用气缸 → 3/2阀**  
> **双作用气缸 → 5/2阀或5/3阀**

## ② 参考表

| 阀型 | 常见用途 |
|---|---|
| **2/2阀** | 简单通断 |
| **3/2阀** | 单作用气缸 |
| **4/2阀** | 双作用气缸换向 |
| **5/2阀** | 双作用气缸常用 |
| **5/3阀** | 中位停止/泄压/封闭 |
| **二位阀** | 两种状态切换 |

## ③ 计算/选型示例

- 已知：  
  气缸为单作用弹簧复位气缸。
- 公式：  
  **单作用气缸 = 一个工作口 + 一个排气口**
- 计算：  
  只需控制进气和排气，不需要A/B两路换向。
- 结论：  
  可选择 **3/2方向控制阀**。

> [!tip] 记忆口诀
> 单作用用三二，双作用选五二；要中位功能，看五三。

# 04 电磁阀

> [!info] 核心结论
> 电磁阀是最常用的气动控制元件，用PLC电信号控制气路通断或换向。

## ① 分类/公式/规则

> [!note] 🟦 核心特点
> 电磁阀由线圈和阀体组成。  
> 线圈得电后产生电磁力，推动阀芯动作，实现气路切换。

> [!warning] 🟧 电磁阀选型公式
> **电磁阀选型 = 电压 + 阀位阀通 + 流量 + 接口 + 复位方式 + 控制方式**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **线圈电压** | DC24V、AC220V等 |
| **阀位阀通** | 3/2、5/2、5/3等 |
| **复位方式** | 单电控弹簧复位、双电控 |
| **接口尺寸** | M5、G1/8、G1/4等 |
| **额定流量** | 决定气缸速度 |
| **手动按钮** | 调试时手动切换 |

## ③ 计算/选型示例

- 已知：  
  PLC输出为DC24V，需要控制双作用气缸。
- 公式：  
  **PLC输出电压 = 电磁阀线圈电压**
- 计算：  
  应选择DC24V线圈的5/2电磁阀。
- 结论：  
  若选成AC220V线圈，PLC无法直接驱动，甚至可能损坏输出模块。

> [!tip] 记忆口诀
> 选电磁阀先看线圈电压，再看阀位阀通。

# 05 单电控与双电控

> [!info] 核心结论
> 单电控断电后会自动复位，双电控具有记忆功能，断电后阀位可能保持。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 单电控：一个线圈控制动作，断电靠弹簧复位。  
> 双电控：两个线圈分别控制两个方向，阀芯位置可保持。

> [!warning] 🟧 选择规则
> **需要断电复位 → 单电控**  
> **需要状态保持 → 双电控**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **单电控** | 断电自动复位 |
| **双电控** | 有记忆功能 |
| **弹簧复位** | 安全状态明确 |
| **气控复位** | 由气压控制 |
| **适合安全回位** | 单电控更常用 |
| **适合保持状态** | 双电控更合适 |

## ③ 计算/选型示例

- 已知：  
  设备急停后要求气缸自动退回安全位置。
- 公式：  
  **断电回安全位 → 单电控弹簧复位**
- 计算：  
  急停切断输出后，弹簧推动阀芯复位，气缸回到设定状态。
- 结论：  
  应优先选择 **单电控电磁阀**，并校核失电状态是否符合安全要求。

> [!tip] 记忆口诀
> 单电控会回家，双电控会记住。

# 06 流量控制阀

> [!info] 核心结论
> 流量控制阀用于调节气缸动作速度，常见形式是单向节流阀。

## ① 分类/公式/规则

> [!note] 🟦 调速逻辑
> 气缸速度主要由进排气流量决定。  
> 流量越大，速度越快；流量越小，速度越慢。

> [!warning] 🟧 基本关系
> **气缸速度 ∝ 流量 ÷ 有效面积**

## ② 参考表

| 类型 | 作用 |
|---|---|
| **节流阀** | 限制气流 |
| **单向节流阀** | 一个方向节流，反向自由流通 |
| **速度控制阀** | 常用于气缸调速 |
| **排气节流** | 动作更稳定 |
| **进气节流** | 简单但稳定性较差 |
| **精密节流阀** | 更细调节 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出速度太快，冲击明显。
- 公式：  
  **降低速度 = 减小排气流量**
- 计算：  
  适当调小排气侧单向节流阀，可降低伸出速度。
- 结论：  
  建议优先采用 **排气节流调速**，动作更平稳。

> [!tip] 记忆口诀
> 调气缸速度，优先调排气。

# 07 单向阀与快速排气阀

> [!info] 核心结论
> 单向阀控制气流只能单向通过，快速排气阀用于让气缸腔体就近排气，提高动作速度。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 单向阀用于防止气体倒流；  
> 快速排气阀把气缸内空气直接从附近排出，不必返回电磁阀排气口。

> [!warning] 🟧 应用规则
> **防倒流 → 单向阀**  
> **要加快动作 → 快速排气阀**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **单向阀** | 防止反向流动 |
| **液控/气控单向阀** | 控制开启条件 |
| **快速排气阀** | 缩短排气路径 |
| **梭阀** | 两路气源取高压 |
| **双压阀** | 两路同时有气才输出 |
| **止回功能** | 保压、防回流 |

## ③ 计算/选型示例

- 已知：  
  大缸径气缸缩回速度慢，排气管路较长。
- 公式：  
  **排气路径长 → 排气阻力大 → 动作慢**
- 计算：  
  在气缸口附近加快速排气阀，可缩短排气路径。
- 结论：  
  可配置 **快速排气阀**，但需校核动作冲击和噪声。

> [!tip] 记忆口诀
> 单向阀防倒流，快排阀提速度。

# 08 压力控制元件

> [!info] 核心结论
> 压力控制元件用于调节、限制或检测气压，影响气缸推力、夹持力和系统安全。

## ① 分类/公式/规则

> [!note] 🟦 压力与力
> 气缸输出力与压力成正比。  
> 压力越高，理论推力越大；压力过高则可能损坏元件或造成冲击。

> [!warning] 🟧 气缸推力公式
> **F = P × A**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **减压阀** | 调节工作压力 |
| **溢流阀** | 限制最高压力 |
| **顺序阀** | 达到压力后动作 |
| **压力开关** | 气压信号反馈PLC |
| **比例压力阀** | 电控连续调压 |
| **压力表** | 现场显示压力 |

## ③ 计算/选型示例

- 已知：  
  气缸有效面积0.001m²，工作压力0.5MPa。
- 公式：  
  **F = P × A**
- 计算：  
  F = 0.5 × 10⁶ × 0.001 = **500N**
- 结论：  
  理论推力约为 **500N**，实际需考虑摩擦和效率。

> [!tip] 记忆口诀
> 压力决定力气，流量决定速度。

# 09 机械阀与手动阀

> [!info] 核心结论
> 机械阀和手动阀不依赖PLC输出，常用于调试、人工操作、机械到位触发和简单气动逻辑。

## ① 分类/公式/规则

> [!note] 🟦 使用场景
> 手动阀适合人工控制气路；  
> 机械阀适合由滑块、挡块、凸轮等机械结构触发；  
> 脚踏阀适合人工操作设备。

> [!warning] 🟧 选择规则
> **人来操作 → 手动阀/脚踏阀**  
> **机构碰到才动作 → 机械阀**

## ② 参考表

| 类型 | 应用 |
|---|---|
| **按钮阀** | 手动点动 |
| **旋钮阀** | 手动切换 |
| **脚踏阀** | 脚踏控制 |
| **滚轮阀** | 机械到位触发 |
| **柱塞阀** | 直接按压触发 |
| **拉杆阀** | 机械拉动控制 |

## ③ 计算/选型示例

- 已知：  
  设备调试时需要人工点动气缸伸出。
- 公式：  
  **调试点动 → 手动控制阀**
- 计算：  
  手动阀可绕开PLC程序，方便检查气缸和气路。
- 结论：  
  可在调试工位配置 **手动按钮阀** 或使用电磁阀手动按钮。

> [!tip] 记忆口诀
> 人控用手动，碰到用机械。

# 10 气动逻辑元件

> [!info] 核心结论
> 气动逻辑元件可以实现“或、与、延时、顺序”等气路逻辑，适合简单气动自动控制。

## ① 分类/公式/规则

> [!note] 🟦 逻辑理解
> 梭阀类似“或门”：两路有一路有气就输出。  
> 双压阀类似“与门”：两路同时有气才输出。

> [!warning] 🟧 气动逻辑规则
> **梭阀 = 或逻辑**  
> **双压阀 = 与逻辑**  
> **延时阀 = 延时动作**

## ② 参考表

| 元件 | 逻辑作用 |
|---|---|
| **梭阀** | OR，任一路有气输出 |
| **双压阀** | AND，两路都有气输出 |
| **延时阀** | 延迟输出 |
| **顺序阀** | 达到压力后动作 |
| **气控阀** | 用气信号控制换向 |
| **记忆阀** | 保持状态 |

## ③ 计算/选型示例

- 已知：  
  设备要求两个按钮同时按下，气缸才能动作。
- 公式：  
  **双手同时信号 = 与逻辑**
- 计算：  
  两路气信号同时进入双压阀，才会输出控制气。
- 结论：  
  可使用 **双压阀** 实现简单气动双手控制，安全要求需按规范校核。

> [!tip] 记忆口诀
> 梭阀像“或”，双压像“与”。

# 11 阀岛与集成控制

> [!info] 核心结论
> 阀岛把多个电磁阀集中安装，并通过多芯线或总线控制，适合多气缸自动化设备。

## ① 分类/公式/规则

> [!note] 🟦 阀岛优势
> 相比单个电磁阀分散安装，阀岛接线更少、维护更集中、扩展更方便。  
> 常见控制方式包括多针接口、IO-Link、PROFINET、EtherCAT等。

> [!warning] 🟧 阀岛选型公式
> **阀岛选型 = 阀位数量 + 阀型 + 流量 + 通讯方式 + 电源 + 扩展需求**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **阀位数量** | 决定可控气缸数量 |
| **阀型组合** | 5/2、5/3、3/2等 |
| **控制方式** | 多芯线、总线、IO-Link |
| **电源** | 阀电源和逻辑电源 |
| **手动操作** | 调试用 |
| **诊断功能** | 断线、短路、通讯报警 |

## ③ 计算/选型示例

- 已知：  
  设备有12个气缸，需要PLC集中控制。
- 公式：  
  **多气缸集中控制 → 阀岛优先**
- 计算：  
  使用单阀接线复杂，阀岛可减少接线并便于维护。
- 结论：  
  可选择 **12位以上阀岛**，并按PLC通讯协议和阀流量校核。

> [!tip] 记忆口诀
> 气缸多，别散装；用阀岛，线少好维护。

# 12 接线与PLC控制

> [!info] 核心结论
> 气动控制元件接PLC时，要确认线圈电压、输出类型、公共端、保护二极管和手动调试方式。

## ① 分类/公式/规则

> [!note] 🟦 接线重点
> DC24V电磁阀常由PLC晶体管输出控制。  
> 线圈是感性负载，断电时会产生反向电压，通常需要浪涌吸收保护。

> [!warning] 🟧 接线匹配公式
> **PLC可控 = 输出类型匹配 + 线圈电压匹配 + 公共端正确 + 电流不超限**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **线圈电压** | DC24V、AC220V等 |
| **输出类型** | PNP/NPN/继电器输出 |
| **公共端COM** | 接错会不动作 |
| **线圈电流** | 不得超过PLC输出能力 |
| **浪涌保护** | 防止反向电压 |
| **指示灯** | 判断线圈是否得电 |

## ③ 计算/选型示例

- 已知：  
  电磁阀线圈电流为0.12A，PLC输出点最大允许0.5A。
- 公式：  
  **输出点电流 ≥ 线圈电流**
- 计算：  
  0.5A ≥ 0.12A，单个输出点电流满足要求。
- 结论：  
  可直接驱动该电磁阀，但多个阀同时动作时还需校核模块总电流。

> [!tip] 记忆口诀
> 阀不动作先看灯，灯不亮查输出和公共端。

# 13 安装与消声

> [!info] 核心结论
> 气动控制元件安装要注意气流方向、排气通畅、消声器、防尘和检修空间。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 阀体通常标有P、A、B、R、S口。  
> P接气源，A/B接执行元件，R/S为排气口。  
> 排气口不能堵死，否则气缸动作会变慢或异常。

> [!warning] 🟧 安装规则
> **P口接气源，A/B接气缸，R/S接消声器或排气管**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **气口标识** | P、A、B、R、S |
| **消声器** | 降低排气噪声 |
| **排气通畅** | 防止动作变慢 |
| **防尘防水** | 保护阀芯和线圈 |
| **安装方向** | 按厂家要求 |
| **维修空间** | 方便换线圈和阀体 |

## ③ 计算/选型示例

- 已知：  
  气缸缩回速度突然变慢，排气口装了堵塞的消声器。
- 公式：  
  **排气不畅 → 气缸动作变慢**
- 计算：  
  排气受阻会让气缸另一侧压力释放变慢。
- 结论：  
  应清理或更换消声器，并检查排气口是否堵塞。

> [!tip] 记忆口诀
> 进气要够，排气要通；消声器堵了，气缸就慢了。

# 14 常见故障排查

> [!info] 核心结论
> 气动控制故障要按“电信号 → 阀动作 → 气源 → 管路 → 执行元件”的顺序排查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看PLC输出和阀灯；  
> 再按手动按钮试阀；  
> 再查P口气压、A/B口输出和排气口；  
> 最后查气缸、气管和机械负载。

> [!warning] 🟧 排错公式
> **气动不动作 = 无电 + 无气 + 阀芯卡滞 + 管路接错 + 气缸卡滞 + 负载过大**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **阀灯不亮** | PLC无输出、公共端错误 |
| **灯亮不动作** | 线圈坏、阀芯卡、无气源 |
| **气缸反向动作** | A/B管接反 |
| **动作慢** | 流量不足、节流太小、消声器堵 |
| **气缸无力** | 压力低、漏气、负载大 |
| **阀漏气** | 密封损坏、杂质卡阀 |
| **动作不稳定** | 气源波动、管路积水、阀污染 |

## ③ 计算/选型示例

- 已知：  
  电磁阀指示灯亮，但气缸不动作。
- 公式：  
  **灯亮不动作 = 线圈/阀芯/气源/气缸问题**
- 计算：  
  可先手动按阀，如果仍不动作，检查P口供气和阀芯是否卡滞。
- 结论：  
  不要只判断PLC程序，应同时检查气源和阀体。

> [!tip] 记忆口诀
> 灯不亮查电，灯亮不动查气和阀。

# 15 气动控制元件快速选型表

> [!info] 核心结论
> 气动控制元件选型先明确控制对象，再确定阀型、流量、接口、控制电压和安装方式。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看控制对象：气缸/气爪/真空/气路  
> ② 看单作用还是双作用  
> ③ 选阀位阀通：3/2、5/2、5/3  
> ④ 看控制方式：电磁/手动/机械/气控  
> ⑤ 看线圈电压和PLC输出  
> ⑥ 看接口尺寸和管径  
> ⑦ 校核额定流量  
> ⑧ 看复位方式：单电控/双电控  
> ⑨ 配调速阀、消声器、单向阀  
> ⑩ 校核环境、防护和维护空间

> [!warning] 🟧 最终选型公式
> **推荐元件 = 阀型 + 控制方式 + 电压 + 流量 + 接口尺寸 + 复位方式 + 附件**

## ② 参考表

| 需求 | 推荐元件 |
|---|---|
| 单作用气缸 | 3/2阀 |
| 双作用气缸 | 5/2阀 |
| 中位停止 | 5/3阀 |
| PLC控制 | 电磁阀 |
| 人工控制 | 手动阀/脚踏阀 |
| 速度调节 | 单向节流阀 |
| 快速动作 | 快速排气阀 |
| 多气缸集中控制 | 阀岛 |

## ③ 计算/选型示例

- 已知：  
  双作用气缸，PLC DC24V控制，要求断电后气缸缩回，接口G1/4。
- 公式：  
  **双作用 → 5/2阀**  
  **断电复位 → 单电控弹簧复位**  
  **PLC DC24V → DC24V线圈**
- 计算：  
  选择5/2单电控电磁阀，DC24V线圈，G1/4接口。
- 结论：  
  推荐配置为 **DC24V 5/2单电控电磁阀 + 单向节流阀 + 消声器**，流量需按气缸速度校核。

> [!tip] 选型口诀
> 先看气缸单双，再选五二三二；PLC控看电压，动作快看流量，速度调节靠节流。
