# 01 集成气动系统总览

> [!info] 核心结论
> 集成气动系统是把 **气源处理、控制阀、执行元件、传感器、管路、通讯和安全保护** 集成在一起的完整气动解决方案。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 单个气缸、电磁阀、传感器只是零件；  
> 集成气动系统关注的是 **整套气路如何稳定、快速、安全、可维护地运行**。

> [!warning] 🟧 通用选型公式
> **集成气动系统选型 = 用气需求 + 动作数量 + 控制方式 + 阀岛/管路 + 传感反馈 + 安全联锁 + 维护方式**

## ② 参考表

| 模块 | 作用 |
|---|---|
| **气源处理** | 过滤、减压、干燥、排水 |
| **控制模块** | 电磁阀、阀岛、比例阀 |
| **执行模块** | 气缸、气爪、旋转缸、吸盘 |
| **检测模块** | 磁性开关、压力/真空开关 |
| **管路模块** | 气管、接头、分气块、消声器 |
| **通讯模块** | IO-Link、PROFINET、EtherCAT等 |
| **安全模块** | 残压释放、急停、压力监控 |

## ③ 计算/选型示例

- 已知：  
  一台设备有12个气缸、4个真空吸盘，需要PLC集中控制。
- 公式：  
  **多执行元件 + 集中控制 = 阀岛/集成气动方案优先**
- 计算：  
  如果每个电磁阀分散安装，接线和维护复杂；阀岛可集中供气和集中控制。
- 结论：  
  可选择 **气源处理 + 阀岛 + 气缸/真空回路 + 传感器反馈 + PLC通讯** 的集成方案。

> [!tip] 记忆口诀
> 单件看功能，系统看稳定；气路、电路、反馈、安全一起算。

# 02 系统组成

> [!info] 核心结论
> 一套完整集成气动系统，通常由 **气源、处理、控制、执行、检测、管路、通讯、安全** 八部分组成。

## ① 分类/公式/规则

> [!note] 🟦 系统链路
> 空压机提供压缩空气，气源处理保证空气质量，阀岛控制气路方向，气缸/气爪执行动作，传感器把状态反馈给PLC。

> [!warning] 🟧 标准链路
> **空压机 → 气源处理 → 阀岛/电磁阀 → 执行元件 → 传感器反馈 → PLC判断**

## ② 参考表

| 系统部分 | 典型元件 |
|---|---|
| **供气源** | 空压机、储气罐、主管路 |
| **处理单元** | 过滤器、减压阀、干燥器 |
| **控制单元** | 电磁阀、阀岛、比例阀 |
| **执行单元** | 气缸、气爪、真空吸盘 |
| **反馈单元** | 磁开、压力开关、真空开关 |
| **连接单元** | 气管、接头、分气块 |
| **控制单元** | PLC、远程IO、总线模块 |

## ③ 计算/选型示例

- 已知：  
  气缸动作无力，PLC输出正常。
- 公式：  
  **动作异常 = 供气 + 控制 + 执行 + 反馈任一环节异常**
- 计算：  
  PLC输出正常，不代表气源压力、阀岛流量、管路和气缸都正常。
- 结论：  
  应按系统链路逐段排查：气源压力 → 阀岛输出 → 管路 → 气缸 → 传感器。

> [!tip] 记忆口诀
> 气从哪里来，怎么走，推谁动，谁反馈，一条链路查到底。

# 03 气源集中处理

> [!info] 核心结论
> 集成气动系统入口必须先处理气源，保证空气 **干净、干燥、压力稳定、流量充足**。

## ① 分类/公式/规则

> [!note] 🟦 处理重点
> 气源处理不是可有可无。  
> 水分、油污、粉尘和压力波动会导致阀卡滞、气缸磨损、比例阀漂移、真空吸附失败。

> [!warning] 🟧 气源配置公式
> **气源处理 = 过滤 + 减压 + 干燥 + 排水 + 压力监控**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **过滤器** | 去水、去杂质 |
| **减压阀** | 稳定工作压力 |
| **干燥器** | 降低空气含水量 |
| **自动排水器** | 排出冷凝水 |
| **压力开关** | 低压报警/联锁 |
| **精密过滤器** | 保护比例阀、精密阀 |

## ③ 计算/选型示例

- 已知：  
  系统最大用气量为1000L/min，取1.3倍余量。
- 公式：  
  **气源处理流量 ≥ 最大用气量 × 余量系数**
- 计算：  
  1000 × 1.3 = **1300L/min**
- 结论：  
  气源处理单元额定流量应不小于 **1300L/min**，并按厂家压降曲线校核。

> [!tip] 记忆口诀
> 气源不稳，后面全抖；气源不净，阀缸都病。

# 04 阀岛集成控制

> [!info] 核心结论
> 阀岛是集成气动系统的核心控制单元，用于集中安装多个电磁阀，实现多气缸集中控制和快速维护。

## ① 分类/公式/规则

> [!note] 🟦 阀岛优势
> 阀岛可以减少接线、减少管路混乱、提高诊断能力。  
> 多气缸设备、自动化产线、机器人末端夹具常优先考虑阀岛。

> [!warning] 🟧 阀岛选型公式
> **阀岛选型 = 阀位数量 + 阀型组合 + 流量 + 通讯协议 + 供电方式 + 扩展能力**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **阀位数量** | 决定可控气缸数量 |
| **阀型** | 3/2、5/2、5/3等 |
| **额定流量** | 决定气缸速度 |
| **通讯方式** | 多针、IO-Link、PROFINET等 |
| **诊断功能** | 断线、短路、低压报警 |
| **扩展能力** | 后期加轴、加阀是否方便 |

## ③ 计算/选型示例

- 已知：  
  设备有10个双作用气缸，每个气缸用一个5/2阀。
- 公式：  
  **所需阀位 ≥ 气缸数量**
- 计算：  
  需要至少 **10个5/2阀位**，建议预留1~2个扩展位。
- 结论：  
  可选 **12位阀岛**，其中10位使用，2位备用。

> [!tip] 记忆口诀
> 气缸多，用阀岛；线少、气路清、维护快。

# 05 分布式气动系统

> [!info] 核心结论
> 分布式气动系统把阀岛、IO和传感器布置在设备现场，减少长距离布线和长气管，提高响应速度。

## ① 分类/公式/规则

> [!note] 🟦 分布式思路
> 传统方式是阀集中在电柜或气控箱中；  
> 分布式方式是把阀岛靠近气缸安装，让气路更短、响应更快、管路更清晰。

> [!warning] 🟧 布局规则
> **电缆可以长一点，气管尽量短一点**

## ② 参考表

| 布局方式 | 特点 |
|---|---|
| **集中式阀组** | 维护集中，气管可能较长 |
| **分布式阀岛** | 响应快，现场布置灵活 |
| **远程IO集成** | 接线少，诊断强 |
| **机旁气控箱** | 适合模块化设备 |
| **末端执行集成** | 适合夹具、机器人末端 |
| **总线控制** | 适合多站点设备 |

## ③ 计算/选型示例

- 已知：  
  电磁阀距离气缸3米，气缸动作响应慢。
- 公式：  
  **气管越长 → 充排气时间越长**
- 计算：  
  长气管增加气路容积，响应变慢。
- 结论：  
  可把阀岛移近气缸，或采用分布式阀岛方案。

> [!tip] 记忆口诀
> 气管越短，动作越快；阀离气缸越近，响应越稳。

# 06 气动执行集成

> [!info] 核心结论
> 气动执行集成要把气缸、气爪、真空吸盘、传感器和安装结构作为一个动作单元整体设计。

## ① 分类/公式/规则

> [!note] 🟦 集成思路
> 执行元件不能只看“能不能动”，还要看 **推力、速度、导向、缓冲、反馈、安装、维护**。

> [!warning] 🟧 执行单元选型公式
> **执行单元 = 执行动作 + 输出力 + 行程/角度 + 速度 + 到位检测 + 缓冲方式**

## ② 参考表

| 动作需求 | 推荐执行元件 |
|---|---|
| **直线推拉** | 标准气缸 |
| **短行程压紧** | 薄型气缸 |
| **抗偏载推送** | 导杆气缸 |
| **高刚性滑移** | 滑台气缸 |
| **长行程移动** | 无杆气缸 |
| **夹取工件** | 气爪 |
| **吸附搬运** | 真空吸盘 |

## ③ 计算/选型示例

- 已知：  
  需要夹取0.8kg工件，垂直搬运，取安全系数3。
- 公式：  
  **夹持力 ≥ m × g × 安全系数**
- 计算：  
  0.8 × 9.8 × 3 = **23.52N**
- 结论：  
  气爪夹持力应大于 **23.52N**，还需考虑摩擦系数、加速度和手指长度。

> [!tip] 记忆口诀
> 推拉看气缸，夹取看气爪，吸附看真空，反馈一定要加。

# 07 传感反馈集成

> [!info] 核心结论
> 集成气动系统必须用传感器确认动作结果，不能只依赖PLC输出或阀岛动作状态。

## ① 分类/公式/规则

> [!note] 🟦 反馈逻辑
> 电磁阀通电只是“发出动作命令”；  
> 气缸磁性开关、压力开关、真空开关才是“动作结果反馈”。

> [!warning] 🟧 标准判断逻辑
> **动作完成 = 输出命令 + 到位反馈 + 未超时 + 压力/真空正常**

## ② 参考表

| 反馈类型 | 用途 |
|---|---|
| **磁性开关** | 气缸伸缩到位 |
| **压力开关** | 气压足够/夹紧确认 |
| **真空开关** | 吸附确认 |
| **流量传感器** | 泄漏和耗气监测 |
| **接近开关** | 工件或机构到位 |
| **模拟量传感器** | 连续压力/流量监测 |

## ③ 计算/选型示例

- 已知：  
  夹爪关闭后需要确认是否夹到工件。
- 公式：  
  **夹爪闭合 ≠ 工件已夹住**
- 计算：  
  夹爪可能空夹，也可能工件偏位。
- 结论：  
  可增加 **压力检测、位置检测或工件有无检测**，提高动作可靠性。

> [!tip] 记忆口诀
> 阀输出是命令，传感器反馈才是结果。

# 08 安全与残压释放

> [!info] 核心结论
> 集成气动系统必须考虑急停、断电、断气、残压释放、防误动作和垂直轴防坠落。

## ① 分类/公式/规则

> [!note] 🟦 安全重点
> 气动系统断电后，管路中可能仍有残压。  
> 急停后若没有安全排气，气缸可能继续动作或保持危险压力。

> [!warning] 🟧 安全气路公式
> **安全停机 = 切断能量 + 释放残压 + 防止意外启动 + 保持危险轴安全**

## ② 参考表

| 安全项 | 作用 |
|---|---|
| **安全切断阀** | 急停切断气源 |
| **残压释放阀** | 排出下游压力 |
| **软启动阀** | 缓慢建立压力 |
| **压力监控** | 判断是否泄压到位 |
| **机械防坠** | 垂直轴防下落 |
| **双通道安全阀** | 高安全等级场景 |

## ③ 计算/选型示例

- 已知：  
  设备急停后，气缸仍保持夹紧压力。
- 公式：  
  **残压未释放 → 执行元件仍有力**
- 计算：  
  下游管路仍有压力，气缸可能继续产生夹紧力。
- 结论：  
  应配置残压释放方案，安全要求需按实际规范和风险评估校核。

> [!tip] 记忆口诀
> 急停不等于没气，残压不放仍危险。

# 09 管路与接口集成

> [!info] 核心结论
> 管路和接头是集成气动系统的“血管”，管径、管长、接头通径和排气能力会直接影响动作速度。

## ① 分类/公式/规则

> [!note] 🟦 管路原则
> 主管要满足总流量，支管要满足单个执行元件流量。  
> 气管越短、弯头越少、接头通径越大，系统响应越好。

> [!warning] 🟧 管路规则
> **短管路 + 合理管径 + 少接头 + 排气通畅 = 动作稳定**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **主管管径** | 影响总供气能力 |
| **支管管径** | 影响单个气缸速度 |
| **气管长度** | 影响响应时间 |
| **接头通径** | 可能成为流量瓶颈 |
| **消声器** | 堵塞会拖慢动作 |
| **分气块** | 内部通径影响多路供气 |

## ③ 计算/选型示例

- 已知：  
  某气缸伸出正常，缩回很慢。
- 公式：  
  **单方向慢 = 对应方向供气/排气受限**
- 计算：  
  可能是缩回侧节流阀开度小、气管折弯、接头堵塞或消声器堵塞。
- 结论：  
  应检查缩回方向的管路、接头、节流阀和排气口。

> [!tip] 记忆口诀
> 管路小会憋气，排气堵会变慢。

# 10 通讯与智能诊断

> [!info] 核心结论
> 现代集成气动系统可通过总线、IO-Link和智能阀岛实现远程控制、状态监测和故障诊断。

## ① 分类/公式/规则

> [!note] 🟦 智能化方向
> 传统气动系统只知道阀有没有输出；  
> 智能气动系统还能知道电源状态、阀岛故障、压力异常、通讯异常和维护信息。

> [!warning] 🟧 通讯选型公式
> **通讯方案 = PLC平台 + 阀岛协议 + IO数量 + 诊断需求 + 扩展需求**

## ② 参考表

| 通讯方式 | 特点 |
|---|---|
| **多针接口** | 简单，诊断少 |
| **IO-Link** | 单设备参数化和诊断方便 |
| **PROFINET** | 西门子系统常见 |
| **EtherCAT** | 高速实时控制 |
| **EtherNet/IP** | 常见工业以太网 |
| **远程IO** | 分布式接线方便 |

## ③ 计算/选型示例

- 已知：  
  设备使用西门子PLC，需要集中控制阀岛并读取诊断信息。
- 公式：  
  **西门子PLC集成 → PROFINET阀岛优先评估**
- 计算：  
  PROFINET可减少I/O接线，并支持站点诊断。
- 结论：  
  可选择 **PROFINET阀岛 + 远程IO/传感器反馈** 的集成方案。

> [!tip] 记忆口诀
> 只要动作，用普通I/O；要诊断维护，用总线阀岛。

# 11 节能与耗气管理

> [!info] 核心结论
> 集成气动系统节能重点是减少漏气、降低不必要压力、优化动作顺序和监测耗气量。

## ① 分类/公式/规则

> [!note] 🟦 节能思路
> 压缩空气成本高，漏气和过高压力会长期增加能耗。  
> 系统集成时应考虑分区供气、停机断气、低压待机和流量监测。

> [!warning] 🟧 节能公式
> **气动能耗 = 用气量 + 泄漏量 + 压力设定 + 动作频率**

## ② 参考表

| 节能方法 | 作用 |
|---|---|
| **检漏维护** | 减少长期损耗 |
| **压力分区** | 不同工位不同压力 |
| **停机断气** | 减少待机耗气 |
| **流量监测** | 找出异常耗气点 |
| **动作优化** | 避免无效吹气 |
| **合理管径** | 减少压降和补偿压力 |

## ③ 计算/选型示例

- 已知：  
  设备停机时仍有稳定耗气流量。
- 公式：  
  **停机仍耗气 = 泄漏或未关闭气路**
- 计算：  
  停机状态下执行元件不应持续用气。
- 结论：  
  应检查接头、阀岛、气缸密封和吹气回路，并考虑停机自动切断气源。

> [!tip] 记忆口诀
> 气动节能先查漏，压力别设太高，吹气别一直吹。

# 12 模块化设计

> [!info] 核心结论
> 集成气动系统推荐模块化设计，把每个工位做成可独立调试、独立维护、独立诊断的气动模块。

## ① 分类/公式/规则

> [!note] 🟦 模块化思路
> 每个模块包括：局部阀岛、执行元件、传感器、调压单元、管路标识和电气接口。  
> 这样便于装配、调试、复制和维修。

> [!warning] 🟧 模块设计公式
> **气动模块 = 动作单元 + 控制阀 + 传感反馈 + 局部调压 + 快速接口**

## ② 参考表

| 模块内容 | 作用 |
|---|---|
| **局部阀组** | 控制本模块动作 |
| **气动执行件** | 完成工艺动作 |
| **传感器** | 反馈动作状态 |
| **调压阀** | 独立设定压力 |
| **快插接口** | 方便拆装维护 |
| **标识标签** | 便于排查气路 |

## ③ 计算/选型示例

- 已知：  
  一台设备有多个相似夹具工位。
- 公式：  
  **重复工位 → 模块化气动单元**
- 计算：  
  每个工位使用相同气动接口和阀组布局，后续复制和维护更方便。
- 结论：  
  应设计标准化气动模块，统一接头、管径、编号和调试流程。

> [!tip] 记忆口诀
> 工位多就模块化，复制快、维护快、排错快。

# 13 调试流程

> [!info] 核心结论
> 集成气动系统调试要按“气源 → 单阀 → 单缸 → 反馈 → 联动 → 安全”的顺序逐步进行。

## ① 分类/公式/规则

> [!note] 🟦 调试顺序
> 先不上自动程序，先手动验证每个阀、每个气缸、每个传感器。  
> 单个动作稳定后，再做联动和节拍优化。

> [!warning] 🟧 调试原则
> **先低压、低速、单动作；再正常压力、正常速度、联动动作**

## ② 参考表

| 步骤 | 检查内容 |
|---|---|
| **气源检查** | 压力、排水、过滤 |
| **阀岛检查** | 通讯、电源、手动按钮 |
| **单缸动作** | 方向、速度、缓冲 |
| **传感反馈** | 到位、压力、真空信号 |
| **联动测试** | 顺序、互锁、超时 |
| **安全测试** | 急停、残压、复位 |

## ③ 计算/选型示例

- 已知：  
  自动模式下动作顺序混乱。
- 公式：  
  **自动异常 = 单动作未验证 + 反馈逻辑错误 + 程序互锁不足**
- 计算：  
  应先切到手动，逐个测试阀和传感器，再检查程序顺序。
- 结论：  
  不要直接调自动流程，先把单动作和反馈状态调正确。

> [!tip] 记忆口诀
> 先单动，再联动；先慢速，再节拍；先反馈，再自动。

# 14 常见故障排查

> [!info] 核心结论
> 集成气动系统故障排查要从系统链路入手，按 **气源、阀岛、管路、执行元件、传感器、程序** 逐层定位。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先确认总气源压力和阀岛供气；  
> 再确认阀岛输出和气缸动作；  
> 最后检查传感器反馈和PLC逻辑。

> [!warning] 🟧 排错公式
> **系统异常 = 气源不足 + 阀岛故障 + 管路问题 + 执行卡滞 + 反馈异常 + 程序互锁错误**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **整机气动无动作** | 总气源关闭、急停切气、阀岛无电 |
| **部分气缸不动** | 单阀故障、管路接错、气缸卡滞 |
| **动作慢** | 流量不足、管径小、消声器堵 |
| **动作无力** | 压力低、缸径小、漏气 |
| **反馈不到位** | 磁开关位置偏、接线错误 |
| **联动异常** | 程序顺序、互锁、超时设置错误 |
| **频繁低压报警** | 用气量大、供气不足、泄漏 |

## ③ 计算/选型示例

- 已知：  
  多个气缸同时动作时，系统低压报警。
- 公式：  
  **低压报警 = 瞬时用气量 > 供气能力**
- 计算：  
  同时动作导致用气峰值过大，气源处理元件、主管或阀岛供气不足。
- 结论：  
  可增大气源处理流量、主管管径，或优化程序错峰动作。

> [!tip] 记忆口诀
> 全部不动查总气，单个不动查单阀，动作慢查流量，报警查反馈。

# 15 集成气动系统快速选型表

> [!info] 核心结论
> 集成气动系统选型要从整机动作和节拍出发，统一校核气源、阀岛、执行件、传感器、管路和安全。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 列出所有气动动作  
> ② 确认气缸、气爪、真空数量  
> ③ 计算总用气量和峰值用气量  
> ④ 选择气源处理规格  
> ⑤ 选择阀岛阀位和通讯方式  
> ⑥ 选择执行元件和传感器  
> ⑦ 规划主管、支管和接头规格  
> ⑧ 配置压力/真空/流量监控  
> ⑨ 设计急停、残压释放和安全联锁  
> ⑩ 做调试、标识、检漏和维护方案

> [!warning] 🟧 最终选型公式
> **推荐系统 = 气源处理 + 阀岛 + 执行元件 + 传感反馈 + 管路附件 + 通讯诊断 + 安全模块**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 多气缸集中控制 | 阀岛 |
| 快速动作 | 阀靠近气缸、管路短 |
| 精密压力控制 | 比例阀 + 压力传感器 |
| 真空搬运 | 真空发生器/泵 + 真空开关 |
| 多工位模块化 | 分布式阀岛/远程IO |
| 低压安全联锁 | 压力开关 |
| 节能管理 | 流量监测 + 检漏维护 |

## ③ 计算/选型示例

- 已知：  
  设备有8个双作用气缸、2个真空吸盘，最大同时用气量900L/min，PLC为PROFINET系统。
- 公式：  
  **气源处理流量 ≥ 900 × 1.3**  
  **阀岛协议 = PLC通讯协议匹配**
- 计算：  
  气源处理流量 ≥ **1170L/min**  
  阀岛可选 **PROFINET通讯阀岛**
- 结论：  
  推荐配置为 **大于1170L/min气源处理单元 + PROFINET阀岛 + 气缸磁性开关 + 真空开关 + 合理管路与残压释放模块**。

> [!tip] 选型口诀
> 先列动作，再算用气；气源要稳，阀岛要够；气管要短，反馈要全；急停泄压，维护有标识。
