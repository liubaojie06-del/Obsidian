# 01 直线导轨总览

> [!info] 核心结论
> 直线导轨用于支撑和引导运动部件做高精度直线运动，核心看 **负载、精度、刚性、行程、安装方式和使用环境**。

## ① 分类/公式/规则

> [!warning] 选型公式
> **直线导轨选型 = 负载大小 + 运动精度 + 刚性要求 + 行程长度 + 安装空间 + 防护润滑 + 寿命要求**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **导轨** | 提供直线运动基准 |
| **滑块** | 承载负载并沿导轨移动 |
| **滚动体** | 降低摩擦，常见滚珠/滚柱 |
| **预压** | 提高刚性，减少间隙 |
| **精度等级** | 影响运动直线度和定位精度 |
| **润滑防护** | 影响寿命和稳定性 |

## ③ 计算/选型示例

- 已知：  
  设备需要承载20kg工件做水平直线移动。
- 公式：  
  **导轨需求 = 负载 + 行程 + 精度 + 刚性**
- 计算：  
  水平移动主要校核重量、加速度、偏心力矩和导轨寿命。
- 结论：  
  可选 **双导轨 + 多滑块结构**，具体规格需按厂家手册校核。

> [!tip] 记忆口诀
> 导轨管方向，滑块管承载；精度看等级，寿命看润滑。

# 02 底层原理

> [!info] 核心结论
> 直线导轨的本质，是用滚珠或滚柱在导轨和滑块之间循环滚动，把滑动摩擦变成滚动摩擦。

## ① 分类/公式/规则

> [!note] 工作逻辑
> 滑块内部的滚珠或滚柱在负载作用下沿滚道滚动，同时通过回流通道循环，实现低摩擦直线运动。

> [!warning] 运动链路
> **负载 → 滑块 → 滚动体 → 导轨滚道 → 机架基准面**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **导轨本体** | 提供刚性和运动基准 |
| **滑块本体** | 承载并连接运动部件 |
| **滚珠/滚柱** | 承载并降低摩擦 |
| **回流器** | 让滚动体循环运动 |
| **密封片** | 防尘、防异物进入 |
| **油嘴/油孔** | 加注润滑脂或润滑油 |

## ③ 计算/选型示例

- 已知：  
  普通滑动副摩擦大，运动发涩。
- 公式：  
  **滚动摩擦 < 滑动摩擦**
- 计算：  
  直线导轨用滚动体承载，可显著降低运动阻力。
- 结论：  
  对需要平稳直线运动的机构，直线导轨比普通滑动导向更适合。

> [!tip] 记忆口诀
> 滚起来才顺，导得准才稳。

# 03 结构组成

> [!info] 核心结论
> 直线导轨通常由 **导轨、滑块、滚动体、端盖、密封件、润滑接口** 组成。

## ① 分类/公式/规则

> [!note] 结构理解
> 导轨负责“走得直”，滑块负责“带负载”，滚动体负责“低摩擦”，密封润滑负责“用得久”。

> [!warning] 结构公式
> **直线导轨组件 = 导轨 + 滑块 + 滚动体 + 回流结构 + 密封润滑**

## ② 参考表

| 结构 | 重点 |
|---|---|
| **导轨** | 长度、宽度、安装孔距 |
| **滑块** | 法兰型、方型、加长型 |
| **滚动体** | 滚珠或滚柱 |
| **端盖** | 滚动体回流 |
| **防尘片** | 防尘、防切屑 |
| **润滑口** | 定期补油维护 |

## ③ 计算/选型示例

- 已知：  
  设备负载较大，且有偏心力矩。
- 公式：  
  **抗偏载能力 = 导轨数量 + 滑块数量 + 滑块间距**
- 计算：  
  单滑块抗力矩有限，增加滑块数量和间距可提高稳定性。
- 结论：  
  推荐使用 **双导轨四滑块** 结构。

> [!tip] 记忆口诀
> 一根导轨能导向，两根导轨更抗扭；滑块拉开，力矩更稳。

# 04 常见类型

> [!info] 核心结论
> 直线导轨常见有滚珠导轨、滚柱导轨、微型导轨、宽幅导轨和交叉滚子导轨，不同类型适合不同负载和精度。

## ① 分类/公式/规则

> [!warning] 选择规则
> **普通自动化 → 滚珠导轨**  
> **高刚性重载 → 滚柱导轨**  
> **小型设备 → 微型导轨**  
> **高精密平台 → 交叉滚子导轨**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **滚珠导轨** | 通用自动化，应用最广 |
| **滚柱导轨** | 高刚性、重载、高精度 |
| **微型导轨** | 小设备、小行程 |
| **宽幅导轨** | 单轨抗力矩较好 |
| **静音导轨** | 低噪声运动 |
| **交叉滚子导轨** | 精密平台、短行程高刚性 |

## ③ 计算/选型示例

- 已知：  
  设备负载中等，要求通用、成本适中。
- 公式：  
  **通用负载 + 普通精度 → 滚珠导轨优先**
- 计算：  
  滚珠导轨规格多，安装维护方便。
- 结论：  
  可优先选择 **滚珠直线导轨**。

> [!tip] 记忆口诀
> 通用选滚珠，重载选滚柱，小型选微型，精密选交叉滚子。

# 05 滚珠导轨

> [!info] 核心结论
> 滚珠导轨是最常用的直线导轨，摩擦小、速度高、成本适中，适合大多数自动化设备。

## ① 分类/公式/规则

> [!note] 核心特点
> 滚珠导轨通过钢球在滚道内循环滚动承载，运行顺畅，适合高速往复和一般精度要求。

> [!warning] 应用规则
> **速度快、负载中等、通用设备 → 滚珠导轨优先**

## ② 参考表

| 项目 | 特点 |
|---|---|
| **摩擦** | 小 |
| **速度** | 较高 |
| **承载** | 中等到较高 |
| **刚性** | 中等 |
| **成本** | 相对适中 |
| **应用** | 直线模组、滑台、搬运机构 |

## ③ 计算/选型示例

- 已知：  
  直线模组速度较高，负载不大。
- 公式：  
  **高速轻中载 → 滚珠导轨**
- 计算：  
  滚珠导轨摩擦小，适合高速往复。
- 结论：  
  可选择滚珠直线导轨，规格按负载和寿命校核。

> [!tip] 记忆口诀
> 滚珠导轨跑得顺，通用设备最常用。

---

# 06 滚柱导轨

> [!info] 核心结论
> 滚柱导轨刚性和承载能力更强，适合重载、高刚性、高精度设备。

## ① 分类/公式/规则

> [!note] 核心特点
> 滚柱与滚道接触面积更大，抗变形能力更强，但成本和安装要求通常更高。

> [!warning] 应用规则
> **重载 + 高刚性 + 高稳定性 → 滚柱导轨优先**

## ② 参考表

| 项目 | 特点 |
|---|---|
| **承载能力** | 强 |
| **刚性** | 高 |
| **抗振性** | 较好 |
| **成本** | 较高 |
| **安装要求** | 较高 |
| **应用** | 机床、压装设备、重载滑台 |

## ③ 计算/选型示例

- 已知：  
  设备存在较大切削力或压装力。
- 公式：  
  **外力大 → 需要高刚性导轨**
- 计算：  
  普通滚珠导轨可能弹性变形较大。
- 结论：  
  可评估 **滚柱直线导轨**，并校核安装基准和预压等级。

> [!tip] 记忆口诀
> 滚珠跑得快，滚柱扛得住。

# 07 精度等级

> [!info] 核心结论
> 直线导轨精度等级影响运动直线度、平行度和安装后运动精度，不能只看导轨尺寸。

## ① 分类/公式/规则

> [!note] 精度理解
> 精度等级越高，导轨制造误差越小，但对安装基准面的要求也更高。  
> 如果安装面不合格，高精度导轨也发挥不出来。

> [!warning] 精度规则
> **导轨精度 + 安装精度 + 机架刚性 = 实际运动精度**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **导轨直线度** | 影响运动轨迹 |
| **滑块高度差** | 影响平台平面度 |
| **成对平行度** | 影响双轨运行阻力 |
| **安装基准面** | 影响最终精度 |
| **螺钉锁紧顺序** | 影响变形 |
| **温度变化** | 影响长度和间隙 |

## ③ 计算/选型示例

- 已知：  
  购买了高精度导轨，但装机后运动发紧。
- 公式：  
  **运动发紧 = 导轨不平行 / 基准面不平 / 安装应力**
- 计算：  
  高精度导轨对安装平面和平行度更敏感。
- 结论：  
  应检查基准面精度、双轨平行度和锁紧顺序。

> [!tip] 记忆口诀
> 导轨精度买得高，不如安装基准做得好。

# 08 预压等级

> [!info] 核心结论
> 预压可以提高导轨刚性和消除间隙，但预压过大会增加摩擦、发热和寿命损耗。

## ① 分类/公式/规则

> [!note] 预压理解
> 预压就是让滚动体在滑块和导轨之间提前承受一定压紧力，使导轨更“紧”、更刚。

> [!warning] 选择规则
> **高速轻载 → 低预压**  
> **高刚性定位 → 中预压**  
> **重载高刚性 → 高预压需谨慎校核**

## ② 参考表

| 预压状态 | 影响 |
|---|---|
| **无预压/轻预压** | 运动轻快，适合高速 |
| **中预压** | 刚性较好，通用精密设备 |
| **高预压** | 刚性强，但阻力大 |
| **预压过大** | 发热、磨损、寿命下降 |
| **预压过小** | 间隙大，刚性不足 |
| **最终选择** | 按厂家等级校核 |

## ③ 计算/选型示例

- 已知：  
  设备要求高速搬运，负载较轻。
- 公式：  
  **高速轻载 → 不宜过高预压**
- 计算：  
  高预压会增加运动阻力和发热。
- 结论：  
  可选择轻预压或普通预压，具体按厂家手册校核。

> [!tip] 记忆口诀
> 预压小跑得轻，预压大刚性强；选太大，寿命伤。

# 09 负载与额定载荷

> [!info] 核心结论
> 直线导轨选型不能只看静态重量，还要看动态载荷、冲击、偏心力矩和寿命。

## ① 分类/公式/规则

> [!warning] 寿命简化公式
> **L =（C ÷ P）³ × 50km**

> [!note] 参数说明
> **L**：额定寿命，常用于滚珠导轨估算  
> **C**：基本动额定载荷  
> **P**：等效工作载荷  
> 不同厂家和导轨类型公式可能不同，需按厂家手册校核。

## ② 参考表

| 参数 | 说明 |
|---|---|
| **C动额定载荷** | 与疲劳寿命相关 |
| **C₀静额定载荷** | 与静态安全相关 |
| **P等效载荷** | 综合实际受力 |
| **冲击系数** | 冲击越大，系数越大 |
| **安全系数** | 防止过载损坏 |
| **力矩载荷** | 偏心负载必须校核 |

## ③ 计算/选型示例

- 已知：  
  导轨动额定载荷C=10kN，等效载荷P=2kN。
- 公式：  
  **L =（10 ÷ 2）³ × 50**
- 计算：  
  L = 5³ × 50 = **6250km**
- 结论：  
  理论额定寿命约6250km，实际需按速度、润滑、冲击和厂家公式校核。

> [!tip] 记忆口诀
> 载荷越小寿命越长，冲击越大寿命越短。

# 10 力矩与偏载

> [!info] 核心结论
> 直线导轨最怕偏心负载，偏载会产生力矩，使滑块受力不均、磨损加快、运动发紧。

## ① 分类/公式/规则

> [!warning] 力矩公式
> **M = F × L**

> [!note] 参数说明
> **M**：力矩  
> **F**：作用力  
> **L**：力臂距离  
> 负载离滑块越远，力矩越大。

## ② 参考表

| 偏载情况 | 风险 |
|---|---|
| **负载悬臂长** | 力矩大 |
| **单滑块承载** | 抗力矩弱 |
| **双滑块间距小** | 稳定性差 |
| **重心偏离中心** | 受力不均 |
| **冲击载荷** | 滚道易损伤 |
| **高速急停** | 惯性力矩大 |

## ③ 计算/选型示例

- 已知：  
  负载力F=100N，离滑块中心距离L=0.2m。
- 公式：  
  **M = 100 × 0.2**
- 计算：  
  M = **20N·m**
- 结论：  
  滑块需要承受20N·m力矩，应校核滑块允许力矩，必要时改为双滑块或双导轨。

> [!tip] 记忆口诀
> 负载越偏，导轨越累；滑块拉开，力矩变小。

---

# 11 双导轨四滑块

> [!info] 核心结论
> 双导轨四滑块是最常见的高稳定直线平台结构，适合承载、抗扭和提高运动平稳性。

## ① 分类/公式/规则

> [!note] 结构特点
> 两根导轨提供宽基准，四个滑块分摊负载。  
> 滑块间距越大，抗俯仰、抗偏摆能力越强。

> [!warning] 设计规则
> **平台越宽、负载越偏、力矩越大 → 越需要双导轨多滑块**

## ② 参考表

| 结构 | 特点 |
|---|---|
| **单轨单滑块** | 简单，但抗力矩弱 |
| **单轨双滑块** | 抗俯仰能力提高 |
| **双轨双滑块** | 稳定性更好 |
| **双轨四滑块** | 通用平台常用 |
| **滑块间距大** | 抗力矩更好 |
| **导轨间距大** | 抗扭更好 |

## ③ 计算/选型示例

- 已知：  
  工作台上安装偏心负载，运行时有晃动。
- 公式：  
  **晃动风险 = 导轨间距小 + 滑块间距小 + 偏心力矩大**
- 计算：  
  增加导轨间距和滑块间距可提高平台稳定性。
- 结论：  
  推荐采用 **双导轨四滑块** 布局。

> [!tip] 记忆口诀
> 单轨能走，双轨更稳；四滑块一上，平台不晃。

# 12 安装基准

> [!info] 核心结论
> 直线导轨安装质量决定最终运动效果，重点是基准面、平行度、锁紧顺序和清洁度。

## ① 分类/公式/规则

> [!note] 安装原则
> 导轨不能靠螺丝强行拉直，也不能装在不平的基准面上。  
> 基准面有毛刺、铁屑、油污，都会影响导轨直线度。

> [!warning] 安装规则
> **清洁基准面 → 靠紧基准边 → 轻锁螺钉 → 校正平行 → 分步锁紧**

## ② 参考表

| 安装项 | 注意 |
|---|---|
| **基准面** | 平面度和粗糙度 |
| **基准边** | 用于导轨靠边定位 |
| **螺钉锁紧** | 按顺序分步锁紧 |
| **双轨平行** | 防止运动发紧 |
| **滑块保护** | 不要随意拆下滑块 |
| **清洁度** | 防止异物压伤滚道 |

## ③ 计算/选型示例

- 已知：  
  导轨装好后滑块局部位置运动很紧。
- 公式：  
  **局部发紧 = 基准面不平 / 双轨不平行 / 异物压入**
- 计算：  
  某一段受力异常会让滑块滚动阻力变大。
- 结论：  
  应检查安装面、导轨平行度和螺钉锁紧状态。

> [!tip] 记忆口诀
> 装导轨先清面，双轨一定调平行。

# 13 润滑与防护

> [!info] 核心结论
> 润滑和防护决定直线导轨寿命，缺油、粉尘、切屑和水汽都会明显缩短寿命。

## ① 分类/公式/规则

> [!note] 维护重点
> 导轨运行时滚动体和滚道不断接触，必须有润滑膜保护。  
> 粉尘、金属屑进入滑块后，会造成滚道划伤和卡滞。

> [!warning] 维护公式
> **长寿命 = 定期润滑 + 防尘密封 + 清洁环境 + 正确负载**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **润滑脂** | 常用维护方式 |
| **润滑油** | 自动润滑系统常用 |
| **端面密封** | 防尘、防异物 |
| **刮屑片** | 防切屑进入 |
| **防尘盖带** | 保护导轨安装孔 |
| **防护罩** | 恶劣环境使用 |

## ③ 计算/选型示例

- 已知：  
  导轨使用一段时间后噪声变大，表面干涩。
- 公式：  
  **噪声变大 + 表面干涩 = 润滑不足风险**
- 计算：  
  缺油会增加滚动体与滚道磨损。
- 结论：  
  应补充指定润滑脂，并检查密封和防尘状态。

> [!tip] 记忆口诀
> 导轨不怕跑，怕的是干跑和脏跑。

# 14 常见故障排查

> [!info] 核心结论
> 直线导轨故障排查按 **安装、润滑、负载、防护、磨损** 的顺序进行。

## ① 分类/公式/规则

> [!note] 排查顺序
> 先看导轨是否装歪，再看是否缺油；  
> 先空载推滑块，再带载运行；  
> 先查异物和防护，再判断导轨损坏。

> [!warning] 排错公式
> **导轨异常 = 安装不良 + 润滑不足 + 负载偏心 + 异物进入 + 滑块磨损**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **运动发紧** | 双轨不平行、预压过大 |
| **噪声变大** | 缺油、异物、滚道损伤 |
| **定位不稳** | 滑块间隙、安装松动 |
| **局部卡顿** | 基准面不平、异物压伤 |
| **寿命短** | 负载过大、润滑差 |
| **滑块发热** | 预压过大、速度高、润滑不足 |
| **锈蚀** | 潮湿、清洁维护不足 |

## ③ 计算/选型示例

- 已知：  
  双导轨平台某段运行卡顿，其余位置正常。
- 公式：  
  **局部卡顿 = 局部安装误差 / 异物 / 滚道损伤**
- 计算：  
  如果只在固定位置异常，通常不是电机问题，而是导轨或安装问题。
- 结论：  
  应检查该段导轨表面、安装螺钉、基准面和滑块滚道。

> [!tip] 记忆口诀
> 全程紧查平行，局部卡查异物；越跑越响，先补润滑。

# 15 直线导轨快速选型表

> [!info] 核心结论
> 直线导轨选型先确定负载和结构，再校核精度、刚性、寿命、安装和防护。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 看运动方向和行程  
> ② 看负载重量  
> ③ 看是否有偏心力矩  
> ④ 确定单轨/双轨结构  
> ⑤ 确定滑块数量和间距  
> ⑥ 选择滚珠/滚柱/微型类型  
> ⑦ 确定精度等级和预压等级  
> ⑧ 校核动载荷、静载荷和寿命  
> ⑨ 校核安装空间和基准面  
> ⑩ 配置润滑、防尘和维护方案

> [!warning] 最终选型公式
> **推荐导轨 = 导轨类型 + 规格尺寸 + 滑块数量 + 精度等级 + 预压等级 + 防护润滑方案**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 普通自动化滑台 | 滚珠导轨 |
| 高刚性重载 | 滚柱导轨 |
| 小型紧凑机构 | 微型导轨 |
| 抗偏心力矩 | 双导轨四滑块 |
| 高精密短行程 | 交叉滚子导轨 |
| 粉尘切屑环境 | 加强密封和防护 |
| 高速往复 | 轻预压、良好润滑 |

## ③ 计算/选型示例

- 已知：  
  平台负载30kg，负载有偏心，要求运行平稳。
- 公式：  
  **偏心负载 + 平稳运行 → 双导轨多滑块**
- 计算：  
  单导轨抗力矩不足，双导轨四滑块可分摊负载并提高抗扭刚性。
- 结论：  
  推荐初步采用 **双导轨四滑块滚珠导轨结构**，再按负载、力矩和寿命用厂家手册校核。

> [!tip] 选型口诀
> 先看负载，再看力矩；普通用滚珠，重载用滚柱；双轨四滑块，平台最稳妥。
