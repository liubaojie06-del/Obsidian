# 01 滚珠丝杠总览

> [!info] 核心结论
> 滚珠丝杠用于把电机旋转运动转换成高效率、高精度的直线运动，核心看 **导程、精度、负载、速度、支撑方式、预压和寿命**。

## ① 分类/公式/规则

> [!warning] 选型公式
> **滚珠丝杠选型 = 负载推力 + 行程 + 速度 + 导程 + 精度等级 + 支撑方式 + 预压 + 寿命**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **丝杠轴** | 旋转传动主体 |
| **螺母** | 输出直线运动 |
| **滚珠** | 降低摩擦、提高效率 |
| **导程** | 电机转一圈走多远 |
| **预压** | 减小间隙，提高刚性 |
| **支撑座** | 固定丝杠并承受轴向力 |

## ③ 计算/选型示例

- 已知：  
  滑台需要行程500mm，速度300mm/s，定位精度较高。
- 公式：  
  **丝杠需求 = 行程 + 速度 + 精度 + 负载**
- 计算：  
  中短行程、高精度定位场景，滚珠丝杠比同步带更适合。
- 结论：  
  可选择 **伺服电机 + 滚珠丝杠 + 直线导轨** 结构，具体规格需按厂家手册校核。

> [!tip] 记忆口诀
> 旋转变直线，精度看丝杠；导程定速度，预压定刚性。

# 02 底层原理

> [!info] 核心结论
> 滚珠丝杠的本质，是用滚珠在丝杠和螺母之间循环滚动，把滑动摩擦变成滚动摩擦。

## ① 分类/公式/规则

> [!note] 工作逻辑
> 电机带动丝杠旋转，滚珠在螺旋滚道内循环滚动，推动螺母沿轴向直线移动。

> [!warning] 运动链路
> **电机旋转 → 丝杠旋转 → 滚珠循环滚动 → 螺母直线移动 → 平台运动**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **丝杠滚道** | 提供螺旋运动路径 |
| **螺母滚道** | 与丝杠配合承载 |
| **滚珠** | 传递载荷并降低摩擦 |
| **回珠器** | 让滚珠循环返回 |
| **密封圈** | 防尘、防异物 |
| **润滑脂/油** | 减少磨损、延长寿命 |

## ③ 计算/选型示例

- 已知：  
  普通梯形丝杆运行阻力大，定位效率低。
- 公式：  
  **滚动摩擦 < 滑动摩擦**
- 计算：  
  滚珠丝杠用滚珠承载，摩擦小，传动效率高。
- 结论：  
  对高效率、高精度定位机构，应优先评估滚珠丝杠。

> [!tip] 记忆口诀
> 梯形丝杆靠滑，滚珠丝杠靠滚；滚起来，效率就高。

# 03 导程与位移

> [!info] 核心结论
> 导程决定电机转一圈，螺母直线移动多少距离。

## ① 分类/公式/规则

> [!warning] 位移公式
> **直线位移 = 电机转数 × 丝杠导程**

> [!warning] 脉冲当量公式
> **脉冲当量 = 丝杠导程 ÷ 每圈脉冲数**

## ② 参考表

| 导程 | 特点 |
|---|---|
| **小导程** | 推力大、速度低、分辨率高 |
| **大导程** | 速度快、推力相对小 |
| **常见导程** | 5mm、10mm、20mm等，需按厂家规格 |
| **高精度定位** | 常选较小导程 |
| **高速搬运** | 可选较大导程 |
| **最终选择** | 需校核速度、推力和临界转速 |

## ③ 计算/选型示例

- 已知：  
  丝杠导程为10mm，电机转8圈。
- 公式：  
  **直线位移 = 8 × 10**
- 计算：  
  直线位移 = **80mm**
- 结论：  
  电机转8圈，滑台理论移动80mm。

> [!tip] 记忆口诀
> 导程就是一圈走多远，小导程有力，大导程跑快。

# 04 速度与转速

> [!info] 核心结论
> 滚珠丝杠的直线速度由电机转速和导程决定。

## ① 分类/公式/规则

> [!warning] 速度公式
> **直线速度 = 电机转速 × 丝杠导程 ÷ 60**

> [!warning] 电机转速公式
> **电机转速 rpm = 直线速度 ÷ 导程 × 60**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **电机转速** | 越高，直线速度越快 |
| **丝杠导程** | 越大，同转速下速度越快 |
| **丝杠长度** | 越长，高速越容易受限 |
| **支撑方式** | 影响临界转速 |
| **螺母负载** | 影响加速能力 |
| **润滑状态** | 影响高速稳定性 |

## ③ 计算/选型示例

- 已知：  
  丝杠导程10mm，目标直线速度500mm/s。
- 公式：  
  **rpm = 500 ÷ 10 × 60**
- 计算：  
  rpm = **3000rpm**
- 结论：  
  电机需要达到约3000rpm，同时必须校核丝杠临界转速。

> [!tip] 记忆口诀
> 速度要快，看转速和导程；丝杠太长，高速要谨慎。

# 05 推力与驱动转矩

> [!info] 核心结论
> 滚珠丝杠可以把电机转矩转换成轴向推力，导程越小，同样转矩下推力越大。

## ① 分类/公式/规则

> [!warning] 转矩推力公式
> **T = F × L ÷（2π × η）**

> [!note] 参数说明
> **T**：电机输出转矩，N·m  
> **F**：轴向推力，N  
> **L**：导程，m/rev  
> **η**：传动效率，需按厂家手册校核

## ② 参考表

| 参数 | 影响 |
|---|---|
| **轴向推力F** | 负载越大，需要推力越大 |
| **导程L** | 导程越大，所需转矩越大 |
| **效率η** | 效率越高，损耗越小 |
| **摩擦力** | 会增加实际转矩 |
| **加速度** | 影响峰值推力 |
| **安全系数** | 防止过载和失步 |

## ③ 计算/选型示例

- 已知：  
  推力需求500N，导程10mm，效率按0.9估算。
- 公式：  
  **T = F × L ÷（2π × η）**
- 计算：  
  L = 0.01m  
  T = 500 × 0.01 ÷（2 × 3.14 × 0.9）≈ **0.88N·m**
- 结论：  
  理论驱动转矩约0.88N·m，实际还需叠加加速转矩和安全余量。

> [!tip] 记忆口诀
> 导程小更有力，导程大更快；推力转矩要一起算。

# 06 精度等级

> [!info] 核心结论
> 滚珠丝杠精度等级影响定位误差，选型时要区分定位精度和重复定位精度。

## ① 分类/公式/规则

> [!note] 精度理解
> 丝杠精度主要影响“走指定距离时实际走了多少”。  
> 但系统最终精度还受导轨、联轴器、安装、温升、控制参数影响。

> [!warning] 精度公式
> **实际定位误差 = 丝杠导程误差 + 反向间隙 + 安装误差 + 热伸长 + 控制误差**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **研磨丝杠** | 精度高，成本高 |
| **轧制丝杠** | 成本低，通用性强 |
| **导程误差** | 影响长距离定位 |
| **重复定位精度** | 多次回同一点一致性 |
| **反向间隙** | 换向时误差来源 |
| **温升变形** | 高速长时间运行需关注 |

## ③ 计算/选型示例

- 已知：  
  设备要求重复定位精度高，但绝对定位精度要求一般。
- 公式：  
  **重复精度高 → 预压和机械刚性重点校核**
- 计算：  
  重复定位主要受间隙、刚性、导轨和控制稳定性影响。
- 结论：  
  可选带预压螺母的滚珠丝杠，并重点控制安装和反向间隙。

> [!tip] 记忆口诀
> 绝对精度看导程，重复精度看间隙和刚性。

# 07 预压与反向间隙

> [!info] 核心结论
> 预压用于消除或减小滚珠丝杠反向间隙，提高刚性和重复定位稳定性。

## ① 分类/公式/规则

> [!note] 预压理解
> 预压就是让滚珠、丝杠和螺母之间提前保持一定压紧力，减少换向时的空行程。

> [!warning] 选择规则
> **高精度定位 → 需要预压**  
> **高速轻载 → 预压不宜过大**  
> **重载高刚性 → 预压需按厂家校核**

## ② 参考表

| 状态 | 影响 |
|---|---|
| **无预压** | 阻力小，但可能有间隙 |
| **轻预压** | 兼顾寿命和精度 |
| **中预压** | 刚性较好 |
| **高预压** | 刚性强，但发热大 |
| **预压过大** | 磨损、温升、寿命下降 |
| **预压不足** | 换向误差增大 |

## ③ 计算/选型示例

- 已知：  
  滑台正向定位准，反向定位有固定偏差。
- 公式：  
  **反向误差 = 反向间隙 + 弹性变形 + 联轴器松动**
- 计算：  
  换向时若螺母存在间隙，会先空走一小段。
- 结论：  
  应检查丝杠预压、联轴器锁紧和传动间隙补偿。

> [!tip] 记忆口诀
> 反向有偏差，先查间隙；要想换向准，预压少不了。

# 08 支撑方式

> [!info] 核心结论
> 滚珠丝杠支撑方式决定轴向刚性、临界转速和长丝杠稳定性。

## ① 分类/公式/规则

> [!note] 支撑理解
> 丝杠两端支撑越强，抗弯、抗振和高速稳定性越好。  
> 常见支撑方式有固定-自由、固定-支撑、固定-固定。

> [!warning] 选择规则
> **短行程低速 → 固定-自由可用**  
> **中长行程 → 固定-支撑常见**  
> **高速长行程 → 固定-固定优先评估**

## ② 参考表

| 支撑方式 | 特点 |
|---|---|
| **固定-自由** | 结构简单，适合短行程 |
| **固定-支撑** | 通用，稳定性较好 |
| **固定-固定** | 刚性高，适合高速长行程 |
| **轴承座BK/BF** | 常见固定/支撑组合 |
| **角接触轴承** | 承受轴向载荷 |
| **预紧轴承** | 提高轴向刚性 |

## ③ 计算/选型示例

- 已知：  
  丝杠较长，运行高速时出现抖动。
- 公式：  
  **长丝杠高速 → 临界转速风险增加**
- 计算：  
  支撑刚性不足时，丝杠容易弯曲振动。
- 结论：  
  可改用固定-固定支撑，或增加丝杠直径、降低转速。

> [!tip] 记忆口诀
> 丝杠越长，支撑越重要；速度越高，固定越要稳。

# 09 临界转速

> [!info] 核心结论
> 长丝杠高速旋转时会发生振动风险，必须校核临界转速。

## ① 分类/公式/规则

> [!note] 工程理解
> 丝杠像一根细长轴，转速过高时会产生弯曲振动。  
> 丝杠越长、直径越小，临界转速越低。

> [!warning] 使用规则
> **工作转速必须低于厂家允许临界转速，并留安全余量**

## ② 参考表

| 影响因素 | 趋势 |
|---|---|
| **丝杠长度增加** | 临界转速降低 |
| **丝杠直径增加** | 临界转速提高 |
| **支撑刚性提高** | 临界转速提高 |
| **转速提高** | 振动风险增加 |
| **偏心安装** | 加剧振动 |
| **润滑不良** | 噪声和发热增加 |

## ③ 计算/选型示例

- 已知：  
  丝杠导程10mm，目标速度600mm/s。
- 公式：  
  **电机转速 = 600 ÷ 10 × 60**
- 计算：  
  电机转速 = **3600rpm**
- 结论：  
  需要确认该长度和支撑方式下，丝杠允许转速是否高于3600rpm。

> [!tip] 记忆口诀
> 丝杠越长越不能猛转，高速先查临界转速。

# 10 压杆稳定性

> [!info] 核心结论
> 当滚珠丝杠承受压缩载荷时，长细丝杠可能发生弯曲失稳，必须校核压杆稳定性。

## ① 分类/公式/规则

> [!note] 稳定性理解
> 丝杠不仅会传动，还可能承受轴向压力。  
> 长丝杠、细丝杠、垂直轴、推压场景都要关注屈曲风险。

> [!warning] 使用规则
> **轴向压力越大、丝杠越长、直径越小 → 屈曲风险越高**

## ② 参考表

| 场景 | 风险 |
|---|---|
| **垂直升降** | 重力形成轴向负载 |
| **压装机构** | 轴向压力大 |
| **长行程推送** | 细长杆易弯 |
| **固定-自由支撑** | 稳定性较弱 |
| **高速冲击** | 动态载荷增加 |
| **偏心受力** | 更容易弯曲 |

## ③ 计算/选型示例

- 已知：  
  丝杠用于垂直压紧，推力较大，行程较长。
- 公式：  
  **压缩载荷大 → 需校核屈曲安全**
- 计算：  
  仅看推力公式不够，还要看丝杠轴是否会弯曲失稳。
- 结论：  
  应按厂家手册校核允许压缩载荷，必要时增大丝杠直径或改变支撑方式。

> [!tip] 记忆口诀
> 拉着用看强度，顶着用看弯不弯。

# 11 寿命计算

> [!info] 核心结论
> 滚珠丝杠寿命与动额定载荷和实际轴向载荷有关，负载越大寿命越短。

## ① 分类/公式/规则

> [!warning] 寿命简化公式
> **L =（Ca ÷ Fa）³ × 10⁶ rev**

> [!note] 参数说明
> **L**：额定寿命，转数  
> **Ca**：基本动额定载荷  
> **Fa**：等效轴向载荷  
> 不同厂家公式可能不同，需按厂家手册校核。

## ② 参考表

| 参数 | 说明 |
|---|---|
| **Ca动额定载荷** | 与疲劳寿命相关 |
| **Fa轴向载荷** | 实际工作载荷 |
| **冲击系数** | 冲击越大，等效载荷越大 |
| **转速** | 影响单位时间寿命 |
| **润滑** | 明显影响寿命 |
| **粉尘异物** | 会缩短寿命 |

## ③ 计算/选型示例

- 已知：  
  Ca = 12000N，Fa = 3000N。
- 公式：  
  **L =（12000 ÷ 3000）³ × 10⁶**
- 计算：  
  L = 4³ × 10⁶ = **64 × 10⁶ rev**
- 结论：  
  理论额定寿命约6400万转，实际需按工况、润滑和厂家手册修正。

> [!tip] 记忆口诀
> 载荷减一点，寿命长很多；冲击大一点，寿命短很多。

# 12 轴承座与联轴器

> [!info] 核心结论
> 滚珠丝杠系统不仅要选丝杠，还要正确匹配轴承座、联轴器和电机安装结构。

## ① 分类/公式/规则

> [!note] 结构理解
> 轴承座负责支撑丝杠和承受轴向力；  
> 联轴器负责连接电机和丝杠，补偿少量安装误差。

> [!warning] 连接公式
> **传动稳定 = 电机同轴 + 联轴器锁紧 + 支撑座刚性 + 丝杠预紧**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **固定端轴承座** | 承受轴向力，定位丝杠 |
| **支撑端轴承座** | 支撑丝杠端部 |
| **联轴器** | 连接电机与丝杠 |
| **锁紧螺母** | 轴承预紧固定 |
| **电机座** | 保证同轴和刚性 |
| **防松结构** | 防止长期运行松动 |

## ③ 计算/选型示例

- 已知：  
  丝杠运行时有周期性振动，联轴器附近异响。
- 公式：  
  **周期振动 = 同轴度差 / 联轴器松动 / 丝杠弯曲**
- 计算：  
  电机轴与丝杠轴不同心，会让联轴器周期受力。
- 结论：  
  应检查电机座同轴度、联轴器锁紧和轴承座安装。

> [!tip] 记忆口诀
> 丝杠准不准，别只看丝杠；联轴器和轴承座也要稳。

# 13 润滑与防护

> [!info] 核心结论
> 滚珠丝杠需要定期润滑和防尘，缺油、粉尘、切屑进入都会导致磨损、卡滞和精度下降。

## ① 分类/公式/规则

> [!note] 维护重点
> 滚珠在滚道内循环承载，高速高频运行时必须保持润滑膜。  
> 防护罩、密封圈、刮刷和定期清洁都很重要。

> [!warning] 维护公式
> **长寿命 = 正确润滑 + 清洁防护 + 合理负载 + 正确安装**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **润滑脂** | 常见维护方式 |
| **润滑油** | 自动润滑系统常用 |
| **密封圈** | 防尘防异物 |
| **防护罩** | 防切屑和粉尘 |
| **定期清洁** | 防止滚道污染 |
| **补油周期** | 需按工况和厂家要求 |

## ③ 计算/选型示例

- 已知：  
  丝杠运行一段时间后噪声变大，表面发干。
- 公式：  
  **噪声变大 + 表面发干 = 润滑不足风险**
- 计算：  
  缺少润滑会增加滚珠和滚道磨损。
- 结论：  
  应补充指定润滑脂，并检查防尘密封是否完好。

> [!tip] 记忆口诀
> 丝杠不怕转，怕干转；不怕忙，怕进脏。

# 14 常见故障排查

> [!info] 核心结论
> 滚珠丝杠故障排查按 **安装、润滑、间隙、支撑、负载、污染** 的顺序进行。

## ① 分类/公式/规则

> [!note] 排查顺序
> 先空载手动移动，再带载运行；  
> 先查润滑和异物，再查预压和轴承座；  
> 先查机械，再怀疑电机和驱动参数。

> [!warning] 排错公式
> **丝杠异常 = 安装不同轴 + 润滑不足 + 反向间隙 + 轴承损坏 + 负载过大 + 异物进入**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **运动发紧** | 安装不同轴、预压过大 |
| **反向误差大** | 间隙、联轴器松动 |
| **噪声变大** | 缺油、滚道损伤 |
| **局部卡顿** | 异物进入、丝杠弯曲 |
| **定位不稳** | 支撑松动、螺母磨损 |
| **高速振动** | 临界转速、同轴度差 |
| **发热明显** | 预压大、润滑差、速度高 |

## ③ 计算/选型示例

- 已知：  
  滑台同一位置反复卡顿，其他位置正常。
- 公式：  
  **局部卡顿 = 局部滚道损伤 / 异物 / 丝杠弯曲**
- 计算：  
  若每次都在同一位置异常，优先怀疑机械局部问题。
- 结论：  
  应检查该段丝杠滚道、螺母循环器、防尘密封和润滑情况。

> [!tip] 记忆口诀
> 全程紧查安装，局部卡查异物；反向偏查间隙，高速抖查临界。

# 15 滚珠丝杠快速选型表

> [!info] 核心结论
> 滚珠丝杠选型先确定负载、速度和精度，再校核导程、直径、支撑方式、寿命和临界转速。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 看运动方向：水平/垂直  
> ② 确认有效行程  
> ③ 计算轴向负载和加速力  
> ④ 确定目标速度  
> ⑤ 选择导程  
> ⑥ 初选丝杠直径  
> ⑦ 选择精度等级和预压  
> ⑧ 选择支撑方式  
> ⑨ 校核临界转速、压杆稳定和寿命  
> ⑩ 配置润滑、防护、轴承座和联轴器

> [!warning] 最终选型公式
> **推荐丝杠 = 丝杠直径 + 导程 + 精度等级 + 预压等级 + 螺母形式 + 支撑方式 + 防护润滑**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 高精度定位 | 小导程 + 预压螺母 |
| 高速搬运 | 大导程 + 校核临界转速 |
| 重载推压 | 大直径 + 高刚性支撑 |
| 垂直升降 | 校核重力、抱闸、防坠 |
| 长行程 | 固定-固定支撑优先评估 |
| 低成本通用 | 轧制滚珠丝杠 |
| 高精密设备 | 研磨滚珠丝杠 |

## ③ 计算/选型示例

- 已知：  
  需要速度400mm/s，导程10mm，轴向推力600N，效率0.9。
- 公式：  
  **rpm = 速度 ÷ 导程 × 60**  
  **T = F × L ÷（2π × η）**
- 计算：  
  rpm = 400 ÷ 10 × 60 = **2400rpm**  
  T = 600 × 0.01 ÷（2 × 3.14 × 0.9）≈ **1.06N·m**
- 结论：  
  电机需满足约 **2400rpm** 和 **1.06N·m以上转矩**，同时校核丝杠临界转速、寿命和支撑方式。

> [!tip] 选型口诀
> 先定行程负载，再定速度导程；精度看等级，刚性看预压；长丝杠查临界，重推力查稳定。
