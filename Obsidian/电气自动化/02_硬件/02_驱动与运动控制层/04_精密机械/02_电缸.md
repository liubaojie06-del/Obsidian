# 01 电缸总览

> [!info] 核心结论
> 电缸是把电机旋转运动转换成直线推拉运动的执行机构，适合 **高精度定位、可控推力、可编程行程、替代气缸/液压缸** 的场景。

## ① 分类/公式/规则

> [!warning] 选型公式
> **电缸选型 = 推力 + 行程 + 速度 + 精度 + 负载方向 + 电机类型 + 丝杠类型 + 安装方式**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **推力** | 决定能推多大负载 |
| **行程** | 决定直线移动距离 |
| **速度** | 决定节拍 |
| **精度** | 决定定位能力 |
| **丝杠** | 决定推力、速度、寿命 |
| **电机** | 伺服/步进/直流等 |
| **安装方式** | 水平、垂直、侧装、铰接 |

## ③ 计算/选型示例

- 已知：  
  设备需要直线推压，推力要求1000N，行程200mm，并且要可调位置。
- 公式：  
  **可控推力 + 可控位置 → 电缸优先**
- 计算：  
  气缸位置控制较粗，电缸可通过电机和编码器实现位置、速度和推力控制。
- 结论：  
  可选择 **伺服电缸**，具体推力、速度和寿命需按厂家手册校核。

> [!tip] 记忆口诀
> 气缸靠气压，电缸靠电机；要精度、要可控，就看电缸。

# 02 底层原理

> [!info] 核心结论
> 电缸的本质，是用电机驱动丝杠或同步带，把旋转运动转换成直线运动。

## ① 分类/公式/规则

> [!note] 工作链路
> 控制器给指令，驱动器控制电机旋转，电机带动丝杠，丝杠推动螺母或推杆做直线运动。

> [!warning] 运动链路
> **控制信号 → 驱动器 → 电机旋转 → 丝杠/传动机构 → 推杆直线运动 → 负载动作**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **电机** | 提供动力 |
| **驱动器** | 控制速度、位置、转矩 |
| **丝杠** | 旋转变直线 |
| **推杆** | 输出直线推拉力 |
| **导向机构** | 保证直线运动 |
| **限位/传感器** | 原点、行程、安全保护 |
| **编码器** | 反馈位置和速度 |

## ③ 计算/选型示例

- 已知：  
  丝杠导程10mm，电机转5圈。
- 公式：  
  **直线位移 = 电机转数 × 丝杠导程**
- 计算：  
  直线位移 = 5 × 10 = **50mm**
- 结论：  
  电机转5圈，电缸推杆理论移动50mm。

> [!tip] 记忆口诀
> 电机负责转，丝杠负责变，推杆负责直线干活。

# 03 常见类型

> [!info] 核心结论
> 电缸常见有伺服电缸、步进电缸、直流电缸、折返式电缸、直连式电缸和缸筒式电缸。

## ① 分类/公式/规则

> [!warning] 类型选择规则
> **高精度高响应 → 伺服电缸**  
> **低成本简单定位 → 步进电缸**  
> **简单伸缩动作 → 直流电动推杆**  
> **安装长度受限 → 折返式电缸**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **伺服电缸** | 精度高、响应快、可闭环 |
| **步进电缸** | 成本低、控制简单 |
| **直流电缸** | 简单推拉，常用于低速动作 |
| **直连式电缸** | 电机与丝杠同轴，结构直接 |
| **折返式电缸** | 电机并排布置，长度短 |
| **缸筒式电缸** | 外形接近气缸，易替换 |

## ③ 计算/选型示例

- 已知：  
  设备空间短，电机不能沿推杆方向伸出太长。
- 公式：  
  **安装长度受限 → 折返式结构**
- 计算：  
  折返式电缸通过同步带或齿轮把电机布置在侧面。
- 结论：  
  可选择 **折返式伺服电缸**。

> [!tip] 记忆口诀
> 精准用伺服，省钱用步进，简单推拉用直流，空间短用折返。

# 04 电缸 vs 气缸

> [!info] 核心结论
> 气缸适合简单快速动作，电缸适合精确位置、速度和推力控制。

## ① 分类/公式/规则

> [!note] 快速区别
> 气缸靠压缩空气推动，结构简单、动作快；  
> 电缸靠电机控制，位置、速度、推力可编程。

> [!warning] 选择规则
> **两点动作、成本敏感 → 气缸**  
> **多位置、精密控制、力控要求 → 电缸**

## ② 参考表

| 对比项 | 气缸 | 电缸 |
|---|---|---|
| 动力源 | 压缩空气 | 电机 |
| 位置控制 | 较粗 | **精确** |
| 推力控制 | 靠气压 | **可闭环控制** |
| 中间位置 | 较难 | **容易实现** |
| 能耗 | 依赖压缩空气 | 按运动用电 |
| 维护 | 气源、密封 | 丝杠、润滑、电机 |

## ③ 计算/选型示例

- 已知：  
  设备需要三个不同位置停止。
- 公式：  
  **多位置停止 → 电缸优先**
- 计算：  
  气缸做多位置控制需要复杂中停机构或多缸组合。
- 结论：  
  电缸更适合多位置可编程控制。

> [!tip] 记忆口诀
> 气缸适合“到位就行”，电缸适合“到哪都准”。

# 05 电缸 vs 液压缸

> [!info] 核心结论
> 液压缸适合超大推力，电缸适合清洁、可控、低维护和精密控制场景。

## ① 分类/公式/规则

> [!note] 快速区别
> 液压缸推力大，但需要液压站、油路和密封维护；  
> 电缸结构更清洁，控制更方便，适合自动化设备。

> [!warning] 选择规则
> **超大推力/重载冲击 → 液压缸**  
> **清洁环境/精密控制 → 电缸**

## ② 参考表

| 对比项 | 液压缸 | 电缸 |
|---|---|---|
| 推力能力 | **很强** | 中小到较大 |
| 清洁度 | 可能漏油 | **清洁** |
| 控制精度 | 需伺服液压 | **易闭环** |
| 系统复杂度 | 液压站、阀组 | 驱动器、电机 |
| 维护点 | 油液、密封、管路 | 丝杠、润滑 |
| 应用 | 压机、重载 | 压装、定位、搬运 |

## ③ 计算/选型示例

- 已知：  
  食品包装设备需要推压动作，不允许油污污染。
- 公式：  
  **洁净环境 + 可控推压 → 电缸优先**
- 计算：  
  液压系统存在漏油风险，电缸更适合洁净设备。
- 结论：  
  可选择电缸替代液压缸，推力需按工艺需求校核。

> [!tip] 记忆口诀
> 要大力看液压，要干净精准看电缸。

# 06 推力计算

> [!info] 核心结论
> 电缸推力由电机转矩、丝杠导程和传动效率共同决定。

## ① 分类/公式/规则

> [!warning] 转矩推力公式
> **F = 2π × η × T ÷ L**

> [!note] 参数说明
> **F**：轴向推力，N  
> **η**：传动效率，需按厂家手册校核  
> **T**：电机输出转矩，N·m  
> **L**：丝杠导程，m/rev

## ② 参考表

| 参数 | 影响 |
|---|---|
| **电机转矩T** | 越大推力越大 |
| **导程L** | 越小推力越大 |
| **效率η** | 越高损耗越小 |
| **摩擦阻力** | 增加实际需求 |
| **加速度** | 增加峰值推力 |
| **安全系数** | 防止过载 |

## ③ 计算/选型示例

- 已知：  
  电机转矩2N·m，丝杠导程10mm，效率0.9。
- 公式：  
  **F = 2π × 0.9 × 2 ÷ 0.01**
- 计算：  
  F ≈ **1130N**
- 结论：  
  理论推力约1130N，实际需按加速度、摩擦、寿命和厂家手册校核。

> [!tip] 记忆口诀
> 转矩越大越有力，导程越小越能推。

# 07 速度与导程

> [!info] 核心结论
> 电缸速度由电机转速和丝杠导程决定，导程越大，同转速下速度越快。

## ① 分类/公式/规则

> [!warning] 速度公式
> **直线速度 = 电机转速 × 导程 ÷ 60**

> [!warning] 转速公式
> **电机转速 rpm = 直线速度 ÷ 导程 × 60**

## ② 参考表

| 导程选择 | 特点 |
|---|---|
| **小导程** | 推力大、速度低、分辨率高 |
| **大导程** | 速度快、推力相对小 |
| **高精度** | 常用较小导程 |
| **高速度** | 常用较大导程 |
| **长行程高速** | 需校核临界转速 |
| **最终选择** | 推力和速度一起校核 |

## ③ 计算/选型示例

- 已知：  
  目标速度400mm/s，丝杠导程10mm。
- 公式：  
  **rpm = 400 ÷ 10 × 60**
- 计算：  
  rpm = **2400rpm**
- 结论：  
  电机需要约2400rpm，同时需校核丝杠允许转速。

> [!tip] 记忆口诀
> 导程大跑得快，导程小推得狠。

# 08 行程与安装长度

> [!info] 核心结论
> 电缸选型不能只看有效行程，还要看本体长度、伸出长度、安装空间和安全余量。

## ① 分类/公式/规则

> [!note] 工程理解
> 电缸行程越长，本体通常越长。  
> 直连式电缸还要考虑电机长度，折返式电缸可缩短总长度。

> [!warning] 安装公式
> **总安装空间 ≥ 电缸本体长度 + 有效行程 + 接头/安装附件空间**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **有效行程** | 实际可运动距离 |
| **本体长度** | 电缸缩回时长度 |
| **伸出长度** | 推杆伸出后总长度 |
| **安全余量** | 避免撞端 |
| **安装附件** | 法兰、耳轴、鱼眼接头 |
| **电机方向** | 直连或折返影响空间 |

## ③ 计算/选型示例

- 已知：  
  设备内部可用长度有限，但需要300mm行程。
- 公式：  
  **空间受限 → 优先评估折返式电缸**
- 计算：  
  折返式把电机布置在侧面，可减少轴向长度。
- 结论：  
  可选择折返式电缸，并校核宽度方向空间。

> [!tip] 记忆口诀
> 行程看能走多远，安装看能不能放得下。

# 09 丝杠类型

> [!info] 核心结论
> 电缸常用滚珠丝杠和梯形丝杠，滚珠丝杠效率高、精度好，梯形丝杠成本低、自锁性更好。

## ① 分类/公式/规则

> [!warning] 类型选择规则
> **高效率、高速度、高精度 → 滚珠丝杠**  
> **低速、低成本、一定自锁需求 → 梯形丝杠**

## ② 参考表

| 对比项 | 滚珠丝杠 | 梯形丝杠 |
|---|---|---|
| 效率 | **高** | 较低 |
| 精度 | **较好** | 一般 |
| 速度 | **较高** | 较低 |
| 成本 | 较高 | 较低 |
| 自锁性 | 通常较弱 | 较好 |
| 维护 | 需润滑防尘 | 也需润滑维护 |

## ③ 计算/选型示例

- 已知：  
  设备需要频繁高速往复定位。
- 公式：  
  **高速高频 → 滚珠丝杠优先**
- 计算：  
  滚珠丝杠效率高、发热低，更适合频繁运动。
- 结论：  
  推荐选择 **滚珠丝杠电缸**。

> [!tip] 记忆口诀
> 要快要准用滚珠，要省要稳看梯形。

# 10 精度与重复定位

> [!info] 核心结论
> 电缸精度由丝杠、导向、编码器、安装刚性和控制参数共同决定。

## ① 分类/公式/规则

> [!note] 精度理解
> 分辨率不等于定位精度。  
> 分辨率只是控制系统能分得多细，实际精度还受机械误差影响。

> [!warning] 误差公式
> **实际误差 = 丝杠误差 + 反向间隙 + 机械变形 + 编码器误差 + 控制误差**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **丝杠精度** | 影响位移准确性 |
| **反向间隙** | 影响换向定位 |
| **预压** | 提高重复定位稳定性 |
| **导向刚性** | 影响偏载下变形 |
| **编码器分辨率** | 影响反馈细分 |
| **温升** | 影响长时间精度 |

## ③ 计算/选型示例

- 已知：  
  理论脉冲当量0.001mm，但实际重复精度达不到0.001mm。
- 公式：  
  **脉冲当量 ≠ 实际定位精度**
- 计算：  
  丝杠间隙、机械变形和导轨误差都会影响实际结果。
- 结论：  
  需综合校核丝杠精度、预压、导向和安装刚性。

> [!tip] 记忆口诀
> 分辨率是能分多细，精度是实际到多准。

# 11 负载方向与安装方式

> [!info] 核心结论
> 电缸水平、垂直、侧向和铰接安装时受力不同，垂直轴尤其要考虑重力和抱闸。

## ① 分类/公式/规则

> [!warning] 负载校核公式
> **实际推力需求 = 工艺力 + 重力分量 + 摩擦力 + 加速惯性力**

## ② 参考表

| 安装方式 | 重点 |
|---|---|
| **水平安装** | 摩擦、加速度、推力 |
| **垂直安装** | 重力、抱闸、防坠 |
| **侧向安装** | 偏载、导向刚性 |
| **铰接安装** | 角度变化、侧向力 |
| **法兰安装** | 同轴度和刚性 |
| **耳轴安装** | 摆动机构常见 |

## ③ 计算/选型示例

- 已知：  
  垂直提升负载20kg。
- 公式：  
  **重力 F = m × g**
- 计算：  
  F = 20 × 9.8 = **196N**
- 结论：  
  电缸至少要克服196N重力，还需叠加加速力、摩擦和安全余量，并考虑抱闸或防坠。

> [!tip] 记忆口诀
> 水平看摩擦，垂直看重力；偏心看导向，断电看抱闸。

# 12 抱闸与防坠

> [!info] 核心结论
> 垂直电缸、升降轴和压紧保持场景，必须考虑断电下滑风险，常需电机抱闸或机械防坠。

## ① 分类/公式/规则

> [!note] 抱闸理解
> 抱闸主要用于断电保持，不是用来频繁动态刹车。  
> 动作时应先建立电机转矩，再释放抱闸。

> [!warning] 动作逻辑
> **使能建立转矩 → 松开抱闸 → 电缸运动**  
> **停止保持 → 抱闸闭合 → 取消使能**

## ② 参考表

| 场景 | 处理方式 |
|---|---|
| **垂直升降** | 抱闸重点考虑 |
| **重物保持** | 抱闸 + 机械防坠 |
| **断电防下滑** | 抱闸或自锁结构 |
| **频繁急停** | 不能只靠抱闸 |
| **安全场景** | 需风险评估 |
| **梯形丝杠** | 可能具备一定自锁，需校核 |

## ③ 计算/选型示例

- 已知：  
  Z轴电缸断电后负载可能下滑。
- 公式：  
  **断电保持 = 抱闸保持力矩 + 机械安全**
- 计算：  
  断电后驱动器不再主动输出转矩，必须有保持机构。
- 结论：  
  应选择 **带抱闸电机电缸**，保持力矩需按负载校核。

> [!tip] 记忆口诀
> 垂直轴不怕动，怕断电掉；抱闸防下滑，安全不能省。

# 13 控制方式

> [!info] 核心结论
> 电缸可实现位置控制、速度控制、推力控制和多段程序动作，控制方式取决于电机和驱动器。

## ① 分类/公式/规则

> [!note] 控制模式
> 位置模式：控制走到哪里。  
> 速度模式：控制跑多快。  
> 转矩/推力模式：控制出多大力。  
> 多点定位：按程序走多个位置。

> [!warning] 模式选择规则
> **定位 → 位置模式**  
> **连续推送 → 速度模式**  
> **压装/夹紧 → 推力模式或转矩限制**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **脉冲控制** | 简单定位 |
| **模拟量控制** | 速度/推力给定 |
| **总线控制** | 多轴联动、参数诊断 |
| **位置模式** | 多点定位 |
| **速度模式** | 平稳推送 |
| **推力模式** | 压装、夹紧、测试 |

## ③ 计算/选型示例

- 已知：  
  电缸需要压到工件后保持固定压力。
- 公式：  
  **压装保持 = 位置控制 + 推力限制/力控反馈**
- 计算：  
  只用位置控制可能因工件尺寸差异导致力不稳定。
- 结论：  
  应选择支持 **推力控制或转矩限制** 的伺服电缸方案。

> [!tip] 记忆口诀
> 到哪里用位置，跑多快用速度，压多重用推力。

# 14 传感器与限位

> [!info] 核心结论
> 电缸必须配置原点、限位和必要的到位反馈，防止超程撞机和位置丢失。

## ① 分类/公式/规则

> [!note] 传感器作用
> 原点用于建立坐标；  
> 正负限位用于防止越界；  
> 编码器用于反馈位置；  
> 压力/力传感器用于闭环力控。

> [!warning] 安全位置逻辑
> **原点建坐标，软限位管范围，硬限位保安全**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **原点开关** | 建立零点 |
| **正限位** | 防止正向超程 |
| **负限位** | 防止反向超程 |
| **编码器** | 位置和速度反馈 |
| **力传感器** | 推力闭环检测 |
| **机械挡块** | 最后保护 |

## ③ 计算/选型示例

- 已知：  
  电缸有效行程300mm，程序可能误发超出行程指令。
- 公式：  
  **安全行程 < 机械极限行程**
- 计算：  
  应设置软件限位，并保留硬限位和机械缓冲空间。
- 结论：  
  电缸系统应配置 **软限位 + 硬限位 + 原点回零**。

> [!tip] 记忆口诀
> 原点定零，限位保命；没有限位，撞机迟早会来。

# 15 维护与润滑

> [!info] 核心结论
> 电缸维护重点是丝杠润滑、防尘、联轴器锁紧、导向磨损、电机线缆和限位开关状态。

## ① 分类/公式/规则

> [!note] 维护重点
> 电缸虽然比液压系统清洁，但内部丝杠和导向仍需要润滑维护。  
> 粉尘、切屑、水汽进入会明显缩短寿命。

> [!warning] 维护公式
> **长寿命 = 正确润滑 + 防尘密封 + 不超载 + 正确安装**

## ② 参考表

| 项目 | 检查内容 |
|---|---|
| **丝杠润滑** | 是否缺油、干涩 |
| **导向机构** | 是否卡滞、磨损 |
| **联轴器** | 是否松动 |
| **电机线缆** | 是否拉扯、破损 |
| **限位开关** | 是否可靠触发 |
| **防护罩** | 是否破损进尘 |

## ③ 计算/选型示例

- 已知：  
  电缸运行噪声变大，推杆有轻微卡顿。
- 公式：  
  **噪声 + 卡顿 = 润滑不足 / 异物 / 导向磨损**
- 计算：  
  应先检查润滑、防尘和导向，而不是直接更换电机。
- 结论：  
  需补充指定润滑脂，并检查丝杠、导向和防护结构。

> [!tip] 记忆口诀
> 电缸不怕用，怕干跑、脏跑、超载跑。

# 16 常见故障排查

> [!info] 核心结论
> 电缸故障排查按 **电源驱动、参数、机械传动、限位反馈、负载** 的顺序进行。

## ① 分类/公式/规则

> [!note] 排查顺序
> 先看驱动器报警，再看限位和使能；  
> 先空载低速运行，再带载测试；  
> 先查机械卡滞，再查控制参数。

> [!warning] 排错公式
> **电缸异常 = 驱动报警 + 参数错误 + 机械卡滞 + 限位触发 + 负载过大 + 润滑不足**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **不动作** | 无使能、报警、限位触发 |
| **动作慢** | 速度参数低、负载大、机械阻力 |
| **定位不准** | 脉冲当量错、间隙、编码器异常 |
| **推力不足** | 电机转矩不足、导程不合适 |
| **运行抖动** | 增益不当、机构共振 |
| **噪声大** | 缺油、丝杠损伤、同轴度差 |
| **过载报警** | 负载过大、卡滞、加速度太高 |

## ③ 计算/选型示例

- 已知：  
  电缸空载正常，带载后过载报警。
- 公式：  
  **带载过载 = 负载推力需求 > 电缸可用推力**
- 计算：  
  空载正常说明驱动和基本机构可运行，带载报警重点查负载、推力和加速度。
- 结论：  
  应降低加速度、检查机械卡滞，或选更大推力电缸。

> [!tip] 记忆口诀
> 空载正常带载报警，先查负载和推力。

# 17 电缸快速选型表

> [!info] 核心结论
> 电缸选型要从工艺动作出发，先算推力和速度，再定行程、精度、丝杠、电机和安装方式。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 看动作形式：推、拉、压、顶升  
> ② 确认安装方向：水平/垂直/铰接  
> ③ 计算最大推力和峰值推力  
> ④ 确认有效行程  
> ⑤ 确认最高速度和加速度  
> ⑥ 选择丝杠类型和导程  
> ⑦ 选择电机：伺服/步进/直流  
> ⑧ 确认精度和重复定位要求  
> ⑨ 配置限位、原点、抱闸和反馈  
> ⑩ 校核寿命、润滑、防护和厂家手册

> [!warning] 最终选型公式
> **推荐电缸 = 推力等级 + 行程 + 速度 + 丝杠导程 + 电机类型 + 安装方式 + 反馈限位 + 防护等级**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 高精度定位 | 伺服电缸 |
| 低成本定位 | 步进电缸 |
| 简单伸缩 | 直流电动推杆 |
| 高速高频 | 滚珠丝杠电缸 |
| 低速自锁 | 梯形丝杠电缸 |
| 垂直升降 | 抱闸 + 防坠 |
| 压装控制 | 推力控制 + 力传感器 |

## ③ 计算/选型示例

- 已知：  
  推力需求800N，速度300mm/s，导程10mm，效率0.9。
- 公式：  
  **rpm = 速度 ÷ 导程 × 60**  
  **T = F × L ÷（2π × η）**
- 计算：  
  rpm = 300 ÷ 10 × 60 = **1800rpm**  
  T = 800 × 0.01 ÷（2 × 3.14 × 0.9）≈ **1.42N·m**
- 结论：  
  电机需满足约 **1800rpm** 和 **1.42N·m以上转矩**，并按加速度、寿命和安全余量校核。

> [!tip] 选型口诀
> 先算推力，再算速度；导程定快慢，电机定力气；垂直要抱闸，压装要力控。
