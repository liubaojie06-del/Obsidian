# 01 低压直流伺服总览

> [!info] 核心结论
> 低压直流伺服通常使用 **DC24V、DC48V、DC60V、DC72V 等直流母线供电**，适合小型设备、移动机器人、AGV、低压安全设备和电池供电场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 低压直流伺服不是“普通直流电机”，而是由 **直流伺服驱动器 + 伺服电机 + 编码器反馈 + 控制器** 组成的闭环运动系统。

> [!warning] 🟧 通用选型公式
> **低压直流伺服选型 = 母线电压 + 电机功率 + 峰值电流 + 转矩需求 + 速度需求 + 编码器类型 + 通讯方式 + 供电能力**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **供电方式** | DC24V / 48V / 60V / 72V 等，需按厂家手册校核 |
| **控制对象** | 低压伺服电机、无刷伺服电机、轮毂伺服等 |
| **核心功能** | 位置、速度、转矩闭环控制 |
| **典型应用** | AGV、AMR、小型机械臂、医疗设备、移动平台 |
| **重点参数** | 电压、电流、转矩、转速、编码器、通讯 |
| **风险点** | 大电流、压降、再生过压、接线极性错误 |

## ③ 计算/选型示例

- 已知：  
  设备使用DC48V电池供电，需要驱动一个移动轮实现速度闭环。
- 公式：  
  **系统需求 = DC48V供电 + 速度闭环 + 编码器反馈 + 控制器通讯**
- 计算：  
  普通电机只能粗略调速，低压直流伺服可通过编码器反馈实现稳定速度控制。
- 结论：  
  可选择 **DC48V低压直流伺服驱动器 + 匹配伺服电机**，电流和通讯方式需按厂家手册校核。

> [!tip] 记忆口诀
> 低压供电看电流，精准运动看闭环。

# 02 底层原理

> [!info] 核心结论
> 低压直流伺服的本质，是把直流电通过驱动器逆变成可控三相电流，再驱动电机精准运动。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 控制器发出目标位置、速度或转矩指令；驱动器根据编码器反馈计算偏差，并调节电机电流，让实际运动跟随目标指令。

> [!warning] 🟧 闭环控制逻辑
> **目标指令 → 驱动器计算 → 电机输出转矩 → 编码器反馈 → 偏差修正**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **直流母线** | 给驱动器提供能量 |
| **逆变功率模块** | 输出三相电流给电机 |
| **电流环** | 控制转矩 |
| **速度环** | 控制转速 |
| **位置环** | 控制位置 |
| **编码器反馈** | 返回实际位置和速度 |

## ③ 计算/选型示例

- 已知：  
  目标速度为1000rpm，实际速度为960rpm。
- 公式：  
  **速度偏差 = 目标速度 - 实际速度**
- 计算：  
  速度偏差 = 1000 - 960 = **40rpm**
- 结论：  
  驱动器会根据速度偏差增加输出电流，使电机速度接近目标值。

> [!tip] 记忆口诀
> 指令给目标，编码器给反馈，驱动器负责修正。

# 03 低压直流伺服 vs 交流伺服

> [!info] 核心结论
> 低压直流伺服适合低压供电和移动设备；交流伺服适合固定设备、高功率和标准工业供电场景。

## ① 分类/公式/规则

> [!note] 🟦 快速区别
> 低压直流伺服多用于 **电池供电、小功率、移动平台**；  
> 交流伺服多用于 **AC220V / AC380V 固定式设备**。

> [!warning] 🟧 选择公式
> **电池供电 / 低压安全 / 移动设备 → 低压直流伺服**  
> **大功率 / 固定产线 / 高动态设备 → 交流伺服**

## ② 参考表

| 对比项 | 低压直流伺服 | 交流伺服 |
|---|---|---|
| 供电 | **DC低压** | AC220V / AC380V常见 |
| 应用 | 移动设备、小型机构 | 工业设备、产线设备 |
| 功率 | 中小功率常见 | 覆盖范围更大 |
| 电流 | 同功率下电流较大 | 同功率下电流较小 |
| 安装 | 适合电池系统 | 适合电柜系统 |
| 重点 | 压降、供电容量、再生能量 | 电源等级、制动、EMC |

## ③ 计算/选型示例

- 已知：  
  一个AGV小车使用48V电池，要求轮子速度闭环控制。
- 公式：  
  **移动设备 + 电池供电 = 低压直流伺服优先**
- 计算：  
  若使用交流伺服，需要逆变供电，系统复杂度更高。
- 结论：  
  该场景优先选择 **低压直流伺服系统**。

> [!tip] 记忆口诀
> 固定设备多用交流，移动设备多看直流。

# 04 母线电压选择

> [!info] 核心结论
> 母线电压决定伺服系统的速度能力和电流大小；同样功率下，电压越低，电流越大。

## ① 分类/公式/规则

> [!warning] 🟧 功率公式
> **P ≈ U × I × η**

> [!note] 🟦 参数说明
> **P**：输出功率  
> **U**：直流母线电压  
> **I**：母线电流  
> **η**：效率  
>  
> 实际电流还与加速、负载、控制策略有关，需按厂家手册校核。

## ② 参考表

| 电压等级 | 常见特点 |
|---|---|
| **DC24V** | 小功率、低速、小型机构 |
| **DC48V** | AGV、AMR、移动平台常见 |
| **DC60V/72V** | 速度和功率能力更高 |
| **低电压** | 安全性较好，但电流大 |
| **高电压** | 电流较小，高速性能更好 |
| **重点校核** | 驱动器允许电压范围 |

## ③ 计算/选型示例

- 已知：  
  伺服系统机械功率约400W，母线电压48V，效率按0.85估算。
- 公式：  
  **I ≈ P ÷（U × η）**
- 计算：  
  I ≈ 400 ÷（48 × 0.85）≈ **9.8A**
- 结论：  
  母线电流约为 **9.8A**，电源和线缆应留有余量。

> [!tip] 记忆口诀
> 电压低，电流大；功率越高，线缆越要粗。

# 05 驱动器选型

> [!info] 核心结论
> 低压直流伺服驱动器选型重点看 **供电电压、连续电流、峰值电流、控制模式、编码器接口和通讯协议**。

## ① 分类/公式/规则

> [!note] 🟦 驱动器重点
> 低压伺服驱动器必须与电机、编码器和控制器匹配。  
> 同一电压等级下，电流能力决定可驱动的转矩和加速能力。

> [!warning] 🟧 驱动器选型公式
> **驱动器规格 = 母线电压 + 连续电流 + 峰值电流 + 编码器接口 + 通讯接口**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **输入电压** | DC24V / 48V / 72V等 |
| **连续电流** | 长时间输出能力 |
| **峰值电流** | 加速、冲击时短时输出 |
| **控制模式** | 位置、速度、转矩 |
| **编码器接口** | 增量、霍尔、绝对值等 |
| **通讯接口** | 脉冲、CANopen、EtherCAT、RS485等 |

## ③ 计算/选型示例

- 已知：  
  电机连续电流为8A，峰值电流为24A，系统供电为DC48V。
- 公式：  
  **驱动器连续电流 ≥ 8A**  
  **驱动器峰值电流 ≥ 24A**
- 计算：  
  驱动器必须满足DC48V输入，并覆盖电机连续和峰值电流。
- 结论：  
  选择时不能只看功率，必须同时校核 **连续电流和峰值电流**。

> [!tip] 记忆口诀
> 长期看连续电流，加速看峰值电流。

# 06 电机选型

> [!info] 核心结论
> 低压直流伺服电机选型要看额定转矩、峰值转矩、额定转速、最高转速、编码器和安装尺寸。

## ① 分类/公式/规则

> [!note] 🟦 电机重点
> 电机功率只是初步参考；真正决定能不能带动负载的是 **转矩、速度和热容量**。

> [!warning] 🟧 电机选型公式
> **电机需求 = 连续转矩 + 峰值转矩 + 最高转速 + 惯量匹配 + 安装结构**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定转矩** | 长时间运行能力 |
| **峰值转矩** | 加速或短时冲击能力 |
| **额定转速** | 常用运行速度 |
| **最高转速** | 短时速度上限 |
| **电机惯量** | 影响响应和稳定性 |
| **编码器** | 反馈位置和速度 |

## ③ 计算/选型示例

- 已知：  
  负载连续需要0.8N·m，启动加速瞬间需要2.4N·m。
- 公式：  
  **额定转矩 ≥ 连续转矩**  
  **峰值转矩 ≥ 加速转矩**
- 计算：  
  电机额定转矩应≥0.8N·m，峰值转矩应≥2.4N·m。
- 结论：  
  若峰值转矩不足，启动或加速时容易跟随误差过大或报警。

> [!tip] 记忆口诀
> 持续运行看额定，启动加速看峰值。

# 07 编码器与霍尔反馈

> [!info] 核心结论
> 编码器用于精确反馈位置和速度，霍尔信号常用于电机换相或低成本位置检测，具体接口必须与驱动器匹配。

## ① 分类/公式/规则

> [!note] 🟦 反馈类型
> 低压直流伺服常见反馈包括：增量编码器、绝对值编码器、霍尔传感器、磁编码器等。  
> 反馈类型不同，控制精度和接线方式不同。

> [!warning] 🟧 反馈匹配公式
> **反馈可用 = 电机反馈类型 + 驱动器反馈接口 + 参数设置一致**

## ② 参考表

| 反馈类型 | 特点 |
|---|---|
| **霍尔信号** | 常用于无刷电机换相 |
| **增量编码器** | 速度/位置反馈常见 |
| **绝对值编码器** | 上电可读当前位置 |
| **磁编码器** | 结构紧凑，抗污染较好 |
| **差分编码器** | 抗干扰较好 |
| **单端编码器** | 接线简单，抗干扰较弱 |

## ③ 计算/选型示例

- 已知：  
  电机编码器分辨率为 **4096 counts/rev**。
- 公式：  
  **每计数角度 = 360° ÷ 4096**
- 计算：  
  每计数角度 ≈ **0.0879°**
- 结论：  
  编码器每变化1个计数，理论角度变化约 **0.0879°**。

> [!tip] 记忆口诀
> 霍尔管换相，编码器管精度；接口不匹配，伺服跑不稳。

# 08 三种控制模式

> [!info] 核心结论
> 低压直流伺服常用位置、速度、转矩三种控制模式，应用场景不同。

## ① 分类/公式/规则

> [!note] 🟦 模式区别
> **位置模式**：控制到指定位置。  
> **速度模式**：控制目标速度。  
> **转矩模式**：控制输出力矩或电流。

> [!warning] 🟧 模式选择公式
> **定位机构 → 位置模式**  
> **移动底盘/输送 → 速度模式**  
> **张力/压紧/力控 → 转矩模式**

## ② 参考表

| 控制模式 | 典型应用 |
|---|---|
| **位置控制** | 滑台、转台、机械臂关节 |
| **速度控制** | AGV车轮、输送机构 |
| **转矩控制** | 张力、压紧、夹持 |
| **脉冲控制** | 简单定位 |
| **CANopen** | 移动机器人、分布式控制 |
| **EtherCAT** | 多轴同步、高速控制 |

## ③ 计算/选型示例

- 已知：  
  AGV驱动轮需要保持目标行驶速度。
- 公式：  
  **移动轮控制 → 速度模式**
- 计算：  
  控制器给定目标速度，驱动器根据编码器反馈稳定轮速。
- 结论：  
  该场景优先使用 **速度控制模式**。

> [!tip] 记忆口诀
> 到哪里用位置，跑多快用速度，出多大力用转矩。

# 09 通讯与控制接口

> [!info] 核心结论
> 低压直流伺服常见控制接口有脉冲、模拟量、CANopen、RS485、EtherCAT 等，选型时必须与控制器匹配。

## ① 分类/公式/规则

> [!note] 🟦 接口选择
> 单轴简单定位可用脉冲；  
> 移动机器人常用CANopen；  
> 多轴同步和高速运动控制优先实时总线。

> [!warning] 🟧 通讯匹配公式
> **系统可控 = 控制器协议 + 驱动器协议 + 波特率/站号 + 数据映射一致**

## ② 参考表

| 接口 | 适合场景 |
|---|---|
| **PUL/DIR脉冲** | 简单位置控制 |
| **0-10V模拟量** | 速度给定 |
| **RS485/Modbus** | 参数读写、简单控制 |
| **CANopen** | AGV、机器人、分布式控制 |
| **EtherCAT** | 多轴同步、高实时性 |
| **数字I/O** | 使能、报警、限位、复位 |

## ③ 计算/选型示例

- 已知：  
  一台移动机器人有4个驱动轮，每个轮子都要独立速度控制。
- 公式：  
  **多驱动轮 = 多轴通讯 + 状态反馈 + 故障诊断**
- 计算：  
  使用脉冲控制接线复杂，CANopen或EtherCAT更适合集中管理。
- 结论：  
  优先选择支持 **CANopen或EtherCAT** 的低压直流伺服驱动器。

> [!tip] 记忆口诀
> 单轴简单用脉冲，多轴分布用总线。

# 10 电源与电池系统

> [!info] 核心结论
> 低压直流伺服对电源要求很高，电源不仅要电压对，还要电流够、抗冲击强、能处理再生能量。

## ① 分类/公式/规则

> [!note] 🟦 电源重点
> 伺服加速时电流大，减速时可能回灌能量。  
> 电池供电系统还要关注BMS限流、欠压保护、线缆压降和母线电容。

> [!warning] 🟧 电源容量公式
> **电源电流 ≥ 伺服连续电流总和 × 余量系数**

## ② 参考表

| 项目 | 关注点 |
|---|---|
| **开关电源** | 输出电流、峰值能力 |
| **电池供电** | BMS限流、欠压保护 |
| **母线电容** | 缓冲瞬时电流 |
| **保险/断路器** | 短路和过流保护 |
| **线缆截面** | 大电流压降和发热 |
| **再生能量** | 减速时母线电压升高 |

## ③ 计算/选型示例

- 已知：  
  系统有2个伺服轴，每轴连续母线电流约8A，建议留1.5倍余量。
- 公式：  
  **电源电流 ≥ 2 × 8 × 1.5**
- 计算：  
  电源电流 ≥ **24A**
- 结论：  
  电源或电池系统应能稳定提供 **24A以上** 电流，峰值电流还需单独校核。

> [!tip] 记忆口诀
> 低压伺服不怕电压低，怕的是电流不够和压降太大。

# 11 再生制动与母线过压

> [!info] 核心结论
> 低压直流伺服减速或下坡时可能产生再生能量，若无法吸收，会导致直流母线过压报警。

## ① 分类/公式/规则

> [!note] 🟦 再生逻辑
> 电机减速时可能变成发电机，把机械能回灌到驱动器母线。  
> 如果电池、电源或制动电阻不能吸收能量，母线电压就会上升。

> [!warning] 🟧 再生风险公式
> **过压风险 = 大惯量 + 快速减速 + 下坡负载 + 电池不吸收回灌**

## ② 参考表

| 场景 | 风险 |
|---|---|
| **快速减速** | 再生能量大 |
| **AGV下坡** | 轮子带动电机发电 |
| **垂直轴下降** | 重力势能回灌 |
| **电池满电** | BMS可能限制充电 |
| **开关电源供电** | 通常不能吸收回灌 |
| **制动电阻** | 可消耗再生能量，需校核 |

## ③ 计算/选型示例

- 已知：  
  AGV满电状态下下坡，驱动器频繁报母线过压。
- 公式：  
  **过压 = 再生能量 > 电池/BMS吸收能力**
- 计算：  
  满电时电池吸收回灌能力下降，母线电压容易升高。
- 结论：  
  需评估制动电阻、能量回收策略或限速策略，具体按厂家手册校核。

> [!tip] 记忆口诀
> 加速吃电，减速发电；电池不收，母线就升。

# 12 线缆压降与接线

> [!info] 核心结论
> 低压直流伺服电流大，线缆压降和接线可靠性非常关键，线太细会导致发热、掉压和报警。

## ① 分类/公式/规则

> [!note] 🟦 接线重点
> 直流母线正负极不能接反；动力线和编码器线应分开布线；大电流回路要选合适线径和可靠端子。

> [!warning] 🟧 压降判断公式
> **线缆压降 ≈ 电流 × 线缆电阻**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **DC+ / DC-** | 极性不能接反 |
| **动力线** | 线径要满足电流 |
| **编码器线** | 屏蔽、远离动力线 |
| **通讯线** | 双绞屏蔽，终端电阻按需配置 |
| **接地** | 外壳和屏蔽按规范处理 |
| **保险保护** | 靠近电源侧配置更合理 |

## ③ 计算/选型示例

- 已知：  
  母线电流为20A，线缆电阻为0.05Ω。
- 公式：  
  **压降 = I × R**
- 计算：  
  压降 = 20 × 0.05 = **1V**
- 结论：  
  DC48V系统中压降1V已经比较明显，线缆长度和截面积需重点校核。

> [!tip] 记忆口诀
> 低压系统电流大，线细压降就明显。

# 13 参数整定

> [!info] 核心结论
> 低压直流伺服调试要在响应速度、稳定性、噪声、发热和跟随误差之间找平衡。

## ① 分类/公式/规则

> [!note] 🟦 整定重点
> 增益太低，响应慢、跟随误差大；  
> 增益太高，容易振动、啸叫、过冲；  
> 电流限制太小，力矩不足；电流限制太大，温升升高。

> [!warning] 🟧 调试公式
> **稳定运行 = 合适增益 + 合理加减速 + 足够电流 + 机械无卡滞**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **响应慢** | 增益低、加速度小 |
| **振动啸叫** | 增益过高、机械共振 |
| **跟随误差大** | 转矩不足、负载过重 |
| **低速抖动** | 摩擦、编码器噪声、参数不合适 |
| **电机发热** | 电流过大、负载重 |
| **过冲明显** | 刚性过高或负载惯量大 |

## ③ 计算/选型示例

- 已知：  
  低速运行平稳，但加速时跟随误差大。
- 公式：  
  **加速跟随误差 = 加速度要求高 + 转矩不足**
- 计算：  
  可适当增加电流限制、降低加速度，或选更大转矩电机。
- 结论：  
  先确认机械无卡滞，再调整加减速和增益参数。

> [!tip] 记忆口诀
> 慢了调增益，抖了降刚性；加速跟不上，先查转矩够不够。

# 14 常见故障排查

> [!info] 核心结论
> 低压直流伺服故障排查要按“电源电压 → 使能 → 编码器 → 通讯 → 负载 → 参数”的顺序检查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 上电无反应先查DC电源和极性；  
> 运行报警先查负载和编码器；  
> 通讯异常先查站号、波特率、终端电阻和线缆。

> [!warning] 🟧 排错公式
> **伺服异常 = 供电异常 + 极性错误 + 编码器异常 + 通讯异常 + 负载过重 + 参数错误**

## ② 参考表

| 故障/现象 | 常见原因 |
|---|---|
| **不上电** | DC电源无输出、极性错误、保险熔断 |
| **欠压报警** | 电池电压低、线缆压降大 |
| **过压报警** | 再生能量、制动不足 |
| **过流报警** | 负载卡死、电机线短路 |
| **编码器报警** | 线缆松动、接口不匹配 |
| **通讯失败** | 地址、波特率、协议、终端电阻错误 |
| **电机抖动** | 相序、编码器方向、参数错误 |

## ③ 计算/选型示例

- 已知：  
  驱动器在加速时欠压报警。
- 公式：  
  **欠压风险 = 加速电流大 + 电源容量不足 + 线缆压降大**
- 计算：  
  加速瞬间电流升高，电源或电池电压被拉低。
- 结论：  
  应检查电源峰值能力、电池BMS限流、线缆截面和加速度参数。

> [!tip] 记忆口诀
> 欠压查电源和线，过压查减速和回灌，抖动查编码器和相序。

# 15 低压直流伺服快速选型表

> [!info] 核心结论
> 低压直流伺服选型先看母线电压和电流，再看转矩、转速、反馈、通讯和再生能量处理。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看供电电压  
> ② 看负载转矩  
> ③ 看最高转速  
> ④ 看连续电流  
> ⑤ 看峰值电流  
> ⑥ 看编码器类型  
> ⑦ 看控制模式  
> ⑧ 看通讯接口  
> ⑨ 看电源/电池能力  
> ⑩ 看制动、散热和线缆压降

> [!warning] 🟧 最终选型公式
> **推荐系统 = DC电压等级 + 电机转矩/转速 + 驱动器电流 + 编码器 + 通讯接口 + 电源容量 + 制动方案**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| AGV驱动轮 | DC48V常见、速度模式、CANopen |
| 小型滑台 | 位置模式、脉冲或总线 |
| 电池供电 | BMS限流、欠压、回灌 |
| 高加速度 | 峰值电流、峰值转矩 |
| 多轴同步 | EtherCAT或实时总线 |
| 怕丢位置 | 绝对值编码器 |
| 下坡/垂直轴 | 制动电阻和母线过压 |

## ③ 计算/选型示例

- 已知：  
  移动平台使用DC48V，单轴机械功率约300W，效率按0.85估算，峰值功率约900W。
- 公式：  
  **连续电流 ≈ 300 ÷（48 × 0.85）**  
  **峰值电流 ≈ 900 ÷（48 × 0.85）**
- 计算：  
  连续电流 ≈ **7.35A**  
  峰值电流 ≈ **22.1A**
- 结论：  
  驱动器和电源需至少覆盖约 **7.35A连续电流、22.1A峰值电流**，并继续校核转矩、散热和厂家手册。

> [!tip] 选型口诀
> 低压先算电流，运动再算转矩；移动设备看电池，多轴系统看通讯，减速下坡看回灌。
