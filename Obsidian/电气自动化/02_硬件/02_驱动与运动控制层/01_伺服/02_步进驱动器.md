# 01 步进驱动器总览

> [!info] 核心结论
> 步进驱动器用于把控制器发出的 **脉冲和方向信号** 转换成电机绕组电流，从而控制步进电机按固定角度一步一步转动。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> PLC或运动控制器负责发脉冲，步进驱动器负责放大电流，步进电机负责执行转动。  
> 步进系统通常是 **开环控制**，结构简单、成本低，但高速和重载时要防止丢步。

> [!warning] 🟧 通用选型公式
> **步进驱动器选型 = 电机相数 + 额定电流 + 电源电压 + 控制方式 + 细分设置 + 负载转矩 + 速度要求**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **设备类型** | 步进电机驱动器 |
| **控制对象** | 两相、三相、五相步进电机等 |
| **常见控制** | 脉冲+方向、CW/CCW、总线控制 |
| **核心参数** | 电压、电流、细分、频率响应 |
| **优点** | 成本低、定位简单、控制方便 |
| **风险点** | 丢步、共振、发热、高速力矩下降 |

## ③ 计算/选型示例

- 已知：  
  两相步进电机额定电流为 **3A**，需要PLC脉冲控制。
- 公式：  
  **驱动器额定输出电流 ≥ 电机额定电流**
- 计算：  
  步进驱动器输出电流应覆盖 **3A**，并支持PLC的脉冲控制方式。
- 结论：  
  可选择支持 **两相步进电机、输出电流≥3A、脉冲+方向控制** 的步进驱动器。

> [!tip] 记忆口诀
> 控制器发脉冲，驱动器放电流，电机一步一步走。

# 02 底层原理

> [!info] 核心结论
> 步进驱动器的本质，是按脉冲顺序给电机绕组通电，让转子按固定步距角转动。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 控制器每发一个脉冲，驱动器就让电机前进一步或一个细分步。  
> 方向信号决定正转或反转，细分设置决定每一步被分成多少小步。

> [!warning] 🟧 控制逻辑
> **脉冲输入 → 驱动器分配电流 → 电机磁场变化 → 转子转动 → 负载移动**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **脉冲信号** | 决定走多少步 |
| **方向信号** | 决定正转/反转 |
| **细分电路** | 让运动更平滑 |
| **功率模块** | 输出绕组电流 |
| **电流环** | 控制电机相电流 |
| **保护电路** | 过流、过压、过热保护 |

## ③ 计算/选型示例

- 已知：  
  步进电机步距角为 **1.8°**。
- 公式：  
  **每圈步数 = 360° ÷ 步距角**
- 计算：  
  每圈步数 = 360 ÷ 1.8 = **200步/圈**
- 结论：  
  不细分时，电机转一圈需要 **200个脉冲**。

> [!tip] 记忆口诀
> 一个脉冲走一步，步距角决定一圈多少步。

# 03 步进驱动器 vs 伺服驱动器

> [!info] 核心结论
> 步进驱动器适合低成本、简单定位；伺服驱动器适合高速度、高精度、高动态和闭环控制。

## ① 分类/公式/规则

> [!note] 🟦 快速区别
> 步进系统通常开环，按脉冲走步；  
> 伺服系统闭环控制，有编码器反馈，能实时修正位置偏差。

> [!warning] 🟧 选择公式
> **简单定位 / 成本敏感 → 步进**  
> **高速高精度 / 防丢步 / 动态响应高 → 伺服**

## ② 参考表

| 对比项 | 步进驱动器 | 伺服驱动器 |
|---|---|---|
| 控制方式 | 多为开环 | **闭环反馈** |
| 成本 | 较低 | 较高 |
| 低速保持 | 较好 | 较好 |
| 高速性能 | 力矩下降明显 | **高速性能好** |
| 丢步风险 | 有 | 较低 |
| 调试难度 | 简单 | 较复杂 |

## ③ 计算/选型示例

- 已知：  
  一个小型滑台只需要低速移动到固定位置，负载较轻。
- 公式：  
  **低速 + 轻载 + 简单定位 = 步进可评估**
- 计算：  
  该场景对动态响应和闭环精度要求不高。
- 结论：  
  可优先选择 **步进驱动器 + 步进电机**，但需校核转矩余量，避免丢步。

> [!tip] 记忆口诀
> 简单低速用步进，高速精准用伺服。

# 04 脉冲控制方式

> [!info] 核心结论
> 步进驱动器常用脉冲信号控制，PLC或运动控制器发多少脉冲，电机就走多少步。

## ① 分类/公式/规则

> [!note] 🟦 常见模式
> **PUL + DIR**：脉冲决定步数，方向信号决定方向。  
> **CW/CCW**：正转脉冲和反转脉冲分开输入。  
> **总线控制**：通过通讯直接发送位置、速度和参数。

> [!warning] 🟧 脉冲控制公式
> **运动距离 = 脉冲数 × 脉冲当量**

## ② 参考表

| 控制方式 | 说明 |
|---|---|
| **PUL + DIR** | 最常见，接线简单 |
| **CW/CCW** | 正反转脉冲分开 |
| **A/B相脉冲** | 类似编码器相位输入 |
| **IO控制** | 简单启停或段位控制 |
| **RS485/总线** | 参数化和位置命令 |
| **使能信号ENA** | 控制驱动器使能/释放 |

## ③ 计算/选型示例

- 已知：  
  脉冲当量为 **0.01mm/脉冲**，PLC发送 **5000个脉冲**。
- 公式：  
  **运动距离 = 脉冲数 × 脉冲当量**
- 计算：  
  运动距离 = 5000 × 0.01 = **50mm**
- 结论：  
  负载理论移动 **50mm**，实际还需考虑机械间隙和是否丢步。

> [!tip] 记忆口诀
> 脉冲管距离，频率管速度，方向管正反。

# 05 细分设置

> [!info] 核心结论
> 细分可以让步进电机运动更平滑、分辨率更高，但细分不等于实际精度无限提高。

## ① 分类/公式/规则

> [!note] 🟦 细分理解
> 细分是驱动器把电机原本的一整步拆成多个小步。  
> 例如200步/圈电机，设置16细分后，每圈需要3200个脉冲。

> [!warning] 🟧 细分公式
> **每圈脉冲数 = 基本步数 × 细分数**

## ② 参考表

| 细分数 | 200步/圈电机对应脉冲数 |
|---|---|
| **1细分** | 200脉冲/圈 |
| **2细分** | 400脉冲/圈 |
| **4细分** | 800脉冲/圈 |
| **8细分** | 1600脉冲/圈 |
| **16细分** | 3200脉冲/圈 |
| **32细分** | 6400脉冲/圈 |

## ③ 计算/选型示例

- 已知：  
  步进电机为200步/圈，驱动器设置 **8细分**。
- 公式：  
  **每圈脉冲数 = 200 × 8**
- 计算：  
  每圈脉冲数 = **1600脉冲/圈**
- 结论：  
  电机转一圈需要PLC发送 **1600个脉冲**。

> [!tip] 记忆口诀
> 细分越大，走得越细；精度不只看细分，还要看机械。

# 06 电流设置

> [!info] 核心结论
> 步进驱动器电流设置要匹配电机额定电流，电流过小力矩不足，电流过大发热严重。

## ① 分类/公式/规则

> [!note] 🟦 设置原则
> 驱动器电流通常通过拨码开关、软件或参数设置。  
> 设置时要区分 **峰值电流、有效值电流、相电流**，以厂家说明为准。

> [!warning] 🟧 电流设置公式
> **推荐驱动电流 ≈ 电机额定相电流**

## ② 参考表

| 电流状态 | 影响 |
|---|---|
| **电流偏小** | 力矩不足，易丢步 |
| **电流合适** | 力矩和温升平衡 |
| **电流偏大** | 电机和驱动器发热 |
| **静止半流** | 降低待机发热 |
| **峰值电流** | 瞬时最大值 |
| **有效值电流** | 发热更相关 |

## ③ 计算/选型示例

- 已知：  
  步进电机额定相电流为 **2.8A**。
- 公式：  
  **驱动器设置电流 ≈ 2.8A**
- 计算：  
  若驱动器档位有2.6A和3.0A，可按厂家建议选择接近档位。
- 结论：  
  电流设置需结合电机温升、负载力矩和厂家手册校核。

> [!tip] 记忆口诀
> 电流小了没力，电流大了发热；刚好够用最稳。

# 07 电源电压选择

> [!info] 核心结论
> 步进驱动器电源电压影响高速性能，电压越高，高速力矩通常越好，但必须在驱动器允许范围内。

## ① 分类/公式/规则

> [!note] 🟦 电压理解
> 步进电机高速运行时，绕组电感会限制电流上升。  
> 较高供电电压有利于电流快速建立，从而改善高速转矩。

> [!warning] 🟧 电压选择公式
> **供电电压 ≤ 驱动器允许最高电压**

## ② 参考表

| 电压情况 | 影响 |
|---|---|
| **电压偏低** | 高速力矩下降明显 |
| **电压合适** | 运行稳定 |
| **电压过高** | 可能损坏驱动器 |
| **电源功率不足** | 加速掉压、报警 |
| **开关电源** | 常用于小功率步进 |
| **制动回灌** | 大惯量减速时需注意 |

## ③ 计算/选型示例

- 已知：  
  步进驱动器允许电压范围为 **DC24～50V**。
- 公式：  
  **实际电源电压必须落在允许范围内**
- 计算：  
  DC48V在允许范围内，DC60V超出范围。
- 结论：  
  可选DC48V电源，不能使用DC60V，除非驱动器明确支持。

> [!tip] 记忆口诀
> 低压跑不快，高压别超限。

# 08 速度与脉冲频率

> [!info] 核心结论
> 步进电机转速由输入脉冲频率决定，频率越高，转速越高，但过高可能失步。

## ① 分类/公式/规则

> [!warning] 🟧 转速公式
> **rpm = 脉冲频率 ÷ 每圈脉冲数 × 60**

> [!note] 🟦 参数说明
> 脉冲频率单位为Hz，也就是每秒脉冲数。  
> 每圈脉冲数由电机步距角和细分数决定。

## ② 参考表

| 参数 | 作用 |
|---|---|
| **脉冲频率** | 决定转速 |
| **每圈脉冲数** | 决定一圈需要多少脉冲 |
| **细分数** | 影响每圈脉冲数 |
| **加速时间** | 影响是否丢步 |
| **负载惯量** | 影响最高可用速度 |
| **转矩曲线** | 高速转矩会下降 |

## ③ 计算/选型示例

- 已知：  
  每圈脉冲数为 **1600脉冲/圈**，PLC输出频率为 **8000Hz**。
- 公式：  
  **rpm = 8000 ÷ 1600 × 60**
- 计算：  
  rpm = **300rpm**
- 结论：  
  步进电机理论转速为 **300rpm**。

> [!tip] 记忆口诀
> 频率越高转得越快，负载越重越容易丢步。

# 09 丢步与堵转

> [!info] 核心结论
> 步进系统开环运行时，如果负载超过电机能力，电机可能丢步或堵转，但控制器未必知道。

## ① 分类/公式/规则

> [!note] 🟦 丢步原因
> 常见原因包括加速度过大、负载过重、转矩不足、速度过高、机械卡滞、电流设置偏小、电源电压偏低。

> [!warning] 🟧 防丢步公式
> **可靠运行 = 转矩余量足够 + 加减速合理 + 机械顺畅 + 电流电压匹配**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **位置越来越偏** | 丢步、同步带打滑、联轴器松动 |
| **启动抖动** | 加速度过大、共振 |
| **高速失步** | 高速转矩不足 |
| **堵转啸叫** | 负载卡死或转矩不足 |
| **发热大** | 电流过大或散热差 |
| **偶发误差** | 干扰、脉冲丢失、机械松动 |

## ③ 计算/选型示例

- 已知：  
  步进电机低速正常，高速时位置偏差越来越大。
- 公式：  
  **高速丢步 = 高速转矩不足 + 加速过快 + 负载惯量大**
- 计算：  
  步进电机高速转矩下降，若负载需要的转矩超过可用转矩，就会丢步。
- 结论：  
  可降低最高速度、延长加速时间、提高合规供电电压，或改用更大电机/闭环步进/伺服。

> [!tip] 记忆口诀
> 步进最怕丢步，速度越高越要留转矩余量。

# 10 共振与振动

> [!info] 核心结论
> 步进电机容易在某些速度区间出现共振，表现为抖动、噪声、失步或运行不平稳。

## ① 分类/公式/规则

> [!note] 🟦 共振理解
> 步进电机是一步一步运动的，天然存在脉动转矩。  
> 当电机步进频率与机械系统固有频率接近时，就可能出现共振。

> [!warning] 🟧 抑制共振公式
> **降低共振 = 合理细分 + 平滑加减速 + 避开共振速度 + 机械刚性优化**

## ② 参考表

| 方法 | 作用 |
|---|---|
| **提高细分** | 运动更平滑 |
| **加减速曲线** | 减少冲击 |
| **避开共振区** | 跳过不稳定速度段 |
| **增加阻尼** | 减少振动 |
| **优化机构刚性** | 降低机械抖动 |
| **闭环步进** | 改善稳定性和反馈能力 |

## ③ 计算/选型示例

- 已知：  
  电机在某一速度附近噪声大、抖动明显。
- 公式：  
  **抖动区间 = 可能接近共振频率**
- 计算：  
  可尝试调整细分、加速度或避开该速度区间。
- 结论：  
  若仍不稳定，应检查机械刚性、联轴器、导轨和负载惯量。

> [!tip] 记忆口诀
> 步进抖不一定坏，可能踩中了共振点。

# 11 开环、闭环步进

> [!info] 核心结论
> 开环步进成本低但不反馈位置，闭环步进带编码器反馈，可检测丢步并改善运行性能。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 开环步进：控制器认为发了多少脉冲就走了多少。  
> 闭环步进：电机带编码器，驱动器可检测实际位置和跟随误差。

> [!warning] 🟧 选择公式
> **成本敏感 / 负载稳定 → 开环步进**  
> **怕丢步 / 负载波动 / 响应更高 → 闭环步进**

## ② 参考表

| 对比项 | 开环步进 | 闭环步进 |
|---|---|---|
| 反馈 | 无 | **有编码器反馈** |
| 成本 | 低 | 较高 |
| 丢步检测 | 不易发现 | **可报警** |
| 发热 | 通常较明显 | 可优化电流 |
| 调试 | 简单 | 略复杂 |
| 应用 | 简单定位 | 防丢步定位 |

## ③ 计算/选型示例

- 已知：  
  设备偶尔卡料，位置一旦丢失会导致产品报废。
- 公式：  
  **位置可靠性需求高 → 反馈优先**
- 计算：  
  开环步进丢步后控制器可能不知道，闭环步进可检测误差并报警。
- 结论：  
  应优先评估 **闭环步进驱动器** 或升级为伺服系统。

> [!tip] 记忆口诀
> 开环便宜但怕丢，闭环贵点能反馈。

# 12 PLC接线与抗干扰

> [!info] 核心结论
> 步进驱动器接PLC时，要确认脉冲电平、输入类型、公共端、屏蔽接地和脉冲频率能力。

## ① 分类/公式/规则

> [!note] 🟦 接线重点
> PLC高速脉冲输出要与驱动器输入方式匹配。  
> 常见问题包括PNP/NPN接法错误、5V/24V电平不匹配、公共端接错、脉冲线受干扰。

> [!warning] 🟧 接线匹配公式
> **脉冲有效 = 电平匹配 + 公共端正确 + 频率足够 + 屏蔽良好**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **PUL** | 脉冲输入 |
| **DIR** | 方向输入 |
| **ENA** | 使能输入 |
| **公共端** | PNP/NPN接法关键 |
| **屏蔽线** | 降低脉冲干扰 |
| **高速输出** | PLC必须支持对应频率 |

## ③ 计算/选型示例

- 已知：  
  需要最高脉冲频率 **100kHz**，PLC高速输出最高只支持 **50kHz**。
- 公式：  
  **PLC输出频率 ≥ 需求频率**
- 计算：  
  50kHz ＜ 100kHz，PLC输出能力不足。
- 结论：  
  需要降低速度/细分，或更换支持更高频率的PLC/运动控制器。

> [!tip] 记忆口诀
> 接线先看公共端，速度先看脉冲频率。

# 13 常见故障排查

> [!info] 核心结论
> 步进驱动器故障排查要按“电源 → 使能 → 脉冲 → 方向 → 电机接线 → 机械负载”的顺序检查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看驱动器报警灯，再查电源电压；  
> 先空载运行，再带载测试；  
> 先低速验证，再逐步提高速度。

> [!warning] 🟧 排错公式
> **步进异常 = 电源异常 + 脉冲无效 + 方向错误 + 电机接线错 + 电流设置错 + 负载过重**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **电机不动** | 无电源、未使能、无脉冲 |
| **只锁住不转** | 脉冲未到、控制模式错误 |
| **方向反了** | DIR逻辑反或电机相序问题 |
| **运行抖动** | 接线错误、共振、加速太快 |
| **定位不准** | 丢步、细分/脉冲当量设置错 |
| **驱动报警** | 过流、过压、过热、电机短路 |

## ③ 计算/选型示例

- 已知：  
  电机上电有保持力，但PLC发脉冲后不转。
- 公式：  
  **有保持力 = 驱动器和电机基本通电**  
  **不转 = 脉冲信号或控制模式异常**
- 计算：  
  可能是PUL接线错误、公共端错误、脉冲电平不匹配或PLC未输出高速脉冲。
- 结论：  
  应检查PLC高速输出、PUL/DIR接线、公共端和驱动器控制模式。

> [!tip] 记忆口诀
> 有力不转查脉冲，没力不动查电源。

# 14 步进驱动器快速选型表

> [!info] 核心结论
> 步进驱动器选型先匹配电机，再校核电压、电流、细分、脉冲频率、负载转矩和是否需要闭环。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看电机相数  
> ② 看电机额定电流  
> ③ 看驱动器供电电压  
> ④ 看控制方式  
> ⑤ 看细分范围  
> ⑥ 看最高脉冲频率  
> ⑦ 看负载转矩和惯量  
> ⑧ 看速度要求  
> ⑨ 看是否需要闭环  
> ⑩ 看安装环境和散热

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电机相数 + 输出电流 + 供电电压 + 细分能力 + 控制方式 + 闭环需求**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单低速定位 | 开环步进驱动器 |
| 防止丢步 | 闭环步进驱动器 |
| 高速运行 | 较高电压、转矩曲线校核 |
| 高分辨率运动 | 细分和脉冲频率 |
| PLC控制 | PUL/DIR电平和频率 |
| 负载较重 | 电流余量和转矩余量 |
| 振动明显 | 细分、加减速、共振抑制 |

## ③ 计算/选型示例

- 已知：  
  两相1.8°步进电机，16细分，丝杆导程5mm，希望脉冲当量是多少？
- 公式：  
  **每圈脉冲数 = 200 × 16**  
  **脉冲当量 = 丝杆导程 ÷ 每圈脉冲数**
- 计算：  
  每圈脉冲数 = **3200脉冲/圈**  
  脉冲当量 = 5 ÷ 3200 = **0.0015625mm/脉冲**
- 结论：  
  PLC每发1个脉冲，理论移动约 **0.00156mm**，实际精度需按机械结构校核。

> [!tip] 选型口诀
> 先配电机相数，再配电流电压；速度看频率，精度看细分；怕丢步，就选闭环。
