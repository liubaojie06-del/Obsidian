# 01 精密机电总览

> [!info] 核心结论
> 精密机电是把 **电机、驱动、传动机构、导向机构、反馈元件和机械结构** 组合起来，实现精准运动的系统。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 伺服电机负责“转”，机械机构负责“传”，导轨负责“导”，编码器负责“反馈”。  
> 精密机电的核心不是单个零件，而是 **运动精度、刚性、寿命、负载、速度、重复定位精度** 的整体匹配。

> [!warning] 🟧 通用选型公式
> **精密机电选型 = 运动方式 + 行程 + 负载 + 速度 + 精度 + 传动方式 + 安装空间 + 环境条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **直线模组** | 实现直线运动 |
| **丝杆模组** | 高精度、中低速、大推力 |
| **同步带模组** | 长行程、高速度、精度一般 |
| **直线电机** | 高速度、高响应、高精度 |
| **减速机** | 提高转矩、降低转速 |
| **联轴器** | 连接电机与机械轴 |
| **导轨滑块** | 承载并保证直线导向 |

## ③ 计算/选型示例

- 已知：  
  设备需要带动工件做500mm直线定位，要求重复定位精度较高。
- 公式：  
  **机构选择 = 行程 + 负载 + 精度 + 速度**
- 计算：  
  500mm行程不算特别长，重复定位精度要求较高，可优先考虑丝杆模组。
- 结论：  
  可选择 **伺服电机 + 滚珠丝杆模组 + 导轨滑块** 方案，具体规格需按厂家手册校核。

> [!tip] 记忆口诀
> 电机给动力，传动定精度；导轨保方向，反馈保位置。

# 02 底层原理

> [!info] 核心结论
> 精密机电的本质，是把电机的旋转运动转换成可控的直线或旋转机械运动。

## ① 分类/公式/规则

> [!note] 🟦 运动链路
> 控制器发出指令，驱动器控制电机，电机输出转矩，机械传动把转矩转换成位移或旋转，反馈元件再把实际位置传回控制系统。

> [!warning] 🟧 运动控制逻辑
> **控制指令 → 驱动器 → 电机 → 传动机构 → 负载运动 → 反馈修正**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **控制器** | 发送位置、速度、转矩指令 |
| **驱动器** | 控制电机运行 |
| **电机** | 输出转矩和转速 |
| **传动机构** | 丝杆、同步带、齿轮、减速机 |
| **导向机构** | 导轨、滑块、轴承 |
| **反馈元件** | 编码器、光栅尺、磁栅尺 |

## ③ 计算/选型示例

- 已知：  
  电机旋转一圈，丝杆导程为10mm。
- 公式：  
  **直线位移 = 电机圈数 × 丝杆导程**
- 计算：  
  电机转5圈，直线位移 = 5 × 10 = **50mm**
- 结论：  
  丝杆把电机旋转运动转换成直线运动，导程决定一圈走多远。

> [!tip] 记忆口诀
> 电机转几圈，丝杆走几段；导程一确定，位移就能算。

# 03 直线模组

> [!info] 核心结论
> 直线模组是精密机电中最常见的执行机构，用于实现自动化设备的直线搬运、定位和推拉动作。

## ① 分类/公式/规则

> [!note] 🟦 常见类型
> 直线模组常见传动方式有 **滚珠丝杆、同步带、齿轮齿条、直线电机**。  
> 不同传动方式决定速度、精度、推力、行程和成本。

> [!warning] 🟧 模组选型公式
> **直线模组选型 = 行程 + 负载 + 速度 + 精度 + 安装方向 + 驱动方式**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **丝杆模组** | 高精度、中短行程 |
| **同步带模组** | 长行程、高速度 |
| **齿轮齿条模组** | 长行程、大负载 |
| **直线电机模组** | 高速、高响应、高精度 |
| **单轴模组** | 简单直线运动 |
| **多轴平台** | XY、XYZ定位系统 |

## ③ 计算/选型示例

- 已知：  
  需要搬运5kg负载，行程1200mm，定位精度要求一般，但速度要求较高。
- 公式：  
  **长行程 + 高速度 → 同步带/齿轮齿条优先**
- 计算：  
  1200mm行程较长，丝杆高速长行程可能受临界转速限制。
- 结论：  
  可优先评估 **同步带直线模组**，精度和刚性需按实际工艺校核。

> [!tip] 记忆口诀
> 短行程高精度看丝杆，长行程高速度看同步带。

# 04 滚珠丝杆

> [!info] 核心结论
> 滚珠丝杆用于把旋转运动转换成高精度直线运动，适合定位、压装、升降和精密调节。

## ① 分类/公式/规则

> [!note] 🟦 丝杆重点
> 丝杆导程决定速度和推力关系。  
> 导程越大，同样转速下速度越快；导程越小，同样电机转矩下推力越大。

> [!warning] 🟧 丝杆速度公式
> **直线速度 = 电机转速 × 丝杆导程 ÷ 60**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **导程** | 一圈前进多少mm |
| **直径** | 影响刚性和承载 |
| **精度等级** | 影响定位精度 |
| **预压** | 减小反向间隙 |
| **临界转速** | 限制长丝杆高速运行 |
| **支撑方式** | 影响刚性和稳定性 |

## ③ 计算/选型示例

- 已知：  
  电机转速3000rpm，丝杆导程10mm。
- 公式：  
  **直线速度 = 3000 × 10 ÷ 60**
- 计算：  
  直线速度 = **500mm/s**
- 结论：  
  该丝杆系统理论直线速度约为 **500mm/s**，实际需校核临界转速和负载。

> [!tip] 记忆口诀
> 导程大速度快，导程小推力大。

# 05 同步带传动

> [!info] 核心结论
> 同步带传动适合长行程和高速搬运，但刚性和定位精度通常不如丝杆。

## ① 分类/公式/规则

> [!note] 🟦 同步带特点
> 同步带通过带齿与同步轮啮合传动，结构轻、速度快、行程长，常用于搬运、上下料、喷涂、码垛等场景。

> [!warning] 🟧 同步带位移公式
> **每圈位移 = 同步轮齿数 × 皮带节距**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **皮带节距** | 相邻齿之间距离 |
| **同步轮齿数** | 决定一圈位移 |
| **皮带宽度** | 影响承载能力 |
| **张紧力** | 影响精度和寿命 |
| **行程长度** | 同步带适合长行程 |
| **刚性** | 低于丝杆和直线电机 |

## ③ 计算/选型示例

- 已知：  
  同步轮齿数为20齿，皮带节距为5mm。
- 公式：  
  **每圈位移 = 20 × 5**
- 计算：  
  每圈位移 = **100mm**
- 结论：  
  电机每转一圈，滑台理论移动 **100mm**。

> [!tip] 记忆口诀
> 同步带跑得快，长行程很划算；精度要很高，先考虑丝杆。

# 06 减速机

> [!info] 核心结论
> 减速机用于降低转速、提高输出转矩，并改善惯量匹配，是伺服系统常见配套部件。

## ① 分类/公式/规则

> [!note] 🟦 减速机作用
> 减速机不是只为了“变慢”，更重要的是 **放大转矩、降低负载惯量折算、提高控制稳定性**。

> [!warning] 🟧 减速关系公式
> **输出转速 = 电机转速 ÷ 减速比**  
> **输出转矩 ≈ 电机转矩 × 减速比 × 效率**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **行星减速机** | 精度高、结构紧凑 |
| **谐波减速机** | 高减速比、机器人常见 |
| **RV减速机** | 高刚性、大负载 |
| **蜗轮蜗杆** | 可大减速，效率需校核 |
| **齿轮箱** | 通用传动 |
| **重点参数** | 减速比、背隙、额定转矩、效率 |

## ③ 计算/选型示例

- 已知：  
  电机转速3000rpm，减速比为10。
- 公式：  
  **输出转速 = 3000 ÷ 10**
- 计算：  
  输出转速 = **300rpm**
- 结论：  
  减速后速度降低，输出转矩约放大10倍再乘以效率。

> [!tip] 记忆口诀
> 减速降速度，顺手放大力；精密传动，还要看背隙。

# 07 联轴器

> [!info] 核心结论
> 联轴器用于连接电机轴和机械轴，并补偿少量偏心、偏角和轴向误差。

## ① 分类/公式/规则

> [!note] 🟦 联轴器作用
> 联轴器不是随便接两根轴。  
> 它需要在传递转矩的同时，保护轴承、吸收微小安装误差，避免硬连接导致振动和损坏。

> [!warning] 🟧 联轴器选型公式
> **联轴器选型 = 轴径 + 转矩 + 转速 + 偏差补偿 + 刚性要求 + 安装空间**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **梅花联轴器** | 通用、减振较好 |
| **膜片联轴器** | 高刚性、响应快 |
| **波纹管联轴器** | 高精度、小惯量 |
| **十字滑块联轴器** | 补偿偏心能力较强 |
| **刚性联轴器** | 要求同轴度高 |
| **安全联轴器** | 过载保护 |

## ③ 计算/选型示例

- 已知：  
  伺服电机轴径14mm，丝杆端轴径12mm，需要高响应定位。
- 公式：  
  **联轴器需求 = 两端孔径匹配 + 高刚性 + 低背隙**
- 计算：  
  高响应定位不宜使用间隙大的联轴器。
- 结论：  
  可选择 **膜片联轴器或波纹管联轴器**，转矩和转速需按厂家手册校核。

> [!tip] 记忆口诀
> 联轴器选不好，振动、断轴、定位漂。

# 08 直线导轨

> [!info] 核心结论
> 直线导轨负责承载和导向，决定直线运动的刚性、顺滑度和寿命。

## ① 分类/公式/规则

> [!note] 🟦 导轨作用
> 传动机构负责“推着走”，导轨负责“直着走”。  
> 导轨滑块选型要看负载、力矩、精度等级、润滑、防尘和安装面刚性。

> [!warning] 🟧 导轨选型公式
> **导轨选型 = 负载 + 力矩 + 行程 + 精度等级 + 安装方式 + 环境防护**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **导轨宽度** | 影响承载和刚性 |
| **滑块数量** | 影响稳定性 |
| **预压等级** | 影响刚性和摩擦 |
| **精度等级** | 影响运动精度 |
| **润滑** | 影响寿命 |
| **防尘件** | 防止粉尘铁屑进入 |

## ③ 计算/选型示例

- 已知：  
  滑台承载较重，并且负载中心偏离导轨中心。
- 公式：  
  **导轨受力 = 竖向载荷 + 偏心力矩**
- 计算：  
  负载偏心会让滑块承受额外力矩，不能只按重量选导轨。
- 结论：  
  应校核滑块额定载荷和额定力矩，必要时增加滑块数量或加宽导轨。

> [!tip] 记忆口诀
> 传动管走，导轨管稳；负载偏心，力矩要算。

# 09 直线电机

> [!info] 核心结论
> 直线电机取消丝杆、皮带等中间传动，直接产生直线推力，适合高速、高响应、高精度场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 直线电机可以理解为“展开的旋转电机”。  
> 它没有传统机械传动间隙，但对光栅尺反馈、安装精度、散热和防护要求更高。

> [!warning] 🟧 直线电机选型公式
> **直线电机选型 = 连续推力 + 峰值推力 + 最高速度 + 加速度 + 行程 + 反馈精度**

## ② 参考表

| 项目 | 特点 |
|---|---|
| **优点** | 高速、高加速度、无反向间隙 |
| **缺点** | 成本高、散热和防护要求高 |
| **反馈** | 常配光栅尺或磁栅尺 |
| **适合场景** | 半导体、激光加工、精密平台 |
| **注意** | 磁吸力、散热、电缆拖链 |
| **控制** | 需要高性能伺服驱动 |

## ③ 计算/选型示例

- 已知：  
  负载质量为20kg，要求加速度为5m/s²。
- 公式：  
  **推力 F = m × a**
- 计算：  
  F = 20 × 5 = **100N**
- 结论：  
  仅加速所需推力为 **100N**，实际还需叠加摩擦、负载工艺力和安全余量。

> [!tip] 记忆口诀
> 要极快、要极准，直线电机很强；但安装散热反馈都要认真做。

# 10 精度与重复定位

> [!info] 核心结论
> 精密机电选型要区分分辨率、定位精度和重复定位精度，三者不是同一个概念。

## ① 分类/公式/规则

> [!note] 🟦 三个概念
> 分辨率：系统能分得多细。  
> 定位精度：实际到达目标位置有多准。  
> 重复定位精度：多次到同一位置的一致性。

> [!warning] 🟧 精度判断公式
> **实际定位误差 = 控制误差 + 传动误差 + 反向间隙 + 刚性变形 + 温度影响**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **分辨率** | 理论最小控制单位 |
| **定位精度** | 到目标位置的误差 |
| **重复定位精度** | 多次到位一致性 |
| **反向间隙** | 换向时产生误差 |
| **热膨胀** | 温度变化导致尺寸变化 |
| **机械刚性** | 受力后变形影响精度 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，每圈指令脉冲10000个。
- 公式：  
  **脉冲当量 = 导程 ÷ 每圈脉冲数**
- 计算：  
  脉冲当量 = 10 ÷ 10000 = **0.001mm/脉冲**
- 结论：  
  0.001mm只是理论控制分辨率，不等于实际定位精度一定达到0.001mm。

> [!tip] 记忆口诀
> 分辨率是能分多细，精度是能到多准。

# 11 常见故障排查

> [!info] 核心结论
> 精密机电异常时，要同时查电气控制、机械传动、安装精度和润滑状态。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查报警和参数，再查机械是否卡滞；  
> 先低速空载运行，再带载验证；  
> 先查传动连接，再查导轨丝杆润滑。

> [!warning] 🟧 排错公式
> **运动异常 = 参数错误 + 电机转矩不足 + 传动松动 + 导轨卡滞 + 间隙过大 + 润滑不良**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **定位不准** | 脉冲当量错、间隙、打滑 |
| **重复性差** | 机械松动、导轨磨损 |
| **运行异响** | 联轴器偏心、轴承损坏 |
| **电机报警** | 负载过大、加速度过高 |
| **低速爬行** | 摩擦大、参数不合适 |
| **发热明显** | 润滑差、预压过大、负载重 |

## ③ 计算/选型示例

- 已知：  
  滑台正向定位准确，反向定位误差明显。
- 公式：  
  **换向误差 = 反向间隙 + 机械弹性变形**
- 计算：  
  丝杆间隙、联轴器间隙或同步带松弛都可能导致反向误差。
- 结论：  
  应检查反向间隙、联轴器、丝杆预压和同步带张紧。

> [!tip] 记忆口诀
> 单向准、反向偏，优先查间隙和松动。

# 12 精密机电快速选型表

> [!info] 核心结论
> 精密机电选型必须从运动需求出发，先定机构，再定电机和控制。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看运动形式：直线/旋转  
> ② 看行程范围  
> ③ 看负载重量  
> ④ 看速度和加速度  
> ⑤ 看精度和重复定位  
> ⑥ 选传动方式  
> ⑦ 选导轨和支撑  
> ⑧ 选电机和减速机  
> ⑨ 校核惯量和转矩  
> ⑩ 校核安装空间和环境

> [!warning] 🟧 最终选型公式
> **推荐方案 = 执行机构 + 传动方式 + 导向机构 + 电机驱动 + 反馈元件 + 安装防护**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 高精度短行程 | 丝杆模组 |
| 长行程高速 | 同步带模组 |
| 高速高响应 | 直线电机 |
| 大转矩低速 | 减速机 |
| 精密旋转 | 伺服 + 减速机/直驱 |
| 垂直升降 | 抱闸、重力、导向 |
| 高重复精度 | 刚性、间隙、反馈 |

## ③ 计算/选型示例

- 已知：  
  设备要求行程400mm，负载10kg，重复定位精度要求高，速度300mm/s。
- 公式：  
  **方案需求 = 中短行程 + 高重复精度 + 中等速度**
- 计算：  
  同步带速度快但精度一般；丝杆模组更适合高重复定位。
- 结论：  
  可选择 **伺服电机 + 滚珠丝杆模组 + 直线导轨**，并继续校核转矩、惯量和安装刚性。

> [!tip] 选型口诀
> 先定运动，再定机构；精度看丝杆，速度看同步带；响应看直线电机，稳定看导轨和刚性。
