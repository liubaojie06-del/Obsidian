# 01 交流伺服系统总览

> [!info] 核心结论
> 交流伺服系统用于实现 **高精度位置、速度、转矩控制**，常见于机械手、丝杆模组、包装机、贴标机、数控设备和自动化产线。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 交流伺服系统不是单独一个电机，而是一套闭环控制系统。  
> 它通常由 **伺服驱动器、伺服电机、编码器、控制器、动力线、编码器线** 组成。

> [!warning] 🟧 通用选型公式
> **伺服系统选型 = 负载惯量 + 转矩需求 + 速度需求 + 精度要求 + 控制方式 + 通讯接口 + 安装环境**

## ② 参考表

| 组成 | 作用 |
|---|---|
| **伺服驱动器** | 控制电机电流、速度、位置 |
| **伺服电机** | 输出转矩和转速 |
| **编码器** | 反馈位置和速度 |
| **PLC/运动控制器** | 发送位置、速度、转矩命令 |
| **动力线** | 驱动器给电机供电 |
| **编码器线** | 电机位置反馈到驱动器 |

## ③ 计算/选型示例

- 已知：  
  设备需要控制滑台精准移动到指定位置。
- 公式：  
  **精准定位 = 控制器指令 + 驱动器控制 + 电机执行 + 编码器反馈**
- 计算：  
  伺服电机运动后，编码器实时反馈位置，驱动器根据偏差修正输出。
- 结论：  
  应选用 **交流伺服系统**，具体功率、转矩、惯量和通讯方式需按厂家手册校核。

> [!tip] 记忆口诀
> 变频管调速，伺服管精准；要位置准，就用伺服闭环。

# 02 底层原理

> [!info] 核心结论
> 交流伺服系统的本质，是通过编码器反馈形成闭环，让实际位置不断追随目标位置。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 控制器给出目标位置或速度，驱动器计算控制量，电机执行动作，编码器反馈实际位置，驱动器根据偏差持续修正。

> [!warning] 🟧 闭环控制逻辑
> **目标指令 → 驱动器计算 → 电机运动 → 编码器反馈 → 偏差修正**

## ② 参考表

| 环节 | 说明 |
|---|---|
| **目标值** | PLC或运动控制器发出的指令 |
| **实际值** | 编码器反馈的位置/速度 |
| **偏差** | 目标值 - 实际值 |
| **控制算法** | 根据偏差调节输出 |
| **电流环** | 控制转矩 |
| **速度环** | 控制速度 |
| **位置环** | 控制位置 |

## ③ 计算/选型示例

- 已知：  
  目标位置为100.00mm，实际位置为99.80mm。
- 公式：  
  **位置偏差 = 目标位置 - 实际位置**
- 计算：  
  位置偏差 = 100.00 - 99.80 = **0.20mm**
- 结论：  
  伺服系统会继续驱动电机运动，直到位置偏差进入允许范围。

> [!tip] 记忆口诀
> 有目标，有反馈，有修正，才叫伺服。

# 03 伺服驱动器

> [!info] 核心结论
> 伺服驱动器是伺服系统的大脑，负责接收指令、控制电机、读取编码器并执行保护。

## ① 分类/公式/规则

> [!note] 🟦 驱动器功能
> 伺服驱动器可完成位置控制、速度控制、转矩控制、电子齿轮、回零、报警保护、通讯控制等功能。

> [!warning] 🟧 驱动器选型公式
> **伺服驱动器选型 = 电机功率 + 电压等级 + 控制模式 + 通讯协议 + I/O需求 + 安全功能**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **电压等级** | 220V、380V等需按型号校核 |
| **额定功率** | 与伺服电机匹配 |
| **额定电流** | 驱动电机的核心参数 |
| **控制模式** | 位置、速度、转矩 |
| **通讯方式** | 脉冲、模拟量、EtherCAT、PROFINET等 |
| **安全功能** | STO等需按型号校核 |

## ③ 计算/选型示例

- 已知：  
  伺服电机功率为750W，电源为AC220V。
- 公式：  
  **驱动器规格 = 电机功率匹配 + 电压等级匹配**
- 计算：  
  应选择支持750W电机、AC220V输入的伺服驱动器。
- 结论：  
  驱动器必须与电机型号匹配，不建议不同系列随意混用。

> [!tip] 记忆口诀
> 电机和驱动要成套，功率电压先对号。

# 04 伺服电机

> [!info] 核心结论
> 伺服电机负责输出机械运动，选型重点是额定转矩、峰值转矩、额定转速和惯量等级。

## ① 分类/公式/规则

> [!note] 🟦 电机重点
> 伺服电机不是只看功率。  
> 选型时更要看 **转矩够不够、速度够不够、惯量配不配、安装尺寸合不合适**。

> [!warning] 🟧 电机选型公式
> **伺服电机选型 = 连续转矩 + 峰值转矩 + 最高转速 + 惯量匹配 + 机械安装**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定转矩** | 长时间可输出转矩 |
| **峰值转矩** | 短时间加速或冲击转矩 |
| **额定转速** | 长时间运行速度 |
| **最高转速** | 短时间允许速度 |
| **转子惯量** | 影响响应和匹配 |
| **抱闸** | 垂直轴或保持位置使用 |

## ③ 计算/选型示例

- 已知：  
  负载连续运行需要2N·m转矩，加速时需要5N·m峰值转矩。
- 公式：  
  **电机额定转矩 ≥ 连续转矩**  
  **电机峰值转矩 ≥ 加速峰值转矩**
- 计算：  
  电机额定转矩应≥2N·m，峰值转矩应≥5N·m。
- 结论：  
  若峰值转矩不足，设备加速时可能报警或跟随误差过大。

> [!tip] 记忆口诀
> 长期看额定，瞬间看峰值；速度看转速，响应看惯量。

# 05 编码器反馈

> [!info] 核心结论
> 编码器是伺服闭环控制的眼睛，用于反馈电机实际位置和速度。

## ① 分类/公式/规则

> [!note] 🟦 编码器作用
> 驱动器通过编码器知道电机转了多少、转到哪里、转得多快。  
> 没有反馈，伺服就无法实现高精度闭环控制。

> [!warning] 🟧 反馈控制公式
> **控制偏差 = 指令位置 - 编码器反馈位置**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **增量编码器** | 断电后通常需回零 |
| **绝对值编码器** | 上电可读取当前位置 |
| **单圈绝对值** | 一圈内位置 |
| **多圈绝对值** | 记录多圈位置 |
| **高分辨率编码器** | 定位更细 |
| **电池多圈** | 需关注电池报警和更换 |

## ③ 计算/选型示例

- 已知：  
  伺服电机编码器分辨率为 **131072 counts/rev**。
- 公式：  
  **每计数角度 = 360° ÷ 编码器计数**
- 计算：  
  每计数角度 = 360 ÷ 131072 ≈ **0.00275°**
- 结论：  
  编码器分辨率越高，位置反馈越细，但最终精度还受机械结构影响。

> [!tip] 记忆口诀
> 编码器看位置，驱动器算偏差；反馈越稳定，伺服越听话。

# 06 三种控制模式

> [!info] 核心结论
> 伺服常见控制模式有 **位置控制、速度控制、转矩控制**，不同模式对应不同应用。

## ① 分类/公式/规则

> [!note] 🟦 模式区别
> 位置控制：我要你到哪里。  
> 速度控制：我要你跑多快。  
> 转矩控制：我要你出多大力。

> [!warning] 🟧 模式选择公式
> **定位设备 → 位置控制**  
> **恒速设备 → 速度控制**  
> **张力/压合 → 转矩控制**

## ② 参考表

| 控制模式 | 典型应用 |
|---|---|
| **位置控制** | 丝杆模组、机械手、分度盘 |
| **速度控制** | 主轴、输送、同步运行 |
| **转矩控制** | 张力、压装、拧紧、收卷 |
| **脉冲控制** | 简单定位 |
| **总线控制** | 多轴联动 |
| **模拟量控制** | 速度/转矩给定 |

## ③ 计算/选型示例

- 已知：  
  设备要求滑台移动到指定坐标。
- 公式：  
  **指定坐标控制 → 位置模式**
- 计算：  
  PLC或运动控制器发出位置指令，伺服驱动器控制电机到达目标位置。
- 结论：  
  该场景应使用 **位置控制模式**。

> [!tip] 记忆口诀
> 到哪里用位置，跑多快用速度，出多大力用转矩。

# 07 电子齿轮与脉冲当量

> [!info] 核心结论
> 电子齿轮用于把控制器发出的脉冲数，换算成电机实际转动量或机械位移。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> PLC发送多少脉冲，伺服转多少距离，不是固定不变的。  
> 它由编码器分辨率、电子齿轮比、机械传动比共同决定。

> [!warning] 🟧 脉冲当量公式
> **脉冲当量 = 每圈机械位移 ÷ 每圈指令脉冲数**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **指令脉冲** | PLC发出的脉冲 |
| **电子齿轮比** | 调整指令与电机转动关系 |
| **丝杆导程** | 电机一圈对应直线位移 |
| **减速比** | 电机到负载的传动比例 |
| **脉冲当量** | 一个脉冲对应多少位移 |
| **定位精度** | 与机械和控制共同相关 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程为10mm，设定每转需要10000个指令脉冲。
- 公式：  
  **脉冲当量 = 10 ÷ 10000**
- 计算：  
  脉冲当量 = **0.001mm/脉冲**
- 结论：  
  PLC每发送1个脉冲，理论移动 **0.001mm**。

> [!tip] 记忆口诀
> 一圈走多远，分成多少脉冲，脉冲当量就出来。

# 08 惯量匹配

> [!info] 核心结论
> 惯量匹配影响伺服响应、稳定性和调试难度，负载惯量过大容易振动或跟随误差大。

## ① 分类/公式/规则

> [!note] 🟦 惯量理解
> 惯量可以理解为“物体不愿意改变运动状态的程度”。  
> 负载越重、半径越大、加减速越快，对伺服要求越高。

> [!warning] 🟧 惯量比公式
> **惯量比 = 负载折算到电机轴的惯量 ÷ 电机转子惯量**

## ② 参考表

| 惯量比 | 含义 |
|---|---|
| **较小** | 响应好，容易调试 |
| **适中** | 常见正常匹配 |
| **过大** | 易振动、响应慢、报警 |
| **减速机** | 可降低折算到电机侧的负载惯量 |
| **刚性设置** | 需按机械情况调试 |
| **最终判断** | 需按厂家推荐范围校核 |

## ③ 计算/选型示例

- 已知：  
  折算到电机轴的负载惯量为 **0.0008kg·m²**，电机转子惯量为 **0.0002kg·m²**。
- 公式：  
  **惯量比 = 0.0008 ÷ 0.0002**
- 计算：  
  惯量比 = **4**
- 结论：  
  该惯量比为4，是否合适需按伺服厂家推荐范围和实际响应要求校核。

> [!tip] 记忆口诀
> 惯量比太大，伺服不好拉；加减速越快，惯量越要查。

# 09 转矩与加速度

> [!info] 核心结论
> 伺服电机加速时需要额外转矩，负载越大、加速度越大，所需转矩越大。

## ① 分类/公式/规则

> [!warning] 🟧 加速转矩公式
> **T = J × α**

> [!note] 🟦 参数说明
> **T**：加速转矩  
> **J**：转动惯量  
> **α**：角加速度  
>  
> 实际选型还要加上摩擦、重力、工艺负载和安全余量。

## ② 参考表

| 项目 | 影响 |
|---|---|
| **负载惯量J** | 越大越难加速 |
| **角加速度α** | 越大所需转矩越大 |
| **摩擦力** | 增加负载转矩 |
| **垂直轴重力** | 需额外保持转矩 |
| **减速机** | 改变转矩和惯量 |
| **峰值转矩** | 决定短时加速能力 |

## ③ 计算/选型示例

- 已知：  
  总惯量 J = **0.01kg·m²**，角加速度 α = **50rad/s²**。
- 公式：  
  **T = J × α**
- 计算：  
  T = 0.01 × 50 = **0.5N·m**
- 结论：  
  仅加速所需转矩约为 **0.5N·m**，实际还需叠加负载和摩擦转矩。

> [!tip] 记忆口诀
> 想加速，就要力；惯量越大，转矩越大。

# 10 丝杆伺服选型

> [!info] 核心结论
> 丝杆伺服选型要把直线速度、推力、导程转换成电机转速和转矩。

## ① 分类/公式/规则

> [!warning] 🟧 丝杆转速公式
> **电机转速 rpm = 直线速度 ÷ 丝杆导程 × 60**

> [!warning] 🟧 丝杆转矩近似公式
> **T = F × L ÷（2π × η）**

> [!note] 🟦 参数说明
> **F**：轴向推力  
> **L**：丝杆导程，单位m/rev  
> **η**：丝杆效率，需按实际结构校核

## ② 参考表

| 参数 | 作用 |
|---|---|
| **丝杆导程** | 一圈走多远 |
| **直线速度** | 决定电机转速 |
| **推力F** | 决定负载转矩 |
| **效率η** | 影响实际转矩 |
| **加速度** | 决定峰值转矩 |
| **行程** | 决定限位和选型 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，直线速度500mm/s。
- 公式：  
  **rpm = 直线速度 ÷ 导程 × 60**
- 计算：  
  rpm = 500 ÷ 10 × 60 = **3000rpm**
- 结论：  
  电机需要达到约 **3000rpm**，需校核伺服额定/最高转速。

> [!tip] 记忆口诀
> 丝杆一圈走导程，速度除导程就是每秒转几圈。

# 11 抱闸与垂直轴

> [!info] 核心结论
> 垂直轴、提升轴、断电后会下滑的机构，通常需要带抱闸伺服电机。

## ① 分类/公式/规则

> [!note] 🟦 抱闸作用
> 伺服电机抱闸主要用于断电保持位置，不是用于频繁动态制动。  
> 运行时应先伺服使能建立转矩，再释放抱闸。

> [!warning] 🟧 抱闸控制逻辑
> **上电使能 → 建立转矩 → 松开抱闸 → 运动**  
> **停止保持 → 抱闸闭合 → 取消使能**

## ② 参考表

| 场景 | 是否关注抱闸 |
|---|---|
| **垂直升降轴** | 必须重点关注 |
| **重物提升** | 必须重点关注 |
| **水平滑台** | 通常按需求选择 |
| **断电防下滑** | 需要抱闸 |
| **安全保持** | 需安全评估 |
| **频繁制动** | 不建议只靠抱闸 |

## ③ 计算/选型示例

- 已知：  
  Z轴断电后负载会下滑。
- 公式：  
  **防下滑 = 伺服保持转矩 + 机械抱闸 + 安全联锁**
- 计算：  
  断电时伺服无输出，必须依靠抱闸或机械结构保持负载。
- 结论：  
  Z轴应选择带抱闸伺服电机，并按负载重量校核保持能力。

> [!tip] 记忆口诀
> 水平轴看需求，垂直轴先看抱闸。

# 12 回零与绝对值

> [!info] 核心结论
> 增量式伺服上电通常需要回零，绝对值伺服可直接读取当前位置。

## ① 分类/公式/规则

> [!note] 🟦 回零逻辑
> 增量系统只知道“走了多少”，不知道“现在在哪”。  
> 绝对值系统能保存位置，上电后可直接知道当前位置。

> [!warning] 🟧 位置建立公式
> **实际坐标 = 机械原点 + 增量位移**

## ② 参考表

| 方案 | 特点 |
|---|---|
| **原点开关回零** | 成本低，精度一般 |
| **Z相信号回零** | 精度较高 |
| **限位回零** | 简单但需谨慎 |
| **绝对值编码器** | 上电读位置 |
| **电池多圈** | 需维护电池 |
| **无电池绝对值** | 需按型号校核 |

## ③ 计算/选型示例

- 已知：  
  设备断电后不能重新回零，且要求继续从当前位置运行。
- 公式：  
  **上电可运行 = 绝对位置可读取**
- 计算：  
  增量系统断电后位置可能丢失，绝对值系统更适合。
- 结论：  
  应优先选择 **绝对值编码器伺服系统**。

> [!tip] 记忆口诀
> 增量要回零，绝对记位置。

# 13 通讯与PLC控制

> [!info] 核心结论
> 伺服系统可通过脉冲、模拟量或总线与PLC/运动控制器连接，其中总线更适合多轴同步和复杂控制。

## ① 分类/公式/规则

> [!note] 🟦 常见控制方式
> 脉冲控制适合简单定位；  
> 模拟量适合速度/转矩给定；  
> EtherCAT、PROFINET、MECHATROLINK等总线适合多轴联动和数据诊断。

> [!warning] 🟧 通讯匹配公式
> **伺服可控 = 控制器协议 + 驱动器协议 + 参数映射 + 控制字逻辑匹配**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **脉冲+方向** | 简单定位 |
| **CW/CCW脉冲** | 正反脉冲控制 |
| **0-10V模拟量** | 速度给定 |
| **±10V模拟量** | 速度/转矩双向给定 |
| **EtherCAT** | 多轴高速同步 |
| **PROFINET** | 西门子系统集成 |
| **Modbus** | 简单参数和监控 |

## ③ 计算/选型示例

- 已知：  
  一台设备有8个伺服轴，需要同步运动。
- 公式：  
  **多轴同步 = 总线通讯 + 同步周期 + 控制器运动功能**
- 计算：  
  脉冲接线复杂，同步性能受限制；总线控制更适合多轴系统。
- 结论：  
  优先选择支持 **实时总线控制** 的伺服系统。

> [!tip] 记忆口诀
> 单轴简单用脉冲，多轴同步用总线。

# 14 参数整定与刚性

> [!info] 核心结论
> 伺服调试的核心是让系统既响应快，又不振动、不过冲、不报警。

## ① 分类/公式/规则

> [!note] 🟦 整定重点
> 刚性太低，响应慢、跟随误差大；  
> 刚性太高，容易振动、啸叫、机械冲击大。  
> 调试要结合机械结构，不能只追求参数越大越好。

> [!warning] 🟧 调试判断公式
> **稳定控制 = 合适增益 + 合适滤波 + 机械刚性 + 负载惯量匹配**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **响应慢** | 增益偏低、加速度太小 |
| **振动啸叫** | 增益过高、机械共振 |
| **定位过冲** | 刚性过高或负载惯量大 |
| **跟随误差大** | 转矩不足、参数偏软 |
| **低速抖动** | 摩擦、机械间隙、参数不合适 |
| **报警频繁** | 加减速过猛或负载异常 |

## ③ 计算/选型示例

- 已知：  
  伺服定位速度慢，但运行平稳。
- 公式：  
  **响应慢 = 增益偏低 / 加速度偏小**
- 计算：  
  可逐步提高刚性或加速度，但每次调整后需观察振动和报警。
- 结论：  
  参数整定要逐步进行，必要时使用厂家自动整定功能。

> [!tip] 记忆口诀
> 刚性低了慢，刚性高了抖；调伺服，要找平衡点。

# 15 常见报警排查

> [!info] 核心结论
> 伺服报警要先看报警代码，再结合发生时刻判断是接线、参数、负载、编码器还是通讯问题。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 上电报警先查接线和参数；  
> 运行报警先查负载和机械；  
> 加速报警先查转矩和加速度；  
> 通讯报警先查协议和地址。

> [!warning] 🟧 排错公式
> **伺服异常 = 电源问题 + 编码器问题 + 参数问题 + 负载问题 + 通讯问题 + 机械问题**

## ② 参考表

| 报警/现象 | 常见原因 |
|---|---|
| **过流** | 负载卡死、接线错误、驱动异常 |
| **过载** | 负载过重、加减速过快 |
| **过压** | 减速太快、再生能量大 |
| **编码器报警** | 线缆、插头、编码器故障 |
| **跟随误差大** | 转矩不足、参数不合适 |
| **通讯报警** | 地址、协议、线缆、映射错误 |
| **位置跑偏** | 脉冲当量、电子齿轮、机械打滑 |

## ③ 计算/选型示例

- 已知：  
  伺服加速时经常过载报警。
- 公式：  
  **加速过载 = 加速度过大 + 负载惯量大 + 电机转矩不足**
- 计算：  
  若机械无卡滞，可先降低加速度或延长加速时间。
- 结论：  
  需检查机械负载、加减速参数、电机容量和惯量匹配。

> [!tip] 记忆口诀
> 上电报错查接线，运动报错查负载；加速报警查转矩，定位不准查当量。

# 16 交流伺服系统快速选型表

> [!info] 核心结论
> 伺服选型必须从机械负载出发，先算速度、转矩和惯量，再选电机、驱动器和控制方式。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看运动方式：旋转/直线  
> ② 看负载重量和摩擦  
> ③ 看最高速度  
> ④ 看加减速时间  
> ⑤ 算连续转矩  
> ⑥ 算峰值转矩  
> ⑦ 校核惯量比  
> ⑧ 选电机功率和抱闸  
> ⑨ 选驱动器和通讯  
> ⑩ 校核安装环境和线缆

> [!warning] 🟧 最终选型公式
> **推荐伺服系统 = 电机转矩 + 电机转速 + 惯量匹配 + 驱动器功率 + 控制模式 + 通讯接口**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 精准定位 | 位置模式、编码器分辨率 |
| 多轴同步 | 实时总线、运动控制器 |
| 垂直轴 | 抱闸、保持转矩、安全 |
| 低速大转矩 | 峰值转矩、刚性调试 |
| 高速往复 | 加速度、惯量比、散热 |
| 压装/张力 | 转矩模式、力控反馈 |
| 断电不回零 | 绝对值编码器 |

## ③ 计算/选型示例

- 已知：  
  丝杆导程10mm，目标速度500mm/s，要求每个脉冲移动0.001mm，垂直轴运行。
- 公式：  
  **电机转速 = 速度 ÷ 导程 × 60**  
  **每圈脉冲数 = 导程 ÷ 脉冲当量**
- 计算：  
  电机转速 = 500 ÷ 10 × 60 = **3000rpm**  
  每圈脉冲数 = 10 ÷ 0.001 = **10000脉冲/圈**
- 结论：  
  应选择满足 **3000rpm运行、10000脉冲/圈指令、带抱闸** 的伺服系统，并继续校核转矩和惯量。

> [!tip] 选型口诀
> 速度看转速，负载看转矩；响应看惯量，垂直看抱闸；多轴看总线，精度看当量。
