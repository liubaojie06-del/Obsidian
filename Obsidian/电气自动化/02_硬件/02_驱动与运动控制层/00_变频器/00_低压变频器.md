# 01 低压变频器总览

> [!info] 核心结论
> 低压变频器用于控制电机的 **启动、停止、转速、方向、转矩和保护**，本质是把固定电源变成可调频率、可调电压的电源。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 工频电源直接驱动电机时，电机基本按固定速度运行。  
> 变频器通过改变输出频率，让电机实现平滑调速。

> [!warning] 🟧 通用选型公式
> **变频器选型 = 电机功率 + 额定电流 + 负载类型 + 电源电压 + 控制方式 + 制动需求 + 环境条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **低压变频器** | 常用于AC 220V/380V/400V等级，需按型号校核 |
| **控制对象** | 三相异步电机、部分同步电机 |
| **核心功能** | 调速、软启动、保护、节能 |
| **常见控制** | V/F控制、矢量控制、转矩控制 |
| **输入信号** | DI、AI、通讯、面板 |
| **输出信号** | DO、AO、继电器、通讯状态 |

## ③ 计算/选型示例

- 已知：  
  一台三相异步电机功率 **2.2kW**，额定电流 **5A**，用于普通风机。
- 公式：  
  **变频器额定电流 ≥ 电机额定电流**
- 计算：  
  变频器输出额定电流应不小于 **5A**，并按负载类型留余量。
- 结论：  
  可选择匹配 **2.2kW等级** 的低压变频器，最终按厂家手册校核电流、过载能力和应用场景。

> [!tip] 记忆口诀
> 选变频，先看电机电流，再看负载轻重，最后看控制和环境。

# 02 底层原理

> [!info] 核心结论
> 变频器的本质，是先把交流电整成直流，再把直流逆变成频率可调的交流电。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 低压变频器一般由 **整流单元、直流母线、逆变单元、控制板、驱动保护电路** 组成。

> [!warning] 🟧 变频器能量转换逻辑
> **AC输入 → 整流成DC → 直流母线储能 → IGBT逆变 → 输出可调AC**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **整流单元** | 把交流变成直流 |
| **直流母线** | 储能、滤波 |
| **逆变单元** | 把直流变成可调交流 |
| **IGBT** | 高频开关器件 |
| **控制板** | 计算频率、电压、保护逻辑 |
| **散热系统** | 带走功率器件热量 |

## ③ 计算/选型示例

- 已知：  
  电机需要从0Hz平滑升到50Hz。
- 公式：  
  **输出频率由变频器控制**
- 计算：  
  变频器逐步提高输出频率，同时按控制方式调整输出电压。
- 结论：  
  电机不会像直接工频启动那样冲击大，而是按设定加速时间平滑启动。

> [!tip] 记忆口诀
> 先进整流成直流，再逆变成可调交流。

# 03 频率与电机转速

> [!info] 核心结论
> 电机转速主要由输出频率决定，频率越高，同步转速越高。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **n = 120 × f ÷ P**

> [!note] 🟦 参数说明
> **n**：同步转速，rpm  
> **f**：电源频率，Hz  
> **P**：电机极数  
>  
> 实际异步电机转速会略低于同步转速，需考虑转差率。

## ② 参考表

| 电机极数 | 50Hz同步转速 |
|---|---|
| **2极** | 3000rpm |
| **4极** | 1500rpm |
| **6极** | 1000rpm |
| **8极** | 750rpm |

## ③ 计算/选型示例

- 已知：  
  4极电机，变频器输出频率为 **30Hz**。
- 公式：  
  **n = 120 × f ÷ P**
- 计算：  
  n = 120 × 30 ÷ 4 = **900rpm**
- 结论：  
  4极电机在30Hz时，同步转速约 **900rpm**，实际转速会略低。

> [!tip] 记忆口诀
> 频率管速度，极数定基准；4极50Hz，大约1500转。

# 04 V/F控制

> [!info] 核心结论
> V/F控制通过保持电压和频率比例关系，让电机磁通基本稳定，适合风机、水泵、输送线等普通调速场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 频率降低时，如果电压不降低，电机容易磁饱和发热。  
> 因此V/F控制会按频率比例降低输出电压。

> [!warning] 🟧 V/F基本关系
> **V ÷ f ≈ 常数**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 简单、稳定、调试容易 |
| **适合负载** | 风机、水泵、普通输送 |
| **低速性能** | 一般 |
| **转矩控制** | 不如矢量控制 |
| **参数要求** | 相对简单 |
| **注意** | 低频大转矩需校核 |

## ③ 计算/选型示例

- 已知：  
  电机额定电压380V，额定频率50Hz，采用线性V/F。
- 公式：  
  **V/f = 380 ÷ 50**
- 计算：  
  V/f = **7.6V/Hz**  
  30Hz时理论电压约 30 × 7.6 = **228V**
- 结论：  
  30Hz时变频器会降低输出电压，维持电机磁通稳定。

> [!tip] 记忆口诀
> 频率降，电压跟着降；V/F成比例，电机不乱跑。

# 05 矢量控制

> [!info] 核心结论
> 矢量控制能更精确地控制电机磁通和转矩，适合低速大转矩、响应要求高的场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> V/F控制像“粗调速度”，矢量控制像“精细控制转矩”。  
> 矢量控制通常需要准确输入电机铭牌参数，必要时做电机自学习。

> [!warning] 🟧 矢量控制适用公式
> **适合矢量控制 = 低速运行 + 转矩要求高 + 速度响应快 + 负载变化大**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **V/F控制** | 风机、水泵、普通调速 |
| **无PG矢量** | 不带编码器，性能较好 |
| **有PG矢量** | 带编码器，速度/转矩更准 |
| **转矩控制** | 卷绕、张力、负载控制 |
| **闭环控制** | 高精度速度或位置反馈 |
| **注意** | 参数不准会影响效果 |

## ③ 计算/选型示例

- 已知：  
  输送设备低速运行时需要较大启动转矩。
- 公式：  
  **低速大转矩需求 → 矢量控制优先**
- 计算：  
  V/F低速可能转矩不足，矢量控制低速性能更好。
- 结论：  
  可选择支持 **无PG矢量或有PG矢量控制** 的变频器，电机参数需准确设置。

> [!tip] 记忆口诀
> 普通调速用V/F，低速有力用矢量。

# 06 变频器选型

> [!info] 核心结论
> 变频器不能只按电机功率选，还要看电机电流、负载类型、过载能力、制动和环境。

## ① 分类/公式/规则

> [!note] 🟦 选型主线
> 风机水泵属于轻载或平方转矩负载；  
> 输送、搅拌、提升、挤出等通常对转矩要求更高；  
> 起停频繁或惯量大时，要考虑制动单元和制动电阻。

> [!warning] 🟧 选型核心公式
> **变频器额定输出电流 ≥ 电机额定电流 × 负载余量系数**

## ② 参考表

| 负载类型 | 选型关注点 |
|---|---|
| **风机/水泵** | 节能、PID、轻载过载能力 |
| **输送线** | 启动转矩、加减速时间 |
| **搅拌机** | 重载、低速转矩 |
| **提升机构** | 制动、安全、转矩余量 |
| **离心机** | 大惯量、减速制动 |
| **多电机** | 总电流和保护方式需校核 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流 **10A**，负载较重，建议留 **1.2倍** 余量。
- 公式：  
  **所需变频器电流 ≥ 10 × 1.2**
- 计算：  
  所需电流 ≥ **12A**
- 结论：  
  应选择输出额定电流不小于 **12A** 的变频器，最终按厂家重载/轻载规格校核。

> [!tip] 记忆口诀
> 功率只是参考，电流才是关键；轻载看节能，重载看余量。

# 07 主回路接线

> [!info] 核心结论
> 变频器主回路接线要分清输入端、输出端、电机端、制动端和接地端，严禁把电源接到输出端。

## ① 分类/公式/规则

> [!note] 🟦 接线主线
> 输入端接电源，输出端接电机，接地端必须可靠接地。  
> 变频器输出端不允许直接接电容补偿柜，也不建议随意接接触器切换运行。

> [!warning] 🟧 主回路接线规则
> **电源 → 变频器输入端**  
> **变频器输出端 → 电机**  
> **PE端 → 可靠接地**

## ② 参考表

| 端子 | 作用 |
|---|---|
| **R/S/T 或 L/N** | 变频器输入电源 |
| **U/V/W** | 变频器输出到电机 |
| **PE** | 保护接地 |
| **P/+、N/-** | 直流母线端，需谨慎 |
| **PB、P/+** | 制动电阻端，按型号确认 |
| **制动单元** | 大功率制动场景使用 |

## ③ 计算/选型示例

- 已知：  
  安装时误把三相电源接到U/V/W输出端。
- 公式：  
  **错误风险 = 外部电源直接进入逆变模块**
- 计算：  
  可能直接损坏IGBT逆变单元。
- 结论：  
  接线前必须核对端子标识，主回路接线完成后再上电。

> [!tip] 记忆口诀
> 输入接电源，输出接电机；U/V/W不能接市电。

# 08 控制端子与PLC控制

> [!info] 核心结论
> 变频器可以通过端子、模拟量、通讯或面板控制，接PLC时要确认控制模式和信号逻辑。

## ① 分类/公式/规则

> [!note] 🟦 常见控制方式
> DI控制启停和方向；  
> AI控制频率给定；  
> DO/继电器反馈运行、故障、到频；  
> 通讯可读取状态、设定频率和故障码。

> [!warning] 🟧 PLC控制公式
> **变频器运行 = 运行命令来源 + 频率给定来源 + 端子逻辑 + 参数设置一致**

## ② 参考表

| 信号 | 作用 |
|---|---|
| **DI数字输入** | 启动、停止、正反转、多段速 |
| **AI模拟输入** | 0-10V、4-20mA频率给定 |
| **DO数字输出** | 运行中、到频、报警 |
| **继电器输出** | 故障、运行状态 |
| **RS485/总线** | 通讯控制和监控 |
| **公共端** | PNP/NPN逻辑需校核 |

## ③ 计算/选型示例

- 已知：  
  PLC输出0-10V控制频率，变频器最大频率设置为50Hz，PLC输出6V。
- 公式：  
  **频率 = 电压 ÷ 10 × 最大频率**
- 计算：  
  频率 = 6 ÷ 10 × 50 = **30Hz**
- 结论：  
  变频器应输出约 **30Hz**，前提是AI量程和频率给定来源设置正确。

> [!tip] 记忆口诀
> 启停看DI，速度看AI；通讯能全控，参数要对齐。

# 09 电机参数与自学习

> [!info] 核心结论
> 变频器调试前必须输入正确的电机铭牌参数，矢量控制尤其依赖电机参数准确性。

## ① 分类/公式/规则

> [!note] 🟦 必填参数
> 常见电机参数包括：额定功率、额定电压、额定电流、额定频率、额定转速、功率因数等。

> [!warning] 🟧 参数设置公式
> **控制效果 = 电机参数准确 + 控制方式匹配 + 自学习完成**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **额定功率** | 匹配电机容量 |
| **额定电压** | 输出电压基准 |
| **额定电流** | 过载保护依据 |
| **额定频率** | 基准频率 |
| **额定转速** | 转差和速度估算 |
| **自学习** | 识别电机等效参数 |

## ③ 计算/选型示例

- 已知：  
  电机铭牌额定电流为 **7.2A**，变频器参数中误设为 **4A**。
- 公式：  
  **保护判断依据 = 变频器设定电机额定电流**
- 计算：  
  设定过小，变频器可能过早报过载。
- 结论：  
  应按电机铭牌准确设置额定电流，参数需按厂家手册校核。

> [!tip] 记忆口诀
> 铭牌参数要抄准，矢量控制才有劲。

# 10 加减速时间

> [!info] 核心结论
> 加减速时间决定电机从低速到高速、从高速到停止的变化快慢，设置不合理容易过流或过压。

## ① 分类/公式/规则

> [!note] 🟦 调试逻辑
> 加速太快，电机需要更大电流，容易过流；  
> 减速太快，负载惯性能量回灌母线，容易过压。

> [!warning] 🟧 加减速判断公式
> **加速过短 → 易过流**  
> **减速过短 → 易过压**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **加速时间** | 启动冲击、电流大小 |
| **减速时间** | 母线电压、制动需求 |
| **S曲线** | 启停更平滑 |
| **大惯量负载** | 减速时间需加长 |
| **重载启动** | 加速时间需校核 |
| **频繁启停** | 散热和制动需校核 |

## ③ 计算/选型示例

- 已知：  
  风机从50Hz减速到0Hz，减速时间设为2秒，经常报过压。
- 公式：  
  **过压风险 = 大惯量 + 减速太快 + 能量回灌**
- 计算：  
  风机惯量较大，快速减速会把能量回灌到直流母线。
- 结论：  
  可适当延长减速时间，必要时配置制动单元和制动电阻。

> [!tip] 记忆口诀
> 加速太急容易过流，减速太猛容易过压。

# 11 制动单元与制动电阻

> [!info] 核心结论
> 大惯量、频繁减速、快速停车或下放负载时，需要考虑制动单元和制动电阻。

## ① 分类/公式/规则

> [!note] 🟦 制动原理
> 电机减速时可能变成发电机，把能量回馈到直流母线。  
> 制动电阻把这部分能量转化为热量，避免母线过压。

> [!warning] 🟧 制动需求公式
> **需要制动 = 减速时间短 + 负载惯量大 + 母线过压风险高**

## ② 参考表

| 场景 | 是否关注制动 |
|---|---|
| **风机水泵自然停** | 通常较少 |
| **离心机快速停机** | 重点关注 |
| **提升下放** | 重点关注 |
| **频繁启停输送** | 需校核 |
| **大惯量转盘** | 需校核 |
| **紧急停车** | 需安全评估 |

## ③ 计算/选型示例

- 已知：  
  转盘惯量大，要求从50Hz在3秒内停车。
- 公式：  
  **快速停车能量 = 机械惯性能量回灌**
- 计算：  
  惯量越大、停车越快，制动能量越大。
- 结论：  
  应校核制动单元容量和制动电阻功率，具体参数按厂家手册选型。

> [!tip] 记忆口诀
> 减速能量去哪儿？不让母线憋着，就让电阻发热。

# 12 风机水泵PID控制

> [!info] 核心结论
> 变频器常用于风机水泵恒压、恒流、恒温控制，通过PID自动调节频率。

## ① 分类/公式/规则

> [!note] 🟦 控制逻辑
> 传感器反馈压力、流量或温度；  
> 变频器比较设定值和反馈值；  
> PID自动调节输出频率，让反馈接近设定值。

> [!warning] 🟧 PID控制公式
> **控制偏差 = 设定值 - 反馈值**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **设定值SP** | 目标压力/流量/温度 |
| **反馈值PV** | 传感器实际测量值 |
| **输出MV** | 变频器输出频率 |
| **P比例** | 反应快慢 |
| **I积分** | 消除静差 |
| **D微分** | 抑制快速变化，实际需谨慎 |

## ③ 计算/选型示例

- 已知：  
  恒压供水设定压力 **0.4MPa**，当前反馈压力 **0.32MPa**。
- 公式：  
  **偏差 = SP - PV**
- 计算：  
  偏差 = 0.4 - 0.32 = **0.08MPa**
- 结论：  
  压力偏低，变频器应提高输出频率，增加水泵转速。

> [!tip] 记忆口诀
> 反馈低了就加速，反馈高了就降速。

# 13 EMC与接地

> [!info] 核心结论
> 变频器是强干扰源，安装时必须重视接地、屏蔽、电缆布线、滤波和电抗器。

## ① 分类/公式/规则

> [!note] 🟦 干扰来源
> 变频器IGBT高速开关会产生高频干扰，可能影响PLC、传感器、编码器、通讯和模拟量信号。

> [!warning] 🟧 抗干扰公式
> **稳定运行 = 可靠接地 + 动力/信号分开 + 屏蔽电缆 + 滤波/电抗器按需配置**

## ② 参考表

| 项目 | 建议 |
|---|---|
| **PE接地** | 变频器和电机可靠接地 |
| **电机电缆** | 长距离建议屏蔽或按规范处理 |
| **信号线** | 远离U/V/W动力线 |
| **模拟量线** | 用屏蔽线，单点接地需按规范 |
| **输入电抗器** | 改善电网侧冲击 |
| **输出电抗器** | 长电缆或特殊电机场景校核 |

## ③ 计算/选型示例

- 已知：  
  变频器运行时，附近编码器计数跳变。
- 公式：  
  **干扰风险 = 动力线干扰 + 屏蔽接地不良 + 信号线距离过近**
- 计算：  
  编码器线若与电机线同槽敷设，容易受到高频干扰。
- 结论：  
  应分开布线、检查屏蔽接地，必要时采用差分信号、滤波器或电抗器。

> [!tip] 记忆口诀
> 变频器一运行，干扰就可能来；动力信号分开走，接地屏蔽不能少。

# 14 常见故障排查

> [!info] 核心结论
> 变频器故障排查要先看故障代码，再按电源、电机、负载、参数、散热和接线逐项检查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看面板故障码，再查主回路；  
> 先空载试运行，再带载测试；  
> 先查参数，再判断硬件故障。

> [!warning] 🟧 排错公式
> **变频器异常 = 电源异常 + 电机异常 + 负载异常 + 参数错误 + 散热不良 + 干扰问题**

## ② 参考表

| 故障 | 常见原因 |
|---|---|
| **过流OC** | 加速太快、负载卡滞、电机短路 |
| **过压OV** | 减速太快、制动不足、电网电压高 |
| **欠压UV** | 输入电压低、瞬时掉电 |
| **过载OL** | 负载过重、电机参数错误 |
| **过热OH** | 风扇坏、散热差、环境温度高 |
| **通讯故障** | 地址、波特率、协议、线缆问题 |

## ③ 计算/选型示例

- 已知：  
  变频器启动时报过流OC。
- 公式：  
  **启动过流 = 加速太快 + 启动转矩不足 + 负载过重 + 电机/线缆异常**
- 计算：  
  若机械负载未卡死，可先适当延长加速时间并检查电机参数。
- 结论：  
  排查机械卡滞、电机绝缘、输出线缆、加速时间和转矩提升参数。

> [!tip] 记忆口诀
> 过流查负载和加速，过压查减速和制动，过热查风扇和散热。

# 15 低压变频器快速选型表

> [!info] 核心结论
> 低压变频器选型先看电机和负载，再看控制方式、制动、通讯、环境和保护要求。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看电源电压  
> ② 看电机功率  
> ③ 看电机额定电流  
> ④ 看负载轻重  
> ⑤ 看控制方式  
> ⑥ 看是否需要制动  
> ⑦ 看是否需要通讯  
> ⑧ 看防护等级和散热  
> ⑨ 看输入输出信号  
> ⑩ 看厂家手册和规范校核

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电压等级 + 输出电流 + 负载类型 + 控制方式 + 制动配置 + 通讯接口 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 风机水泵节能 | V/F、PID、轻载规格 |
| 输送线调速 | 启停、加减速、端子控制 |
| 低速大转矩 | 矢量控制、参数自学习 |
| 提升机构 | 制动、安全、重载余量 |
| 多段速控制 | DI端子或通讯控制 |
| PLC通讯 | RS485、PROFINET、Modbus等 |
| 强干扰现场 | 接地、屏蔽、电抗器、滤波 |

## ③ 计算/选型示例

- 已知：  
  380V三相电机，功率 **7.5kW**，额定电流 **15A**，用于搅拌负载，需要低速有力。
- 公式：  
  **型号需求 = 380V + 输出电流≥15A + 重载能力 + 矢量控制**
- 计算：  
  搅拌属于较重负载，不能只按轻载风机水泵规格选型。
- 结论：  
  应选择满足 **重载电流和矢量控制** 的低压变频器，过载能力、制动、散热和防护等级需按厂家手册校核。

> [!tip] 选型口诀
> 电压先匹配，电流要够用；轻载看节能，重载看转矩；减速怕过压，制动要校核。
