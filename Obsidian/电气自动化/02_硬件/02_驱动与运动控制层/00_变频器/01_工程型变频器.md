# 01 工程型变频器总览

> [!info] 核心结论
> 工程型变频器适合 **重载、高动态、高精度、多电机协同、复杂工艺控制** 场景，不只是简单调速。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 普通变频器偏向单机调速；  
> 工程型变频器更强调 **系统控制能力、过载能力、通讯能力、扩展能力、安全能力和工艺适配能力**。

> [!warning] 🟧 通用选型公式
> **工程型变频器选型 = 电机参数 + 负载特性 + 控制精度 + 过载能力 + 制动/回馈 + 通讯总线 + 安全功能**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **工程型变频器** | 面向复杂工业传动系统 |
| **典型应用** | 起重、冶金、造纸、线缆、卷绕、测试台 |
| **控制方式** | V/F、无PG矢量、有PG矢量、转矩控制 |
| **扩展能力** | 编码器、通讯卡、I/O扩展、安全模块 |
| **重点参数** | 电流、过载、响应、制动、通讯、安全 |
| **校核要求** | 需按厂家手册/项目规范校核 |

## ③ 计算/选型示例

- 已知：  
  一台提升设备要求低速大转矩、频繁启停、快速制动。
- 公式：  
  **型号需求 = 重载能力 + 闭环矢量 + 制动能力 + 安全功能**
- 计算：  
  普通风机水泵型变频器可能无法满足低速转矩和频繁制动需求。
- 结论：  
  应优先选择 **工程型变频器**，并校核过载能力、制动单元、编码器反馈和安全功能。

> [!tip] 记忆口诀
> 普通调速看功率，工程传动看系统。

# 02 底层架构

> [!info] 核心结论
> 工程型变频器本质上仍是AC-DC-AC结构，但控制板、扩展模块、通讯和保护能力更强。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 输入交流电先整流成直流，再通过直流母线储能，最后由IGBT逆变输出可控交流电给电机。

> [!warning] 🟧 能量转换逻辑
> **AC输入 → 整流 → DC母线 → 逆变 → 电机输出转矩**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **整流单元** | 将交流变直流 |
| **直流母线** | 储能、滤波、能量交换 |
| **逆变单元** | 输出可调电压和频率 |
| **控制板** | 运行控制算法 |
| **扩展卡** | 通讯、编码器、I/O、安全 |
| **保护单元** | 过流、过压、过热、缺相等保护 |

## ③ 计算/选型示例

- 已知：  
  负载减速时经常报母线过压。
- 公式：  
  **母线过压 = 再生能量 > 变频器吸收能力**
- 计算：  
  负载惯量大，减速时电机发电，能量回灌到直流母线。
- 结论：  
  需校核制动电阻、制动单元或回馈单元配置。

> [!tip] 记忆口诀
> 加速电网给能量，减速负载还能量。

# 03 工程型 vs 通用型

> [!info] 核心结论
> 工程型变频器更适合重载和复杂控制，通用型变频器更适合普通风机、水泵、输送等简单调速。

## ① 分类/公式/规则

> [!note] 🟦 快速区分
> 通用型：关注调速和节能。  
> 工程型：关注 **转矩、同步、制动、通讯、闭环、安全和工艺控制**。

> [!warning] 🟧 区分公式
> **普通负载 → 通用型**  
> **重载/高精度/多轴协同 → 工程型**

## ② 参考表

| 对比项 | 通用型变频器 | 工程型变频器 |
|---|---|---|
| 主要用途 | 普通调速 | **复杂传动** |
| 低速转矩 | 一般 | **更强** |
| 扩展能力 | 基础 | **较强** |
| 通讯能力 | 常规 | **多总线支持** |
| 制动能力 | 简单 | **可扩展制动/回馈** |
| 应用场景 | 风机水泵 | 起重、卷绕、冶金、测试台 |

## ③ 计算/选型示例

- 已知：  
  设备要求多台电机速度同步，且需要PLC通过总线实时控制。
- 公式：  
  **系统需求 = 多机同步 + 通讯控制 + 参数联动**
- 计算：  
  通用型变频器可能通讯和同步能力不足。
- 结论：  
  应选择支持总线通讯和同步控制的 **工程型变频器**。

> [!tip] 记忆口诀
> 单机简单用通用，多机复杂用工程。

# 04 典型应用场景

> [!info] 核心结论
> 工程型变频器常用于负载重、工艺复杂、停机要求高或需要闭环控制的设备。

## ① 分类/公式/规则

> [!note] 🟦 场景主线
> 只要现场出现 **重载启动、低速大转矩、快速制动、多电机同步、张力控制、位置控制**，就应考虑工程型变频器。

> [!warning] 🟧 应用判断公式
> **适合工程型 = 负载复杂 + 控制要求高 + 系统联动多**

## ② 参考表

| 场景 | 关注点 |
|---|---|
| **起重提升** | 制动、安全、低速转矩 |
| **卷绕收放卷** | 张力、转矩、卷径计算 |
| **冶金轧机** | 同步、动态响应、重载 |
| **造纸/薄膜** | 多轴速度同步 |
| **测试台** | 转矩控制、能量回馈 |
| **输送主线** | 多段速、总线控制、联动 |

## ③ 计算/选型示例

- 已知：  
  收卷设备需要随着卷径变大自动调整转速。
- 公式：  
  **线速度 = 角速度 × 卷径半径**
- 计算：  
  卷径变大时，如果转速不降低，线速度会升高，导致张力变化。
- 结论：  
  应选支持 **卷径计算、张力控制或转矩控制** 的工程型变频器。

> [!tip] 记忆口诀
> 起重看制动，卷绕看张力，多轴看同步。

# 05 控制方式选择

> [!info] 核心结论
> 工程型变频器控制方式要按负载要求选择，低要求用V/F，高要求用矢量或闭环控制。

## ① 分类/公式/规则

> [!note] 🟦 控制方式
> V/F控制适合普通调速；  
> 无PG矢量适合多数低速转矩场景；  
> 有PG矢量适合高精度速度、转矩和位置控制。

> [!warning] 🟧 控制方式公式
> **控制精度要求越高 → 越倾向闭环矢量控制**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **V/F控制** | 风机、水泵、普通输送 |
| **无PG矢量** | 低速转矩、一般重载 |
| **有PG矢量** | 起重、卷绕、精密速度 |
| **转矩控制** | 张力、测试台、卷绕 |
| **位置控制** | 简单定位、同步控制 |
| **主从控制** | 多电机联动 |

## ③ 计算/选型示例

- 已知：  
  提升机构要求低速运行平稳，不能溜钩。
- 公式：  
  **提升稳定 = 低速转矩 + 编码器反馈 + 制动逻辑**
- 计算：  
  无反馈控制可能在低速和重载时转矩控制不足。
- 结论：  
  建议选择 **有PG闭环矢量控制** 的工程型变频器。

> [!tip] 记忆口诀
> 普通速度用V/F，重载低速用矢量，精密控制上闭环。

# 06 容量与电流选型

> [!info] 核心结论
> 工程型变频器选型必须以电机额定电流和负载过载需求为核心，不能只看kW。

## ① 分类/公式/规则

> [!note] 🟦 选型主线
> 同样功率下，不同电机额定电流可能不同；  
> 同样电机下，不同负载对过载能力要求不同。

> [!warning] 🟧 核心公式
> **变频器额定输出电流 ≥ 电机额定电流 × 负载余量系数**

## ② 参考表

| 负载 | 建议关注 |
|---|---|
| **轻载** | 风机、水泵，节能优先 |
| **恒转矩** | 输送、搅拌，看电流余量 |
| **重载** | 挤出、破碎，看过载能力 |
| **提升** | 制动、转矩、安全逻辑 |
| **大惯量** | 减速能量和制动 |
| **频繁启停** | 热容量和散热 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流 **22A**，重载应用建议留 **1.3倍** 余量。
- 公式：  
  **所需电流 ≥ 22 × 1.3**
- 计算：  
  所需电流 ≥ **28.6A**
- 结论：  
  应选择输出额定电流不小于 **28.6A** 的工程型变频器，最终按厂家重载规格校核。

> [!tip] 记忆口诀
> kW看等级，电流看真相；工程选型，余量要算上。

# 07 过载能力

> [!info] 核心结论
> 工程型变频器必须校核短时过载能力，尤其是启动、冲击负载、提升和堵转风险场景。

## ① 分类/公式/规则

> [!note] 🟦 过载理解
> 过载能力表示变频器在短时间内输出超过额定电流的能力。  
> 工程场景常见 **150% 60s、180% 10s** 等规格，具体以厂家手册为准。

> [!warning] 🟧 过载判断公式
> **峰值负载电流 ≤ 变频器允许过载电流**

## ② 参考表

| 场景 | 过载关注点 |
|---|---|
| **重载启动** | 启动峰值电流 |
| **提升起升** | 起动转矩和抱闸逻辑 |
| **冲击负载** | 瞬时电流冲击 |
| **挤出机** | 长时间高负载 |
| **破碎机** | 堵转和冲击 |
| **大惯量设备** | 加速电流和热容量 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流20A，启动峰值可能达到额定的 **160%**。
- 公式：  
  **峰值电流 = 20 × 1.6**
- 计算：  
  峰值电流 = **32A**
- 结论：  
  变频器短时过载能力必须覆盖 **32A**，持续时间也需满足启动过程。

> [!tip] 记忆口诀
> 额定电流看长期，过载能力看瞬间。

# 08 制动与回馈

> [!info] 核心结论
> 工程型变频器常需要处理再生能量，快速停车、提升下放、大惯量减速时要重点校核制动或回馈。

## ① 分类/公式/规则

> [!note] 🟦 两种处理方式
> **制动电阻**：把能量变成热量消耗掉。  
> **回馈单元**：把能量回馈电网，适合能量较大或频繁再生场景。

> [!warning] 🟧 制动需求公式
> **再生风险 = 大惯量 + 快速减速 + 下放负载 + 频繁启停**

## ② 参考表

| 方案 | 适合场景 |
|---|---|
| **制动电阻** | 中小功率、短时制动 |
| **制动单元** | 控制制动电阻投入 |
| **回馈单元** | 大功率、频繁再生 |
| **共直流母线** | 多传动能量共享 |
| **能耗制动** | 简单可靠但发热 |
| **再生回馈** | 节能但系统复杂 |

## ③ 计算/选型示例

- 已知：  
  起重机构下放重物时母线电压升高。
- 公式：  
  **下放工况 = 电机发电 + 能量回灌母线**
- 计算：  
  重物下放时机械势能转为电能，若无法消耗，会导致母线过压。
- 结论：  
  应配置制动电阻或回馈单元，具体容量按负载惯量、下降速度和工作制校核。

> [!tip] 记忆口诀
> 减速会发电，下放更明显；能量不处理，母线就过压。

# 09 编码器闭环

> [!info] 核心结论
> 闭环矢量控制需要编码器反馈速度或位置，适合高精度、高响应和防失速场景。

## ① 分类/公式/规则

> [!note] 🟦 闭环逻辑
> 变频器发出控制指令后，通过编码器读取实际速度，再修正输出转矩和频率。

> [!warning] 🟧 闭环控制公式
> **控制偏差 = 目标速度 - 实际速度**

## ② 参考表

| 编码器类型 | 关注点 |
|---|---|
| **增量编码器** | A/B/Z，测速常用 |
| **绝对值编码器** | 上电可读位置 |
| **差分输出** | 抗干扰强 |
| **PNP/NPN输出** | 需匹配PG卡 |
| **编码器卡** | 与变频器型号匹配 |
| **屏蔽接地** | 防止计数干扰 |

## ③ 计算/选型示例

- 已知：  
  目标转速为500rpm，编码器反馈实际转速为480rpm。
- 公式：  
  **速度偏差 = 500 - 480**
- 计算：  
  速度偏差 = **20rpm**
- 结论：  
  闭环控制会根据偏差增加输出转矩，尽量让实际速度接近目标速度。

> [!tip] 记忆口诀
> 开环靠估算，闭环看反馈；编码器越稳，控制越准。

# 10 通讯与PLC集成

> [!info] 核心结论
> 工程型变频器常通过现场总线与PLC集成，实现启停、频率、转矩、状态、故障和参数读写。

## ① 分类/公式/规则

> [!note] 🟦 常见通讯
> 常见通讯包括 Modbus RTU/TCP、PROFINET、PROFIBUS、EtherCAT、EtherNet/IP、CANopen 等，具体取决于品牌和选件。

> [!warning] 🟧 通讯匹配公式
> **通讯成功 = 协议一致 + 地址正确 + 参数映射正确 + 控制字/状态字逻辑正确**

## ② 参考表

| 数据 | 作用 |
|---|---|
| **控制字** | 启停、复位、方向 |
| **状态字** | 运行、故障、到频 |
| **频率给定** | PLC设定速度 |
| **转矩给定** | 张力/测试台控制 |
| **实际频率** | 反馈运行状态 |
| **故障码** | 诊断变频器报警 |

## ③ 计算/选型示例

- 已知：  
  PLC通过通讯给定频率，变频器最大频率50Hz，通讯给定值按0～10000对应0～50Hz。
- 公式：  
  **频率 = 给定值 ÷ 10000 × 50**
- 计算：  
  给定值6000时，频率 = 6000 ÷ 10000 × 50 = **30Hz**
- 结论：  
  PLC写入6000，变频器应输出约 **30Hz**，前提是通讯比例和控制源设置正确。

> [!tip] 记忆口诀
> 通讯控变频，控制字先行；状态字看反馈，故障码找原因。

# 11 多电机与主从控制

> [!info] 核心结论
> 多电机系统要关注速度同步、转矩分配、主从关系和通讯延迟。

## ① 分类/公式/规则

> [!note] 🟦 主从控制
> 主驱动提供速度或位置基准；  
> 从驱动跟随主驱动速度、转矩或位置，实现机械同步。

> [!warning] 🟧 同步控制公式
> **从轴给定 = 主轴给定 × 同步比例 + 补偿量**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **速度同步** | 多输送、多辊 |
| **转矩均衡** | 双电机拖动同一负载 |
| **主从控制** | 主轴带从轴 |
| **电子齿轮** | 比例同步 |
| **电子凸轮** | 按曲线同步 |
| **共直流母线** | 多驱能量共享 |

## ③ 计算/选型示例

- 已知：  
  主轴速度为1000rpm，从轴按0.8倍同步。
- 公式：  
  **从轴速度 = 主轴速度 × 同步比例**
- 计算：  
  从轴速度 = 1000 × 0.8 = **800rpm**
- 结论：  
  从轴目标速度应为 **800rpm**，实际还需考虑机械传动比和反馈误差。

> [!tip] 记忆口诀
> 主轴定节奏，从轴跟比例；多机联动，参数要统一。

# 12 张力与卷绕控制

> [!info] 核心结论
> 卷绕类设备重点不是单纯调速，而是保持材料张力稳定。

## ① 分类/公式/规则

> [!note] 🟦 张力控制逻辑
> 收卷时卷径不断变大，放卷时卷径不断变小。  
> 变频器需要根据卷径、线速度、张力反馈或转矩模型调整速度和转矩。

> [!warning] 🟧 转矩关系公式
> **转矩 T = 张力 F × 卷径半径 r**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **张力F** | 材料拉力 |
| **卷径D** | 卷材直径 |
| **半径r** | D/2 |
| **线速度** | 材料实际运行速度 |
| **转矩给定** | 保持张力 |
| **张力反馈** | 传感器或摆杆反馈 |

## ③ 计算/选型示例

- 已知：  
  目标张力 F = **100N**，当前卷径半径 r = **0.2m**。
- 公式：  
  **T = F × r**
- 计算：  
  T = 100 × 0.2 = **20N·m**
- 结论：  
  当前理论转矩约为 **20N·m**，实际需考虑机械损耗、加减速补偿和厂家控制模型。

> [!tip] 记忆口诀
> 卷径越大，同样张力要更大转矩。

# 13 安全功能

> [!info] 核心结论
> 工程型变频器常集成或支持安全功能，如STO、安全停止、安全限速等，需与安全系统配合使用。

## ① 分类/公式/规则

> [!note] 🟦 常见安全功能
> STO用于安全转矩关闭；  
> SS1用于安全停止1；  
> SLS用于安全限速；  
> 具体功能和等级需按变频器型号、安全模块和项目规范校核。

> [!warning] 🟧 安全链路公式
> **安全功能有效 = 安全输入 + 安全逻辑 + 变频器安全功能 + 执行机构验证**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **STO** | 安全转矩关闭 |
| **SS1** | 受控减速后STO |
| **SLS** | 安全限速 |
| **SDI** | 安全方向 |
| **安全I/O** | 接安全PLC或安全继电器 |
| **安全认证** | 需按项目要求校核 |

## ③ 计算/选型示例

- 已知：  
  安全门打开后，设备要求先减速停车，再切断转矩。
- 公式：  
  **安全停机 = 受控减速 + STO**
- 计算：  
  直接STO可能导致自由停车，不一定满足工艺或安全要求。
- 结论：  
  可评估 **SS1 + STO** 安全方案，具体等级和接线需按安全规范校核。

> [!tip] 记忆口诀
> STO管无转矩，SS1管先停再断。

# 14 柜体与EMC设计

> [!info] 核心结论
> 工程型变频器系统往往功率大、线缆多、干扰强，柜体散热、接地、屏蔽和布线必须重点设计。

## ① 分类/公式/规则

> [!note] 🟦 设计重点
> 变频器柜要关注进出风、散热空间、动力线与信号线分离、屏蔽接地、制动电阻散热和通讯抗干扰。

> [!warning] 🟧 稳定运行公式
> **稳定系统 = 散热足够 + 接地可靠 + 动力信号分离 + 屏蔽规范 + EMC器件匹配**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **散热** | 风道、风扇、柜内温升 |
| **接地** | PE可靠连接 |
| **屏蔽** | 电机线、编码器线、通讯线 |
| **布线** | 动力线与信号线分槽 |
| **电抗器** | 输入/输出按需配置 |
| **滤波器** | EMC要求高时校核 |

## ③ 计算/选型示例

- 已知：  
  变频器运行时通讯偶发掉线。
- 公式：  
  **通讯异常 = 变频干扰 + 屏蔽接地差 + 动力线距离近**
- 计算：  
  通讯线若与电机电缆同槽长距离并行，易受干扰。
- 结论：  
  应分开布线、规范屏蔽接地，必要时增加滤波和隔离措施。

> [!tip] 记忆口诀
> 工程变频功率大，散热接地和屏蔽别省。

# 15 调试流程

> [!info] 核心结论
> 工程型变频器调试要先空载、再带载；先本地、再通讯；先低速、再高速。

## ① 分类/公式/规则

> [!note] 🟦 调试步骤
> ① 核对主回路  
> ② 输入电机铭牌参数  
> ③ 设置控制方式  
> ④ 做电机自学习  
> ⑤ 空载低速运行  
> ⑥ 带载测试  
> ⑦ 通讯联调  
> ⑧ 保护和安全功能验证

> [!warning] 🟧 调试原则公式
> **可靠调试 = 参数正确 + 低速验证 + 逐步加载 + 故障记录**

## ② 参考表

| 阶段 | 重点 |
|---|---|
| **上电前** | 接线、绝缘、接地 |
| **首次上电** | 电源电压、故障码 |
| **参数设置** | 电机参数、控制源 |
| **空载测试** | 方向、频率、声音 |
| **带载测试** | 电流、转矩、温升 |
| **联机测试** | PLC通讯、保护、安全 |

## ③ 计算/选型示例

- 已知：  
  电机空载运行正常，带载后过流。
- 公式：  
  **带载过流 = 负载过重 + 加速太快 + 转矩参数不足 + 机械卡滞**
- 计算：  
  空载正常说明主回路大概率正常，带载过流更可能与负载或参数有关。
- 结论：  
  检查机械负载、加速时间、转矩提升、电机参数和控制方式。

> [!tip] 记忆口诀
> 空载看方向，带载看电流；通讯看状态，安全看联锁。

# 16 常见故障排查

> [!info] 核心结论
> 工程型变频器故障排查，要结合故障码、运行工况、负载状态、通讯状态和安全链路一起判断。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看故障代码，再看发生工况；  
> 启动时报错查加速和负载；  
> 减速时报错查制动和母线；  
> 通讯报错查协议和映射。

> [!warning] 🟧 排错公式
> **故障原因 = 电源 + 电机 + 负载 + 参数 + 通讯 + 安全 + 环境**

## ② 参考表

| 故障 | 常见原因 |
|---|---|
| **过流OC** | 加速快、负载重、短路、参数错 |
| **过压OV** | 减速快、制动不足、回馈能量大 |
| **欠压UV** | 电源跌落、输入缺相 |
| **过载OL** | 负载过重、电机参数错 |
| **过热OH** | 散热差、风扇坏、柜温高 |
| **通讯故障** | 地址、协议、线缆、映射错误 |
| **编码器故障** | 接线、PG卡、屏蔽、方向错误 |

## ③ 计算/选型示例

- 已知：  
  设备减速时频繁报过压OV。
- 公式：  
  **减速过压 = 再生能量 + 减速时间过短 + 制动能力不足**
- 计算：  
  负载惯量越大、减速越快，回灌能量越多。
- 结论：  
  可延长减速时间，或校核制动电阻、制动单元、回馈单元配置。

> [!tip] 记忆口诀
> 启动过流查负载，减速过压查制动；通讯掉线查线缆，闭环报警查编码器。

# 17 工程型变频器快速选型表

> [!info] 核心结论
> 工程型变频器选型必须从系统工艺出发，先定负载和控制方式，再定容量、制动、通讯和安全。

## ① 分类/公式/规则

> [!note] 🟦 十步选型法
> ① 看电源电压  
> ② 看电机功率和电流  
> ③ 看负载类型  
> ④ 看控制精度  
> ⑤ 看是否闭环  
> ⑥ 看制动/回馈需求  
> ⑦ 看通讯协议  
> ⑧ 看I/O扩展  
> ⑨ 看安全功能  
> ⑩ 看柜体散热和EMC

> [!warning] 🟧 最终选型公式
> **推荐型号 = 电压等级 + 输出电流 + 过载等级 + 控制方式 + 制动方案 + 通讯选件 + 安全功能**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 重载启动 | 过载能力、矢量控制 |
| 低速大转矩 | 闭环矢量、编码器 |
| 快速停车 | 制动电阻/回馈单元 |
| 起重提升 | 抱闸逻辑、安全功能 |
| 卷绕张力 | 转矩控制、卷径计算 |
| 多电机同步 | 主从控制、总线通讯 |
| PLC集成 | 控制字、状态字、故障码 |

## ③ 计算/选型示例

- 已知：  
  380V系统，电机额定电流 **30A**，起重负载，要求闭环控制、快速制动、PLC总线控制。
- 公式：  
  **型号需求 = 380V + 输出电流≥30A×余量 + 闭环矢量 + 制动方案 + 总线通讯**
- 计算：  
  起重属于高风险重载应用，不能只按普通功率等级选型。
- 结论：  
  应选择支持 **重载过载能力、编码器闭环、制动/回馈、安全功能和PLC通讯** 的工程型变频器，并按厂家手册校核。

> [!tip] 选型口诀
> 先看工艺，再看负载；低速要矢量，制动要校核；多轴看通讯，安全别漏掉。
