# 01 高压变频器总览

> [!info] 核心结论
> 高压变频器用于控制 **3kV、6kV、10kV 等中高压电机** 的启动、调速、节能和保护，常用于大功率风机、水泵、压缩机、皮带机等设备。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 低压变频器常用于小中功率电机；  
> 高压变频器用于大功率高压电机，重点关注 **绝缘、散热、谐波、旁路、保护和安全操作**。

> [!warning] 🟧 通用选型公式
> **高压变频器选型 = 电压等级 + 电机功率 + 额定电流 + 负载类型 + 调速范围 + 旁路方案 + 冷却方式**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **常见电压** | 3kV、6kV、10kV等 |
| **控制对象** | 高压异步电机、部分同步电机 |
| **典型负载** | 风机、水泵、压缩机、皮带机 |
| **核心功能** | 调速、软启动、节能、保护 |
| **常见结构** | 移相变压器 + 功率单元 + 控制系统 |
| **重点校核** | 电压、电流、功率、散热、谐波、旁路 |

## ③ 计算/选型示例

- 已知：  
  现场高压电机为 **10kV、1000kW**，用于循环水泵调速。
- 公式：  
  **高压变频器容量 ≥ 电机容量 × 负载余量**
- 计算：  
  至少选择满足10kV电压等级、1000kW容量和水泵负载特性的高压变频器。
- 结论：  
  可选 **10kV等级高压变频器**，具体容量、电流、旁路柜和冷却方式需按厂家手册校核。

> [!tip] 记忆口诀
> 高压变频看三件：电压等级、电机功率、旁路保护。

# 02 底层原理

> [!info] 核心结论
> 高压变频器的本质，是把高压交流电转换成可控频率、可控电压的高压交流电。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 高压变频器通常先通过移相变压器降压并分相供电，再由多个功率单元整流、逆变，最后串联叠加输出高压可变频电源。

> [!warning] 🟧 能量转换逻辑
> **高压输入 → 移相变压器 → 功率单元整流 → 直流母线 → 单元逆变 → 串联叠加输出**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **移相变压器** | 隔离、降压、移相、降低谐波 |
| **功率单元** | 完成整流和逆变 |
| **直流母线** | 储能和滤波 |
| **控制系统** | 计算频率、电压和保护逻辑 |
| **输出侧** | 向高压电机提供可调电源 |
| **旁路系统** | 变频故障时切换工频运行 |

## ③ 计算/选型示例

- 已知：  
  电机需要从0Hz平滑升至50Hz。
- 公式：  
  **电机转速 ∝ 输出频率**
- 计算：  
  变频器逐步提高输出频率，电机转速随之上升。
- 结论：  
  高压变频器可实现高压电机平滑启动，降低直接启动冲击。

> [!tip] 记忆口诀
> 先变成直流，再逆变调频；多单元叠加，输出高压交流。

# 03 高压变频器结构

> [!info] 核心结论
> 高压变频器通常由 **变压器柜、功率柜、控制柜、旁路柜、冷却系统** 组成。

## ① 分类/公式/规则

> [!note] 🟦 柜体组成
> 高压变频器不是一个小型驱动器，而是一套完整柜体系统，需要考虑高压隔离、散热、维护空间和安全联锁。

> [!warning] 🟧 系统组成公式
> **高压变频系统 = 变压器柜 + 功率单元柜 + 控制柜 + 旁路柜 + 冷却系统**

## ② 参考表

| 柜体/模块 | 主要作用 |
|---|---|
| **变压器柜** | 输入隔离、移相、供电 |
| **功率柜** | 功率单元整流逆变 |
| **控制柜** | 控制、保护、通讯、人机界面 |
| **旁路柜** | 变频/工频切换 |
| **冷却系统** | 风冷或水冷散热 |
| **高压开关柜** | 上级供电和保护 |

## ③ 计算/选型示例

- 已知：  
  用户要求变频器故障时电机不能长期停机。
- 公式：  
  **连续运行需求 = 变频运行 + 工频旁路**
- 计算：  
  当变频器故障时，可通过旁路柜切换到工频运行。
- 结论：  
  应配置合适的 **工频旁路方案**，切换逻辑需按厂家手册和现场规范校核。

> [!tip] 记忆口诀
> 高压变频一整套，变压、功率、控制、旁路不能少。

# 04 常见拓扑类型

> [!info] 核心结论
> 工业现场常见高压变频器多采用 **功率单元串联多电平结构**，输出波形更接近正弦。

## ① 分类/公式/规则

> [!note] 🟦 拓扑理解
> 高压器件直接承受全电压难度大，因此常把多个低压功率单元串联起来，逐级叠加形成高压输出。

> [!warning] 🟧 多电平逻辑
> **单元电压叠加 = 相电压输出**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **功率单元串联型** | 工业现场常见，模块化强 |
| **多电平结构** | 输出波形好，电机冲击小 |
| **电流源型** | 特定场景使用，需按厂家方案 |
| **中性点钳位型** | 多电平拓扑之一 |
| **级联H桥型** | 常见单元串联方案 |
| **水冷大功率型** | 高功率密度场景 |

## ③ 计算/选型示例

- 已知：  
  某相由多个功率单元串联，每个单元输出一定电压。
- 公式：  
  **相电压 ≈ 单元电压 × 串联单元数量**
- 计算：  
  单元数量越多，可叠加得到更高输出电压，同时输出波形更细。
- 结论：  
  高压变频器通过多单元串联实现高压输出，具体单元数量需按厂家设计校核。

> [!tip] 记忆口诀
> 一个单元电压低，多个单元串起来就是高压。

# 05 频率与电机转速

> [!info] 核心结论
> 高压电机转速主要由变频器输出频率决定，频率越高，同步转速越高。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **n = 120 × f ÷ P**

> [!note] 🟦 参数说明
> **n**：同步转速，rpm  
> **f**：输出频率，Hz  
> **P**：电机极数  
>  
> 异步电机实际转速会略低于同步转速，需考虑转差率。

## ② 参考表

| 电机极数 | 50Hz同步转速 |
|---|---|
| **2极** | 3000rpm |
| **4极** | 1500rpm |
| **6极** | 1000rpm |
| **8极** | 750rpm |

## ③ 计算/选型示例

- 已知：  
  高压电机为 **6极**，变频器输出频率为 **40Hz**。
- 公式：  
  **n = 120 × f ÷ P**
- 计算：  
  n = 120 × 40 ÷ 6 = **800rpm**
- 结论：  
  6极电机在40Hz时，同步转速约 **800rpm**。

> [!tip] 记忆口诀
> 调频就是调速，极数决定基准转速。

# 06 风机水泵节能

> [!info] 核心结论
> 高压变频器在风机、水泵负载中常用于节能，降低转速可显著降低功率消耗。

## ① 分类/公式/规则

> [!note] 🟦 节能原理
> 风机、水泵属于平方转矩类负载，流量与转速近似成正比，功率与转速的三次方近似相关。

> [!warning] 🟧 风机水泵相似定律
> **Q ∝ n**  
> **H ∝ n²**  
> **P ∝ n³**

## ② 参考表

| 参数 | 与转速关系 |
|---|---|
| **流量Q** | 约与转速成正比 |
| **压力/扬程H** | 约与转速平方成正比 |
| **轴功率P** | 约与转速三次方成正比 |
| **节流调节** | 能耗较高 |
| **变频调节** | 节能效果明显 |
| **低速运行** | 需校核工艺下限 |

## ③ 计算/选型示例

- 已知：  
  风机转速降到额定的 **80%**。
- 公式：  
  **功率比例 ≈ 0.8³**
- 计算：  
  功率比例 ≈ 0.512 = **51.2%**
- 结论：  
  理论轴功率约降至原来的 **51.2%**，实际节能需结合系统效率和工况校核。

> [!tip] 记忆口诀
> 风机水泵降一点速，功率按三次方往下降。

# 07 容量与电流选型

> [!info] 核心结论
> 高压变频器选型必须以 **电机额定电压、额定功率、额定电流、负载特性** 为核心。

## ① 分类/公式/规则

> [!note] 🟦 选型重点
> 高压变频器不能只看kW，还要看电机额定电流、过载能力、应用场景和冷却条件。

> [!warning] 🟧 电流选型公式
> **变频器额定输出电流 ≥ 电机额定电流 × 负载余量系数**

## ② 参考表

| 参数 | 选型意义 |
|---|---|
| **电压等级** | 必须与电机一致 |
| **电机功率** | 初步确定容量 |
| **额定电流** | 核心校核参数 |
| **负载类型** | 决定轻载/重载配置 |
| **过载能力** | 启动和冲击负载校核 |
| **环境温度** | 影响降额使用 |

## ③ 计算/选型示例

- 已知：  
  高压电机额定电流 **120A**，负载较重，建议按 **1.15倍** 校核。
- 公式：  
  **所需输出电流 ≥ 120 × 1.15**
- 计算：  
  所需输出电流 ≥ **138A**
- 结论：  
  应选择输出额定电流满足 **138A以上** 的高压变频器，最终按厂家重载规格校核。

> [!tip] 记忆口诀
> 高压选型别只看功率，额定电流才是硬指标。

# 08 旁路柜与切换逻辑

> [!info] 核心结论
> 旁路柜用于在变频器检修或故障时，让电机切换到工频运行，提高系统连续性。

## ① 分类/公式/规则

> [!note] 🟦 旁路作用
> 高压变频器常用于关键设备。  
> 如果变频器故障后设备必须继续运行，就需要设计工频旁路方案。

> [!warning] 🟧 旁路逻辑公式
> **运行连续性 = 变频运行 + 故障切换 + 工频旁路 + 电气联锁**

## ② 参考表

| 旁路类型 | 说明 |
|---|---|
| **手动旁路** | 人工操作切换，结构简单 |
| **自动旁路** | 故障后自动切换，逻辑复杂 |
| **一拖一旁路** | 一台变频器对应一台电机 |
| **一拖多方案** | 一台变频器服务多台电机，需复杂联锁 |
| **工频接触器** | 工频运行回路 |
| **变频接触器** | 变频运行回路 |

## ③ 计算/选型示例

- 已知：  
  高压水泵属于关键工艺设备，不允许因变频器故障长期停运。
- 公式：  
  **可靠供水 = 变频节能 + 工频备用**
- 计算：  
  变频器正常时调速节能，故障或检修时通过旁路工频运行。
- 结论：  
  建议配置 **旁路柜**，并严格设计机械和电气联锁，避免变频与工频同时接入。

> [!tip] 记忆口诀
> 变频管节能，旁路保连续；联锁不到位，高压很危险。

# 09 谐波与电网影响

> [!info] 核心结论
> 高压变频器会对电网产生谐波影响，需要通过移相变压器、多脉冲整流、滤波等方式控制。

## ① 分类/公式/规则

> [!note] 🟦 谐波来源
> 整流环节会产生非正弦电流，引起谐波。  
> 高压变频器常通过移相变压器形成多脉冲整流，降低输入谐波。

> [!warning] 🟧 谐波治理公式
> **谐波控制 = 移相整流 + 多脉冲结构 + 滤波措施 + 电网校核**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **输入谐波** | 影响电网电能质量 |
| **移相变压器** | 降低谐波的重要部件 |
| **多脉冲整流** | 脉冲数越高，谐波通常越低 |
| **滤波器** | 进一步改善电能质量 |
| **功率因数** | 需按工况校核 |
| **电网容量** | 弱电网需重点评估 |

## ③ 计算/选型示例

- 已知：  
  高压变频器接入电网后，电能质量要求较高。
- 公式：  
  **电能质量 = 谐波含量 + 功率因数 + 电压波动**
- 计算：  
  单纯接入变频器可能无法满足要求，需要校核谐波指标。
- 结论：  
  应按项目规范进行谐波评估，必要时配置滤波或优化整流方案。

> [!tip] 记忆口诀
> 高压变频接电网，谐波评估不能忘。

# 10 高压电机与绝缘

> [!info] 核心结论
> 高压变频器输出含PWM波形，必须关注电机绝缘、du/dt、电缆长度和轴承电流风险。

## ① 分类/公式/规则

> [!note] 🟦 绝缘风险
> 变频输出不是纯正弦波，快速开关可能带来电压应力。  
> 高压电机长期变频运行时，需要确认电机是否适合变频供电。

> [!warning] 🟧 电机适配公式
> **电机可用 = 绝缘等级满足 + 温升满足 + 冷却满足 + 电缆距离满足**

## ② 参考表

| 项目 | 风险点 |
|---|---|
| **电机绝缘** | 承受PWM电压应力 |
| **du/dt** | 电压变化率过高可能损伤绝缘 |
| **电缆长度** | 长电缆会加重反射过压 |
| **轴承电流** | 可能造成轴承电蚀 |
| **低速散热** | 自冷电机低速散热变差 |
| **电机改造** | 老电机需谨慎校核 |

## ③ 计算/选型示例

- 已知：  
  老旧高压电机计划加装高压变频器。
- 公式：  
  **改造风险 = 旧电机绝缘 + PWM应力 + 冷却条件**
- 计算：  
  老电机绝缘状态不明，长期变频运行存在风险。
- 结论：  
  改造前应检测电机绝缘，必要时做绝缘升级、加输出滤波或更换变频专用电机。

> [!tip] 记忆口诀
> 高压变频不只看柜子，还要看电机能不能扛。

# 11 冷却与散热

> [!info] 核心结论
> 高压变频器功率大、发热高，冷却方式和现场环境直接影响可靠性。

## ① 分类/公式/规则

> [!note] 🟦 常见冷却
> 常见冷却方式有 **强迫风冷、水冷、空水冷** 等。  
> 高温、粉尘、潮湿环境会影响散热和绝缘。

> [!warning] 🟧 散热可靠公式
> **可靠散热 = 冷却方式匹配 + 风道畅通 + 环境温度合格 + 滤网维护**

## ② 参考表

| 冷却方式 | 特点 |
|---|---|
| **强迫风冷** | 结构常见，维护方便 |
| **水冷** | 散热能力强，系统复杂 |
| **空水冷** | 适合特定环境 |
| **柜内风道** | 影响热量排出 |
| **滤网** | 防尘但需定期清理 |
| **空调房** | 提高环境稳定性 |

## ③ 计算/选型示例

- 已知：  
  变频器运行一段时间后报过热。
- 公式：  
  **过热风险 = 环境温度高 + 风道堵塞 + 滤网脏 + 风机异常**
- 计算：  
  滤网堵塞会降低风量，功率单元温度升高。
- 结论：  
  应检查风机、滤网、风道、柜内温度和空调系统。

> [!tip] 记忆口诀
> 高压变频怕高温，风道滤网要勤看。

# 12 控制与PLC通讯

> [!info] 核心结论
> 高压变频器可通过硬接线或通讯与PLC/DCS集成，实现启停、频率给定、状态反馈和故障诊断。

## ① 分类/公式/规则

> [!note] 🟦 常见信号
> 常见控制包括启停、急停、频率给定、运行状态、故障状态、旁路状态、就地/远程、频率反馈、电流反馈等。

> [!warning] 🟧 控制匹配公式
> **远程控制正常 = 控制源设置 + 给定源设置 + 联锁条件 + 通讯/端子正常**

## ② 参考表

| 信号/数据 | 作用 |
|---|---|
| **运行命令** | 启动/停止 |
| **频率给定** | 设定转速 |
| **运行反馈** | 变频器运行状态 |
| **故障反馈** | 报警或跳闸状态 |
| **旁路状态** | 工频/变频运行判断 |
| **电流/频率反馈** | 监控负载运行情况 |

## ③ 计算/选型示例

- 已知：  
  DCS输出4-20mA给定频率，设定范围0～50Hz，当前电流为 **12mA**。
- 公式：  
  **频率 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  频率 =（12 - 4）÷ 16 × 50 = **25Hz**
- 结论：  
  变频器应接收约 **25Hz** 频率给定，前提是量程和给定源设置正确。

> [!tip] 记忆口诀
> 远程能不能控，先看控制源和给定源。

# 13 保护功能

> [!info] 核心结论
> 高压变频器保护不仅保护自身，还要保护高压电机、变压器、旁路系统和工艺设备。

## ① 分类/公式/规则

> [!note] 🟦 常见保护
> 高压变频器通常具备过流、过压、欠压、过载、过热、缺相、接地、单元故障、变压器过温、门禁联锁等保护。

> [!warning] 🟧 保护逻辑公式
> **保护动作 = 故障检测 + 联锁判断 + 报警/跳闸 + 记录故障**

## ② 参考表

| 保护类型 | 可能原因 |
|---|---|
| **过流** | 负载重、加速快、短路 |
| **过压** | 减速快、能量回馈 |
| **欠压** | 电网波动、输入异常 |
| **过热** | 散热差、风机故障 |
| **单元故障** | 功率单元异常 |
| **接地故障** | 电机或电缆绝缘问题 |
| **门禁联锁** | 柜门打开或联锁异常 |

## ③ 计算/选型示例

- 已知：  
  启动过程中频繁报过流。
- 公式：  
  **启动过流 = 加速时间短 + 负载过重 + 参数不合适 + 电机异常**
- 计算：  
  可先检查负载是否卡滞，再适当延长加速时间。
- 结论：  
  排查机械负载、电机绝缘、电缆、加速时间和变频器参数。

> [!tip] 记忆口诀
> 过流查负载，过压查减速，过热查散热，接地查绝缘。

# 14 高压安全操作

> [!info] 核心结论
> 高压变频器涉及高电压和大能量，操作维护必须执行停电、验电、放电、接地和挂牌流程。

## ① 分类/公式/规则

> [!note] 🟦 安全重点
> 高压变频器停机后，直流母线和电容仍可能残留电压。  
> 维护前必须确认完全放电，并按电气安全规程操作。

> [!warning] 🟧 安全操作公式
> **安全维护 = 停电 + 验电 + 放电 + 接地 + 挂牌 + 专人监护**

## ② 参考表

| 操作项 | 重点 |
|---|---|
| **停电** | 断开上级电源 |
| **验电** | 确认无电压 |
| **放电** | 等待母线电容放电 |
| **接地** | 挂接地线 |
| **挂牌** | 防止误送电 |
| **门禁联锁** | 不得随意短接 |

## ③ 计算/选型示例

- 已知：  
  变频器刚停机，维护人员准备打开功率柜。
- 公式：  
  **开柜条件 = 电源断开 + 母线电压安全 + 联锁允许**
- 计算：  
  停机不等于内部无电，电容可能仍有残压。
- 结论：  
  必须按厂家规定等待放电完成，并由合格人员确认后再开柜。

> [!tip] 记忆口诀
> 高压设备不怕慢，就怕没验电；停机不等于无电。

# 15 调试流程

> [!info] 核心结论
> 高压变频器调试要从低风险步骤开始，先检查系统，再空载，再带载，最后联动运行。

## ① 分类/公式/规则

> [!note] 🟦 调试步骤
> ① 核对一次接线  
> ② 检查二次控制和联锁  
> ③ 检查绝缘和接地  
> ④ 上电自检  
> ⑤ 参数设置  
> ⑥ 空载/低频试运行  
> ⑦ 带载调试  
> ⑧ PLC/DCS联动  
> ⑨ 保护和旁路测试

> [!warning] 🟧 调试原则公式
> **可靠调试 = 接线正确 + 联锁有效 + 参数正确 + 逐步升频 + 记录数据**

## ② 参考表

| 阶段 | 重点 |
|---|---|
| **上电前** | 绝缘、接地、端子、柜门 |
| **上电自检** | 控制电源、风机、单元状态 |
| **低频运行** | 电机方向、振动、声音 |
| **升频运行** | 电流、温度、负载 |
| **联动测试** | PLC/DCS启停和反馈 |
| **旁路测试** | 工频/变频切换逻辑 |

## ③ 计算/选型示例

- 已知：  
  电机低频运行方向反了。
- 公式：  
  **反向原因 = 输出相序与电机方向不一致**
- 计算：  
  可通过参数反向设置或调整电机相序处理，需按厂家调试规程执行。
- 结论：  
  调试初期应先低频点动确认方向，避免高速反转造成工艺风险。

> [!tip] 记忆口诀
> 先低频看方向，再升频看电流；先单机稳定，再系统联动。

# 16 常见故障排查

> [!info] 核心结论
> 高压变频器故障排查要结合故障代码、发生工况、输入电网、电机状态、功率单元和冷却系统判断。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看故障码和历史记录；  
> 再看故障发生在启动、运行、减速还是切换旁路；  
> 最后逐项检查电网、负载、电机、单元和散热。

> [!warning] 🟧 排错公式
> **高压变频异常 = 电网问题 + 负载问题 + 电机问题 + 单元问题 + 散热问题 + 参数问题**

## ② 参考表

| 故障 | 常见原因 |
|---|---|
| **启动失败** | 联锁未满足、控制源错误 |
| **过流跳闸** | 负载重、加速快、电机异常 |
| **过压跳闸** | 减速快、负载回馈 |
| **单元故障** | 功率单元异常、通讯异常 |
| **变压器过温** | 散热差、负载高 |
| **输出缺相** | 单元、接线、电机故障 |
| **旁路失败** | 接触器联锁或控制逻辑异常 |

## ③ 计算/选型示例

- 已知：  
  高压变频器运行一段时间后报过热。
- 公式：  
  **过热 = 损耗发热 - 冷却能力**
- 计算：  
  冷却风机故障、滤网堵塞或环境温度高都会降低冷却能力。
- 结论：  
  应检查风机、滤网、风道、环境温度、柜体散热和负载电流。

> [!tip] 记忆口诀
> 启动查联锁，运行查负载，过热查风冷，单元报警查模块。

# 17 高压变频器快速选型表

> [!info] 核心结论

> ④ 看调速范围
