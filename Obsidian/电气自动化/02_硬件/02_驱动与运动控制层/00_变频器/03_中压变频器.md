# 01 中压变频器总览

> [!info] 核心结论
> 中压变频器用于控制中等电压等级电机的启动、调速、节能和保护，常见于大功率风机、水泵、压缩机、输送机和矿山设备。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 中压变频器介于低压变频器和高压变频器之间，既要关注变频调速性能，也要关注电压等级、绝缘、谐波、散热和系统保护。

> [!warning] 🟧 通用选型公式
> **中压变频器选型 = 电压等级 + 电机功率 + 额定电流 + 负载类型 + 调速范围 + 冷却方式 + 通讯/旁路需求**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **设备类型** | 中压交流变频调速装置 |
| **常见电压** | 690V、1140V、2.3kV、3.3kV、4.16kV等，需按项目/厂家定义校核 |
| **控制对象** | 中压异步电机、部分同步电机 |
| **典型负载** | 风机、水泵、压缩机、皮带机、破碎机 |
| **核心功能** | 调速、软启动、节能、保护、通讯 |
| **重点校核** | 电流、过载、绝缘、散热、谐波、旁路 |

## ③ 计算/选型示例

- 已知：  
  一台中压电机额定电压 **1140V**，功率 **400kW**，用于矿用皮带输送机。
- 公式：  
  **变频器额定输出电流 ≥ 电机额定电流 × 负载余量系数**
- 计算：  
  皮带机属于较重负载，应重点校核启动转矩、过载能力和制动需求。
- 结论：  
  应选择匹配 **1140V电压等级**、具备重载能力的中压变频器，具体电流和保护功能需按厂家手册校核。

> [!tip] 记忆口诀
> 中压变频先看电压，再看电流；轻载看节能，重载看转矩。

# 02 底层原理

> [!info] 核心结论
> 中压变频器的本质，是把固定频率电源变成频率和电压可调的交流电源。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 中压变频器通常通过整流、直流母线、逆变三个核心环节，把输入电源转换成可调频率输出。

> [!warning] 🟧 能量转换逻辑
> **AC输入 → 整流成DC → 直流母线储能 → 逆变成可调AC → 驱动电机**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **整流单元** | 把交流电变成直流电 |
| **直流母线** | 储能、滤波、能量交换 |
| **逆变单元** | 输出可调频率和电压 |
| **功率器件** | IGBT、IGCT等需按型号确认 |
| **控制系统** | 运行算法、保护、通讯 |
| **冷却系统** | 风冷、水冷或空水冷 |

## ③ 计算/选型示例

- 已知：  
  电机需要从0Hz平滑升至50Hz。
- 公式：  
  **电机转速 ∝ 输出频率**
- 计算：  
  变频器逐步提升输出频率，电机转速同步提升，启动冲击明显小于直接工频启动。
- 结论：  
  中压变频器可实现大功率电机平滑启动和节能调速。

> [!tip] 记忆口诀
> 先整流，再逆变；频率一调，电机速度就变。

# 03 中压 vs 低压 vs 高压

> [!info] 核心结论
> 中压变频器适合低压容量不足、高压系统又不必要或不经济的中大功率场景。

## ① 分类/公式/规则

> [!note] 🟦 快速区分
> 低压变频器常见于普通设备和中小功率电机；  
> 中压变频器适合中大功率设备；  
> 高压变频器多用于6kV、10kV等大型高压电机系统。

> [!warning] 🟧 选型判断公式
> **电压等级由电机决定，容量等级由电流和负载决定**

## ② 参考表

| 类型 | 常见特点 |
|---|---|
| **低压变频器** | 380V、400V、690V等，应用广 |
| **中压变频器** | 1140V、2.3kV、3.3kV等场景较多，需按项目定义 |
| **高压变频器** | 3kV、6kV、10kV等大型电机 |
| **中压优势** | 电流比低压小，系统容量更容易提升 |
| **中压难点** | 绝缘、安全、散热、谐波要求更高 |
| **最终依据** | 电机铭牌和系统电压等级 |

## ③ 计算/选型示例

- 已知：  
  同样功率下，电压提高，电流会降低。
- 公式：  
  **P ≈ √3 × U × I × cosφ × η**
- 计算：  
  功率P相近时，电压U越高，电流I越小。
- 结论：  
  中压系统可降低大功率场景下的电流压力，但绝缘和安全要求更高。

> [!tip] 记忆口诀
> 功率一定，电压升高，电流就降；电流降了，绝缘要求上来了。

# 04 典型应用场景

> [!info] 核心结论
> 中压变频器常用于功率较大、运行时间长、调速节能明显或启动冲击大的设备。

## ① 分类/公式/规则

> [!note] 🟦 场景主线
> 风机水泵看节能；  
> 皮带输送看启动转矩；  
> 压缩机看工艺稳定；  
> 破碎机看重载冲击；  
> 矿山设备看可靠性和防护。

> [!warning] 🟧 应用判断公式
> **适合中压变频 = 中大功率电机 + 需要调速/软启动/节能/保护**

## ② 参考表

| 场景 | 关注点 |
|---|---|
| **风机** | 节能、调速范围、谐波 |
| **水泵** | 恒压、PID、旁路 |
| **压缩机** | 稳定运行、工艺联锁 |
| **皮带机** | 软启动、低速大转矩 |
| **破碎机** | 过载能力、冲击负载 |
| **矿山设备** | 防护等级、散热、可靠性 |

## ③ 计算/选型示例

- 已知：  
  一台风机长期通过挡板调节风量。
- 公式：  
  **节能潜力 = 调速替代节流**
- 计算：  
  挡板调节会造成较大能量损失，变频调速可降低电机转速。
- 结论：  
  风机类负载适合评估中压变频节能改造。

> [!tip] 记忆口诀
> 风机水泵看节能，皮带破碎看转矩。

# 05 频率与转速

> [!info] 核心结论
> 中压变频器通过改变输出频率来调节电机转速。

## ① 分类/公式/规则

> [!warning] 🟧 同步转速公式
> **n = 120 × f ÷ P**

> [!note] 🟦 参数说明
> **n**：同步转速，rpm  
> **f**：输出频率，Hz  
> **P**：电机极数  
>  
> 异步电机实际转速会略低于同步转速，需考虑转差率。

## ② 参考表

| 电机极数 | 50Hz同步转速 |
|---|---|
| **2极** | 3000rpm |
| **4极** | 1500rpm |
| **6极** | 1000rpm |
| **8极** | 750rpm |

## ③ 计算/选型示例

- 已知：  
  电机为 **4极**，变频器输出频率为 **40Hz**。
- 公式：  
  **n = 120 × 40 ÷ 4**
- 计算：  
  n = **1200rpm**
- 结论：  
  4极电机在40Hz时，同步转速约为 **1200rpm**，实际转速略低。

> [!tip] 记忆口诀
> 频率管速度，极数定基准；4极50Hz，大约1500转。

# 06 风机水泵节能

> [!info] 核心结论
> 中压变频器在风机、水泵类负载中节能效果明显，因为功率近似随转速三次方变化。

## ① 分类/公式/规则

> [!note] 🟦 节能原理
> 风机水泵属于平方转矩负载。  
> 转速降低后，流量、压力和功率都会下降，其中功率下降最明显。

> [!warning] 🟧 相似定律
> **Q ∝ n**  
> **H ∝ n²**  
> **P ∝ n³**

## ② 参考表

| 参数 | 与转速关系 |
|---|---|
| **流量Q** | 约与转速成正比 |
| **压力/扬程H** | 约与转速平方成正比 |
| **轴功率P** | 约与转速三次方成正比 |
| **节流调节** | 能耗较高 |
| **变频调节** | 节能明显 |
| **工艺下限** | 需按系统要求校核 |

## ③ 计算/选型示例

- 已知：  
  水泵转速降至额定的 **90%**。
- 公式：  
  **功率比例 ≈ 0.9³**
- 计算：  
  功率比例 ≈ **72.9%**
- 结论：  
  理论轴功率约降至额定的 **72.9%**，实际节能需结合系统效率和工况校核。

> [!tip] 记忆口诀
> 转速降一点，功率降一截；风机水泵最吃变频节能。

# 07 容量与电流选型

> [!info] 核心结论
> 中压变频器选型不能只看功率，必须校核电机额定电流和负载余量。

## ① 分类/公式/规则

> [!note] 🟦 选型主线
> 电压等级必须与电机一致；  
> 输出电流必须覆盖电机额定电流；  
> 重载设备还要校核短时过载能力。

> [!warning] 🟧 电流选型公式
> **变频器额定输出电流 ≥ 电机额定电流 × 负载余量系数**

## ② 参考表

| 参数 | 选型意义 |
|---|---|
| **电压等级** | 必须与电机匹配 |
| **电机功率** | 初步确定容量 |
| **额定电流** | 核心校核参数 |
| **负载类型** | 决定轻载/重载选型 |
| **过载能力** | 启动、冲击、堵转校核 |
| **环境温度** | 影响降额使用 |

## ③ 计算/选型示例

- 已知：  
  电机额定电流 **180A**，皮带机重载启动，建议留 **1.2倍** 余量。
- 公式：  
  **所需输出电流 ≥ 180 × 1.2**
- 计算：  
  所需输出电流 ≥ **216A**
- 结论：  
  应选择输出额定电流不小于 **216A** 的中压变频器，最终按厂家重载规格校核。

> [!tip] 记忆口诀
> 功率看型号，电流定容量；重载不留余量，启动容易报警。

# 08 控制方式

> [!info] 核心结论
> 中压变频器控制方式应按负载要求选择，普通调速用V/F，重载和精密控制优先矢量控制。

## ① 分类/公式/规则

> [!note] 🟦 控制主线
> V/F控制简单可靠，适合风机水泵；  
> 无PG矢量适合一般重载；  
> 有PG矢量适合低速大转矩、高动态响应和闭环控制。

> [!warning] 🟧 控制选择公式
> **控制要求越高 → 越倾向矢量/闭环控制**

## ② 参考表

| 控制方式 | 适合场景 |
|---|---|
| **V/F控制** | 风机、水泵、普通调速 |
| **无PG矢量** | 输送、搅拌、一般重载 |
| **有PG矢量** | 提升、张力、高精度速度 |
| **转矩控制** | 卷绕、测试台 |
| **PID控制** | 恒压、恒流、恒温 |
| **多段速控制** | 输送线、工艺段速 |

## ③ 计算/选型示例

- 已知：  
  皮带输送机需要低速重载启动。
- 公式：  
  **低速重载 → 矢量控制优先**
- 计算：  
  V/F低频转矩可能不足，矢量控制可提升低速转矩表现。
- 结论：  
  应选择支持 **无PG矢量或有PG矢量** 的中压变频器。

> [!tip] 记忆口诀
> 风泵用V/F，重载用矢量；低速要有力，参数要调准。

# 09 旁路与工频切换

> [!info] 核心结论
> 关键设备使用中压变频器时，常需要配置工频旁路，保证变频器故障或检修时设备可继续运行。

## ① 分类/公式/规则

> [!note] 🟦 旁路作用
> 变频器正常时调速节能；  
> 变频器故障或检修时，可切换到工频运行，提高系统连续性。

> [!warning] 🟧 旁路安全公式
> **旁路可靠 = 变频回路 + 工频回路 + 电气联锁 + 机械联锁 + 操作逻辑**

## ② 参考表

| 旁路方式 | 说明 |
|---|---|
| **无旁路** | 成本低，但故障停机风险高 |
| **手动旁路** | 人工切换，逻辑简单 |
| **自动旁路** | 自动切换，控制复杂 |
| **一拖一** | 一台变频器对应一台电机 |
| **一拖多** | 一台变频器服务多台电机，联锁复杂 |
| **工频接触器** | 工频运行回路关键元件 |

## ③ 计算/选型示例

- 已知：  
  一台循环泵为关键设备，不能因变频器故障长时间停运。
- 公式：  
  **运行连续性 = 变频运行 + 工频备用**
- 计算：  
  正常运行时用变频调速，变频器故障时切换到工频运行。
- 结论：  
  应评估配置 **旁路柜**，并严格设计联锁，避免变频和工频同时投入。

> [!tip] 记忆口诀
> 变频管节能，旁路保生产；联锁不到位，切换有风险。

# 10 制动与回馈

> [!info] 核心结论
> 大惯量、快速停车、下放负载和频繁启停场景，需要重点校核中压变频器的制动或回馈能力。

## ① 分类/公式/规则

> [!note] 🟦 能量去向
> 电机减速时可能处于发电状态，能量回灌到直流母线。  
> 若能量无法消耗或回馈，容易引起母线过压。

> [!warning] 🟧 制动需求公式
> **再生风险 = 大惯量 + 快速减速 + 下放负载 + 频繁启停**

## ② 参考表

| 方案 | 适合场景 |
|---|---|
| **延长减速时间** | 简单、低成本 |
| **制动电阻** | 中小能量消耗制动 |
| **制动单元** | 控制制动电阻投入 |
| **回馈单元** | 大功率频繁再生 |
| **共直流母线** | 多传动能量共享 |
| **机械制动** | 提升、保持、停车安全 |

## ③ 计算/选型示例

- 已知：  
  大惯量风机减速时报过压。
- 公式：  
  **过压风险 = 减速时间短 + 惯量大 + 再生能量高**
- 计算：  
  风机惯量大，快速减速时回灌能量升高。
- 结论：  
  可先延长减速时间，必要时配置制动或回馈方案，具体参数按厂家手册校核。

> [!tip] 记忆口诀
> 加速吃电，减速发电；能量不处理，母线就过压。

# 11 谐波与电网影响

> [!info] 核心结论
> 中压变频器接入电网时，需要关注输入谐波、功率因数、电压波动和电磁兼容。

## ① 分类/公式/规则

> [!note] 🟦 谐波来源
> 变频器整流环节会产生非正弦电流，可能影响电网电能质量。  
> 大容量中压系统尤其需要做谐波和电网容量评估。

> [!warning] 🟧 谐波控制公式
> **电能质量 = 谐波控制 + 功率因数 + 电网容量 + 滤波措施**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **输入谐波** | 影响电网电能质量 |
| **输出谐波** | 影响电机温升和绝缘 |
| **滤波器** | 降低谐波或EMI |
| **电抗器** | 改善电流冲击和电网影响 |
| **功率因数** | 需按工况校核 |
| **弱电网** | 更容易受变频器影响 |

## ③ 计算/选型示例

- 已知：  
  工厂电网容量较小，计划新增一台大功率中压变频器。
- 公式：  
  **接入风险 = 变频器容量 ÷ 电网容量**
- 计算：  
  变频器容量占电网容量比例越高，对电压波动和谐波影响越明显。
- 结论：  
  应进行电能质量评估，必要时配置电抗器、滤波器或优化整流方案。

> [!tip] 记忆口诀
> 大容量变频接电网，谐波和电压波动要先算。

# 12 电机绝缘与电缆

> [!info] 核心结论
> 中压变频器输出PWM波形，需关注电机绝缘、du/dt、电缆长度和轴承电流。

## ① 分类/公式/规则

> [!note] 🟦 适配重点
> 变频输出不是纯正弦波，快速开关会带来电压应力。  
> 老旧电机或长电缆系统尤其需要校核绝缘和过电压风险。

> [!warning] 🟧 电机适配公式
> **电机可用 = 绝缘等级满足 + 冷却满足 + 电缆长度合理 + 输出滤波校核**

## ② 参考表

| 项目 | 风险点 |
|---|---|
| **电机绝缘** | PWM电压应力 |
| **du/dt** | 电压变化率影响绝缘 |
| **长电缆** | 可能产生反射过压 |
| **轴承电流** | 可能引起轴承电蚀 |
| **低速散热** | 自冷电机低速散热变差 |
| **老电机改造** | 需检测绝缘状态 |

## ③ 计算/选型示例

- 已知：  
  老旧中压电机准备加装变频器。
- 公式：  
  **改造风险 = 老旧绝缘 + PWM应力 + 冷却不足**
- 计算：  
  电机原本按工频运行设计，长期变频运行可能增加绝缘和散热风险。
- 结论：  
  改造前应检测电机绝缘，必要时配置输出滤波、du/dt滤波器或更换变频专用电机。

> [!tip] 记忆口诀
> 变频器能带电机，不代表老电机一定扛得住。

- 计算：  
  柜内热量无法排出时，功率模块温度升高。
- 结论：  
  应检查风机、滤网、风道、柜内温度和负载电流。

# 14 PLC/DCS控制

> [!info] 核心结论
> 中压变频器可通过硬接线或通讯接入PLC/DCS，实现远程启停、频率给定、状态反馈和故障诊断。

## ① 分类/公式/规则

> [!note] 🟦 常见控制
> DI用于启停、复位、急停；

> DO/AO用于运行、故障、电流、频率反馈；  
> 通讯可读取更多状态和故障信息。

> [!warning] 🟧 远程控制公式
> **远程控制正常 = 控制源正确 + 给定源正确 + 联锁满足 + 通讯/端子正常**

## ② 参考表

| 信号/数据 | 作用 |
|---|---|
| **启动/停止** | 远程运行控制 |
| **频率给定** | 调速命令 |
| **运行反馈** | 变频器运行状态 |
| **故障反馈** | 报警或跳闸状态 |
| **电流反馈** | 负载监测 |
| **旁路状态** | 工频/变频状态判断 |

## ③ 计算/选型示例

- 已知：  
  DCS输出4-20mA，设定范围0～50Hz，当前输出 **16mA**。
- 公式：  
  **频率 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  频率 =（16 - 4）÷ 16 × 50 = **37.5Hz**
- 结论：  
  变频器应接收约 **37.5Hz** 给定，前提是AI量程和给定源参数一致。

> [!tip] 记忆口诀

# 15 安全与维护

> [!info] 核心结论
> 中压变频器涉及较高电压和大能量，维护必须执行停电、验电、放电、接地和挂牌流程。

## ① 分类/公式/规则

> [!note] 🟦 安全重点
> 变频器停机后，直流母线电容可能仍有残压。  
> 维护前必须等待放电完成，并由合格人员确认无电。

> [!warning] 🟧 安全维护公式
> **安全维护 = 停电 + 验电 + 放电 + 接地 + 挂牌 + 专人监护**

## ② 参考表

| 操作项 | 重点 |
|---|---|
| **停电** | 断开上级电源 |
| **验电** | 确认无电压 |
| **放电** | 等待母线电容放电 |
| **接地** | 挂接地线 |
| **挂牌** | 防止误送电 |
| **门禁联锁** | 不得随意短接 |

## ③ 计算/选型示例

- 已知：  
  变频器刚停机，准备打开功率柜检查。
- 公式：  
  **开柜条件 = 电源断开 + 母线电压安全 + 联锁允许**
- 计算：  
  停机不等于内部无电，直流母线可能仍有危险电压。
- 结论：  
  必须按厂家手册等待放电完成，并按电气安全规程操作。

> [!tip] 记忆口诀
> 停机不等于无电，开柜之前先验电。

# 16 常见故障排查

> [!info] 核心结论
> 中压变频器故障排查要结合故障码、发生工况、输入电网、电机负载、散热和通讯状态综合判断。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看故障代码；  
> 再看故障发生在启动、运行、减速还是切换；  
> 最后查电网、负载、电机、参数、冷却和通讯。

> [!warning] 🟧 排错公式
> **中压变频异常 = 电源问题 + 负载问题 + 电机问题 + 参数问题 + 散热问题 + 通讯问题**

## ② 参考表

| 故障 | 常见原因 |
|---|---|
| **启动失败** | 联锁未满足、控制源错误 |
| **过流OC** | 加速快、负载重、电机异常 |
| **过压OV** | 减速快、制动不足、回馈能量大 |
| **欠压UV** | 输入电压低、电网波动 |
| **过热OH** | 风机坏、滤网堵、柜温高 |
| **通讯故障** | 地址、协议、线缆、映射错误 |
| **旁路失败** | 接触器联锁或控制逻辑异常 |

## ③ 计算/选型示例

- 已知：  
  中压变频器减速时频繁报过压。
- 公式：  
  **减速过压 = 再生能量 + 减速时间过短 + 制动能力不足**
- 计算：  
  大惯量负载快速减速时，回灌能量会抬高直流母线电压。
- 结论：  
  可延长减速时间，或校核制动单元、制动电阻、回馈单元配置。

> [!tip] 记忆口诀
> 启动过流查负载，减速过压查制动；运行过热查风冷，远程失控查控制源。
