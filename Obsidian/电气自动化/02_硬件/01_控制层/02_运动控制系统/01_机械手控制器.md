# 01 机械手控制器总览

> [!info] 核心结论
> 机械手控制器用于控制机械手完成 **取放、搬运、码垛、上下料、装配、检测、点胶、焊接** 等动作，本质是多轴运动控制、逻辑控制、轨迹规划和安全联锁的综合控制平台。

## ① 分类/公式/规则

> [!warning] 选型公式
> **机械手控制器选型 = 机械手类型 + 轴数 + 负载 + 运动轨迹 + 控制精度 + 通讯接口 + I/O需求 + 安全功能**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制对象** | 桁架机械手、SCARA、六轴机器人、协作机器人、专机机械手 |
| **核心任务** | 多轴运动、轨迹规划、插补、回零、坐标变换 |
| **外部设备** | 夹爪、吸盘、视觉、输送线、PLC、传感器 |
| **常见接口** | EtherCAT、PROFINET、EtherNet/IP、Modbus、IO-Link |
| **重点参数** | 轴数、负载、速度、重复精度、节拍、安全等级 |
| **典型应用** | 机床上下料、注塑取件、码垛、装配、包装、检测 |

## ③ 计算/选型示例

- 已知：  
  设备需要三轴直角坐标机械手完成取放，末端带真空吸盘。
- 公式：  
  **三轴取放 = X/Y/Z多轴控制 + 末端I/O控制**
- 计算：  
  控制器至少需要支持3轴伺服控制、吸盘电磁阀输出、真空开关输入和安全限位。
- 结论：  
  可选择 **三轴以上机械手控制器或运动控制器**，并预留扩展I/O。

> [!tip] 记忆口诀
> 机械手控制器管三件事：轴怎么走、手怎么抓、什么时候安全停。

# 02 底层原理

> [!info] 核心结论
> 机械手控制器的底层原理，是根据目标点位和工艺动作，计算各轴运动轨迹，再驱动伺服、步进或机器人关节运动。

## ① 分类/公式/规则

> [!note] 工作链路
> 控制器接收程序指令，完成坐标计算和轨迹规划，通过驱动器控制电机，同时读取编码器、限位、夹爪、真空和视觉反馈。

> [!warning] 控制链路
> **程序指令 → 坐标计算 → 轨迹规划 → 伺服驱动 → 机械结构运动 → 传感器反馈 → 下一步动作**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **任务程序** | 定义取放、等待、判断、循环 |
| **坐标系统** | 管理机械坐标、工件坐标、工具坐标 |
| **轨迹规划** | 生成速度、加速度、路径 |
| **伺服控制** | 控制各轴位置和速度 |
| **末端执行器** | 夹爪、吸盘、焊枪、点胶头 |
| **反馈系统** | 编码器、限位、真空、压力、视觉 |

## ③ 计算/选型示例

- 已知：  
  机械手从A点移动到B点，需要避开中间障碍。
- 公式：  
  **安全路径 = 起点 + 中间过渡点 + 终点**
- 计算：  
  控制器可先抬升Z轴，再移动X/Y，最后下降取放。
- 结论：  
  点位规划不能只设起点和终点，复杂场景需设置安全过渡点。

> [!tip] 记忆口诀
> 机械手不是直冲目标，而是按路径、速度和安全条件运动。

# 03 常见类型

> [!info] 核心结论
> 机械手控制器要根据机械手结构选择，不同结构对应不同控制算法和运动方式。

## ① 分类/公式/规则

> [!warning] 类型选择规则
> **直线搬运 → 桁架/直角坐标机械手**  
> **平面高速取放 → SCARA机械手**  
> **复杂空间姿态 → 六轴机器人**  
> **人机协作 → 协作机器人**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **直角坐标机械手** | X/Y/Z直线轴，结构简单，适合取放 |
| **桁架机械手** | 行程长、负载大，常用于机床上下料 |
| **SCARA机械手** | 平面速度快，适合装配、搬运 |
| **六轴机器人** | 姿态灵活，适合复杂空间动作 |
| **协作机器人** | 安全协作，部署灵活 |
| **专机机械手** | 针对单一工艺定制，效率高 |

## ③ 计算/选型示例

- 已知：  
  工件只需要在水平面内高速取放，并且Z轴上下运动。
- 公式：  
  **平面取放 + 高速节拍 → SCARA或直角坐标机械手**
- 计算：  
  若动作轨迹简单，直角坐标更易维护；若要求高速灵活，SCARA更合适。
- 结论：  
  控制器需匹配机械结构，不能只看轴数。

> [!tip] 记忆口诀
> 行程长看桁架，平面快看SCARA，姿态复杂看六轴。

# 04 轴数与自由度

> [!info] 核心结论
> 轴数决定控制器能控制多少个运动自由度，自由度决定机械手能完成多复杂的空间动作。

## ① 分类/公式/规则

> [!warning] 轴数选型公式
> **控制器轴数 ≥ 机械手运动轴数 + 外部辅助轴数 + 预留轴数**

## ② 参考表

| 轴数 | 常见能力 |
|---|---|
| **1轴** | 单轴送料、升降 |
| **2轴** | 平面搬运、XY平台 |
| **3轴** | XYZ取放、直角坐标机械手 |
| **4轴** | XYZ + 旋转轴 |
| **5轴** | 增加姿态调整 |
| **6轴** | 空间位置和姿态控制 |
| **7轴及以上** | 机器人本体 + 外部轴/滑轨 |

## ③ 计算/选型示例

- 已知：  
  机械手有X、Y、Z三轴，末端还有一个旋转R轴。
- 公式：  
  **所需轴数 = X + Y + Z + R**
- 计算：  
  所需轴数 = **4轴**
- 结论：  
  控制器至少要支持4轴，若后期增加外部输送同步轴，建议选6轴或更高。

> [!tip] 记忆口诀
> 轴数别刚好，机械手后期最容易加旋转轴、夹爪轴和外部轴。

# 05 坐标系

> [!info] 核心结论
> 机械手控制器通过坐标系管理运动位置，常见有机械坐标、工件坐标、工具坐标和用户坐标。

## ① 分类/公式/规则

> [!note] 坐标理解
> 机械坐标是设备本体坐标；  
> 工件坐标是以工件为基准；  
> 工具坐标是以夹爪、吸盘、焊枪或点胶头为基准。

> [!warning] 坐标规则
> **实际运动位置 = 机械坐标 + 工件坐标偏移 + 工具坐标偏移**

## ② 参考表

| 坐标系 | 作用 |
|---|---|
| **机械坐标** | 机械手本体基准 |
| **世界坐标** | 统一空间参考 |
| **工件坐标** | 以工件或治具为基准 |
| **工具坐标TCP** | 以末端工具中心点为基准 |
| **用户坐标** | 用户自定义操作平面 |
| **视觉坐标** | 相机检测坐标，需要标定转换 |

## ③ 计算/选型示例

- 已知：  
  更换夹爪后，机械手取料位置整体偏移。
- 公式：  
  **工具变化 → TCP变化**
- 计算：  
  夹爪长度或安装位置变化，会让工具中心点改变。
- 结论：  
  应重新标定工具坐标TCP，而不是盲目修改所有点位。

> [!tip] 记忆口诀
> 点位不准不一定是程序错，先看工件坐标和工具坐标。

# 06 轨迹规划

> [!info] 核心结论
> 轨迹规划决定机械手如何从一个点运动到另一个点，直接影响节拍、平稳性和安全性。

## ① 分类/公式/规则

> [!warning] 轨迹规划公式
> **运动轨迹 = 起点 + 终点 + 路径类型 + 速度 + 加速度 + 过渡方式**

## ② 参考表

| 轨迹类型 | 适合场景 |
|---|---|
| **点到点PTP** | 快速移动，不严格要求路径 |
| **直线运动LIN** | 末端沿直线运动 |
| **圆弧运动CIR** | 圆弧轨迹 |
| **样条轨迹** | 平滑复杂路径 |
| **关节运动** | 机器人关节空间运动 |
| **连续路径CP** | 焊接、点胶、涂胶、切割 |

## ③ 计算/选型示例

- 已知：  
  点胶头必须沿工件边缘直线移动。
- 公式：  
  **工艺路径严格 → 直线插补或连续轨迹**
- 计算：  
  如果用普通点到点运动，末端路径可能不是直线。
- 结论：  
  应选择支持直线插补和连续轨迹的机械手控制器。

> [!tip] 记忆口诀
> 取放看点位，点胶焊接看轨迹。

# 07 速度、加速度与节拍

> [!info] 核心结论
> 机械手节拍不仅由最高速度决定，还受加速度、路径长度、等待时间、夹爪动作和传感反馈影响。

## ① 分类/公式/规则

> [!warning] 节拍公式
> **单循环时间 = 移动时间 + 抓取时间 + 等待确认时间 + 放料时间 + 安全延时**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **最高速度** | 决定长距离移动能力 |
| **加速度** | 决定起停响应 |
| **S曲线** | 减小冲击和振动 |
| **路径长度** | 路径越短节拍越快 |
| **夹爪响应** | 夹取和释放时间 |
| **传感确认** | 真空/夹紧/到位反馈时间 |

## ③ 计算/选型示例

- 已知：  
  机械手移动0.8s，夹取0.2s，放料0.2s，等待确认0.1s。
- 公式：  
  **节拍 = 0.8 + 0.2 + 0.2 + 0.1**
- 计算：  
  单循环时间 = **1.3s**
- 结论：  
  如果要提升节拍，不能只提高速度，也要优化夹取、等待和路径。

> [!tip] 记忆口诀
> 机械手快不快，不只看速度，还要看抓放和等待。

# 08 末端执行器控制

> [!info] 核心结论
> 机械手控制器不仅控制轴运动，还要控制末端夹爪、真空吸盘、电磁铁、焊枪、点胶阀等执行器。

## ① 分类/公式/规则

> [!warning] 末端控制公式
> **末端动作完成 = 输出动作 + 反馈确认 + 超时保护**

## ② 参考表

| 末端执行器 | 反馈方式 |
|---|---|
| **气动夹爪** | 开到位、关到位、压力开关 |
| **电动夹爪** | 位置、力矩、夹持状态 |
| **真空吸盘** | 真空开关、真空传感器 |
| **电磁铁** | 电流检测、工件检测 |
| **点胶阀** | 阀开反馈、压力反馈 |
| **焊枪** | 焊接电源状态、气体状态 |

## ③ 计算/选型示例

- 已知：  
  机械手吸取工件后偶尔掉料。
- 公式：  
  **吸取成功 = 真空值达到阈值 + 搬运中保持稳定**
- 计算：  
  只控制真空阀ON，不能证明工件已经吸住。
- 结论：  
  应接入真空开关，并在程序中加入吸附确认和掉压报警。

> [!tip] 记忆口诀
> 机械手会动只是第一步，抓得住、放得开才算完整动作。

# 09 I/O接口

> [!info] 核心结论
> 机械手控制器需要丰富I/O，用于连接夹具、传感器、按钮、限位、气动元件、安全信号和外部设备。

## ① 分类/公式/规则

> [!warning] I/O需求公式
> **I/O总需求 = 本体信号 + 末端信号 + 夹具信号 + 外部联锁 + 预留点数**

## ② 参考表

| I/O类型 | 用途 |
|---|---|
| **DI输入** | 原点、限位、夹紧、真空、工件有无 |
| **DO输出** | 夹爪、真空阀、吹气阀、指示灯 |
| **AI输入** | 压力、力、位移、真空模拟量 |
| **AO输出** | 比例阀、速度给定、压力给定 |
| **高速输入** | 编码器、视觉触发、位置锁存 |
| **安全输入** | 急停、安全门、光栅，需安全系统处理 |

## ③ 计算/选型示例

- 已知：  
  末端有2个真空阀、2个破真空阀、4个真空开关。
- 公式：  
  **末端I/O = 4DO + 4DI**
- 计算：  
  至少需要4个输出点和4个输入点。
- 结论：  
  机械手控制器或远程I/O应预留足够末端I/O，避免后期接线不足。

> [!tip] 记忆口诀
> 机械手I/O别只算轴，本体、夹具、末端都要算。

# 10 通讯与外部联动

> [!info] 核心结论
> 机械手通常要与PLC、视觉系统、输送线、机床、注塑机、扫码枪和MES进行通讯联动。

## ① 分类/公式/规则

> [!warning] 通讯选型公式
> **通讯方案 = 主控系统协议 + 联动设备协议 + 数据量 + 实时性 + 安全联锁需求**

## ② 参考表

| 通讯对象 | 常见内容 |
|---|---|
| **PLC** | 启停、允许、完成、报警、配方号 |
| **视觉系统** | 坐标、角度、OK/NG结果 |
| **伺服驱动** | 位置、速度、报警、使能 |
| **输送线** | 到位、跟踪、编码器同步 |
| **机床/注塑机** | 开门、允许取件、夹紧完成 |
| **HMI/MES** | 参数、产量、报警、追溯数据 |

## ③ 计算/选型示例

- 已知：  
  机械手要从视觉系统获取X/Y/角度补正量。
- 公式：  
  **视觉坐标 → 机械手坐标转换**
- 计算：  
  控制器需要支持通讯读取视觉数据，并执行坐标补偿。
- 结论：  
  应确认控制器支持视觉通讯接口和坐标偏移功能。

> [!tip] 记忆口诀
> 机械手不是孤岛，必须和PLC、视觉、设备状态联动。

# 11 视觉引导

> [!info] 核心结论
> 视觉引导机械手可以实现工件定位、偏移补偿、角度识别、抓取点计算和质量检测。

## ① 分类/公式/规则

> [!warning] 视觉引导公式
> **抓取坐标 = 基准坐标 + 视觉偏移量 + 工具补偿量**

## ② 参考表

| 视觉功能 | 应用 |
|---|---|
| **工件有无** | 判断是否抓取 |
| **坐标定位** | 输出X/Y偏移 |
| **角度识别** | 输出旋转补偿 |
| **缺陷检测** | OK/NG判断 |
| **条码识别** | 产品追溯 |
| **相机标定** | 视觉坐标转机械坐标 |

## ③ 计算/选型示例

- 已知：  
  工件基准抓取点为X=100mm，视觉检测偏移为+2.5mm。
- 公式：  
  **实际抓取X = 100 + 2.5**
- 计算：  
  实际抓取X = **102.5mm**
- 结论：  
  机械手应按视觉补偿后的坐标抓取。

> [!tip] 记忆口诀
> 视觉负责看偏多少，机械手负责补多少。

# 12 安全控制

> [!info] 核心结论
> 机械手属于高风险运动设备，必须配置急停、安全门、光栅、限位、软限位、速度限制和安全复位。

## ① 分类/公式/规则

> [!warning] 安全规则
> **普通程序不能替代安全回路，安全功能必须按风险评估设计**

## ② 参考表

| 安全功能 | 作用 |
|---|---|
| **急停** | 停止危险运动 |
| **安全门** | 防止人员进入危险区 |
| **安全光栅** | 监控人员进入 |
| **STO** | 伺服安全转矩关闭 |
| **软限位** | 程序限制运动范围 |
| **硬限位** | 机械/电气最终保护 |
| **安全复位** | 防止恢复后误启动 |

## ③ 计算/选型示例

- 已知：  
  人员打开安全门后，机械手必须停止并禁止重新启动。
- 公式：  
  **安全门打开 → 安全回路断开 → 伺服STO触发**
- 计算：  
  不能只靠普通控制程序暂停运动。
- 结论：  
  应使用安全门开关、安全继电器或安全PLC，并控制伺服STO。

> [!tip] 记忆口诀
> 机械手运动快、力大，安全回路不能省。

# 13 示教与编程

> [!info] 核心结论
> 机械手控制器通常支持示教点位、流程编程、坐标偏移、条件判断、循环和子程序。

## ① 分类/公式/规则

> [!note] 编程理解
> 示教是把实际点位记录到控制器中；  
> 程序是按顺序调用点位和动作，并根据传感器反馈判断下一步。

> [!warning] 程序结构
> **回原点 → 等待允许 → 到取料点 → 抓取确认 → 到放料点 → 放料确认 → 返回待机**

## ② 参考表

| 编程元素 | 作用 |
|---|---|
| **点位示教** | 记录取料、放料、过渡点 |
| **速度设定** | 控制不同动作速度 |
| **条件判断** | 判断工件有无、夹紧到位 |
| **循环执行** | 自动连续运行 |
| **子程序** | 复用取放、报警、回零逻辑 |
| **变量参数** | 配方、坐标偏移、计数 |
| **报警处理** | 异常停止和提示原因 |

## ③ 计算/选型示例

- 已知：  
  机械手取料点固定，但放料点有10个位置。
- 公式：  
  **多放料点 → 点位数组或配方参数**
- 计算：  
  程序可按计数器选择不同放料点。
- 结论：  
  控制器应支持多点位存储和参数化编程。

> [!tip] 记忆口诀
> 示教记录位置，程序决定顺序，传感器决定能不能走下一步。

# 14 回零与校准

> [!info] 核心结论
> 机械手上电后通常需要回零或读取绝对位置，确保控制器坐标与机械实际位置一致。

## ① 分类/公式/规则

> [!warning] 回零规则
> **无绝对位置反馈 → 上电需要回零**  
> **绝对值编码器 → 可读取当前位置，但仍需定期校准**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **原点开关** | 建立轴零点 |
| **限位开关** | 防止超程 |
| **绝对值编码器** | 断电后保留位置 |
| **机械基准块** | 校准位置基准 |
| **工具标定** | 校准TCP |
| **工件标定** | 校准工件坐标 |

## ③ 计算/选型示例

- 已知：  
  机械手断电重启后，点位整体偏移。
- 公式：  
  **位置偏移 = 回零异常 / 原点松动 / 机械打滑 / 工具偏移**
- 计算：  
  原点开关位置变化或同步带打滑都会导致坐标偏移。
- 结论：  
  应检查回零流程、原点开关、联轴器、同步带和工具标定。

> [!tip] 记忆口诀
> 点位漂了先别改程序，先查原点和机械基准。

# 15 常见故障排查

> [!info] 核心结论
> 机械手控制器故障排查按 **电源、急停、安全、伺服、原点限位、程序、I/O、机械结构** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **机械手异常 = 安全未解除 + 伺服报警 + 原点异常 + 点位错误 + I/O异常 + 程序互锁 + 机械卡滞**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **无法启动** | 急停未复位、安全门未关、伺服未使能 |
| **轴不动** | 驱动报警、限位触发、使能异常 |
| **回零失败** | 原点开关、方向、限位、速度设置错误 |
| **位置不准** | 原点偏移、工具坐标错误、机械松动 |
| **抓取失败** | 夹爪不到位、真空不足、工件偏位 |
| **路径异常** | 坐标系错误、点位错误、插补参数异常 |
| **节拍变慢** | 等待信号慢、速度低、加速度低、机构阻力大 |
| **频繁报警** | 负载过大、干扰、线缆松动、机械卡滞 |

## ③ 计算/选型示例

- 已知：  
  机械手程序运行到取料点后一直等待。
- 公式：  
  **一直等待 = 等待条件未满足**
- 计算：  
  可能等待真空到位、夹紧到位、PLC允许或工件检测信号。
- 结论：  
  应查看当前程序等待的具体输入点，再排查对应传感器和接线。

> [!tip] 记忆口诀
> 卡在哪一步，就查那一步的等待条件。

# 16 机械手控制器快速选型表

> [!info] 核心结论
> 机械手控制器选型要从机械结构和工艺动作出发，统一校核轴数、轨迹、负载、通讯、I/O、安全和示教编程能力。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 确认机械手类型：桁架、直角坐标、SCARA、六轴  
> ② 统计运动轴数和外部轴  
> ③ 确认负载、速度、节拍和重复精度  
> ④ 确认轨迹类型：点位、直线、圆弧、连续路径  
> ⑤ 确认末端执行器：夹爪、吸盘、工具  
> ⑥ 统计DI/DO/AI/AO和安全信号  
> ⑦ 确认通讯对象：PLC、视觉、HMI、MES、驱动器  
> ⑧ 确认坐标系、TCP、视觉补偿和配方需求  
> ⑨ 确认急停、安全门、光栅、STO和限位  
> ⑩ 确认示教、调试、报警、备份和维护工具

> [!warning] 最终选型公式
> **推荐机械手控制器 = 轴数 + 轨迹功能 + 通讯协议 + I/O容量 + 安全功能 + 示教编程 + 扩展余量**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单三轴取放 | 三轴/四轴机械手控制器 |
| 桁架上下料 | 长行程、多点位、机床联动 |
| SCARA装配 | 高速平面运动、视觉补偿 |
| 六轴搬运 | 姿态控制、TCP、机器人程序 |
| 视觉引导抓取 | 相机标定、坐标转换 |
| 真空吸附 | 真空开关、破真空、掉料报警 |
| 高安全要求 | 安全PLC、STO、安全门、光栅 |
| 多工位切换 | 配方、点位数组、参数管理 |

## ③ 计算/选型示例

- 已知：  
  设备需要4轴机械手，带1个气动夹爪，8个传感器，6个电磁阀，需要与PLC通讯。
- 公式：  
  **控制器需求 = 4轴 + 8DI + 6DO + PLC通讯 + 安全限位**
- 计算：  
  控制器至少支持4轴运动、8点输入、6点输出和对应通讯协议。
- 结论：  
  推荐选择 **4轴以上机械手控制器 + 远程I/O或本体I/O + PLC通讯接口**，并预留后期扩展点位。

> [!tip] 选型口诀
> 先定机械结构，再数轴和点；要抓取看末端，要定位看坐标；要安全看STO，要联动看通讯。
