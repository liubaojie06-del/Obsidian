# 01 EVO系列PLC总览

> [!info] 核心结论
> EVO系列PLC适合 **小型设备、单机自动化、顺序控制、I/O控制、简单通讯和基础运动控制** 场景。

## ① 分类/公式/规则

> [!warning] 选型公式
> **EVO系列PLC选型 = I/O点数 + 输出类型 + 通讯接口 + 扩展能力 + 高速功能 + 程序容量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **定位** | 小型PLC / 单机控制器 |
| **主要功能** | 顺序控制、定时、计数、互锁、报警 |
| **输入对象** | 按钮、传感器、限位、压力开关 |
| **输出对象** | 电磁阀、继电器、指示灯、蜂鸣器 |
| **通讯对象** | HMI、变频器、仪表、扫码枪 |
| **适用设备** | 小型包装机、夹具、输送线、测试台、简易工装 |

## ③ 计算/选型示例

- 已知：  
  设备有12个输入、10个输出，需要连接1台触摸屏。
- 公式：  
  **PLC点数 ≥ 实际I/O点数 + 预留余量**
- 计算：  
  实际为12DI + 10DO，建议预留10%~20%余量。
- 结论：  
  可选择点数更高一档的EVO系列PLC，并确认通讯口支持HMI连接。

> [!tip] 记忆口诀
> 点数够用是基础，通讯匹配才好用，余量不留后期愁。

# 02 底层原理

> [!info] 核心结论
> EVO系列PLC通过循环扫描方式工作：先读取输入，再执行程序，最后刷新输出。

## ① 分类/公式/规则

> [!warning] 扫描流程
> **输入采样 → 程序运算 → 输出刷新 → 通讯处理 → 状态诊断**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **输入采样** | 读取现场开关量信号 |
| **程序运算** | 执行梯形图、功能块、定时计数 |
| **输出刷新** | 控制电磁阀、继电器、指示灯 |
| **通讯处理** | 与HMI、变频器、仪表交换数据 |
| **故障诊断** | 检测运行、通讯、程序异常 |
| **数据保持** | 保存参数、计数值、状态 |

## ③ 计算/选型示例

- 已知：  
  传感器检测到工件后，PLC控制气缸伸出。
- 公式：  
  **响应时间 = 输入响应 + 扫描周期 + 输出响应 + 气缸动作时间**
- 计算：  
  普通顺控设备通常可满足；高速检测需校核扫描周期。
- 结论：  
  EVO系列PLC适合常规顺序控制，高速场景需按手册确认响应能力。

> [!tip] 记忆口诀
> PLC不是直接跳变，它是一圈一圈扫描执行。

# 03 EVO系列PLC适用场景

> [!info] 核心结论
> EVO系列PLC适合控制规模不大、动作逻辑清晰、成本敏感的小型自动化设备。

## ① 分类/公式/规则

> [!warning] 应用判断规则
> **少量I/O + 简单顺控 + 基础通讯 → EVO系列PLC可评估**

## ② 参考表

| 场景 | 适用性 |
|---|---|
| **单机设备** | 适合 |
| **小型输送线** | 适合 |
| **气动夹具** | 适合 |
| **简易包装机** | 适合 |
| **温度/压力简单控制** | 可评估 |
| **复杂多轴同步** | 不建议硬上，需评估更高平台 |
| **大型产线控制** | 更适合中型PLC或控制器 |

## ③ 计算/选型示例

- 已知：  
  设备只有气缸顺序动作、按钮操作和少量传感器。
- 公式：  
  **简单逻辑 + 少量点数 = 小型PLC**
- 计算：  
  不涉及复杂运动控制和大量通讯。
- 结论：  
  EVO系列PLC可以作为优先方案。

> [!tip] 记忆口诀
> 单机简单用EVO，多轴产线看更高一级PLC。

# 04 输入点DI

> [!info] 核心结论
> DI输入点用于采集按钮、限位、接近开关、光电、磁性开关等开关量信号。

## ① 分类/公式/规则

> [!warning] DI匹配公式
> **DI可用 = 信号电压 + PNP/NPN类型 + 公共端接法 + 输入响应时间匹配**

## ② 参考表

| 输入对象 | 说明 |
|---|---|
| **启动按钮** | 启动设备流程 |
| **停止按钮** | 停止设备动作 |
| **复位按钮** | 清除报警或回初始位 |
| **接近开关** | 检测金属或机构位置 |
| **光电开关** | 检测工件有无 |
| **磁性开关** | 检测气缸伸出/缩回 |
| **压力开关** | 检测气源或夹紧压力 |

## ③ 计算/选型示例

- 已知：  
  现场有8个传感器、3个按钮、2个限位开关。
- 公式：  
  **DI点数 ≥ 8 + 3 + 2**
- 计算：  
  DI点数 ≥ **13点**
- 结论：  
  选型时至少需要13点DI，并建议预留余量。

> [!tip] 记忆口诀
> 输入是PLC的眼睛，先数清楚再选型号。

# 05 输出点DO

> [!info] 核心结论
> DO输出点用于控制电磁阀、继电器、接触器、指示灯、蜂鸣器等负载。

## ① 分类/公式/规则

> [!warning] DO匹配公式
> **DO可用 = 负载电压 + 负载电流 + 输出类型 + 公共端接法**

## ② 参考表

| 输出对象 | 注意 |
|---|---|
| **电磁阀** | 注意线圈电流和浪涌 |
| **继电器** | 可用于隔离和扩流 |
| **接触器** | 建议中间继电器隔离 |
| **指示灯** | 电流较小 |
| **蜂鸣器** | 注意工作电压 |
| **报警灯塔** | 多色输出需多点DO |
| **小型执行器** | 校核输出点电流 |

## ③ 计算/选型示例

- 已知：  
  电磁阀线圈电流0.12A，PLC单点输出允许0.5A。
- 公式：  
  **单点输出能力 ≥ 负载电流**
- 计算：  
  0.5A ≥ 0.12A，单点能力满足。
- 结论：  
  单个输出点可驱动该电磁阀，但需校核公共端总电流。

> [!tip] 记忆口诀
> 输出点不是无限带负载，电流超了就要加继电器。

# 06 继电器输出与晶体管输出

> [!info] 核心结论
> EVO系列PLC选型时要区分继电器输出和晶体管输出：继电器通用，晶体管适合高速和直流负载。

## ① 分类/公式/规则

> [!warning] 选择规则
> **普通电磁阀/指示灯 → 继电器或晶体管均可评估**  
> **高速脉冲/步进控制 → 晶体管输出优先**

## ② 参考表

| 对比项 | 继电器输出 | 晶体管输出 |
|---|---|---|
| 负载类型 | AC/DC较通用 | 多用于DC负载 |
| 动作速度 | 较慢 | **快** |
| 使用寿命 | 有机械触点寿命 | 无触点，寿命较长 |
| 高频动作 | 不适合 | **适合** |
| 脉冲输出 | 不适合 | **适合** |
| 维护重点 | 触点寿命 | 过流保护 |

## ③ 计算/选型示例

- 已知：  
  需要用PLC输出脉冲控制步进驱动器。
- 公式：  
  **高速脉冲输出 → 晶体管型PLC**
- 计算：  
  继电器触点速度慢，不能稳定输出高速脉冲。
- 结论：  
  应选支持高速脉冲的晶体管输出型号，具体频率按厂家手册校核。

> [!tip] 记忆口诀
> 继电器管普通开关，晶体管管高速脉冲。

# 07 模拟量功能

> [!info] 核心结论
> 当设备需要读取压力、温度、流量、位移，或控制比例阀、变频器时，需要模拟量AI/AO功能。

## ① 分类/公式/规则

> [!warning] 模拟量换算公式
> **实际工程值 = 数字量 ÷ 满量程数字量 × 工程量满量程**

## ② 参考表

| 模拟量类型 | 常见应用 |
|---|---|
| **0-10V输入** | 压力、位移、速度反馈 |
| **4-20mA输入** | 压力、液位、流量 |
| **热电偶输入** | 温度检测 |
| **热电阻输入** | 精密温度检测 |
| **0-10V输出** | 变频器、比例阀给定 |
| **4-20mA输出** | 工业仪表或比例控制 |

## ③ 计算/选型示例

- 已知：  
  压力传感器0-10V对应0-1.0MPa，PLC读到6V。
- 公式：  
  **压力 = 6 ÷ 10 × 1.0**
- 计算：  
  压力 = **0.6MPa**
- 结论：  
  当前压力约0.6MPa，程序中需做量程换算。

> [!tip] 记忆口诀
> 开关量只看有无，模拟量必须换算。

# 08 通讯接口

> [!info] 核心结论
> EVO系列PLC通讯接口主要用于连接HMI、变频器、仪表、扫码枪、远程模块等设备。

## ① 分类/公式/规则

> [!warning] 通讯成功公式
> **通讯成功 = 协议一致 + 地址正确 + 参数一致 + 接线正确 + 数据格式正确**

## ② 参考表

| 通讯对象 | 重点 |
|---|---|
| **HMI触摸屏** | 协议、站号、IP或串口参数 |
| **变频器** | Modbus、频率给定、运行控制 |
| **仪表** | 地址、寄存器、量程 |
| **扫码枪** | 串口参数、数据格式 |
| **上位机** | 以太网或串口协议 |
| **远程I/O** | 通讯周期、站点数量 |

## ③ 计算/选型示例

- 已知：  
  EVO系列PLC需要连接1台HMI和2台变频器。
- 公式：  
  **通讯口数量和协议 ≥ 外部设备需求**
- 计算：  
  HMI可走以太网或串口，变频器可走RS485 Modbus。
- 结论：  
  选型时需确认PLC是否有足够通讯口，或能否扩展通讯模块。

> [!tip] 记忆口诀
> 通讯先看协议，再看接口；参数不一致，接线也白接。

# 09 高速计数与脉冲输出

> [!info] 核心结论
> EVO系列PLC若用于编码器计数、步进定位或简单伺服控制，需要关注高速输入和高速输出能力。

## ① 分类/公式/规则

> [!warning] 高速功能选型公式
> **高速功能可用 = 频率范围 + 通道数量 + 输出类型 + 指令功能匹配**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **高速计数HSC** | 读取编码器、计数轮、测速 |
| **高速脉冲PTO** | 控制步进/伺服位置 |
| **PWM输出** | 简单占空比控制 |
| **方向信号DIR** | 控制运动方向 |
| **原点输入** | 建立轴坐标 |
| **限位输入** | 防止超程 |

## ③ 计算/选型示例

- 已知：  
  步进驱动器需要最高20kHz脉冲，PLC高速输出最高100kHz。
- 公式：  
  **PLC高速输出频率 ≥ 驱动器需求频率**
- 计算：  
  100kHz ≥ 20kHz，频率满足。
- 结论：  
  可用于该步进轴控制，但需确认加减速、方向、原点和限位功能。

> [!tip] 记忆口诀
> 普通输出控阀，高速输出控轴。

# 10 与HMI配合

> [!info] 核心结论
> EVO系列PLC常与HMI触摸屏配合，实现参数设定、手动调试、报警显示、配方切换和状态监控。

## ① 分类/公式/规则

> [!warning] HMI功能公式
> **HMI功能 = 操作按钮 + 参数设定 + 状态显示 + 报警记录 + 配方管理**

## ② 参考表

| HMI功能 | 作用 |
|---|---|
| **启动/停止** | 设备基本操作 |
| **手动模式** | 单动作调试 |
| **自动模式** | 运行生产流程 |
| **参数设定** | 时间、计数、速度、温度 |
| **报警显示** | 提示故障原因 |
| **配方管理** | 多产品切换 |
| **产量统计** | 记录生产数量 |

## ③ 计算/选型示例

- 已知：  
  设备需要3种产品切换，每种产品延时参数不同。
- 公式：  
  **多产品参数 → HMI配方管理**
- 计算：  
  HMI保存不同产品参数，PLC根据产品号调用。
- 结论：  
  可设计配方页面，减少人工修改程序风险。

> [!tip] 记忆口诀
> PLC负责控制，HMI负责操作和参数。

# 11 程序设计

> [!info] 核心结论
> EVO系列PLC程序应保持结构简单清晰，重点做好手自动、互锁、报警、超时和复位逻辑。

## ① 分类/公式/规则

> [!warning] 程序结构
> **初始化 → 手动模式 → 自动流程 → 互锁保护 → 报警超时 → 复位处理**

## ② 参考表

| 程序模块 | 作用 |
|---|---|
| **初始化** | 上电状态整理 |
| **手动控制** | 调试单个动作 |
| **自动流程** | 按步骤运行 |
| **互锁条件** | 防止错误动作 |
| **超时报警** | 防止一直等待 |
| **复位逻辑** | 报警后恢复 |
| **参数管理** | 保存工艺设定 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出指令发出后，3秒内未收到伸出到位。
- 公式：  
  **输出已发出 + 到位未反馈 + 时间超限 = 动作超时报警**
- 计算：  
  PLC应停止后续动作，并提示“气缸伸出不到位”。
- 结论：  
  自动流程必须加超时保护，不能无限等待。

> [!tip] 记忆口诀
> 动作要有反馈，等待要有超时，报警要能复位。

# 12 典型接线

> [!info] 核心结论
> EVO系列PLC接线重点是电源、输入公共端、输出公共端、负载电流、通讯屏蔽和接地。

## ① 分类/公式/规则

> [!warning] 接线规则
> **稳定接线 = 电源稳定 + COM正确 + 强弱电分离 + 接地可靠 + 标识清楚**

## ② 参考表

| 接线项 | 注意 |
|---|---|
| **24V电源** | 容量足够，纹波小 |
| **输入COM** | 与PNP/NPN方式匹配 |
| **输出COM** | 与负载电源匹配 |
| **电磁阀线圈** | 注意浪涌吸收 |
| **模拟量线** | 使用屏蔽线，远离动力线 |
| **通讯线** | 注意终端电阻和屏蔽 |
| **PE接地** | 控制柜可靠接地 |

## ③ 计算/选型示例

- 已知：  
  程序监控输出点ON，但现场电磁阀不动作。
- 公式：  
  **程序ON ≠ 端子有电**
- 计算：  
  需要用万用表测量PLC输出端子和电磁阀线圈电压。
- 结论：  
  若端子无电，查输出COM和模块；若端子有电，查电磁阀和气路。

> [!tip] 记忆口诀
> 程序亮看端子，端子有电看负载。

# 13 安装与维护

> [!info] 核心结论
> EVO系列PLC安装要注意电柜散热、抗干扰、接地、端子固定、模块扩展空间和程序备份。

## ① 分类/公式/规则

> [!warning] 维护公式
> **稳定运行 = 良好电源 + 良好接地 + 合理散热 + 规范布线 + 定期备份**

## ② 参考表

| 项目 | 维护重点 |
|---|---|
| **电源** | 检查24V稳定性 |
| **端子** | 防松动、虚接 |
| **散热** | 柜内温度不过高 |
| **接地** | 屏蔽线和PE处理 |
| **程序备份** | 防止设备故障后无法恢复 |
| **参数备份** | 配方、计数、设定值 |
| **通讯线** | 防松、防干扰 |

## ③ 计算/选型示例

- 已知：  
  设备运行时偶发通讯中断，控制柜内有变频器。
- 公式：  
  **通讯异常 = 干扰 + 接地不良 + 线缆布线不当**
- 计算：  
  通讯线与动力线长距离并行，可能受到干扰。
- 结论：  
  应分开布线，检查屏蔽接地，并确认通讯参数和终端电阻。

> [!tip] 记忆口诀
> PLC怕干扰，通讯线和动力线别贴太近。

# 14 常见故障排查

> [!info] 核心结论
> EVO系列PLC故障排查按 **电源、RUN状态、输入、输出、通讯、程序、现场设备** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **PLC异常 = 电源异常 + 接线错误 + 输入异常 + 输出异常 + 通讯失败 + 程序互锁 + 现场故障**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **PLC不亮** | 电源无电、保险损坏、电源故障 |
| **RUN不亮** | 程序错误、CPU停止 |
| **输入不亮** | 传感器、电源、公共端、PNP/NPN错误 |
| **输出不动作** | 输出COM、负载、电磁阀、继电器异常 |
| **HMI连不上** | 协议、IP、站号、线缆、端口错误 |
| **自动不启动** | 启动条件不满足、报警未复位 |
| **动作不到位** | 气源、机构、传感器、程序超时 |

## ③ 计算/选型示例

- 已知：  
  启动按钮按下后，PLC输入灯不亮。
- 公式：  
  **输入灯不亮 = 按钮/线路/电源/输入类型异常**
- 计算：  
  先测按钮输出电压，再测PLC输入端电压。
- 结论：  
  若PLC端子已有正确电压但灯不亮，需检查输入类型和模块状态。

> [!tip] 记忆口诀
> 输入不亮查前端，输出不动查后端，通讯不通查参数。

# 15 EVO系列PLC快速选型表

> [!info] 核心结论
> EVO系列PLC选型先统计I/O，再确认输出类型、通讯接口、模拟量、高速功能和扩展余量。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 统计DI输入点数  
> ② 统计DO输出点数  
> ③ 确认继电器输出还是晶体管输出  
> ④ 确认传感器PNP/NPN类型  
> ⑤ 确认是否需要AI/AO模拟量  
> ⑥ 确认是否需要高速计数/脉冲输出  
> ⑦ 确认HMI通讯方式  
> ⑧ 确认是否连接变频器、仪表、扫码枪  
> ⑨ 确认程序容量和扩展模块  
> ⑩ 预留10%~20%点数和通讯余量

> [!warning] 最终选型公式
> **推荐EVO系列PLC = 本体点数 + 输出类型 + 通讯接口 + 模拟量扩展 + 高速功能 + 扩展余量**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单顺序控制 | 基础EVO本体 |

## ③ 计算/选型示例

- 已知：  
  设备有16DI、12DO、2AI、1个HMI、1台变频器。
- 公式：  
  **PLC配置 ≥ 16DI + 12DO + 2AI + 通讯接口**
- 计算：  
  本体I/O需满足28点以上，模拟量需2AI，通讯需支持HMI和变频器。
- 结论：  
  推荐选择 **EVO系列PLC本体 + 模拟量扩展 + 通讯接口**，并按厂家手册确认具体型号。
