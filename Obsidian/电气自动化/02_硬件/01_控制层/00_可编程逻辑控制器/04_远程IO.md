# 01 柜内IO与柜外IO总览

> [!info] 核心结论
> 柜内IO安装在控制柜内，适合集中接线和稳定环境；柜外IO安装在设备现场，适合分布式接线、减少线缆和模块化设备。

## ① 分类/公式/规则

> [!warning] 选型公式
> **IO方案选型 = 点位数量 + 现场距离 + 环境等级 + 通讯协议 + 接线成本 + 维护方式 + 响应要求**

## ② 参考表

| 类型 | 主要特点 |
|---|---|
| **柜内IO** | 集中安装在电柜内，接线规整，维护集中 |
| **柜外IO** | 靠近现场传感器/执行器，减少长距离布线 |
| **本地IO** | 靠近PLC CPU安装 |
| **远程IO** | 通过现场总线或工业以太网连接PLC |
| **分布式IO** | 分散在各设备模块或工位附近 |
| **防护型IO** | 常见IP65/IP67，用于柜外恶劣环境 |

## ③ 计算/选型示例

- 已知：  
  一台设备有5个工位，每个工位30个传感器和执行器。
- 公式：  
  **点位分散 + 线缆长 → 柜外/远程IO优先**
- 计算：  
  如果全部拉回控制柜，线缆数量多、排查困难。
- 结论：  
  可在每个工位附近布置柜外远程IO模块，通过总线回PLC。

> [!tip] 记忆口诀
> 点位集中用柜内，点位分散用柜外；线越长，越该考虑远程IO。

# 02 柜内IO

> [!info] 核心结论
> 柜内IO通常安装在PLC机架、远程IO站或电控柜DIN导轨上，适合控制柜集中布线和维护。

## ① 分类/公式/规则

> [!note] 核心理解
> 柜内IO是传统自动化控制中最常见的方式。  
> 现场传感器、电磁阀、按钮、继电器等信号通过电缆进入控制柜，再接到IO模块。

> [!warning] 柜内IO选型公式
> **柜内IO选型 = DI/DO/AI/AO点数 + 电压等级 + 输出类型 + 模块通道数 + 端子形式 + 扩展余量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **安装位置** | 控制柜内DIN导轨或PLC机架 |
| **环境要求** | 柜内干燥、防尘、温度受控 |
| **接线方式** | 端子排集中接线 |
| **优点** | 维护集中、模块保护好 |
| **缺点** | 现场线缆多、柜内接线复杂 |
| **适合场景** | 单机设备、小中型控制柜、点位集中设备 |

## ③ 计算/选型示例

- 已知：  
  设备所有传感器距离控制柜较近，I/O总点数不多。
- 公式：  
  **点位集中 + 距离短 → 柜内IO优先**
- 计算：  
  线缆数量和长度可控，柜内集中接线方便维护。
- 结论：  
  可选择 **PLC本地IO或柜内远程IO模块**。

> [!tip] 记忆口诀
> 柜内IO好维护，适合线短点位集中。

---

# 03 柜外IO

> [!info] 核心结论
> 柜外IO安装在设备现场或工位附近，常用M8/M12接口和工业总线，适合分布式设备和模块化产线。

## ① 分类/公式/规则

> [!note] 核心理解
> 柜外IO把IO模块从控制柜搬到现场，现场传感器和执行器就近接入，再通过一根通讯线和电源线回到PLC。

> [!warning] 柜外IO选型公式
> **柜外IO选型 = 防护等级 + 通讯协议 + 接口类型 + 点数 + 供电容量 + 环境适应性**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **安装位置** | 设备现场、机架、工位附近 |
| **防护等级** | 常见IP65/IP67 |
| **接口方式** | M8、M12、快插接头 |
| **优点** | 减少线缆、模块化强、排查方便 |
| **缺点** | 成本较高，对环境和接插件要求高 |
| **适合场景** | 产线、多工位、机器人周边、输送系统 |

## ③ 计算/选型示例

- 已知：  
  现场有一个夹具模块，需要8个输入和8个输出，距离电柜15米。
- 公式：  
  **远距离模块化点位 → 柜外IO优先**
- 计算：  
  柜外IO可减少16根以上信号线长距离回柜。
- 结论：  
  可选择 **IP67柜外IO模块 + M12接插件 + 总线通讯**。

> [!tip] 记忆口诀
> 柜外IO靠近现场，线少、模块清、换机快。

# 04 柜内IO vs 柜外IO

> [!info] 核心结论
> 柜内IO适合集中控制，柜外IO适合分布式控制；不是谁替代谁，而是按设备结构选择。

## ① 分类/公式/规则

> [!warning] 选择规则
> **点位集中、环境普通、成本敏感 → 柜内IO**  
> **点位分散、线缆长、模块化强、防护要求高 → 柜外IO**

## ② 参考表

| 对比项 | 柜内IO | 柜外IO |
|---|---|---|
| 安装位置 | 控制柜内 | 设备现场 |
| 接线方式 | 端子排集中 | M8/M12快插 |
| 防护要求 | 依赖控制柜 | 模块自身防护 |
| 线缆数量 | 较多 | **较少** |
| 维护方式 | 柜内集中维护 | 现场就近维护 |
| 适合设备 | 点位集中 | 点位分散、多工位 |
| 成本 | 模块成本较低 | 模块和接插件成本较高 |

## ③ 计算/选型示例

- 已知：  
  两台设备I/O点数相同，但一台点位集中在电柜附近，另一台分布在20米长产线上。
- 公式：  
  **同样点数，不同布局 → IO方案不同**
- 计算：  
  点位集中适合柜内IO，点位分散适合柜外IO。
- 结论：  
  IO选型不能只看点数，还要看现场分布。

> [!tip] 记忆口诀
> 点数只是数量，分布才决定柜内还是柜外。

# 05 DI输入模块

> [!info] 核心结论
> DI模块用于读取开关量信号，柜内和柜外都必须匹配传感器类型、电压等级和PNP/NPN逻辑。

## ① 分类/公式/规则

> [!warning] DI匹配公式
> **DI可用 = 电压等级 + PNP/NPN类型 + 输入响应 + 公共端接法**

## ② 参考表

| 输入对象 | 说明 |
|---|---|
| **按钮** | 启动、停止、复位 |
| **接近开关** | 检测金属或机构位置 |
| **光电传感器** | 工件有无检测 |
| **磁性开关** | 气缸伸缩到位 |
| **压力开关** | 气源或夹紧压力检测 |
| **安全信号** | 需进入安全模块，不建议普通DI替代 |

## ③ 计算/选型示例

- 已知：  
  现场有18个PNP传感器，控制器输入模块为PNP型。
- 公式：  
  **DI点数 ≥ 18 + 预留余量**
- 计算：  
  若取20%余量，18 × 1.2 = **21.6点**
- 结论：  
  可选择24点DI输入容量，并确认PNP接线方式匹配。

> [!tip] 记忆口诀
> 输入先分PNP/NPN，再数点数和余量。

# 06 DO输出模块

> [!info] 核心结论
> DO模块用于控制电磁阀、继电器、指示灯、蜂鸣器等负载，重点校核输出电流和公共端容量。

## ① 分类/公式/规则

> [!warning] DO匹配公式
> **DO可用 = 负载电压 + 单点电流 + 公共端总电流 + 输出类型**

## ② 参考表

| 输出对象 | 注意 |
|---|---|
| **电磁阀** | 注意线圈电流和浪涌吸收 |
| **继电器** | 可用于隔离和扩流 |
| **指示灯** | 负载较小 |
| **蜂鸣器** | 注意电源电压 |
| **报警灯塔** | 多色灯需多个输出点 |
| **接触器** | 建议中间继电器隔离 |

## ③ 计算/选型示例

- 已知：  
  一个DO公共端下接8个电磁阀，每个电磁阀0.12A。
- 公式：  
  **公共端总电流 = 单个电流 × 同时动作数量**
- 计算：  
  总电流 = 0.12 × 8 = **0.96A**
- 结论：  
  该公共端需要支持至少0.96A，并留余量；否则应分组供电或加继电器。

> [!tip] 记忆口诀
> 单点够不够看单阀，总线够不够看同时动作。

# 07 模拟量IO

> [!info] 核心结论
> 模拟量IO用于读取连续信号或输出连续控制量，常见于压力、温度、流量、液位、比例阀和变频器。

## ① 分类/公式/规则

> [!warning] 模拟量换算公式
> **工程值 = 原始数字量 ÷ 满量程数字量 × 工程量满量程**

## ② 参考表

| 类型 | 常见信号 |
|---|---|
| **AI电压输入** | 0-10V、0-5V |
| **AI电流输入** | 4-20mA、0-20mA |
| **RTD输入** | PT100、PT1000 |
| **TC输入** | K型、J型热电偶 |
| **AO电压输出** | 0-10V比例阀、变频器 |
| **AO电流输出** | 4-20mA仪表控制 |

## ③ 计算/选型示例

- 已知：  
  压力传感器0-10V对应0-1.0MPa，PLC采集到7V。
- 公式：  
  **压力 = 7 ÷ 10 × 1.0**
- 计算：  
  压力 = **0.7MPa**
- 结论：  
  当前压力约0.7MPa，程序中需要做量程换算和滤波。

> [!tip] 记忆口诀
> 开关量看通断，模拟量看数值；读进来以后一定要换算。

# 08 通讯协议

> [!info] 核心结论
> 柜内IO和柜外IO都可能通过通讯连接PLC，常见协议有PROFINET、EtherCAT、EtherNet/IP、Modbus、IO-Link等。

## ① 分类/公式/规则

> [!warning] 通讯选型公式
> **通讯方案 = PLC协议 + IO模块协议 + 站点数量 + 刷新周期 + 诊断需求**

## ② 参考表

| 协议 | 常见用途 |
|---|---|
| **PROFINET** | 西门子系统、远程IO、阀岛 |
| **EtherCAT** | 高速IO、多轴运动系统 |
| **EtherNet/IP** | 工业以太网设备 |
| **Modbus TCP/RTU** | 通用IO、仪表、变频器 |
| **IO-Link** | 智能传感器、参数化设备 |
| **CANopen** | 部分运动控制和设备网络 |

## ③ 计算/选型示例

- 已知：  
  PLC为PROFINET系统，现场需要柜外IO。
- 公式：  
  **PLC协议 = IO通讯协议匹配**
- 计算：  
  应选择支持PROFINET的柜外IO站点，或通过网关转换。
- 结论：  
  优先选择与PLC原生协议一致的IO模块。

> [!tip] 记忆口诀
> 协议不一致，点数再够也接不上。

# 09 IP防护等级

> [!info] 核心结论
> 柜外IO必须重点关注防护等级，柜内IO主要依赖控制柜防护。

## ① 分类/公式/规则

> [!warning] 防护选择规则
> **柜内干燥环境 → 普通模块即可**  
> **柜外粉尘/水汽/油污环境 → IP65/IP67模块优先**

## ② 参考表

| 安装环境 | 推荐关注点 |
|---|---|
| **控制柜内** | 散热、防尘、端子可靠 |
| **设备机架外部** | IP65/IP67、防震 |
| **潮湿区域** | 防水接插件、防腐蚀 |
| **粉尘区域** | 密封、防尘帽 |
| **油污区域** | 耐油线缆和接头 |
| **焊接区域** | 耐高温、防飞溅线缆 |

## ③ 计算/选型示例

- 已知：  
  IO模块安装在设备外部，现场经常有水汽和油雾。
- 公式：  
  **柜外潮湿油污 → 高防护IO**
- 计算：  
  普通柜内模块直接外置容易进水、腐蚀或短路。
- 结论：  
  应选择IP67柜外IO，并使用匹配M12防护接插件。

> [!tip] 记忆口诀
> 柜内靠柜子防护，柜外靠模块自己防护。

# 10 接线方式

> [!info] 核心结论
> 柜内IO多用端子排接线，柜外IO多用M8/M12快插接线。

## ① 分类/公式/规则

> [!warning] 接线选择公式
> **接线方式 = 安装位置 + 防护要求 + 维护频率 + 信号类型**

## ② 参考表

| 接线方式 | 特点 |
|---|---|
| **端子排接线** | 柜内常用，成本低，接线密集 |
| **弹簧端子** | 接线效率高，抗振较好 |
| **螺钉端子** | 通用，需注意松动 |
| **M8接头** | 小型传感器常见 |
| **M12接头** | 工业现场常见，可靠性较好 |
| **预制线缆** | 快速安装，维护方便 |

## ③ 计算/选型示例

- 已知：  
  一个现场模块经常需要拆装更换。
- 公式：  
  **频繁维护 → 快插接插件更方便**
- 计算：  
  使用M12预制线缆可减少现场剥线和压线时间。
- 结论：  
  柜外IO推荐使用标准快插线缆，提高维护效率。

> [!tip] 记忆口诀
> 柜内端子排，柜外M12；想快换，用预制线。

# 11 供电与分组

> [!info] 核心结论
> IO模块供电要区分逻辑电源和负载电源，输出负载多时必须校核总电流和分组保护。

## ① 分类/公式/规则

> [!warning] 供电校核公式
> **电源容量 ≥ 模块自身耗电 + 所有负载电流 × 同时系数 + 余量**

## ② 参考表

| 电源类型 | 作用 |
|---|---|
| **逻辑电源** | 给IO模块内部电路供电 |
| **传感器电源** | 给接近、光电、磁开供电 |
| **负载电源** | 给电磁阀、继电器、灯等负载供电 |
| **分组保险** | 防止单路短路影响全站 |
| **电源诊断** | 低压、短路、过载报警 |
| **公共端COM** | 必须按PNP/NPN和负载方式接线 |

## ③ 计算/选型示例

- 已知：  
  柜外IO带16个电磁阀，每个0.1A，最多同时动作10个。
- 公式：  
  **负载电流 = 0.1 × 10**
- 计算：  
  负载电流 = **1.0A**
- 结论：  
  负载电源至少要满足1.0A并留余量，同时校核IO模块输出总电流。

> [!tip] 记忆口诀
> 逻辑电源管模块，负载电源管动作；别把两者混着算。

# 12 诊断功能

> [!info] 核心结论
> 现代IO模块不仅传输信号，还可以诊断短路、断线、掉站、低电压和模块故障。

## ① 分类/公式/规则

> [!warning] 诊断价值公式
> **诊断能力越强 → 故障定位越快 → 停机时间越短**

## ② 参考表

| 诊断项 | 说明 |
|---|---|
| **模块掉站** | 通讯中断 |
| **通道短路** | 输出或传感器供电短路 |
| **断线检测** | 部分模拟量或智能IO支持 |
| **低电压报警** | 供电不足 |
| **过载报警** | 负载电流过大 |
| **IO-Link诊断** | 可读传感器状态和参数 |
| **通道级诊断** | 精确定位到具体点位 |

## ③ 计算/选型示例

- 已知：  
  某个柜外IO输出通道频繁报警。
- 公式：  
  **通道报警 = 负载短路 / 过流 / 接线异常**
- 计算：  
  可直接定位到该通道，检查对应电磁阀线圈和线缆。
- 结论：  
  带通道诊断的IO模块可显著提高排故效率。

> [!tip] 记忆口诀
> 没诊断靠人找，有诊断直接报点位。

# 13 柜内IO常见问题

> [!info] 核心结论
> 柜内IO常见问题集中在端子松动、公共端错误、线号混乱、强弱电干扰和柜内空间不足。

## ① 分类/公式/规则

> [!warning] 排错公式
> **柜内IO异常 = 接线松动 + COM错误 + 模块故障 + 干扰 + 地址映射错误**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **输入不亮** | 传感器无电、COM错误、线断 |
| **输出不动作** | 输出COM错误、负载无电、保险断 |
| **模拟量波动** | 屏蔽接地差、动力线干扰 |
| **模块报警** | 短路、供电异常、模块故障 |
| **地址不对** | 硬件组态或程序映射错误 |
| **维护困难** | 线号不清、端子排拥挤 |

## ③ 计算/选型示例

- 已知：  
  程序中输入点ON/OFF正常，但现场实际设备状态对不上。
- 公式：  
  **状态不一致 = 接线点位 / 地址映射 / 程序标签错误**
- 计算：  
  可能传感器接到了错误端子或程序地址写错。
- 结论：  
  应按电气图逐点核对端子号、PLC地址和程序变量名。

> [!tip] 记忆口诀
> 柜内问题多查线号、端子、COM和地址。

# 14 柜外IO常见问题

> [!info] 核心结论
> 柜外IO常见问题集中在通讯掉站、接插件进水、线缆损伤、供电不足和现场干扰。

## ① 分类/公式/规则

> [!warning] 排错公式
> **柜外IO异常 = 通讯异常 + 电源不足 + 接插件松动 + 进水腐蚀 + 线缆破损 + 地址错误**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **整站掉线** | 网线、电源、站号、通讯配置 |
| **部分点异常** | 接插件松、线缆断、传感器坏 |
| **频繁报警** | 供电波动、干扰、接地不良 |
| **进水短路** | 防护帽未装、接头未锁紧 |
| **输出无力** | 负载电源不足或压降过大 |
| **模块发热** | 过载、环境温度高 |

## ③ 计算/选型示例

- 已知：  
  柜外IO雨后频繁报短路。
- 公式：  
  **潮湿后短路 = 接插件密封不良 / 防护帽缺失**
- 计算：  
  未使用的M12接口如果未装防护帽，容易进水。
- 结论：  
  应检查所有接插件锁紧状态，并给未用接口安装防护帽。

> [!tip] 记忆口诀
> 柜外IO怕水、怕松、怕线缆被磨破。

# 15 快速选型表

> [!info] 核心结论
> 柜内IO和柜外IO选型要同时考虑点数、位置、通讯、防护、供电、诊断和维护方式。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 统计DI/DO/AI/AO点数  
> ② 判断点位集中还是分散  
> ③ 确认柜内安装还是柜外安装  
> ④ 确认PLC通讯协议  
> ⑤ 确认PNP/NPN和电压等级  
> ⑥ 确认输出负载电流和分组供电  
> ⑦ 确认防护等级和接插件  
> ⑧ 确认刷新周期和响应要求  
> ⑨ 确认诊断功能和维护方式  
> ⑩ 预留10%~20%点位和通讯余量

> [!warning] 最终选型公式
> **推荐IO方案 = 柜内/柜外位置 + 点位类型 + 通讯协议 + 防护等级 + 供电分组 + 诊断能力 + 扩展余量**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 点位集中 | 柜内IO |
| 点位分散 | 柜外IO/远程IO |
| 控制柜空间充足 | 柜内模块 |
| 现场模块化设备 | 柜外IP67 IO |
| 高速响应 | 校核总线周期和输入滤波 |
| 恶劣环境 | 高防护、耐油耐磨线缆 |
| 快速维护 | M12快插和通道诊断 |
| 低成本 | 柜内IO更常见 |

## ③ 计算/选型示例

- 已知：  
  一台设备有40DI、32DO，其中大部分点位集中在电柜附近，只有远端工位有12DI和8DO。
- 公式：  
  **集中点位用柜内，远端点位用柜外**
- 计算：  
  柜内处理主要I/O，远端工位配置一个柜外IO站。
- 结论：  
  推荐 **柜内IO + 柜外远程IO混合方案**，既降低成本，又减少远端布线。

> [!tip] 选型口诀
> 集中进柜，分散到现场；短线走端子，长线走远程；防护看环境，诊断看维护。
