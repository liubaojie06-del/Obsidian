# 01 小型PLC总览

> [!info] 核心结论
> 小型PLC适合 **单机设备、简单顺序控制、少量I/O、低成本自动化控制** 场景。

## ① 分类/公式/规则

> [!warning] 选型公式
> **小型PLC选型 = I/O点数 + 电源电压 + 输出类型 + 通讯接口 + 扩展模块 + 程序容量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **控制规模** | 小型单机、简单设备 |
| **I/O点数** | 通常点数较少 |
| **结构形式** | 一体式为主，可少量扩展 |
| **控制内容** | 顺序、互锁、定时、计数 |
| **常见设备** | 小型输送线、夹具、包装机、测试台 |
| **优势** | 成本低、接线简单、调试快 |

## ③ 计算/选型示例

- 已知：  
  设备有10个输入、8个输出，需要控制气缸和电磁阀。
- 公式：  
  **PLC点数 ≥ 实际I/O点数 + 预留余量**
- 计算：  
  实际总点数 = 10DI + 8DO，建议预留20%左右。
- 结论：  
  可选择 **24点左右小型PLC**，并确认输出类型能驱动电磁阀或继电器。

> [!tip] 记忆口诀
> 小设备、小点数、小成本，优先看小型PLC。

# 02 底层原理

> [!info] 核心结论
> 小型PLC通过循环扫描方式，读取输入、执行程序、刷新输出，从而控制设备动作。

## ① 分类/公式/规则

> [!warning] 扫描流程
> **输入采样 → 程序执行 → 输出刷新 → 通讯处理 → 下一轮扫描**

## ② 参考表

| 环节 | 作用 |
|---|---|
| **输入采样** | 读取按钮、传感器、限位 |
| **程序执行** | 执行梯形图/功能块逻辑 |
| **输出刷新** | 控制继电器、电磁阀、指示灯 |
| **通讯处理** | 与HMI、变频器、仪表通讯 |
| **诊断监控** | 检查运行、报警、错误状态 |
| **数据保持** | 保存计数、配方、参数 |

## ③ 计算/选型示例

- 已知：  
  按钮按下后，PLC控制电磁阀动作。
- 公式：  
  **动作响应 = 输入响应 + PLC扫描周期 + 输出响应**
- 计算：  
  若扫描周期较短，普通按钮控制通常能满足。
- 结论：  
  高速响应场景需额外确认输入滤波、扫描周期和输出响应时间。

> [!tip] 记忆口诀
> PLC一圈圈扫描，先看输入，再算逻辑，最后给输出。

# 03 小型PLC vs 中型PLC

> [!info] 核心结论
> 小型PLC适合简单单机，中型PLC适合点数更多、通讯更多、轴控更多的中大型设备。

## ① 分类/公式/规则

> [!warning] 选择规则
> **简单单机 → 小型PLC**  
> **多工位、多通讯、多轴、多扩展 → 中型PLC**

## ② 参考表

| 对比项 | 小型PLC | 中型PLC |
|---|---|---|
| 控制规模 | 小型单机 | 中型设备/产线 |
| 成本 | **较低** | 较高 |
| 扩展能力 | 有限 | 更强 |
| 通讯能力 | 基础为主 | 更丰富 |
| 程序容量 | 较小 | 较大 |
| 运动控制 | 简单定位 | 多轴更强 |

## ③ 计算/选型示例

- 已知：  
  设备只有20个I/O点、1台HMI、无伺服轴。
- 公式：  
  **少点数 + 简单逻辑 = 小型PLC**
- 计算：  
  控制需求不复杂，小型PLC即可满足。
- 结论：  
  可选择小型PLC，不必过度配置中型PLC。

> [!tip] 记忆口诀
> 单机少点用小型，产线多站看中型。

# 04 系统组成

> [!info] 核心结论
> 小型PLC通常集成CPU、电源、输入输出点和基础通讯接口，部分型号可扩展模块。

## ① 分类/公式/规则

> [!warning] 系统公式
> **小型PLC系统 = PLC本体 + 扩展I/O + HMI + 电源 + 传感器 + 执行器**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **CPU** | 执行程序 |
| **DI输入** | 接按钮、传感器、限位 |
| **DO输出** | 控制灯、继电器、电磁阀 |
| **AI/AO扩展** | 采集压力、温度或输出模拟量 |
| **通讯口** | 连接HMI、变频器、仪表 |
| **存储区** | 保存程序、参数和数据 |

## ③ 计算/选型示例

- 已知：  
  小型设备需要读取压力传感器0-10V信号。
- 公式：  
  **模拟量信号 → 需要AI模块或带AI的PLC**
- 计算：  
  普通DI不能读取连续电压值。
- 结论：  
  应选择带模拟量输入的小型PLC，或增加模拟量扩展模块。

> [!tip] 记忆口诀
> 开关量接DI/DO，连续量看AI/AO。

# 05 输入点DI

> [!info] 核心结论
> DI输入点用于读取按钮、开关、传感器、限位、到位信号，是PLC判断设备状态的入口。

## ① 分类/公式/规则

> [!warning] DI匹配规则
> **DI可用 = 电压等级 + PNP/NPN类型 + 公共端接法 + 输入响应匹配**

## ② 参考表

| 输入对象 | 说明 |
|---|---|
| **按钮开关** | 启动、停止、复位 |
| **接近开关** | 工件有无、机构到位 |
| **光电传感器** | 检测物体位置 |
| **磁性开关** | 气缸伸缩到位 |
| **限位开关** | 行程保护 |
| **压力开关** | 气压到位或低压报警 |

## ③ 计算/选型示例

- 已知：  
  现场传感器为PNP输出，小型PLC输入为NPN型。
- 公式：  
  **输入类型不匹配 → PLC可能读不到信号**
- 计算：  
  PNP/NPN输出逻辑不同，公共端接法也不同。
- 结论：  
  应选择匹配输入类型的PLC，或使用中间继电器/转换模块。

> [!tip] 记忆口诀
> 输入先看电压，再分PNP/NPN。

# 06 输出点DO

> [!info] 核心结论
> DO输出点用于控制电磁阀、继电器、接触器、指示灯等执行元件，重点看输出类型和电流容量。

## ① 分类/公式/规则

> [!warning] DO选型公式
> **输出可用 = 负载电压 + 负载电流 + 输出类型 + 公共端接法**

## ② 参考表

| 输出类型 | 特点 |
|---|---|
| **继电器输出** | 交直流都可，速度慢，有寿命 |
| **晶体管输出** | DC负载，高速，适合脉冲 |
| **晶闸管输出** | AC负载，部分场景使用 |
| **DO接继电器** | 适合隔离和扩流 |
| **DO接电磁阀** | 注意线圈电流和浪涌 |
| **高速输出** | 控制步进/伺服脉冲 |

## ③ 计算/选型示例

- 已知：  
  电磁阀线圈DC24V，电流0.15A；PLC单点输出最大0.5A。
- 公式：  
  **PLC输出电流 ≥ 负载电流**
- 计算：  
  0.5A ≥ 0.15A，单点电流满足。
- 结论：  
  可直接驱动，但多个输出同时动作时还需校核PLC公共端总电流。

> [!tip] 记忆口诀
> 输出先看负载电流，别让PLC小点位硬扛大负载。

# 07 继电器输出与晶体管输出

> [!info] 核心结论
> 小型PLC常见输出类型是继电器输出和晶体管输出，继电器通用，晶体管适合高速和直流控制。

## ① 分类/公式/规则

> [!warning] 选择规则
> **普通低速开关控制 → 继电器输出**  
> **高速脉冲/频繁动作/DC负载 → 晶体管输出**

## ② 参考表

| 对比项 | 继电器输出 | 晶体管输出 |
|---|---|---|
| 负载类型 | AC/DC都可 | 主要DC |
| 动作速度 | 较慢 | **快** |
| 寿命 | 机械触点寿命 | 无触点寿命长 |
| 脉冲输出 | 不适合高速 | **适合** |
| 抗冲击 | 触点需保护 | 需防过流 |
| 成本 | 常见 | 常见 |

## ③ 计算/选型示例

- 已知：  
  PLC需要输出高速脉冲控制步进驱动器。
- 公式：  
  **高速脉冲 → 晶体管输出**
- 计算：  
  继电器输出速度慢，不能用于高速脉冲。
- 结论：  
  应选择带高速晶体管输出的小型PLC。

> [!tip] 记忆口诀
> 继电器管通用，晶体管管高速。

# 08 模拟量扩展

> [!info] 核心结论
> 小型PLC本体模拟量能力有限，遇到压力、温度、流量、比例阀控制时常需要扩展AI/AO模块。

## ① 分类/公式/规则

> [!warning] 模拟量换算公式
> **实际值 = 数字量 ÷ 满量程数字值 × 工程量满量程**

## ② 参考表

| 模拟量类型 | 用途 |
|---|---|
| **0-10V输入** | 压力、位移、速度反馈 |
| **4-20mA输入** | 压力、流量、液位 |
| **热电偶输入** | 温度检测 |
| **热电阻输入** | 精密温度检测 |
| **0-10V输出** | 控制比例阀、变频器 |
| **4-20mA输出** | 工业模拟量控制 |

## ③ 计算/选型示例

- 已知：  
  0-10V压力传感器对应0-1.0MPa，PLC读到5V。
- 公式：  
  **压力 = 5 ÷ 10 × 1.0**
- 计算：  
  压力 = **0.5MPa**
- 结论：  
  当前压力约0.5MPa，实际需按模块量程和标定值换算。

> [!tip] 记忆口诀
> 开关量只分有无，模拟量要做换算。

# 09 通讯接口

> [!info] 核心结论
> 小型PLC常用通讯接口包括以太网、RS485、RS232、Modbus、CANopen等，主要用于连接HMI、变频器、仪表和扫码枪。

## ① 分类/公式/规则

> [!warning] 通讯公式
> **通讯成功 = 协议一致 + 地址正确 + 波特率一致 + 接线正确 + 数据格式一致**

## ② 参考表

| 接口/协议 | 常见用途 |
|---|---|
| **以太网** | HMI、上位机、远程维护 |
| **RS485** | 变频器、仪表、多站通讯 |
| **RS232** | 扫码枪、打印机、单设备通讯 |
| **Modbus RTU** | 仪表、变频器常见 |
| **Modbus TCP** | 以太网设备通讯 |
| **CANopen** | 部分运动和现场设备 |

## ③ 计算/选型示例

- 已知：  
  小型PLC需要连接HMI和2台变频器。
- 公式：  
  **通讯口数量和协议必须满足设备需求**
- 计算：  
  HMI可走以太网，变频器可走RS485 Modbus。
- 结论：  
  选PLC时应确认是否同时具备以太网和RS485，或是否支持扩展通讯模块。

> [!tip] 记忆口诀
> 通讯先看协议，再看接口；参数不同，连了也不通。

# 10 简单运动控制

> [!info] 核心结论
> 小型PLC可实现简单步进、伺服定位控制，但复杂多轴同步不宜勉强使用。

## ① 分类/公式/规则

> [!warning] 运动选择规则
> **单轴/少轴简单定位 → 小型PLC可用**  
> **多轴同步/插补/电子凸轮 → 中型PLC或运动控制器**

## ② 参考表

| 功能 | 小型PLC适用性 |
|---|---|
| **单轴点位** | 适合 |
| **步进脉冲控制** | 常见 |
| **伺服脉冲控制** | 可用，需高速输出 |
| **回原点** | 常见 |
| **多轴同步** | 需谨慎 |
| **插补轨迹** | 通常不适合 |
| **电子凸轮** | 通常不适合 |

## ③ 计算/选型示例

- 已知：  
  步进驱动器需要脉冲频率20kHz，PLC高速输出最高100kHz。
- 公式：  
  **PLC最高脉冲频率 ≥ 驱动器所需频率**
- 计算：  
  100kHz ≥ 20kHz，频率能力满足。
- 结论：  
  可用于该步进轴控制，但还需确认方向信号、回原点和加减速功能。

> [!tip] 记忆口诀
> 小型PLC能控轴，但别拿它硬做复杂运动。

# 11 HMI与参数设置

> [!info] 核心结论
> 小型PLC常与触摸屏HMI配合，用于显示状态、设置参数、手动调试和报警提示。

## ① 分类/公式/规则

> [!warning] HMI功能公式
> **HMI功能 = 状态显示 + 参数设定 + 手动操作 + 报警显示 + 配方切换**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **启动/停止** | 设备操作 |
| **手动模式** | 单动作调试 |
| **参数设置** | 时间、速度、计数 |
| **报警显示** | 提示故障原因 |
| **产量统计** | 记录数量 |
| **配方切换** | 多产品快速切换 |

## ③ 计算/选型示例

- 已知：  
  设备有3种产品，每种产品延时时间不同。
- 公式：  
  **多产品参数 → HMI配方**
- 计算：  
  通过HMI选择产品配方，PLC调用对应参数。
- 结论：  
  可在HMI中设计配方页面，减少人工改程序风险。

> [!tip] 记忆口诀
> PLC管控制，HMI管操作；参数放屏上，调试更方便。

# 12 程序设计

> [!info] 核心结论
> 小型PLC程序应保持简单清晰，重点做好手自动、互锁、报警、复位和状态步进。

## ① 分类/公式/规则

> [!warning] 程序结构
> **初始化 → 手动模式 → 自动流程 → 报警处理 → 复位逻辑 → 数据保存**

## ② 参考表

| 程序模块 | 作用 |
|---|---|
| **启动条件** | 判断设备是否允许运行 |
| **手动控制** | 调试单个动作 |
| **自动顺序** | 按步骤执行流程 |
| **互锁保护** | 防止错误动作 |
| **报警超时** | 动作不到位报警 |
| **复位逻辑** | 恢复初始状态 |
| **参数管理** | 保存时间、计数、配方 |

## ③ 计算/选型示例

- 已知：  
  气缸伸出后3秒内未收到到位信号。
- 公式：  
  **输出已发出 + 反馈未到位 + 时间超限 = 动作超时报警**
- 计算：  
  PLC应停止后续动作并提示伸出不到位。
- 结论：  
  自动程序必须加入超时报警，不能一直等待。

> [!tip] 记忆口诀
> 动作要有反馈，等待要有超时，报警要能复位。

# 13 安装与接线

> [!info] 核心结论
> 小型PLC安装要注意电源容量、公共端接法、接地、强弱电分离和端子标识。

## ① 分类/公式/规则

> [!warning] 接线规则
> **稳定运行 = 电源稳定 + COM正确 + 接地可靠 + 强弱电分开 + 标识清楚**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **24V电源** | 容量足够，纹波小 |
| **COM公共端** | 输入输出接法正确 |
| **接地PE** | 控制柜可靠接地 |
| **继电器负载** | 加浪涌吸收 |
| **模拟量线** | 屏蔽并远离动力线 |
| **端子编号** | 方便维护排查 |

## ③ 计算/选型示例

- 已知：  
  PLC输出点ON，但电磁阀不动作。
- 公式：  
  **程序ON ≠ 端子有电**
- 计算：  
  需要实测输出端子电压，并检查公共端和电磁阀线圈。
- 结论：  
  不要只看程序监控，必须结合现场接线和电压判断。

> [!tip] 记忆口诀
> PLC亮不代表设备动，端子有电才算真正输出。

# 14 常见故障排查

> [!info] 核心结论
> 小型PLC故障排查按 **电源、运行状态、输入、输出、通讯、程序、现场设备** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **PLC异常 = 电源异常 + 接线错误 + 输入不对 + 输出无效 + 通讯失败 + 程序互锁 + 现场故障**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **PLC不亮** | 无电源、保险损坏、电源故障 |
| **RUN不亮** | CPU停止、程序错误 |
| **输入无信号** | 传感器坏、接线错、PNP/NPN不匹配 |
| **输出不动作** | 公共端错、负载坏、输出点损坏 |
| **HMI连不上** | IP、站号、协议、线缆问题 |
| **自动不启动** | 条件不满足、报警未复位 |
| **动作不到位** | 气源、机构、传感器问题 |

## ③ 计算/选型示例

- 已知：  
  启动按钮按下后，PLC输入指示灯不亮。
- 公式：  
  **输入灯不亮 = 按钮/电源/接线/输入点异常**
- 计算：  
  先测按钮两端电压，再测PLC输入端电压。
- 结论：  
  若端子无电，查按钮和线路；若端子有电但输入不亮，查PLC输入类型或模块。

> [!tip] 记忆口诀
> 输入不亮查前端，输出不动查后端，通讯不通查参数。

# 15 小型PLC快速选型表

> [!info] 核心结论
> 小型PLC选型先统计I/O点数和输出类型，再确认通讯、模拟量、高速脉冲和扩展余量。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 统计DI输入点数  
> ② 统计DO输出点数  
> ③ 确认输出类型：继电器/晶体管  
> ④ 确认传感器PNP/NPN  
> ⑤ 确认是否需要AI/AO  
> ⑥ 确认是否需要高速计数/高速脉冲  
> ⑦ 确认通讯接口和协议  
> ⑧ 确认HMI连接方式  
> ⑨ 确认程序容量和扩展模块  
> ⑩ 预留10%~20%点数余量

> [!warning] 最终选型公式
> **推荐小型PLC = CPU点数 + 输出类型 + 通讯接口 + 模拟量扩展 + 高速功能 + 扩展余量**

## ③ 计算/选型示例

- 已知：  
  设备有14DI、10DO、2AI、1台HMI、1台变频器。
- 公式：  
  **PLC配置 ≥ 14DI + 10DO + 2AI + 通讯接口**
- 计算：  
  可选择24点或以上小型PLC，并增加2AI模块；HMI和变频器需确认通讯口是否够用。
- 结论：  
  推荐 **带通讯接口的小型PLC + 模拟量输入扩展模块**，并预留后期I/O余量。
