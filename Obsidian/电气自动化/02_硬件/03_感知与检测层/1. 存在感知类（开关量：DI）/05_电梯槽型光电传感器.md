# 01 电梯槽型光电开关总览

> [!info] 核心结论
> 电梯槽型光电开关通过检测遮光片是否进入U型槽，用于电梯 **平层、限位、位置确认、门区检测** 等场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 槽型光电开关把发射端和接收端集成在同一个U型结构内。  
> 当遮光片进入槽口时，光束被遮断，传感器输出状态发生变化。

> [!warning] 🟧 通用选型公式
> **槽型光电选型 = 槽宽 + 槽深 + 检测方式 + 输出类型 + 电源电压 + 安装方式 + 环境要求**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **传感器类型** | 槽型光电开关 |
| **典型结构** | U型槽结构 |
| **检测对象** | 遮光片、挡片、金属片、塑料片 |
| **常见应用** | 电梯平层、限位、门区、位置检测 |
| **输出类型** | PNP、NPN、继电器型等需校核 |
| **电源电压** | DC / AC 需按型号确认 |

## ③ 计算/选型示例

- 已知：  
  电梯机构中有一个遮光片，需要进入槽型光电开关后给控制系统一个到位信号。
- 公式：  
  **检测可靠性 = 槽宽余量 + 遮光面积 + 安装精度 + 输出匹配**
- 计算：  
  遮光片能稳定进入槽口，并能完全遮断发射端与接收端之间的光束。
- 结论：  
  可选 **电梯槽型光电开关**，具体槽宽、输出类型、电源电压、安装孔距需按厂家手册校核。

> [!tip] 记忆口诀
> 遮光片进槽，信号就变号；槽宽留余量，动作才稳当。

# 02 底层原理

> [!info] 核心结论
> 槽型光电开关的本质，是检测U型槽内的光束是否被遮挡。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> U型槽一侧是发光端，另一侧是接收端。  
> 正常情况下，接收端能收到光；当遮光片进入槽口，光被挡住，接收端光强下降，内部电路判断后输出ON/OFF信号。

> [!warning] 🟧 光电检测逻辑
> **发光端发光 → 接收端收光 → 遮光片进入 → 光束被挡 → 输出状态变化**

## ② 参考表

| 状态 | 光路情况 | 输出变化 |
|---|---|---|
| **无遮挡** | 接收端能收到光 | 按L-ON/D-ON逻辑输出 |
| **有遮挡** | 光束被遮断 | 输出状态翻转 |
| **遮挡不充分** | 光强变化不明显 | 可能误动作 |
| **槽内有灰尘** | 光强变弱 | 可能误判 |
| **安装偏移** | 遮光片未完全挡光 | 检测不稳定 |

## ③ 计算/选型示例

- 已知：  
  遮光片进入槽口后，偶尔PLC输入信号闪烁。
- 公式：  
  **信号稳定性 = 遮光充分 + 光轴稳定 + 安装牢固 + 无污染**
- 计算：  
  如果遮光片只挡住部分光束，或者安装位置抖动，接收光强会在阈值附近波动。
- 结论：  
  应调整遮光片位置，保证完全遮光，并检查槽口是否有灰尘或油污。

> [!tip] 记忆口诀
> 槽型看遮光，挡得越完整，信号越稳定。

# 03 电梯典型应用

> [!info] 核心结论
> 电梯槽型光电开关常用于判断轿厢、门机或机构部件是否到达指定位置。

## ① 分类/公式/规则

> [!note] 🟦 常见用途
> 电梯系统中，槽型光电开关常用于：  
> 平层检测、门区检测、限位检测、位置确认、速度或位置辅助判断。

> [!warning] 🟧 应用判断公式
> **适合槽型光电 = 有遮光片 + 运动路径固定 + 需要明确位置点信号**

## ② 参考表

| 应用位置 | 作用 |
|---|---|
| **平层位置** | 判断轿厢是否到达楼层基准位置 |
| **门区检测** | 判断轿厢是否处于允许开门区域 |
| **限位位置** | 判断机构是否到达极限位置 |
| **门机机构** | 判断门开到位、关到位 |
| **速度/位置辅助** | 配合控制系统判断运行状态 |

## ③ 计算/选型示例

- 已知：  
  电梯需要在轿厢到达楼层位置时，输出一个平层确认信号。
- 公式：  
  **平层信号 = 遮光片位置 + 光电开关安装位置 + 控制器判断逻辑**
- 计算：  
  当轿厢运行到指定位置，遮光片进入槽口，传感器输出信号给控制器。
- 结论：  
  槽型光电开关可用于平层检测，但安装位置和机械精度需按电梯设计规范校核。

> [!tip] 记忆口诀
> 电梯到没到位，看遮光片进没进槽。

# 04 槽宽与遮光片选型

> [!info] 核心结论
> 槽宽必须大于遮光片厚度和运动偏差，否则容易刮碰；但槽宽过大也可能影响检测稳定性。

## ① 分类/公式/规则

> [!note] 🟦 槽宽原则
> 遮光片进入槽口时，要既能顺利通过，又能可靠遮断光束。  
> 所以槽宽要考虑遮光片厚度、安装偏差、振动偏差和安全余量。

> [!warning] 🟧 槽宽选择公式
> **推荐槽宽 ≥ 遮光片厚度 + 2 × 最大偏移量 + 安全余量**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **遮光片厚度** | 实际进入槽口的材料厚度 |
| **最大偏移量** | 运行中左右晃动或安装误差 |
| **安全余量** | 防止刮碰和长期磨损 |
| **槽深** | 决定遮光片进入深度 |
| **遮光面积** | 决定能否充分挡光 |

## ③ 计算/选型示例

- 已知：  
  遮光片厚度为 **2mm**，机械偏移量为 **±1mm**，希望预留 **1mm** 安全余量。
- 公式：  
  **推荐槽宽 ≥ 2 + 2 × 1 + 1**
- 计算：  
  推荐槽宽 ≥ **5mm**
- 结论：  
  槽宽应不小于 **5mm**，实际还需结合传感器结构和厂家手册校核。

> [!tip] 记忆口诀
> 槽宽不够会刮片，槽宽太大要看遮光面。

# 05 输出类型与PLC接线

> [!info] 核心结论
> 槽型光电开关接PLC时，重点确认 **PNP/NPN、NO/NC、L-ON/D-ON、电源电压** 是否匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出逻辑
> PNP输出通常给PLC输入端提供正电压信号；  
> NPN输出通常把PLC输入端拉到0V；  
> L-ON表示接收到光时输出ON；  
> D-ON表示光被遮挡时输出ON。

> [!warning] 🟧 接线匹配公式
> **PLC输入正常 = 输出类型匹配 + 电源电压正确 + 公共端接法正确 + 程序逻辑一致**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **PNP** | 输出正电信号 |
| **NPN** | 输出0V信号 |
| **NO常开** | 条件成立后输出ON |
| **NC常闭** | 条件成立后输出OFF |
| **L-ON** | 亮通，收光时ON |
| **D-ON** | 暗通，遮光时ON |
| **DC24V** | 常见工业控制电压，需校核 |

## ③ 计算/选型示例

- 已知：  
  PLC输入模块要求PNP信号，希望遮光片进入槽口时PLC输入ON。
- 公式：  
  **遮光ON = PNP输出 + D-ON逻辑**
- 计算：  
  遮光片进入后光被挡住，属于暗状态；需要暗通输出。
- 结论：  
  可选择 **PNP输出、D-ON逻辑** 的槽型光电开关，具体接线按PLC和传感器手册校核。

> [!tip] 记忆口诀
> 遮光要ON选暗通，收光要ON选亮通；接PLC先分PNP/NPN。

# 06 安装与调试

> [!info] 核心结论
> 电梯槽型光电开关安装时，重点是固定牢靠、遮光片居中、避免刮碰、信号稳定。

## ① 分类/公式/规则

> [!note] 🟦 安装步骤
> ① 固定传感器本体  
> ② 调整遮光片进入方向  
> ③ 保证遮光片居中通过槽口  
> ④ 检查进入深度  
> ⑤ 慢速运行观察信号  
> ⑥ 再进行正常速度验证

> [!warning] 🟧 安装可靠公式
> **可靠安装 = 传感器固定 + 遮光片居中 + 进入深度足够 + 无机械碰撞 + 信号无抖动**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **安装方向** | 遮光片能顺利进入槽口 |
| **槽口间隙** | 不刮碰、不偏斜 |
| **固定强度** | 防止振动后偏移 |
| **线缆固定** | 避免拉扯和磨损 |
| **指示灯** | 用于判断输出状态 |
| **运行验证** | 慢速、正常速度都要测试 |

## ③ 计算/选型示例

- 已知：  
  慢速调试时信号正常，但正常运行时偶尔丢信号。
- 公式：  
  **丢信号原因 = 机械抖动 + 遮光时间短 + 响应时间不足 + 安装偏移**
- 计算：  
  正常速度下遮光片经过时间变短，若安装偏差大或响应慢，可能漏检。
- 结论：  
  应检查响应时间、遮光片宽度、进入深度和安装牢固度。

> [!tip] 记忆口诀
> 先慢速看位置，再高速看稳定。

# 07 常见故障排查

> [!info] 核心结论
> 槽型光电开关故障，多数来自遮光不充分、接线错误、安装偏移、污染或输出逻辑设置错误。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看电源和指示灯，再看遮光片位置，然后查PNP/NPN和PLC输入状态。

> [!warning] 🟧 排错公式
> **检测异常 = 电源异常 + 遮光不足 + 接线错误 + 输出逻辑错误 + 槽口污染 + 安装松动**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无输出** | 电源未接、线接错、传感器损坏 |
| **一直有信号** | 槽口被挡、D-ON/L-ON逻辑不符 |
| **信号闪烁** | 遮光片边缘抖动、安装松动 |
| **PLC无输入** | PNP/NPN不匹配或公共端错误 |
| **偶发误动作** | 灰尘油污、振动、光路污染 |
| **机械刮碰** | 槽宽不足或遮光片偏移 |

## ③ 计算/选型示例

- 已知：  
  传感器指示灯变化正常，但PLC输入没有变化。
- 公式：  
  **PLC无信号 = 传感器输出正常但接线/输入模块异常**
- 计算：  
  传感器本体动作正常，问题更可能在PNP/NPN匹配、公共端接线或PLC输入地址。
- 结论：  
  应检查输出类型、PLC输入模块公共端和程序读取地址。

> [!tip] 记忆口诀
> 灯变PLC不变，先查接线和类型。

# 08 快速选型表

> [!info] 核心结论
> 电梯槽型光电开关选型，先看槽口尺寸和遮光片，再看输出类型、电源和安装环境。

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看遮光片厚度  
> ② 看运动偏移量  
> ③ 看槽宽槽深  
> ④ 看输出类型  
> ⑤ 看电源电压  
> ⑥ 看安装孔位和环境

> [!warning] 🟧 最终选型公式
> **推荐型号 = 槽宽 + 槽深 + PNP/NPN + NO/NC或L-ON/D-ON + 电源电压 + 安装尺寸**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 电梯平层检测 | **槽型光电开关** |
| 遮光片检测 | 槽宽需留余量 |
| 遮光时输出ON | D-ON逻辑 |
| 收光时输出ON | L-ON逻辑 |
| PLC输入为PNP | 选PNP输出 |
| PLC输入为NPN | 选NPN输出 |
| 振动环境 | 固定强度和防松要校核 |

## ③ 计算/选型示例

- 已知：  
  遮光片厚度 **3mm**，最大偏移量 **±1mm**，PLC要求PNP输入，遮光时需要ON。
- 公式：  
  **型号需求 = 槽宽≥3+2×1+安全余量 + PNP + D-ON**
- 计算：  
  槽宽至少大于 **5mm**，再增加安全余量更稳妥。
- 结论：  
  可选 **槽宽满足余量、PNP输出、D-ON逻辑** 的电梯槽型光电开关，具体型号需按厂家手册校核。

> [!tip] 选型口诀
> 先量遮光片，再定槽宽；再看PNP/NPN，最后确认亮通暗通。
