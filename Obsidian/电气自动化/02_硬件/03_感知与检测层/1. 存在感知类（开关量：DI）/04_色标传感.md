# 01 色标传感器总览

> [!info] 核心结论
> 色标传感器用于检测包装、标签、薄膜、纸张上的 **色标、印刷标记、定位线**，常用于高速定位和切割对位。

## ① 分类/公式/规则

> [!note] 🟦 典型用途
> 色标传感器主要不是检测“有没有物体”，而是检测 **目标表面颜色或明暗差异是否变化**。

> [!warning] 🟧 通用选型公式
> **色标传感器选型 = 色标颜色 + 背景颜色 + 检测距离 + 光斑大小 + 响应速度 + 输出类型**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **CL系列** | 色标检测传感器 |
| **检测对象** | 色标、印刷线、标签标记 |
| **典型行业** | 包装、印刷、贴标、制袋 |
| **核心参数** | 响应时间、光斑、距离、重复精度 |
| **输出类型** | PNP、NPN、NO/NC |
| **调试方式** | 示教、阈值设定、按键调节 |

## ③ 计算/选型示例

- 已知：  
  包装膜上有黑色定位色标，背景为浅色，需要控制切刀位置。
- 公式：  
  **检测可靠性 = 色标与背景反差 + 光斑稳定 + 响应速度满足**
- 计算：  
  黑色标记与浅色背景反差明显，适合色标检测。
- 结论：  
  可优先选择 **CL系列色标传感器**，具体响应速度、检测距离和输出方式需按厂家手册校核。

> [!tip] 记忆口诀
> 色标不是看物体，是看颜色和背景差异。

# 02 底层原理

> [!info] 核心结论
> 色标传感器的本质，是通过发光照射目标，再分析反射光强或颜色差异，判断是否遇到色标。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 传感器发出特定光源照射包装膜或标签表面。  
> 不同颜色反射光强不同，接收器检测反射光变化，内部电路与设定阈值比较后输出信号。

> [!warning] 🟧 色标检测逻辑
> **发光 → 照射色标/背景 → 反射光变化 → 阈值比较 → 输出ON/OFF**

## ② 参考表

| 项目 | 原理说明 |
|---|---|
| **发光端** | 发出白光、红光、绿光或蓝光 |
| **接收端** | 接收目标表面反射光 |
| **色标差异** | 不同颜色反射率不同 |
| **阈值判断** | 反射信号超过或低于设定值 |
| **输出信号** | 给PLC或控制器定位使用 |

## ③ 计算/选型示例

- 已知：  
  背景反射值为 **800**，色标反射值为 **250**。
- 公式：  
  **推荐阈值 ≈（背景反射值 + 色标反射值）÷ 2**
- 计算：  
  阈值 ≈（800 + 250）÷ 2 = **525**
- 结论：  
  可将判断阈值设置在 **525附近**，再结合现场速度、抖动和材质变化进行微调。

> [!tip] 记忆口诀
> 背景一个值，色标一个值，中间设阈值。

# 03 CL系列色标传感器

> [!info] 核心结论
> CL系列色标传感器适合包装机、制袋机、贴标机等需要高速识别色标的位置控制场景。

## ① 分类/公式/规则

> [!note] 🟦 定位
> CL系列主要用于识别连续材料上的色标，帮助设备完成 **定位、切断、封口、贴标、计数** 等动作。

> [!warning] 🟧 CL系列适用公式
> **适合CL系列 = 有明显色标 + 背景稳定 + 需要高速定位 + 安装距离可控**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **检测对象** | 包装膜、标签、纸张、印刷品 |
| **检测内容** | 色标、色线、印刷定位点 |
| **应用设备** | 包装机、贴标机、制袋机 |
| **关键参数** | 响应时间、重复精度、光斑尺寸 |
| **调试方式** | 示教背景和色标 |
| **注意** | 需现场测试颜色反差 |

## ③ 计算/选型示例

- 已知：  
  包装机运行速度较快，需要在色标经过时及时输出切刀信号。
- 公式：  
  **定位可靠性 = 响应时间 + 色标宽度 + 材料速度**
- 计算：  
  材料速度越快，色标经过传感器的时间越短，对响应时间要求越高。
- 结论：  
  应选择响应速度满足设备节拍的 **CL系列色标传感器**，并按厂家手册校核响应时间。

> [!tip] 记忆口诀
> 包装定位看色标，色标检测看CL。

# 04 颜色与背景差异

> [!info] 核心结论
> 色标传感器能否稳定检测，关键看 **色标和背景之间的反射差异是否足够大**。

## ① 分类/公式/规则

> [!note] 🟦 反差判断
> 色标颜色和背景颜色越接近，越难检测；  
> 颜色差异越明显，检测越稳定。

> [!warning] 🟧 反差判断公式
> **检测余量 = |背景反射值 - 色标反射值|**

## ② 参考表

| 色标与背景 | 检测难度 |
|---|---|
| **黑标 + 白底** | 容易 |
| **深色标 + 浅色底** | 较容易 |
| **浅色标 + 深色底** | 较容易 |
| **相近颜色** | 较难 |
| **透明膜背景** | 需现场测试 |
| **高反光材料** | 需注意角度和误反射 |

## ③ 计算/选型示例

- 已知：  
  背景反射值为 **700**，色标反射值为 **620**。
- 公式：  
  **检测余量 = |背景反射值 - 色标反射值|**
- 计算：  
  检测余量 = |700 - 620| = **80**
- 结论：  
  差异较小，检测可能不稳定；建议更换色标颜色、调整光源模式或现场重新示教。

> [!tip] 记忆口诀
> 色差越大越稳定，颜色越近越难分。

# 05 示教与阈值设置

> [!info] 核心结论
> 色标传感器调试的核心，是让传感器分别记住 **背景值** 和 **色标值**，再设置合适判断阈值。

## ① 分类/公式/规则

> [!note] 🟦 示教步骤
> ① 对准背景示教  
> ② 对准色标示教  
> ③ 自动或手动生成阈值  
> ④ 运行材料测试  
> ⑤ 微调灵敏度或输出逻辑

> [!warning] 🟧 阈值设置公式
> **推荐阈值 ≈（背景值 + 色标值）÷ 2**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **背景值** | 非色标区域反射值 |
| **色标值** | 色标区域反射值 |
| **阈值** | 判断色标/背景的分界点 |
| **灵敏度** | 决定检测变化的敏感程度 |
| **L-ON/D-ON** | 决定亮时ON还是暗时ON |
| **响应时间** | 决定高速检测能力 |

## ③ 计算/选型示例

- 已知：  
  背景值 **900**，色标值 **300**。
- 公式：  
  **阈值 =（900 + 300）÷ 2**
- 计算：  
  阈值 = **600**
- 结论：  
  可先将阈值设为 **600**，再观察高速运行时是否有漏检或误检。

> [!tip] 记忆口诀
> 先教背景，再教色标，中间阈值最可靠。

# 06 安装距离与光斑

> [!info] 核心结论
> 色标传感器安装时，要保证检测距离稳定，并让光斑完整落在色标区域内。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 色标检测不是光斑越大越好。  
> 光斑过大可能同时照到色标和背景，导致反射值混合，判断不稳定。

> [!warning] 🟧 光斑匹配公式
> **光斑尺寸 ≤ 色标有效宽度**

## ② 参考表

| 安装项 | 检查重点 |
|---|---|
| **检测距离** | 保持在额定范围内 |
| **光斑大小** | 不宜超过色标宽度 |
| **安装角度** | 避免强反光直射接收端 |
| **材料抖动** | 会导致信号波动 |
| **传感器固定** | 防止振动造成偏移 |
| **线缆方向** | 避免拉扯和干扰 |

## ③ 计算/选型示例

- 已知：  
  色标宽度为 **5mm**，传感器光斑直径为 **8mm**。
- 公式：  
  **光斑尺寸 ≤ 色标有效宽度**
- 计算：  
  8mm ＞ 5mm，光斑大于色标宽度。
- 结论：  
  可能同时照到色标和背景，建议选择更小光斑型号或增大色标宽度。

> [!tip] 记忆口诀
> 光斑要小于色标，检测才不跑偏。

# 07 输出类型与PLC接线

> [!info] 核心结论
> 色标传感器接PLC时，要确认 **PNP/NPN、NO/NC、L-ON/D-ON、电源电压** 与PLC输入一致。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> PNP输出一般给PLC输入提供正电压信号；  
> NPN输出一般把PLC输入拉到0V；  
> L-ON/D-ON决定亮区或暗区触发输出。

> [!warning] 🟧 接线匹配公式
> **PLC输入正常 = 传感器输出类型 + PLC输入公共端 + 电源极性 + 输出逻辑**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **PNP** | 输出正电信号 |
| **NPN** | 输出0V信号 |
| **NO常开** | 检测条件成立后ON |
| **NC常闭** | 检测条件成立后OFF |
| **L-ON** | 亮通，反射光强时ON |
| **D-ON** | 暗通，反射光弱时ON |
| **DC24V** | 工业现场常见供电，需校核 |

## ③ 计算/选型示例

- 已知：  
  黑色标记比白色背景反射光弱，PLC希望检测到黑标时输入ON。
- 公式：  
  **黑标ON = 暗区触发输出**
- 计算：  
  黑标反射弱，属于暗区；应选择或设置为 **D-ON**。
- 结论：  
  可将输出逻辑设置为 **D-ON**，并确认PNP/NPN与PLC输入模块匹配。

> [!tip] 记忆口诀
> 黑标多用暗通，白标多用亮通；接PLC先看PNP/NPN。

# 08 常见问题排查

> [!info] 核心结论
> 色标检测不稳定，多数原因是颜色反差不足、光斑过大、材料抖动、距离变化或阈值设置不合理。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看色差，再看光斑；  
> 先看安装，再看阈值；  
> 先低速调稳，再高速验证。

> [!warning] 🟧 排错公式
> **检测异常 = 色差不足 + 光斑不合适 + 距离变化 + 材料抖动 + 参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **漏检色标** | 色差小、响应慢、阈值不合适 |
| **误检背景** | 反光干扰、灵敏度过高 |
| **高速时不稳定** | 响应时间不够或材料抖动 |
| **换卷后异常** | 印刷颜色或背景变化 |
| **检测透明膜不稳** | 反射弱或背景干扰 |
| **PLC无输入** | PNP/NPN或接线错误 |

## ③ 计算/选型示例

- 已知：  
  低速运行正常，高速运行时偶尔漏检色标。
- 公式：  
  **高速检测可靠性 = 色标经过时间 - 传感器响应时间**
- 计算：  
  速度越快，色标停留在光斑下的时间越短。
- 结论：  
  应校核传感器响应时间，必要时增大色标宽度、优化安装位置或选择更高速型号。

> [!tip] 记忆口诀
> 低速能检不算稳，高速不漏才算准。

# 09 色标传感器快速选型表

> [!info] 核心结论

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看色标颜色  
> ② 看背景颜色  
> ③ 看材料速度  
> ④ 看色标宽度  
> ⑤ 看安装距离  
> ⑥ 看输出类型

> [!warning] 🟧 最终选型公式
> **推荐型号 = 色标反差 + 光斑尺寸 + 响应时间 + 检测距离 + PNP/NPN + L-ON/D-ON**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 包装膜色标检测 | **CL系列色标传感器** |
| 印刷定位检测 | **CL系列色标传感器** |
| 高速包装机 | 校核响应时间 |
| 小色标检测 | 校核光斑尺寸 |
| 黑标白底 | D-ON常用 |
| 白标深底 | L-ON常用 |
| 接PLC输入 | 校核PNP/NPN |

## ③ 计算/选型示例

- 已知：  
  色标宽度 **4mm**，材料高速运行，PLC输入要求PNP，检测黑色标记。

  **型号需求 = 小光斑 + 高速响应 + PNP输出 + D-ON逻辑**
- 计算：  
  光斑应小于4mm，响应时间需满足材料速度，黑标一般适合暗通逻辑。
- 结论：  
  可选择 **CL系列色标传感器**，重点校核光斑尺寸、响应时间、PNP输出和D-ON设置。

> [!tip] 选型口诀
> 色差先看清，光斑要合适；高速看响应，接线看输出。
