# 01 光电传感器总览

> [!info] 核心结论
> 光电传感器通过 **发光端发射光线、接收端检测光变化** 来判断物体是否存在。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 标准安装看 **PS系列标准方形**；  
> 小空间安装看 **PSB超薄型 / PSM迷你方形**；  
> 物体通过缝隙检测看 **SV / SW / PSE槽型**；  
> 圆孔或螺纹安装看 **PSPC / PSMC圆柱形**。

> [!warning] 🟧 通用选型公式
> **光电选型 = 检测方式 + 检测距离 + 物体颜色/材质 + 安装空间 + 输出类型 + 环境条件**

## ② 参考表

| 系列 | 主要定位 |
|---|---|
| **PS系列** | 标准方形光电传感器 |
| **SV / SW / PSE系列** | 槽型光电传感器 |
| **PSB系列** | 超薄型光电传感器 |
| **PSM系列** | 迷你方形光电传感器 |
| **PSPC / PSMC系列** | 圆柱形光电传感器 |

## ③ 计算/选型示例

- 已知：  
  需要检测输送线上是否有包装盒经过。
- 公式：  
  **传感器需求 = 检测距离 + 物体材质 + 安装空间 + PLC输入类型**
- 计算：  
  包装盒体积较大，安装空间充足，可用标准方形光电。
- 结论：  
  可优先考虑 **PS系列标准方形光电传感器**，具体距离、输出、电源、防护等级需按厂家手册校核。

> [!tip] 记忆口诀
> 有空间用标准，小空间用迷你，过槽检测用槽型，螺纹安装用圆柱。

# 02 底层原理

> [!info] 核心结论
> 光电传感器的本质，是把 **光线是否被遮挡、反射或接收** 转换成电信号。

## ① 分类/公式/规则

> [!note] 🟦 基本原理
> 发射器发出红光、红外光或激光，接收器检测返回光或透过光。  
> 当物体进入检测区域，光强发生变化，内部电路判断后输出ON/OFF信号。

> [!warning] 🟧 光电检测逻辑
> **发光 → 光路变化 → 接收光强变化 → 电路判断 → 输出信号**

## ② 参考表

| 检测方式 | 光路变化 |
|---|---|
| **对射型** | 物体遮断发射端到接收端的光 |
| **镜反射型** | 物体遮断反光板返回的光 |
| **漫反射型** | 物体把光反射回传感器 |
| **槽型** | 物体进入槽口遮断光束 |
| **背景抑制型** | 通过距离差异判断目标 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个薄片是否通过固定位置。
- 公式：  
  **检测可靠性 = 光束稳定 + 遮光明显 + 安装位置固定**
- 计算：  
  薄片通过槽口时能稳定遮断光束。
- 结论：  
  优先选 **槽型光电传感器**，安装间隙和响应时间需按厂家手册校核。

> [!tip] 记忆口诀
> 光被挡、光被反、光变弱，传感器就能判断。

# 03 光电检测方式

> [!info] 核心结论
> 光电传感器选型时，先确定检测方式，再确定外形系列。

## ① 分类/公式/规则

> [!note] 🟦 三大常用方式
> **对射型**：距离远、最稳定，但需要发射端和接收端对准。  
> **镜反射型**：安装较方便，需要反光板。  
> **漫反射型**：只需一个传感器，但受颜色、材质、表面影响更大。

> [!warning] 🟧 检测方式选择公式
> **检测方式 = 距离要求 + 安装条件 + 物体反光特性 + 稳定性要求**

## ② 参考表

| 检测方式 | 优点 | 注意 |
|---|---|---|
| **对射型** | 距离远、稳定 | 需两端安装对准 |
| **镜反射型** | 接线少、距离较远 | 需反光板 |
| **漫反射型** | 安装简单 | 受颜色和材质影响 |
| **槽型** | 定位准、调试简单 | 受槽宽限制 |
| **背景抑制型** | 可减少背景误检 | 距离需精确校核 |

## ③ 计算/选型示例

- 已知：  
  检测距离较远，现场两侧都能安装传感器。
- 公式：  
  **远距离稳定检测 = 对射安装 + 光轴对准 + 抗干扰校核**
- 计算：  
  对射型光路明确，抗物体颜色干扰能力较好。
- 结论：  
  优先考虑 **对射型光电传感器**，具体有效距离需按厂家手册校核。

> [!tip] 记忆口诀
> 距离远用对射，安装省用反射，简单检测用漫反射。

# 04 PS系列标准方形光电传感器

> [!info] 核心结论
> PS系列标准方形光电传感器适合常规设备检测，是应用最广的一类通用光电传感器。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 标准方形结构安装方便、适用范围广，常用于输送线、包装机、装配线、物料有无检测。

> [!warning] 🟧 PS系列判断公式
> **适合PS系列 = 安装空间充足 + 检测对象常规 + 需要稳定通用检测**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 外形 | **标准方形** |
| 场景 | 输送、包装、装配、分拣 |
| 检测方式 | 对射、反射、漫反射等需按型号确认 |
| 输出 | PNP、NPN、NO/NC |
| 电源 | 常见DC 24V，需按型号校核 |
| 安装 | 支架安装、螺钉固定 |

## ③ 计算/选型示例

- 已知：  
  需要检测输送线上有无纸箱，安装空间充足。
- 公式：  
  **标准检测 = 常规距离 + 常规物体 + 稳定安装**
- 计算：  
  纸箱尺寸较大，检测位置固定，标准方形结构便于调节。
- 结论：  
  可优先选择 **PS系列标准方形光电传感器**，检测距离和检测方式需按实际工况校核。

> [!tip] 记忆口诀
> 常规检测不纠结，标准方形先考虑。

# 05 SV / SW / PSE系列槽型光电传感器

> [!info] 核心结论
> 槽型光电传感器适合检测薄片、标签、齿轮片、挡片等通过槽口的物体。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 槽型光电的发射端和接收端固定在同一个U型槽结构中，物体进入槽口后遮断光束，检测稳定、调试简单。

> [!warning] 🟧 槽型判断公式
> **适合槽型 = 物体可穿过槽口 + 位置固定 + 需要精准边缘检测**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 外形 | **U型 / 槽型** |
| 检测对象 | 薄片、标签、挡片、齿轮 |
| 优点 | 光轴固定，不易跑偏 |
| 重点参数 | 槽宽、槽深、响应时间 |
| 安装 | 被测物需能进入槽口 |
| 注意 | 槽宽必须大于物体厚度和偏移量 |

## ③ 计算/选型示例

- 已知：  
  检测一个厚度 **2mm** 的金属挡片，安装偏移可能为 **±1mm**。
- 公式：  
  **推荐槽宽 ≥ 物体厚度 + 2 × 偏移量 + 安全余量**
- 计算：  
  2mm + 2 × 1mm = **4mm**，再加安全余量。
- 结论：  
  槽宽建议大于 **4mm**，具体型号需按厂家手册和机械偏差校核。

> [!tip] 记忆口诀
> 进槽就检测，槽宽要留余量。

# 06 PSB系列超薄型光电传感器

> [!info] 核心结论
> PSB系列超薄型光电传感器适合安装空间很小、设备结构紧凑的检测点。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 超薄型光电适合安装在狭窄位置，例如小型机构、紧凑治具、夹具侧边、设备内部空间受限区域。

> [!warning] 🟧 超薄型判断公式
> **适合PSB = 空间狭窄 + 安装厚度受限 + 检测距离较短**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 外形 | **超薄型** |
| 优点 | 节省安装空间 |
| 场景 | 夹具、治具、小型设备 |
| 检测距离 | 通常需重点校核 |
| 安装 | 注意固定强度和线缆弯折 |
| 风险 | 空间小，调节余量也小 |

## ③ 计算/选型示例

- 已知：  
  设备内部只剩很薄的安装空间，需要检测小零件到位。
- 公式：  
  **可安装性 = 传感器厚度 + 接线空间 + 调节空间 ≤ 设备可用空间**
- 计算：  
  如果普通方形传感器装不下，应选择超薄型结构。
- 结论：  
  可评估 **PSB系列超薄型光电传感器**，检测距离、安装间隙和线缆出线方向需按厂家手册校核。

> [!tip] 记忆口诀
> 空间薄、位置窄，超薄光电来安排。

# 07 PSM系列迷你方形光电传感器

> [!info] 核心结论
> PSM系列迷你方形光电传感器适合小型设备、小物体检测和安装空间有限的场景。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 迷你方形比标准方形更小，适合小型自动化设备、电子装配、轻工机械和精密机构检测。

> [!warning] 🟧 迷你型判断公式
> **适合PSM = 小空间 + 小目标 + 中短距离检测 + 需要方形安装**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 外形 | **迷你方形** |
| 场景 | 小型设备、电子装配 |
| 优点 | 体积小、安装灵活 |
| 目标物 | 小零件、小包装、小挡片 |
| 输出 | PNP/NPN、NO/NC需校核 |
| 注意 | 小体积不等于远距离 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个小塑料件是否到位，安装位置靠近机械手夹爪。
- 公式：  
  **小空间检测 = 小体积传感器 + 合适检测距离 + 抗误触发安装**
- 计算：  
  标准传感器体积偏大，迷你方形更容易布置在夹爪附近。
- 结论：  
  可选 **PSM系列迷你方形光电传感器**，但需现场校核目标颜色、反光性和安装角度。

> [!tip] 记忆口诀
> 设备小、目标小，迷你方形刚刚好。

# 08 PSPC / PSMC系列圆柱形光电传感器

> 圆柱形光电传感器适合孔位安装、螺纹固定和需要轴向检测的场景。

> [!warning] 🟧 圆柱型判断公式
> **适合圆柱型 = 需要孔安装 + 螺纹固定 + 轴向检测 + 安装位置明确**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 外形 | **圆柱形** |
| 安装 | 螺纹孔、支架孔、螺母固定 |
| 场景 | 到位检测、物料检测、狭窄安装 |
| 重点参数 | 直径、长度、检测距离 |
| 接线 | 电缆式、接插件式需校核 |
| 注意 | 安装角度影响检测稳定性 |

## ③ 计算/选型示例

- 已知：  
  设备支架已有圆孔，需要安装一个光电传感器检测零件。
- 公式：  
  **圆柱安装匹配 = 安装孔径 + 传感器直径 + 锁紧空间**
- 计算：  
  若孔径与传感器直径匹配，圆柱形结构可直接安装并锁紧。
- 结论：  
  可选择 **PSPC / PSMC系列圆柱形光电传感器**，外径和检测距离需按机械图纸及厂家手册校核。

> [!tip] 记忆口诀
> 有孔位、要锁紧，圆柱光电更省心。

# 09 输出类型与PLC接线

> [!info] 核心结论
> 光电传感器接PLC时，必须确认 **PNP/NPN、NO/NC、L-ON/D-ON、电源电压** 是否匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> PNP输出一般给PLC输入提供正电压信号；  
> NPN输出一般把PLC输入拉到0V；  
> NO/NC或L-ON/D-ON决定检测到物体时输出状态。

> [!warning] 🟧 接线匹配公式
> **PLC输入正常 = 传感器输出类型 + PLC输入公共端 + 电源极性 + 程序逻辑**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **PNP** | 常见于PLC源型输入场景 |
| **NPN** | 常见于PLC漏型输入场景 |
| **NO常开** | 检测到物体后输出ON |
| **NC常闭** | 未检测到物体时输出ON |
| **L-ON** | 亮通，接收到光时ON |
| **D-ON** | 暗通，光被遮挡时ON |
| **DC24V** | 工业传感器常见供电，需校核 |

## ③ 计算/选型示例

- 已知：  
  PLC输入模块要求PNP信号，现场光电传感器为NPN输出。
- 公式：  
  **输入有效 = 输出类型匹配 + 接线方式匹配**
- 计算：  
  NPN与PNP输入要求不匹配，PLC可能无法正确识别信号。
- 结论：  
  应更换为 **PNP输出光电传感器**，或按PLC模块手册调整接线方案。

> [!tip] 记忆口诀
> PNP/NPN先看清，接错PLC没反应。

# 10 安装与抗干扰

> [!info] 核心结论
> 光电传感器检测不稳定，多数不是传感器坏了，而是安装角度、背景反光、距离或干扰没处理好。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 光电传感器要注意光轴对准、检测距离、背景反光、强光干扰、目标颜色、线缆走向和固定强度。

> [!warning] 🟧 稳定检测公式
> **稳定检测 = 合理距离 + 正确角度 + 避免背景反光 + 光轴对准 + 输出抗干扰**

## ② 参考表

| 问题 | 处理方法 |
|---|---|
| **检测忽有忽无** | 调整距离和灵敏度 |
| **背景误检** | 避开反光背景或用背景抑制型 |
| **阳光干扰** | 避免直射，增加遮光 |
| **目标颜色变化** | 现场测试深色、浅色样品 |
| **对射不稳定** | 重新对准发射端和接收端 |
| **线缆干扰** | 远离动力线和变频器线 |

## ③ 计算/选型示例

- 已知：  
  漫反射光电检测白色盒子正常，检测黑色盒子不稳定。
- 公式：  
  **反射信号 = 目标反光能力 + 距离 + 入射角**
- 计算：  
  黑色目标反射光弱，接收端信号可能低于判断阈值。
- 结论：  
  可缩短检测距离、调整角度、提高灵敏度，或改用对射型/镜反射型方案。

> [!tip] 记忆口诀
> 光电怕乱光，安装先对光；颜色会影响，现场要试样。

# 11 光电传感器快速选型表

> [!info] 核心结论
> 光电选型先看检测对象和安装空间，再看检测方式、输出类型和环境条件。

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看检测对象  
> ② 看检测距离  
> ③ 看安装空间  
> ④ 看检测方式  
> ⑤ 看输出类型  
> ⑥ 看环境干扰

> [!warning] 🟧 最终选型公式
> **推荐型号 = 系列外形 + 检测方式 + 检测距离 + PNP/NPN + NO/NC + 电源 + 防护等级**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 常规设备检测 | **PS标准方形** |
| 狭窄空间检测 | **PSB超薄型** |
| 小型设备检测 | **PSM迷你方形** |
| 薄片/挡片通过检测 | **SV / SW / PSE槽型** |
| 孔位螺纹安装 | **PSPC / PSMC圆柱形** |
| 远距离稳定检测 | 对射型方案 |
| 安装简单检测 | 漫反射或镜反射方案 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个小型零件，安装空间有限，PLC输入要求PNP，使用DC24V。
- 公式：  
  **型号需求 = 小型外形 + 合适距离 + PNP + DC24V + 稳定检测**
- 计算：  
  小空间优先考虑PSM迷你方形或PSB超薄型，具体看安装厚度和检测方向。
- 结论：  
  可优先评估 **PSM系列迷你方形** 或 **PSB系列超薄型**，最终型号需按检测距离、目标颜色、输出类型和厂家手册校核。

> [!tip] 选型口诀
> 常规PS，小型PSM，超薄PSB；过槽选槽型，孔装选圆柱。
