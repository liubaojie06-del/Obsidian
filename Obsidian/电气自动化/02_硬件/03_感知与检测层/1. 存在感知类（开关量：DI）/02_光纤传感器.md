# 01 光纤开关总览

> [!info] 核心结论
> 光纤开关适合 **小物体、狭小空间、高精度、远离电气干扰区域** 的检测场景。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 光纤开关通常由 **光纤放大器 + 光纤头** 组成。  
> 放大器负责发光、接收、判断和输出；  
> 光纤头负责把光引到检测位置。

> [!warning] 🟧 通用选型公式
> **光纤开关选型 = 放大器类型 + 光纤头形式 + 检测方式 + 检测距离 + 输出类型 + 环境条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **FA/FC系列** | 光纤传感器 / 光纤放大器系列 |
| **光纤放大器** | 判断光强变化并输出信号 |
| **光纤头** | 将光线引到检测点 |
| **检测方式** | 对射型、反射型 |
| **典型优势** | 小型化、抗干扰、安装灵活 |
| **常见输出** | PNP、NPN、NO/NC |

## ③ 计算/选型示例

- 已知：  
  需要检测一个很小的零件，普通光电传感器安装空间不足。
- 公式：  
  **光纤适用性 = 小空间 + 小目标 + 精准检测 + 放大器远离现场**
- 计算：  
  光纤头体积小，可安装在狭窄检测位置；放大器可安装在控制柜或安全位置。
- 结论：  
  可优先考虑 **FA/FC系列光纤传感器**，具体光纤头、检测距离、输出方式需按厂家手册校核。

> [!tip] 记忆口诀
> 空间小、目标小、位置刁，光纤开关最可靠。

# 02 底层原理

> [!info] 核心结论
> 光纤开关的本质，是通过光纤把光送到检测点，再根据 **光强变化** 判断物体是否存在。

## ① 分类/公式/规则

> [!note] 🟦 基本原理
> 放大器发出光，经光纤传到检测端。  
> 当物体进入检测区域后，光被遮挡或反射发生变化，放大器检测到光强变化后输出ON/OFF信号。

> [!warning] 🟧 光纤检测逻辑
> **发光 → 光纤传输 → 目标改变光强 → 放大器判断 → 输出信号**

## ② 参考表

| 检测方式 | 原理 |
|---|---|
| **对射型光纤** | 物体遮断发射端到接收端的光 |
| **反射型光纤** | 物体反射光回接收端 |
| **透过检测** | 有物体时光被挡住 |
| **反射检测** | 有物体时反射光增强 |
| **放大器判断** | 根据设定阈值输出ON/OFF |

## ③ 计算/选型示例

- 已知：  
  需要检测一个黑色小零件是否到位。
- 公式：  
  **检测可靠性 = 光强变化量 + 阈值设置 + 安装稳定性**
- 计算：  
  黑色物体反光弱，若使用反射型可能不稳定；对射型遮光更明显。
- 结论：  
  如果机械结构允许，优先考虑 **对射型光纤**；若只能单侧安装，再评估反射型并现场调试灵敏度。

> [!tip] 记忆口诀
> 光纤不直接判断物体，判断的是光强有没有变。

# 03 FA/FC系列光纤传感器

> [!info] 核心结论
> FA/FC系列光纤传感器适合高精度、小空间、复杂安装位置的有无检测。

## ① 分类/公式/规则

> [!note] 🟦 定位
> FA/FC系列可用于连接不同类型光纤头，实现小零件、透明体、薄片、孔位、边缘、到位等检测。

> [!warning] 🟧 FA/FC适用公式
> **适合FA/FC = 检测点小 + 安装空间窄 + 需要灵敏度调节 + 需要放大器显示/设定**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **检测对象** | 小零件、薄片、标签、透明件 |
| **检测方式** | 对射型 / 反射型 |
| **安装优势** | 光纤头小，布置灵活 |
| **调试方式** | 放大器调灵敏度和阈值 |
| **输出类型** | PNP/NPN需校核 |
| **重点参数** | 响应时间、检测距离、光纤长度 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个直径约 **1mm** 的小孔是否通过。
- 公式：  
  **小目标检测 = 小光斑 + 稳定光路 + 高灵敏度放大器**
- 计算：  
  普通光电光斑可能过大，难以稳定识别小孔变化。
- 结论：  
  可选 **FA/FC系列光纤传感器 + 精密光纤头**，具体光斑尺寸和响应时间需按厂家手册校核。

> [!tip] 记忆口诀
> 检测越精细，越适合用光纤。

# 04 对射型光纤

> [!info] 核心结论
> 对射型光纤检测稳定性高，适合小物体、透明体、薄片和高速通过检测。

## ① 分类/公式/规则

> [!note] 🟦 工作方式
> 对射型光纤有发射端和接收端。  
> 物体通过时遮断光路，放大器根据接收光变弱判断物体存在。

> [!warning] 🟧 对射型判断公式
> **适合对射型 = 可两侧安装 + 需要稳定检测 + 目标能遮光**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 稳定、抗颜色影响较好 |
| **缺点** | 需要两侧安装和对准 |
| **适合对象** | 薄片、小件、透明件、挡片 |
| **重点参数** | 光轴对准、安装距离、响应时间 |
| **风险点** | 光纤头偏移会导致误检 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个厚度 **0.5mm** 的薄片高速通过。
- 公式：  
  **检测可靠性 = 遮光面积 + 响应时间 + 光轴稳定性**
- 计算：  
  薄片能够短时间遮断光束，需要放大器响应足够快。
- 结论：  
  优先选择 **对射型光纤**，并校核响应时间和光纤安装稳定性。

> [!tip] 记忆口诀
> 能两边装就选对射，稳定性通常更踏实。

# 05 反射型光纤

> [!info] 核心结论
> 反射型光纤只需单侧安装，适合空间受限场景，但受目标颜色、材质和角度影响更大。

## ① 分类/公式/规则

> [!note] 🟦 工作方式
> 反射型光纤从同一端发光和收光。  
> 物体靠近后把光反射回来，放大器根据反射光变化输出信号。

> [!warning] 🟧 反射型判断公式
> **适合反射型 = 只能单侧安装 + 目标反光稳定 + 检测距离较短**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **优点** | 单侧安装，结构简单 |
| **缺点** | 受颜色、表面、角度影响 |
| **适合对象** | 普通小零件、包装件、工件到位 |
| **不利对象** | 黑色、透明、高反光、弧面物体 |
| **重点参数** | 灵敏度、背景反光、距离 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个白色塑料件，只有一侧能安装传感器。
- 公式：  
  **反射检测可靠性 = 目标反光能力 + 检测距离 + 背景差异**
- 计算：  
  白色塑料件反光相对较强，单侧安装可行。
- 结论：  
  可选择 **反射型光纤**，但需现场校核背景反光和物体位置偏差。

> [!tip] 记忆口诀
> 只能一边装，反射来帮忙；颜色会影响，调试要现场。

# 06 光纤头选型

> [!info] 核心结论
> 光纤头决定检测点的安装方式、光斑大小、检测距离和机械适配性。

## ① 分类/公式/规则

> [!note] 🟦 常见光纤头
> 光纤头常见形式有：  
> 圆柱螺纹型、扁平型、耐弯折型、耐高温型、同轴型、区域检测型等。

> [!warning] 🟧 光纤头选型公式
> **光纤头型号 = 安装空间 + 检测对象 + 检测距离 + 环境要求 + 光斑需求**

## ② 参考表

| 光纤头类型 | 适合场景 |
|---|---|
| **圆柱螺纹型** | 孔位安装、螺母锁紧 |
| **扁平型** | 狭小空间、贴面安装 |
| **同轴型** | 小目标、位置精密检测 |
| **耐弯折型** | 频繁运动或弯曲场景 |
| **耐高温型** | 高温环境检测 |
| **区域型** | 宽范围通过检测 |

## ③ 计算/选型示例

- 已知：  
  检测点在狭窄夹具内部，普通传感器无法安装。
- 公式：  
  **可安装性 = 光纤头尺寸 + 弯曲半径 + 固定空间 ≤ 现场空间**
- 计算：  
  光纤头小，但光纤线不能过度弯折，需要预留弯曲空间。
- 结论：  
  选择合适光纤头时，既要看头部尺寸，也要看光纤最小弯曲半径。

> [!tip] 记忆口诀
> 光纤好不好用，一半看放大器，一半看光纤头。

# 07 放大器参数设置

> [!info] 核心结论
> 光纤开关调试的核心，是通过放大器设置合适的阈值，让“有物体”和“无物体”光强差异足够明显。

## ① 分类/公式/规则

> [!note] 🟦 调试重点
> 放大器通常可设置灵敏度、阈值、亮通/暗通、响应速度、定时功能、输出模式等参数。

> [!warning] 🟧 阈值设置公式
> **推荐阈值 ≈（有物体光强 + 无物体光强）÷ 2**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **阈值** | 判断ON/OFF的分界点 |
| **灵敏度** | 决定检测变化的敏感程度 |
| **L-ON** | 亮通，接收到光时ON |
| **D-ON** | 暗通，光被遮挡时ON |
| **响应时间** | 决定高速检测能力 |
| **定时功能** | 延时、保持、滤波 |

## ③ 计算/选型示例

- 已知：  
  无物体时光强显示 **800**，有物体时光强显示 **200**。
- 公式：  
  **阈值 ≈（800 + 200）÷ 2**
- 计算：  
  阈值 ≈ **500**
- 结论：  
  可将阈值设置在约 **500** 附近，再现场验证抖动和误触发情况。

> [!tip] 记忆口诀
> 有无差值越大，光纤检测越稳。

# 08 输出类型与PLC接线

> [!info] 核心结论
> 光纤放大器接PLC时，必须确认 **PNP/NPN、NO/NC、L-ON/D-ON、电源电压** 是否匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> PNP输出一般给PLC输入提供正电压信号；  
> NPN输出一般把PLC输入拉到0V；  
> L-ON/D-ON决定光强状态与输出ON的关系。

> [!warning] 🟧 接线匹配公式
> **PLC输入正常 = 放大器输出类型 + PLC输入公共端 + 电源极性 + 参数逻辑**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **PNP** | 输出正电信号 |
| **NPN** | 输出0V信号 |
| **NO常开** | 检测条件成立后ON |
| **NC常闭** | 检测条件成立后OFF |
| **L-ON** | 亮通，光强高时ON |
| **D-ON** | 暗通，光被遮挡时ON |
| **DC24V** | 工业现场常见供电，需校核 |

## ③ 计算/选型示例

- 已知：  
  PLC输入模块要求PNP输入，但光纤放大器为NPN输出。
- 公式：  
  **输入有效 = 输出类型匹配 + 公共端接线正确**
- 计算：  
  输出类型不匹配，PLC可能无法正常读取信号。
- 结论：  
  应选择 **PNP输出型光纤放大器**，或按PLC模块手册调整接线方案。

> [!tip] 记忆口诀
> 光纤接PLC，先看PNP/NPN；信号反不反，再看L-ON/D-ON。

# 09 安装与抗干扰

> [!info] 核心结论
> 光纤开关检测不稳定，常见原因是光纤头松动、光轴偏移、弯曲过度、背景反光或阈值设置不合理。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 光纤头要固定牢靠，光纤线要避免过度弯折，检测面要保持清洁，放大器阈值要留有余量。

> [!warning] 🟧 稳定检测公式
> **稳定检测 = 光路固定 + 光纤无损伤 + 阈值合理 + 环境干扰少 + 安装无松动**

## ② 参考表

| 问题 | 处理方法 |
|---|---|
| **检测忽有忽无** | 检查光纤头固定和阈值 |
| **无物体也触发** | 检查背景反光和灵敏度 |
| **有物体不触发** | 调整距离、角度、阈值 |
| **信号抖动** | 增加滤波或延时 |
| **光强变低** | 清洁光纤端面 |
| **频繁断光** | 检查光纤弯折或损伤 |

## ③ 计算/选型示例

- 已知：  
  光纤检测小零件时，设备振动后信号不稳定。
- 公式：  
  **不稳定原因 = 光纤头位移 + 光轴偏移 + 阈值余量不足**
- 计算：  
  振动可能让光纤头位置变化，导致光强接近阈值。
- 结论：  
  应加固光纤头，重新示教阈值，并检查光强余量是否足够。

> [!tip] 记忆口诀
> 光纤怕松、怕弯、怕脏，调试先看光强。

# 10 光纤开关快速选型表

> [!info] 核心结论
> 光纤开关选型先看检测方式和光纤头，再看放大器输出、响应速度和环境适应性。

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看检测对象  
> ② 看检测方式  
> ③ 看安装空间  
> ④ 看光纤头形式  
> ⑤ 看放大器输出  
> ⑥ 看响应速度和环境

> [!warning] 🟧 最终选型公式
> **推荐型号 = 放大器系列 + 光纤头类型 + 对射/反射 + 检测距离 + PNP/NPN + 响应时间**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 小零件检测 | **光纤传感器** |
| 狭小空间检测 | **小型光纤头** |
| 稳定遮断检测 | **对射型光纤** |
| 单侧安装检测 | **反射型光纤** |
| 高速通过检测 | **高速响应放大器** |
| 高温位置检测 | **耐高温光纤头** |
| 接PLC输入 | 校核PNP/NPN和NO/NC |

## ③ 计算/选型示例

- 已知：  
  需要检测高速通过的小标签，安装空间很小，PLC输入要求PNP。
- 公式：  
  **型号需求 = 对射/反射判断 + 小型光纤头 + 高速响应 + PNP输出**
- 计算：  
  标签高速通过，对响应时间要求高；空间小，需要光纤头小型化。
- 结论：  
  可选 **FA/FC系列光纤放大器 + 小型光纤头**，并重点校核响应时间、安装距离和输出类型。

> [!tip] 选型口诀
> 小空间用光纤，小目标看光斑；能对射更稳定，只能单侧选反射。
