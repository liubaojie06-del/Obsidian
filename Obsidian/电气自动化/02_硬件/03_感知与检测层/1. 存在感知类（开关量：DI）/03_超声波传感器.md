# 01 超声波传感器总览

> [!info] 核心结论
> 超声波传感器通过发射和接收声波来检测物体，适合 **距离测量、液位检测、透明物检测、单双张检测** 等场景。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 需要测量距离、液位、物体远近：选 **超声波测距传感器**。  
> 需要判断纸张、薄膜、金属箔是否单张或双张：选 **超声波单双张传感器**。

> [!warning] 🟧 通用选型公式
> **超声波选型 = 检测对象 + 检测距离 + 输出类型 + 盲区 + 声束角 + 环境条件**

## ② 参考表

| 类型 | 主要用途 |
|---|---|
| **超声波测距传感器** | 距离、液位、料位、有无检测 |
| **超声波单双张传感器** | 单张/双张/缺张检测 |
| **开关量输出** | 到位、有无、超限判断 |
| **模拟量输出** | 连续距离或液位测量 |
| **IO-Link输出** | 参数化、诊断、数字通讯 |
| **PNP/NPN输出** | 接PLC输入模块 |

## ③ 计算/选型示例

- 已知：  
  需要检测料箱中物料高度，物料表面颜色变化较大。
- 公式：  
  **适用性 = 非接触测量 + 不怕颜色变化 + 距离范围满足**
- 计算：  
  光电传感器可能受颜色影响，超声波主要看声波反射，更适合此类场景。
- 结论：  
  可优先考虑 **超声波测距传感器**，具体量程、盲区、输出方式需按厂家手册校核。

> [!tip] 记忆口诀
> 颜色透明别慌张，超声波看反射声波强不强。

# 02 底层原理

> [!info] 核心结论
> 超声波传感器的本质，是通过 **声波飞行时间或声波衰减变化** 来判断距离或材料状态。

## ① 分类/公式/规则

> [!note] 🟦 测距原理
> 传感器发出超声波，声波遇到物体后反射回来。  
> 传感器测量发射到接收的时间，再根据声速计算距离。

> [!warning] 🟧 测距核心公式
> **距离 S = 声速 v × 往返时间 t ÷ 2**

> [!note] 🟦 单双张检测原理
> 超声波穿过纸张、薄膜或片材时会产生衰减。  
> 单张、双张、无料时，接收到的声波强度不同，传感器通过阈值判断状态。

> [!warning] 🟧 单双张检测逻辑
> **无料声强最大 → 单张声强变弱 → 双张声强更弱 → 电路判断输出**

## ② 参考表

| 功能 | 底层变化 | 判断依据 |
|---|---|---|
| **测距** | 声波往返时间变化 | 时间差 |
| **液位检测** | 液面反射声波 | 距离变化 |
| **有无检测** | 有物体时产生反射 | 回波强度 |
| **单双张检测** | 声波穿透材料后衰减 | 接收声强 |
| **透明物检测** | 不依赖颜色透明度 | 声波反射 |

## ③ 计算/选型示例

- 已知：  
  声速约为 **340m/s**，声波从发出到接收共用时 **10ms**。
- 公式：  
  **S = v × t ÷ 2**
- 计算：  
  S = 340 × 0.01 ÷ 2 = **1.7m**
- 结论：  
  目标物距离传感器约 **1.7m**，实际测量需考虑温度补偿和厂家精度参数。

> [!tip] 记忆口诀
> 测距看时间，单双张看衰减。

# 03 超声波测距传感器

> [!info] 核心结论
> 超声波测距传感器适合非接触测量距离，常用于液位、料位、位置、有无检测。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 测距型超声波传感器通过回波时间计算距离，对颜色、透明度、表面图案不敏感。

> [!warning] 🟧 测距型判断公式
> **适合测距型 = 需要非接触 + 目标能反射声波 + 距离范围满足 + 环境干扰可控**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 检测对象 | 固体、液体、颗粒料 |
| 典型应用 | **液位、料位、距离、有无** |
| 输出类型 | 开关量、模拟量、IO-Link |
| 重点参数 | 量程、盲区、分辨率、重复精度 |
| 环境影响 | 温度、气流、粉尘、泡沫 |
| 安装重点 | 垂直对准目标面，避开强干扰 |

## ③ 计算/选型示例

- 已知：  
  水箱液位变化范围为 **0.3m～1.5m**，需要连续输出液位。
- 公式：  
  **传感器量程 ≥ 最大检测距离，并避开盲区**
- 计算：  
  最大距离1.5m，最小距离0.3m，需选择量程覆盖该范围且盲区小于0.3m的型号。
- 结论：  
  可选 **模拟量输出超声波测距传感器**，盲区和量程必须按厂家手册校核。

> [!tip] 记忆口诀
> 测距先看量程，再看盲区，最后看输出。

# 04 超声波单双张传感器

> [!info] 核心结论
> 超声波单双张传感器用于判断片状材料是 **无料、单张、双张或多张**，常见于印刷、包装、送料设备。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 单双张传感器不是用来测距离，而是用声波穿透材料后的强度变化判断材料层数。

> [!warning] 🟧 单双张判断公式
> **材料层数判断 = 接收声强 + 设定阈值 + 材料厚度 + 材料声学特性**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 检测对象 | **纸张、薄膜、金属箔、标签** |
| 检测状态 | 无料、单张、双张 |
| 典型场景 | 印刷、包装、贴标、送料 |
| 优点 | 不依赖颜色和透明度 |
| 重点参数 | 材料厚度、间隙、响应时间 |
| 注意 | 需按材料类型现场示教 |

## ③ 计算/选型示例

- 已知：  
  送料机构需要检测纸张是否一次吸起两张。
- 公式：  
  **异常判断 = 检测结果为双张 → 停机或报警**
- 计算：  
  单张时声波衰减较小，双张时衰减更大，传感器输出双张信号。
- 结论：  
  应选择 **超声波单双张传感器**，并根据纸张厚度和材料进行示教校准。

> [!tip] 记忆口诀
> 一张正常，两张报警；送料防重张，超声最擅长。

# 05 测距型 vs 单双张型

> [!info] 核心结论
> 测距型看“目标离我多远”，单双张型看“材料有几层”，两者用途不同，不能混用。

## ① 分类/公式/规则

> [!note] 🟦 快速区分
> 测距型：看回波时间，输出距离或位置。  
> 单双张型：看声波穿透衰减，输出单张/双张状态。

> [!warning] 🟧 选型判断公式
> **测距离 → 测距型**  
> **判层数 → 单双张型**

## ② 参考表

| 对比项 | 测距传感器 | 单双张传感器 |
|---|---|---|
| 核心功能 | **测距离** | **判层数** |
| 原理重点 | 回波时间 | 声波衰减 |
| 典型对象 | 液位、物体、料位 | 纸张、薄膜、片材 |
| 输出结果 | 距离值/开关点 | 单张/双张/无料 |
| 安装方式 | 面向目标反射面 | 材料穿过检测区 |
| 校核重点 | 量程、盲区 | 材料厚度、间隙 |

## ③ 计算/选型示例

- 已知：  
  需要检测纸张是否双张，而不是纸张距离传感器多远。
- 公式：  
  **需求类型 = 层数判断，而非距离测量**
- 计算：  
  测距型只能判断距离变化，不一定能可靠区分单张和双张。
- 结论：  
  应选 **超声波单双张传感器**，不要用普通测距传感器替代。

> [!tip] 记忆口诀
> 测远近用测距，防双张用单双张。

# 06 输出类型与PLC接线

> [!info] 核心结论
> 超声波传感器接PLC时，要确认输出是 **开关量、模拟量、IO-Link、PNP还是NPN**。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 开关量输出适合到位、有无、超限判断；  
> 模拟量输出适合连续距离或液位；  
> IO-Link适合参数化和诊断；  
> PNP/NPN需与PLC输入模块匹配。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型匹配 + 电源电压正确 + 公共端正确 + 参数设置正确**

## ② 参考表

| 输出类型 | 适合用途 |
|---|---|
| **PNP/NPN开关量** | 有无、到位、报警 |
| **0-10V模拟量** | 距离、液位连续测量 |
| **4-20mA模拟量** | 工业现场抗干扰较好 |
| **IO-Link** | 参数设置、状态诊断 |
| **双开关量** | 上限/下限、单张/双张 |
| **NO/NC** | 输出逻辑选择 |

## ③ 计算/选型示例

- 已知：  
  需要把水箱液位连续传给PLC模拟量模块，PLC支持4-20mA输入。
- 公式：  
  **信号匹配 = 传感器输出4-20mA + PLC输入4-20mA**
- 计算：  
  若传感器输出0-10V，而PLC模块是4-20mA输入，则不匹配。
- 结论：  
  应选择 **4-20mA输出型超声波测距传感器**，并按模块手册接线和量程换算。

> [!tip] 记忆口诀
> 开关量判有无，模拟量测连续；PNP/NPN别接反。

# 07 安装与盲区

> [!info] 核心结论
> 超声波传感器安装时必须避开盲区，并保证目标面能稳定反射声波。

## ① 分类/公式/规则

> [!note] 🟦 盲区概念
> 盲区是传感器头部附近无法可靠检测的区域。  
> 目标物如果太近，传感器可能无法正确接收回波。

> [!warning] 🟧 安装距离公式
> **可靠检测距离 ≥ 传感器盲区 + 安全余量**

## ② 参考表

| 安装项 | 检查重点 |
|---|---|
| **盲区** | 目标不能进入盲区 |
| **检测角度** | 尽量垂直目标面 |
| **目标表面** | 避免过度倾斜或吸声 |
| **安装空间** | 避免侧壁反射干扰 |
| **气流影响** | 避开强风、压缩空气 |
| **粉尘泡沫** | 可能削弱回波 |

## ③ 计算/选型示例

- 已知：  
  传感器盲区为 **100mm**，最近目标距离为 **120mm**。
- 公式：  
  **安全余量 = 最近目标距离 - 盲区**
- 计算：  
  安全余量 = 120 - 100 = **20mm**
- 结论：  
  余量较小，可能不稳定；建议调整安装距离或选择盲区更小的型号。

> [!tip] 记忆口诀
> 超声有盲区，太近测不准。

# 08 环境影响与抗干扰

> [!info] 核心结论
> 超声波传感器不怕颜色和透明度，但会受温度、气流、粉尘、泡沫、目标角度影响。

## ① 分类/公式/规则

> [!note] 🟦 影响因素
> 声速会随温度变化；强气流会扰动声波；粉尘、泡沫和软材料可能吸收声波；斜面会把声波反射到别处。

> [!warning] 🟧 稳定检测公式
> **稳定超声检测 = 温度补偿 + 目标反射稳定 + 避开气流 + 合理安装角度**

## ② 参考表

| 问题 | 可能影响 |
|---|---|
| **温度变化** | 影响声速和距离精度 |
| **强气流** | 声波路径不稳定 |
| **粉尘** | 回波衰减或误检 |
| **泡沫液面** | 反射不稳定 |
| **软材料** | 吸收声波 |
| **斜面目标** | 回波偏离传感器 |

## ③ 计算/选型示例

- 已知：  
  超声波测量液位时，液面有大量泡沫。
- 公式：  
  **回波可靠性 = 液面反射能力 - 泡沫吸收影响**
- 计算：  
  泡沫会削弱或散射声波，导致液位信号跳动。
- 结论：  
  需要现场测试，必要时增加导波管、调整安装位置，或改用其他液位检测方式。

> [!tip] 记忆口诀
> 超声不怕颜色，怕风、泡沫和斜面。

# 09 超声波快速选型表

> [!info] 核心结论
> 超声波选型先分清是 **测距** 还是 **判单双张**，再看量程、盲区、输出和环境。

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看检测目的  
> ② 看检测对象  
> ③ 看检测距离或材料厚度  
> ④ 看盲区和安装空间  
> ⑤ 看输出类型  
> ⑥ 看环境干扰

> [!warning] 🟧 最终选型公式
> **推荐型号 = 测距/单双张 + 量程/材料范围 + 输出类型 + 安装方式 + 防护等级**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 距离检测 | **超声波测距传感器** |
| 液位检测 | **超声波测距传感器** |
| 透明物有无 | **超声波测距或开关型** |
| 纸张双张检测 | **超声波单双张传感器** |
| 薄膜重叠检测 | **超声波单双张传感器** |
| 连续量输出 | 模拟量 / IO-Link |
| 简单到位判断 | PNP/NPN开关量 |

## ③ 计算/选型示例

- 已知：  
  需要检测输送线上纸张是否双张，PLC输入为PNP，现场DC24V供电。
- 公式：  
  **型号需求 = 单双张检测 + PNP输出 + DC24V + 材料厚度匹配**
- 计算：  
  该需求重点是判断纸张层数，不是测距离。
- 结论：  
  选择 **超声波单双张传感器**，并按纸张厚度、速度、安装间隙和厂家手册校核。

> [!tip] 选型口诀
> 测距离看量程，防双张看材料；盲区要避开，输出配PLC。
