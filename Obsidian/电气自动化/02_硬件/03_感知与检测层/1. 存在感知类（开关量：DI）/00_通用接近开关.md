# 01 接近开关总览

> [!info] 核心结论
> 接近开关用于 **无接触检测物体是否靠近**，常见类型主要有 **电感式** 和 **电容式**。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> 检测金属：优先选 **电感接近开关**。  
> 检测非金属、液体、粉料、塑料：优先选 **电容接近开关**。

> [!warning] 🟧 通用选型公式
> **接近开关选型 = 检测对象 + 检测距离 + 安装方式 + 输出类型 + 电源电压 + 环境条件**

## ② 参考表

| 类型 | 主要检测对象 |
|---|---|
| **电感接近开关** | 金属物体 |
| **电容接近开关** | 金属、非金属、液体、粉末 |
| **PNP输出** | 常用于PLC源型输入 |
| **NPN输出** | 常用于PLC漏型输入 |
| **NO常开** | 检测到物体后输出ON |
| **NC常闭** | 未检测时输出ON |

## ③ 计算/选型示例

- 已知：  
  需要检测输送线上的金属挡块是否到位。
- 公式：  
  **传感器类型 = 检测对象材料 + 安装距离 + PLC输入类型**
- 计算：  
  检测对象是金属，距离较近，适合电感式。
- 结论：  
  优先选 **电感接近开关**，输出类型按PLC输入模块选择，具体感应距离需按厂家手册校核。

> [!tip] 记忆口诀
> 金属用电感，非金属用电容；接PLC看PNP/NPN。

# 02 电感接近开关

> [!info] 核心结论
> 电感接近开关主要用于检测 **金属目标物**，常用于气缸到位、金属工件检测、机械限位检测。

## ① 分类/公式/规则

> [!note] 🟦 工作特点
> 电感式传感器利用电磁感应原理，对金属物体最敏感，不能可靠检测塑料、玻璃、木材、液体等非金属物体。

> [!warning] 🟧 电感式判断公式
> **适合电感式 = 金属目标 + 距离较近 + 环境较稳定 + 需要高可靠到位检测**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 检测对象 | **金属** |
| 常见外形 | 圆柱型、方形 |
| 常见螺纹 | M8、M12、M18、M30 |
| 安装方式 | 齐平 / 非齐平 |
| 输出类型 | PNP、NPN、二线制 |
| 接点逻辑 | NO常开、NC常闭 |
| 电源 | 常见DC 24V，需按型号校核 |

## ③ 计算/选型示例

- 已知：  
  需要检测一块铁质工件，到传感器距离约 **3mm**。
- 公式：  
  **推荐感应距离 ≥ 实际检测距离 × 1.5**
- 计算：  
  3mm × 1.5 = **4.5mm**
- 结论：  
  建议选择额定感应距离 **≥5mm** 的电感接近开关，实际可靠距离需按目标材料和厂家手册校核。

> [!tip] 记忆口诀
> 电感只认金属，距离要留余量。

# 03 电容接近开关

> [!info] 核心结论
> 电容接近开关可检测 **金属和非金属**，特别适合液位、粉料、塑料、玻璃、纸张等检测场景。

## ① 分类/公式/规则

> [!note] 🟦 工作特点
> 电容式传感器通过检测电容量变化判断物体接近，对材料介电常数、湿度、灰尘、安装环境更敏感。

> [!warning] 🟧 电容式判断公式
> **适合电容式 = 非金属目标 + 液体/粉料/塑料 + 允许灵敏度调节 + 环境可控**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| 检测对象 | **金属 / 非金属 / 液体 / 粉料** |
| 常见用途 | 液位、料位、包装物检测 |
| 灵敏度 | 通常可调 |
| 环境影响 | 湿度、粉尘、残留物影响较大 |
| 安装方式 | 需避免误触发 |
| 输出类型 | PNP、NPN、NO、NC |
| 校核重点 | 介质、壁厚、距离、干扰 |

## ③ 计算/选型示例

- 已知：  
  需要隔着塑料桶壁检测桶内是否有液体。
- 公式：  
  **电容检测可靠性 = 介质差异 + 桶壁厚度 + 灵敏度调节 + 安装稳定性**
- 计算：  
  液体和空气介电特性不同，电容式可用于液位有无检测。
- 结论：  
  可选 **电容接近开关**，但需现场调灵敏度，并按桶壁材料、厚度和厂家手册校核。

> [!tip] 记忆口诀
> 电容不挑金属，液体粉料也能测。

# 04 电感 vs 电容

> [!info] 核心结论
> 电感式更稳定但只能检测金属；电容式检测范围更广，但更容易受环境影响。

## ① 分类/公式/规则

> [!note] 🟦 对比规则
> 选型时先问一句：  
> **我要检测的东西是不是金属？**

> [!warning] 🟧 快速判断公式
> **金属目标 → 电感式**  
> **非金属 / 液体 / 粉料 → 电容式**

## ② 参考表

| 对比项 | 电感接近开关 | 电容接近开关 |
|---|---|---|
| 检测金属 | **适合** | 适合 |
| 检测塑料 | 不适合 | **适合** |
| 检测液体 | 不适合 | **适合** |
| 检测粉料 | 不适合 | **适合** |
| 抗干扰 | 较好 | 较敏感 |
| 灵敏度调节 | 一般较少 | 常见可调 |
| 典型场景 | 金属到位 | 液位、料位 |

## ③ 计算/选型示例

- 已知：  
  设备中既要检测金属挡块，又要检测塑料瓶。
- 公式：  
  **传感器类型 = 目标材料**
- 计算：  
  金属挡块 → 电感式；塑料瓶 → 电容式。
- 结论：  
  两个检测点不能混用同一种传感器，应分别选 **电感接近开关** 和 **电容接近开关**。

> [!tip] 记忆口诀
> 电感稳，专盯金属；电容广，环境要防。

# 05 输出类型：PNP / NPN

> [!info] 核心结论
> 接近开关接PLC时，最容易出错的是 **PNP/NPN输出类型和PLC输入类型不匹配**。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> PNP输出通常给PLC输入端提供正电压信号；  
> NPN输出通常把PLC输入端拉到0V。

> [!warning] 🟧 匹配公式
> **传感器输出类型 = PLC输入模块类型 + 公共端接法**

## ② 参考表

| 输出类型 | 常见理解 | 典型接法 |
|---|---|---|
| **PNP** | 输出正电 | 负载接0V侧 |
| **NPN** | 输出0V | 负载接24V侧 |
| **二线制** | 串接负载 | 注意漏电流 |
| **三线制** | 电源+、电源-、信号线 | 最常见 |
| **四线制** | 常开+常闭输出 | 功能更灵活 |

## ③ 计算/选型示例

- 已知：  
  PLC输入模块要求使用PNP传感器，现场误买了NPN传感器。
- 公式：  
  **通讯输入正常 = 输出类型匹配 + 接线正确 + 电压正确**
- 计算：  
  输出类型不匹配，即使传感器本身动作，PLC也可能收不到正确输入。
- 结论：  
  选型时必须确认 **PNP/NPN**，并与PLC输入模块手册一致。

> [!tip] 记忆口诀
> PNP给正，NPN给零；PLC要什么，传感器就选什么。

# 06 常开 / 常闭

> [!info] 核心结论
> 常开NO和常闭NC决定传感器在检测到物体前后的输出状态，直接影响PLC程序逻辑。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> NO常开：没有检测到物体时OFF，检测到后ON。  
> NC常闭：没有检测到物体时ON，检测到后OFF。

> [!warning] 🟧 逻辑选择公式
> **输出逻辑 = 设备安全要求 + 程序习惯 + 故障检测需求**

## ② 参考表

| 类型 | 无物体 | 有物体 |
|---|---|---|
| **NO常开** | OFF | **ON** |
| **NC常闭** | **ON** | OFF |
| **NO适合** | 普通到位检测 | 检测到才动作 |
| **NC适合** | 断线检测、安全类思路 | 异常更容易发现 |

## ③ 计算/选型示例

- 已知：  
  气缸到位后PLC需要收到一个ON信号。
- 公式：  
  **到位信号 = 检测到物体后输出ON**
- 计算：  
  需要检测到目标后ON，因此选择NO常开更直观。
- 结论：  
  普通到位检测可选 **NO常开**；若需要断线更容易暴露，可评估 **NC常闭**。

> [!tip] 记忆口诀
> 常开检测才亮，常闭平时就亮。

# 07 安装方式：齐平 / 非齐平

> [!info] 核心结论
> 接近开关安装方式会影响感应距离和误动作风险，电感式尤其要注意齐平和非齐平。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 齐平安装：传感器头部可与金属安装面齐平。  
> 非齐平安装：传感器头部周围需要留出空间，避免被安装金属影响。

> [!warning] 🟧 安装判断公式
> **可靠安装 = 安装方式正确 + 周围间距足够 + 感应面无遮挡 + 目标运动方向合理**

## ② 参考表

| 安装方式 | 特点 |
|---|---|
| **齐平安装** | 可嵌入金属支架 |
| **非齐平安装** | 周围需留空间 |
| **侧面靠金属太近** | 可能误触发 |
| **正面距离太小** | 可能长期ON |
| **目标偏心经过** | 可能检测不稳定 |

## ③ 计算/选型示例

- 已知：  
  传感器需要安装在金属支架孔内，头部与支架表面平齐。
- 公式：  
  **安装匹配 = 齐平安装需求 + 对应型号**
- 计算：  
  如果误选非齐平型号，金属支架可能影响传感器检测。
- 结论：  
  应选择支持 **齐平安装** 的接近开关，具体安装间距需按厂家手册校核。

> [!tip] 记忆口诀
> 装进金属看齐平，露出安装看间距。

# 08 接近开关快速选型表

> [!info] 核心结论
> 接近开关选型不要只看外形，要同时确认检测对象、距离、输出、电源、安装和环境。

## ① 分类/公式/规则

> [!note] 🟦 六步选型法
> ① 看检测对象  
> ② 看检测距离  
> ③ 看安装空间  
> ④ 看输出类型  
> ⑤ 看电源电压  
> ⑥ 看环境防护

> [!warning] 🟧 最终选型公式
> **推荐型号 = 类型 + 外形尺寸 + 感应距离 + 输出方式 + NO/NC + 接线方式 + 防护等级**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 金属到位检测 | **电感接近开关** |
| 塑料瓶检测 | **电容接近开关** |
| 液位有无检测 | **电容接近开关** |
| 粉料料位检测 | **电容接近开关** |
| PLC常规输入 | 三线制PNP/NPN |
| 安装在金属孔内 | 齐平型需校核 |
| 潮湿粉尘环境 | 防护等级需校核 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个金属零件，实际检测距离约 **4mm**，PLC输入要求PNP，现场使用DC 24V。
- 公式：  
  **型号需求 = 电感式 + ≥6mm感应距离 + PNP + DC24V + 合适外形**
- 计算：  
  4mm × 1.5 = **6mm**，建议选择额定感应距离不小于6mm的型号。
- 结论：  
  可选 **DC24V、PNP、NO、感应距离≥6mm** 的电感接近开关，具体型号需按厂家手册校核。

> [!tip] 选型口诀
> 先看材料，再看距离；输出配PLC，安装看空间，环境看防护。

# 09 底层原理

> [!info] 核心结论
> 接近开关的本质，是把“物体靠近”转化成 **电磁场或电场变化**，再由电路判断是否输出信号。

## ① 分类/公式/规则

> [!note] 🟦 电感接近开关原理
> 电感式内部有一个高频振荡线圈，传感器前方会形成交变磁场。  
> 当 **金属物体靠近** 时，金属内部产生涡流，吸收磁场能量，使振荡变弱，电路检测到变化后输出信号。

> [!warning] 🟧 电感式核心逻辑
> **金属靠近 → 产生涡流 → 振荡减弱 → 电路判断 → 输出ON/OFF**

> [!note] 🟦 电容接近开关原理
> 电容式内部形成一个电场，传感器和目标物之间相当于一个“电容”。  
> 当物体靠近时，介电常数发生变化，电容量改变，电路检测到变化后输出信号。

> [!warning] 🟧 电容式核心逻辑
> **物体靠近 → 电容量变化 → 振荡状态改变 → 电路判断 → 输出ON/OFF**

## ② 参考表

| 类型 | 底层变化 | 对什么敏感 |
|---|---|---|
| **电感式** | 磁场被金属吸收 | **金属、铁、铝、铜等** |
| **电容式** | 电容量发生变化 | **金属、塑料、液体、粉料** |
| 电感式关键现象 | 涡流效应 | 金属越大越容易检测 |
| 电容式关键现象 | 介电常数变化 | 介质差异越大越容易检测 |
| 共同点 | 都是无接触检测 | 都需要电路判断阈值 |

## ③ 计算/选型示例

- 已知：  
  需要隔着塑料容器检测里面有没有液体。
- 公式：  
  **检测变化 = 空容器电容量 - 有液体时电容量**
- 计算：  
  空气和液体的介电特性不同，液体靠近传感器后会明显改变电容量。
- 结论：  
  这种场景适合使用 **电容接近开关**；但灵敏度、桶壁厚度、液体种类需按厂家手册校核。

> [!tip] 记忆口诀
> 电感看磁场，金属来扰场；电容看电场，材料变化就能量。
