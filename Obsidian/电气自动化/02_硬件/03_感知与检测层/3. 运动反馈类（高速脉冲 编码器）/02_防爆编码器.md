# 01 防爆编码器总览

> [!info] 核心结论
> 防爆编码器用于有爆炸风险的场所，在满足位置、速度检测的同时，还要满足 **防爆等级、区域等级、气体/粉尘环境** 要求。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 普通编码器只看信号和精度；  
> 防爆编码器除了看信号和精度，还必须看 **防爆认证、防爆型式、适用区域、温度组别**。

> [!warning] 🟧 通用选型公式
> **防爆编码器选型 = 防爆等级 + 危险区域 + 编码器类型 + 输出信号 + 分辨率 + 机械安装 + 环境条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **类型** | 防爆型编码器 |
| **检测对象** | 旋转轴、卷筒、电机、阀门、提升机构 |
| **主要功能** | 测速度、测位置、测方向 |
| **常见类型** | 增量型、绝对值型、总线型 |
| **重点参数** | 防爆等级、PPR/分辨率、输出、电源、轴型 |
| **应用场景** | 石化、煤化工、油气、喷涂、粉尘环境 |

## ③ 计算/选型示例

- 已知：  
  现场为易燃气体环境，需要检测电机轴转速。
- 公式：  
  **适用性 = 危险区域匹配 + 防爆认证匹配 + 信号类型匹配**
- 计算：  
  普通编码器不能直接用于危险区域，必须选对应防爆等级产品。
- 结论：  
  应选择 **防爆增量编码器**，并按现场防爆区域、防爆标志、输出类型和厂家手册校核。

> [!tip] 记忆口诀
> 普通场所看精度，防爆场所先看等级。

# 02 底层原理

> [!info] 核心结论
> 防爆编码器的检测原理和普通编码器类似，区别在于外壳、结构和电气设计要防止引燃危险环境。

## ① 分类/公式/规则

> [!note] 🟦 编码原理
> 编码器内部通过光电、磁电或其他检测方式，把轴旋转转换成脉冲、位置值或总线数据。

> [!note] 🟦 防爆原理
> 防爆设计的目标不是让设备“不发热”，而是让设备在规定条件下 **不成为点燃源**。

> [!warning] 🟧 防爆核心逻辑
> **旋转检测 + 防爆外壳/本安设计 + 温升控制 + 认证匹配 = 危险区可用**

## ② 参考表

| 部分 | 作用 |
|---|---|
| **检测单元** | 获取轴的角度或转速 |
| **信号电路** | 输出脉冲、模拟量或总线数据 |
| **防爆外壳** | 限制火花或高温外泄风险 |
| **密封结构** | 防止危险气体/粉尘进入关键部位 |
| **温度控制** | 避免表面温度成为点火源 |
| **接地与屏蔽** | 降低静电和电磁干扰风险 |

## ③ 计算/选型示例

- 已知：  
  编码器安装在含可燃气体的区域，电机最高转速为1500rpm。
- 公式：  
  **可用性 = 防爆等级满足 + 最高转速满足 + 信号频率满足**
- 计算：  
  不仅要看编码器能否测1500rpm，还要确认防爆等级适合现场环境。
- 结论：  
  防爆编码器必须同时满足 **安全要求 + 测量要求**。

> [!tip] 记忆口诀
> 编码靠检测，防爆靠结构；两者都满足，现场才安全。

# 03 防爆等级与危险区域

> [!info] 核心结论
> 防爆编码器选型第一步，不是看PPR，而是确认现场属于什么危险区域、气体组别和温度组别。

## ① 分类/公式/规则

> [!note] 🟦 选型顺序
> 先确认现场防爆要求，再选择编码器型号。  
> 不能用普通工业编码器替代防爆编码器。

> [!warning] 🟧 防爆匹配公式
> **防爆匹配 = 区域等级 + 防爆型式 + 气体/粉尘组别 + 温度组别 + 认证文件**

## ② 参考表

| 项目 | 重点 |
|---|---|
| **危险区域** | Zone 0 / 1 / 2 或粉尘区域等 |
| **防爆型式** | 隔爆型、本安型等需按项目要求 |
| **气体组别** | 与现场可燃气体匹配 |
| **粉尘环境** | 需考虑粉尘防爆等级 |
| **温度组别** | 表面温度不能点燃环境 |
| **认证资料** | 必须按项目规范校核 |

## ③ 计算/选型示例

- 已知：  
  项目要求设备用于Zone 1危险区。
- 公式：  
  **型号可用 = 编码器防爆证书覆盖该区域**
- 计算：  
  如果编码器只适用于普通区域或较低风险区域，则不能用于Zone 1。
- 结论：  
  必须选择防爆证书覆盖现场区域要求的编码器，具体以项目规范和厂家证书为准。

> [!tip] 记忆口诀
> 防爆先看证，区域不匹配，一律不能用。

# 04 增量型防爆编码器

> [!info] 核心结论
> 增量型防爆编码器输出A/B/Z脉冲，适合危险区域内的转速、方向和相对位置检测。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 增量型防爆编码器适合电机测速、皮带测速、卷筒计数、相对位移检测等场景。

> [!warning] 🟧 增量型选型公式
> **增量防爆编码器 = 防爆等级 + PPR + ABZ信号 + 输出类型 + 最高频率 + 机械安装**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **信号** | A/B/Z |
| **分辨率** | PPR |
| **方向判断** | A/B相位差 |
| **零位信号** | Z相每圈一次 |
| **PLC要求** | 高速计数输入 |
| **注意** | 断电后通常需回零 |

## ③ 计算/选型示例

- 已知：  
  防爆电机最高转速为3000rpm，编码器为1024PPR。
- 公式：  
  **最高频率 f = rpm × PPR ÷ 60**
- 计算：  
  f = 3000 × 1024 ÷ 60 = **51200Hz = 51.2kHz**
- 结论：  
  PLC高速计数输入频率必须大于 **51.2kHz**，并留有余量。

> [!tip] 记忆口诀
> 增量防爆数脉冲，频率先算清。

# 05 绝对值防爆编码器

> [!info] 核心结论
> 绝对值防爆编码器能直接输出当前位置，适合危险区域内不允许频繁回零的设备。

## ① 分类/公式/规则

> [!note] 🟦 定位
> 绝对值防爆编码器适合阀门开度、提升机构、卷扬系统、转台定位等需要上电知道位置的场景。

> [!warning] 🟧 绝对值选型公式
> **绝对值防爆编码器 = 防爆等级 + 单圈/多圈 + 分辨率 + 通讯接口 + 掉电位置保持**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **单圈绝对值** | 旋转角度不超过一圈 |
| **多圈绝对值** | 升降、卷绕、长行程 |
| **总线输出** | PLC直接读取位置 |
| **SSI输出** | 点对点绝对位置传输 |
| **模拟量输出** | 简单开度反馈场景 |
| **防爆外壳** | 危险区使用必查 |

## ③ 计算/选型示例

- 已知：  
  需要检测防爆区域内阀门开度，阀门行程小于一圈。
- 公式：  
  **需求 = 单圈角度位置 + 上电可读 + 防爆等级匹配**
- 计算：  
  阀门不超过一圈，单圈绝对值编码器即可满足位置范围。
- 结论：  
  可选 **单圈绝对值防爆编码器**，输出协议需与控制系统匹配。

> [!tip] 记忆口诀
> 要上电知道位置，就选绝对值；超过一圈，看多圈。

# 06 输出信号与系统匹配

> [!info] 核心结论
> 防爆编码器不仅要防爆等级匹配，输出信号也要和PLC、DCS、变频器或安全栅匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 增量型常见PNP、NPN、推挽、差分RS422；  
> 绝对值型常见SSI、CANopen、PROFIBUS、PROFINET、模拟量等。

> [!warning] 🟧 信号匹配公式
> **系统读取正常 = 输出类型 + 电源电压 + 通讯协议 + 安全栅/隔离器 + 接线方式匹配**

## ② 参考表

| 输出类型 | 适合场景 |
|---|---|
| **PNP/NPN** | PLC高速输入 |
| **推挽输出** | 通用高速输入 |
| **差分RS422** | 长距离、高速、抗干扰 |
| **SSI** | 绝对值位置点对点传输 |
| **PROFINET/PROFIBUS** | PLC总线系统 |
| **4-20mA** | 简单位置或开度反馈 |

## ③ 计算/选型示例

- 已知：  
  防爆编码器为差分RS422输出，但PLC只有24V PNP高速输入。
- 公式：  
  **输入有效 = 信号类型一致**
- 计算：  
  RS422差分信号不能直接接普通24V PNP输入。
- 结论：  
  需要选择匹配输入类型的编码器，或增加信号转换模块，且转换设备也要符合现场防爆要求。

> [!tip] 记忆口诀
> 防爆看证书，信号看接口；两边都匹配，系统才读数。

# 07 机械安装与联轴器

> [!info] 核心结论
> 防爆编码器安装时，要避免轴偏心、机械冲击、联轴器过硬和外壳受力变形。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 编码器轴不能承受过大的径向力和轴向力。  
> 电机或设备轴与编码器轴之间通常需要弹性联轴器。

> [!warning] 🟧 安装可靠公式
> **可靠安装 = 同轴度良好 + 联轴器合适 + 固定牢靠 + 轴承负载不过限**

## ② 参考表

| 项目 | 检查重点 |
|---|---|
| **轴型** | 实心轴、空心轴、盲孔轴 |
| **轴径** | 与机械连接匹配 |
| **联轴器** | 补偿微小偏心 |
| **安装支架** | 刚性足够，不松动 |
| **振动冲击** | 需按现场工况校核 |
| **防爆外壳** | 不可私自改孔、打磨、拆改 |

## ③ 计算/选型示例

- 已知：  
  编码器安装后运行一段时间出现位置漂移。
- 公式：  
  **漂移风险 = 联轴器松动 + 机械打滑 + 轴偏心 + 丢脉冲**
- 计算：  
  如果联轴器锁紧不良，编码器轴转动和实际机械轴转动不同步。
- 结论：  
  应检查联轴器、轴键、紧定螺钉和机械同轴度。

> [!tip] 记忆口诀
> 编码器测的是轴，轴若打滑数就假。

# 08 防爆安装注意事项

> [!info] 核心结论
> 防爆编码器安装不能随意开盖、改线、改密封或替换电缆，必须按防爆规范和厂家要求施工。

## ① 分类/公式/规则

> [!note] 🟦 关键要求
> 防爆设备的电缆引入口、密封圈、接线腔、接地端子、外壳螺纹和铭牌信息都属于安全相关内容。

> [!warning] 🟧 防爆安装公式
> **安全安装 = 正确电缆引入 + 密封可靠 + 接地可靠 + 不破坏防爆结构 + 证书一致**

## ② 参考表

| 项目 | 注意事项 |
|---|---|
| **电缆入口** | 使用匹配防爆格兰头 |
| **接地** | 外壳和屏蔽按规范处理 |
| **密封** | 不得破坏密封结构 |
| **开盖操作** | 按断电和安全流程执行 |
| **铭牌信息** | 防爆标志必须可追溯 |
| **维修更换** | 使用原厂或认证配件 |

## ③ 计算/选型示例

- 已知：  
  现场需要更换防爆编码器电缆接头。
- 公式：  
  **更换可行 = 接头防爆等级匹配 + 螺纹规格匹配 + 密封满足**
- 计算：  
  普通电缆接头不能替代防爆格兰头，否则可能破坏防爆性能。
- 结论：  
  必须选用符合防爆要求的电缆引入装置，并按规范安装。

> [!tip] 记忆口诀
> 防爆件别乱改，电缆入口最关键。

# 09 常见故障排查

> [!info] 核心结论
> 防爆编码器异常时，既要排查普通编码器问题，也要检查防爆接线、屏蔽接地和安全栅/隔离器。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查电源和信号，再查防爆接线；  
> 先查控制系统参数，再查机械连接；  
> 危险区内开盖操作必须按安全流程执行。

> [!warning] 🟧 排错公式
> **编码器异常 = 供电错误 + 信号不匹配 + 屏蔽接地差 + 机械打滑 + 安全栅错误 + 参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无信号** | 电源错误、接线错误、输出类型不匹配 |
| **计数跳变** | 干扰、屏蔽不良、线缆过长 |
| **方向相反** | A/B相接反或参数设置反 |
| **通讯掉线** | 总线地址、屏蔽、终端或设备文件问题 |
| **位置漂移** | 联轴器松动、机械打滑、丢脉冲 |
| **无法在危险区使用** | 防爆等级或证书不匹配 |

## ③ 计算/选型示例

- 已知：  
  编码器本体输出正常，但PLC侧没有计数。
- 公式：  
  **PLC无计数 = 信号未到PLC + 输入类型不匹配 + 高速计数未配置**
- 计算：  
  现场可能存在安全栅、隔离器或中间端子，需逐段测量信号是否传递。
- 结论：  
  应检查编码器输出、安全栅/隔离器、PLC高速输入类型和程序计数配置。

> [!tip] 记忆口诀
> 防爆编码器查故障，先安全，再信号，后机械。

# 10 防爆编码器快速选型表

> [!info] 核心结论
> 防爆编码器选型先确认防爆要求，再确认编码器类型、信号接口、分辨率和安装结构。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 看危险区域等级  
> ② 看气体/粉尘环境  
> ③ 看防爆型式和证书  
> ④ 看增量还是绝对值  
> ⑤ 看PPR或分辨率  
> ⑥ 看输出接口  
> ⑦ 看轴型和安装尺寸  
> ⑧ 看温度、防护和线缆

> [!warning] 🟧 最终选型公式
> **推荐型号 = 防爆等级 + 编码器类型 + 分辨率 + 输出类型 + 电源 + 轴型 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 危险区域使用 | 防爆证书和防爆标志 |
| 测转速 | 增量型防爆编码器 |
| 上电读位置 | 绝对值防爆编码器 |
| 长行程位置 | 多圈绝对值 |
| 接PLC高速输入 | PNP/NPN/推挽/差分匹配 |
| 接总线系统 | PROFINET、PROFIBUS等 |
| 强干扰环境 | 差分信号、屏蔽、接地 |

## ③ 计算/选型示例

- 已知：  
  危险区域内一台卷扬设备，需要检测卷筒位置，断电后不能丢失位置。
- 公式：  
  **型号需求 = 防爆等级匹配 + 多圈绝对值 + 通讯接口匹配 + 机械安装匹配**
- 计算：  
  卷筒会旋转多圈，且断电后不能丢位置，因此增量型不够方便。
- 结论：  
  可选择 **多圈绝对值防爆编码器**，并按防爆证书、通讯协议和机械安装尺寸校核。

> [!tip] 选型口诀
> 防爆先看证，类型看功能；测速用增量，定位用绝对，总线看协议。
