# 01 涡街流量计总览

> [!info] 核心结论
> FV系列涡街流量计适合测量 **蒸汽、气体、液体** 的流量，常用于压缩空气、饱和蒸汽、过热蒸汽、循环水等工况。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 涡街流量计不是靠叶轮旋转测流量，而是利用流体经过阻流体后产生的 **卡门涡街频率** 来计算流速和流量。

> [!warning] 🟧 通用选型公式
> **涡街流量计选型 = 介质类型 + 管径DN + 流量范围 + 温度压力 + 输出信号 + 安装方式 + 补偿需求**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **系列** | FV系列涡街流量计 |
| **适用介质** | 蒸汽、气体、液体 |
| **典型应用** | 蒸汽计量、压缩空气、循环水 |
| **常见输出** | 4-20mA、脉冲、RS485等需校核 |
| **安装方式** | 法兰式、夹持式、插入式需校核 |
| **重点参数** | 管径、流量、压力、温度、密度、雷诺数 |

## ③ 计算/选型示例

- 已知：  
  需要测量一条蒸汽管道的瞬时流量，并把信号送到PLC。
- 公式：  
  **适用性 = 蒸汽介质 + 流速足够 + 管道满管 + 安装直管段满足**
- 计算：  
  蒸汽属于涡街流量计常见应用，但需要温压补偿才能得到更准确的质量流量。
- 结论：  
  可评估 **FV系列涡街流量计**，具体管径、温压补偿、量程和输出方式需按厂家手册校核。

> [!tip] 记忆口诀
> 蒸汽气体常找涡街，温压补偿别忘记。

# 02 底层原理

> [!info] 核心结论
> 涡街流量计的本质，是通过检测流体经过阻流体后产生的 **涡街频率** 来计算流速。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 流体经过三角柱或类似阻流体时，会在后方交替产生旋涡。  
> 流速越快，旋涡产生频率越高。  
> 传感器检测这个频率，再换算成流速和流量。

> [!warning] 🟧 涡街核心公式
> **f = St × v ÷ d**  
> f：涡街频率  
> St：斯特劳哈尔数  
> v：流体平均流速  
> d：阻流体特征宽度

## ② 参考表

| 参数 | 含义 |
|---|---|
| **f** | 涡街频率 |
| **St** | 斯特劳哈尔数，和结构有关 |
| **v** | 流体平均流速 |
| **d** | 阻流体特征宽度 |
| **Q** | 体积流量 |
| **核心关系** | 频率越高，流速越大 |

## ③ 计算/选型示例

- 已知：  
  阻流体结构固定，涡街频率升高。
- 公式：  
  **f = St × v ÷ d**
- 计算：  
  St和d基本固定时，f与v近似成正比。
- 结论：  
  涡街流量计本质上先测 **频率**，再换算 **流速和流量**。

> [!tip] 记忆口诀
> 流得越快，涡旋越密；频率越高，流量越大。

# 03 流量计算公式

> [!info] 核心结论
> 涡街流量计通过涡街频率得到流速，再用管道截面积计算体积流量。

## ① 分类/公式/规则

> [!warning] 🟧 流速公式
> **v = f × d ÷ St**

> [!warning] 🟧 体积流量公式
> **Q = v × A**

> [!warning] 🟧 圆管截面积公式
> **A = π × D² ÷ 4**

## ② 参考表

| 符号 | 含义 |
|---|---|
| **v** | 平均流速，m/s |
| **f** | 涡街频率，Hz |
| **d** | 阻流体特征宽度，m |
| **St** | 斯特劳哈尔数 |
| **Q** | 体积流量，m³/s |
| **D** | 管道内径，m |

## ③ 计算/选型示例

- 已知：  
  管道内径 D = **0.1m**，平均流速 v = **8m/s**。
- 公式：  
  **Q = v × π × D² ÷ 4**
- 计算：  
  Q = 8 × 3.14 × 0.1² ÷ 4  
  Q = 8 × 3.14 × 0.01 ÷ 4  
  Q ≈ **0.0628m³/s**  
  换算为小时流量：0.0628 × 3600 ≈ **226m³/h**
- 结论：  
  DN100管道在8m/s流速下，体积流量约为 **226m³/h**。

> [!tip] 记忆口诀
> 先由频率算流速，再由面积算流量。

# 04 适用介质

> [!info] 核心结论
> 涡街流量计适用范围广，但更适合 **清洁、单相、流动稳定** 的蒸汽、气体和液体。

## ① 分类/公式/规则

> [!note] 🟦 介质判断
> 涡街流量计要求流体流过阻流体后能形成稳定涡街。  
> 如果介质含大量杂质、强腐蚀、强振动或两相流明显，测量稳定性会下降。

> [!warning] 🟧 适用判断公式
> **适合涡街 = 单相流体 + 流速足够 + 流场稳定 + 介质不易堵塞**

## ② 参考表

| 介质 | 是否适合 |
|---|---|
| **饱和蒸汽** | 适合，常需温压补偿 |
| **过热蒸汽** | 适合，常需温压补偿 |
| **压缩空气** | 适合 |
| **氮气/氧气等气体** | 需按工况校核 |
| **清水/循环水** | 适合 |
| **高粘度液体** | 需谨慎校核 |
| **含大量杂质浆液** | 通常不优先 |
| **气液两相流** | 不建议直接使用 |

## ③ 计算/选型示例

- 已知：  
  现场要测量压缩空气用量。
- 公式：  
  **适用性 = 气体介质 + 流速范围满足 + 压力温度参数明确**
- 计算：  
  压缩空气属于涡街流量计常见应用，但气体密度受压力和温度影响明显。
- 结论：  
  可选 **FV系列涡街流量计**，建议结合压力、温度进行补偿或换算。

> [!tip] 记忆口诀
> 蒸汽空气水可用，杂质两相要慎重。

# 05 温压补偿

> [!info] 核心结论
> 测蒸汽和气体时，温度、压力会影响密度，因此常需要温压补偿来得到更准确的质量流量。

## ① 分类/公式/规则

> [!note] 🟦 为什么要补偿
> 涡街流量计通常先得到体积流量。  
> 但蒸汽和气体的密度会随温度、压力变化，若不补偿，质量流量或标况流量可能偏差较大。

> [!warning] 🟧 质量流量公式
> **质量流量 qm = 体积流量 Q × 密度 ρ**

## ② 参考表

| 场景 | 是否建议补偿 |
|---|---|
| **液体体积流量** | 一般不强制 |
| **压缩空气标况流量** | 建议 |
| **饱和蒸汽质量流量** | 建议 |
| **过热蒸汽质量流量** | 强烈建议 |
| **温压波动大** | 强烈建议 |
| **只做简单趋势监控** | 可按项目要求简化 |

## ③ 计算/选型示例

- 已知：  
  蒸汽体积流量 Q = **100m³/h**，工况密度 ρ = **3kg/m³**。
- 公式：  
  **qm = Q × ρ**
- 计算：  
  qm = 100 × 3 = **300kg/h**
- 结论：  
  当前质量流量约为 **300kg/h**，实际密度需根据蒸汽压力和温度计算或由补偿仪表获得。

> [!tip] 记忆口诀
> 测体积看流速，测质量看密度；蒸汽气体要补偿。

# 06 管径与量程选型

> [!info] 核心结论
> 涡街流量计不能只按管径选，还要校核最小流量和最大流量是否在可测范围内。

## ① 分类/公式/规则

> [!note] 🟦 选型重点
> 流速太低，涡街信号弱，可能测不出来；  
> 流速太高，压损、噪声和磨损风险增加。

> [!warning] 🟧 流速校核公式
> **v = Q ÷ A = 4Q ÷（π × D²）**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **最小流量** | 是否大于仪表下限 |
| **常用流量** | 是否在稳定测量区 |
| **最大流量** | 是否超过仪表上限 |
| **管径DN** | 与工艺管道匹配 |
| **介质密度** | 气体/蒸汽尤其重要 |
| **压力损失** | 高速流动需关注 |

## ③ 计算/选型示例

- 已知：  
  DN50管道，最大体积流量为 **30m³/h**。
- 公式：  
  **v = Q ÷ A**
- 计算：  
  D = 0.05m  
  Q = 30 ÷ 3600 = **0.00833m³/s**  
  A = 3.14 × 0.05² ÷ 4 ≈ **0.00196m²**  
  v = 0.00833 ÷ 0.00196 ≈ **4.25m/s**
- 结论：  
  最大流速约 **4.25m/s**，是否合适需按介质类型和厂家量程表校核。

> [!tip] 记忆口诀
> 涡街怕低速没信号，量程上下都要查。

# 07 安装要求

> [!info] 核心结论
> 涡街流量计安装时，要保证流场稳定、直管段足够、方向正确，并尽量避开强振动。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 涡街流量计依靠稳定涡街测量，如果安装在弯头、阀门、泵出口附近，流场紊乱会影响精度。

> [!warning] 🟧 安装可靠公式
> **可靠测量 = 稳定流场 + 足够直管段 + 正确流向 + 避免振动 + 参数正确**

## ② 参考表

| 安装项 | 建议 |
|---|---|
| **流向** | 与表体箭头一致 |
| **直管段** | 上游、下游按厂家手册校核 |
| **安装位置** | 避开弯头、阀门、泵出口 |
| **管道振动** | 强振动需加支撑 |
| **满管液体** | 测液体时必须满管 |
| **蒸汽工况** | 注意冷凝水和保温 |

## ③ 计算/选型示例

- 已知：  
  流量计安装在调节阀后面很近的位置，显示波动明显。
- 公式：  
  **波动原因 = 阀后扰流 + 涡街信号不稳定**
- 计算：  
  调节阀后流速分布不均，容易破坏稳定涡街。
- 结论：  
  应调整安装位置，保证足够直管段，具体要求按厂家手册校核。

> [!tip] 记忆口诀
> 涡街要稳流，阀门弯头要避开。

# 08 输出信号与PLC接线

> [!info] 核心结论
> FV系列涡街流量计接PLC时，要确认输出类型、量程设置、供电方式和屏蔽接地。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 4-20mA用于瞬时流量；  
> 脉冲输出用于累计流量；  
> RS485/HART可用于通讯和参数读取；  
> 继电器输出可用于报警或上下限。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型一致 + 量程一致 + 接线正确 + 屏蔽接地良好**

## ② 参考表

| 输出类型 | 用途 |
|---|---|
| **4-20mA** | 瞬时流量模拟量 |
| **脉冲输出** | 累计流量 |
| **RS485** | 数字通讯 |
| **HART** | 参数配置和智能通讯 |
| **继电器输出** | 报警、上下限 |
| **本地显示** | 查看瞬时和累计流量 |

## ③ 计算/选型示例

- 已知：  
  仪表量程为0～500m³/h，对应4-20mA输出，PLC读取电流为 **10.4mA**。
- 公式：  
  **当前流量 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  当前流量 =（10.4 - 4）÷ 16 × 500  
  = 6.4 ÷ 16 × 500  
  = **200m³/h**
- 结论：  
  PLC应显示约 **200m³/h**，前提是PLC量程和仪表量程设置一致。

> [!tip] 记忆口诀
> 4mA零点，20mA满量程；仪表PLC量程要同名。

# 09 常见故障排查

> [!info] 核心结论
> 涡街流量计异常时，先查流速是否过低、管道是否振动、直管段是否不足、量程是否设置错误。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查工况流量，再查安装位置；  
> 先看仪表本地显示，再查PLC换算；  
> 先查振动干扰，再查传感器故障。

> [!warning] 🟧 排错公式
> **测量异常 = 流速过低 + 流场紊乱 + 管道振动 + 参数错误 + 信号干扰 + 介质不适合**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无流量显示** | 流速低于下限、接线错误 |
| **小流量不显示** | 涡街信号太弱 |
| **显示波动大** | 振动、阀后扰流、气液两相 |
| **显示偏大** | 振动干扰、参数错误 |
| **PLC显示不准** | 4-20mA量程换算错误 |
| **通讯失败** | 地址、波特率、协议不匹配 |

## ③ 计算/选型示例

- 已知：  
  管道有流体流动，但仪表小流量时不显示。
- 公式：  
  **涡街可测 = 流速 ≥ 仪表下限流速**
- 计算：  
  如果流速太低，旋涡不稳定，仪表可能无法识别有效频率。
- 结论：  
  应校核最小流量是否低于仪表下限，必要时缩小管径或更换其他类型流量计。

> [!tip] 记忆口诀
> 涡街不是零速表，小流量下限要记牢。

# 10 快速选型表

> [!info] 核心结论
> FV系列涡街流量计选型时，先确认介质和流量范围，再确认温压补偿、安装条件和输出信号。

## ① 分类/公式/规则

> [!note] 🟦 七步选型法
> ① 看介质是蒸汽、气体还是液体  
> ② 看最小/常用/最大流量  
> ③ 看管径DN  
> ④ 看压力和温度  
> ⑤ 看是否需要温压补偿  
> ⑥ 看安装方式和直管段  
> ⑦ 看输出信号

> [!warning] 🟧 最终选型公式
> **推荐型号 = DN管径 + 介质类型 + 流量范围 + 温压等级 + 输出方式 + 安装形式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 蒸汽流量计量 | 涡街 + 温压补偿 |
| 压缩空气计量 | 涡街 + 压力温度参数 |
| 循环水测量 | 校核满管和流速 |
| 瞬时流量给PLC | 4-20mA输出 |
| 累计流量 | 脉冲输出 |
| 上位机通讯 | RS485/HART等 |
| 强振动现场 | 谨慎选型并加强支撑 |

## ③ 计算/选型示例

- 已知：  
  DN80蒸汽管道，需要测瞬时流量和累计流量，并送PLC显示。
- 公式：  
  **型号需求 = DN80 + 蒸汽介质 + 温压补偿 + 4-20mA + 脉冲输出**
- 计算：  
  蒸汽密度受压力温度影响，若要质量流量或准确计量，建议配置温压补偿。
- 结论：  
  可评估 **FV系列涡街流量计**，重点校核量程、温压补偿、直管段和输出信号。

> [!tip] 选型口诀
> 蒸汽气体看涡街，流速下限先校核；温压补偿配得好，计量结果更可靠。
