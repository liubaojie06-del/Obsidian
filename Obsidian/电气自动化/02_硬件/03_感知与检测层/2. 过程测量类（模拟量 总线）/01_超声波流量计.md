# 01 超声波流量计总览

> [!info] 核心结论
> FU300系列超声波流量计通常用于 **非接触式测量管道内液体流量**，适合不方便开孔、不断管、不停产改造的场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 超声波流量计通过安装在管道外部或管道上的换能器，利用超声波在流体中的传播变化来计算流速和流量。

> [!warning] 🟧 通用选型公式
> **超声波流量计选型 = 管径 + 管材 + 壁厚 + 介质类型 + 流量范围 + 安装方式 + 输出信号**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **系列** | FU300系列超声波流量计 |
| **测量对象** | 液体流量 |
| **典型安装** | 外夹式 / 插入式 / 管段式需按型号校核 |
| **核心优势** | 非接触、安装方便、压损小 |
| **常见输出** | 4-20mA、脉冲、RS485等需校核 |
| **重点参数** | 管径、管材、壁厚、介质、温度 |

## ③ 计算/选型示例

- 已知：  
  现场已有一根供水管道，不方便切管安装流量计。
- 公式：  
  **适用性 = 液体介质 + 管道可贴装探头 + 管材参数已知**
- 计算：  
  若管道外表面可安装探头，并且介质为水类液体，可优先考虑外夹式超声波流量计。
- 结论：  
  可评估 **FU300系列超声波流量计**，具体安装方式、管径范围、输出信号需按厂家手册校核。

> [!tip] 记忆口诀
> 不想开孔、不想停产，外夹超声先看看。

# 02 底层原理

> [!info] 核心结论
> 超声波流量计的本质，是通过超声波在顺流和逆流方向的传播时间差，计算管道内流体速度。

## ① 分类/公式/规则

> [!note] 🟦 时差法原理
> 两个超声波换能器分别发射和接收声波。  
> 顺着水流传播时，声波走得更快；逆着水流传播时，声波走得更慢。  
> 仪表通过两者时间差计算流速。

> [!warning] 🟧 核心逻辑
> **顺流时间短 → 逆流时间长 → 产生时间差 → 计算流速 → 换算流量**

> [!warning] 🟧 体积流量公式
> **Q = v × A**

## ② 参考表

| 项目 | 含义 |
|---|---|
| **顺流传播** | 声波方向与液体流向一致 |
| **逆流传播** | 声波方向与液体流向相反 |
| **时间差** | 顺流和逆流传播时间不同 |
| **流速 v** | 根据时间差计算得到 |
| **流量 Q** | 流速 × 管道截面积 |
| **关键条件** | 管道满管、声波耦合良好 |

## ③ 计算/选型示例

- 已知：  
  管道内径 D = **0.1m**，仪表计算得到平均流速 v = **1.5m/s**。
- 公式：  
  **Q = v × π × D² ÷ 4**
- 计算：  
  Q = 1.5 × 3.14 × 0.1² ÷ 4  
  Q ≈ **0.0118m³/s**  
  换算为小时流量：0.0118 × 3600 ≈ **42.5m³/h**
- 结论：  
  该管道当前流量约为 **42.5m³/h**。

> [!tip] 记忆口诀
> 顺流快、逆流慢，时间一差流量算。

# 03 外夹式安装特点

> [!info] 核心结论
> 外夹式超声波流量计最大的优势是 **不用切管、不接触介质、无压力损失**。

## ① 分类/公式/规则

> [!note] 🟦 外夹式特点
> 探头夹在管道外壁，通过耦合剂把超声波传入管道和介质。  
> 它不直接接触液体，适合改造项目、临时测量、腐蚀性液体或不方便停机的管线。

> [!warning] 🟧 外夹式适用公式
> **适合外夹式 = 管道可接触 + 管壁参数已知 + 满管液体 + 声波可穿透**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **安装方式** | 管外夹装 |
| **是否切管** | 不需要 |
| **是否接触介质** | 不接触 |
| **压力损失** | 基本无压损 |
| **维护便利性** | 较好 |
| **关键风险** | 探头安装、耦合、管材参数 |

## ③ 计算/选型示例

- 已知：  
  现场管道为不锈钢管，内部介质为水，不能停产开孔。
- 公式：  
  **安装可行性 = 管道外壁可安装 + 介质可传声 + 管道参数可输入**
- 计算：  
  不锈钢管和水一般可用于超声波传播，且外壁可安装探头。
- 结论：  
  可优先评估 **外夹式FU300超声波流量计**，但需输入管材、壁厚、管径、衬里等参数。

> [!tip] 记忆口诀
> 外夹不破管，关键看耦合和参数。

# 04 适用介质与管道

> [!info] 核心结论
> 超声波流量计适合声波能稳定传播的液体管道，气泡、杂质、非满管会明显影响测量。

## ① 分类/公式/规则

> [!note] 🟦 适用逻辑
> 时差法超声波流量计要求液体能传播声波，并且管道内尽量满管、流态稳定、气泡少。

> [!warning] 🟧 介质判断公式
> **适合超声波 = 液体 + 满管 + 气泡少 + 固体颗粒少 + 管道参数明确**

## ② 参考表

| 对象 | 是否适合 |
|---|---|
| **自来水** | 适合 |
| **冷却水** | 适合 |
| **污水** | 视气泡和杂质情况校核 |
| **酸碱液** | 外夹式可减少接触腐蚀风险 |
| **油类液体** | 需按声学特性校核 |
| **气体/蒸汽** | 通常不适合普通液体型 |
| **非满管流动** | 不适合 |

## ③ 计算/选型示例

- 已知：  
  管道中液体含有大量气泡，流量显示跳动。
- 公式：  
  **信号稳定性 = 声波穿透能力 - 气泡散射影响**
- 计算：  
  气泡会散射和衰减超声波，导致接收信号不稳定。
- 结论：  
  应避开气泡多的位置安装，必要时改用其他测量方案。

> [!tip] 记忆口诀
> 超声怕气泡，满管才可靠。

# 05 管径与流量计算

> [!info] 核心结论
> 超声波流量计最终仍然是通过 **流速 × 管道面积** 得到体积流量。

## ① 分类/公式/规则

> [!warning] 🟧 管道截面积公式
> **A = π × D² ÷ 4**

> [!warning] 🟧 体积流量公式
> **Q = v × A**

> [!warning] 🟧 合并公式
> **Q = v × π × D² ÷ 4**

## ② 参考表

| 符号 | 含义 |
|---|---|
| **Q** | 体积流量，m³/s |
| **v** | 平均流速，m/s |
| **A** | 管道截面积，m² |
| **D** | 管道内径，m |
| **π** | 约等于3.14 |

## ③ 计算/选型示例

- 已知：  
  管道内径 D = **0.2m**，平均流速 v = **1m/s**。
- 公式：  
  **Q = v × π × D² ÷ 4**
- 计算：  
  Q = 1 × 3.14 × 0.2² ÷ 4  
  Q = 3.14 × 0.04 ÷ 4  
  Q ≈ **0.0314m³/s**  
  换算为小时流量：0.0314 × 3600 ≈ **113m³/h**
- 结论：  
  管径200mm、流速1m/s时，流量约为 **113m³/h**。

> [!tip] 记忆口诀
> 管径进平方，管径变大流量猛涨。

# 06 探头安装距离

> [!info] 核心结论
> 超声波探头安装距离必须按仪表计算值设置，距离不准会导致信号弱、读数不稳或无流量。

## ① 分类/公式/规则

> [!note] 🟦 探头距离来源
> 仪表通常会根据 **管径、壁厚、管材、介质、安装方式** 自动计算探头间距。  
> 现场必须按计算结果准确安装。

> [!warning] 🟧 探头间距公式
> **探头间距 = 管径参数 + 壁厚参数 + 管材声速 + 介质声速 + 安装方式**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **管径** | 影响声路长度 |
| **壁厚** | 影响声波穿管路径 |
| **管材** | 不同材料声速不同 |
| **介质** | 不同液体声速不同 |
| **安装方式** | V法、Z法等需按型号校核 |
| **耦合剂** | 影响信号强度 |

## ③ 计算/选型示例

- 已知：  
  仪表计算探头间距为 **128mm**，现场实际装成 **115mm**。
- 公式：  
  **安装误差 = 实际距离 - 计算距离**
- 计算：  
  安装误差 = 115 - 128 = **-13mm**
- 结论：  
  探头距离误差较大，可能导致信号弱或流量不准，应重新按 **128mm** 调整安装。

> [!tip] 记忆口诀
> 探头距离别凭感觉，仪表算多少就装多少。

# 07 输出信号与PLC接线

> [!info] 核心结论
> FU300接PLC或上位机时，重点确认 **4-20mA、脉冲、RS485、继电器输出** 与系统是否匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 4-20mA用于瞬时流量；  
> 脉冲用于累计流量；  
> RS485用于数字通讯；  
> 继电器输出可用于报警、空管、上下限。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型一致 + 量程一致 + 接线正确 + 屏蔽接地良好**

## ② 参考表

| 输出类型 | 用途 |
|---|---|
| **4-20mA** | 瞬时流量连续信号 |
| **脉冲输出** | 累计流量 |
| **RS485** | 通讯读取流量和参数 |
| **继电器输出** | 报警、上下限、状态 |
| **本地显示** | 查看瞬时和累计流量 |
| **按键设置** | 参数配置和调试 |

## ③ 计算/选型示例

- 已知：  
  流量计量程为0～200m³/h，对应4-20mA，PLC读取电流为 **8mA**。
- 公式：  
  **当前流量 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  当前流量 =（8 - 4）÷ 16 × 200  
  = 4 ÷ 16 × 200  
  = **50m³/h**
- 结论：  
  PLC应显示约 **50m³/h**，前提是PLC量程与仪表量程一致。

> [!tip] 记忆口诀
> 4mA是零，20mA是满；量程错了，显示全乱。

# 08 安装要求

> [!info] 核心结论
> 超声波流量计安装时，要保证满管、直管段、探头耦合、避开气泡和强干扰。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 外夹式探头需要清洁管道外壁，涂抹耦合剂，固定牢靠。  
> 安装位置要避开阀门、弯头、泵出口、气泡聚集点和强振动点。

> [!warning] 🟧 安装可靠公式
> **可靠测量 = 满管 + 稳定流场 + 探头耦合良好 + 直管段足够 + 参数输入正确**

## ② 参考表

| 安装项 | 建议 |
|---|---|
| **满管** | 必须保证液体充满管道 |
| **直管段** | 上游下游长度按厂家手册校核 |
| **管壁处理** | 清洁、除锈、去油漆鼓包 |
| **耦合剂** | 涂抹充分，避免气隙 |
| **探头固定** | 防止滑动和振动 |
| **安装位置** | 避开气泡、泵、阀门扰动 |

## ③ 计算/选型示例

- 已知：  
  探头安装在弯头后很近的位置，流量波动明显。
- 公式：  
  **波动原因 = 流场紊乱 + 探头信号不稳定**
- 计算：  
  弯头后流体速度分布不均，影响测量稳定性。
- 结论：  
  应调整到更长直管段位置，具体直管段要求需按厂家手册校核。

> [!tip] 记忆口诀
> 超声测得准，满管、直管、好耦合。

# 09 常见故障排查

> [!info] 核心结论
> FU300测量异常时，优先检查参数输入、探头间距、耦合状态、满管情况和输出量程。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查工艺条件，再查探头安装；  
> 先看仪表信号强度，再看PLC量程换算。

> [!warning] 🟧 排错公式
> **测量异常 = 管道参数错误 + 探头距离错误 + 耦合不良 + 非满管 + 气泡干扰 + 输出量程错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无流量显示** | 无介质、非满管、探头无信号 |
| **信号弱** | 耦合剂不足、管壁脏、间距错 |
| **读数波动** | 气泡、流场紊乱、安装松动 |
| **读数偏大/偏小** | 管径、壁厚、量程参数错误 |
| **PLC显示不准** | 4-20mA缩放错误 |
| **通讯失败** | 地址、波特率、协议不匹配 |

## ③ 计算/选型示例

- 已知：  
  仪表显示正常为80m³/h，但PLC显示为40m³/h。
- 公式：  
  **显示不一致 = 仪表量程 ≠ PLC量程**
- 计算：  
  如果仪表4-20mA对应0～200m³/h，而PLC按0～100m³/h换算，PLC显示会偏小。
- 结论：  
  应检查PLC模拟量缩放参数，确保与仪表4-20mA量程一致。

> [!tip] 记忆口诀
> 仪表准、PLC错，先查模拟量换算。

# 10 快速选型表

> [!info] 核心结论
> FU300系列超声波流量计选型时，先确认管道和介质能不能测，再确认量程、安装方式和输出信号。

## ① 分类/公式/规则

> [!note] 🟦 七步选型法
> ① 看介质是否适合声波传播  
> ② 看是否满管  
> ③ 看管径和壁厚  
> ④ 看管材和衬里  
> ⑤ 看流量范围  
> ⑥ 看安装方式  
> ⑦ 看输出信号

> [!warning] 🟧 最终选型公式
> **推荐型号 = 管径范围 + 介质类型 + 安装方式 + 流量范围 + 输出方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 不开孔测流量 | **外夹式超声波流量计** |
| 临时测量 | 便携或外夹式方案 |
| 连续流量给PLC | 4-20mA输出 |
| 累计流量 | 脉冲输出 |
| 上位机读取 | RS485等通讯 |
| 管道有气泡 | 需换位置或换方案 |
| 管道参数不清 | 需先测管径和壁厚 |

## ③ 计算/选型示例

- 已知：  
  DN100供水管道，需要不断管测量瞬时流量，并把数据送到PLC。
- 公式：  
  **型号需求 = DN100 + 液体满管 + 外夹式 + 4-20mA输出**
- 计算：  
  DN100供水管道通常适合外夹式超声波测量，但需输入管径、壁厚、管材和介质参数。
- 结论：  
  可评估 **FU300系列外夹式超声波流量计，4-20mA输出**，具体型号需按厂家手册校核。

> [!tip] 选型口诀
> 先看管道满不满，再看介质传不传；探头装得准，流量才可信。
