# 01 电磁流量计总览

> [!info] 核心结论
> FE700系列电磁流量计用于测量 **导电液体的体积流量**，常见于水处理、化工、食品、制药、环保等场景。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 电磁流量计不是靠叶轮转动测流量，而是通过导电液体切割磁力线产生感应电压，再换算流速和流量。

> [!warning] 🟧 通用选型公式
> **电磁流量计选型 = 介质导电性 + 管径DN + 流量范围 + 衬里材料 + 电极材料 + 输出信号 + 安装环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **系列** | FE700系列电磁流量计 |
| **测量对象** | 导电液体 |
| **不适合对象** | 气体、蒸汽、油类、非导电介质 |
| **常见输出** | 4-20mA、脉冲、RS485、HART等需按型号校核 |
| **安装方式** | 法兰式、夹装式等需按型号校核 |
| **重点参数** | 管径、流量范围、压力、温度、导电率 |

## ③ 计算/选型示例

- 已知：  
  需要测量一条清水管道的瞬时流量和累计流量。
- 公式：  
  **适用性 = 液体导电 + 满管流动 + 流速在合理范围**
- 计算：  
  清水通常具有一定导电性，适合电磁流量计测量。
- 结论：  
  可选 **FE700系列电磁流量计**，具体管径、衬里、电极、输出方式需按厂家手册校核。

> [!tip] 记忆口诀
> 导电液体用电磁，不导电介质别乱用。

# 02 底层原理

> [!info] 核心结论
> 电磁流量计基于 **法拉第电磁感应定律**，导电液体在磁场中流动会产生感应电压。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 流量计内部产生磁场，导电液体流过测量管时，相当于导体切割磁力线。  
> 电极检测到感应电压，仪表根据电压大小换算出流速，再计算体积流量。

> [!warning] 🟧 感应电压公式
> **E = k × B × D × v**  
> E：感应电压  
> k：仪表系数  
> B：磁感应强度  
> D：管道内径  
> v：平均流速

## ② 参考表

| 参数 | 含义 |
|---|---|
| **E** | 电极检测到的感应电压 |
| **B** | 磁场强度 |
| **D** | 测量管内径 |
| **v** | 介质平均流速 |
| **Q** | 体积流量 |
| **关键点** | 电压与流速成正比 |

## ③ 计算/选型示例

- 已知：  
  管径固定，磁场强度固定，液体流速变大。
- 公式：  
  **E = k × B × D × v**
- 计算：  
  当 v 增大时，E 同比例增大。
- 结论：  
  电磁流量计本质上先测 **流速**，再根据管径换算成 **流量**。

> [!tip] 记忆口诀
> 液体切磁线，电极生电压；电压看流速，流速算流量。

# 03 流量计算公式

> [!info] 核心结论
> 电磁流量计的体积流量由 **流速 × 管道截面积** 得到。

## ① 分类/公式/规则

> [!warning] 🟧 体积流量公式
> **Q = v × A**

> [!warning] 🟧 圆管截面积公式
> **A = π × D² ÷ 4**

> [!warning] 🟧 合并公式
> **Q = v × π × D² ÷ 4**

## ② 参考表

| 符号 | 含义 |
|---|---|
| **Q** | 体积流量，m³/s |
| **v** | 平均流速，m/s |
| **A** | 管道截面积，m² |
| **D** | 管道内径，m |
| **π** | 约等于3.14 |

## ③ 计算/选型示例

- 已知：  
  管道内径 D = **0.1m**，平均流速 v = **2m/s**。
- 公式：  
  **Q = v × π × D² ÷ 4**
- 计算：  
  Q = 2 × 3.14 × 0.1² ÷ 4  
  Q = 2 × 3.14 × 0.01 ÷ 4  
  Q ≈ **0.0157m³/s**  
  换算为小时流量：0.0157 × 3600 ≈ **56.5m³/h**
- 结论：  
  该管道在2m/s流速下，流量约为 **56.5m³/h**。

> [!tip] 记忆口诀
> 流量等于流速乘面积，管径一变流量大变。

# 04 适用介质

> [!info] 核心结论
> 电磁流量计只能测 **导电液体**，介质不导电时无法正常测量。

## ① 分类/公式/规则

> [!note] 🟦 适用逻辑
> 电磁流量计依赖导电介质产生感应电压。  
> 如果介质不导电，电极无法获得有效信号。

> [!warning] 🟧 介质判断公式
> **适合电磁流量计 = 液体 + 导电 + 满管 + 流动稳定**

## ② 参考表

| 介质 | 是否适合 |
|---|---|
| **自来水** | 适合 |
| **污水** | 适合 |
| **酸碱液** | 需选合适衬里和电极 |
| **浆液** | 需校核磨损和沉积 |
| **油类** | 通常不适合 |
| **气体/蒸汽** | 不适合 |
| **纯水/超纯水** | 需校核导电率 |

## ③ 计算/选型示例

- 已知：  
  现场介质为柴油，需要测量管道流量。
- 公式：  
  **电磁适用性 = 介质导电性**
- 计算：  
  柴油通常导电性很低，不适合电磁感应测量。
- 结论：  
  不建议选电磁流量计，应评估涡轮、椭圆齿轮、科里奥利等其他流量计方案。

> [!tip] 记忆口诀
> 电磁流量计怕不导电，油气蒸汽别硬选。

# 05 管径与量程选型

> [!info] 核心结论
> 管径选型不能只按管道尺寸，还要看实际流量对应的流速是否合适。

## ① 分类/公式/规则

> [!note] 🟦 选型思路
> 管径太大，流速过低，信号弱；  
> 管径太小，流速过高，压损和冲刷风险增加。

> [!warning] 🟧 管径校核公式
> **v = Q ÷ A = 4Q ÷（π × D²）**

> [!warning] 🟧 常用经验
> 液体常用流速范围可参考 **0.5～10m/s**，具体需按介质和厂家手册校核。

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **DN管径** | 与工艺管道匹配 |
| **最小流量** | 低流量时是否能稳定测量 |
| **最大流量** | 高流量时是否超量程 |
| **常用流量** | 应落在仪表较稳定区间 |
| **浆液介质** | 注意沉积和磨损 |
| **腐蚀介质** | 注意衬里和电极材料 |

## ③ 计算/选型示例

- 已知：  
  最大流量 Q = **50m³/h**，管道内径 D = **0.1m**。
- 公式：  
  **v = Q ÷ A**
- 计算：  
  Q = 50 ÷ 3600 ≈ **0.0139m³/s**  
  A = 3.14 × 0.1² ÷ 4 ≈ **0.00785m²**  
  v = 0.0139 ÷ 0.00785 ≈ **1.77m/s**
- 结论：  
  流速约 **1.77m/s**，通常属于较合理范围，最终需按厂家手册校核。

> [!tip] 记忆口诀
> 选管径先算流速，流速合适测量才稳。

# 06 衬里与电极材料

> [!info] 核心结论
> 衬里和电极直接接触介质，选错会导致腐蚀、磨损、泄漏或测量不稳定。

## ① 分类/公式/规则

> [!note] 🟦 材料选择逻辑
> 衬里主要承受介质腐蚀、温度、磨损；  
> 电极主要承受电化学腐蚀和介质污染。

> [!warning] 🟧 材料选型公式
> **材料选择 = 介质腐蚀性 + 温度 + 压力 + 磨损性 + 卫生要求**

## ② 参考表

| 部件 | 选型关注点 |
|---|---|
| **衬里** | 耐腐蚀、耐温、耐磨 |
| **电极** | 耐腐蚀、抗污染 |
| **接地环** | 介质接地和信号稳定 |
| **法兰** | 压力等级和连接标准 |
| **密封件** | 介质兼容性 |
| **外壳** | 防护等级和安装环境 |

## ③ 计算/选型示例

- 已知：  
  介质为腐蚀性酸液，需要长期连续测量。
- 公式：  
  **可靠性 = 衬里耐腐蚀 + 电极耐腐蚀 + 密封可靠**
- 计算：  
  普通材料可能被腐蚀，导致电极损坏或衬里失效。
- 结论：  
  必须按介质化学性质选择衬里和电极，具体材料需按厂家选型表校核。

> [!tip] 记忆口诀
> 流量准不准看信号，寿命长不长看材料。

# 07 安装要求

> [!info] 核心结论
> 电磁流量计安装时必须保证 **满管、接地良好、流向正确、直管段合理**。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 电磁流量计最怕非满管、气泡、强电磁干扰、接地不良和前后流场紊乱。

> [!warning] 🟧 安装可靠公式
> **可靠测量 = 满管流动 + 稳定流场 + 良好接地 + 正确流向 + 合理直管段**

## ② 参考表

| 安装项 | 建议 |
|---|---|
| **流向** | 与表体箭头一致 |
| **满管** | 必须保证测量管充满液体 |
| **直管段** | 上游、下游长度需按厂家手册校核 |
| **安装姿态** | 避免电极处积气或沉积 |
| **接地** | 接地环或接地线按要求连接 |
| **避开干扰** | 远离大功率变频器和强电磁源 |

## ③ 计算/选型示例

- 已知：  
  流量计安装在管道最高点，现场测量值经常跳动。
- 公式：  
  **测量异常 = 非满管 + 气泡 + 信号不稳定**
- 计算：  
  管道最高点容易积气，导致电极接触不稳定，测量信号波动。
- 结论：  
  不建议安装在最高点，应调整到能保证满管的位置。

> [!tip] 记忆口诀
> 电磁要满管，接地要可靠；箭头别装反，气泡要避开。

# 08 输出信号与PLC接线

> [!info] 核心结论
> FE700电磁流量计接PLC或上位机时，要确认输出类型、供电方式、量程对应关系和通讯协议。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 4-20mA用于瞬时流量连续信号；  
> 脉冲输出用于累计流量；  
> RS485/HART等用于数字通讯和参数读取。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型匹配 + 供电正确 + 量程设置一致 + 屏蔽接地正确**

## ② 参考表

| 输出类型 | 用途 |
|---|---|
| **4-20mA** | 瞬时流量模拟量 |
| **脉冲输出** | 累计流量计数 |
| **继电器输出** | 报警、空管、上下限 |
| **RS485** | 数字通讯，需协议匹配 |
| **HART** | 参数配置和智能通讯 |
| **显示屏** | 本地查看瞬时/累计流量 |

## ③ 计算/选型示例

- 已知：  
  仪表量程设置为 0～100m³/h，对应4-20mA输出。PLC读取电流为12mA。
- 公式：  
  **当前流量 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  当前流量 =（12 - 4）÷ 16 × 100  
  = 8 ÷ 16 × 100  
  = **50m³/h**
- 结论：  
  PLC当前应显示约 **50m³/h**，前提是PLC量程和仪表量程设置一致。

> [!tip] 记忆口诀
> 4mA是零点，20mA是满量程；量程不一致，显示就不准。

# 09 常见故障排查

> [!info] 核心结论
> 电磁流量计故障排查，优先检查介质是否满管、接地是否可靠、参数是否一致、信号线是否受干扰。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查工艺条件，再查安装接地；  
> 先查仪表显示，再查PLC量程；  
> 先查现场信号，再查程序换算。

> [!warning] 🟧 排错公式
> **测量异常 = 非满管 + 接地不良 + 气泡 + 电极污染 + 参数错误 + 信号干扰**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无流量显示** | 无介质、空管、接线错误 |
| **显示波动大** | 气泡、非满管、接地差 |
| **显示偏小** | 管径参数错、电极污染、量程不匹配 |
| **显示偏大** | 零点漂移、干扰、参数错误 |
| **PLC显示不准** | 4-20mA量程换算错误 |
| **通讯失败** | 地址、波特率、协议不匹配 |

## ③ 计算/选型示例

- 已知：  
  仪表本地显示为30m³/h，但PLC显示为60m³/h。
- 公式：  
  **显示不一致 = 仪表量程 ≠ PLC模拟量量程**
- 计算：  
  如果仪表4-20mA对应0～100m³/h，而PLC程序按0～200m³/h换算，PLC会显示偏大。
- 结论：  
  应检查PLC模拟量缩放参数，确保与仪表量程一致。

> [!tip] 记忆口诀
> 仪表准、PLC不准，先查量程换算。

# 10 快速选型表

> [!info] 核心结论
> FE700系列电磁流量计选型时，先确认介质能不能测，再确认管径、流量、材料、输出和安装条件。

## ① 分类/公式/规则

> [!note] 🟦 七步选型法
> ① 看介质是否导电  
> ② 看管径DN  
> ③ 看最小/常用/最大流量  
> ④ 看压力和温度  
> ⑤ 看衬里和电极材料  
> ⑥ 看输出信号  
> ⑦ 看安装环境

> [!warning] 🟧 最终选型公式
> **推荐型号 = DN管径 + 流量范围 + 衬里材料 + 电极材料 + 压力等级 + 输出方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 清水/污水测量 | 常规电磁流量计 |
| 腐蚀性液体 | 衬里和电极重点校核 |
| 浆液测量 | 耐磨和沉积重点校核 |
| 连续流量给PLC | 4-20mA输出 |
| 累计流量计数 | 脉冲输出 |
| 上位机通讯 | RS485/HART等 |
| 户外安装 | 防护等级和防雷接地 |

## ③ 计算/选型示例

- 已知：  
  管道DN100，介质为污水，常用流量约50m³/h，需要PLC读取瞬时流量。
- 公式：  
  **型号需求 = DN100 + 污水介质 + 50m³/h量程 + 4-20mA输出**
- 计算：  
  DN100在50m³/h时流速约1.77m/s，通常较合理。
- 结论：  
  可评估 **FE700系列DN100电磁流量计，4-20mA输出**，衬里、电极、压力、温度、防护等级需按厂家手册校核。

> [!tip] 选型口诀
> 先看导电，再算流速；材料抗介质，输出配系统。
