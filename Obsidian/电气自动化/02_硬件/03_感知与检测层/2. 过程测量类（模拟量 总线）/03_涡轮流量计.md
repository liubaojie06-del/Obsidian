# 01 涡轮流量计总览

> [!info] 核心结论
> FT系列涡轮流量计通过测量叶轮转速来计算流量，适合 **清洁、低粘度、稳定流动** 的液体或气体计量。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 涡轮流量计内部有叶轮，流体通过时推动叶轮旋转。  
> 流速越快，叶轮转得越快，传感器输出的脉冲频率越高。

> [!warning] 🟧 通用选型公式
> **涡轮流量计选型 = 介质类型 + 管径DN + 流量范围 + 粘度 + 压力温度 + 输出信号 + 安装条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **系列** | FT系列涡轮流量计 |
| **测量原理** | 叶轮转速与流速成比例 |
| **适合介质** | 清洁液体、部分气体 |
| **不适合介质** | 高粘度、含杂质、强腐蚀、浆液 |
| **常见输出** | 脉冲、4-20mA、RS485等需校核 |
| **重点参数** | 流量范围、粘度、压力、温度、精度 |

## ③ 计算/选型示例

- 已知：  
  需要测量一条清水管道的累计流量。
- 公式：  
  **适用性 = 清洁液体 + 流速稳定 + 流量范围满足**
- 计算：  
  清水粘度低、杂质少，适合涡轮流量计。
- 结论：  
  可评估 **FT系列涡轮流量计**，具体管径、量程、输出方式和安装要求需按厂家手册校核。

> [!tip] 记忆口诀
> 清洁低粘选涡轮，脏污浆液要慎用。

# 02 底层原理

> [!info] 核心结论
> 涡轮流量计的本质，是把流体速度转换成 **叶轮转速**，再把转速转换成 **脉冲频率**。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 流体进入测量管后推动叶轮旋转。  
> 叶轮上的磁性元件或叶片经过检测线圈/传感器时，会产生脉冲信号。  
> 流速越大，单位时间内脉冲越多。

> [!warning] 🟧 涡轮检测逻辑
> **流体流动 → 推动叶轮 → 叶轮旋转 → 产生脉冲 → 换算流量**

## ② 参考表

| 参数 | 含义 |
|---|---|
| **叶轮转速** | 反映流体流速 |
| **脉冲频率 f** | 单位时间内输出脉冲数 |
| **仪表系数 K** | 每单位体积对应的脉冲数 |
| **瞬时流量 Q** | 根据频率换算 |
| **累计流量** | 根据脉冲总数累加 |
| **关键点** | 频率越高，流量越大 |

## ③ 计算/选型示例

- 已知：  
  流体流速升高，叶轮转速也随之升高。
- 公式：  
  **流量 ∝ 叶轮转速 ∝ 脉冲频率**
- 计算：  
  在仪表线性范围内，流速越大，脉冲频率越高。
- 结论：  
  涡轮流量计本质上是通过 **频率信号** 反映流量大小。

> [!tip] 记忆口诀
> 流体推叶轮，叶轮发脉冲；脉冲越密，流量越大。

# 03 流量计算公式

> [!info] 核心结论
> 涡轮流量计常用 **仪表系数K** 把脉冲频率换算成瞬时流量。

## ① 分类/公式/规则

> [!warning] 🟧 频率流量公式
> **Q = f ÷ K**

> [!warning] 🟧 累计流量公式
> **V = N ÷ K**

> [!note] 🟦 参数说明
> Q：瞬时流量  
> f：脉冲频率  
> K：仪表系数  
> V：累计体积  
> N：累计脉冲数

## ② 参考表

| 符号 | 含义 |
|---|---|
| **Q** | 瞬时流量 |
| **f** | 脉冲频率，Hz |
| **K** | 仪表系数，脉冲/单位体积 |
| **V** | 累计体积 |
| **N** | 累计脉冲数 |
| **注意** | 单位必须统一 |

## ③ 计算/选型示例

- 已知：  
  仪表系数 K = **1000 pulse/L**，当前频率 f = **50Hz**。
- 公式：  
  **Q = f ÷ K**
- 计算：  
  Q = 50 ÷ 1000 = **0.05L/s**  
  换算为分钟流量：0.05 × 60 = **3L/min**
- 结论：  
  当前瞬时流量约为 **3L/min**，实际计算需确认K系数单位。

> [!tip] 记忆口诀
> 频率算瞬时，脉冲算累计；K值不对，流量全错。

# 04 适用介质

> [!info] 核心结论
> 涡轮流量计更适合干净、低粘度、流动稳定的介质，不适合杂质多或粘度大的工况。

## ① 分类/公式/规则

> [!note] 🟦 介质判断
> 涡轮流量计内部有运动部件，介质越干净、润滑性越合适、流动越稳定，测量越可靠。

> [!warning] 🟧 适用判断公式
> **适合涡轮 = 清洁介质 + 低粘度 + 无大量颗粒 + 流速在量程内**

## ② 参考表

| 介质 | 是否适合 |
|---|---|
| **清水** | 适合 |
| **柴油/轻油** | 需按粘度校核 |
| **化学液体** | 需校核材质兼容 |
| **压缩空气** | 需选气体型并校核 |
| **高粘度油** | 不优先 |
| **含颗粒污水** | 不建议 |
| **浆液** | 不适合 |
| **强腐蚀介质** | 需特殊材质校核 |

## ③ 计算/选型示例

- 已知：  
  现场介质中含有较多颗粒杂质。
- 公式：  
  **风险 = 颗粒杂质 + 叶轮卡滞 + 轴承磨损**
- 计算：  
  杂质可能堵塞叶轮或磨损轴承，导致测量不准甚至损坏。
- 结论：  
  不建议直接使用涡轮流量计；如必须使用，应加过滤器并按厂家手册校核。

> [!tip] 记忆口诀
> 涡轮怕脏怕粘，介质干净才耐用。

# 05 管径与量程选型

> [!info] 核心结论
> 涡轮流量计必须让常用流量落在仪表线性范围内，过低或过高都会影响测量。

## ① 分类/公式/规则

> [!note] 🟦 选型重点
> 流量太小，叶轮转不稳；  
> 流量太大，叶轮磨损、压损和噪声会增加。

> [!warning] 🟧 流速校核公式
> **v = Q ÷ A = 4Q ÷（π × D²）**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **最小流量** | 是否高于仪表下限 |
| **常用流量** | 应落在线性稳定区 |
| **最大流量** | 不应超过仪表上限 |
| **管径DN** | 与工艺管道匹配 |
| **压损** | 高流速时需关注 |
| **粘度** | 影响叶轮转动和系数 |

## ③ 计算/选型示例

- 已知：  
  DN25管道，最大流量为 **6m³/h**。
- 公式：  
  **v = Q ÷ A**
- 计算：  
  D = 0.025m  
  Q = 6 ÷ 3600 = **0.00167m³/s**  
  A = 3.14 × 0.025² ÷ 4 ≈ **0.00049m²**  
  v = 0.00167 ÷ 0.00049 ≈ **3.4m/s**
- 结论：  
  最大流速约 **3.4m/s**，是否合适需按介质和厂家量程表校核。

> [!tip] 记忆口诀
> 选涡轮先查量程，小流量别低于下限。

# 06 过滤器与直管段

> [!info] 核心结论
> 涡轮流量计对流场和介质清洁度比较敏感，通常需要前置过滤和足够直管段。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 叶轮需要稳定流场推动。  
> 如果前方有弯头、阀门、泵出口或杂质冲击，容易导致读数波动或叶轮磨损。

> [!warning] 🟧 安装可靠公式
> **可靠测量 = 过滤器 + 稳定流场 + 足够直管段 + 正确流向 + 无强振动**

## ② 参考表

| 项目 | 建议 |
|---|---|
| **过滤器** | 建议安装在上游 |
| **流向** | 与表体箭头一致 |
| **上游直管段** | 按厂家手册校核 |
| **下游直管段** | 按厂家手册校核 |
| **阀门位置** | 尽量不要紧贴流量计前端 |
| **泵出口** | 避免直接安装在强扰流位置 |

## ③ 计算/选型示例

- 已知：  
  涡轮流量计安装在泵出口后很近的位置，读数波动明显。
- 公式：  
  **波动原因 = 泵出口扰流 + 叶轮受力不稳定**
- 计算：  
  泵出口流场紊乱，叶轮转速会不稳定。
- 结论：  
  应调整安装位置，增加直管段或整流措施，具体要求按厂家手册校核。

> [!tip] 记忆口诀
> 涡轮要吃干净水，也要吃稳定流。

# 07 输出信号与PLC接线

> [!info] 核心结论
> FT系列涡轮流量计常见输出有 **脉冲、4-20mA、RS485**，接PLC时要确认信号类型和量程换算。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 脉冲输出适合累计流量和频率计算；  
> 4-20mA适合瞬时流量连续显示；  
> RS485适合上位机或PLC通讯读取。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型一致 + 电源正确 + K系数正确 + 量程一致 + 屏蔽接地良好**

## ② 参考表

| 输出类型 | 用途 |
|---|---|
| **脉冲输出** | 累计流量、频率流量 |
| **4-20mA** | 瞬时流量 |
| **RS485** | 通讯读取数据 |
| **继电器输出** | 报警或上下限 |
| **本地显示** | 查看瞬时/累计流量 |
| **屏蔽线** | 抗干扰，建议规范接地 |

## ③ 计算/选型示例

- 已知：  
  仪表量程0～60m³/h对应4-20mA，PLC读取电流为 **12mA**。
- 公式：  
  **当前流量 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  当前流量 =（12 - 4）÷ 16 × 60  
  = 8 ÷ 16 × 60  
  = **30m³/h**
- 结论：  
  PLC应显示约 **30m³/h**，前提是PLC模拟量量程与仪表一致。

> [!tip] 记忆口诀
> 脉冲看K值，模拟看量程；接线不匹配，PLC读不准。

# 08 安装注意事项

> [!info] 核心结论
> 涡轮流量计安装时，要避免反向安装、空管运行、强冲击流、强振动和杂质进入。

## ① 分类/公式/规则

> [!note] 🟦 安装步骤
> ① 确认流向  
> ② 清理管道杂质  
> ③ 安装过滤器  
> ④ 保证直管段  
> ⑤ 缓慢开阀充液  
> ⑥ 检查泄漏和信号

> [!warning] 🟧 安装检查公式
> **安装合格 = 流向正确 + 管道清洁 + 过滤有效 + 流场稳定 + 接线正确**

## ② 参考表

| 检查项 | 建议 |
|---|---|
| **流向箭头** | 必须与实际流向一致 |
| **管道清洁** | 安装前冲洗管道 |
| **过滤器** | 防止杂质卡叶轮 |
| **阀门操作** | 避免瞬间冲击 |
| **安装姿态** | 按厂家要求 |
| **维护空间** | 方便拆洗和检修 |

## ③ 计算/选型示例

- 已知：  
  新管道安装后未冲洗，直接运行流量计。
- 公式：  
  **损坏风险 = 焊渣杂质 + 叶轮卡滞 + 轴承磨损**
- 计算：  
  管道残留焊渣、铁屑可能直接损坏叶轮。
- 结论：  
  新管道投运前必须冲洗，并建议安装过滤器。

> [!tip] 记忆口诀
> 新管先冲洗，涡轮再上岗。

# 09 常见故障排查

> [!info] 核心结论
> 涡轮流量计异常时，优先检查叶轮是否卡滞、介质是否干净、流量是否低于下限、K系数是否正确。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查现场流量，再查叶轮状态；  
> 先看仪表显示，再查PLC换算；  
> 先查介质杂质，再查传感器故障。

> [!warning] 🟧 排错公式
> **测量异常 = 叶轮卡滞 + 杂质堵塞 + 流量过低 + K系数错误 + 接线错误 + 信号干扰**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无流量显示** | 叶轮不转、流量太小、接线错误 |
| **显示偏小** | 叶轮磨损、杂质卡阻、K系数错误 |
| **显示偏大** | 干扰脉冲、参数错误 |
| **读数波动** | 流场不稳、气泡、阀门扰动 |
| **累计不准** | K系数或脉冲倍率错误 |
| **PLC无信号** | 输出类型或公共端错误 |

## ③ 计算/选型示例

- 已知：  
  仪表本地累计流量正常，但PLC累计偏大。
- 公式：  
  **PLC累计 = 脉冲数 ÷ K系数**
- 计算：  
  如果PLC中K系数设置过小，同样脉冲数会换算出更大的累计流量。
- 结论：  
  应核对仪表铭牌K系数、PLC程序K系数和单位是否一致。

> [!tip] 记忆口诀
> 涡轮不准先看轮，累计不准先看K。

# 10 快速选型表

> [!info] 核心结论
> FT系列涡轮流量计选型时，先确认介质是否适合叶轮式测量，再确认量程、K系数、输出和安装条件。

## ① 分类/公式/规则

> [!note] 🟦 七步选型法
> ① 看介质是否清洁  
> ② 看粘度是否合适  
> ③ 看最小/常用/最大流量  
> ④ 看管径DN  
> ⑤ 看压力和温度  
> ⑥ 看输出信号  
> ⑦ 看过滤和安装条件

> [!warning] 🟧 最终选型公式
> **推荐型号 = DN管径 + 流量范围 + 介质类型 + 压力等级 + 输出方式 + K系数 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 清水计量 | 涡轮流量计适合 |
| 轻油计量 | 校核粘度和材质 |
| 累计流量 | 脉冲输出 + K系数 |
| 瞬时流量给PLC | 4-20mA输出 |
| 上位机通讯 | RS485等通讯 |
| 介质有杂质 | 加过滤器或换方案 |
| 小流量测量 | 校核最小启动流量 |

## ③ 计算/选型示例

- 已知：  
  管道DN25，介质为清水，常用流量3m³/h，需要PLC累计流量。
- 公式：  
  **型号需求 = DN25 + 清水 + 3m³/h + 脉冲输出 + K系数输入PLC**
- 计算：  
  清水适合涡轮测量，PLC累计需要正确设置K系数。
- 结论：  
  可评估 **FT系列DN25涡轮流量计，脉冲输出**，量程范围、K系数、压力温度需按厂家手册校核。

> [!tip] 选型口诀
> 介质先看干不干净，量程再看够不够用；脉冲记住K，安装记住过滤。
