# 01 激光传感器总览

> [!info] 核心结论
> 激光传感器利用激光光束检测 **距离、位移、位置、有无、尺寸、轮廓和高度差**，适合小目标、高精度、远距离或复杂表面检测。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 普通光电传感器多用于“有无检测”；  
> 激光传感器除了能检测有无，还常用于 **精密测距、位移检测、高度检测、尺寸检测**。

> [!warning] 🟧 通用选型公式
> **激光传感器选型 = 检测目的 + 测量范围 + 精度要求 + 目标材质 + 输出方式 + 安装环境**

## ② 参考表

| 类型 | 主要用途 |
|---|---|
| **激光位移传感器** | 高度、厚度、位置、振动 |
| **激光测距传感器** | 中远距离距离测量 |
| **激光光电传感器** | 小目标有无、边缘检测 |
| **激光轮廓传感器** | 轮廓、宽度、高度差 |
| **激光对射传感器** | 精准遮挡检测 |
| **激光漫反射传感器** | 单侧安装检测目标 |

## ③ 计算/选型示例

- 已知：  
  需要检测一个零件是否装到位，并且到位误差要求很小。
- 公式：  
  **检测需求 = 小目标 + 高精度 + 位置判断**
- 计算：  
  普通光电可能只能判断有无，无法稳定判断细微高度差。
- 结论：  
  可选择 **激光位移传感器** 或高精度激光检测传感器，具体量程和精度需按厂家手册校核。

> [!tip] 记忆口诀
> 普通光电看有无，激光还能看距离和高度。

# 02 底层原理

> [!info] 核心结论
> 激光传感器的本质，是发射一束激光，再根据反射光的位置、时间或强度变化计算目标状态。

## ① 分类/公式/规则

> [!note] 🟦 常见原理
> 激光传感器常见原理有：  
> **三角测量法**：适合短距离高精度位移检测。  
> **飞行时间法 TOF**：适合中远距离测距。  
> **遮光检测法**：适合对射、边缘、有无检测。

> [!warning] 🟧 检测逻辑
> **发射激光 → 照射目标 → 接收反射/遮挡信号 → 电路计算 → 输出距离或开关信号**

## ② 参考表

| 原理 | 适合场景 |
|---|---|
| **三角测量法** | 高精度短距离位移 |
| **TOF飞行时间法** | 中远距离测距 |
| **相位法** | 稳定测距，需按型号校核 |
| **对射遮光** | 小物体通过检测 |
| **漫反射** | 单侧检测目标 |
| **轮廓扫描** | 尺寸、形状、高度差 |

## ③ 计算/选型示例

- 已知：  
  需要检测产品表面高度变化，测量范围在几十毫米内。
- 公式：  
  **高精度短距离测量 → 三角测量法**
- 计算：  
  三角测量法通过反射光落点变化判断距离，适合精密位移检测。
- 结论：  
  优先考虑 **三角测量激光位移传感器**。

> [!tip] 记忆口诀
> 近距离高精度看三角，远距离测距看TOF。

# 03 激光位移传感器

> [!info] 核心结论
> 激光位移传感器用于测量目标表面到传感器的距离变化，常用于高度、厚度、平面度和位置检测。

## ① 分类/公式/规则

> [!note] 🟦 典型用途
> 激光位移传感器常用于：  
> 零件高度检测、压装到位检测、板材厚度检测、振动检测、胶高检测、工件翘曲检测。

> [!warning] 🟧 位移检测公式
> **位移变化 = 当前距离 - 基准距离**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **测量范围** | 可测距离区间 |
| **基准距离** | 推荐安装中心距离 |
| **分辨率** | 能分辨的最小变化 |
| **重复精度** | 多次测量一致性 |
| **线性精度** | 测量值与真实值偏差 |
| **采样周期** | 影响高速检测能力 |

## ③ 计算/选型示例

- 已知：  
  基准距离为 **50.00mm**，当前测量距离为 **49.85mm**。
- 公式：  
  **位移变化 = 当前距离 - 基准距离**
- 计算：  
  位移变化 = 49.85 - 50.00 = **-0.15mm**
- 结论：  
  当前目标比基准位置靠近传感器 **0.15mm**。

> [!tip] 记忆口诀
> 基准先定好，差值就是位移。

# 04 激光测距传感器

> [!info] 核心结论
> 激光测距传感器适合测量较远距离，常用于堆垛机、AGV、料位、行车定位和大尺寸设备定位。

## ① 分类/公式/规则

> [!note] 🟦 使用场景
> 当普通光电距离不够、超声波受环境影响较大，或需要较远距离定位时，可考虑激光测距。

> [!warning] 🟧 测距判断公式
> **适合激光测距 = 距离较远 + 目标反射稳定 + 需要连续距离输出**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **量程** | 最大测量距离 |
| **盲区** | 最近可测距离 |
| **精度** | 距离误差范围 |
| **目标反射率** | 黑色、亮面会影响 |
| **输出方式** | 模拟量、开关量、通讯 |
| **环境光** | 强光环境需校核 |

## ③ 计算/选型示例

- 已知：  
  需要检测堆垛机与货架之间距离，最大距离约 **8m**。
- 公式：  
  **传感器量程 ≥ 最大距离 × 安全系数**
- 计算：  
  8m × 1.2 = **9.6m**
- 结论：  
  建议选择量程 **≥10m** 的激光测距传感器，反射板或目标反射率需按厂家手册校核。

> [!tip] 记忆口诀
> 远距离定位，用激光更清晰。

# 05 激光光电检测

> [!info] 核心结论
> 激光光电传感器光斑小、方向性强，适合检测小目标、细线、孔位、边缘和窄缝。

## ① 分类/公式/规则

> [!note] 🟦 核心优势
> 激光光斑比普通LED光电更集中，适合小物体或高定位精度检测。

> [!warning] 🟧 小目标检测公式
> **可靠检测 = 光斑尺寸 ≤ 目标有效尺寸**

## ② 参考表

| 检测对象 | 推荐方向 |
|---|---|
| **小零件** | 小光斑激光检测 |
| **细线材** | 激光对射或小光斑 |
| **孔位检测** | 小光斑定位 |
| **边缘检测** | 激光对射/轮廓检测 |
| **透明件** | 需按型号和现场测试 |
| **高反光件** | 需控制角度和反光 |

## ③ 计算/选型示例

- 已知：  
  需要检测宽度 **1mm** 的小零件，传感器光斑直径为 **2mm**。
- 公式：  
  **光斑尺寸 ≤ 目标有效尺寸**
- 计算：  
  2mm ＞ 1mm，光斑大于目标尺寸。
- 结论：  
  检测可能不稳定，应选择更小光斑型号或改变检测方式。

> [!tip] 记忆口诀
> 小目标要小光斑，光斑太大容易漏判。

# 06 目标材质影响

> [!info] 核心结论
> 激光传感器容易受到目标颜色、反光率、表面粗糙度、透明度和倾斜角度影响。

## ① 分类/公式/规则

> [!note] 🟦 影响因素
> 黑色目标反射弱；  
> 镜面目标容易反光过强；  
> 透明目标可能透光或折射；  
> 粗糙表面可能散射；  
> 倾斜表面可能让反射光偏离接收器。

> [!warning] 🟧 稳定检测公式
> **稳定信号 = 反射强度合适 + 角度合理 + 表面状态稳定**

## ② 参考表

| 目标表面 | 可能问题 |
|---|---|
| **黑色表面** | 反射弱，信号小 |
| **白色表面** | 反射较强，通常较好 |
| **镜面金属** | 容易过曝或反射偏离 |
| **透明材料** | 可能透射或多重反射 |
| **粗糙表面** | 反射散射，信号波动 |
| **倾斜表面** | 回光角度不稳定 |

## ③ 计算/选型示例

- 已知：  
  同一传感器检测白色零件正常，检测黑色零件不稳定。
- 公式：  
  **接收光强 = 发射光强 × 目标反射率 × 安装角度影响**
- 计算：  
  黑色目标反射率低，接收光强可能低于稳定阈值。
- 结论：  
  应缩短距离、调整角度、提高增益，或选用适合低反射目标的型号。

> [!tip] 记忆口诀
> 黑色反光弱，亮面会乱反；角度一不对，信号就发散。

# 07 输出方式与PLC接线

> [!info] 核心结论
> 激光传感器接PLC时，要确认是开关量、模拟量还是通讯输出，并正确匹配PNP/NPN、4-20mA或0-10V。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 开关量用于到位、有无、超限判断；  
> 模拟量用于连续距离/位移；  
> 通讯输出用于读取测量值、状态和参数。

> [!warning] 🟧 接线匹配公式
> **PLC读取正常 = 输出类型匹配 + 电源正确 + 量程一致 + 参数设置正确**

## ② 参考表

| 输出类型 | 适合用途 |
|---|---|
| **PNP/NPN** | 到位、有无、上下限 |
| **0-10V** | 连续距离输出 |
| **4-20mA** | 工业现场抗干扰较好 |
| **RS485** | 数字读取距离/参数 |
| **IO-Link** | 参数化、诊断、测量值 |
| **Ethernet** | 上位机或复杂系统 |

## ③ 计算/选型示例

- 已知：  
  激光位移传感器量程为0～100mm，对应4-20mA输出，PLC读取电流为 **12mA**。
- 公式：  
  **当前距离 =（电流 - 4）÷ 16 × 满量程**
- 计算：  
  当前距离 =（12 - 4）÷ 16 × 100  
  = 8 ÷ 16 × 100  
  = **50mm**
- 结论：  
  PLC应显示约 **50mm**，前提是PLC量程与传感器量程设置一致。

> [!tip] 记忆口诀
> 开关量判到位，模拟量测距离；量程不一致，PLC数值必偏。

# 08 安装与调试

> [!info] 核心结论
> 激光传感器安装时，重点是检测距离、光斑位置、目标角度、固定强度和环境干扰。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 激光必须打在目标有效区域；  
> 传感器应安装在推荐距离附近；  
> 目标表面应尽量稳定，避免强反光、强振动和遮挡。

> [!warning] 🟧 安装可靠公式
> **可靠检测 = 距离合适 + 光斑落点正确 + 角度稳定 + 固定牢靠 + 环境干扰少**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **检测距离** | 在额定范围内 |
| **光斑位置** | 落在目标有效区域 |
| **安装角度** | 避免镜面直反 |
| **固定方式** | 防止振动偏移 |
| **线缆走向** | 远离动力线和变频器 |
| **防护措施** | 防尘、防油、防水雾 |

## ③ 计算/选型示例

- 已知：  
  激光检测点在设备运行时偶尔偏离目标边缘。
- 公式：  
  **信号异常 = 目标抖动 + 光斑偏移 + 安装不牢**
- 计算：  
  光斑落在边缘附近时，轻微振动就可能导致信号变化。
- 结论：  
  应调整光斑到目标中心区域，并加固传感器支架。

> [!tip] 记忆口诀
> 激光打得准，检测才稳定；光斑打边缘，信号易漂移。

# 09 安全注意事项

> [!info] 核心结论
> 激光传感器调试时要注意激光安全，避免直视激光束或反射光。

## ① 分类/公式/规则

> [!note] 🟦 安全提醒
> 激光传感器通常有激光等级标识，不同等级对眼睛和皮肤的风险不同。  
> 调试时不要用眼睛直视发光口，也不要让镜面反射光进入眼睛。

> [!warning] 🟧 安全使用公式
> **安全调试 = 不直视激光 + 避免镜面反射 + 按等级防护 + 断电安装**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **直视激光** | 避免 |
| **镜面反射** | 高风险 |
| **调试距离** | 不要贴近发射端观察 |
| **防护眼镜** | 按激光等级和波长选择 |
| **警示标识** | 保持清晰可见 |
| **安装维护** | 按厂家安全说明执行 |

## ③ 计算/选型示例

- 已知：  
  传感器检测镜面金属件，现场反射光较强。
- 公式：  
  **风险 = 激光直射 + 镜面反射**
- 计算：  
  镜面目标可能把激光反射到人员视线方向。
- 结论：  
  应调整角度，避免反射光朝向人员，并按激光等级采取防护措施。

> [!tip] 记忆口诀
> 激光能检测，也要防直视；亮面会反光，调试要避开。

# 10 常见故障排查

> [!info] 核心结论
> 激光传感器异常时，优先检查光斑位置、目标表面、检测距离、输出量程、环境光和接线。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看光斑有没有打到目标；  
> 再看距离是否在量程内；  
> 然后检查目标材质、输出方式和PLC换算。

> [!warning] 🟧 排错公式
> **检测异常 = 光斑偏移 + 目标反光异常 + 距离超量程 + 参数错误 + 接线错误 + 环境干扰**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无信号** | 未对准、超量程、接线错误 |
| **数值跳动** | 振动、表面反光、距离边界 |
| **黑色件不稳定** | 反射弱 |
| **亮面件异常** | 镜面反射或过曝 |
| **PLC数值不准** | 模拟量量程设置错误 |
| **高速漏检** | 响应时间或采样周期不足 |

## ③ 计算/选型示例

- 已知：  
  传感器本体显示距离为30mm，但PLC显示为60mm。
- 公式：  
  **显示不一致 = 传感器量程 ≠ PLC缩放量程**
- 计算：  
  如果传感器4-20mA对应0～50mm，而PLC按0～100mm换算，PLC会显示偏大。
- 结论：  
  应检查PLC模拟量缩放参数和传感器输出量程设置。

> [!tip] 记忆口诀
> 本体准、PLC错，先查量程换算。

# 11 激光传感器快速选型表

> [!info] 核心结论
> 激光传感器选型先看检测目的，再看量程、精度、目标表面、输出方式和安装环境。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 看检测目的：有无、测距、位移、轮廓  
> ② 看测量范围  
> ③ 看精度和分辨率  
> ④ 看目标颜色和反光率  
> ⑤ 看运动速度  
> ⑥ 看输出方式  
> ⑦ 看安装空间  
> ⑧ 看环境防护和激光安全

> [!warning] 🟧 最终选型公式
> **推荐型号 = 检测类型 + 量程 + 精度 + 光斑尺寸 + 输出方式 + 响应时间 + 防护等级**

## ② 参考表

| 需求 | 推荐方向 |
|---|---|
| 小目标有无检测 | 小光斑激光光电 |
| 高度/位移检测 | 激光位移传感器 |
| 远距离测距 | 激光测距传感器 |
| 边缘/宽度检测 | 激光对射或轮廓传感器 |
| 高速检测 | 校核响应时间 |
| 黑色/亮面目标 | 现场测试反射稳定性 |
| 接PLC模拟量 | 4-20mA或0-10V匹配 |

## ③ 计算/选型示例

- 已知：  
  需要检测零件高度，变化范围为 **45～55mm**，要求分辨率 **0.01mm**，PLC支持4-20mA。
- 公式：  
  **型号需求 = 量程覆盖45～55mm + 分辨率≤0.01mm + 4-20mA输出**
- 计算：  
  测量范围至少覆盖10mm变化区间，并留安装余量。
- 结论：  
  可选择 **激光位移传感器，量程覆盖45～55mm，分辨率≤0.01mm，4-20mA输出**，具体精度和响应时间需按厂家手册校核。

> [!tip] 选型口诀
> 先定测什么，再定量程；精度看分辨率，稳定看表面；输出配PLC，安装看光斑。
