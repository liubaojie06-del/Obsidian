# 01 光栅尺总览

> [!info] 核心结论
> 光栅尺用于检测 **直线位移、位置反馈、长度测量和闭环控制**，常见于数控机床、直线模组、测量平台和精密设备。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 光栅尺可以理解为“直线型编码器”。  
> 编码器测旋转角度，光栅尺测直线位置。

> [!warning] 🟧 通用选型公式
> **光栅尺选型 = 测量行程 + 分辨率 + 精度等级 + 输出信号 + 安装方式 + 防护环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **类型** | 直线位移测量传感器 |
| **检测对象** | 工作台、滑台、刀架、模组 |
| **常见输出** | TTL、RS422、1Vpp、SSI、BiSS等需校核 |
| **常见用途** | 位置反馈、长度测量、闭环控制 |
| **重点参数** | 量程、分辨率、精度、重复精度 |
| **安装重点** | 平行度、间隙、防尘、防油 |

## ③ 计算/选型示例

- 已知：  
  直线滑台行程为 **500mm**，需要实时反馈位置给控制器。
- 公式：  
  **光栅尺量程 ≥ 实际行程 + 安装余量**
- 计算：  
  实际行程500mm，建议选择量程略大于500mm的光栅尺。
- 结论：  
  可选 **500mm以上量程光栅尺**，分辨率、输出信号和安装尺寸需按厂家手册校核。

> [!tip] 记忆口诀
> 编码器测旋转，光栅尺测直线。

# 02 底层原理

> [!info] 核心结论
> 光栅尺的本质，是通过光线经过光栅条纹产生的明暗变化，转换成位置脉冲或绝对位置数据。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 光栅尺通常由 **标尺光栅、读数头、光源、接收器和信号处理电路** 组成。  
> 当读数头沿标尺移动时，光栅条纹产生周期性光强变化，电路把这种变化转换成位移信号。

> [!warning] 🟧 光栅检测逻辑
> **直线移动 → 光栅条纹变化 → 光强变化 → 电路整形/插补 → 输出位置信号**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **标尺光栅** | 提供精密刻线基准 |
| **读数头** | 读取光栅变化 |
| **光源** | 提供稳定光信号 |
| **接收器** | 检测明暗变化 |
| **信号电路** | 输出脉冲或数字位置 |
| **控制器** | 计算位置和速度 |

## ③ 计算/选型示例

- 已知：  
  光栅尺分辨率为 **1μm**。
- 公式：  
  **最小位移单位 = 分辨率**
- 计算：  
  1μm = **0.001mm**
- 结论：  
  控制器每变化1个计数，对应位移约 **0.001mm**。

> [!tip] 记忆口诀
> 光栅看条纹，条纹变位置；分辨率越小，测得越细。

# 03 增量式与绝对式

> [!info] 核心结论
> 增量式光栅尺输出脉冲变化量，绝对式光栅尺直接输出当前位置。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> **增量式**：需要回零，靠脉冲计数得到位置。  
> **绝对式**：上电即可读取当前位置，不依赖回零。

> [!warning] 🟧 类型选择公式
> **是否需要回零 = 增量式 / 绝对式的关键区别**

## ② 参考表

| 对比项 | 增量式光栅尺 | 绝对式光栅尺 |
|---|---|---|
| 输出 | A/B/Z脉冲 | 数字位置值 |
| 上电位置 | 通常需回零 | **可直接读取** |
| 成本 | 相对较低 | 相对较高 |
| 接口 | TTL、RS422常见 | SSI、BiSS等常见 |
| 适合场景 | 普通测长、相对定位 | 高端闭环、断电保位置 |

## ③ 计算/选型示例

- 已知：  
  设备断电后重新上电，不允许重新回零。
- 公式：  
  **上电可用 = 绝对位置保持**
- 计算：  
  增量式断电后通常丢失当前位置，绝对式可直接读取位置。
- 结论：  
  该场景优先选择 **绝对式光栅尺**。

> [!tip] 记忆口诀
> 增量要回零，绝对上电知位置。

# 04 分辨率与精度

> [!info] 核心结论
> 分辨率表示能分多细，精度表示测得准不准，二者不能混为一谈。

## ① 分类/公式/规则

> [!note] 🟦 关键区别
> 分辨率越小，位置显示越细；  
> 精度越高，实际误差越小。  
> 例如分辨率1μm，不代表全行程误差一定只有1μm。

> [!warning] 🟧 位置换算公式
> **位移 = 计数值 × 分辨率**

## ② 参考表

| 参数 | 含义 |
|---|---|
| **分辨率** | 最小可显示位移 |
| **精度** | 实际测量误差范围 |
| **重复精度** | 多次回到同一点的一致性 |
| **线性误差** | 全行程位置偏差 |
| **插补倍数** | 将原始信号细分 |
| **温度影响** | 热胀冷缩会影响测量 |

## ③ 计算/选型示例

- 已知：  
  光栅尺分辨率为 **5μm**，控制器计数变化为 **2000**。
- 公式：  
  **位移 = 计数值 × 分辨率**
- 计算：  
  位移 = 2000 × 5μm = 10000μm = **10mm**
- 结论：  
  当前位移变化为 **10mm**。

> [!tip] 记忆口诀
> 分辨率看细不细，精度看准不准。

# 05 输出信号类型

> [!info] 核心结论
> 光栅尺接控制器时，必须确认输出信号类型是否匹配，否则可能无法计数或位置异常。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 增量式常见 **TTL、RS422差分、1Vpp正弦信号**；  
> 绝对式常见 **SSI、BiSS、EnDat等数字接口**。

> [!warning] 🟧 接口匹配公式
> **系统可用 = 光栅尺输出类型 + 控制器输入接口 + 电源电压 + 信号频率匹配**

## ② 参考表

| 输出类型 | 特点 |
|---|---|
| **TTL** | 脉冲信号，距离较短 |
| **RS422差分** | 抗干扰较好，适合高速长线 |
| **1Vpp** | 正弦模拟信号，可高倍插补 |
| **SSI** | 绝对值数字位置传输 |
| **BiSS** | 高速绝对值接口 |
| **A/B/Z** | 增量式常见信号 |

## ③ 计算/选型示例

- 已知：  
  控制器支持RS422差分输入，现场光栅尺为TTL单端输出。
- 公式：  
  **输入匹配 = 信号类型一致**
- 计算：  
  TTL单端不能直接等同RS422差分，接错可能无法识别或抗干扰差。
- 结论：  
  应选择 **RS422差分输出光栅尺**，或增加信号转换模块。

> [!tip] 记忆口诀
> 信号类型先对上，控制器才数得准。

# 06 量程与安装行程

> [!info] 核心结论
> 光栅尺量程必须覆盖机械实际运动范围，并预留安全余量，避免撞尺或超行程。

## ① 分类/公式/规则

> [!note] 🟦 量程选择
> 光栅尺有效量程要大于机械实际行程。  
> 同时还要考虑安装位置、限位余量、读数头长度和机械误差。

> [!warning] 🟧 量程公式
> **推荐量程 ≥ 实际行程 + 两端安全余量**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **实际行程** | 设备运动范围 |
| **有效量程** | 光栅尺可测长度 |
| **安装余量** | 两端预留空间 |
| **机械限位** | 防止超行程撞坏读数头 |
| **读数头位置** | 需保证全程不干涉 |
| **电缆出线** | 避免运动拉扯 |

## ③ 计算/选型示例

- 已知：  
  设备直线行程为 **800mm**，两端各预留 **20mm** 余量。
- 公式：  
  **推荐量程 ≥ 800 + 20 × 2**
- 计算：  
  推荐量程 ≥ **840mm**
- 结论：  
  应选择有效量程不小于 **840mm** 的光栅尺，实际规格需按厂家系列选择。

> [!tip] 记忆口诀
> 行程别卡满，两端留余量。

# 07 安装要求

> [!info] 核心结论
> 光栅尺安装精度直接影响测量稳定性，重点是平行度、间隙、刚性和防污染。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 标尺要与运动方向平行，读数头和标尺间隙要符合要求。  
> 安装面要平整，支架要稳定，避免振动、油污、铁屑进入。

> [!warning] 🟧 安装可靠公式
> **可靠测量 = 标尺平行 + 间隙正确 + 固定牢靠 + 防尘防油 + 电缆不受力**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **平行度** | 与运动方向一致 |
| **读数头间隙** | 按厂家手册校核 |
| **安装面** | 平整、刚性足够 |
| **防护罩** | 防油、防尘、防铁屑 |
| **电缆固定** | 避免拖拽和折断 |
| **机械限位** | 防止读数头撞端部 |

## ③ 计算/选型示例

- 已知：  
  光栅尺安装后，设备运行到某一位置时读数跳变。
- 公式：  
  **读数异常 = 间隙变化 + 平行度差 + 振动 + 污染**
- 计算：  
  某段位置可能因安装面不平或读数头间隙变化导致信号不稳定。
- 结论：  
  应检查该段安装平行度、间隙和标尺表面污染情况。

> [!tip] 记忆口诀
> 光栅尺怕歪、怕脏、怕撞，安装比选型还关键。

# 08 回零与参考点

> [!info] 核心结论
> 增量式光栅尺通常需要回零或寻找参考点，才能建立准确的机械坐标。

## ① 分类/公式/规则

> [!note] 🟦 回零逻辑
> 增量式光栅尺只记录位移变化。  
> 设备上电后，需要通过原点开关、Z相信号或参考标记建立坐标零点。

> [!warning] 🟧 位置公式
> **实际位置 = 参考点位置 + 增量位移**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **原点开关** | 粗定位参考 |
| **Z相信号** | 精确参考点 |
| **参考标记** | 建立绝对基准 |
| **回零方向** | 决定寻找原点过程 |
| **回零速度** | 影响精度和安全 |
| **软限位** | 防止超行程 |

## ③ 计算/选型示例

- 已知：  
  设备断电后重新上电，当前位置未知。
- 公式：  
  **增量式上电 = 先回零，再定位**
- 计算：  
  PLC计数清零不代表机械在零点，必须执行回零动作。
- 结论：  
  增量式光栅尺系统应设计可靠回零流程。

> [!tip] 记忆口诀
> 增量只记走了多少，不记现在在哪。

# 09 常见故障排查

> [!info] 核心结论
> 光栅尺异常时，优先检查供电、信号线、读数头间隙、污染、接地屏蔽和控制器参数。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查供电和接口，再查信号和参数；  
> 先查安装间隙，再查标尺污染；  
> 先查屏蔽接地，再怀疑硬件损坏。

> [!warning] 🟧 排错公式
> **光栅尺异常 = 供电错误 + 信号不匹配 + 间隙异常 + 污染遮挡 + 接地不良 + 参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无位置变化** | 无供电、信号线断、接口不匹配 |
| **位置跳变** | 干扰、接地差、读数头间隙不稳 |
| **局部异常** | 标尺污染、划伤、安装变形 |
| **方向反了** | A/B相接反或方向参数反 |
| **回零失败** | 原点信号或Z相信号异常 |
| **高速丢数** | 输入频率超限或线缆干扰 |

## ③ 计算/选型示例

- 已知：  
  光栅尺移动速度为1000mm/s，分辨率为1μm。
- 公式：  
  **脉冲频率 = 速度 ÷ 分辨率**
- 计算：  
  1000mm/s ÷ 0.001mm = **1,000,000Hz = 1MHz**
- 结论：  
  控制器输入频率必须支持 **1MHz以上**，否则可能高速丢数。

> [!tip] 记忆口诀
> 高速丢数先算频率，局部跳数先查尺面。

# 10 光栅尺快速选型表

> [!info] 核心结论
> 光栅尺选型先看行程和精度，再看输出接口、安装环境和控制器匹配。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 看测量行程  
> ② 看分辨率要求  
> ③ 看精度等级  
> ④ 看增量式还是绝对式  
> ⑤ 看输出信号  
> ⑥ 看安装空间  
> ⑦ 看防护环境  
> ⑧ 看控制器接口和频率能力

> [!warning] 🟧 最终选型公式
> **推荐型号 = 量程 + 分辨率 + 精度 + 输出信号 + 安装方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 普通位移测量 | 增量式光栅尺 |
| 上电直接读位置 | 绝对式光栅尺 |
| 高精度闭环 | 高精度等级和稳定安装 |
| 长距离接线 | RS422差分优先 |
| 高速运动 | 输入频率必须校核 |
| 油污粉尘环境 | 防护罩、防护等级 |
| 机床应用 | 精度、热稳定、防污染重点校核 |

## ③ 计算/选型示例

- 已知：  
  直线平台行程600mm，要求显示分辨率0.005mm，控制器支持RS422输入。
- 公式：  
  **型号需求 = 量程≥600mm + 分辨率≤0.005mm + RS422输出**
- 计算：  
  0.005mm = **5μm**，光栅尺分辨率应不大于5μm。
- 结论：  
  可选择 **量程≥600mm、5μm或更高分辨率、RS422输出光栅尺**，安装尺寸和精度等级需按厂家手册校核。

> [!tip] 选型口诀
> 行程定量程，精度定分辨率；接口配控制器，安装保稳定。
