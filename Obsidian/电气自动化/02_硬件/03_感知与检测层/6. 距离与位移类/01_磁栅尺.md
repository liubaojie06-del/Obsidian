# 01 磁栅尺总览

> [!info] 核心结论
> 磁栅尺用于检测 **直线位移、位置反馈、长度测量和运动控制**，相比光栅尺更耐油污、粉尘和振动。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 磁栅尺可以理解为“用磁信号测直线位置”的线性编码器。  
> 它通常由 **磁尺/磁带 + 读数头 + 信号处理电路 + 控制器** 组成。

> [!warning] 🟧 通用选型公式
> **磁栅尺选型 = 测量行程 + 分辨率 + 精度 + 输出信号 + 安装间隙 + 环境条件**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **类型** | 直线位移测量传感器 |
| **核心部件** | 磁尺、读数头 |
| **检测对象** | 滑台、模组、输送机构、测长机构 |
| **常见输出** | A/B/Z、TTL、RS422、SSI等需校核 |
| **优势** | 抗油污、抗粉尘、安装相对方便 |
| **重点参数** | 量程、分辨率、精度、安装间隙 |

## ③ 计算/选型示例

- 已知：  
  直线模组行程为 **1000mm**，需要位置反馈。
- 公式：  
  **磁栅尺量程 ≥ 实际行程 + 安装余量**
- 计算：  
  实际行程1000mm，建议选择量程略大于1000mm的磁栅尺。
- 结论：  
  可选 **1000mm以上量程磁栅尺**，输出信号和安装间隙需按厂家手册校核。

> [!tip] 记忆口诀
> 光栅看光，磁栅看磁；直线位置，都能检测。

# 02 底层原理

> [!info] 核心结论
> 磁栅尺的本质，是读数头检测磁尺上的周期性磁极变化，并转换成位置脉冲或数字位置值。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 磁尺表面按固定节距排列N极、S极。  
> 读数头移动时，会感应到磁场周期变化，内部电路将磁信号转换成A/B相脉冲或绝对位置数据。

> [!warning] 🟧 磁栅检测逻辑
> **直线移动 → 磁极变化 → 读数头感应 → 电路处理 → 输出位置信号**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **磁尺/磁带** | 提供磁极位置基准 |
| **读数头** | 感应磁场变化 |
| **磁极节距** | 决定原始信号周期 |
| **信号电路** | 插补、整形、输出信号 |
| **控制器** | 计数并换算位置 |

## ③ 计算/选型示例

- 已知：  
  磁栅尺分辨率为 **10μm**。
- 公式：  
  **最小位移单位 = 分辨率**
- 计算：  
  10μm = **0.01mm**
- 结论：  
  控制器每变化1个计数，对应位移约 **0.01mm**。

> [!tip] 记忆口诀
> 磁极一格格，读头一路数；数得越多，走得越远。

# 03 磁栅尺 vs 光栅尺

> [!info] 核心结论
> 光栅尺精度通常更高，磁栅尺环境适应性通常更强。

## ① 分类/公式/规则

> [!note] 🟦 快速区别
> 光栅尺靠光学条纹读取位置，怕油污和粉尘；  
> 磁栅尺靠磁极变化读取位置，抗污染能力更好。

> [!warning] 🟧 选型判断公式
> **高精度洁净环境 → 光栅尺**  
> **油污粉尘振动环境 → 磁栅尺**

## ② 参考表

| 对比项 | 光栅尺 | 磁栅尺 |
|---|---|---|
| 原理 | 光学条纹 | **磁极变化** |
| 精度 | 通常更高 | 中高精度 |
| 抗油污 | 较弱 | **较强** |
| 抗粉尘 | 较弱 | **较强** |
| 安装要求 | 较高 | 相对宽松 |
| 应用 | 精密机床、测量平台 | 模组、机床、工业现场 |

## ③ 计算/选型示例

- 已知：  
  设备现场有油雾、铁屑和振动，精度要求不是超高精密。
- 公式：  
  **环境适应性 = 抗油污 + 抗粉尘 + 抗振动**
- 计算：  
  光栅尺可能因污染导致读数异常，磁栅尺更适合恶劣环境。
- 结论：  
  可优先选择 **磁栅尺**，具体精度和防护等级需按厂家手册校核。

> [!tip] 记忆口诀
> 要极高精度看光栅，要抗脏抗振看磁栅。

# 04 增量式与绝对式

> [!info] 核心结论
> 增量式磁栅尺靠脉冲计数得到位置，绝对式磁栅尺可直接输出当前位置。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> **增量式**：断电后通常需要回零。  
> **绝对式**：上电可读取当前位置，不必每次回零。

> [!warning] 🟧 类型选择公式
> **允许回零 → 增量式**  
> **不允许回零 → 绝对式**

## ② 参考表

| 对比项 | 增量式磁栅尺 | 绝对式磁栅尺 |
|---|---|---|
| 输出 | A/B/Z脉冲 | 数字位置值 |
| 上电位置 | 通常需回零 | **可直接读取** |
| 成本 | 相对较低 | 相对较高 |
| 接口 | TTL、RS422常见 | SSI、BiSS等需校核 |
| 适合场景 | 普通定位、测长 | 断电保位置、闭环控制 |

## ③ 计算/选型示例

- 已知：  
  设备断电后重新上电，不能重新回零。
- 公式：  
  **上电可用 = 绝对位置保持**
- 计算：  
  增量式断电后位置基准可能丢失，绝对式可直接读当前位置。
- 结论：  
  应优先选择 **绝对式磁栅尺**。

> [!tip] 记忆口诀
> 增量要找零，绝对直接读。

# 05 分辨率与精度

> [!info] 核心结论
> 分辨率决定显示细不细，精度决定测量准不准，磁栅尺选型不能只看分辨率。

## ① 分类/公式/规则

> [!note] 🟦 参数区别
> 分辨率越小，位置显示越细；  
> 精度越高，实际位置误差越小。  
> 例如分辨率1μm，不代表全行程误差一定只有1μm。

> [!warning] 🟧 位移换算公式
> **位移 = 计数值 × 分辨率**

## ② 参考表

| 参数 | 含义 |
|---|---|
| **分辨率** | 最小显示单位 |
| **精度** | 实际测量误差 |
| **重复精度** | 多次定位一致性 |
| **磁极节距** | 磁尺原始周期 |
| **插补倍数** | 把原始信号细分 |
| **温度影响** | 温度变化会影响尺寸稳定性 |

## ③ 计算/选型示例

- 已知：  
  磁栅尺分辨率为 **0.01mm**，控制器计数变化为 **3500**。
- 公式：  
  **位移 = 计数值 × 分辨率**
- 计算：  
  位移 = 3500 × 0.01mm = **35mm**
- 结论：  
  当前位移变化为 **35mm**。

> [!tip] 记忆口诀
> 分辨率看小数点，精度看真实误差。

# 06 输出信号类型

> [!info] 核心结论
> 磁栅尺接PLC、数显表或运动控制器时，必须确认输出信号和输入接口匹配。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 增量式磁栅尺常见 **A/B相、Z相、TTL、RS422差分**；  
> 绝对式磁栅尺可能使用 **SSI、BiSS、总线接口** 等。

> [!warning] 🟧 接口匹配公式
> **系统可用 = 磁栅尺输出类型 + 控制器输入类型 + 电源电压 + 最高频率匹配**

## ② 参考表

| 输出类型 | 特点 |
|---|---|
| **A/B相** | 增量计数和方向判断 |
| **Z相** | 参考点或零位信号 |
| **TTL** | 单端脉冲信号 |
| **RS422差分** | 抗干扰强，适合长线 |
| **SSI** | 绝对位置数据 |
| **BiSS** | 高速绝对值接口 |

## ③ 计算/选型示例

- 已知：  
  控制器支持RS422差分输入，现场磁栅尺为TTL输出。
- 公式：  
  **输入匹配 = 信号类型一致**
- 计算：  
  TTL单端信号不能直接等同于RS422差分信号。
- 结论：  
  应选择 **RS422输出磁栅尺**，或增加信号转换模块。

> [!tip] 记忆口诀
> 尺子信号要对口，控制器才能数得准。

# 07 安装间隙与方向

> [!info] 核心结论
> 磁栅尺安装最关键的是读数头与磁尺之间的间隙、平行度和方向。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 读数头离磁尺太远，信号变弱；  
> 离得太近，可能刮碰；  
> 方向装反或偏斜，也会导致读数异常。

> [!warning] 🟧 安装可靠公式
> **可靠读取 = 间隙合适 + 平行度良好 + 方向正确 + 固定牢靠**

## ② 参考表

| 安装项 | 重点 |
|---|---|
| **读头间隙** | 按厂家手册校核 |
| **平行度** | 与运动方向一致 |
| **安装方向** | 读头方向和磁尺方向匹配 |
| **固定强度** | 防止振动松动 |
| **机械限位** | 防止读头撞尺 |
| **电缆固定** | 避免拖拽和折断 |

## ③ 计算/选型示例

- 已知：  
  设备运行到某一段位置时，磁栅尺读数跳变。
- 公式：  
  **读数跳变 = 间隙变化 + 平行度差 + 磁尺污染/损伤 + 干扰**
- 计算：  
  某段位置可能因安装面不平导致读头间隙变大，信号变弱。
- 结论：  
  应检查该段读数头间隙、磁尺表面和安装平行度。

> [!tip] 记忆口诀
> 磁尺不怕脏一点，最怕间隙乱一点。

# 08 磁尺与磁带安装

> [!info] 核心结论
> 磁尺或磁带必须固定平整、方向正确，并避免强磁场、高温和机械刮擦。

## ① 分类/公式/规则

> [!note] 🟦 安装要求
> 磁带通常贴在直线导轨或基准面上，外部可能加保护带。  
> 安装时要防止气泡、翘边、扭曲、刮伤和磁极方向错误。

> [!warning] 🟧 磁带安装公式
> **可靠磁尺 = 基准面平整 + 粘贴牢固 + 保护带完整 + 避免强磁退磁**

## ② 参考表

| 项目 | 注意事项 |
|---|---|
| **基准面** | 清洁、平整、无油污 |
| **粘贴方向** | 按磁尺标记方向安装 |
| **保护带** | 防止磨损和污染 |
| **端部固定** | 防止翘边 |
| **强磁环境** | 避免退磁或干扰 |
| **温度环境** | 需按厂家手册校核 |

## ③ 计算/选型示例

- 已知：  
  磁带贴在油污未清理干净的安装面上。
- 公式：  
  **粘贴风险 = 油污 + 粘接力下降 + 磁带翘起**
- 计算：  
  磁带翘起会导致读数头间隙变化，影响信号稳定。
- 结论：  
  安装前必须清洁基准面，并按厂家工艺粘贴和固定。

> [!tip] 记忆口诀
> 磁带贴得平，读数才安宁。

# 09 回零与参考点

> [!info] 核心结论
> 增量式磁栅尺通常需要回零，才能建立设备坐标基准。

## ① 分类/公式/规则

> [!note] 🟦 回零逻辑
> 增量式磁栅尺只记录移动了多少。  
> 上电后若当前位置未知，需要通过原点开关、Z相信号或参考磁标建立零点。

> [!warning] 🟧 位置公式
> **实际位置 = 参考点位置 + 增量位移**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **原点开关** | 建立粗略回零位置 |
| **Z相信号** | 精确参考点 |
| **参考磁标** | 建立固定位置基准 |
| **回零方向** | 决定寻找原点路径 |
| **回零速度** | 影响回零稳定性 |
| **软限位** | 防止超行程 |

## ③ 计算/选型示例

- 已知：  
  增量式磁栅尺断电后重新上电，PLC计数为0。
- 公式：  
  **PLC计数为0 ≠ 机械位置为0**
- 计算：  
  如果机械停在中间位置，上电清零会导致实际位置和显示位置不一致。
- 结论：  
  增量式磁栅尺系统应设计 **回零流程** 或掉电位置保存方案。

> [!tip] 记忆口诀
> 增量尺不记现在在哪，上电回零才可靠。

# 10 常见故障排查

> [!info] 核心结论
> 磁栅尺异常时，优先检查供电、信号类型、读头间隙、磁带损伤、屏蔽接地和控制器参数。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先查供电和接口，再查读头和磁带；  
> 先查安装间隙，再查接地屏蔽；  
> 先查参数，再判断硬件损坏。

> [!warning] 🟧 排错公式
> **磁栅尺异常 = 供电错误 + 信号不匹配 + 间隙不良 + 磁带损伤 + 干扰 + 参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无位置变化** | 无供电、信号线断、接口不匹配 |
| **位置跳变** | 间隙不稳、屏蔽不良、磁带损伤 |
| **方向反了** | A/B相接反或方向参数反 |
| **局部异常** | 磁带划伤、退磁、翘起 |
| **高速丢数** | 控制器输入频率不够 |
| **回零失败** | Z相或参考点信号异常 |

## ③ 计算/选型示例

- 已知：  
  磁栅尺移动速度为 **500mm/s**，分辨率为 **1μm**。
- 公式：  
  **脉冲频率 = 速度 ÷ 分辨率**
- 计算：  
  500mm/s ÷ 0.001mm = **500,000Hz = 500kHz**
- 结论：  
  控制器输入频率必须支持 **500kHz以上**，否则可能高速丢数。

> [!tip] 记忆口诀
> 跳数查间隙，丢数算频率，局部异常看磁带。

# 11 磁栅尺快速选型表

> [!info] 核心结论
> 磁栅尺选型先看行程和环境，再看分辨率、精度、输出接口和安装间隙。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 看测量行程  
> ② 看分辨率要求  
> ③ 看精度要求  
> ④ 看增量式还是绝对式  
> ⑤ 看输出信号  
> ⑥ 看安装空间和间隙  
> ⑦ 看油污、粉尘、振动环境  
> ⑧ 看控制器接口和最高频率

> [!warning] 🟧 最终选型公式
> **推荐型号 = 量程 + 分辨率 + 精度 + 输出信号 + 安装间隙 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 油污粉尘现场 | 磁栅尺优先评估 |
| 高精密测量 | 精度等级重点校核 |
| 普通位移反馈 | 增量式磁栅尺 |
| 上电直接读位置 | 绝对式磁栅尺 |
| 长距离接线 | RS422差分优先 |
| 高速运动 | 输入频率必须校核 |
| 机械振动大 | 读头固定和间隙稳定 |

## ③ 计算/选型示例

- 已知：  
  直线平台行程1200mm，要求显示分辨率0.01mm，控制器支持RS422输入。
- 公式：  
  **型号需求 = 量程≥1200mm + 分辨率≤0.01mm + RS422输出**
- 计算：  
  0.01mm = **10μm**，磁栅尺分辨率应不大于10μm。
- 结论：  
  可选择 **量程≥1200mm、10μm或更高分辨率、RS422输出磁栅尺**，安装尺寸和精度需按厂家手册校核。

> [!tip] 选型口诀
> 行程定量程，精度定分辨率；环境脏乱选磁栅，接口一定配控制器。
