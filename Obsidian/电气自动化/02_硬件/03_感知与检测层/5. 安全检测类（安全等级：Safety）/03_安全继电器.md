# 01 安全继电器总览

> [!info] 核心结论
> 安全继电器用于监控急停、安全门、安全光幕等安全信号，并在危险条件出现时可靠切断设备危险动作。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 安全继电器不是普通继电器。  
> 普通继电器主要做通断控制；安全继电器用于 **安全信号监控、故障检测、双通道判断、安全输出切断**。

> [!warning] 🟧 通用选型公式
> **安全继电器选型 = 安全输入类型 + 安全等级要求 + 输出触点数量 + 复位方式 + EDM反馈 + 电源电压**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **安全继电器** | 处理安全输入并输出安全控制信号 |
| **输入设备** | 急停、安全门锁、安全光幕、双手按钮 |
| **输出对象** | 接触器、变频器STO、安全PLC输入 |
| **核心功能** | 双通道监控、短路检测、复位控制 |
| **常见电源** | DC24V / AC电源需按型号校核 |
| **重点参数** | 安全等级、输入类型、输出点数、响应时间 |

## ③ 计算/选型示例

- 已知：  
  一台设备有急停按钮和安全门，需要在触发时切断电机接触器。
- 公式：  
  **安全停机 = 安全输入 + 安全继电器判断 + 安全输出切断**
- 计算：  
  急停或安全门动作后，安全继电器断开输出，接触器失电，电机停止。
- 结论：  
  应使用 **安全继电器** 构建安全回路，具体安全等级和接线方式需按实际规范/厂家手册校核。

> [!tip] 记忆口诀
> 普通继电器管动作，安全继电器管保命。

# 02 底层原理

> [!info] 核心结论
> 安全继电器的本质，是用冗余通道和自检逻辑判断安全条件是否成立，再控制安全输出。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 安全设备输出双通道信号，安全继电器同时监控两个通道。  
> 当两个通道状态正确、复位条件满足、反馈回路正常时，安全输出闭合；  
> 当任一通道异常或安全条件不满足时，安全输出断开。

> [!warning] 🟧 安全判断逻辑
> **安全输入正常 + 双通道一致 + 复位满足 + 反馈正常 = 安全输出允许**

## ② 参考表

| 状态 | 安全继电器动作 |
|---|---|
| **安全输入正常** | 允许输出 |
| **急停按下** | 输出断开 |
| **安全门打开** | 输出断开 |
| **光幕遮挡** | 输出断开 |
| **通道短路** | 报警或无法复位 |
| **接触器粘连** | EDM反馈异常，禁止启动 |

## ③ 计算/选型示例

- 已知：  
  急停按钮释放后，设备仍不能启动。
- 公式：  
  **启动允许 = 急停释放 + 复位完成 + 输出反馈正常**
- 计算：  
  急停释放只是输入恢复，安全继电器还需要复位和反馈确认。
- 结论：  
  应检查复位按钮、EDM反馈回路和安全继电器状态指示灯。

> [!tip] 记忆口诀
> 安全输入恢复，不等于设备立刻能启动。

# 03 双通道输入

> [!info] 核心结论
> 安全继电器常用双通道输入，用于发现断线、短路、触点粘连和通道不一致等故障。

## ① 分类/公式/规则

> [!note] 🟦 双通道意义
> 双通道不是简单把两根线并联，而是用两个独立通道分别监控同一个安全动作，提高故障发现能力。

> [!warning] 🟧 双通道公式
> **安全输入可靠 = 通道1正常 + 通道2正常 + 动作同步 + 无短路粘连**

## ② 参考表

| 输入方式 | 说明 |
|---|---|
| **单通道** | 结构简单，诊断能力较弱 |
| **双通道** | 常用于安全回路 |
| **常闭触点NC** | 急停、安全门常见 |
| **OSSD输入** | 安全光幕、安全传感器常见 |
| **交叉短路检测** | 检测两个通道是否异常短接 |
| **同步时间检测** | 判断两个通道动作是否合理 |

## ③ 计算/选型示例

- 已知：  
  急停按钮有两组常闭触点。
- 公式：  
  **急停双通道 = NC1 + NC2 + 安全继电器双输入**
- 计算：  
  急停按下时，两组常闭触点同时断开，安全继电器切断输出。
- 结论：  
  急停按钮建议使用双通道接入安全继电器，具体接线需按产品手册校核。

> [!tip] 记忆口诀
> 两路一起看，故障更容易发现。

# 04 安全输出

> [!info] 核心结论
> 安全继电器输出通常用于切断接触器、驱动器STO或其他危险能量控制回路。

## ① 分类/公式/规则

> [!note] 🟦 输出理解
> 安全继电器的安全输出一般是强制导向触点或安全半导体输出，用于控制后级执行元件。

> [!warning] 🟧 输出控制公式
> **危险动作停止 = 安全输出断开 + 执行元件断电 + 能量释放/停止**

## ② 参考表

| 输出类型 | 常见用途 |
|---|---|
| **安全常开触点** | 控制接触器线圈 |
| **辅助触点** | 状态反馈给PLC |
| **半导体安全输出** | 控制安全输入或驱动STO |
| **多组输出** | 同时切断多个危险源 |
| **延时输出** | 延时停机或顺序停机 |
| **反馈回路** | 监控接触器是否释放 |

## ③ 计算/选型示例

- 已知：  
  设备有两个电机接触器KM1、KM2，需要同时切断。
- 公式：  
  **安全输出需求 = 接触器数量 + 安全触点数量 + 反馈监控**
- 计算：  
  至少需要满足两路接触器控制，并把接触器辅助触点接入反馈回路。
- 结论：  
  选择安全继电器时要确认安全输出点数足够，并设计EDM反馈。

> [!tip] 记忆口诀
> 安全输出切危险，辅助输出给状态。

# 05 手动复位与自动复位

> [!info] 核心结论
> 安全继电器复位方式决定安全条件恢复后，设备是否需要人工确认才能重新启动。

## ① 分类/公式/规则

> [!note] 🟦 复位方式
> **自动复位**：安全条件恢复后自动允许输出。  
> **手动复位**：需要按复位按钮确认后才允许输出。  
> 人员可能进入危险区的场景，通常需要手动复位。

> [!warning] 🟧 手动复位公式
> **重新允许运行 = 安全条件恢复 + 危险区确认 + 手动复位**

## ② 参考表

| 复位方式 | 适合场景 |
|---|---|
| **自动复位** | 风险较低或无人进入区域 |
| **手动复位** | 人员可能进入危险区域 |
| **监控复位** | 检测复位按钮是否粘连 |
| **启动联锁** | 防止上电自动运行 |
| **重启联锁** | 防止安全条件恢复后自动启动 |
| **复位按钮位置** | 应能看到危险区域 |

## ③ 计算/选型示例

- 已知：  
  操作员打开安全门进入设备内部处理卡料。
- 公式：  
  **安全恢复 = 人员离开 + 门关闭 + 人工确认复位**
- 计算：  
  门关上后不能让设备自动启动，需要确认危险区无人。
- 结论：  
  应采用 **手动复位**，复位按钮位置需能观察危险区域。

> [!tip] 记忆口诀
> 人能进去的地方，复位不能自动来。

# 06 EDM反馈监控

> [!info] 核心结论
> EDM反馈用于监控接触器、继电器等执行元件是否真的断开，防止触点粘连造成危险。

## ① 分类/公式/规则

> [!note] 🟦 EDM理解
> EDM是外部设备监控。  
> 安全继电器输出断开后，会通过接触器辅助触点检查后级接触器是否释放。

> [!warning] 🟧 EDM判断公式
> **允许复位 = 安全输入正常 + 接触器反馈已断开 + 复位条件满足**

## ② 参考表

| 项目 | 作用 |
|---|---|
| **接触器主触点** | 切断电机或负载 |
| **接触器辅助触点** | 返回状态给安全继电器 |
| **触点粘连** | 接触器失电但主触点未断开 |
| **EDM回路** | 发现后级执行元件异常 |
| **复位禁止** | 反馈异常时不允许再次启动 |
| **故障诊断** | 帮助定位接触器问题 |

## ③ 计算/选型示例

- 已知：  
  急停后接触器KM1没有释放，安全继电器无法复位。
- 公式：  
  **无法复位 = EDM反馈异常**
- 计算：  
  安全继电器检测到接触器辅助触点状态不符合预期，判断后级可能粘连。
- 结论：  
  应检查KM1接触器、辅助触点接线和反馈回路。

> [!tip] 记忆口诀
> 输出断开不够，还要确认接触器真断开。

# 07 常见输入设备接法

> [!info] 核心结论
> 安全继电器可以接急停、安全门、安全光幕等，但不同输入设备接线逻辑不同，不能随意混接。

## ① 分类/公式/规则

> [!note] 🟦 输入设备分类
> 急停和机械安全门常用双通道常闭触点；  
> 安全光幕常用双OSSD输出；  
> 安全门锁可能同时有门状态、锁状态和解锁控制。

> [!warning] 🟧 输入匹配公式
> **安全输入匹配 = 设备输出类型 + 安全继电器输入类型 + 检测功能一致**

## ② 参考表

| 输入设备 | 常见接法 |
|---|---|
| **急停按钮** | 双NC触点输入 |
| **安全门开关** | 双通道触点输入 |
| **安全门锁** | 安全触点/OSSD + 锁定反馈 |
| **安全光幕** | OSSD1 + OSSD2 |
| **双手按钮** | 需专用双手控制安全模块 |
| **安全地毯** | 需匹配对应安全输入模块 |

## ③ 计算/选型示例

- 已知：  
  安全光幕输出为OSSD1、OSSD2，需要接安全继电器。
- 公式：  
  **光幕接入 = OSSD输出 + 支持OSSD输入的安全继电器**
- 计算：  
  如果安全继电器只支持机械触点输入，可能无法直接匹配OSSD。
- 结论：  
  应选择支持安全光幕OSSD输入的安全继电器或安全控制器。

> [!tip] 记忆口诀
> 触点接触点，OSSD接OSSD，输入类型要对上。

# 08 安全等级与风险评估

> [!info] 核心结论
> 安全继电器选型不能只看价格和输出点数，必须根据设备风险评估确定安全等级要求。

## ① 分类/公式/规则

> [!note] 🟦 风险评估思路
> 风险越高，对安全回路的可靠性、诊断能力、冗余结构和故障检测要求越高。

> [!warning] 🟧 安全等级匹配公式
> **安全系统等级 = 输入设备等级 + 安全继电器等级 + 输出执行元件等级 + 接线结构**

## ② 参考表

| 风险因素 | 关注点 |
|---|---|
| **伤害严重度** | 轻伤、重伤、致命 |
| **暴露频率** | 偶尔进入、频繁进入 |
| **避免可能性** | 能否及时避开危险 |
| **危险停止时间** | 停机越慢风险越大 |
| **故障诊断能力** | 是否能发现单点故障 |
| **系统结构** | 单通道、双通道、带监控 |

## ③ 计算/选型示例

- 已知：  
  设备危险动作可能造成严重伤害，人员需要频繁进入。
- 公式：  
  **高风险 = 严重伤害 + 高频暴露 + 难以避免**
- 计算：  
  该场景不能只用普通继电器或单通道安全回路。
- 结论：  
  应按风险评估选择满足要求的安全继电器、安全输入和安全输出结构。

> [!tip] 记忆口诀
> 风险越高，安全回路越不能省。

# 09 常见故障排查

> [!info] 核心结论
> 安全继电器故障排查，要按“电源 → 输入 → 复位 → 反馈 → 输出”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看电源和指示灯，再看安全输入；  
> 先查复位按钮，再查EDM反馈；  
> 最后检查安全输出和后级接触器。

> [!warning] 🟧 排错公式
> **安全继电器异常 = 电源异常 + 输入未闭合 + 复位不满足 + EDM异常 + 输出回路故障**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无法启动** | 急停未释放、安全门未关 |
| **无法复位** | 复位回路错误、EDM异常 |
| **输出不闭合** | 输入条件不满足或内部故障 |
| **一复位就掉** | 输入抖动、通道不同步 |
| **偶发停机** | 线缆松动、触点抖动、干扰 |
| **PLC有状态但设备不动** | 接触器、驱动STO或主回路问题 |

## ③ 计算/选型示例

- 已知：  
  安全继电器输入灯正常，但输出灯不亮。
- 公式：  
  **输出允许 = 输入正常 + 复位正常 + EDM正常**
- 计算：  
  输入灯正常只代表安全输入恢复，可能复位或反馈条件未满足。
- 结论：  
  应检查复位按钮接线、复位模式和EDM反馈回路。

> [!tip] 记忆口诀
> 输入亮不等于输出亮，复位反馈也要看。

# 10 安全继电器快速选型表

> [!info] 核心结论
> 安全继电器选型时，先确认安全功能和风险等级，再确认输入类型、输出数量、复位方式和反馈监控。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 做风险评估  
> ② 确认安全输入设备  
> ③ 确认单通道/双通道  
> ④ 确认是否支持OSSD  
> ⑤ 确认安全输出点数  
> ⑥ 确认复位方式  
> ⑦ 确认EDM反馈  
> ⑧ 校核电源、响应时间和认证

> [!warning] 🟧 最终选型公式
> **推荐型号 = 安全等级 + 输入类型 + 输出触点数 + 复位方式 + EDM + 电源电压**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 急停回路 | 双NC输入安全继电器 |
| 安全门回路 | 双通道监控 |
| 安全光幕 | 支持OSSD输入 |
| 切接触器 | 安全触点数量足够 |
| 防止自动重启 | 手动复位 |
| 监控接触器 | EDM反馈 |
| 多安全设备 | 安全PLC或多功能安全模块 |

## ③ 计算/选型示例

- 已知：  
  设备有急停、安全门、安全光幕，并需要控制两个接触器。
- 公式：  
  **型号需求 = 多安全输入 + 双通道 + OSSD支持 + 2路安全输出 + EDM**
- 计算：  
  如果单个安全继电器输入能力不足，可考虑多个安全继电器或安全PLC。
- 结论：  
  需按安全回路结构选择合适型号，最终接线和安全等级需按实际规范/厂家手册校核。

> [!tip] 选型口诀
> 输入看设备，输出看接触器；复位防重启，反馈防粘连。
