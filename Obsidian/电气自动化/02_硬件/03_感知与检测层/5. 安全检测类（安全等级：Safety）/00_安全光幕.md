# 01 安全光幕总览

> [!info] 核心结论
> 安全光幕用于检测人员是否进入危险区域，一旦光束被遮挡，应通过安全回路让设备进入安全状态。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 安全光幕不是普通光电开关，而是用于人员安全防护的安全传感器。  
> 它通常由 **发射器、接收器、安全输出、控制单元、安全回路** 组成。

> [!warning] 🟧 通用选型公式
> **安全光幕选型 = 防护高度 + 光轴间距 + 安全距离 + 响应时间 + 安全等级 + 安装环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **发射器** | 发出多束红外光 |
| **接收器** | 接收光束并判断遮挡 |
| **OSSD输出** | 安全半导体输出 |
| **安全PLC/继电器** | 处理安全信号 |
| **防护对象** | 手指、手掌、手臂、人体 |
| **典型场景** | 冲压、机械手、输送线、装配设备 |

## ③ 计算/选型示例

- 已知：  
  操作员需要频繁伸手取放工件，设备动作区存在夹伤风险。
- 公式：  
  **安全需求 = 人员进入检测 + 快速停机 + 防止误启动**
- 计算：  
  普通光电只能检测有无，不能承担安全防护功能。
- 结论：  
  应选择 **安全光幕 + 安全继电器/安全PLC**，安全等级和安装距离需按实际规范/厂家手册校核。

> [!tip] 记忆口诀
> 普通光电管检测，安全光幕管人命。

# 02 底层原理

> [!info] 核心结论
> 安全光幕的本质，是用多束光形成一道“看不见的安全栅栏”。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 发射器连续发出多束光，接收器逐束接收。  
> 当人体、手或物体遮挡任意关键光束时，安全光幕立即切断安全输出，通知安全控制系统停机。

> [!warning] 🟧 安全检测逻辑
> **光束正常 → OSSD输出正常**  
> **光束被遮挡 → OSSD断开 → 安全回路动作 → 设备停机**

## ② 参考表

| 状态 | 光束情况 | 输出结果 |
|---|---|---|
| **无人遮挡** | 光束全部接收 | OSSD正常输出 |
| **有人进入** | 部分光束被挡 | OSSD断开 |
| **光轴未对准** | 接收不稳定 | 无法正常运行 |
| **光幕故障** | 自检异常 | 安全输出断开 |
| **线路异常** | 输出异常 | 安全系统报警 |

## ③ 计算/选型示例

- 已知：  
  光幕正常对准时设备允许运行，手伸入危险区后设备必须停止。
- 公式：  
  **安全动作 = 遮光检测 + OSSD断开 + 安全控制停机**
- 计算：  
  手遮挡光束后，安全输出断开，安全PLC或安全继电器切断危险动作。
- 结论：  
  安全光幕必须接入安全回路，不能只接普通PLC输入作为提示信号。

> [!tip] 记忆口诀
> 光不断，机可动；光一断，机要停。

# 03 关键参数

> [!info] 核心结论
> 安全光幕最关键的参数是 **保护高度、分辨率、检测距离、响应时间、安全等级**。

## ① 分类/公式/规则

> [!note] 🟦 参数主线
> 保护高度决定覆盖范围；  
> 分辨率决定能检测多小的物体；  
> 响应时间影响安全距离；  
> 安全等级决定是否满足风险评估要求。

> [!warning] 🟧 参数选型公式
> **光幕规格 = 防护高度 + 光轴间距/分辨率 + 检测距离 + 响应时间 + 安全等级**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **保护高度** | 覆盖危险开口高度 |
| **分辨率** | 检测手指、手掌、手臂或人体 |
| **检测距离** | 发射端到接收端距离 |
| **响应时间** | 遮挡到输出动作时间 |
| **OSSD输出** | 接安全回路 |
| **安全等级** | 需按风险评估校核 |

## ③ 计算/选型示例

- 已知：  
  需要防止操作员手进入危险区。
- 公式：  
  **防护对象 = 手指/手掌/手臂/人体**
- 计算：  
  如果要检测手指或手掌，光幕分辨率应更小；如果只做人体通道防护，分辨率可相对更大。
- 结论：  
  分辨率必须按防护对象选择，具体参数需按厂家手册和安全规范校核。

> [!tip] 记忆口诀
> 防手指看分辨率，防人体看保护高；距离越近，响应越要快。

# 04 安全距离计算

> [!info] 核心结论
> 安全光幕不能贴着危险点安装，必须留出足够安全距离，保证人到危险点前设备已经停止。

## ① 分类/公式/规则

> [!warning] 🟧 核心公式
> **S = K × T + C**

> [!note] 🟦 参数说明
> **S**：安全距离  
> **K**：人体接近速度参数  
> **T**：总响应时间  
> **C**：附加距离  
>  
> 具体K值、C值和计算方法必须按实际安全规范/厂家手册校核。

## ② 参考表

| 参数 | 说明 |
|---|---|
| **S** | 光幕到危险点距离 |
| **K** | 人体接近速度参数 |
| **T** | 光幕、控制器、机器停机总时间 |
| **C** | 与分辨率、防护方式相关 |
| **停机时间** | 需实测或按设备数据确认 |
| **安装位置** | 必须满足计算结果 |

## ③ 计算/选型示例

- 已知：  
  某设备总响应时间 T = 0.2s，K和C按项目规范取值。
- 公式：  
  **S = K × T + C**
- 计算：  
  将实际K、T、C代入后得到最小安全距离。
- 结论：  
  光幕实际安装距离必须 **不小于计算出的S值**，并需由专业人员按规范确认。

> [!tip] 记忆口诀
> 光幕不是装得越近越好，安全距离算够才可靠。

# 05 OSSD安全输出

> [!info] 核心结论
> 安全光幕通常使用OSSD安全输出，必须接入安全继电器或安全PLC。

## ① 分类/公式/规则

> [!note] 🟦 OSSD理解
> OSSD是安全输出信号，通常为双通道输出，用于检测短路、断线、交叉故障等异常。

> [!warning] 🟧 接线匹配公式
> **安全回路可靠 = 双通道OSSD + 安全控制器 + 正确复位 + 故障诊断**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **OSSD1** | 第一安全输出通道 |
| **OSSD2** | 第二安全输出通道 |
| **安全继电器** | 接收并处理OSSD信号 |
| **安全PLC** | 实现安全逻辑和诊断 |
| **EDM反馈** | 监控接触器是否正常断开 |
| **复位按钮** | 防止危险区恢复后自动启动 |

## ③ 计算/选型示例

- 已知：  
  光幕有两个OSSD输出，现场只接了一个到普通PLC输入。
- 公式：  
  **安全功能 = 双通道 + 安全控制 + 故障检测**
- 计算：  
  单通道普通输入无法满足安全回路要求，也不能可靠检测故障。
- 结论：  
  应将OSSD双通道接入安全PLC或安全继电器，接线需按厂家手册校核。

> [!tip] 记忆口诀
> OSSD要双通道，普通输入不算安全。

# 06 手动复位与防重启

> [!info] 核心结论
> 光幕被遮挡后，危险消除不代表设备可以自动重新启动，通常需要手动复位。

## ① 分类/公式/规则

> [!note] 🟦 防重启逻辑
> 当人员进入危险区后，光幕断开安全输出。  
> 人员离开后，光束恢复，但设备不应自动启动，必须确认危险区无人后再手动复位。

> [!warning] 🟧 安全复位公式
> **重新启动 = 光幕恢复 + 危险区确认 + 手动复位 + 启动命令**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **自动复位** | 光束恢复后自动允许运行，风险较高 |
| **手动复位** | 需要人工确认后恢复 |
| **启动联锁** | 防止上电自动运行 |
| **重启联锁** | 防止遮挡解除后自动启动 |
| **复位按钮位置** | 应能看到危险区 |
| **复位后启动** | 复位不等于直接启动 |

## ③ 计算/选型示例

- 已知：  
  操作员进入危险区处理卡料，离开后光幕恢复。
- 公式：  
  **安全恢复 = 人员离开 + 目视确认 + 手动复位**
- 计算：  
  如果光幕一恢复设备就自动动作，可能造成二次伤害。
- 结论：  
  应配置手动复位和防重启逻辑，复位按钮位置需按规范设计。

> [!tip] 记忆口诀
> 光束恢复不等于能启动，人工确认才放心。

# 07 安装与对准

> [!info] 核心结论
> 安全光幕安装时，要保证光轴对准、覆盖完整、固定牢靠，并防止人员从旁边、下面或上面绕过。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 光幕安装不是只要亮灯即可，还要确认危险区域是否被完整覆盖，是否存在绕过、钻入、跨越的可能。

> [!warning] 🟧 安装可靠公式
> **可靠防护 = 光轴对准 + 安全距离足够 + 覆盖完整 + 无绕行通道 + 固定牢靠**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **光轴对准** | 发射器和接收器对齐 |
| **安全距离** | 不小于计算值 |
| **防护高度** | 覆盖危险开口 |
| **下方间隙** | 防止从底部进入 |
| **侧面绕行** | 需要机械防护补充 |
| **固定强度** | 防止振动后偏移 |

## ③ 计算/选型示例

- 已知：  
  光幕正面能检测人员，但侧面有空隙可以绕进去。
- 公式：  
  **防护有效 = 光幕检测 + 机械防护 + 无绕过路径**
- 计算：  
  如果人员能绕过光幕进入危险区，光幕本身再正常也不能形成完整保护。
- 结论：  
  需增加侧面防护栏、挡板或调整光幕位置，确保无法绕过。

> [!tip] 记忆口诀
> 光幕只防光路，绕过去就失效。

# 08 Muting 与 Blanking

> [!info] 核心结论
> Muting和Blanking是安全光幕的高级功能，必须谨慎使用，避免把人员防护误屏蔽。

## ① 分类/公式/规则

> [!note] 🟦 功能区别
> **Muting屏蔽**：在特定条件下临时屏蔽光幕，例如物料通过。  
> **Blanking消隐**：允许固定物体或特定区域遮挡光束。  
> 这些功能必须通过安全逻辑严格控制。

> [!warning] 🟧 使用判断公式
> **允许屏蔽 = 物料通过 + 人员不能进入 + 条件受控 + 安全逻辑验证**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **Muting** | 临时允许物料通过 |
| **Fixed Blanking** | 固定遮挡区域消隐 |
| **Floating Blanking** | 允许一定范围移动遮挡 |
| **Muting传感器** | 判断物料进入方向和时序 |
| **屏蔽指示灯** | 提示光幕处于屏蔽状态 |
| **风险点** | 设置错误可能导致防护失效 |

## ③ 计算/选型示例

- 已知：  
  输送线上托盘需要穿过光幕，但人员不能跟随进入危险区。
- 公式：  
  **Muting有效 = 托盘识别 + 时序正确 + 人员不可进入**
- 计算：  
  需要通过额外传感器判断是托盘而不是人员，并限制屏蔽时间和范围。
- 结论：  
  可评估Muting功能，但必须按安全规范设计和验证。

> [!tip] 记忆口诀
> 屏蔽物料可以，屏蔽人员不行。

# 09 常见故障排查

> [!info] 核心结论
> 安全光幕故障排查，要先确认对准、遮挡、接线、OSSD状态、复位逻辑和安全控制器报警。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看指示灯，再看光轴；  
> 先查遮挡，再查接线；  
> 先查OSSD，再查安全PLC/继电器。

> [!warning] 🟧 排错公式
> **光幕异常 = 光轴未对准 + 有遮挡 + 接线错误 + OSSD故障 + 复位条件不满足 + 安全控制器报警**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无法复位** | 光束未恢复、复位回路错误 |
| **一直报警** | 光轴未对准或有遮挡 |
| **设备不能启动** | OSSD未输出、安全回路未闭合 |
| **偶发停机** | 振动、灰尘、反光、线缆松动 |
| **PLC无信号** | 接线或安全模块参数错误 |
| **复位后又跳停** | 接触器反馈EDM异常 |

## ③ 计算/选型示例

- 已知：  
  光幕指示灯显示对准正常，但安全PLC仍然报警。
- 公式：  
  **安全回路正常 = 光幕正常 + OSSD正常 + 反馈回路正常 + 复位条件满足**
- 计算：  
  光幕正常不代表整条安全链正常，可能是EDM反馈或安全程序条件不满足。
- 结论：  
  应继续检查OSSD输入、安全PLC诊断、复位按钮和接触器反馈。

> [!tip] 记忆口诀
> 光幕绿灯只是开始，安全回路全通才算完成。

# 10 安全光幕快速选型表

> [!info] 核心结论
> 安全光幕选型必须以风险评估为前提，不能只按尺寸和价格选择。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 做设备风险评估  
> ② 确认防护对象  
> ③ 选择分辨率  
> ④ 确认保护高度  
> ⑤ 计算安全距离  
> ⑥ 校核响应时间  
> ⑦ 选择安全等级  
> ⑧ 确认接线和复位逻辑

> [!warning] 🟧 最终选型公式
> **推荐型号 = 安全等级 + 分辨率 + 保护高度 + 检测距离 + 响应时间 + OSSD输出 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 防手指进入 | 小分辨率光幕 |
| 防手掌进入 | 分辨率需校核 |
| 防人体进入 | 保护高度和安装距离 |
| 冲压设备 | 响应时间和安全距离重点校核 |
| 输送线进出口 | 可能涉及Muting |
| 潮湿粉尘环境 | 防护等级和清洁维护 |
| 接安全PLC | OSSD双通道和诊断 |

## ③ 计算/选型示例

- 已知：  
  设备开口高度800mm，需要防止人员伸手进入危险区，设备停机时间较长。
- 公式：  
  **型号需求 = 保护高度≥开口高度 + 合适分辨率 + 足够安全距离**
- 计算：  
  停机时间越长，光幕距离危险点应越远。
- 结论：  
  应选择保护高度覆盖开口的安全光幕，并按停机时间计算安全距离，最终需专业人员按规范校核。

> [!tip] 选型口诀
> 先评风险，再定等级；先算距离，再装光幕；安全回路，不能偷步。
