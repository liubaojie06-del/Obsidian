# 01 电梯光幕总览

> [!info] 核心结论
> 电梯光幕用于检测乘客或物体是否处于电梯门关闭区域，防止电梯门夹人、夹物。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 电梯光幕通常安装在轿门两侧，一边为发射端，一边为接收端。  
> 当人、手、行李或其他物体遮挡光束时，光幕输出信号，控制电梯门重新打开或停止关门。

> [!warning] 🟧 通用选型公式
> **电梯光幕选型 = 光束数量 + 保护高度 + 检测距离 + 响应时间 + 输出方式 + 安装尺寸**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **类型** | 电梯门区保护光幕 |
| **安装位置** | 电梯轿门两侧 |
| **检测对象** | 人体、手、行李、推车、物体 |
| **核心功能** | 防夹、防撞、门区保护 |
| **常见输出** | 继电器、NPN、PNP等需校核 |
| **重点参数** | 光束数、保护高度、响应时间、抗干扰 |

## ③ 计算/选型示例

- 已知：  
  电梯门关闭过程中，需要检测是否有人或物体挡在门口。
- 公式：  
  **门区保护 = 光束检测 + 遮挡判断 + 门机反开控制**
- 计算：  
  当乘客手臂遮挡光束，接收端检测到光束中断，控制系统应停止关门并重新开门。
- 结论：  
  应选用适合门宽、门高和控制接口的 **电梯光幕**，具体参数需按电梯规范/厂家手册校核。

> [!tip] 记忆口诀
> 门口有遮挡，光幕先发现；关门不硬夹，安全靠反开。

# 02 底层原理

> [!info] 核心结论
> 电梯光幕的本质，是用多束红外光形成一道“看不见的门区保护网”。

## ① 分类/公式/规则

> [!note] 🟦 工作原理
> 发射端连续发出多束红外光，接收端逐束接收。  
> 当乘客或物体进入门区时，部分光束被遮挡，接收端光信号变化，光幕控制器输出遮挡信号。

> [!warning] 🟧 检测逻辑
> **发射红外光 → 接收端收光 → 物体遮挡 → 光束中断 → 输出防夹信号**

## ② 参考表

| 状态 | 光束情况 | 门机动作 |
|---|---|---|
| **无遮挡** | 光束正常接收 | 允许关门 |
| **有人遮挡** | 光束被切断 | 停止关门 / 反开 |
| **光轴偏移** | 接收不稳定 | 可能误动作 |
| **灰尘遮挡** | 光强变弱 | 可能误报警 |
| **线路故障** | 信号异常 | 需检查接线 |

## ③ 计算/选型示例

- 已知：  
  电梯关门时，乘客伸手挡门。
- 公式：  
  **防夹动作 = 遮光检测 + 输出信号 + 门机反开**
- 计算：  
  手臂进入门区后遮挡多束光，光幕输出遮挡信号。
- 结论：  
  门机控制系统应立即停止关门并执行反开动作。

> [!tip] 记忆口诀
> 光束一断，门机反转；光束恢复，才可关门。

# 03 关键参数

> [!info] 核心结论
> 电梯光幕最重要的参数是 **光束数量、保护高度、检测距离、响应时间和输出方式**。

## ① 分类/公式/规则

> [!note] 🟦 参数主线
> 光束数量越多，检测越密；  
> 保护高度越大，覆盖范围越高；  
> 响应时间越短，防夹动作越及时。

> [!warning] 🟧 参数选型公式
> **光幕规格 = 光束数量 + 保护高度 + 门宽距离 + 响应时间 + 输出接口**

## ② 参考表

| 参数 | 作用 |
|---|---|
| **光束数量** | 决定检测密度 |
| **保护高度** | 覆盖电梯门区高度 |
| **检测距离** | 适配门宽 |
| **响应时间** | 遮挡后多久输出 |
| **输出方式** | 与门机控制器匹配 |
| **抗光干扰** | 防止阳光、灯光误触发 |

## ③ 计算/选型示例

- 已知：  
  电梯门宽为900mm，需要安装光幕。
- 公式：  
  **检测距离 ≥ 电梯门宽 + 安装余量**
- 计算：  
  光幕有效检测距离必须覆盖900mm门宽，并考虑安装位置和偏差。
- 结论：  
  应选择检测距离满足门宽要求的电梯光幕，具体型号需按门机结构和厂家手册校核。

> [!tip] 记忆口诀
> 门多宽，光幕距离要够；门多高，保护高度要够。

# 04 光束数量与保护高度

> [!info] 核心结论
> 光束数量和保护高度决定电梯光幕能否完整覆盖乘客容易进入的门区范围。

## ① 分类/公式/规则

> [!note] 🟦 选型逻辑
> 光束越密，对小物体越敏感；  
> 保护高度不足，可能出现检测盲区；  
> 安装位置偏移，也会造成局部漏检。

> [!warning] 🟧 覆盖判断公式
> **有效保护 = 光束密度足够 + 保护高度足够 + 安装位置正确**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **光束少** | 检测间隙大 |
| **光束多** | 检测更细密 |
| **保护高度低** | 上部或下部可能漏检 |
| **安装过偏** | 门区覆盖不完整 |
| **底部盲区** | 小物体可能漏检 |
| **顶部盲区** | 高位物体可能漏检 |

## ③ 计算/选型示例

- 已知：  
  现场经常有儿童、行李箱、推车通过电梯门。
- 公式：  
  **防护能力 = 光束密度 + 覆盖高度 + 响应速度**
- 计算：  
  检测对象尺寸差异大，应选择光束较密、保护高度合适的光幕。
- 结论：  
  电梯光幕不宜只看价格，应重点校核光束数量和保护范围。

> [!tip] 记忆口诀
> 光束越密，漏检越少；高度不够，保护打折。

# 05 输出信号与门机控制

> [!info] 核心结论
> 电梯光幕必须与门机控制器输入方式匹配，否则可能出现不反开、误反开或信号异常。

## ① 分类/公式/规则

> [!note] 🟦 常见输出
> 电梯光幕常见输出包括继电器触点、NPN、PNP等。  
> 不同门机控制器对输入信号要求不同，接线前必须核对手册。

> [!warning] 🟧 接线匹配公式
> **门机识别正常 = 光幕输出类型 + 门机输入类型 + 电源电压 + 信号逻辑一致**

## ② 参考表

| 输出类型 | 说明 |
|---|---|
| **继电器输出** | 干接点，兼容性较强 |
| **NPN输出** | 输出0V信号 |
| **PNP输出** | 输出正电信号 |
| **常开NO** | 遮挡后闭合或输出有效 |
| **常闭NC** | 遮挡后断开或输出无效 |
| **电源电压** | 常见DC24V，需按型号校核 |

## ③ 计算/选型示例

- 已知：  
  门机控制器要求常闭触点信号，现场光幕输出为NPN。
- 公式：  
  **信号匹配 = 输出方式一致 + 逻辑一致**
- 计算：  
  NPN信号不能直接等同于常闭触点，可能需要转换模块或更换型号。
- 结论：  
  应选择与门机控制器匹配的光幕输出方式，接线按厂家手册校核。

> [!tip] 记忆口诀
> 光幕能检测不代表门机会反开，输出逻辑必须对。

# 06 安装与对准

> [!info] 核心结论
> 电梯光幕安装时，重点是发射端和接收端对准、固定牢靠、线缆不受挤压，并避开门刀和机械干涉。

## ① 分类/公式/规则

> [!note] 🟦 安装步骤
> ① 确认安装位置  
> ② 固定发射端和接收端  
> ③ 调整光轴对准  
> ④ 检查门区覆盖  
> ⑤ 接线测试输出  
> ⑥ 多次开关门验证

> [!warning] 🟧 安装可靠公式
> **稳定工作 = 光轴对准 + 支架牢固 + 无机械干涉 + 线缆固定 + 门机逻辑正确**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **发射/接收方向** | 不要装反 |
| **光轴对准** | 保证接收稳定 |
| **安装高度** | 覆盖门区 |
| **固定强度** | 防止振动偏移 |
| **线缆走向** | 避免夹线、拉扯 |
| **机械间隙** | 避免与轿门运动干涉 |

## ③ 计算/选型示例

- 已知：  
  光幕安装后，电梯门关闭过程中偶尔误反开。
- 公式：  
  **误反开原因 = 光轴偏移 + 机械振动 + 线缆接触不良 + 灰尘遮挡**
- 计算：  
  关门振动可能导致光轴瞬间偏移或线缆接触不良。
- 结论：  
  应重新检查安装固定、光轴对准、线缆和门机运行干涉。

> [!tip] 记忆口诀
> 光幕装得正，门机才稳定；光轴一偏，误动作连连。

# 07 常见故障排查

> [!info] 核心结论
> 电梯光幕故障排查，要按“供电 → 对准 → 遮挡 → 输出 → 门机逻辑”的顺序检查。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看电源和指示灯，再看光轴和遮挡；  
> 先确认光幕是否输出，再确认门机是否响应。

> [!warning] 🟧 排错公式
> **光幕异常 = 供电异常 + 光轴未对准 + 灰尘遮挡 + 接线错误 + 输出逻辑不匹配 + 门机参数错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **一直反开** | 光轴未对准、长期遮挡、输出逻辑反 |
| **不反开** | 无输出、接线错误、门机输入异常 |
| **偶发误动作** | 振动、灰尘、阳光干扰、线缆松动 |
| **指示灯不亮** | 电源故障或接线错误 |
| **只在关门时异常** | 机械干涉或线缆被拉扯 |
| **更换后不兼容** | 输出类型或电压不匹配 |

## ③ 计算/选型示例

- 已知：  
  光幕指示灯状态正常变化，但电梯门不反开。
- 公式：  
  **门机不反开 = 光幕输出未进入门机有效输入**
- 计算：  
  光幕本体检测正常，问题可能在输出接线、门机输入端或控制参数。
- 结论：  
  应检查输出端子、门机输入类型、常开/常闭逻辑和门机参数设置。

> [!tip] 记忆口诀
> 灯变门不动，先查输出和门机输入。

# 08 电梯光幕快速选型表

> [!info] 核心结论
> 电梯光幕选型先看门宽和保护高度，再看光束数量、输出方式、安装尺寸和抗干扰能力。

## ① 分类/公式/规则

> [!note] 🟦 七步选型法
> ① 看电梯门宽  
> ② 看门区保护高度  
> ③ 看光束数量  
> ④ 看响应时间  
> ⑤ 看输出类型  
> ⑥ 看电源电压  
> ⑦ 看安装尺寸和线缆方式

> [!warning] 🟧 最终选型公式
> **推荐型号 = 门宽距离 + 保护高度 + 光束数量 + 输出方式 + 电源电压 + 安装结构**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 防夹人 | 光束密度和响应时间 |
| 防夹行李 | 保护高度和底部覆盖 |
| 老电梯改造 | 输出兼容性 |
| 新梯配套 | 门机接口匹配 |
| 误动作多 | 抗干扰和安装稳定性 |
| 门宽较大 | 检测距离需满足 |
| 线缆易运动 | 线缆固定和耐弯折 |

## ③ 计算/选型示例

- 已知：  
  电梯门宽900mm，门高2100mm，需要更换门区保护光幕。
- 公式：  
  **型号需求 = 检测距离≥900mm + 保护高度覆盖主要门区 + 输出匹配门机**
- 计算：  
  门宽决定检测距离，门区高度决定保护高度，门机输入决定输出方式。
- 结论：  
  应选择适配门宽、保护高度和门机输入的电梯光幕，最终按电梯厂家手册校核。

> [!tip] 选型口诀
> 门宽定距离，门高定保护；输出配门机，安装看结构。
