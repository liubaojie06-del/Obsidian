# 01 安全门锁总览

> [!info] 核心结论
> 安全门锁用于监控防护门状态，必要时锁住防护门，防止人员在危险运动未停止前进入危险区域。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 安全门锁不是普通门磁开关，而是设备安全回路的一部分。  
> 它通常用于 **安全门、防护罩、围栏门、机器人工作站、危险运动区域**。

> [!warning] 🟧 通用选型公式
> **安全门锁选型 = 风险等级 + 是否需要锁定 + 锁紧力 + 安全输出 + 解锁方式 + 安装环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **安全门锁** | 监控门状态并实现联锁保护 |
| **安全门开关** | 只检测门开/关 |
| **带锁安全门锁** | 门关闭后可锁定 |
| **安全输出** | OSSD、双通道触点等 |
| **控制对象** | 安全PLC、安全继电器 |
| **典型场景** | 机器人、冲压、包装机、围栏门 |

## ③ 计算/选型示例

- 已知：  
  机器人工作站有防护门，机器人停止需要一定时间。
- 公式：  
  **安全需求 = 门状态检测 + 危险运动停止 + 防止提前开门**
- 计算：  
  如果只检测门开关，人员可能在机器人未完全停止前打开门进入。
- 结论：  
  应使用 **带锁安全门锁**，并接入安全PLC或安全继电器；安全等级需按风险评估和厂家手册校核。

> [!tip] 记忆口诀
> 门开要停机，危险没停门不开。

# 02 底层原理

> [!info] 核心结论
> 安全门锁的本质，是把“门是否关闭、锁是否到位、是否允许进入”转换成安全信号。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 防护门关闭后，门锁识别执行件或钥匙位置。  
> 如果条件满足，门锁输出安全信号；如果门打开、锁未到位或内部故障，安全输出断开。

> [!warning] 🟧 安全联锁逻辑
> **门关闭 → 锁到位 → 安全输出正常 → 设备允许运行**  
> **门打开 / 锁异常 → 安全输出断开 → 设备安全停机**

## ② 参考表

| 状态 | 门锁判断 | 安全输出 |
|---|---|---|
| **门关闭且锁定** | 正常 | 允许运行 |
| **门打开** | 危险区域可进入 | 安全输出断开 |
| **门未锁紧** | 锁定无效 | 安全输出断开 |
| **急停或故障** | 安全条件不满足 | 安全输出断开 |
| **需要解锁** | 等待安全条件满足 | 按逻辑解锁 |

## ③ 计算/选型示例

- 已知：  
  防护门已关闭，但设备仍无法启动。
- 公式：  
  **启动允许 = 门关闭 + 锁定到位 + 安全回路正常 + 复位完成**
- 计算：  
  门关闭不代表锁定到位，安全PLC可能仍未收到锁定确认信号。
- 结论：  
  应检查门锁到位信号、安全输出、复位条件和安全控制器诊断。

> [!tip] 记忆口诀
> 门关只是第一步，锁住才算安全门。

# 03 安全门锁类型

> [!info] 核心结论
> 安全门锁按功能可分为 **无锁型安全门开关** 和 **带锁型安全门锁**。

## ① 分类/公式/规则

> [!note] 🟦 类型区别
> 无锁型只检测门开关状态；  
> 带锁型既检测门状态，又能在危险未消除前保持门锁定。

> [!warning] 🟧 类型选择公式
> **是否需要锁定 = 危险运动停止时间 + 人员进入风险 + 工艺要求**

## ② 参考表

| 类型 | 适合场景 |
|---|---|
| **无锁安全门开关** | 门一开设备立即安全停机 |
| **带锁安全门锁** | 危险运动有停止时间 |
| **机械钥匙式** | 结构简单，安装直观 |
| **非接触式** | 磨损少，适合频繁开关门 |
| **电磁锁定式** | 可由控制系统控制解锁 |
| **编码式门锁** | 防旁路能力更强 |

## ③ 计算/选型示例

- 已知：  
  设备开门后刀盘还会惯性旋转5秒。
- 公式：  
  **锁定需求 = 危险停止时间 > 人员进入时间**
- 计算：  
  如果门立即打开，人员可能接触未停止刀盘。
- 结论：  
  应选择 **带锁安全门锁**，直到刀盘停止后才允许解锁。

> [!tip] 记忆口诀
> 危险能立停，可用门开关；危险有惯性，必须门锁上。

# 04 锁定方式

> [!info] 核心结论
> 安全门锁常见锁定方式有通电解锁、通电锁定和机械保持，不同方式适合不同安全策略。

## ① 分类/公式/规则

> [!note] 🟦 常见方式
> **通电解锁**：断电时保持锁住，安全性高。  
> **通电锁定**：通电时锁住，断电可打开。  
> **机械保持**：通过机械结构保持锁定状态。

> [!warning] 🟧 解锁策略公式
> **解锁允许 = 危险已停止 + 安全条件满足 + 控制系统发出解锁命令**

## ② 参考表

| 锁定方式 | 特点 |
|---|---|
| **通电解锁** | 断电保持锁定，常用于保护人员 |
| **通电锁定** | 通电时锁门，断电可释放 |
| **弹簧锁定** | 机械弹簧保持锁住 |
| **电磁解锁** | 控制系统允许后解锁 |
| **手动应急解锁** | 维修或紧急情况下使用 |
| **逃生释放** | 人员被困时从内部释放 |

## ③ 计算/选型示例

- 已知：  
  设备断电时危险部件仍可能因惯性继续运动。
- 公式：  
  **断电安全 = 断电后门仍不能立即打开**
- 计算：  
  如果断电门自动解锁，可能存在进入风险。
- 结论：  
  应评估 **通电解锁 / 弹簧锁定型安全门锁**，具体方式需按风险评估和规范校核。

> [!tip] 记忆口诀
> 断电还危险，门锁不能随便开。

# 05 关键参数

> [!info] 核心结论
> 安全门锁选型重点看 **锁紧力、安全等级、输出类型、解锁方式、防护等级和安装尺寸**。

## ① 分类/公式/规则

> [!note] 🟦 参数主线
> 锁紧力决定抗拉能力；  
> 输出类型决定接入安全控制器方式；  
> 防护等级决定是否适合粉尘、油污、潮湿环境。

> [!warning] 🟧 参数选型公式
> **门锁规格 = 锁紧力 + 安全输出 + 电源电压 + 解锁方式 + 机械接口 + 防护等级**

## ② 参考表

| 参数 | 说明 |
|---|---|
| **锁紧力** | 门锁能承受的拉力 |
| **安全输出** | OSSD或双通道触点 |
| **辅助输出** | 门状态、锁状态反馈 |
| **电源电压** | 常见DC24V，需按型号校核 |
| **防护等级** | IP等级、防尘防水 |
| **机械寿命** | 开关门次数能力 |

## ③ 计算/选型示例

- 已知：  
  防护门较大，人员可能误拉门，现场有振动。
- 公式：  
  **可靠锁定 = 锁紧力足够 + 安装牢固 + 门体刚性满足**
- 计算：  
  门体越大、振动越强，对锁紧力和安装强度要求越高。
- 结论：  
  应选择锁紧力满足要求的安全门锁，并校核门框刚性和安装支架强度。

> [!tip] 记忆口诀
> 门大看锁力，环境差看防护，接系统看输出。

# 06 安全输出与接线

> [!info] 核心结论
> 安全门锁必须接入安全PLC或安全继电器，不能只接普通PLC输入当作安全保护。

## ① 分类/公式/规则

> [!note] 🟦 常见信号
> 安全门锁常见信号包括：安全输出、门关闭反馈、锁定反馈、故障信号、解锁控制信号。

> [!warning] 🟧 接线匹配公式
> **安全回路可靠 = 双通道安全输出 + 安全控制器 + 反馈监控 + 正确复位逻辑**

## ② 参考表

| 信号 | 作用 |
|---|---|
| **OSSD1/OSSD2** | 安全输出双通道 |
| **门关闭信号** | 判断门是否关好 |
| **锁定信号** | 判断锁舌是否锁住 |
| **解锁输入** | 控制门锁释放 |
| **故障输出** | 诊断门锁异常 |
| **EDM反馈** | 监控接触器或执行元件 |

## ③ 计算/选型示例

- 已知：  
  门锁安全输出只接到普通PLC输入，用程序判断停机。
- 公式：  
  **安全功能 = 安全输入 + 安全逻辑 + 安全输出**
- 计算：  
  普通PLC输入无法可靠诊断短路、断线和输出故障。
- 结论：  
  安全门锁应接入 **安全PLC或安全继电器**，普通PLC只能做状态显示。

> [!tip] 记忆口诀
> 普通PLC能显示，安全PLC管停机。

# 07 解锁逻辑

> [!info] 核心结论
> 安全门锁不能随意解锁，必须在危险运动停止、能量释放或安全条件满足后才允许开门。

## ① 分类/公式/规则

> [!note] 🟦 解锁条件
> 常见解锁条件包括：设备停止、速度为零、压力释放、温度安全、机器人回安全位、操作员请求开门。

> [!warning] 🟧 解锁允许公式
> **允许解锁 = 停机完成 + 危险能量消除 + 安全控制确认 + 人工请求**

## ② 参考表

| 条件 | 说明 |
|---|---|
| **速度为零** | 电机、主轴、刀盘停止 |
| **压力释放** | 气压、液压释放到安全状态 |
| **温度安全** | 高温区域降到安全范围 |
| **机器人安全位** | 机器人离开危险位置 |
| **开门请求** | 操作员按下请求按钮 |
| **安全确认** | 安全PLC逻辑允许解锁 |

## ③ 计算/选型示例

- 已知：  
  操作员按下开门请求按钮，但主轴还在减速。
- 公式：  
  **解锁 = 开门请求 + 主轴停止确认**
- 计算：  
  开门请求只是申请，不能代表危险已消除。
- 结论：  
  安全PLC应等主轴停止确认后，再允许门锁解锁。

> [!tip] 记忆口诀
> 申请开门不等于马上开，危险停稳才解锁。

# 08 安装与防旁路

> [!info] 核心结论
> 安全门锁安装必须防止被轻易旁路、误触发或因门体偏移导致安全失效。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> 执行件与门锁要对准，门体关闭后锁舌进入顺畅。  
> 安装位置要避免人员用工具、备用钥匙或简单磁铁轻易旁路安全功能。

> [!warning] 🟧 安装可靠公式
> **可靠防护 = 对准准确 + 固定牢靠 + 防旁路 + 门体强度足够 + 无绕行入口**

## ② 参考表

| 检查项 | 重点 |
|---|---|
| **对准** | 执行件进入顺畅，不别劲 |
| **固定** | 螺钉防松，支架牢靠 |
| **门体间隙** | 防止晃动导致误报警 |
| **防旁路** | 避免使用备用执行件欺骗门锁 |
| **逃生释放** | 人员可能被困时必须考虑 |
| **机械防护** | 门锁不能替代围栏结构强度 |

## ③ 计算/选型示例

- 已知：  
  设备门关闭后偶尔报警，轻推门体报警又消失。
- 公式：  
  **异常原因 = 门体晃动 + 执行件偏移 + 锁定不到位**
- 计算：  
  门体或支架刚性不足，会导致门锁检测信号不稳定。
- 结论：  
  应调整门锁位置，加固门体和执行件支架，确认锁定状态稳定。

> [!tip] 记忆口诀
> 门锁怕偏、怕松、怕旁路，安装牢靠才安全。

# 09 常见故障排查

> [!info] 核心结论
> 安全门锁故障排查，要按“门是否关好 → 锁是否到位 → 安全输出是否正常 → 安全控制器是否允许”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先看门体和执行件，再看指示灯和锁定信号；  
> 先查门锁本体，再查安全PLC或安全继电器诊断。

> [!warning] 🟧 排错公式
> **门锁异常 = 门未对准 + 锁舌卡滞 + 输出故障 + 解锁条件不满足 + 接线错误 + 安全逻辑未复位**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **无法启动设备** | 门未关好、锁未到位、安全回路断开 |
| **无法解锁** | 危险未停止、解锁信号未给出 |
| **频繁报警** | 门体晃动、执行件偏移 |
| **PLC无信号** | 接线错误、输出类型不匹配 |
| **锁不住门** | 锁舌损坏、机械干涉 |
| **复位失败** | 门锁状态或EDM反馈不满足 |

## ③ 计算/选型示例

- 已知：  
  门锁指示灯显示锁定正常，但安全PLC仍不允许启动。
- 公式：  
  **启动允许 = 门锁安全输出 + 其他安全条件 + 复位完成**
- 计算：  
  门锁正常不代表急停、光幕、接触器反馈等其他安全条件也正常。
- 结论：  
  应查看安全PLC诊断，确认整条安全链路是否全部满足。

> [!tip] 记忆口诀
> 门锁正常不代表安全链正常，诊断要看全回路。

# 10 安全门锁快速选型表

> [!info] 核心结论
> 安全门锁选型要以风险评估为前提，先确认是否需要锁定，再选输出、锁紧力和解锁方式。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 做设备风险评估  
> ② 判断是否需要锁门  
> ③ 确认危险停止时间  
> ④ 选择安全等级  
> ⑤ 确认锁紧力  
> ⑥ 选择输出类型  
> ⑦ 设计解锁和复位逻辑  
> ⑧ 校核安装环境和防旁路

> [!warning] 🟧 最终选型公式
> **推荐型号 = 安全等级 + 带锁/无锁 + 锁紧力 + OSSD/触点输出 + 解锁方式 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 门开立即停机 | 无锁安全门开关可评估 |
| 危险停止有延时 | 带锁安全门锁 |
| 机器人工作站 | 带锁门锁 + 安全PLC |
| 频繁开关门 | 非接触或耐久型 |
| 断电仍有危险 | 弹簧锁定/通电解锁型 |
| 人员可能被困 | 逃生释放必须考虑 |
| 接安全系统 | OSSD或双通道安全触点 |

## ③ 计算/选型示例

- 已知：  
  防护门后方有高速旋转机构，停机时间为3秒，人员可能立即开门进入。
- 公式：  
  **型号需求 = 带锁安全门锁 + 停机确认 + 延时/零速解锁**
- 计算：  
  停机3秒内仍有危险，门不能立即打开。
- 结论：  
  应选 **带锁安全门锁**，由安全PLC确认机构停止后再解锁，具体安全等级需按规范校核。

> [!tip] 选型口诀
> 先评风险，再定门锁；危险没停稳，门就不能开。
