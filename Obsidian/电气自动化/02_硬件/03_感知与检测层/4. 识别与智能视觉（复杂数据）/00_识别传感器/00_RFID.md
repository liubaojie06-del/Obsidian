# 01 RFID 总览

> [!info] 核心结论
> RFID 用于 **非接触识别物体身份和数据**，常用于工装托盘、物料追踪、产线防错、仓储物流和设备追溯。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> RFID系统通常由 **读写器、天线、电子标签、通讯接口、控制系统** 组成。

> [!warning] 🟧 通用选型公式
> **RFID选型 = 识别距离 + 标签类型 + 频率 + 数据容量 + 通讯接口 + 安装环境**

## ② 参考表

| 组成 | 作用 |
|---|---|
| **读写器** | 读取或写入标签数据 |
| **天线** | 发射和接收射频信号 |
| **电子标签** | 存储ID或工艺数据 |
| **PLC/上位机** | 处理识别结果 |
| **通讯接口** | PROFINET、EtherNet/IP、RS485等 |

## ③ 计算/选型示例

- 已知：  
  产线托盘需要记录产品型号、工序状态和加工次数。
- 公式：  
  **RFID需求 = 托盘识别 + 数据写入 + 工序追踪**
- 计算：  
  只识别编号可用条码；需要反复写入工艺数据，更适合RFID。
- 结论：  
  可选 **工业RFID读写器 + 可写电子标签**，频率、容量和通讯接口需按厂家手册校核。

> [!tip] 记忆口诀
> 条码看得见才读，RFID隔空也能识别。

# 02 底层原理

> [!info] 核心结论
> RFID 的本质，是读写器通过无线射频信号与电子标签交换数据。

## ① 分类/公式/规则

> [!note] 🟦 工作流程
> 读写器发出射频信号，电子标签进入感应区域后被激活或响应。  
> 标签把自身ID或存储数据返回给读写器，读写器再传给PLC或上位机。

> [!warning] 🟧 RFID识别逻辑
> **读写器发射信号 → 标签响应 → 数据返回 → 控制系统判断**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **读** | 读取标签ID或数据区 |
| **写** | 把工艺数据写入标签 |
| **感应区域** | 标签可被识别的空间 |
| **数据区** | 标签内部可存储信息 |
| **防碰撞** | 多标签同时识别时的处理机制 |

## ③ 计算/选型示例

- 已知：  
  一个托盘进入工位后，PLC需要读取托盘编号。
- 公式：  
  **识别成功 = 标签进入感应区 + 通讯稳定 + 数据格式正确**
- 计算：  
  标签进入读写器范围后，读写器读取ID并发送给PLC。
- 结论：  
  RFID适合自动识别托盘身份，实际读取距离和稳定性需现场测试。

> [!tip] 记忆口诀
> RFID不是看图案，而是读芯片里的数据。

# 03 RFID 频率分类

> [!info] 核心结论
> RFID常见频段有低频、高频、超高频，不同频段的识别距离、抗金属能力和应用场景不同。

## ① 分类/公式/规则

> [!note] 🟦 频率选择
> 近距离、抗干扰、工业托盘常看 **低频/高频**；  
> 远距离、多标签、物流仓储常看 **超高频UHF**。

> [!warning] 🟧 频率选型公式
> **频率选择 = 识别距离 + 环境金属 + 标签数量 + 读取速度**

## ② 参考表

| 频段 | 特点 |
|---|---|
| **LF低频** | 距离短，抗干扰较好 |
| **HF高频** | 工业应用常见，距离中短 |
| **UHF超高频** | 距离较远，可多标签读取 |
| **近距离工位** | LF/HF常见 |
| **物流仓储** | UHF常见 |
| **金属环境** | 需选抗金属标签并校核 |

## ③ 计算/选型示例

- 已知：  
  工位只需在托盘到位后近距离读取标签。
- 公式：  
  **频率需求 = 近距离 + 单标签 + 工业现场**
- 计算：  
  不需要远距离批量读取，重点是稳定和抗干扰。
- 结论：  
  可优先评估 **HF工业RFID**，实际频率和型号需按设备系统校核。

> [!tip] 记忆口诀
> 近距离看稳定，远距离看UHF。

# 04 标签类型

> [!info] 核心结论
> RFID标签要按安装环境、数据需求和安装材料选择，尤其要注意是否贴在金属上。

## ① 分类/公式/规则

> [!note] 🟦 常见标签
> 标签可分为只读标签、可读写标签、抗金属标签、耐高温标签、柔性标签、螺钉固定标签等。

> [!warning] 🟧 标签选型公式
> **标签型号 = 安装材料 + 数据容量 + 温度环境 + 防护等级 + 固定方式**

## ② 参考表

| 标签类型 | 适合场景 |
|---|---|
| **只读标签** | 只识别ID |
| **可读写标签** | 工艺数据写入 |
| **抗金属标签** | 金属托盘、金属工装 |
| **耐高温标签** | 烘箱、喷涂、热处理 |
| **螺钉固定标签** | 振动或长期使用 |
| **柔性标签** | 包装、纸箱、物流 |

## ③ 计算/选型示例

- 已知：  
  标签需要安装在金属工装夹具上。
- 公式：  
  **标签可用性 = 抗金属设计 + 固定可靠 + 读取距离满足**
- 计算：  
  普通标签贴在金属上可能读取距离明显变短，甚至无法读取。
- 结论：  
  应选择 **抗金属RFID标签**，并按安装方向和读取距离现场校核。

> [!tip] 记忆口诀
> 标签贴金属，普通标签容易失灵。

# 05 读写器与天线

> [!info] 核心结论
> 读写器和天线决定RFID的识别距离、识别范围、通讯速度和现场稳定性。

## ① 分类/公式/规则

> [!note] 🟦 读写器选择
> 近距离单点识别可选一体式读写头；  
> 多点或远距离识别可选读写器加外接天线方案。

> [!warning] 🟧 读写器选型公式
> **读写器选型 = 频率 + 读取距离 + 标签数量 + 通讯接口 + 安装方式**

## ② 参考表

| 项目 | 关注点 |
|---|---|
| **一体式读写头** | 安装简单，适合单工位 |
| **分体式读写器** | 可接多天线，适合复杂系统 |
| **天线方向** | 影响读取范围 |
| **功率设置** | 影响距离和误读范围 |
| **通讯接口** | 与PLC或上位机匹配 |
| **防护等级** | 适合现场环境 |

## ③ 计算/选型示例

- 已知：  
  只需要在一个工位读取一个托盘标签。
- 公式：  
  **系统需求 = 单工位 + 单标签 + 近距离读取**
- 计算：  
  不需要多天线和远距离覆盖。
- 结论：  
  可选择 **一体式RFID读写头**，通讯接口按PLC系统选择。

> [!tip] 记忆口诀
> 单点读写用一体，多点覆盖用分体。

# 06 RFID 与 PLC 通讯

> [!info] 核心结论
> RFID接PLC时，重点确认通讯协议、数据地址、读写命令和数据格式。

## ① 分类/公式/规则

> [!note] 🟦 通讯内容
> PLC通常需要发送读命令、写命令、复位命令，并接收标签ID、数据区内容、状态字和错误码。

> [!warning] 🟧 通讯成功公式
> **RFID通讯成功 = 接口匹配 + 地址正确 + 命令正确 + 数据格式正确 + 状态正常**

## ② 参考表

| 通讯方式 | 常见用途 |
|---|---|
| **PROFINET** | 西门子PLC常用 |
| **EtherNet/IP** | 罗克韦尔系统常用 |
| **Modbus TCP/RTU** | 通用控制系统 |
| **RS232/RS485** | 简单串口通讯 |
| **IO-Link** | 小型识别和参数化 |
| **TCP/IP** | 上位机或MES对接 |

## ③ 计算/选型示例

- 已知：  
  RFID读写器通过PROFINET连接S7-1200 PLC。
- 公式：  
  **PLC读取 = 硬件组态 + 设备名/IP + 数据区映射 + 程序解析**
- 计算：  
  设备在线后，PLC需要按输入输出区读取状态和数据。
- 结论：  
  应导入设备文件并正确配置数据长度，具体地址和指令按厂家手册校核。

> [!tip] 记忆口诀
> 设备在线只是第一步，数据解析才算真正读到。

# 07 读写数据设计

> [!info] 核心结论
> RFID系统要提前规划标签数据结构，否则后期容易出现数据混乱、写错地址、追溯困难。

## ① 分类/公式/规则

> [!note] 🟦 数据规划
> 常见标签数据包括：产品编号、托盘编号、工序号、状态位、时间戳、加工次数、配方号等。

> [!warning] 🟧 数据结构公式
> **标签数据 = ID区 + 工艺数据区 + 状态区 + 校验/备用区**

## ② 参考表

| 数据项 | 用途 |
|---|---|
| **托盘ID** | 识别载具 |
| **产品型号** | 区分产品类型 |
| **工序状态** | 判断当前加工阶段 |
| **加工次数** | 追踪循环次数 |
| **配方编号** | 自动调用工艺参数 |
| **校验数据** | 防止读写错误 |

## ③ 计算/选型示例

- 已知：  
  标签需要存储托盘ID、产品型号、当前工序、加工次数。
- 公式：  
  **标签容量 ≥ 所有数据字节数 + 备用容量**
- 计算：  
  若当前数据约需要32字节，建议选容量大于32字节并留有余量的标签。
- 结论：  
  标签容量不要刚好够用，建议预留扩展空间，具体容量需按编码方式校核。

> [!tip] 记忆口诀
> 标签不是随便写，地址规划要先做。

# 08 安装与抗干扰

> [!info] 核心结论
> RFID读取不稳定，常见原因是金属干扰、距离不合适、方向不对、多个标签互相干扰或功率设置不合理。

## ① 分类/公式/规则

> [!note] 🟦 安装重点
> RFID安装要关注读写器与标签的距离、角度、金属环境、遮挡物、移动速度和识别区域边界。

> [!warning] 🟧 稳定识别公式
> **稳定识别 = 合理距离 + 正确方向 + 金属隔离 + 功率合适 + 标签唯一**

## ② 参考表

| 问题 | 处理方法 |
|---|---|
| **读不到标签** | 缩短距离、调整方向 |
| **偶尔漏读** | 降低速度、增加读取时间 |
| **误读旁边标签** | 降低功率、缩小读取区域 |
| **金属上读不到** | 使用抗金属标签 |
| **多标签冲突** | 启用防碰撞或限制标签数量 |
| **读写失败** | 检查写保护、地址和容量 |

## ③ 计算/选型示例

- 已知：  
  托盘移动速度较快，经过读写器时偶尔漏读。
- 公式：  
  **可读取时间 = 识别区域长度 ÷ 移动速度**
- 计算：  
  速度越快，标签停留在读取区域的时间越短。
- 结论：  
  可加长识别区域、降低速度、优化天线方向或提前触发读取。

> [!tip] 记忆口诀
> RFID读不稳，先查距离、方向、金属和速度。

# 09 常见故障排查

> [!info] 核心结论
> RFID故障排查要按“设备在线 → 标签可读 → 数据正确 → PLC解析正确”的顺序进行。

## ① 分类/公式/规则

> [!note] 🟦 排查顺序
> 先确认读写器供电和通讯，再确认标签位置和类型，最后检查PLC程序和数据地址。

> [!warning] 🟧 排错公式
> **RFID异常 = 通讯异常 + 标签不匹配 + 距离过远 + 金属干扰 + 数据地址错误 + 程序解析错误**

## ② 参考表

| 问题 | 可能原因 |
|---|---|
| **读写器不在线** | 电源、网络、设备名/IP错误 |
| **读不到标签** | 距离、方向、频率或标签类型错误 |
| **读到错误标签** | 读取范围太大或旁边有标签 |
| **写入失败** | 写保护、地址错误、容量不足 |
| **PLC数据乱码** | 字节顺序或编码格式错误 |
| **偶发漏读** | 速度快、金属干扰、识别时间短 |

## ③ 计算/选型示例

- 已知：  
  读写器在线，指示灯显示已读到标签，但PLC显示数据异常。
- 公式：  
  **数据异常 = 通讯读取成功但解析错误**
- 计算：  
  可能是PLC读取地址不对、字节顺序不对，或字符串编码方式不一致。
- 结论：  
  应检查PLC数据映射、字节顺序、数据长度和编码格式。

> [!tip] 记忆口诀
> 读不到查现场，读错了查程序。

# 10 RFID快速选型表

> [!info] 核心结论
> RFID选型先看识别距离和安装环境，再看频率、标签、读写器接口和数据容量。

## ① 分类/公式/规则

> [!note] 🟦 八步选型法
> ① 看识别距离  
> ② 看单标签还是多标签  
> ③ 看标签安装材料  
> ④ 看是否需要写数据  
> ⑤ 看频率类型  
> ⑥ 看通讯接口  
> ⑦ 看移动速度  
> ⑧ 看环境防护

> [!warning] 🟧 最终选型公式
> **推荐RFID系统 = 读写器 + 天线 + 标签 + 频率 + 通讯接口 + 数据容量 + 防护等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 托盘识别 | 工业HF/UHF RFID |
| 金属工装识别 | 抗金属标签 |
| 高速物流识别 | UHF和读取速度 |
| 工艺数据写入 | 可读写标签 |
| PLC直接通讯 | PROFINET/Modbus等 |
| 小型单点识别 | 一体式读写头 |
| 多点覆盖 | 分体式读写器+天线 |

## ③ 计算/选型示例

- 已知：  
  产线托盘为金属材质，需要在每个工位读取托盘ID，并写入工序状态。
- 公式：  
  **型号需求 = 抗金属标签 + 可读写 + 工位读写器 + PLC通讯**
- 计算：  
  金属托盘要求抗金属标签；B
