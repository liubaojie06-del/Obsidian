# 01 工业相机总览

> [!info] 核心结论
> 工业相机是机器视觉系统的“眼睛”，常见分类包括 **线阵相机、面阵相机、板级相机、工业相机客户端**。

## ① 分类/公式/规则

> [!note] 🟦 分类主线
> **线阵相机**：一行一行扫描，适合连续运动物体。  
> **面阵相机**：一次拍一整张图，适合静态或间歇运动检测。  
> **板级相机**：小型化、裸板结构，适合嵌入式设备。  
> **工业相机客户端**：用于相机配置、调试、采图和参数管理。

> [!warning] 🟧 通用选型公式
> **工业相机选型 = 检测对象 + 视野范围 + 精度要求 + 运动方式 + 接口类型 + 光源镜头 + 软件平台**

## ② 参考表

| 分类 | 核心用途 |
|---|---|
| **工业线阵相机** | 连续扫描、幅面检测 |
| **工业面阵相机** | 拍整幅图、定位测量 |
| **板级相机** | 嵌入式、小空间集成 |
| **工业相机客户端** | 参数设置、采图调试 |
| **镜头** | 决定视野和清晰度 |
| **光源** | 决定图像稳定性 |

## ③ 计算/选型示例

- 已知：  
  需要检测输送线上连续移动的布料表面缺陷。
- 公式：  
  **相机类型 = 物体运动方式 + 检测幅面 + 图像采集方式**
- 计算：  
  布料连续运动且幅面较宽，适合一行一行连续扫描。
- 结论：  
  优先考虑 **工业线阵相机**，分辨率、行频、镜头和光源需按厂家手册校核。

> [!tip] 记忆口诀
> 连续运动选线阵，单张拍照选面阵；空间很小选板级，调试参数看客户端。

# 02 工业线阵相机

> [!info] 核心结论
> 工业线阵相机一次只采集一行图像，通过物体运动拼接成完整图像，适合高速连续检测。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 线阵相机不是一次拍整张图，而是像扫描仪一样 **逐行采集**。  
> 物体移动一段，相机采一行，最终拼成完整图像。

> [!warning] 🟧 线阵适用公式
> **适合线阵 = 连续运动 + 长幅面 + 高分辨率 + 稳定速度 + 编码器同步**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **采集方式** | 一行一行扫描 |
| **典型对象** | 布料、纸张、薄膜、钢板、玻璃 |
| **关键参数** | 分辨率、行频、像元尺寸 |
| **同步方式** | 编码器触发常见 |
| **光源要求** | 线光源常见 |
| **难点** | 速度同步、光照均匀 |

## ③ 计算/选型示例

- 已知：  
  输送带速度为 **1000mm/s**，希望纵向分辨率为 **0.1mm/行**。
- 公式：  
  **所需行频 = 物体速度 ÷ 每行对应距离**
- 计算：  
  行频 = 1000 ÷ 0.1 = **10000行/s**
- 结论：  
  线阵相机行频应不低于 **10kHz**，并建议用编码器同步采集。

> [!tip] 记忆口诀
> 线阵靠运动成像，速度和行频必须对上。

# 03 工业面阵相机

> [!info] 核心结论
> 工业面阵相机一次采集一整幅图像，适合定位、测量、读码、缺陷检测和机器人引导。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 面阵相机类似普通相机，一次拍出二维图像。  
> 它适合产品静止拍照，或通过短曝光冻结运动画面。

> [!warning] 🟧 面阵适用公式
> **适合面阵 = 单个工件 + 固定视野 + 需要二维图像 + 拍照节拍可控**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **采集方式** | 一次拍整幅图 |
| **典型用途** | 定位、尺寸、缺陷、读码 |
| **关键参数** | 分辨率、帧率、快门类型 |
| **常见接口** | USB3.0、GigE、Camera Link等 |
| **光源要求** | 环形光、背光、同轴光等 |
| **难点** | 运动拖影、景深、反光 |

## ③ 计算/选型示例

- 已知：  
  视野宽度为 **200mm**，相机水平像素为 **2448px**。
- 公式：  
  **单像素尺寸 = 视野宽度 ÷ 水平像素数**
- 计算：  
  单像素尺寸 = 200 ÷ 2448 ≈ **0.082mm/px**
- 结论：  
  若要检测0.1mm级缺陷，像素占比偏少，需缩小视野、提高分辨率或优化光源。

> [!tip] 记忆口诀
> 面阵一次拍整图，定位测量最常用。

# 04 板级相机

> [!info] 核心结论
> 板级相机是去掉外壳或采用小型电路板结构的工业相机，适合空间受限和嵌入式设备。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 板级相机通常体积小、重量轻、便于嵌入设备内部。  
> 它更像工业相机的“核心板”，需要设备结构提供安装、防护和散热。

> [!warning] 🟧 板级相机适用公式
> **适合板级 = 空间受限 + 需要嵌入 + 重量敏感 + 可自行设计防护结构**

## ② 参考表

| 项目 | 选型关注点 |
|---|---|
| **结构** | 裸板或小型模块 |
| **优势** | 小体积、易嵌入 |
| **场景** | 小型设备、医疗、仪器、机器人 |
| **安装** | 需定制支架 |
| **防护** | 需外部结构保护 |
| **注意** | 防尘、防静电、散热 |

## ③ 计算/选型示例

- 已知：  
  设备内部安装空间很小，普通外壳工业相机放不下。
- 公式：  
  **可安装性 = 相机尺寸 + 镜头空间 + 接线空间 ≤ 设备内部空间**
- 计算：  
  普通工业相机体积较大，板级相机更容易嵌入。
- 结论：  
  可选择 **板级相机**，但需重点设计固定、防护、散热和线缆应力释放。

> [!tip] 记忆口诀
> 空间小、要嵌入，板级相机更合适。

# 05 工业相机客户端

> [!info] 核心结论
> 工业相机客户端用于相机发现、参数设置、图像预览、采图测试、固件维护和故障诊断。

## ① 分类/公式/规则

> [!note] 🟦 核心理解
> 客户端不是相机本体，而是相机调试软件。  
> 它常用于设置曝光、增益、触发模式、帧率、IP地址、图像格式等参数。

> [!warning] 🟧 调试流程公式
> **相机调试 = 连接相机 + 设置参数 + 预览图像 + 保存配置 + 联机测试**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **设备搜索** | 查找在线相机 |
| **参数设置** | 曝光、增益、帧率、触发 |
| **图像预览** | 查看实时图像 |
| **采图保存** | 保存样张用于分析 |
| **网络配置** | 设置IP、网卡、带宽 |
| **诊断维护** | 查看状态、升级固件 |

## ③ 计算/选型示例

- 已知：  
  相机已经接网线，但客户端搜索不到设备。
- 公式：  
  **搜索成功 = 网卡正确 + IP同网段 + 驱动正常 + 防火墙允许**
- 计算：  
  如果电脑网卡IP与相机不在同一网段，客户端可能无法发现相机。
- 结论：  
  应检查网卡、IP地址、防火墙、驱动和相机供电状态。

> [!tip] 记忆口诀
> 相机连不上，先查客户端；图像不稳定，先调曝光和触发。

# 06 四类快速对比

> [!info] 核心结论
> 线阵看连续，面阵看整图，板级看嵌入，客户端看调试。

## ① 分类/公式/规则

> [!note] 🟦 快速判断
> **连续材料检测**：优先线阵相机。  
> **单个产品检测**：优先面阵相机。  
> **小空间设备集成**：优先板级相机。  
> **参数配置和预览调试**：使用工业相机客户端。

> [!warning] 🟧 最终选择公式
> **推荐方案 = 检测方式 + 安装空间 + 图像要求 + 通讯接口 + 软件支持**

## ② 参考表

| 类别 | 最适合 |
|---|---|
| **线阵相机** | 连续扫描、卷材检测 |
| **面阵相机** | 定位、测量、读码 |
| **板级相机** | 嵌入式、小空间 |
| **相机客户端** | 配置、调试、采图 |
| **复杂视觉系统** | 相机 + 镜头 + 光源 + 软件 |
| **稳定检测核心** | 图像质量 + 算法匹配 |

## ③ 计算/选型示例

- 已知：  
  项目需要检测手机外壳尺寸，产品到位后停止拍照。
- 公式：  
  **相机类型 = 静止产品 + 二维尺寸检测**
- 计算：  
  产品停止后拍整幅图，面阵相机更适合。
- 结论：  
  优先选择 **工业面阵相机**，再根据视野、精度、镜头和光源进一步选型。

> [!tip] 选型口诀
> 卷料长图选线阵，工件拍照选面阵；设备嵌入选板级，参数调试找客户端。
