# 01 工业交换机总览

> [!info] 核心结论
> 工业交换机用于连接PLC、HMI、工控机、伺服驱动、远程IO、视觉相机、机器人、仪表和上位系统，是工业以太网通信的核心网络设备。

## ① 分类/公式/规则

> [!warning] 选型公式
> **工业交换机选型 = 端口数量 + 端口速率 + 通讯协议 + 管理功能 + 冗余要求 + 供电方式 + 防护等级 + 安装环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **端口数量** | 连接多少台设备 |
| **端口速率** | 100M、千兆、万兆等 |
| **端口类型** | 电口、光口、SFP、PoE |
| **管理方式** | 非管理型、管理型 |
| **冗余功能** | 环网、RSTP、MRP等 |
| **工业适应性** | 宽温、抗振、抗干扰 |
| **安装方式** | DIN导轨、壁挂、机架式 |

## ③ 计算/选型示例

- 已知：  
  一台设备需要连接PLC、HMI、工控机、4个远程IO站、2台相机。
- 公式：  
  **端口数 ≥ 设备数量 + 预留端口**
- 计算：  
  设备数量 = 1 + 1 + 1 + 4 + 2 = **9个端口**
- 结论：  
  建议选择 **12口或16口工业交换机**，不要刚好选8口。

> [!tip] 记忆口诀
> 交换机先数口，再看速率；现场要稳定，就选工业型。

# 02 底层原理

> [!info] 核心结论
> 工业交换机的底层作用，是根据MAC地址把以太网数据帧转发到正确端口，减少网络冲突，提高通信效率。

## ① 分类/公式/规则

> [!note] 工作逻辑
> 交换机会学习每个端口连接设备的MAC地址。  
> 当PLC、HMI、IO、相机之间通信时，交换机根据MAC地址表决定数据从哪个端口发出。

> [!warning] 转发链路
> **设备发送数据帧 → 交换机读取MAC地址 → 查表 → 转发到目标端口**

## ② 参考表

| 概念 | 说明 |
|---|---|
| **MAC地址** | 网络设备的硬件地址 |
| **交换表** | 记录MAC地址对应端口 |
| **数据帧** | 以太网传输的基本单位 |
| **广播** | 发给同一网络内所有设备 |
| **单播** | 发给指定设备 |
| **全双工** | 收发可同时进行 |
| **存储转发** | 接收完整帧后再转发 |

## ③ 计算/选型示例

- 已知：  
  PLC与HMI通信，PLC还要和远程IO通信。
- 公式：  
  **交换机 = 多设备数据转发中心**
- 计算：  
  交换机让不同设备的数据流分别转发，不需要所有数据都广播到每个端口。
- 结论：  
  工业交换机能提高网络通信效率和稳定性。

> [!tip] 记忆口诀
> 集线器是大家一起喊，交换机是知道该发给谁。

# 03 工业交换机 vs 普通交换机

> [!info] 核心结论
> 普通交换机适合办公环境，工业交换机适合车间、电柜、设备现场等高温、振动、强干扰、长期运行环境。

## ① 分类/公式/规则

> [!warning] 选择规则
> **办公网络 → 普通交换机**  
> **工业现场 / 控制柜 / 产线设备 → 工业交换机**

## ② 参考表

| 对比项 | 普通交换机 | 工业交换机 |
|---|---|---|
| 使用环境 | 办公室 | 工业现场 |
| 工作温度 | 较窄 | **宽温更常见** |
| 抗干扰 | 一般 | **更强** |
| 安装方式 | 桌面/机架 | **DIN导轨/壁挂/机架** |
| 供电方式 | 常见AC适配器 | **DC24V、冗余电源常见** |
| 可靠性 | 普通办公要求 | **长期连续运行** |
| 冗余功能 | 视型号 | **工业环网常见** |

## ③ 计算/选型示例

- 已知：  
  交换机要装在设备控制柜内，旁边有变频器和伺服驱动。
- 公式：  
  **强干扰工业环境 → 工业交换机**
- 计算：  
  普通交换机抗干扰和供电适应性可能不足。
- 结论：  
  应选择工业交换机，并做好接地、屏蔽和布线。

> [!tip] 记忆口诀
> 普通交换机管办公，工业交换机管现场。

# 04 非管理型交换机

> [!info] 核心结论
> 非管理型工业交换机即插即用，适合简单设备联网，但不支持复杂网络配置和诊断。

## ① 分类/公式/规则

> [!warning] 选择规则
> **简单连接、点数少、无需诊断 → 非管理型交换机可用**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **配置方式** | 基本无需配置 |
| **成本** | 相对低 |
| **维护** | 简单 |
| **VLAN** | 通常不支持 |
| **环网冗余** | 通常不支持或能力有限 |
| **网络诊断** | 较弱 |
| **适合场景** | 小型设备、简单PLC-HMI网络 |

## ③ 计算/选型示例

- 已知：  
  一台小设备只有PLC、HMI和一个工控机需要联网。
- 公式：  
  **少量设备 + 简单通信 = 非管理型交换机**
- 计算：  
  不需要VLAN、环网、端口镜像和远程诊断。
- 结论：  
  可选择 **5口或8口非管理型工业交换机**。

> [!tip] 记忆口诀
> 非管理型就是插上能用，简单便宜，但诊断能力弱。

# 05 管理型交换机

> [!info] 核心结论
> 管理型工业交换机支持VLAN、QoS、环网冗余、端口镜像、网络诊断和远程管理，适合复杂产线和关键网络。

## ① 分类/公式/规则

> [!warning] 选择规则
> **设备多、网络复杂、需要诊断/冗余/隔离 → 管理型交换机**

## ② 参考表

| 功能 | 作用 |
|---|---|
| **VLAN** | 网络隔离 |
| **QoS** | 优先保证关键数据 |
| **环网冗余** | 提高网络可靠性 |
| **端口镜像** | 抓包分析故障 |
| **IGMP Snooping** | 优化组播流量 |
| **SNMP** | 网络监控 |
| **Web/CLI管理** | 配置和维护 |

## ③ 计算/选型示例

- 已知：  
  产线有PLC、机器人、视觉相机、MES网络，且停机成本高。
- 公式：  
  **复杂产线网络 → 管理型交换机优先**
- 计算：  
  管理型交换机可实现网络隔离、诊断和冗余。
- 结论：  
  推荐使用管理型工业交换机，并规划VLAN和环网。

> [!tip] 记忆口诀
> 简单联网用非管理，复杂产线用管理。

# 06 端口数量

> [!info] 核心结论
> 端口数量必须覆盖当前设备，并预留后期调试、扩展和维护端口。

## ① 分类/公式/规则

> [!warning] 端口数量公式
> **推荐端口数 ≥ 当前设备数 + 备用端口 + 调试端口**

## ② 参考表

| 端口数量 | 常见应用 |
|---|---|
| **5口** | 小型单机设备 |
| **8口** | 常见小设备网络 |
| **12口** | 中等设备或工位 |
| **16口** | 多设备产线单元 |
| **24口** | 中大型控制柜 |
| **48口** | 大型网络机柜或集中网络 |

## ③ 计算/选型示例

- 已知：  
  当前需要10个网络设备，预留2个维护端口。
- 公式：  
  **端口数 ≥ 10 + 2**
- 计算：  
  端口数 ≥ **12口**
- 结论：  
  可选择12口或16口工业交换机。

> [!tip] 记忆口诀
> 端口别卡满，调试和扩展都要留口。

# 07 端口速率

> [!info] 核心结论
> 端口速率影响数据吞吐能力，PLC和IO常用100M即可，视觉相机、数据采集和上位机常需要千兆。

## ① 分类/公式/规则

> [!warning] 带宽选择规则
> **普通控制网络 → 100M常见**  
> **相机/工控机/数据上传 → 千兆优先**

## ② 参考表

| 速率 | 适合场景 |
|---|---|
| **10/100M** | PLC、HMI、远程IO、仪表 |
| **千兆电口** | 工控机、视觉相机、服务器 |
| **千兆光口** | 远距离骨干链路 |
| **万兆端口** | 大量图像数据、核心汇聚 |
| **自适应端口** | 自动匹配10/100/1000M |
| **全双工** | 工业网络常用 |

## ③ 计算/选型示例

- 已知：  
  工业相机单路数据约60MB/s。
- 公式：  
  **网络带宽需求 = 数据量 × 8**
- 计算：  
  60MB/s × 8 = **480Mbps**
- 结论：  
  单个100M端口不够，应使用千兆端口。

> [!tip] 记忆口诀
> 控制数据小，百兆能跑；图像数据大，千兆起步。

# 08 电口、光口与SFP

> [!info] 核心结论
> 电口适合短距离普通网线连接，光口适合远距离、强干扰和跨区域网络连接。

## ① 分类/公式/规则

> [!warning] 端口选择规则
> **短距离柜内/设备内 → 电口**  
> **远距离/强干扰/跨车间 → 光口或SFP**

## ② 参考表

| 端口类型 | 特点 |
|---|---|
| **RJ45电口** | 常用网线连接，方便 |
| **光纤口** | 距离远，抗电磁干扰 |
| **SFP槽位** | 可插不同光模块 |
| **单模光纤** | 距离更远 |
| **多模光纤** | 中短距离常见 |
| **光电混合交换机** | 电口连接设备，光口做上联 |

## ③ 计算/选型示例

- 已知：  
  两个车间控制柜相距300米，中间有强电设备。
- 公式：  
  **长距离 + 强干扰 → 光纤链路**
- 计算：  
  普通网线距离和抗干扰能力不适合。
- 结论：  
  应选择带光口或SFP的工业交换机，用光纤连接。

> [!tip] 记忆口诀
> 近距离用网线，远距离用光纤；干扰大，也用光纤。

# 09 PoE工业交换机

> [!info] 核心结论
> PoE工业交换机可以通过网线同时传输数据和供电，常用于工业相机、无线AP、IP摄像头和部分传感器。

## ① 分类/公式/规则

> [!warning] PoE供电公式
> **PoE总功率 ≥ 所有PoE设备功率之和 + 余量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **PoE** | 网线供电 |
| **PoE+** | 单端口供电能力更高 |
| **PoE++** | 更高功率设备使用 |
| **总功率预算** | 所有PoE端口可用总功率 |
| **工业相机** | 常见PoE供电设备 |
| **无线AP** | 常见PoE供电设备 |
| **线缆质量** | 影响供电压降和稳定性 |

## ③ 计算/选型示例

- 已知：  
  4台PoE相机，每台功率12W。
- 公式：  
  **总功率 = 4 × 12**
- 计算：  
  总功率 = **48W**
- 结论：  
  PoE交换机总功率应大于48W，并留余量，例如选择60W或更高功率预算。

> [!tip] 记忆口诀
> PoE不只看几个口，还要看总功率够不够。

# 10 工业环网冗余

> [!info] 核心结论
> 工业环网冗余用于提高网络可靠性，当一段网线断开时，网络可从另一方向恢复通信。

## ① 分类/公式/规则

> [!warning] 环网规则
> **环网可用 = 交换机支持环网协议 + 拓扑正确 + 参数一致**

## ② 参考表

| 冗余方式 | 说明 |
|---|---|
| **RSTP** | 通用生成树协议 |
| **MRP** | PROFINET系统常见环网协议 |
| **ERPS** | 以太网环网保护 |
| **厂商私有环网** | 恢复速度快，但需同品牌支持 |
| **双上联** | 提高核心链路可靠性 |
| **双电源** | 提高供电可靠性 |

## ③ 计算/选型示例

- 已知：  
  一条产线有多个控制柜，任意一段网线断开不能导致整线停机。
- 公式：  
  **高可靠网络 → 环网冗余**
- 计算：  
  线型拓扑中间断线会导致后面设备掉线，环网可绕路通信。
- 结论：  
  应选择支持环网协议的管理型工业交换机。

> [!tip] 记忆口诀
> 线型网络一断一片，环网网络断线可绕行。

# 11 VLAN网络隔离

> [!info] 核心结论
> VLAN用于把同一台交换机上的不同设备划分到不同逻辑网络，减少干扰并提高安全性。

## ① 分类/公式/规则

> [!warning] VLAN规则
> **不同VLAN默认不能直接互通，需要三层路由或网关**

## ② 参考表

| VLAN用途 | 示例 |
|---|---|
| **控制网络隔离** | PLC、远程IO独立 |
| **视觉网络隔离** | 相机和工控机独立 |
| **办公网络隔离** | MES、办公网分开 |
| **调试网络隔离** | 维护端口单独配置 |
| **广播控制** | 减少无关广播流量 |
| **安全管理** | 限制非授权访问 |

## ③ 计算/选型示例

- 已知：  
  相机数据量大，影响PLC通信稳定。
- 公式：  
  **大流量视觉网络 → VLAN隔离**
- 计算：  
  把相机和工控机放到视觉VLAN，把PLC和IO放到控制VLAN。
- 结论：  
  使用管理型交换机配置VLAN，可降低相机流量对控制网络的影响。

> [!tip] 记忆口诀
> 视觉流量大，控制要稳定；VLAN能把网络分开。

# 12 QoS优先级

> [!info] 核心结论
> QoS用于给关键通信数据更高优先级，避免普通数据占用带宽影响控制通信。

## ① 分类/公式/规则

> [!warning] QoS规则
> **关键控制数据优先，普通数据后传**

## ② 参考表

| 数据类型 | 优先级建议 |
|---|---|
| **PLC与远程IO** | 高 |
| **运动控制数据** | 高 |
| **报警和联锁数据** | 高 |
| **HMI显示数据** | 中 |
| **相机图片传输** | 视场景配置 |
| **MES报表数据** | 较低 |
| **普通文件传输** | 较低 |

## ③ 计算/选型示例

- 已知：  
  网络中同时有PLC控制数据和大量图片上传。
- 公式：  
  **控制优先 → QoS配置**
- 计算：  
  图片上传可能占用带宽，导致控制数据延迟。
- 结论：  
  可通过QoS提高控制数据优先级。

> [!tip] 记忆口诀
> 网络拥堵时，关键控制数据要先走。

# 13 IGMP Snooping与组播

> [!info] 核心结论
> IGMP Snooping用于管理组播流量，常见于工业协议、相机、多媒体和某些设备发现机制中。

## ① 分类/公式/规则

> [!warning] 组播管理规则
> **有组播流量 → 管理型交换机应评估IGMP Snooping**

## ② 参考表

| 概念 | 说明 |
|---|---|
| **单播** | 一对一通信 |
| **广播** | 发给所有设备 |
| **组播** | 发给一组设备 |
| **IGMP Snooping** | 让组播只发给需要的端口 |
| **广播风暴** | 大量广播导致网络拥塞 |
| **组播泛洪** | 组播被发到不需要的端口 |

## ③ 计算/选型示例

- 已知：  
  接入多个网络设备后，网络负载突然升高。
- 公式：  
  **异常流量 = 广播/组播过多**
- 计算：  
  如果交换机不管理组播，组播可能泛洪到多个端口。
- 结论：  
  应检查是否需要启用IGMP Snooping或优化网络拓扑。

> [!tip] 记忆口诀
> 广播发给所有人，组播发给一群人；管理不好就会堵网。

# 14 工业协议兼容性

> [!info] 核心结论
> 工业交换机通常不直接“识别工艺逻辑”，但必须满足PROFINET、EtherNet/IP、Modbus TCP、工业相机等网络的实时性和稳定性要求。

## ① 分类/公式/规则

> [!warning] 协议选型公式
> **交换机适配 = 协议要求 + 实时性 + 认证/兼容性 + 网络拓扑**

## ② 参考表

| 协议/应用 | 交换机关注点 |
|---|---|
| **PROFINET** | MRP、QoS、诊断、认证 |
| **EtherNet/IP** | IGMP、组播、QoS |
| **Modbus TCP** | 通用TCP通信，要求相对低 |
| **EtherCAT** | 一般不通过普通交换机随意转发 |
| **工业相机** | 千兆、低丢包、大带宽 |
| **OPC UA/MES** | 稳定上联、网络隔离 |
| **远程IO** | 周期通信稳定性 |

## ③ 计算/选型示例

- 已知：  
  现场是西门子PROFINET网络，并要求环网冗余。
- 公式：  
  **PROFINET环网 → MRP支持**
- 计算：  
  普通非管理交换机可能不支持MRP环网。
- 结论：  
  应选择支持PROFINET/MRP的管理型工业交换机。

> [!tip] 记忆口诀
> 交换机不控制工艺，但会影响工业协议跑得稳不稳。

# 15 供电与冗余电源

> [!info] 核心结论
> 工业交换机常用DC24V供电，关键网络建议使用双电源冗余和电源报警输出。

## ① 分类/公式/规则

> [!warning] 供电公式
> **电源容量 ≥ 交换机功耗 + PoE功耗 + 余量**

## ② 参考表

| 供电项 | 说明 |
|---|---|
| **DC24V** | 工业控制柜常见 |
| **宽压输入** | 适应电源波动 |
| **双电源输入** | 提高可靠性 |
| **电源报警继电器** | 反馈电源故障 |
| **PoE供电** | 给相机/AP供电 |
| **接地端子** | 提高抗干扰能力 |
| **浪涌保护** | 雷击、强干扰场景考虑 |

## ③ 计算/选型示例

- 已知：  
  交换机本体功耗10W，PoE设备总功率48W。
- 公式：  
  **总功率 = 10 + 48**
- 计算：  
  总功率 = **58W**
- 结论：  
  电源应大于58W并留余量，建议选择更高功率DC电源。

> [!tip] 记忆口诀
> 普通交换机看网口，PoE交换机还要算功率。

# 16 安装与布线

> [!info] 核心结论
> 工业交换机安装布线要注意电柜散热、DIN导轨固定、接地、网线等级、强弱电分离和端口标识。

## ① 分类/公式/规则

> [!warning] 稳定布线公式
> **网络稳定 = 合格线缆 + 正确接地 + 强弱电分离 + 接头可靠 + 标识清楚**

## ② 参考表

| 项目 | 注意 |
|---|---|
| **DIN导轨安装** | 控制柜常用 |
| **网线等级** | 百兆/千兆需匹配线缆 |
| **屏蔽网线** | 强干扰环境更常用 |
| **光纤保护** | 防折弯、防污染 |
| **接地** | 屏蔽和设备接地要规范 |
| **线缆分离** | 网线远离动力线 |
| **端口标签** | 方便维护排查 |

## ③ 计算/选型示例

- 已知：  
  网络通信偶发中断，网线与变频器动力线长距离并行。
- 公式：  
  **通信异常 = 干扰 + 接地/屏蔽不良 + 线缆质量差**
- 计算：  
  动力线产生的干扰可能耦合到网线。
- 结论：  
  应分开布线，使用工业网线，并检查屏蔽接地。

> [!tip] 记忆口诀
> 网络故障不一定是交换机，线缆和接地经常是根因。

# 17 常见故障排查

> [!info] 核心结论
> 工业交换机故障排查按 **电源、端口灯、网线、IP地址、协议配置、环网、带宽、干扰** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **网络异常 = 供电异常 + 端口故障 + 网线问题 + IP冲突 + 配置错误 + 带宽拥塞 + 环网风暴 + 干扰**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **交换机不亮** | 电源无电、接线错误、保险断 |
| **端口灯不亮** | 网线断、设备未上电、端口损坏 |
| **设备Ping不通** | IP错误、网段不一致、防火墙 |
| **通信时断时续** | 网线接触不良、干扰、电源波动 |
| **整网很慢** | 广播风暴、环路、带宽占满 |
| **相机掉线** | 带宽不足、网线差、PoE不足 |
| **环网异常** | 协议配置错误、环路未阻塞 |
| **远程IO掉站** | 周期过短、网络抖动、交换机不适配 |

## ③ 计算/选型示例

- 已知：  
  接入一根新网线后，全网突然变慢。
- 公式：  
  **全网变慢 = 可能形成网络环路**
- 计算：  
  如果两台交换机之间重复连接多根线，且没有正确启用环网/生成树，可能形成广播风暴。
- 结论：  
  应立即检查拓扑，拔除可疑回路，并确认环网协议配置。

> [!tip] 记忆口诀
> 端口不亮查线，Ping不通查IP，全网变慢先查环路。

# 18 工业交换机快速选型表

> [!info] 核心结论
> 工业交换机选型要从网络拓扑出发，统一确认端口、速率、光电口、PoE、管理功能、冗余、协议和环境。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 统计需要联网的设备数量  
> ② 确认端口数量和预留端口  
> ③ 确认百兆、千兆或更高速率  
> ④ 确认电口、光口、SFP需求  
> ⑤ 确认是否需要PoE供电  
> ⑥ 确认非管理型还是管理型  
> ⑦ 确认是否需要VLAN、QoS、IGMP  
> ⑧ 确认是否需要环网冗余和双电源  
> ⑨ 确认工业协议兼容性和网络拓扑  
> ⑩ 确认安装方式、工作温度、防护和抗干扰

> [!warning] 最终选型公式
> **推荐工业交换机 = 端口数 + 速率 + 光/电/PoE端口 + 管理功能 + 冗余协议 + 工业环境等级**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单PLC-HMI网络 | 非管理型交换机 |
| 多设备产线 | 管理型交换机 |
| 视觉相机 | 千兆端口、大带宽 |
| 远距离联网 | 光口/SFP |
| 相机/AP供电 | PoE交换机 |
| 高可靠产线 | 环网冗余、双电源 |
| 网络隔离 | VLAN |
| 网络诊断 | SNMP、端口镜像、Web管理 |

## ③ 计算/选型示例

- 已知：  
  项目有PLC、HMI、工控机、6个远程IO、2台千兆相机，需要接入MES，并希望产线断一根网线不全线掉站。
- 公式：  
  **端口需求 = 1 + 1 + 1 + 6 + 2 + 1 = 12口**  
  **相机需求 = 千兆端口**  
  **高可靠需求 = 环网冗余**
- 计算：  
  至少12口，建议16口；相机端口用千兆；产线网络用管理型环网。
- 结论：  
  推荐选择 **16口管理型工业交换机，含千兆端口，支持环网冗余、VLAN、QoS和双电源输入**。

> [!tip] 选型口诀
> 先数设备定端口，再看数据定速率；远距离用光纤，相机多用千兆；关键产线要管理，可靠网络要环网。
