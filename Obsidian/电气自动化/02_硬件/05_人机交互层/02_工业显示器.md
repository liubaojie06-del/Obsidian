# 01 工业显示器总览

> [!info] 核心结论
> 工业显示器用于工业现场的 **画面显示、设备监控、工控机显示、视觉检测结果显示、产线看板、操作界面扩展**，相比普通显示器更强调稳定性、防护、接口、抗干扰和长期运行。

## ① 分类/公式/规则

> [!warning] 选型公式
> **工业显示器选型 = 尺寸 + 分辨率 + 亮度 + 接口 + 触摸方式 + 防护等级 + 安装方式 + 工作环境**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **显示尺寸** | 7寸、10寸、12寸、15寸、17寸、19寸、21寸以上 |
| **分辨率** | 决定画面细腻度和显示空间 |
| **亮度** | 决定强光环境下是否看得清 |
| **接口** | HDMI、VGA、DVI、DP、USB、Type-C等 |
| **触摸功能** | 可选电阻触摸、电容触摸或无触摸 |
| **防护等级** | 前面板常见IP65，部分整机高防护 |
| **应用场景** | 工控机显示、视觉检测、设备监控、产线看板 |

## ③ 计算/选型示例

- 已知：  
  现场需要连接工业计算机，显示视觉检测画面和报警状态。
- 公式：  
  **工控机输出 + 现场显示 → 工业显示器**
- 计算：  
  普通办公显示器在防尘、防振、长期运行和安装方式上可能不足。
- 结论：  
  推荐选择 **工业显示器**，并确认接口与工控机输出匹配。

> [!tip] 记忆口诀
> 工控机负责算，工业显示器负责把结果稳定显示出来。

# 02 底层原理

> [!info] 核心结论
> 工业显示器本质上是把工控机、控制器或视频源输出的图像信号转换成可视画面，部分型号还把触摸信号回传给主机。

## ① 分类/公式/规则

> [!note] 工作链路
> 工控机或控制器输出视频信号，显示器接收并转换为屏幕画面；  
> 如果带触摸功能，触摸屏会通过USB、串口或其他接口把触摸坐标传回主机。

> [!warning] 信号链路
> **主机视频输出 → 显示器接口 → 显示驱动板 → 液晶面板显示**  
> **手指触摸 → 触摸控制板 → USB/串口回传 → 主机识别操作**

## ② 参考表

| 部件 | 作用 |
|---|---|
| **液晶面板** | 显示图像 |
| **背光模组** | 提供亮度 |
| **驱动板** | 处理HDMI/VGA/DP等视频信号 |
| **触摸屏** | 接收触摸操作 |
| **触摸控制板** | 转换触摸坐标 |
| **外壳结构** | 防护、安装和散热 |
| **电源模块** | 给显示器供电 |

## ③ 计算/选型示例

- 已知：  
  工控机HDMI连接显示器后有画面，但触摸无效。
- 公式：  
  **触摸有效 = 视频线 + 触摸通讯线 + 驱动识别**
- 计算：  
  HDMI只传视频，不一定传触摸数据；触摸通常还需要USB线连接主机。
- 结论：  
  应检查USB触摸线、驱动和系统设备识别状态。

> [!tip] 记忆口诀
> 视频线管显示，USB线管触摸；有画面不代表能触摸。

# 03 工业显示器 vs 普通显示器

> [!info] 核心结论
> 普通显示器适合办公环境，工业显示器适合设备现场长期运行，重点差异在防护、接口、安装、抗干扰和稳定性。

## ① 分类/公式/规则

> [!warning] 选择规则
> **办公显示 → 普通显示器**  
> **工业现场 / 控制柜 / 设备面板 → 工业显示器**

## ② 参考表

| 对比项 | 普通显示器 | 工业显示器 |
|---|---|---|
| 使用环境 | 办公环境 | 工业现场 |
| 安装方式 | 桌面为主 | 嵌入式、壁挂、VESA、机架 |
| 防护能力 | 较弱 | **更强** |
| 抗干扰 | 一般 | **更适合现场** |
| 长期运行 | 普通使用 | **适合7×24小时运行** |
| 接口 | 办公接口为主 | **工业接口更丰富** |
| 外壳结构 | 塑料较多 | 金属外壳常见 |

## ③ 计算/选型示例

- 已知：  
  显示器要安装在设备操作面板上，现场有粉尘和振动。
- 公式：  
  **现场安装 + 粉尘振动 → 工业显示器**
- 计算：  
  普通显示器不方便嵌入安装，也不适合长期暴露在现场环境中。
- 结论：  
  应选择嵌入式工业显示器，并确认前面板防护等级。

> [!tip] 记忆口诀
> 普通屏给办公室，工业屏给设备现场。

# 04 工业显示器 vs 工业HMI

> [!info] 核心结论
> 工业显示器主要负责显示，工业HMI不仅显示，还能直接与PLC通讯、做画面工程、报警和配方。

## ① 分类/公式/规则

> [!warning] 区别规则
> **工业显示器 = 显示终端**  
> **工业HMI = 显示 + 触摸 + PLC通讯 + 画面工程 + 数据功能**

## ② 参考表

| 对比项 | 工业显示器 | 工业HMI |
|---|---|---|
| 是否独立运行画面 | 通常不能 | 可以 |
| 是否需要主机 | 通常需要工控机 | 可直接连接PLC |
| PLC通讯 | 依赖主机软件 | **内置通讯驱动** |
| 报警/配方 | 依赖上位机软件 | **常见内置功能** |
| 适合场景 | 工控机显示、视觉显示 | 设备操作界面 |
| 本质 | 显示设备 | 人机界面控制终端 |

## ③ 计算/选型示例

- 已知：  
  现场只有PLC，没有工控机，但需要做操作画面。
- 公式：  
  **无工控机 + 直接连PLC → 工业HMI优先**
- 计算：  
  工业显示器本身不能直接做PLC画面逻辑。
- 结论：  
  应选择工业HMI，而不是单纯工业显示器。

> [!tip] 记忆口诀
> 有工控机用显示器，直接连PLC用HMI。

# 05 屏幕尺寸

> [!info] 核心结论
> 屏幕尺寸决定可视面积和操作空间，画面越复杂、距离越远、参数越多，尺寸越不能太小。

## ① 分类/公式/规则

> [!warning] 尺寸选择规则
> **简单状态显示 → 小尺寸**  
> **复杂监控 / 视觉图像 / 趋势曲线 → 大尺寸**

## ② 参考表

| 尺寸 | 常见应用 |
|---|---|
| **7寸** | 小型设备状态显示 |
| **10.1寸** | 单机设备、简单上位机 |
| **12.1寸** | 参数和画面较多设备 |
| **15寸** | 工控机常用显示尺寸 |
| **17/19寸** | 视觉检测、产线监控 |
| **21寸以上** | 看板、大画面、复杂监控 |

## ③ 计算/选型示例

- 已知：  
  需要显示视觉图像、OK/NG结果、报警列表和生产统计。
- 公式：  
  **显示内容多 → 大尺寸屏幕**
- 计算：  
  小屏幕会导致图像缩小、文字密集、操作不方便。
- 结论：  
  可选择 **15寸或更大工业显示器**。

> [!tip] 记忆口诀
> 只看状态小屏够，看图像和数据就要大屏。

# 06 分辨率

> [!info] 核心结论
> 分辨率决定显示细节和画面空间，工业视觉、趋势曲线和多窗口监控通常需要更高分辨率。

## ① 分类/公式/规则

> [!warning] 分辨率规则
> **分辨率越高，同尺寸下可显示的信息越多，但文字和按钮可能变小**

## ② 参考表

| 分辨率 | 说明 |
|---|---|
| **800×480** | 小屏常见，适合简单显示 |
| **1024×600** | 7~10寸常见 |
| **1024×768** | 传统4:3工业屏常见 |
| **1280×800** | 宽屏设备常见 |
| **1920×1080** | 高清显示，适合视觉和看板 |
| **更高分辨率** | 多窗口、精细图像、复杂监控 |

## ③ 计算/选型示例

- 已知：  
  工控机输出分辨率为1920×1080，显示器只支持1024×768。
- 公式：  
  **显示器分辨率必须支持主机输出**
- 计算：  
  分辨率不匹配可能导致无显示、拉伸或显示不完整。
- 结论：  
  应选择支持1920×1080的工业显示器，或调整主机输出分辨率。

> [!tip] 记忆口诀
> 尺寸决定有多大，分辨率决定有多细。

# 07 亮度与可视角度

> [!info] 核心结论
> 亮度决定强光环境下能否看清，可视角度决定从侧面观察时画面是否清楚。

## ① 分类/公式/规则

> [!warning] 亮度选择规则
> **室内普通环境 → 常规亮度**  
> **强光、户外、远距离观察 → 高亮屏**

## ② 参考表

| 参数 | 影响 |
|---|---|
| **亮度cd/m²** | 越高越适合强光环境 |
| **对比度** | 影响文字和图像清晰度 |
| **可视角度** | 影响侧面观看效果 |
| **防眩光玻璃** | 减少反光 |
| **自动亮度** | 部分场景可调节 |
| **背光寿命** | 影响长期显示亮度衰减 |

## ③ 计算/选型示例

- 已知：  
  显示器安装在靠窗设备旁，白天反光严重。
- 公式：  
  **强光反射 → 高亮度 + 防眩光**
- 计算：  
  普通低亮度屏幕可能看不清画面。
- 结论：  
  可选择高亮工业显示器，并使用防眩光面板或调整安装角度。

> [!tip] 记忆口诀
> 光线强就要高亮，反光重就要防眩。

# 08 触摸方式

> [!info] 核心结论
> 工业显示器可选无触摸、电阻触摸或电容触摸；触摸方式要根据手套、油污、水汽和操作习惯选择。

## ① 分类/公式/规则

> [!warning] 触摸选择规则
> **戴手套、油污环境 → 电阻触摸可评估**  
> **操作体验好、多点触控 → 电容触摸优先**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **无触摸显示器** | 只显示，需键鼠或外部操作 |
| **电阻触摸** | 可用手套、触笔，耐油污较好 |
| **电容触摸** | 灵敏、体验好，常见多点触控 |
| **红外触摸** | 大尺寸常见 |
| **防水触摸** | 潮湿场景需确认 |
| **手套触摸** | 需确认控制板和面板支持 |

## ③ 计算/选型示例

- 已知：  
  操作员长期戴厚手套操作设备。
- 公式：  
  **戴手套操作 → 触摸方式需匹配**
- 计算：  
  普通电容屏可能识别不稳定。
- 结论：  
  可选择支持手套操作的电容屏，或选择电阻触摸工业显示器。

> [!tip] 记忆口诀
> 手套油污看电阻，体验流畅看电容。

# 09 视频接口

> [!info] 核心结论
> 工业显示器必须与工控机、控制器或视频源的输出接口匹配，常见接口有HDMI、VGA、DVI、DP和Type-C。

## ① 分类/公式/规则

> [!warning] 接口匹配公式
> **显示成功 = 视频接口匹配 + 分辨率支持 + 刷新率支持 + 线缆可靠**

## ② 参考表

| 接口 | 特点 |
|---|---|
| **HDMI** | 常见高清数字接口 |
| **VGA** | 老设备常见模拟接口 |
| **DVI** | 工业设备中仍常见 |
| **DisplayPort** | 高分辨率、高刷新场景 |
| **Type-C** | 部分新型设备支持 |
| **BNC/AV视频** | 特殊监控或旧设备场景 |

## ③ 计算/选型示例

- 已知：  
  工控机只有DP接口，显示器只有VGA接口。
- 公式：  
  **接口不一致 → 需转换器或更换设备**
- 计算：  
  DP是数字接口，VGA是模拟接口，普通线缆不能直接等效转换。
- 结论：  
  应选择带DP接口的工业显示器，或使用可靠的主动转换器。

> [!tip] 记忆口诀
> 先看主机出什么口，再看显示器进什么口。

# 10 触摸接口

> [!info] 核心结论
> 带触摸的工业显示器通常需要额外触摸通讯接口，最常见是USB，部分旧设备使用RS232。

## ① 分类/公式/规则

> [!warning] 触摸连接公式
> **触摸可用 = 触摸线连接 + 系统识别 + 驱动正常 + 校准正确**

## ② 参考表

| 接口 | 说明 |
|---|---|
| **USB触摸** | 最常见，接入方便 |
| **RS232触摸** | 老设备或特殊工业屏常见 |
| **I2C/内部接口** | 嵌入式系统内部使用 |
| **免驱触摸** | 系统直接识别HID设备 |
| **需驱动触摸** | 需要安装厂家驱动 |
| **触摸校准** | 电阻屏或特殊屏幕可能需要 |

## ③ 计算/选型示例

- 已知：  
  显示器画面正常，但点按位置偏移。
- 公式：  
  **触摸偏移 = 校准错误 / 分辨率映射错误 / 多屏设置错误**
- 计算：  
  多显示器环境下，触摸可能映射到错误屏幕。
- 结论：  
  应重新校准触摸，并检查系统显示器编号和触摸映射。

> [!tip] 记忆口诀
> 触摸不准先校准，多屏系统查映射。

# 11 安装方式

> [!info] 核心结论
> 工业显示器常见安装方式有嵌入式、壁挂式、VESA支架、机架式、桌面式和悬臂式。

## ① 分类/公式/规则

> [!warning] 安装选择公式
> **安装方式 = 使用位置 + 操作习惯 + 防护需求 + 维护空间 + 结构强度**

## ② 参考表

| 安装方式 | 适合场景 |
|---|---|
| **嵌入式安装** | 电柜门、操作面板 |
| **VESA安装** | 支架、悬臂、壁挂 |
| **机架式安装** | 机柜、服务器柜 |
| **桌面式安装** | 调试台、实验台 |
| **悬臂式安装** | 需要调整角度和位置 |
| **面板安装** | 设备外壳集成 |

## ③ 计算/选型示例

- 已知：  
  显示器要安装在控制柜门上。
- 公式：  
  **柜门安装 → 嵌入式面板显示器**
- 计算：  
  需要按厂家开孔尺寸加工柜门，并保证密封。
- 结论：  
  应选择嵌入式工业显示器，并确认开孔尺寸和固定夹具。

> [!tip] 记忆口诀
> 装柜门看开孔，装支架看VESA，装机柜看机架。

# 12 防护等级

> [!info] 核心结论
> 工业显示器防护等级要根据现场粉尘、水汽、油污和清洗方式选择，前面板防护和整机防护要分清。

## ① 分类/公式/规则

> [!warning] 防护规则
> **前面板IP65 ≠ 整机IP65**  
> **嵌入安装后，柜内侧仍需靠控制柜防护**

## ② 参考表

| 防护需求 | 说明 |
|---|---|
| **IP54** | 基础防尘防溅 |
| **IP65前面板** | 面板侧防尘、防喷水 |
| **IP67整机** | 更高现场防护 |
| **防油污** | 需看面板和密封材质 |
| **防腐蚀** | 化工、食品场景需关注 |
| **防清洗** | 食品、医药设备需确认 |

## ③ 计算/选型示例

- 已知：  
  显示器正面会被水枪冲洗。
- 公式：  
  **水洗环境 → 高防护前面板或整机防护**
- 计算：  
  如果只支持普通防护，水可能从面板缝隙进入。
- 结论：  
  应选择满足现场清洗要求的高防护工业显示器，并确认密封安装。

> [!tip] 记忆口诀
> 前面防水看面板，后面防水看整机或电柜。

# 13 供电方式

> [!info] 核心结论
> 工业显示器常见供电为DC12V、DC24V或AC220V，工业设备中DC24V更常见。

## ① 分类/公式/规则

> [!warning] 电源匹配公式
> **供电可用 = 电压匹配 + 功率足够 + 接地可靠 + 抗干扰良好**

## ② 参考表

| 供电方式 | 特点 |
|---|---|
| **DC12V** | 小型显示器常见 |
| **DC24V** | 工业控制柜常见 |
| **AC220V** | 大屏或独立显示器常见 |
| **宽压输入** | 适应现场电源波动 |
| **电源适配器** | 外置供电，注意固定 |
| **端子供电** | 工业柜内接线常见 |

## ③ 计算/选型示例

- 已知：  
  显示器额定功率30W，使用DC24V供电。
- 公式：  
  **电流 = 功率 ÷ 电压**
- 计算：  
  电流 = 30 ÷ 24 ≈ **1.25A**
- 结论：  
  24V电源至少要提供1.25A并留余量。

> [!tip] 记忆口诀
> 电压要匹配，电流要够用，电源要稳定。

# 14 环境适应性

> [!info] 核心结论
> 工业显示器要适应温度、湿度、振动、粉尘、电磁干扰和长期运行条件。

## ① 分类/公式/规则

> [!warning] 环境选型公式
> **环境适应性 = 工作温度 + 湿度 + 防护等级 + 抗振 + EMC + 散热能力**

## ② 参考表

| 环境因素 | 影响 |
|---|---|
| **高温** | 背光寿命下降，电子元件故障风险增加 |
| **低温** | 液晶响应变慢，启动困难 |
| **湿度** | 结露、腐蚀、触摸异常 |
| **振动** | 接线松动、结构疲劳 |
| **粉尘** | 散热差、接口污染 |
| **电磁干扰** | 画面闪烁、触摸误动作 |

## ③ 计算/选型示例

- 已知：  
  显示器安装在变频器附近，偶发画面闪烁。
- 公式：  
  **画面闪烁 = 电源干扰 / 信号线干扰 / 接地不良**
- 计算：  
  视频线和动力线并行过长，可能引入干扰。
- 结论：  
  应分开布线、使用屏蔽线，并检查接地和电源质量。

> [!tip] 记忆口诀
> 工业屏怕热、怕潮、怕干扰，安装布线要规范。

# 16 常见故障排查

> [!info] 核心结论
> 工业显示器故障排查按 **电源、视频线、输入源、分辨率、触摸线、驱动、主机状态** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **显示器异常 = 电源异常 + 信号线异常 + 输入源错误 + 分辨率不支持 + 触摸驱动异常 + 主机故障**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **屏幕不亮** | 无电源、电源适配器故障、背光故障 |
| **无信号** | 视频线松动、输入源错误、主机无输出 |
| **画面闪烁** | 干扰、线缆差、分辨率或刷新率异常 |
| **显示不完整** | 分辨率比例不匹配 |
| **触摸无效** | USB未接、驱动未装、触摸屏故障 |
| **触摸偏移** | 校准错误、多屏映射错误 |
| **画面发暗** | 亮度设置低、背光老化 |
| **偶发黑屏** | 电源波动、线缆松动、主机休眠 |

## ③ 计算/选型示例

- 已知：  
  显示器提示“无信号”。
- 公式：  
  **无信号 = 主机无输出 / 线缆异常 / 输入源错误**
- 计算：  
  先检查工控机是否开机，再检查HDMI/VGA线和显示器输入源设置。
- 结论：  
  不要直接判断显示器坏，先确认视频源和线缆。

> [!tip] 记忆口诀
> 屏不亮查电源，无信号查线和输入源，触摸不动查USB。

# 17 工业显示器快速选型表

> [!info] 核心结论
> 工业显示器选型要先确认使用场景和主机接口，再确定尺寸、分辨率、触摸、防护、供电和安装方式。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 确认用途：工控机显示、视觉显示、看板、触摸操作  
> ② 确认尺寸：7寸、10寸、15寸、19寸等  
> ③ 确认分辨率：是否匹配主机输出  
> ④ 确认亮度和可视角度  
> ⑤ 确认是否需要触摸  
> ⑥ 确认触摸方式：电阻/电容/无触摸  
> ⑦ 确认视频接口：HDMI/VGA/DVI/DP  
> ⑧ 确认安装方式：嵌入/VESA/机架/悬臂  
> ⑨ 确认防护等级和环境温度  
> ⑩ 确认供电方式、线缆长度和维护空间

> [!warning] 最终选型公式
> **推荐工业显示器 = 尺寸 + 分辨率 + 亮度 + 接口 + 触摸方式 + 防护等级 + 安装方式 + 供电方式**

## ② 参考表

| 需求 | 推荐关注点 |
|---|---|
| 简单设备显示 | 7寸~10寸 |
| 工控机上位机 | 12寸~15寸 |
| 视觉检测 | 15寸以上、高分辨率 |
| 产线看板 | 大尺寸、高亮度 |
| 戴手套操作 | 电阻屏或手套电容屏 |
| 潮湿粉尘环境 | 高防护前面板 |
| 柜门安装 | 嵌入式面板安装 |
| 支架安装 | VESA标准孔位 |

## ③ 计算/选型示例

- 已知：  
  工控机有HDMI输出，现场需要15寸触摸显示器，安装在电柜门上，使用DC24V供电。
- 公式：  
  **显示器需求 = 15寸 + HDMI + 触摸 + 嵌入安装 + DC24V**
- 计算：  
  还需确认USB触摸接口、开孔尺寸、防护等级和供电电流。
- 结论：  
  推荐选择 **15寸嵌入式工业触摸显示器，支持HDMI视频输入、USB触摸、DC24V供电和前面板IP65防护**。

> [!tip] 选型口诀
> 先看主机接口，再看尺寸分辨率；要操作看触摸，要上柜看开孔；现场恶劣看防护。
