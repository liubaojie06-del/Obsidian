# 01 低端物理交互总览

> [!info] 核心结论
> 低端物理交互指通过 **按钮、旋钮、开关、指示灯、蜂鸣器、脚踏开关、信号灯塔、数码管、拨码开关** 等低成本硬件实现人与设备的基础交互。

## ① 分类/公式/规则

> [!warning] 选型公式
> **低端物理交互选型 = 操作动作 + 信号类型 + 电压等级 + 安装方式 + 防护等级 + 寿命 + 安全要求**

## ② 参考表

| 元件 | 作用 |
|---|---|
| **按钮** | 启动、停止、复位、确认 |
| **选择开关** | 手动/自动、模式选择 |
| **急停按钮** | 紧急停止危险动作 |
| **指示灯** | 显示运行、报警、电源状态 |
| **蜂鸣器** | 声音报警和提示 |
| **旋钮/电位器** | 调节速度、压力、亮度等 |
| **脚踏开关** | 脚控启动、夹紧、点动 |
| **信号灯塔** | 远距离显示设备状态 |

## ③ 计算/选型示例

- 已知：  
  一台小设备只需要启动、停止、复位和报警提示。
- 公式：  
  **简单操作 + 少量状态显示 → 物理按钮 + 指示灯**
- 计算：  
  不一定需要HMI触摸屏，按钮和指示灯即可满足基本交互。
- 结论：  
  可配置 **启动按钮、停止按钮、复位按钮、报警指示灯、蜂鸣器**。

> [!tip] 记忆口诀
> 简单设备不一定上屏，按钮灯蜂鸣器就能完成基础交互。

# 02 底层原理

> [!info] 核心结论
> 低端物理交互的本质，是把人的动作转换成电信号，把设备状态转换成光、声、机械反馈。

## ① 分类/公式/规则

> [!note] 工作链路
> 人按按钮或转旋钮，触点或传感元件产生电信号，PLC读取后执行逻辑；  
> PLC输出信号点亮灯、驱动蜂鸣器或控制继电器，反馈设备状态。

> [!warning] 交互链路
> **人操作 → 物理元件 → 电信号 → PLC判断 → 输出反馈 → 人看到/听到状态**

## ② 参考表

| 交互方向 | 典型元件 |
|---|---|
| **人给设备命令** | 按钮、开关、脚踏、旋钮 |
| **设备给人反馈** | 指示灯、蜂鸣器、信号灯塔 |
| **连续调节** | 电位器、编码器旋钮 |
| **离散选择** | 选择开关、拨码开关 |
| **安全动作** | 急停按钮、安全门开关 |
| **确认动作** | 复位按钮、确认按钮 |

## ③ 计算/选型示例

- 已知：  
  操作员按下启动按钮后，PLC输入点ON。
- 公式：  
  **按钮按下 = 触点闭合 = PLC输入有效**
- 计算：  
  PLC程序检测启动条件满足后，输出运行信号。
- 结论：  
  物理交互的核心就是 **触点状态与PLC逻辑的对应关系**。

> [!tip] 记忆口诀
> 人按下去是机械动作，PLC看到的是电信号。

# 03 按钮

> [!info] 核心结论
> 按钮是最基础的人机输入元件，常用于启动、停止、复位、确认、点动和报警消音。

## ① 分类/公式/规则

> [!warning] 按钮选型公式
> **按钮选型 = 功能 + 颜色 + 触点形式 + 电压等级 + 安装孔径 + 防护等级**

## ② 参考表

| 按钮类型 | 适合场景 |
|---|---|
| **常开按钮NO** | 启动、确认、复位 |
| **常闭按钮NC** | 停止、断线更易发现 |
| **自复位按钮** | 点动、确认 |
| **自锁按钮** | 保持状态 |
| **带灯按钮** | 操作与状态合一 |
| **蘑菇头按钮** | 停止或明显操作 |
| **防水按钮** | 潮湿、粉尘环境 |

## ③ 计算/选型示例

- 已知：  
  设备需要一个启动按钮，按下后PLC收到启动信号。
- 公式：  
  **启动按钮 → 常开触点NO**
- 计算：  
  平时输入OFF，按下后触点闭合，PLC输入ON。
- 结论：  
  启动按钮常用 **绿色常开自复位按钮**。

> [!tip] 记忆口诀
> 启动常开，停止常闭；绿色启动，红色停止。

# 04 选择开关

> [!info] 核心结论
> 选择开关用于模式选择，例如手动/自动、本地/远程、正转/反转、单步/连续。

## ① 分类/公式/规则

> [!warning] 选择开关选型公式
> **选择开关选型 = 档位数量 + 保持/自复位 + 触点数量 + 安装孔径 + 标识方式**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **二位选择开关** | 手动/自动、本地/远程 |
| **三位选择开关** | 左/停/右，手动/停/自动 |
| **钥匙开关** | 权限控制 |
| **自复位选择开关** | 点动、临时操作 |
| **保持型选择开关** | 模式保持 |
| **多触点开关** | 同时给PLC和指示灯信号 |

## ③ 计算/选型示例

- 已知：  
  设备需要手动模式和自动模式切换。
- 公式：  
  **模式选择 = 二位保持选择开关**
- 计算：  
  开关保持在手动或自动位置，PLC根据输入点判断当前模式。
- 结论：  
  可选 **二位保持型选择开关**。

> [!tip] 记忆口诀
> 按钮给瞬时命令，选择开关给长期模式。

# 05 急停按钮

> [!info] 核心结论
> 急停按钮用于紧急切断危险动作，属于安全相关元件，不能只当普通PLC输入使用。

## ① 分类/公式/规则

> [!warning] 安全规则
> **急停信号 → 安全继电器/安全PLC/STO/安全回路**  
> **普通PLC程序不能替代安全回路**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **红色蘑菇头** | 急停常见外观 |
| **黄色底座** | 提高识别度 |
| **常闭触点NC** | 断线或按下都会断开 |
| **旋转复位** | 防止误复位 |
| **双通道触点** | 安全回路常用 |
| **安全继电器** | 监控急停回路 |
| **STO** | 伺服安全转矩关闭 |

## ③ 计算/选型示例

- 已知：  
  伺服设备急停后必须停止输出转矩。
- 公式：  
  **急停按下 → 安全回路断开 → STO触发**
- 计算：  
  不能只让PLC停止程序，因为PLC程序本身不是安全功能。
- 结论：  
  急停应接入安全回路，并控制伺服STO或主动力切断。

> [!tip] 记忆口诀
> 急停不是普通按钮，是安全链路的一部分。

# 06 指示灯

> [!info] 核心结论
> 指示灯用于显示设备状态，例如电源、运行、停止、报警、待机、完成和故障。

## ① 分类/公式/规则

> [!warning] 指示灯选型公式
> **指示灯选型 = 颜色含义 + 电压等级 + 安装孔径 + 亮度 + 防护等级**

## ② 参考表

| 颜色 | 常见含义 |
|---|---|
| **绿色** | 运行、正常、允许 |
| **红色** | 故障、停止、危险 |
| **黄色/橙色** | 警告、待处理 |
| **蓝色** | 特殊状态、维护 |
| **白色** | 电源、照明、通用提示 |
| **闪烁** | 需要关注或状态变化 |

## ③ 计算/选型示例

- 已知：  
  设备报警时需要操作员快速发现。
- 公式：  
  **报警状态 → 红灯闪烁 + 蜂鸣器**
- 计算：  
  只有画面报警可能不够醒目。
- 结论：  
  可配置红色报警指示灯，并配合蜂鸣器提示。

> [!tip] 记忆口诀
> 灯是设备表情，颜色要让人一眼看懂。

# 07 蜂鸣器

> [!info] 核心结论
> 蜂鸣器用于声音提示，适合报警、启动提示、完成提示、异常提醒和安全警告。

## ① 分类/公式/规则

> [!warning] 蜂鸣器选型公式
> **蜂鸣器选型 = 声压大小 + 工作电压 + 声音模式 + 安装方式 + 环境噪声**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **连续音蜂鸣器** | 简单报警 |
| **间歇音蜂鸣器** | 更容易引起注意 |
| **多音调报警器** | 区分不同状态 |
| **带灯蜂鸣器** | 声光一体 |
| **可调音量** | 适应不同现场 |
| **高声压报警器** | 噪声大的车间 |

## ③ 计算/选型示例

- 已知：  
  车间噪声较大，普通蜂鸣器听不清。
- 公式：  
  **报警声压 > 环境噪声**
- 计算：  
  环境噪声越大，蜂鸣器声压要求越高。
- 结论：  
  应选择更高声压的工业报警器，或配合信号灯塔。

> [!tip] 记忆口诀
> 灯给眼睛看，蜂鸣器给耳朵听。

# 08 旋钮与电位器

> [!info] 核心结论
> 旋钮和电位器用于连续调节，例如速度、压力、温度设定、亮度、频率或比例阀开度。

## ① 分类/公式/规则

> [!warning] 模拟量换算公式
> **工程值 = 输入比例 × 工程量满量程**

## ② 参考表

| 类型 | 应用 |
|---|---|
| **电位器旋钮** | 模拟量设定 |
| **多圈电位器** | 精细调节 |
| **编码器旋钮** | 数字增减调节 |
| **带刻度旋钮** | 直观设定 |
| **带开关旋钮** | 调节 + 开关 |
| **防误触旋钮** | 关键参数调节 |

## ③ 计算/选型示例

- 已知：  
  电位器输出0-10V，对应输送速度0-1000mm/s，PLC读到6V。
- 公式：  
  **速度 = 6 ÷ 10 × 1000**
- 计算：  
  速度 = **600mm/s**
- 结论：  
  当前旋钮设定速度约为600mm/s。

> [!tip] 记忆口诀
> 按钮选状态，旋钮调大小。

# 09 编码器旋钮

> [!info] 核心结论
> 编码器旋钮用于数字化参数调节，常见于菜单选择、数值增减、速度微调和设备面板操作。

## ① 分类/公式/规则

> [!warning] 编码器判断规则
> **旋转方向 = A/B相先后顺序**  
> **调节步数 = 脉冲数量 × 每格增量**

## ② 参考表

| 项目 | 说明 |
|---|---|
| **A/B相输出** | 判断方向和步数 |
| **带按压功能** | 确认操作 |
| **有档位手感** | 便于逐格调节 |
| **无绝对位置** | 上电后只知道增量 |
| **抗抖处理** | 防止误增减 |
| **适合数字菜单** | 比电位器更适合参数输入 |

## ③ 计算/选型示例

- 已知：  
  编码器旋钮每格对应速度增加10mm/s，向右旋转5格。
- 公式：  
  **速度增量 = 5 × 10**
- 计算：  
  速度增加 **50mm/s**
- 结论：  
  编码器旋钮适合做参数微调。

> [!tip] 记忆口诀
> 电位器给绝对模拟值，编码器旋钮给增量步数。

> [!info] 核心结论
> 脚踏开关用于脚控设备动作，常见于夹紧、冲压、点焊、点胶、缝纫、测试启动等场景。

## ① 分类/公式/规则

> [!warning] 脚踏开关选型公式
> **脚踏开关选型 = 触点形式 + 防护罩 + 防滑结构 + 机械寿命 + 安全等级**

## ② 参考表

| 类型 | 特点 |
|---|---|
| **普通脚踏开关** | 简单脚控启动 |
| **带防护罩脚踏** | 防止误踩 |
| **双脚踏开关** | 部分安全场景使用 |
| **保持型脚踏** | 踩下保持状态 |
| **自复位脚踏** | 松开即断开 |
| **防水脚踏** | 潮湿环境使用 |

## ③ 计算/选型示例

- 已知：  
  操作员双手需要扶工件，启动动作需要脚控制。
- 公式：  
  **双手忙 + 需要启动 → 脚踏开关**
- 计算：  
  脚踏可以释放双手，但必须防止误踩造成危险。
- 结论：  
  可选择带防护罩脚踏开关，并在PLC中加入安全互锁。

> [!tip] 记忆口诀
> 脚踏方便，但最怕误踩。

# 11 信号灯塔

> [!info] 核心结论
> 信号灯塔用于远距离显示设备状态，常用于产线、设备群、无人值守设备和车间看板辅助提示。

## ① 分类/公式/规则

> [!warning] 灯塔选型公式
> **信号灯塔选型 = 层数 + 颜色 + 蜂鸣功能 + 电压 + 安装方式 + 可视距离**

## ② 参考表

| 颜色/状态 | 常见含义 |
|---|---|
| **绿灯常亮** | 正常运行 |
| **黄灯常亮** | 待机或等待 |
| **红灯常亮** | 停机或故障 |
| **红灯闪烁** | 严重报警 |
| **蜂鸣开启** | 需要人工处理 |
| **蓝灯** | 维护、缺料、呼叫 |

## ③ 计算/选型示例

- 已知：  
  设备无人值守，异常时需要远处人员发现。
- 公式：  
  **远距离状态提示 → 信号灯塔**
- 计算：  
  单个面板指示灯只能近距离查看。
- 结论：  
  应配置多层信号灯塔，并定义统一颜色含义。

> [!tip] 记忆口诀
> 面板灯给近处看，信号灯塔给远处看。

> [!warning] 显示选型公式
> **显示器件选型 = 显示变量 + 输入信号 + 位数 + 精度 + 安装尺寸 + 可读距离**

|---|---|

## ③ 计算/选型示例

- 已知：

# 13 拨码开关

> [!info] 核心结论
> 拨码开关用于设置地址、模式、编号、站号和简单参数，适合不频繁修改的硬件配置。

## ① 分类/公式/规则

> [!warning] 拨码规则
> **拨码值 = 各位权重相加**

## ② 参考表

| 用途 | 示例 |
|---|---|
| **站号设置** | 通讯地址 |
| **模式选择** | 工作模式、调试模式 |
| **功能开关** | 启用/禁用某功能 |
| **硬件配置** | 终端电阻、通讯速率 |
| **设备编号** | 工位号、模块号 |
| **简单参数** | 倍率、档位、区域号 |

## ③ 计算/选型示例

- 已知：  
  4位拨码开关采用二进制权重1、2、4、8，拨到ON的是1和4。
- 公式：  
  **拨码值 = 1 + 4**
- 计算：  
  拨码值 = **5**
- 结论：  
  当前硬件地址或模式值为5。

> [!tip] 记忆口诀
> 拨码适合少改的设置，别拿它做高频操作。

# 14 接线与PLC输入输出

> [!info] 核心结论
> 低端物理交互元件多数通过DI/DO接入PLC，必须匹配电压、公共端、触点形式和负载电流。

## ① 分类/公式/规则

> [!warning] 接线公式
> **输入可靠 = 电压匹配 + COM正确 + 触点形式正确**  
> **输出可靠 = 电压匹配 + 电流足够 + 浪涌保护**

## ② 参考表

| 元件 | PLC连接方式 |
|---|---|
| **按钮/开关** | DI输入 |
| **急停** | 安全回路，必要时反馈PLC |
| **指示灯** | DO输出 |
| **蜂鸣器** | DO输出或继电器驱动 |
| **电位器** | AI模拟量输入 |
| **编码器旋钮** | 高速DI或专用输入 |
| **灯塔** | 多路DO输出 |

## ③ 计算/选型示例

- 已知：  
  一个蜂鸣器DC24V，电流0.8A，PLC输出点最大0.5A。
- 公式：  
  **负载电流 > PLC输出能力**
- 计算：  
  0.8A > 0.5A，PLC不能直接驱动。
- 结论：  
  应通过中间继电器或MOSFET模块驱动蜂鸣器。

> [!tip] 记忆口诀
> 输入看信号，输出看电流；带不动就加继电器。

# 15 防护与寿命

> [!info] 核心结论
> 物理交互元件长期被人按、转、踩，必须考虑机械寿命、防护等级、触点容量和现场环境。

## ① 分类/公式/规则

> [!warning] 寿命选型公式
> **元件寿命需求 = 每天动作次数 × 使用天数 × 安全余量**

## ② 参考表

| 项目 | 影响 |
|---|---|
| **机械寿命** | 按钮、开关可动作次数 |
| **电气寿命** | 触点带负载通断寿命 |
| **防护等级** | 防水、防尘、防油 |
| **触点容量** | 可承受电压和电流 |
| **安装牢固性** | 防止松动 |
| **标识耐久性** | 防止文字磨损 |

## ③ 计算/选型示例

- 已知：  
  一个按钮每天按1000次，设备计划使用3年，每年300天。
- 公式：  
  **动作次数 = 1000 × 3 × 300**
- 计算：  
  动作次数 = **900000次**
- 结论：  
  按钮机械寿命应明显高于90万次，并留安全余量。

> [!tip] 记忆口诀
> 人天天按的元件，寿命不能只看能不能亮。

# 16 常见故障排查

> [!info] 核心结论
> 低端物理交互故障排查按 **电源、触点、接线、PLC输入输出、负载、程序逻辑** 的顺序进行。

## ① 分类/公式/规则

> [!warning] 排错公式
> **物理交互异常 = 元件损坏 + 触点接触不良 + 接线松动 + COM错误 + PLC地址错误 + 负载损坏**

## ② 参考表

| 现象 | 可能原因 |
|---|---|
| **按钮按下无反应** | 触点坏、线断、PLC输入无电 |
| **指示灯不亮** | 灯坏、输出无电、公共端错误 |
| **蜂鸣器不响** | 电流不足、蜂鸣器坏、输出未动作 |
| **旋钮数值跳变** | 电位器磨损、模拟量干扰 |
| **选择开关状态不对** | 触点接错、程序地址错 |
| **急停无法复位** | 急停未旋回、安全回路未闭合 |
| **灯塔颜色混乱** | 输出点接错或程序定义错 |

## ③ 计算/选型示例

- 已知：  
  按钮按下后，PLC输入灯不亮。
- 公式：  
  **输入灯不亮 = 按钮触点 / 电源 / 接线 / COM / PLC输入异常**
- 计算：  
  先用万用表测按钮前后电压，再测PLC输入端电压。
- 结论：  
  端子有电但PLC不亮，查输入类型和模块；端子无电，查按钮和线路。

> [!tip] 记忆口诀
> 按钮不灵查输入，灯不亮查输出，声音不响查电流。

# 17 低端物理交互快速选型表

> [!info] 核心结论
> 低端物理交互选型要从操作功能出发，统一确认按钮、开关、灯、蜂鸣器、旋钮、脚踏、安全和PLC接口。

## ① 分类/公式/规则

> [!note] 十步选型法
> ① 明确操作功能：启动、停止、复位、模式、调节  
> ② 确认反馈方式：灯光、声音、数码显示  
> ③ 确认是否涉及安全：急停、安全门、脚踏危险动作  
> ④ 确认电压等级：DC24V、AC220V等  
> ⑤ 确认触点形式：NO/NC、自复位/自锁  
> ⑥ 确认PLC输入输出点数  
> ⑦ 确认负载电流是否需要继电器  
> ⑧ 确认安装孔径和面板布局  
> ⑨ 确认防护等级和现场环境  
> ⑩ 确认标识、颜色、权限和维护备件

> [!warning] 最终选型公式
> **推荐交互方案 = 按钮/开关 + 指示灯/蜂鸣器 + 旋钮/显示表 + PLC DI/DO/AI + 安全回路**

## ② 参考表

| 需求 | 推荐元件 |
|---|---|
| 启动停止 | 按钮 |
| 模式切换 | 选择开关 |
| 紧急停机 | 急停按钮 + 安全回路 |
| 状态提示 | 指示灯 |
| 远距离状态 | 信号灯塔 |
| 声音报警 | 蜂鸣器 |
| 连续调节 | 电位器/编码器旋钮 |
| 脚控动作 | 脚踏开关 |
| 单数值显示 | 数码管/面板表 |

## ③ 计算/选型示例

- 已知：  
  一台简易压装设备需要启动、停止、急停、手自动切换、压力设定、运行/报警提示。
- 公式：  
  **交互需求 = 按钮 + 急停 + 选择开关 + 电位器 + 指示灯 + 蜂鸣器**
- 计算：  
  输入侧需要启动、停止、急停反馈、手自动、压力设定；输出侧需要运行灯、报警灯、蜂鸣器。
- 结论：  
  推荐配置 **启动按钮、停止按钮、急停按钮、二位选择开关、压力设定旋钮、运行指示灯、报警灯、蜂鸣器**，急停进入安全回路。

> [!tip] 选型口诀
> 能按就用按钮，能选就用开关，能调就用旋钮；要看用灯，要听用蜂鸣；涉及安全必须走安全回路。
