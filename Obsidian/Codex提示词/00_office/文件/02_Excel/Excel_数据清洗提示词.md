# Excel 数据清洗提示词

## 适用场景

用于整理重复、缺失、格式不统一、字段命名混乱的 Excel 数据表。

## 推荐工具

- Excel
- Codex 或其他 AI 助手

## 使用前准备

- 准备字段说明和样例数据。
- 明确清洗目标，例如去重、拆分姓名、统一日期、规范分类。
- 标记不能修改的关键字段。

## 完整提示词

```text
请帮我设计一套 Excel 数据清洗方案。

表格用途：
字段列表：
样例数据：
当前问题：
清洗目标：

请输出：
1. 数据质量问题清单。
2. 每个问题的处理规则和 Excel 操作步骤。
3. 可使用的公式、Power Query 操作或辅助列方案。
4. 清洗前后的字段对照表。
5. 最后给出核验方法，确保没有误删、误改关键数据。

请不要编造不存在的数据；不确定的地方请标记为“需人工确认”。
```

## 使用方法

1. 粘贴字段列表和几行样例数据。
2. 先确认清洗规则，再执行公式或 Power Query 操作。
3. 保留原始数据备份，再输出清洗结果。
