# Excel 公式生成提示词

## 适用场景

用于根据业务规则生成 Excel 公式，并解释公式逻辑、适用版本和可能的错误。

## 推荐工具

- Excel
- Codex 或其他 AI 助手

## 使用前准备

- 明确 Excel 版本。
- 准备列名、单元格位置和样例数据。
- 描述需要计算、匹配、统计或判断的业务规则。

## 完整提示词

```text
请帮我生成 Excel 公式。

Excel 版本：
表格字段：
单元格范围：
想实现的结果：
样例数据：
特殊规则：

请输出：
1. 推荐公式。
2. 公式逐段解释。
3. 如果有多个方案，请比较普通公式、动态数组公式、XLOOKUP/FILTER/SUMIFS 等方案。
4. 说明公式应填在哪个单元格，以及如何向下填充。
5. 列出常见错误和排查方法。

请优先使用我当前 Excel 版本支持的公式。
```

## 使用方法

1. 补充 Excel 版本和字段名。
2. 复制推荐公式到测试单元格。
3. 用样例数据验证结果后再应用到整张表。
