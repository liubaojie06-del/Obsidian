# Word 文档整理提示词

## 适用场景

用于整理 Word 报告、制度文件、论文格式、说明书、学习资料。

## 对应笔记类型

Word 文档整理模板、办公软件提示词笔记。

## 使用前准备

- 提供 Word 文档路径或原始文字。
- 说明目标格式：报告、论文、制度、说明书、笔记。
- 说明是否需要目录、标题编号、表格、引用格式。

## 完整提示词

```text
请帮我整理一个 Word 文档，使它结构清晰、格式统一、适合正式使用。

请先分析内容结构，不要直接乱改。
如果需要修改现有文件，请先输出整理方案，等我确认后再执行。
不要删除原文中的重要信息。
如需另存，请保留原文件，生成新版本。

请按下面流程：

1. 识别文档类型
- 报告
- 论文
- 制度文件
- 操作说明
- 学习资料
- 其他

2. 输出整理方案

| 项目 | 当前问题 | 修改建议 |
|---|---|---|
| 标题层级 |  |  |
| 段落格式 |  |  |
| 表格 |  |  |
| 图片 / 图注 |  |  |
| 目录 |  |  |
| 页眉页脚 |  |  |
| 编号 |  |  |

3. 统一结构
请使用清晰标题层级：
- 一级标题
- 二级标题
- 三级标题

4. 优化内容
- 修正表达不清的句子。
- 合并重复内容。
- 保留专业术语。
- 不改变原意。

5. 格式要求
- 标题层级统一。
- 正文格式统一。
- 表格对齐清楚。
- 需要时添加目录。
- 需要时添加页码。

6. 输出结果
请给出：
- 修改摘要。
- 新文件路径。
- 仍需人工确认的地方。
```

## 使用方法

把文档路径或原文贴到提示词后面，适合做正式文档前的结构整理。

## 输出格式要求

- 先分析，后修改。
- 不覆盖原文件。
- 结构和格式要统一。
