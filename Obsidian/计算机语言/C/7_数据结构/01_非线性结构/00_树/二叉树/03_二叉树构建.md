# 01 | 构建总览：先确定根，再划分左右子树

> [!abstract] 核心结论  
> 二叉树构建的核心是找到根结点，并确定左子树和右子树的范围。

## ① 底层原理

前序第一个是根，后序最后一个是根，中序可以用根划分左右子树。

---

## ② 基本语法 / 代码用法

```c
typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;
```

---

## ③ 含义 / 说明

```text
构建之前先定义结点结构，之后所有构建函数都返回 TreeNode*。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只知道根，但不知道左右子树边界。构建题最关键的是范围。

> [!tip] 记忆口诀  
> **先找根，再分家。**

# 02 | 手动创建结点：malloc 申请空间

> [!abstract] 核心结论  
> 动态创建二叉树结点时，要用 malloc 申请空间，并初始化左右孩子为 NULL。

## ① 底层原理

新结点刚创建时一般没有孩子，所以 left 和 right 都应先置空。

---

## ② 基本语法 / 代码用法

```c
TreeNode *createNode(int value) {
    TreeNode *node = (TreeNode *)malloc(sizeof(TreeNode));
    if (node == NULL) return NULL;
    node->data = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}
```

---

## ③ 含义 / 说明

```text
createNode 返回新结点地址。malloc 失败返回 NULL，防止继续访问空指针。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：malloc 后不判断 NULL，或者忘记初始化 left/right。

> [!tip] 记忆口诀  
> **新结点，先清左右。**

# 03 | 手动连接二叉树：父结点指向孩子

> [!abstract] 核心结论  
> 构建父子关系时，本质是让父结点的 left 或 right 保存孩子地址。

## ① 底层原理

树的结构不在数据值里，而在指针关系里。连接错指针，树结构就错。

---

## ② 基本语法 / 代码用法

```c
TreeNode *root = createNode(1);
TreeNode *left = createNode(2);
TreeNode *right = createNode(3);
root->left = left;
root->right = right;
```

---

## ③ 含义 / 说明

```text
root->left = left 表示把 left 结点接到 root 左边。
```

---

## ④ 输出

```text
形成一棵根为 1，左右孩子为 2 和 3 的二叉树。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：root->left->data = 2；如果 left 还是 NULL，会空指针访问。

> [!tip] 记忆口诀  
> **先造孩子，再接父亲。**

# 04 | 递归输入构建：遇到空标记返回 NULL

> [!abstract] 核心结论  
> 前序递归构建常用特殊值表示空结点，例如 -1。

## ① 底层原理

输入顺序是根、左子树、右子树。遇到 -1 说明当前位置没有结点，递归停止。

---

## ② 基本语法 / 代码用法

```c
TreeNode *createTree(void) {
    int value;
    scanf("%d", &value);
    if (value == -1) return NULL;
    TreeNode *root = createNode(value);
    root->left = createTree();
    root->right = createTree();
    return root;
}
```

---

## ③ 含义 / 说明

```text
先读当前根，再递归创建左子树和右子树。
```

---

## ④ 输出

```text
输入 1 2 -1 -1 3 -1 -1 可构建根 1、左右孩子 2 和 3。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：没有 -1 这样的结束标记，递归无法停止。

> [!tip] 记忆口诀  
> **遇空返回，递归才停。**

# 05 | 根据前序和中序构建：前序找根，中序切分

> [!abstract] 核心结论  
> 前序第一个元素是根，中序中根左边是左子树，右边是右子树。

## ① 底层原理

只要知道左子树长度，就能在前序数组中切出左右子树范围。

---

## ② 基本语法 / 代码用法

```c
TreeNode *buildPreIn(int pre[], int in[], int pl, int pr, int il, int ir) {
    if (pl > pr) return NULL;
    int rootVal = pre[pl];
    TreeNode *root = createNode(rootVal);
    int k = findIndex(in, il, ir, rootVal);
    int leftSize = k - il;
    root->left = buildPreIn(pre, in, pl + 1, pl + leftSize, il, k - 1);
    root->right = buildPreIn(pre, in, pl + leftSize + 1, pr, k + 1, ir);
    return root;
}
```

---

## ③ 含义 / 说明

```text
pl/pr 是前序范围，il/ir 是中序范围。k 是根在中序中的位置。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：左子树长度算错，导致递归范围错位。

> [!tip] 记忆口诀  
> **前序拿根，中序分段。**

# 06 | 根据后序和中序构建：后序最后是根

> [!abstract] 核心结论  
> 后序最后一个元素是根，中序仍然负责划分左右子树。

## ① 底层原理

后序顺序是左、右、根，所以 rootVal = post[postRight]。

---

## ② 基本语法 / 代码用法

```c
TreeNode *buildPostIn(int post[], int in[], int pl, int pr, int il, int ir) {
    if (pl > pr) return NULL;
    int rootVal = post[pr];
    TreeNode *root = createNode(rootVal);
    int k = findIndex(in, il, ir, rootVal);
    int leftSize = k - il;
    root->left = buildPostIn(post, in, pl, pl + leftSize - 1, il, k - 1);
    root->right = buildPostIn(post, in, pl + leftSize, pr - 1, k + 1, ir);
    return root;
}
```

---

## ③ 含义 / 说明

```text
右子树范围在后序中位于左子树之后、根结点之前。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：把 post[pl] 当根。后序的根在最后，不在最前。

> [!tip] 记忆口诀  
> **后序看末尾，中序分左右。**

# 07 | 数组构建完全二叉树：按下标找孩子

> [!abstract] 核心结论  
> 完全二叉树可以按层序放在数组中，通过下标递归构建。

## ① 底层原理

下标 i 的左孩子是 2i+1，右孩子是 2i+2。越界表示没有孩子。

---

## ② 基本语法 / 代码用法

```c
TreeNode *buildComplete(int arr[], int n, int i) {
    if (i >= n) return NULL;
    TreeNode *root = createNode(arr[i]);
    root->left = buildComplete(arr, n, 2 * i + 1);
    root->right = buildComplete(arr, n, 2 * i + 2);
    return root;
}
```

---

## ③ 含义 / 说明

```text
index 超过数组长度时返回 NULL。
```

---

## ④ 输出

```text
数组 {1,2,3} 构建根 1，左右孩子 2 和 3。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：下标从 0 开始时仍使用 2i 和 2i+1。

> [!tip] 记忆口诀  
> **零号作根，左右加一加二。**

# 08 | 序列化：把树保存成序列

> [!abstract] 核心结论  
> 序列化是把二叉树转换成字符串或数组，方便保存和传输。

## ① 底层原理

必须记录空结点，否则结构信息会丢失。常用 # 表示 NULL。

---

## ② 基本语法 / 代码用法

```c
void serialize(TreeNode *root) {
    if (root == NULL) {
        printf("# ");
        return;
    }
    printf("%d ", root->data);
    serialize(root->left);
    serialize(root->right);
}
```

---

## ③ 含义 / 说明

```text
这里使用前序序列化：根、左、右，并用 # 保存空指针。
```

---

## ④ 输出

```text
树 1(2,3) 可输出：1 2 # # 3 # #
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只保存非空结点，导致反序列化无法确定结构。

> [!tip] 记忆口诀  
> **序列化，空也要记。**

---

# 09 | 反序列化：按序列重建树

> [!abstract] 核心结论  
> 反序列化是根据序列化结果重新创建二叉树。

## ① 底层原理

读取到 # 返回 NULL；读取到数字就创建结点，再递归构建左右子树。

---

## ② 基本语法 / 代码用法

```c
TreeNode *deserialize(char **tokens, int *index) {
    if (tokens[*index][0] == '#') {
        (*index)++;
        return NULL;
    }
    int value = atoi(tokens[*index]);
    (*index)++;
    TreeNode *root = createNode(value);
    root->left = deserialize(tokens, index);
    root->right = deserialize(tokens, index);
    return root;
}
```

---

## ③ 含义 / 说明

```text
index 记录当前读到哪个 token，每处理一个 token 就向后移动。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：递归时不推进 index，导致一直读取同一个元素。

> [!tip] 记忆口诀  
> **读一个，走一步。**

# 10 | 构建主题总结：边界比代码更重要

> [!abstract] 核心结论  
> 构建二叉树最容易错在递归边界、数组范围、空结点标记。

## ① 底层原理

每一次递归都应该明确当前子树在数组中的范围，以及什么时候返回 NULL。

---

## ② 基本语法 / 代码用法

```c
if (left > right) {
    return NULL;
}
```

---

## ③ 含义 / 说明

```text
范围为空时说明当前子树不存在，应返回 NULL。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：边界条件写成 >=，导致单个结点不能被创建。

> [!tip] 记忆口诀  
> **构建看范围，范围空就停。**
