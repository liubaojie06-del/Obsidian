# 03 | 二叉树构建：根据遍历结果或数组恢复树

> [!abstract] 核心结论  
> 构建二叉树的关键是确定根结点和左右子树范围。前序 + 中序、后序 + 中序都可以唯一确定一棵二叉树；层序数组常用于构建完全二叉树。

## ① 底层原理

| 构建方式 | 核心思路 |
|---|---|
| 前序 + 中序 | 前序第一个是根，中序用根分左右子树 |
| 后序 + 中序 | 后序最后一个是根，中序用根分左右子树 |
| 数组构建完全二叉树 | 根据下标关系连接父子 |
| 序列化 / 反序列化 | 用特殊标记保存空结点 |

注意：

```text
只有前序 + 后序，一般不能唯一确定普通二叉树。
```

---

## ② 根据前序 + 中序构建

```text
前序：根 左 右
中序：左 根 右
```

步骤：

```text
1. 前序第一个元素是根
2. 在中序中找到根
3. 根左边是左子树
4. 根右边是右子树
5. 递归构建左右子树
```

```c
#include <stdlib.h>

typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

TreeNode *createNode(int value) {
    TreeNode *node = (TreeNode *)malloc(sizeof(TreeNode));
    node->data = value;
    node->left = NULL;
    node->right = NULL;
    return node;
}

int findIndex(int arr[], int left, int right, int value) {
    for (int i = left; i <= right; i++) {
        if (arr[i] == value) {
            return i;
        }
    }
    return -1;
}

TreeNode *buildPreIn(int preorder[], int inorder[], int preLeft, int preRight, int inLeft, int inRight) {
    if (preLeft > preRight) {
        return NULL;
    }

    int rootValue = preorder[preLeft];
    TreeNode *root = createNode(rootValue);

    int rootIndex = findIndex(inorder, inLeft, inRight, rootValue);
    int leftSize = rootIndex - inLeft;

    root->left = buildPreIn(preorder, inorder, preLeft + 1, preLeft + leftSize, inLeft, rootIndex - 1);
    root->right = buildPreIn(preorder, inorder, preLeft + leftSize + 1, preRight, rootIndex + 1, inRight);

    return root;
}
```

## ③ 根据后序 + 中序构建

```text
后序：左 右 根
中序：左 根 右
```

```c
TreeNode *buildPostIn(int postorder[], int inorder[], int postLeft, int postRight, int inLeft, int inRight) {
    if (postLeft > postRight) {
        return NULL;
    }

    int rootValue = postorder[postRight];
    TreeNode *root = createNode(rootValue);

    int rootIndex = findIndex(inorder, inLeft, inRight, rootValue);
    int leftSize = rootIndex - inLeft;

    root->left = buildPostIn(postorder, inorder, postLeft, postLeft + leftSize - 1, inLeft, rootIndex - 1);
    root->right = buildPostIn(postorder, inorder, postLeft + leftSize, postRight - 1, rootIndex + 1, inRight);

    return root;
}
```

## ④ 根据数组构建完全二叉树

如果数组下标从 `0` 开始：

```text
当前结点 i
左孩子：2 * i + 1
右孩子：2 * i + 2
父结点：(i - 1) / 2
```

```c
TreeNode *buildCompleteTree(int arr[], int n, int index) {
    if (index >= n) {
        return NULL;
    }

    TreeNode *root = createNode(arr[index]);
    root->left = buildCompleteTree(arr, n, 2 * index + 1);
    root->right = buildCompleteTree(arr, n, 2 * index + 2);

    return root;
}
```

## ⑤ 序列化与反序列化

序列化就是把树变成字符串或数组。

```text
1 2 # # 3 # #
```

其中 `#` 表示空结点，对应：

```text
        1
       / \
      2   3
```

如果不保存空结点，很多不同结构可能得到相同遍历结果。

---

## ⑥ 重点 / 注意 / 常见错误

| 问题 | 关键点 |
|---|---|
| 前序 + 中序 | 前序找根，中序分左右 |
| 后序 + 中序 | 后序找根，中序分左右 |
| 完全二叉树数组 | 下标关系构建 |
| 序列化 | 必须记录空结点 |
| 递归构建 | 范围边界最容易错 |

> [!tip] 记忆口诀  
> **前后找根，中序分家。**
