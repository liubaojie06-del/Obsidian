# 08 | 二叉树综合应用：翻转、对称、合并、序列化、DFS/BFS

> [!abstract] 核心结论  
> 二叉树综合题本质上还是递归、DFS、BFS 的组合。先判断当前结点，再把问题拆给左子树和右子树，是最常见的解题思路。

## ① 底层原理

| 题型 | 核心思路 |
|---|---|
| 翻转二叉树 | 交换每个结点的左右孩子 |
| 对称二叉树 | 比较左子树和右子树是否镜像 |
| 合并二叉树 | 同位置结点相加 |
| 序列化 / 反序列化 | 遍历时记录空结点 |
| 最近公共祖先 | 左右子树递归返回结果 |
| DFS 综合题 | 递归或栈 |
| BFS 综合题 | 队列 |

二叉树综合题通用思路：

```text
1. 当前结点为空怎么办
2. 当前结点要做什么
3. 左子树递归做什么
4. 右子树递归做什么
5. 左右结果如何合并
```

## ② 翻转二叉树

翻转前：

```text
        1
       / \
      2   3
```

翻转后：

```text
        1
       / \
      3   2
```

代码：

```c
typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

TreeNode *invertTree(TreeNode *root) {
    if (root == NULL) {
        return NULL;
    }

    TreeNode *temp = root->left;
    root->left = root->right;
    root->right = temp;

    invertTree(root->left);
    invertTree(root->right);

    return root;
}
```

## ③ 对称二叉树

对称二叉树要求左子树和右子树互为镜像。

```c
int isMirror(TreeNode *left, TreeNode *right) {
    if (left == NULL && right == NULL) {
        return 1;
    }

    if (left == NULL || right == NULL) {
        return 0;
    }

    if (left->data != right->data) {
        return 0;
    }

    return isMirror(left->left, right->right) &&
           isMirror(left->right, right->left);
}

int isSymmetric(TreeNode *root) {
    if (root == NULL) {
        return 1;
    }

    return isMirror(root->left, root->right);
}
```

## ④ 合并二叉树

同位置结点相加，如果一边为空，就返回另一边。

```c
TreeNode *mergeTrees(TreeNode *root1, TreeNode *root2) {
    if (root1 == NULL) {
        return root2;
    }

    if (root2 == NULL) {
        return root1;
    }

    root1->data += root2->data;
    root1->left = mergeTrees(root1->left, root2->left);
    root1->right = mergeTrees(root1->right, root2->right);

    return root1;
}
```

## ⑤ 序列化与反序列化思想

序列化必须记录空结点。

例如：

```text
        1
       / \
      2   3
```

前序序列化可以写成：

```text
1 2 # # 3 # #
```

其中：

```text
# 表示 NULL
```

如果不记录空结点，结构信息会丢失。

## ⑥ DFS 与 BFS 综合选择

| 场景 | 优先选择 |
|---|---|
| 需要走到叶子 | DFS |
| 需要记录路径 | DFS + 回溯 |
| 需要逐层处理 | BFS |
| 需要最短层数 | BFS |
| 需要左右子树合并结果 | 递归 DFS |

DFS 模板：

```c
void dfs(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    dfs(root->left);
    dfs(root->right);
}
```

## ⑦ 最近公共祖先

```c
TreeNode *lowestCommonAncestor(TreeNode *root, TreeNode *p, TreeNode *q) {
    if (root == NULL || root == p || root == q) {
        return root;
    }

    TreeNode *left = lowestCommonAncestor(root->left, p, q);
    TreeNode *right = lowestCommonAncestor(root->right, p, q);

    if (left != NULL && right != NULL) {
        return root;
    }

    return left != NULL ? left : right;
}
```

## ⑧ 重点 / 注意 / 常见错误

| 题型 | 易错点 |
|---|---|
| 翻转二叉树 | 只交换根，没有递归交换子树 |
| 对称二叉树 | 比较方向错，应比较外侧和内侧 |
| 合并二叉树 | 忘记处理一边为空的情况 |
| 序列化 | 忘记记录空结点 |
| LCA | 不理解递归返回值含义 |
| BFS | 忘记入队左右孩子前判断非空 |

对称二叉树比较方向：

```text
left->left  对比 right->right
left->right 对比 right->left
```

不是：

```text
left->left  对比 right->left
left->right 对比 right->right
```

> [!tip] 记忆口诀  
> **综合题别慌，先判空，再分左右。**
