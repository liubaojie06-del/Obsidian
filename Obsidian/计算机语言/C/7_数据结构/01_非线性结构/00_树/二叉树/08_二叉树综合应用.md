# 02 | 翻转二叉树：交换每个结点左右孩子

> [!abstract] 核心结论  
> 翻转二叉树就是对每个结点交换 left 和 right。

## ① 底层原理

只交换根不够，必须递归交换所有子树。

---

## ② 基本语法 / 代码用法

```c
TreeNode *invertTree(TreeNode *root) {
    if (root == NULL) return NULL;
    TreeNode *tmp = root->left;
    root->left = root->right;
    root->right = tmp;
    invertTree(root->left);
    invertTree(root->right);
    return root;
}
```

---

## ③ 含义 / 说明

```text
先交换当前结点，再递归处理交换后的左右子树。
```

---

## ④ 输出

```text
整棵树左右镜像翻转。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只交换 root 的左右孩子，忘记递归子树。

> [!tip] 记忆口诀  
> **每个结点都交换。**

# 03 | 对称二叉树：比较镜像位置

> [!abstract] 核心结论  
> 对称二叉树要求左子树和右子树互为镜像。

## ① 底层原理

外侧对外侧，内侧对内侧：left->left 比 right->right，left->right 比 right->left。

---

## ② 基本语法 / 代码用法

```c
int isMirror(TreeNode *a, TreeNode *b) {
    if (a == NULL && b == NULL) return 1;
    if (a == NULL || b == NULL) return 0;
    if (a->data != b->data) return 0;
    return isMirror(a->left, b->right) && isMirror(a->right, b->left);
}
```

---

## ③ 含义 / 说明

```text
两个空树对称，一个空一个非空不对称。
```

---

## ④ 输出

```text
对称返回 1，否则返回 0。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：比较 a->left 和 b->left，这是相同方向，不是镜像方向。

> [!tip] 记忆口诀  
> **对称看镜像，外外内内。**

---

# 04 | 合并二叉树：同位置相加

> [!abstract] 核心结论  
> 合并二叉树时，同位置都有结点就相加，只有一边有就保留那边。

## ① 底层原理

递归处理相同位置的左右孩子。

---

## ② 基本语法 / 代码用法

```c
TreeNode *mergeTrees(TreeNode *a, TreeNode *b) {
    if (a == NULL) return b;
    if (b == NULL) return a;
    a->data += b->data;
    a->left = mergeTrees(a->left, b->left);
    a->right = mergeTrees(a->right, b->right);
    return a;
}
```

---

## ③ 含义 / 说明

```text
这里直接把结果合并到 a 树上。
```

---

## ④ 输出

```text
返回合并后的树根。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：没有处理一边为空的情况。

> [!tip] 记忆口诀  
> **都有就加，单边就留。**

---

# 05 | 相同二叉树：结构和值都相同

> [!abstract] 核心结论  
> 判断两棵树是否相同，需要同时比较结构和值。

## ① 底层原理

两个空树相同，一个空一个非空不同，两个非空则值相同且左右子树相同。

---

## ② 基本语法 / 代码用法

```c
int isSameTree(TreeNode *a, TreeNode *b) {
    if (a == NULL && b == NULL) return 1;
    if (a == NULL || b == NULL) return 0;
    return a->data == b->data &&
           isSameTree(a->left, b->left) &&
           isSameTree(a->right, b->right);
}
```

---

## ③ 含义 / 说明

```text
相同树比较的是同方向位置。
```

---

## ④ 输出

```text
相同返回 1，不同返回 0。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只比较遍历结果，不比较空结点结构。

> [!tip] 记忆口诀  
> **相同看同位。**

---

# 06 | 子树判断：当前相同或递归向下找

> [!abstract] 核心结论  
> 判断 sub 是否是 root 的子树，要么当前开始完全相同，要么在左右子树里继续找。

## ① 底层原理

子树问题通常结合相同二叉树函数。

---

## ② 基本语法 / 代码用法

```c
int isSubtree(TreeNode *root, TreeNode *sub) {
    if (root == NULL) return sub == NULL;
    if (isSameTree(root, sub)) return 1;
    return isSubtree(root->left, sub) || isSubtree(root->right, sub);
}
```

---

## ③ 含义 / 说明

```text
先尝试以当前结点为起点匹配，再递归尝试左右子树。
```

---

## ④ 输出

```text
是子树返回 1，否则返回 0。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只比较根值相等，不继续比较整棵结构。

> [!tip] 记忆口诀  
> **先同树，再下找。**

# 07 | 序列化综合应用：保存结构必须记空

> [!abstract] 核心结论  
> 序列化用于保存或传输二叉树，必须保存空结点信息。

## ① 底层原理

没有空标记，就无法区分很多不同结构的树。

---

## ② 基本语法 / 代码用法

```c
void serialize(TreeNode *root) {
    if (root == NULL) { printf("# "); return; }
    printf("%d ", root->data);
    serialize(root->left);
    serialize(root->right);
}
```

---

## ③ 含义 / 说明

```text
前序序列化输出根、左、右，空结点用 # 表示。
```

---

## ④ 输出

```text
1 2 # # 3 # #
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：只输出 1 2 3，反序列化时结构不唯一。

> [!tip] 记忆口诀  
> **空也输出，结构才在。**

# 08 | LCA 综合应用：递归返回查找结果

> [!abstract] 核心结论  
> 最近公共祖先通过左右子树查找结果判断。

## ① 底层原理

左右都找到目标时，当前结点就是最近公共祖先。

---

## ② 基本语法 / 代码用法

```c
TreeNode *lca(TreeNode *root, TreeNode *p, TreeNode *q) {
    if (root == NULL || root == p || root == q) return root;
    TreeNode *left = lca(root->left, p, q);
    TreeNode *right = lca(root->right, p, q);
    if (left && right) return root;
    return left ? left : right;
}
```

---

## ③ 含义 / 说明

```text
返回非空表示当前子树中找到了目标或答案。
```

---

## ④ 输出

```text
返回 LCA 结点地址。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：找到一个目标就直接当最终答案，但另一个可能在其子树里。

> [!tip] 记忆口诀  
> **左右都回，当前是祖。**

# 09 | BFS 综合题：逐层处理

> [!abstract] 核心结论  
> 需要层数、每层平均值、右视图等问题时，优先考虑 BFS。

## ① 底层原理

BFS 通过队列保存当前层结点，可以一层一层处理。

---

## ② 基本语法 / 代码用法

```c
void bfs(TreeNode *root) {
    if (root == NULL) return;
    TreeNode *q[100]; int front = 0, rear = 0;
    q[rear++] = root;
    while (front < rear) {
        TreeNode *cur = q[front++];
        if (cur->left) q[rear++] = cur->left;
        if (cur->right) q[rear++] = cur->right;
    }
}
```

---

## ③ 含义 / 说明

```text
如果要分层，需要在每轮循环开始记录当前队列长度。
```

---

## ④ 输出

```text
无固定输出，重点看函数返回值或树结构变化。
```

---

## ⑤ 重点 / 注意 / 常见错误

常见错误：没有记录每层大小，导致所有结点混在一起。

> [!tip] 记忆口诀  
> **要按层，用队列。**
