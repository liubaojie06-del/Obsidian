# 05 | 二叉树路径问题：从根到叶、路径总和、最大路径和、LCA

> [!abstract] 核心结论  
> 路径问题通常用 DFS。根到叶路径要记录当前路径；路径总和要记录当前累计值；最大路径和要区分“向上贡献值”和“全局答案”；最近公共祖先要利用递归返回值。

## ① 底层原理

| 类型 | 核心思路 |
|---|---|
| 根到叶路径 | DFS + 回溯 |
| 所有路径 | 记录当前路径，到叶子时输出 |
| 路径总和 | 累加当前路径和 |
| 最大路径和 | 后序递归，维护全局最大值 |
| 最近公共祖先 LCA | 左右子树分别查找目标结点 |

路径题最常用框架：

```text
进入结点：加入当前路径
递归左右：继续向下搜索
离开结点：撤销当前选择
```

这就是回溯。

## ② 根到叶路径总和

判断是否存在一条从根到叶的路径，使路径和等于 `target`。

```c
typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

int hasPathSum(TreeNode *root, int target) {
    if (root == NULL) {
        return 0;
    }

    if (root->left == NULL && root->right == NULL) {
        return root->data == target;
    }

    int rest = target - root->data;

    return hasPathSum(root->left, rest) || hasPathSum(root->right, rest);
}
```

## ③ 输出所有根到叶路径

```c
#include <stdio.h>

void printPath(int path[], int length) {
    for (int i = 0; i < length; i++) {
        printf("%d ", path[i]);
    }
    printf("\n");
}

void allPaths(TreeNode *root, int path[], int length) {
    if (root == NULL) {
        return;
    }

    path[length] = root->data;
    length++;

    if (root->left == NULL && root->right == NULL) {
        printPath(path, length);
        return;
    }

    allPaths(root->left, path, length);
    allPaths(root->right, path, length);
}
```

> [!note] 代码备注  
> 这里 `length` 是值传递，所以递归返回后不需要手动 `length--`。如果使用全局数组和全局长度，就需要回溯撤销。

## ④ 最大路径和

最大路径和可以不经过根，但路径必须连续。

```text
向上返回：只能选一边，因为路径向父结点延伸不能分叉。
全局答案：可以左 + 根 + 右，因为路径可以在当前结点拐弯。
```

代码框架：

```c
int maxValue(int a, int b) {
    return a > b ? a : b;
}

int maxPathSumValue(TreeNode *root, int *answer) {
    if (root == NULL) {
        return 0;
    }

    int left = maxPathSumValue(root->left, answer);
    int right = maxPathSumValue(root->right, answer);

    if (left < 0) {
        left = 0;
    }

    if (right < 0) {
        right = 0;
    }

    int current = root->data + left + right;
    if (current > *answer) {
        *answer = current;
    }

    return root->data + maxValue(left, right);
}
```

## ⑤ 最近公共祖先 LCA

如果 `p` 和 `q` 分别出现在当前结点的左右子树，那么当前结点就是最近公共祖先。

```c
TreeNode *lowestCommonAncestor(TreeNode *root, TreeNode *p, TreeNode *q) {
    if (root == NULL || root == p || root == q) {
        return root;
    }

    TreeNode *left = lowestCommonAncestor(root->left, p, q);
    TreeNode *right = lowestCommonAncestor(root->right, p, q);

    if (left != NULL && right != NULL) {
        return root;
    }

    return left != NULL ? left : right;
}
```

## ⑥ 重点 / 注意 / 常见错误

| 题型 | 关键点 |
|---|---|
| 根到叶路径 | 必须到叶子才算一条完整路径 |
| 所有路径 | DFS + 回溯 |
| 路径总和 | 递归传递剩余目标值 |
| 最大路径和 | 返回贡献值，更新全局答案 |
| LCA | 左右子树返回值合并 |

错误理解：

```text
路径总和只要中间某一段等于 target 就可以。
```

正确理解：

```text
如果题目说根到叶路径，就必须从根开始，到叶子结束。
```
