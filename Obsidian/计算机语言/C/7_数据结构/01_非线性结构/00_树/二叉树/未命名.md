# 二叉树基础

## 1. 什么是二叉树

**二叉树（Binary Tree）** 是一种树形数据结构，其中每个结点最多有两个孩子：

- 左孩子
- 右孩子

二叉树可以为空，也可以由一个根结点和两棵互不相交的子树组成：

- 左子树
- 右子树

```text
        A
       / \
      B   C
     / \
    D   E
```

## 2. 二叉树的基本概念

| 概念 | 含义 |
|---|---|
| 根结点 | 树的起始结点 |
| 父结点 | 拥有子结点的结点 |
| 子结点 | 某个结点的直接后继 |
| 叶子结点 | 没有子结点的结点 |
| 度 | 一个结点拥有的子树数量 |
| 层次 | 根结点为第 1 层，向下递增 |
| 高度 / 深度 | 树中结点的最大层数 |
| 左子树 | 根结点左边的子树 |
| 右子树 | 根结点右边的子树 |

## 3. 二叉树的性质

### 性质 1：第 i 层最多有多少个结点

二叉树第 `i` 层最多有：

```text
2^(i - 1)
```

个结点。

例如：

| 层数 | 最多结点数 |
|---|---|
| 第 1 层 | 1 |
| 第 2 层 | 2 |
| 第 3 层 | 4 |
| 第 4 层 | 8 |

### 性质 2：深度为 k 的二叉树最多有多少个结点

深度为 `k` 的二叉树最多有：

```text
2^k - 1
```

个结点。

### 性质 3：叶子结点与度为 2 的结点关系

对于任意一棵非空二叉树，如果：

- `n0` 表示叶子结点数
- `n2` 表示度为 2 的结点数

则有：

```text
n0 = n2 + 1
```

## 4. 特殊二叉树

### 4.1 满二叉树

如果一棵深度为 `k` 的二叉树有 `2^k - 1` 个结点，则称为**满二叉树**。

```text
        A
       / \
      B   C
     / \ / \
    D  E F  G
```

特点：

- 每一层的结点数都达到最大值
- 所有叶子结点都在最后一层

### 4.2 完全二叉树

如果一棵二叉树除了最后一层外，其余层都是满的，并且最后一层的结点从左到右连续排列，则称为**完全二叉树**。

```text
        A
       / \
      B   C
     / \ /
    D  E F
```

特点：

- 结点编号可以和数组下标建立关系
- 堆通常使用完全二叉树实现

### 4.3 二叉搜索树

**二叉搜索树（Binary Search Tree，BST）** 是一种特殊二叉树，满足：

- 左子树所有结点值小于根结点
- 右子树所有结点值大于根结点
- 左右子树也都是二叉搜索树

```text
        8
       / \
      3   10
     / \    \
    1   6    14
```

## 5. 二叉树的存储方式

## 5.1 顺序存储

顺序存储通常使用数组，适合存储**完全二叉树**。

如果根结点下标从 `1` 开始：

| 结点 | 下标关系 |
|---|---|
| 父结点 | `i / 2` |
| 左孩子 | `2 * i` |
| 右孩子 | `2 * i + 1` |

如果根结点下标从 `0` 开始：

| 结点 | 下标关系 |
|---|---|
| 父结点 | `(i - 1) / 2` |
| 左孩子 | `2 * i + 1` |
| 右孩子 | `2 * i + 2` |

## 5.2 链式存储

普通二叉树更常用链式存储。

```c
typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;
```

其中：

- `data`：结点数据
- `left`：指向左孩子
- `right`：指向右孩子

## 6. 二叉树的遍历

二叉树遍历是指按照某种规则访问树中所有结点。

常见遍历方式：

| 遍历方式 | 访问顺序 |
|---|---|
| 前序遍历 | 根 → 左 → 右 |
| 中序遍历 | 左 → 根 → 右 |
| 后序遍历 | 左 → 右 → 根 |
| 层序遍历 | 从上到下、从左到右 |

假设二叉树为：

```text
        A
       / \
      B   C
     / \
    D   E
```

遍历结果：

| 遍历方式 | 结果 |
|---|---|
| 前序遍历 | A B D E C |
| 中序遍历 | D B E A C |
| 后序遍历 | D E B C A |
| 层序遍历 | A B C D E |

## 7. 递归遍历代码

### 7.1 前序遍历

```c
void preorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}
```

### 7.2 中序遍历

```c
void inorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}
```

### 7.3 后序遍历

```c
void postorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->data);
}
```

## 8. 创建二叉树示例

下面使用递归方式创建一棵简单二叉树。

输入规则：

- 输入整数表示结点值
- 输入 `-1` 表示空结点

例如输入：

```text
1 2 -1 -1 3 -1 -1
```

表示：

```text
    1
   / \
  2   3
```

代码：

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

TreeNode *createTree() {
    int value;
    scanf("%d", &value);

    if (value == -1) {
        return NULL;
    }

    TreeNode *node = (TreeNode *)malloc(sizeof(TreeNode));
    node->data = value;
    node->left = createTree();
    node->right = createTree();

    return node;
}

void preorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}

int main() {
    TreeNode *root = createTree();

    preorder(root);

    return 0;
}
```

## 9. 常见操作

### 9.1 统计结点个数

```c
int countNodes(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    return 1 + countNodes(root->left) + countNodes(root->right);
}
```

### 9.2 统计叶子结点个数

```c
int countLeaves(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    if (root->left == NULL && root->right == NULL) {
        return 1;
    }

    return countLeaves(root->left) + countLeaves(root->right);
}
```

### 9.3 求二叉树高度

```c
int height(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    int leftHeight = height(root->left);
    int rightHeight = height(root->right);

    return leftHeight > rightHeight ? leftHeight + 1 : rightHeight + 1;
}
```
