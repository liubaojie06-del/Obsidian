# 02 | 二叉树遍历：前序、中序、后序、层序

> [!abstract] 核心结论  
> 二叉树遍历就是按照某种顺序访问所有结点。前序、中序、后序属于 DFS；层序遍历属于 BFS，通常使用队列实现。

## ① 底层原理

| 遍历方式 | 访问顺序 | 常用实现 |
|---|---|---|
| 前序遍历 | 根 → 左 → 右 | 递归 / 栈 |
| 中序遍历 | 左 → 根 → 右 | 递归 / 栈 |
| 后序遍历 | 左 → 右 → 根 | 递归 / 栈 |
| 层序遍历 | 从上到下，从左到右 | 队列 |

示例二叉树：

```text
        1
       / \
      2   3
     / \
    4   5
```

遍历结果：

| 遍历方式 | 结果 |
|---|---|
| 前序 | `1 2 4 5 3` |
| 中序 | `4 2 5 1 3` |
| 后序 | `4 5 2 3 1` |
| 层序 | `1 2 3 4 5` |

## ② 递归遍历代码

```c
#include <stdio.h>

typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

void preorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}

void inorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    inorder(root->left);
    printf("%d ", root->data);
    inorder(root->right);
}

void postorder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    postorder(root->left);
    postorder(root->right);
    printf("%d ", root->data);
}
```

## ③ 层序遍历代码

```c
#define MAX_SIZE 100

void levelOrder(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    TreeNode *queue[MAX_SIZE];
    int front = 0;
    int rear = 0;

    queue[rear++] = root;

    while (front < rear) {
        TreeNode *current = queue[front++];
        printf("%d ", current->data);

        if (current->left != NULL) {
            queue[rear++] = current->left;
        }

        if (current->right != NULL) {
            queue[rear++] = current->right;
        }
    }
}
```

## ④ 迭代前序遍历

```c
#define MAX_SIZE 100

void preorderIter(TreeNode *root) {
    if (root == NULL) {
        return;
    }

    TreeNode *stack[MAX_SIZE];
    int top = -1;

    stack[++top] = root;

    while (top != -1) {
        TreeNode *current = stack[top--];
        printf("%d ", current->data);

        if (current->right != NULL) {
            stack[++top] = current->right;
        }

        if (current->left != NULL) {
            stack[++top] = current->left;
        }
    }
}
```

> [!note] 代码备注  
> 栈是后进先出，所以前序遍历迭代版要先压右孩子，再压左孩子。

## ⑤ 重点 / 注意 / 常见错误

| 问题 | 关键点 |
|---|---|
| 前序 | 根在前 |
| 中序 | 根在中 |
| 后序 | 根在后 |
| 层序 | 队列实现 |
| 递归遍历 | 一定先判断 `root == NULL` |
| 迭代遍历 | 通常用栈模拟递归 |

错误写法：

```c
void preorder(TreeNode *root) {
    printf("%d ", root->data);
    preorder(root->left);
    preorder(root->right);
}
```

问题是：没有判断 `root == NULL`，会出现空指针访问。

> [!tip] 记忆口诀  
> **前中后看根，层序靠队列。**
