# 04 | 二叉树属性：深度、高度、结点数、平衡判断

> [!abstract] 核心结论  
> 二叉树属性题大多可以用递归解决。核心套路是：空树返回基础值，非空树根据左右子树结果合并答案。

## ① 底层原理

| 属性 | 含义 |
|---|---|
| 最大深度 | 根到最深叶子结点的层数 |
| 最小深度 | 根到最近叶子结点的层数 |
| 结点数量 | 整棵树有多少个结点 |
| 叶子数量 | 没有左右孩子的结点数量 |
| 树高 | 通常等同最大深度 |
| 是否平衡 | 任意结点左右子树高度差不超过 1 |
| 路径长度 | 从根到某结点经过的边数或结点数 |

递归通用框架：

```c
if (root == NULL) {
    return 基础值;
}

左结果 = 递归(root->left);
右结果 = 递归(root->right);
return 合并结果;
```

## ② 最大深度 / 树高

```c
typedef struct TreeNode {
    int data;
    struct TreeNode *left;
    struct TreeNode *right;
} TreeNode;

int maxDepth(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    int leftDepth = maxDepth(root->left);
    int rightDepth = maxDepth(root->right);

    return leftDepth > rightDepth ? leftDepth + 1 : rightDepth + 1;
}
```

含义：

```text
最大深度 = max(左子树深度, 右子树深度) + 1
```

## ③ 最小深度

```c
int minDepth(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    if (root->left == NULL && root->right == NULL) {
        return 1;
    }

    if (root->left == NULL) {
        return minDepth(root->right) + 1;
    }

    if (root->right == NULL) {
        return minDepth(root->left) + 1;
    }

    int leftDepth = minDepth(root->left);
    int rightDepth = minDepth(root->right);

    return leftDepth < rightDepth ? leftDepth + 1 : rightDepth + 1;
}
```

> [!note] 代码备注  
> 最小深度必须到叶子结点才算结束。如果一个结点只有一边子树，不能把空子树深度 `0` 当成最小值。

## ④ 结点数量与叶子数量

```c
int countNodes(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    return 1 + countNodes(root->left) + countNodes(root->right);
}

int countLeaves(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    if (root->left == NULL && root->right == NULL) {
        return 1;
    }

    return countLeaves(root->left) + countLeaves(root->right);
}
```

## ⑤ 平衡二叉树判断

```c
int checkHeight(TreeNode *root) {
    if (root == NULL) {
        return 0;
    }

    int leftHeight = checkHeight(root->left);
    if (leftHeight == -1) {
        return -1;
    }

    int rightHeight = checkHeight(root->right);
    if (rightHeight == -1) {
        return -1;
    }

    int diff = leftHeight - rightHeight;
    if (diff < 0) {
        diff = -diff;
    }

    if (diff > 1) {
        return -1;
    }

    return leftHeight > rightHeight ? leftHeight + 1 : rightHeight + 1;
}

int isBalanced(TreeNode *root) {
    return checkHeight(root) != -1;
}
```

## ⑥ 重点 / 注意 / 常见错误

| 属性题 | 递归返回值 |
|---|---|
| 最大深度 | 当前树最大层数 |
| 最小深度 | 到最近叶子的层数 |
| 结点数量 | 当前树结点总数 |
| 叶子数量 | 当前树叶子总数 |
| 平衡判断 | 高度或不平衡标记 |

最小深度常见错误：

```c
return min(leftDepth, rightDepth) + 1;
```

问题是：如果一边为空，`0` 不能代表到叶子的有效路径。

> [!tip] 记忆口诀  
> **属性看左右，递归往上收。**
