# 01 | C 语言运算符：对数据进行计算和判断的符号

> [!abstract] 核心结论
> 运算符是 C 语言中用来操作数据的符号，例如加减乘除、比较大小、逻辑判断、赋值、自增自减、位运算等。表达式的结果取决于运算符、操作数、优先级和结合性。

### ① 底层原理：运算符到底做了什么

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #2C3E50; color: #333; margin-bottom: 10px;">
<strong>运算符的本质：</strong>
<ul>
  <li><strong>操作数：</strong> 被运算的数据，例如 <code>a + b</code> 中的 <code>a</code> 和 <code>b</code>。</li>
  <li><strong>运算符：</strong> 对数据执行操作的符号，例如 <code>+</code>、<code>-</code>、<code>*</code>、<code>/</code>。</li>
  <li><strong>表达式：</strong> 由操作数和运算符组成，并产生一个结果。</li>
  <li><strong>优先级：</strong> 决定哪个运算先执行，例如 <code>*</code> 比 <code>+</code> 先执行。</li>
  <li><strong>结合性：</strong> 当优先级相同时，决定从左往右算，还是从右往左算。</li>
</ul>
</div>

### ② 运算符运行时的内存理解

示例：

```c
int a = 10;
int b = 3;
int c = a + b;
```

运行时可以理解为：

```text
1. 栈区创建变量 a，值为 10。
2. 栈区创建变量 b，值为 3。
3. CPU 从内存中读取 a 和 b 的值。
4. CPU 执行加法运算，得到 13。
5. 栈区创建变量 c。
6. 把结果 13 存入 c。
```

内存示意：

```text
栈区 Stack
┌──────────────┐
│ a = 10        │
│ b = 3         │
│ c = 13        │
└──────────────┘
```

> [!tip] 一句话理解
> **变量在内存里存值，运算符让 CPU 取出这些值进行计算，再把结果写回内存。**

# 02 | 算术运算符：加减乘除取余

> [!abstract] 核心结论
> 算术运算符用于数值计算，包括加、减、乘、除、取余。整数除法会舍弃小数部分，取余只能用于整数。

### ① 基本运算符

| 运算符 | 含义 | 示例 |
| :--- | :--- | :--- |
| `+` | 加法 | `a + b` |
| `-` | 减法 | `a - b` |
| `*` | 乘法 | `a * b` |
| `/` | 除法 | `a / b` |
| `%` | 取余 | `a % b` |

### ② 示例代码

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 3;

    printf("%d\n", a + b);
    printf("%d\n", a - b);
    printf("%d\n", a * b);
    printf("%d\n", a / b);
    printf("%d\n", a % b);

    return 0;
}
```

输出：

```text
13
7
30
3
1
```

### ③ 整数除法

```c
int a = 10;
int b = 3;

printf("%d\n", a / b);
```

输出：

```text
3
```

说明：

```text
两个整数相除，结果仍然是整数。
小数部分会被直接舍弃。
```

想得到小数：

```c
int a = 10;
int b = 3;

printf("%f\n", a * 1.0 / b);
```

输出：

```text
3.333333
```

### ④ 取余运算

```c
int a = 10;
int b = 3;

printf("%d\n", a % b);
```

输出：

```text
1
```

说明：

```text
10 除以 3，商是 3，余数是 1。
```

常见用途：

```text
判断奇偶：
num % 2 == 0  表示偶数
num % 2 != 0  表示奇数
```

示例：

```c
#include <stdio.h>

int main() {
    int num = 7;

    if (num % 2 == 0) {
        printf("偶数\n");
    } else {
        printf("奇数\n");
    }

    return 0;
}
```

# 03 | 赋值运算符：把结果存进变量

> [!abstract] 核心结论
> 赋值运算符用于把右边表达式的结果写入左边变量。赋值不是“相等”，而是“把右边的值存到左边的内存空间”。

### ① 基本赋值

```c
int a;
a = 10;
```

理解：

```text
把 10 存入变量 a 对应的内存空间。
```

### ② 连续赋值

```c
int a, b, c;

a = b = c = 10;
```

理解：

```text
赋值运算符从右向左结合。
先 c = 10。
再 b = c。
最后 a = b。
```

结果：

```text
a = 10
b = 10
c = 10
```

### ③ 复合赋值运算符

| 运算符 | 等价写法 |
| :--- | :--- |
| `a += b` | `a = a + b` |
| `a -= b` | `a = a - b` |
| `a *= b` | `a = a * b` |
| `a /= b` | `a = a / b` |
| `a %= b` | `a = a % b` |

示例：

```c
#include <stdio.h>

int main() {
    int a = 10;

    a += 5;
    printf("%d\n", a);

    a *= 2;
    printf("%d\n", a);

    return 0;
}
```

输出：

```text
15
30
```

> [!tip] 记忆口诀
> **赋值看右边，结果存左边；等号不是相等，是写入内存。**

# 04 | 自增自减运算符：让变量自己加一或减一

> [!abstract] 核心结论
> `++` 和 `--` 用于让变量加一或减一。前置形式先变化再使用，后置形式先使用再变化。

### ① 基本用法

```c
int a = 10;

a++;
++a;

a--;
--a;
```

含义：

```text
a++  等价于 a = a + 1
++a  等价于 a = a + 1
a--  等价于 a = a - 1
--a  等价于 a = a - 1
```

### ② 前置和后置区别

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = ++a;

    printf("a = %d\n", a);
    printf("b = %d\n", b);

    return 0;
}
```

输出：

```text
a = 11
b = 11
```

说明：

```text
++a：先让 a 加 1，再把新值给 b。
```

---

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = a++;

    printf("a = %d\n", a);
    printf("b = %d\n", b);

    return 0;
}
```

输出：

```text
a = 11
b = 10
```

说明：

```text
a++：先把 a 原来的值给 b，再让 a 加 1。
```

### ③ 不推荐复杂写法

不推荐：

```c
int a = 1;
int b = a++ + ++a;
```

原因：

```text
这种写法可读性差，而且涉及求值顺序问题。
不同情况可能产生难以理解的结果。
```

推荐：

```c
int a = 1;

a++;
++a;

int b = a;
```

> [!tip] 记忆口诀
> **前加先变，后加后变；写代码别炫技，清楚永远第一。**

# 05 | 关系运算符：比较两个数据的大小关系

> [!abstract] 核心结论
> 关系运算符用于比较两个值，结果只有两种：真或假。在 C 语言中，真通常用 `1` 表示，假用 `0` 表示。

### ① 基本关系运算符

| 运算符 | 含义 |
| :--- | :--- |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |
| `==` | 等于 |
| `!=` | 不等于 |

### ② 示例代码

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;

    printf("%d\n", a > b);
    printf("%d\n", a < b);
    printf("%d\n", a == b);
    printf("%d\n", a != b);

    return 0;
}
```

输出：

```text
0
1
0
1
```

### ③ 等于和赋值不要混淆

错误写法：

```c
if (a = 10) {
    printf("成立\n");
}
```

问题：

```text
a = 10 是赋值，不是判断相等。
赋值表达式的结果是 10，非 0 为真。
所以条件会成立。
```

正确写法：

```c
if (a == 10) {
    printf("成立\n");
}
```

> [!tip] 记忆口诀
> **一个等号是赋值，两个等号才比较。**

# 06 | 逻辑运算符：组合多个条件

> [!abstract] 核心结论
> 逻辑运算符用于连接多个条件，常用于 `if`、`while`、`for` 等控制语句。C 语言中，0 表示假，非 0 表示真。

### ① 基本逻辑运算符
| 运算符  | 含义     | 示例               |     |     |     |     |     |
| :--- | :----- | :--------------- | --- | --- | --- | --- | --- |
| `&&` | 逻辑与，并且 | `a > 0 && b > 0` |     |     |     |     |     |
| \|\| | 逻辑或，或者 |  a > 0 \|\| > 0  |     |     |     |     |     |
| `!`  | 逻辑非，取反 | `!flag`          |     |     |     |     |     |
|      |        |                  |     |     |     |     |     |
### ② 逻辑与 &&

```c
#include <stdio.h>

int main() {
    int age = 20;
    int score = 90;

    if (age >= 18 && score >= 60) {
        printf("通过\n");
    }

    return 0;
}
```

说明：

```text
&& 两边都为真，整体才为真。
```

### ③ 逻辑或 ||

```c
#include <stdio.h>

int main() {
    int vip = 1;
    int coupon = 0;

    if (vip || coupon) {
        printf("可以优惠\n");
    }

    return 0;
}
```

说明：

```text
|| 只要有一边为真，整体就为真。
```

### ④ 逻辑非 !

```c
#include <stdio.h>

int main() {
    int flag = 0;

    if (!flag) {
        printf("flag 是假\n");
    }

    return 0;
}
```

输出：

```text
flag 是假
```

### ⑤ 短路求值

```c
int a = 0;
int b = 10;

if (a != 0 && b / a > 1) {
    printf("成立\n");
}
```

说明：

```text
&& 左边为假时，右边不会再执行。
所以 b / a 不会发生除以 0。
```

再看：

```c
int a = 1;
int b = 0;

if (a == 1 || b++ > 0) {
    printf("成立\n");
}

printf("%d\n", b);
```

输出：

```text
成立
0
```

说明：

```text
|| 左边为真时，右边不会再执行。
所以 b++ 没有执行。
```

> [!tip] 记忆口诀
> **与要全真，或要一真；非会取反，短路会偷懒。**

# 07 | 条件运算符：三目表达式

> [!abstract] 核心结论
> 条件运算符 `?:` 是 C 语言中唯一的三目运算符，可以根据条件在两个结果中选择一个。

### ① 基本语法

```c
条件 ? 表达式1 : 表达式2
```

执行逻辑：

```text
如果条件为真，结果是表达式1。
如果条件为假，结果是表达式2。
```

### ② 示例代码

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;

    int max = a > b ? a : b;

    printf("%d\n", max);

    return 0;
}
```

输出：

```text
20
```

等价于：

```c
int max;

if (a > b) {
    max = a;
} else {
    max = b;
}
```

### ③ 判断奇偶

```c
#include <stdio.h>

int main() {
    int num = 7;

    printf("%s\n", num % 2 == 0 ? "偶数" : "奇数");

    return 0;
}
```

输出：

```text
奇数
```

> [!tip] 记忆口诀
> **问号前面放条件，冒号两边选答案。**

# 08 | 位运算符：直接操作二进制位

> [!abstract] 核心结论
> 位运算符直接操作整数的二进制位，常用于底层开发、权限控制、状态标记、嵌入式开发和高性能计算。

### ① 基本位运算符

| 运算符  | 含义   |     |
| :--- | :--- | --- |
| `&`  | 按位与  |     |
| \|   | 按位或  |     |
| `^`  | 按位异或 |     |
| `~`  | 按位取反 |     |
| `<<` | 左移   |     |
| `>>` | 右移   |     |

### ② 按位与 &

```c
int a = 6;  // 二进制 110
int b = 3;  // 二进制 011

int c = a & b;
```

结果：

```text
110
011
---
010
```

所以：

```text
c = 2
```

常见用途：

```text
判断某一位是否为 1。
```

### ③ 按位或 |

```c
int a = 6;  // 110
int b = 3;  // 011

int c = a | b;
```

结果：

```text
110
011
---
111
```

所以：

```text
c = 7
```

常见用途：

```text
把某一位设置为 1。
```

### ④ 按位异或 ^

```c
int a = 6;  // 110
int b = 3;  // 011

int c = a ^ b;
```

结果：

```text
110
011
---
101
```

所以：

```text
c = 5
```

规则：

```text
相同为 0，不同为 1。
```

### ⑤ 左移 <<

```c
int a = 3;      // 二进制 0011
int b = a << 1; // 二进制 0110
```

结果：

```text
b = 6
```

理解：

```text
左移 1 位，在不溢出的情况下，相当于乘以 2。
```

### ⑥ 右移 >>

```c
int a = 8;      // 二进制 1000
int b = a >> 1; // 二进制 0100
```

结果：

```text
b = 4
```

理解：

```text
右移 1 位，在常见情况下，相当于除以 2。
```

# 09 | 逗号运算符、sizeof、强制类型转换

> [!abstract] 核心结论
> C 语言还有一些特殊运算符，例如逗号运算符、`sizeof`、强制类型转换，它们经常出现在循环、内存计算和类型转换中。

### ① 逗号运算符

```c
int a = 1;
int b = 2;
int c;

c = (a++, b++, a + b);
```

执行逻辑：

```text
从左到右依次执行。
整个逗号表达式的结果是最后一个表达式的值。
```

结果：

```text
a = 2
b = 3
c = 5
```

### ② sizeof 运算符

```c
#include <stdio.h>

int main() {
    int a = 10;

    printf("%zu\n", sizeof(a));
    printf("%zu\n", sizeof(int));

    return 0;
}
```

说明：

```text
sizeof 用来计算变量或类型占多少字节。
sizeof 不是函数，是运算符。
```

### ③ 强制类型转换

```c
int a = 10;
int b = 3;

double result = (double)a / b;
```

说明：

```text
(double)a 把 a 临时转换成 double。
这样除法结果就是小数。
```

输出示例：

```text
3.333333
```

# 10 | 运算符优先级、常见错误与总结

> [!abstract] 核心结论
> 运算符优先级决定表达式中谁先算。写复杂表达式时，不要过度依赖记忆，直接加括号最稳。

### ① 常见优先级顺序

从高到低大致记忆：

```text
1. 括号：()
2. 自增自减、逻辑非、sizeof：++ -- ! sizeof
3. 乘除取余：* / %
4. 加减：+ -
5. 移位：<< >>
6. 关系比较：< <= > >=
7. 相等比较：== !=
8. 位与：&
9. 位异或：^
10. 位或：|
11. 逻辑与：&&
12. 逻辑或：||
13. 条件运算：?:
14. 赋值：= += -= *= /= %=
15. 逗号：,
```

### ② 加括号是最稳写法

不推荐：

```c
if (a + b > c && d || e)
```

推荐：

```c
if (((a + b) > c) && (d || e))
```

说明：

```text
括号能让表达式的含义更清晰。
尤其适合初学阶段和复杂判断。
```

### ③ 常见错误

**错误 1：把赋值当比较**

```c
if (a = 10)
```

正确：

```c
if (a == 10)
```

---

**错误 2：整数除法误以为有小数**

```c
int a = 10;
int b = 3;

printf("%f\n", a / b);
```

正确：

```c
printf("%f\n", a * 1.0 / b);
```

---

**错误 3：取余用于小数**

错误：

```c
double x = 5.5;
double y = 2.0;

printf("%f\n", x % y);
```

说明：

```text
% 只能用于整数，不能用于小数。
```

---

**错误 4：复杂自增表达式**

不推荐：

```c
int b = a++ + ++a;
```

推荐：

```c
a++;
++a;
int b = a;
```

---

**错误 5：逻辑运算和位运算混淆**

```c
a && b   // 逻辑与，判断真假
a & b    // 按位与，操作二进制位
```

### ④ 运算符总结

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #27AE60; color: #333; margin-bottom: 10px;">
<strong>C 语言运算符核心总结：</strong>
<ul>
  <li><strong>算术运算符：</strong> <code>+</code>、<code>-</code>、<code>*</code>、<code>/</code>、<code>%</code>。</li>
  <li><strong>赋值运算符：</strong> <code>=</code>、<code>+=</code>、<code>-=</code>、<code>*=</code>。</li>
  <li><strong>关系运算符：</strong> <code>&gt;</code>、<code>&lt;</code>、<code>==</code>、<code>!=</code>。</li>
  <li><strong>逻辑运算符：</strong> <code>&amp;&amp;</code>、<code>||</code>、<code>!</code>。</li>
  <li><strong>位运算符：</strong> <code>&amp;</code>、<code>|</code>、<code>^</code>、<code>~</code>、<code>&lt;&lt;</code>、<code>&gt;&gt;</code>。</li>
  <li><strong>特殊运算符：</strong> <code>sizeof</code>、<code>?:</code>、<code>,</code>、强制类型转换。</li>
</ul>
</div>

> [!tip] 记忆口诀
> **加减乘除算数值，大小等号做判断；与或非管真假，位运算看二进制；优先级记不牢，括号一加最可靠。**
