# 01 | C 语言数组：一组相同类型数据的连续空间

> [!abstract] 核心结论
> 数组是用来存放一组相同类型数据的容器。它在内存中通常是一段连续空间，通过下标访问元素，下标从 `0` 开始。

### ① 底层原理：数组在内存中怎么存

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #2C3E50; color: #333; margin-bottom: 10px;">
<strong>数组的本质：</strong>
<ul>
  <li><strong>数组是一段连续内存：</strong> 多个相同类型的数据挨着存放。</li>
  <li><strong>数组名代表起始位置：</strong> 在大多数表达式中，数组名会退化为首元素地址。</li>
  <li><strong>下标从 0 开始：</strong> 第一个元素是 <code>arr[0]</code>，不是 <code>arr[1]</code>。</li>
  <li><strong>元素类型统一：</strong> 一个 <code>int</code> 数组只能放 <code>int</code> 类型数据。</li>
  <li><strong>数组大小固定：</strong> 普通数组定义后，长度通常不能随意改变。</li>
</ul>
</div>

### ② 数组内存图

示例：

```c
int arr[5] = {10, 20, 30, 40, 50};
```

内存理解：

```text
数组名：arr
元素类型：int
元素个数：5
每个 int 常见占 4 字节
整个数组常见占 5 × 4 = 20 字节
```

内存示意：

```text
栈区 Stack

arr[0]     arr[1]     arr[2]     arr[3]     arr[4]
┌────────┬────────┬────────┬────────┬────────┐
│   10   │   20   │   30   │   40   │   50   │
└────────┴────────┴────────┴────────┴────────┘
   ↑
   arr 数组起始地址
```

访问规律：

```text
arr[0] 表示第 1 个元素
arr[1] 表示第 2 个元素
arr[2] 表示第 3 个元素
arr[3] 表示第 4 个元素
arr[4] 表示第 5 个元素
```

> [!tip] 一句话理解
> **数组就是一排连续的小格子，每个格子存一个同类型数据，下标就是格子的编号。**

# 02 | 一维数组：最基础的数据列表

> [!abstract] 核心结论
> 一维数组适合存放一列数据，例如成绩列表、年龄列表、编号列表。定义数组时需要指定类型和元素个数。

### ① 基本语法

```c
数据类型 数组名[元素个数];
```

示例：

```c
int nums[5];
double scores[10];
char chars[20];
```

说明：

```text
int nums[5] 表示定义一个 int 数组。
这个数组有 5 个元素。
下标范围是 0 到 4。
```

### ② 定义并初始化

```c
int arr[5] = {10, 20, 30, 40, 50};
```

也可以省略长度：

```c
int arr[] = {10, 20, 30, 40, 50};
```

说明：

```text
省略长度时，编译器会根据初始化数据个数自动推导数组长度。
```

### ③ 部分初始化

```c
int arr[5] = {10, 20};
```

实际结果：

```text
arr[0] = 10
arr[1] = 20
arr[2] = 0
arr[3] = 0
arr[4] = 0
```

说明：

```text
数组部分初始化时，剩余元素自动初始化为 0。
```

### ④ 全部初始化为 0

```c
int arr[5] = {0};
```

说明：

```text
这是常用写法，表示把数组所有元素初始化为 0。
```

# 03 | 数组元素访问与修改

> [!abstract] 核心结论
> 数组通过下标访问元素。读取数组元素就是从指定位置取值，修改数组元素就是向指定位置写入新值。

### ① 访问数组元素

```c
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    printf("%d\n", arr[0]);
    printf("%d\n", arr[2]);
    printf("%d\n", arr[4]);

    return 0;
}
```

输出：

```text
10
30
50
```

### ② 修改数组元素

```c
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    arr[0] = 100;
    arr[3] = 400;

    printf("%d\n", arr[0]);
    printf("%d\n", arr[3]);

    return 0;
}
```

输出：

```text
100
400
```

### ③ 下标从 0 开始

```text
int arr[5];

合法下标：
arr[0]
arr[1]
arr[2]
arr[3]
arr[4]

非法下标：
arr[5]
arr[6]
arr[-1]
```

注意：

```text
C 语言不会自动帮你检查数组越界。
越界访问可能导致数据异常、程序崩溃或隐藏 bug。
```

# 04 | 数组遍历：用循环处理每个元素

> [!abstract] 核心结论
> 数组通常配合 `for` 循环使用。数组下标从 `0` 开始，所以循环条件常写成 `i < len`。

### ① 遍历数组

```c
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    for (int i = 0; i < 5; i++) {
        printf("%d\n", arr[i]);
    }

    return 0;
}
```

输出：

```text
10
20
30
40
50
```

### ② 使用 sizeof 计算数组长度

```c
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};

    int len = sizeof(arr) / sizeof(arr[0]);

    for (int i = 0; i < len; i++) {
        printf("%d\n", arr[i]);
    }

    return 0;
}
```

长度公式：

```text
数组长度 = 数组总字节数 / 单个元素字节数
```

对应代码：

```c
int len = sizeof(arr) / sizeof(arr[0]);
```

### ③ 数组求和

```c
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int len = sizeof(arr) / sizeof(arr[0]);
    int sum = 0;

    for (int i = 0; i < len; i++) {
        sum += arr[i];
    }

    printf("sum = %d\n", sum);

    return 0;
}
```

输出：

```text
sum = 150
```

### ④ 找最大值

```c
#include <stdio.h>

int main() {
    int arr[] = {12, 45, 7, 89, 23};
    int len = sizeof(arr) / sizeof(arr[0]);

    int max = arr[0];

    for (int i = 1; i < len; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }

    printf("max = %d\n", max);

    return 0;
}
```

输出：

```text
max = 89
```

> [!tip] 记忆口诀
> **数组遍历从零起，小于长度最稳妥；求和累加，最大先拿第一个。**

# 05 | 数组输入：用 scanf 给数组赋值

> [!abstract] 核心结论
> 数组输入通常用循环配合 `scanf`。普通数组元素是变量，所以输入时需要写 `&arr[i]`。

### ① 输入 5 个整数

```c
#include <stdio.h>

int main() {
    int arr[5];

    for (int i = 0; i < 5; i++) {
        scanf("%d", &arr[i]);
    }

    for (int i = 0; i < 5; i++) {
        printf("%d\n", arr[i]);
    }

    return 0;
}
```

输入：

```text
10 20 30 40 50
```

输出：

```text
10
20
30
40
50
```

### ② 为什么要加 &

```c
scanf("%d", &arr[i]);
```

理解：

```text
arr[i] 是第 i 个元素。
&arr[i] 是第 i 个元素的地址。
scanf 需要地址，才能把输入数据写进去。
```

### ③ 输入时检查返回值

```c
#include <stdio.h>

int main() {
    int arr[5];

    for (int i = 0; i < 5; i++) {
        if (scanf("%d", &arr[i]) != 1) {
            printf("输入错误\n");
            return 1;
        }
    }

    printf("输入完成\n");

    return 0;
}
```

说明：

```text
scanf 成功读取一个整数时返回 1。
检查返回值可以避免非法输入导致程序异常。
```

# 06 | 字符数组与字符串

> [!abstract] 核心结论
> C 语言没有专门的字符串类型，字符串通常用字符数组保存。字符串末尾会有隐藏的 `'\0'`，表示字符串结束。

### ① 字符数组

```c
char str[5] = {'H', 'e', 'l', 'l', 'o'};
```

说明：

```text
这是普通字符数组，不一定是标准字符串。
因为末尾没有 '\0'。
```

### ② 字符串数组

```c
char str[] = "Hello";
```

实际存储：

```text
'H' 'e' 'l' 'l' 'o' '\0'
```

所以：

```c
char str[] = "Hello";
```

数组长度是：

```text
6
```

不是：

```text
5
```

### ③ 输出字符串

```c
#include <stdio.h>

int main() {
    char name[] = "Tom";

    printf("%s\n", name);

    return 0;
}
```

输出：

```text
Tom
```

### ④ 输入字符串

```c
#include <stdio.h>

int main() {
    char name[20];

    scanf("%19s", name);

    printf("%s\n", name);

    return 0;
}
```

说明：

```text
name 是数组名，本身表示首元素地址。
所以 scanf("%s", name) 不需要加 &。
%19s 表示最多读取 19 个字符，给 '\0' 留 1 个位置。
```

### ⑤ 字符数组和字符串区别

```text
char arr[5] = {'H','e','l','l','o'};
不是标准字符串，因为没有 '\0'。

char str[] = "Hello";
是标准字符串，末尾自动有 '\0'。
```

> [!tip] 记忆口诀
> **字符数组存字符，字符串多零尾；`%s` 遇零停止，输入限宽防越界。**

# 07 | 二维数组：表格型数据

> [!abstract] 核心结论
> 二维数组可以理解为“表格”，由行和列组成。它适合存放矩阵、成绩表、棋盘等数据。

### ① 基本语法

```c
数据类型 数组名[行数][列数];
```

示例：

```c
int matrix[3][4];
```

理解：

```text
matrix 是一个 3 行 4 列的二维数组。
一共有 3 × 4 = 12 个元素。
```

### ② 定义并初始化

```c
int arr[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```

内存逻辑：

```text
第 0 行：1 2 3
第 1 行：4 5 6
```

访问：

```c
arr[0][0]  // 第 0 行第 0 列，值为 1
arr[0][2]  // 第 0 行第 2 列，值为 3
arr[1][0]  // 第 1 行第 0 列，值为 4
arr[1][2]  // 第 1 行第 2 列，值为 6
```

### ③ 遍历二维数组

```c
#include <stdio.h>

int main() {
    int arr[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", arr[i][j]);
        }

        printf("\n");
    }

    return 0;
}
```

输出：

```text
1 2 3
4 5 6
```

### ④ 二维数组内存本质

```text
二维数组逻辑上是表格。
但在内存中仍然是连续存放的。
C 语言按行优先顺序存储。
```

示例：

```c
int arr[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```

内存顺序：

```text
1 2 3 4 5 6
```

图示：

```text
逻辑结构：

arr[0][0] arr[0][1] arr[0][2]
arr[1][0] arr[1][1] arr[1][2]

内存顺序：

arr[0][0] → arr[0][1] → arr[0][2] → arr[1][0] → arr[1][1] → arr[1][2]
```

> [!tip] 记忆口诀
> **二维数组像表格，外层管行内层列；逻辑二维，内存连续。**

# 08 | 数组名、地址与指针关系

> [!abstract] 核心结论
> 在大多数表达式中，数组名会退化为首元素地址。`arr` 通常表示 `&arr[0]`，但 `sizeof(arr)` 和 `&arr` 是两个特殊情况。

### ① 数组名表示首元素地址

```c
#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};

    printf("%p\n", (void*)arr);
    printf("%p\n", (void*)&arr[0]);

    return 0;
}
```

说明：

```text
arr 和 &arr[0] 的地址值通常一样。
都表示数组第一个元素的位置。
```

### ② 数组名不是普通指针变量

```c
int arr[3] = {10, 20, 30};
```

注意：

```text
arr 可以代表首元素地址。
但 arr 不是一个可以被重新赋值的指针变量。
```

错误写法：

```c
arr = arr + 1;
```

原因：

```text
数组名不是普通变量，不能被赋值。
```

### ③ sizeof(arr) 不会退化

```c
#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};

    printf("%zu\n", sizeof(arr));
    printf("%zu\n", sizeof(arr[0]));

    return 0;
}
```

常见输出：

```text
12
4
```

说明：

```text
sizeof(arr) 得到整个数组大小。
不是指针大小。
```

### ④ 函数参数中数组会退化

```c
#include <stdio.h>

void test(int arr[]) {
    printf("%zu\n", sizeof(arr));
}

int main() {
    int nums[5] = {1, 2, 3, 4, 5};

    test(nums);

    return 0;
}
```

64 位环境常见输出：

```text
8
```

原因：

```text
函数参数 int arr[] 等价于 int *arr。
在函数内部，sizeof(arr) 得到的是指针大小，不是数组总大小。
```

正确做法：

```c
void print_array(int arr[], int len) {
    for (int i = 0; i < len; i++) {
        printf("%d\n", arr[i]);
    }
}
```

调用：

```c
int nums[5] = {1, 2, 3, 4, 5};
int len = sizeof(nums) / sizeof(nums[0]);

print_array(nums, len);
```

# 09 | 数组常见算法

> [!abstract] 核心结论
> 数组常见操作包括求和、求平均值、找最大最小值、查找元素、逆序输出、交换元素。这些是后续排序和查找算法的基础。

### ① 求平均值

```c
#include <stdio.h>

int main() {
    int arr[] = {80, 90, 75, 85, 100};
    int len = sizeof(arr) / sizeof(arr[0]);
    int sum = 0;

    for (int i = 0; i < len; i++) {
        sum += arr[i];
    }

    double avg = sum * 1.0 / len;

    printf("avg = %.2f\n", avg);

    return 0;
}
```

输出：

```text
avg = 86.00
```

### ② 找最小值

```c
#include <stdio.h>

int main() {
    int arr[] = {12, 45, 7, 89, 23};
    int len = sizeof(arr) / sizeof(arr[0]);

    int min = arr[0];

    for (int i = 1; i < len; i++) {
        if (arr[i] < min) {
            min = arr[i];
        }
    }

    printf("min = %d\n", min);

    return 0;
}
```

输出：

```text
min = 7
```

### ③ 查找元素

```c
#include <stdio.h>

int main() {
    int arr[] = {12, 45, 7, 89, 23};
    int len = sizeof(arr) / sizeof(arr[0]);
    int target = 89;
    int index = -1;

    for (int i = 0; i < len; i++) {
        if (arr[i] == target) {
            index = i;
            break;
        }
    }

    if (index != -1) {
        printf("找到了，下标是 %d\n", index);
    } else {
        printf("没找到\n");
    }

    return 0;
}
```

输出：

```text
找到了，下标是 3
```

### ④ 逆序输出

```c
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int len = sizeof(arr) / sizeof(arr[0]);

    for (int i = len - 1; i >= 0; i--) {
        printf("%d\n", arr[i]);
    }

    return 0;
}
```

输出：

```text
50
40
30
20
10
```

### ⑤ 交换两个元素

```c
int temp = arr[0];
arr[0] = arr[1];
arr[1] = temp;
```

理解：

```text
交换两个变量必须借助临时变量。
否则一个值会被覆盖掉。
```

# 10 | 数组常见错误与总结

> [!abstract] 核心结论
> 数组最常见的问题是下标越界、长度计算错误、函数参数退化成指针、字符串忘记 `'\0'`、输入字符串不限制宽度。

### ① 常见错误

**错误 1：数组越界**

```c
int arr[5] = {1, 2, 3, 4, 5};

printf("%d\n", arr[5]);
```

问题：

```text
arr[5] 越界。
合法下标只有 0 到 4。
```

正确：

```c
printf("%d\n", arr[4]);
```

---

**错误 2：循环条件写错**

```c
for (int i = 0; i <= 5; i++) {
    printf("%d\n", arr[i]);
}
```

问题：

```text
如果数组长度是 5，最大下标是 4。
i <= 5 会访问 arr[5]，发生越界。
```

正确：

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", arr[i]);
}
```

---

**错误 3：函数里用 sizeof 计算数组长度**

```c
void test(int arr[]) {
    int len = sizeof(arr) / sizeof(arr[0]);
}
```

问题：

```text
函数参数中的 arr 已经退化成指针。
sizeof(arr) 得到的是指针大小，不是数组总大小。
```

正确：

```c
void test(int arr[], int len) {
    // 使用外部传进来的 len
}
```

---

**错误 4：字符串数组空间不够**

```c
char str[5] = "Hello";
```

问题：

```text
"Hello" 需要 6 个字符空间：
'H' 'e' 'l' 'l' 'o' '\0'
```

正确：

```c
char str[6] = "Hello";
```

或者：

```c
char str[] = "Hello";
```

---

**错误 5：scanf 输入字符串不限制宽度**

危险：

```c
char name[20];
scanf("%s", name);
```

更安全：

```c
char name[20];
scanf("%19s", name);
```

说明：

```text
最多读取 19 个字符，留 1 个位置给 '\0'。
```

### ② 数组总结

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #27AE60; color: #333; margin-bottom: 10px;">
<strong>C 语言数组核心总结：</strong>
<ul>
  <li><strong>数组本质：</strong> 一段连续内存空间。</li>
  <li><strong>元素类型：</strong> 所有元素类型必须相同。</li>
  <li><strong>下标规则：</strong> 从 <code>0</code> 开始，到 <code>长度 - 1</code> 结束。</li>
  <li><strong>遍历方式：</strong> 常用 <code>for</code> 循环，条件写 <code>i &lt; len</code>。</li>
  <li><strong>数组长度：</strong> 可用 <code>sizeof(arr) / sizeof(arr[0])</code> 计算。</li>
  <li><strong>字符串：</strong> 字符数组保存，末尾必须有 <code>'\0'</code>。</li>
  <li><strong>函数传参：</strong> 数组作为参数会退化成指针，长度要单独传。</li>
</ul>
</div>

```c
void print_array(int arr[], int len) {
    for (int i = 0; i < len; i++) {
        printf("%d\n", arr[i]);
    }
}
```

> [!tip] 记忆口诀
> **数组连续一排格，下标从零开始数；长度用 sizeof，遍历小于 len；传参会变指针，长度必须另传入。**
