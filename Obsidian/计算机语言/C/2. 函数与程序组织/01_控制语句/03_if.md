# 01 | if 语句：让程序根据条件选择执行路径

> [!abstract] 核心结论
> `if` 是 C 语言中最基础的条件判断语句。它的作用是：当条件成立时执行某段代码，不成立时跳过，或者执行另一段代码。

### ① 底层原理：if 到底做了什么

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #2C3E50; color: #333; margin-bottom: 10px;">
<strong>if 的本质：</strong>
<ul>
  <li><strong>if 是分支控制语句：</strong> 根据条件真假决定程序走哪条路。</li>
  <li><strong>条件表达式会产生真假结果：</strong> 在 C 语言中，<code>0</code> 表示假，非 <code>0</code> 表示真。</li>
  <li><strong>if 控制代码是否执行：</strong> 条件为真时执行大括号内代码，条件为假时跳过。</li>
  <li><strong>底层是跳转逻辑：</strong> 编译后会变成条件判断和跳转指令。</li>
</ul>
</div>

### ② if 运行时的内存与执行理解

示例：

```c
int age = 18;

if (age >= 18) {
    printf("成年人\n");
}
```

运行时可以理解为：

```text
1. 栈区创建变量 age。
2. age 保存数值 18。
3. CPU 读取 age 的值。
4. 执行 age >= 18 的比较。
5. 比较结果为真。
6. 程序进入 if 代码块。
7. 执行 printf("成年人\n")。
```

内存示意：

```text
栈区 Stack
┌──────────────┐
│ age = 18      │
└──────────────┘

常量区 Rodata
┌──────────────┐
│ "成年人\n"    │
└──────────────┘

代码执行路径
age >= 18 为真
      ↓
执行 if 代码块
```

> [!tip] 一句话理解
> **if 就是程序的岔路口：条件为真走进去，条件为假绕过去。**

# 02 | 单分支 if：条件成立才执行

> [!abstract] 核心结论
> 单分支 `if` 只有一条判断路径：条件成立就执行，条件不成立就什么都不做。

### ① 基本语法

```c
if (条件表达式) {
    条件成立时执行的代码;
}
```

### ② 示例代码

```c
#include <stdio.h>

int main() {
    int age = 18;

    if (age >= 18) {
        printf("成年人\n");
    }

    return 0;
}
```

输出：

```text
成年人
```

### ③ 条件不成立时

```c
#include <stdio.h>

int main() {
    int age = 16;

    if (age >= 18) {
        printf("成年人\n");
    }

    printf("程序结束\n");

    return 0;
}
```

输出：

```text
程序结束
```

说明：

```text
age >= 18 为假。
所以 if 代码块被跳过。
后面的 printf 继续执行。
```

# 03 | if else：二选一执行

> [!abstract] 核心结论
> `if else` 用于二选一场景：条件成立执行 `if` 代码块，否则执行 `else` 代码块。

### ① 基本语法

```c
if (条件表达式) {
    条件成立时执行;
} else {
    条件不成立时执行;
}
```

### ② 示例代码

```c
#include <stdio.h>

int main() {
    int age = 16;

    if (age >= 18) {
        printf("成年人\n");
    } else {
        printf("未成年人\n");
    }

    return 0;
}
```

输出：

```text
未成年人
```

### ③ 执行流程

```text
判断 age >= 18
        ↓
如果为真：执行 if 代码块
        ↓
如果为假：执行 else 代码块
```

流程理解：

```text
age = 16
age >= 18 为假
        ↓
跳过 if
        ↓
执行 else
```

# 04 | if else if：多条件分支

> [!abstract] 核心结论
> `if else if else` 用于多个条件依次判断。程序会从上到下检查，一旦某个条件成立，就执行对应代码块，后面的条件不再判断。

### ① 基本语法

```c
if (条件1) {
    条件1成立时执行;
} else if (条件2) {
    条件2成立时执行;
} else if (条件3) {
    条件3成立时执行;
} else {
    所有条件都不成立时执行;
}
```

### ② 示例代码：成绩等级

```c
#include <stdio.h>

int main() {
    int score = 85;

    if (score >= 90) {
        printf("优秀\n");
    } else if (score >= 80) {
        printf("良好\n");
    } else if (score >= 60) {
        printf("及格\n");
    } else {
        printf("不及格\n");
    }

    return 0;
}
```

输出：

```text
良好
```

### ③ 执行过程

```text
score = 85

判断 score >= 90
结果为假，继续往下

判断 score >= 80
结果为真，执行“良好”

后面的 score >= 60 和 else 不再执行
```

### ④ 顺序很重要

正确写法：

```c
if (score >= 90) {
    printf("优秀\n");
} else if (score >= 80) {
    printf("良好\n");
} else if (score >= 60) {
    printf("及格\n");
}
```

错误写法：

```c
if (score >= 60) {
    printf("及格\n");
} else if (score >= 80) {
    printf("良好\n");
} else if (score >= 90) {
    printf("优秀\n");
}
```

问题：

```text
如果 score = 95，score >= 60 已经成立。
程序会直接输出“及格”。
后面的“良好”和“优秀”永远没有机会执行。
```

> [!tip] 记忆口诀
> **多条件从上往下判，谁先成立谁先干。范围判断要小心，大范围通常放后面。**

# 05 | 嵌套 if：条件里面继续判断

> [!abstract] 核心结论
> 嵌套 `if` 表示在一个条件成立后，再继续判断另一个条件。适合处理“先满足大条件，再判断小条件”的场景。

### ① 基本语法

```c
if (条件1) {
    if (条件2) {
        条件1和条件2都成立时执行;
    }
}
```

### ② 示例代码：账号和密码判断

```c
#include <stdio.h>

int main() {
    int account = 123456;
    int password = 888888;

    if (account == 123456) {
        if (password == 888888) {
            printf("登录成功\n");
        } else {
            printf("密码错误\n");
        }
    } else {
        printf("账号不存在\n");
    }

    return 0;
}
```

输出：

```text
登录成功
```

### ③ 执行流程

```text
先判断账号是否正确
        ↓
账号正确，再判断密码
        ↓
密码正确，登录成功
        ↓
密码错误，提示密码错误
        ↓
账号错误，提示账号不存在
```

### ④ 嵌套不要太深

不推荐：

```c
if (a) {
    if (b) {
        if (c) {
            if (d) {
                printf("太深了\n");
            }
        }
    }
}
```

推荐：

```c
if (a && b && c && d) {
    printf("更清晰\n");
}
```

说明：

```text
嵌套太深会降低可读性。
能用逻辑运算符合并时，可以适当合并。
```

# 06 | if 与逻辑运算符

> [!abstract] 核心结论
> `if` 经常和逻辑运算符一起使用。`&&` 表示并且，`||` 表示或者，`!` 表示取反。

### ① 逻辑与 &&

```c
#include <stdio.h>

int main() {
    int age = 20;
    int score = 80;

    if (age >= 18 && score >= 60) {
        printf("满足条件\n");
    }

    return 0;
}
```

说明：

```text
&& 两边都为真，整体才为真。
```

### ② 逻辑或 ||

```c
#include <stdio.h>

int main() {
    int vip = 0;
    int coupon = 1;

    if (vip || coupon) {
        printf("可以优惠\n");
    }

    return 0;
}
```

说明：

```text
|| 只要一边为真，整体就为真。
```

### ③ 逻辑非 !

```c
#include <stdio.h>

int main() {
    int is_login = 0;

    if (!is_login) {
        printf("请先登录\n");
    }

    return 0;
}
```

输出：

```text
请先登录
```

### ④ 短路求值

```c
int a = 0;
int b = 10;

if (a != 0 && b / a > 1) {
    printf("成立\n");
}
```

说明：

```text
&& 左边为假时，右边不会执行。
所以不会发生 b / a 除以 0。
```

再看：

```c
int vip = 1;
int count = 0;

if (vip || count++ > 0) {
    printf("通过\n");
}

printf("count = %d\n", count);
```

输出：

```text
通过
count = 0
```

说明：

```text
|| 左边为真时，右边不会执行。
所以 count++ 没有执行。
```

> [!tip] 记忆口诀
> **与要全真，或要一真；非会反转，短路偷懒。**

# 07 | if 与 scanf 输入判断

> [!abstract] 核心结论
> `if` 常用于判断用户输入是否合法。配合 `scanf` 的返回值，可以判断输入是否成功。

### ① 判断输入是否成功

```c
#include <stdio.h>

int main() {
    int age;

    printf("请输入年龄：");

    if (scanf("%d", &age) == 1) {
        printf("输入成功：%d\n", age);
    } else {
        printf("输入失败\n");
    }

    return 0;
}
```

说明：

```text
scanf 返回成功读取的变量个数。
如果成功读取一个整数，返回值就是 1。
```

### ② 判断范围是否合法

```c
#include <stdio.h>

int main() {
    int age;

    printf("请输入年龄：");
    scanf("%d", &age);

    if (age >= 0 && age <= 120) {
        printf("年龄合法\n");
    } else {
        printf("年龄不合法\n");
    }

    return 0;
}
```

### ③ 判断奇偶

```c
#include <stdio.h>

int main() {
    int num;

    printf("请输入一个整数：");
    scanf("%d", &num);

    if (num % 2 == 0) {
        printf("偶数\n");
    } else {
        printf("奇数\n");
    }

    return 0;
}
```

# 08 | if 与三目运算符

> [!abstract] 核心结论
> 简单的 `if else` 可以用三目运算符 `?:` 简化，但复杂逻辑仍然推荐使用 `if else`，可读性更好。

### ① if else 写法

```c
int a = 10;
int b = 20;
int max;

if (a > b) {
    max = a;
} else {
    max = b;
}
```

### ② 三目运算符写法

```c
int a = 10;
int b = 20;

int max = a > b ? a : b;
```

理解：

```text
条件 ? 条件成立的结果 : 条件不成立的结果
```

### ③ 适合三目运算符的场景

```c
printf("%s\n", age >= 18 ? "成年人" : "未成年人");
```

说明：

```text
简单二选一适合三目运算符。
复杂多分支仍然推荐 if else。
```

# 09 | if 常见错误

> [!abstract] 核心结论
> `if` 最常见的错误包括：把 `=` 当成 `==`、条件后面误加分号、范围判断写错、浮点数直接比较、else 匹配错误。

### ① 错误 1：把赋值当比较

错误写法：

```c
int a = 5;

if (a = 10) {
    printf("成立\n");
}
```

问题：

```text
a = 10 是赋值。
赋值表达式结果是 10。
非 0 为真，所以 if 一定成立。
```

正确写法：

```c
if (a == 10) {
    printf("成立\n");
}
```

---

### ② 错误 2：if 后面误加分号

错误写法：

```c
int age = 16;

if (age >= 18);
{
    printf("成年人\n");
}
```

问题：

```text
if 后面的分号表示空语句。
后面的大括号已经不受 if 控制。
所以 printf 总会执行。
```

正确写法：

```c
if (age >= 18) {
    printf("成年人\n");
}
```

---

### ③ 错误 3：范围判断写错

错误写法：

```c
if (60 <= score <= 100) {
    printf("合法成绩\n");
}
```

问题：

```text
C 语言不会这样判断连续范围。
60 <= score 会先得到 0 或 1。
然后再拿 0 或 1 去和 100 比较。
结果几乎总是真。
```

正确写法：

```c
if (score >= 60 && score <= 100) {
    printf("合法成绩\n");
}
```

---

### ④ 错误 4：浮点数直接比较相等

不推荐：

```c
double x = 0.1 + 0.2;

if (x == 0.3) {
    printf("相等\n");
}
```

原因：

```text
浮点数在计算机中可能存在精度误差。
直接用 == 判断可能失败。
```

推荐：

```c
#include <math.h>

if (fabs(x - 0.3) < 1e-6) {
    printf("近似相等\n");
}
```

---

### ⑤ 错误 5：else 匹配错误

```c
if (a > 0)
    if (b > 0)
        printf("a 和 b 都大于 0\n");
else
    printf("这是谁的 else？\n");
```

说明：

```text
else 默认匹配最近的未匹配 if。
为了避免歧义，推荐永远加大括号。
```

推荐写法：

```c
if (a > 0) {
    if (b > 0) {
        printf("a 和 b 都大于 0\n");
    } else {
        printf("b 不大于 0\n");
    }
}
```

# 10 | if 总结

> [!abstract] 核心结论
> `if` 是程序产生分支的核心工具。它让程序不再只会从上到下执行，而是能根据条件选择不同路径。

### ① if 核心总结

<div style="background-color: #F7F9FC; padding: 15px; border-radius: 8px; border-left: 5px solid #27AE60; color: #333; margin-bottom: 10px;">
<strong>if 语句核心总结：</strong>
<ul>
  <li><strong>if：</strong> 条件成立才执行。</li>
  <li><strong>if else：</strong> 二选一执行。</li>
  <li><strong>if else if：</strong> 多条件依次判断。</li>
  <li><strong>嵌套 if：</strong> 条件里面继续判断条件。</li>
  <li><strong>条件真假：</strong> <code>0</code> 为假，非 <code>0</code> 为真。</li>
  <li><strong>范围判断：</strong> 使用 <code>&amp;&amp;</code> 连接条件。</li>
  <li><strong>建议：</strong> 即使只有一行代码，也尽量写大括号。</li>
</ul>
</div>

### ② if 模板速查

**单分支：**

```c
if (条件) {
    执行代码;
}
```

**双分支：**

```c
if (条件) {
    条件成立执行;
} else {
    条件不成立执行;
}
```

**多分支：**

```c
if (条件1) {
    执行1;
} else if (条件2) {
    执行2;
} else {
    默认执行;
}
```

**范围判断：**

```c
if (score >= 0 && score <= 100) {
    printf("合法成绩\n");
}
```

> [!tip] 记忆口诀
> **if 看真假，真进假跳；else 管否则，多分支从上往下找；一个等号是赋值，两个等号才比较。**
